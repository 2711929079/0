# 玻瑞亚斯与流浪骑士

（前见#安德留斯回到挪德卡莱）来自一个覆灭的勇士之国的无名流浪骑士旅经挪德卡莱，邀王狼一同踏上归途，见证记忆中已然模糊的国土。温柔的狼不愿回绝人的恳盼，割下一半的灵魂与骑士作伴（后续见蒙德#玻瑞亚斯与无名骑士「北风」至冬#挪德卡莱·猎魔骑士）[31]。
六七百年前菲林斯诞生苍焰妖精克里洛·楚德米洛维奇·菲林斯诞生于距今六七百年[6]。在传说中，克里洛原本是冰原上引渡亡魂的一缕蓝火[32]（后续见#菲林斯沉睡）。
