# 社奉行-神里家

继承稻妻城中至为尊崇的三家名门之一——神里家族的，是一对兄妹。其中，哥哥神里绫人出任「家主」一职，掌管政务，妹妹神里绫华贵为「公主」，平日主理家族内外事宜。由于神里绫华常出现在社交场合，与民间交集也较多。因此，更被人们所熟悉的她反而获得了高于兄长的名望，被雅称为「白鹭公主」。
