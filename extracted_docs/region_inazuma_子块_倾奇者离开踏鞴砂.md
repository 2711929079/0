# 倾奇者离开踏鞴砂

400年前，博士奉丑角命令前往踏鞴砂制造混乱嵌入稻妻的运转。原本的目的只是毁掉踏鞴砂，让对稻妻极为重要的锻造业没落。博士伪装成了枫丹的技师埃舍尔提供新的锻造技术，但这个技术随着时间过去会导致祟神之力肆虐毁掉整个锻造业的核心踏鞴砂。然而刀匠们都不知道毁掉踏鞴砂的力量是来源自博士提供的锻造技术。「我这么怪异，他们却笨的看不清真相，真是可惜。」—《倾落伽蓝》 博士易容成埃舍尔进入踏鞴砂时的内心独白。地脉树有各种功能，比如当年蒙德四处皆大雪，如今龙脊雪山地区得以因忍冬之树庇佑而苍绿葱翠。再比如帝君用伏龙树镇压若陀龙王。而踏鞴砂的地脉树是雳木，如今只剩下残枝断干，但也能抵御踏鞴砂过量的雷元素。因此其剩断干的原因可能是愚人众为了确保引发踏鞴砂灾变时灾变更严重而为之。如今世界任务医樱中愚人众也曾指使盗宝团破坏踏鞴砂周围雷樱树。破坏踏鞴砂的计划一切顺利，然而他在踏鞴砂发现一个拥有人类情感的人偶。博士对人做了无数实验，而这个与众不同的实验素材引起博士注意。博士不顾原本的计划，打造了能净化祟神之力的装置。踏鞴砂最高负责人，同时也是那个人偶的朋友丹羽久秀发现了博士的阴谋前往与他对质。他认为博士的计划是制造踏鞴砂的危机，然后自己再开发净化装置让别人带着装置进入炉心赴死，以此净化炉心。而成功开发净化装置的埃舍尔能因此在锻造业获得更好的名声地位什么的。丹羽认为博士为此制造灾难，他认了，他会带着装置勇敢赴死，但他认为博士的目的不只是这样，一定有更深层的阴谋。然后博士直接对丹羽捅了一刀，在丹羽咽气前告知丹羽他完全猜错自己的计划了，那个装置根本不是为他准备的。「你要是乐意，就当我是个怪物，是个鬼魅好了。这样你就算死，也不是因为愚蠢而被人所害，只不过是输给了无法超越之物。…丹羽大人？…哎呀，已经咽气了啊。好可惜。」之后博士找到了外出求援后归来的人偶，谎称踏鞴砂出了事情身为最高负责人的丹羽一定要负责，但他已经畏罪潜逃了。人偶听到好友畏罪潜逃虽然伤心气愤，但还是自愿带着装置冒死进入炉心净化了祟神之力。之后人偶出于好奇询问埃舍尔，为什么让踏鞴砂所有人束手无策的祟神之力能被净化装置净化掉，想明白装置的原理构造，而埃舍尔打开装置内部，取出了一颗血淋淋的心脏。人偶虽然拥有情感和自我意识，但他一直认为自己不是真正的人类。他想融入踏鞴砂的人们，像正常人一样能经历生老病死，他想成为真正的人。而他认为人偶与人最大的区别是人偶没有心脏，因此想获得一颗心。埃舍尔取出心脏后告诉他，保护他进入炉心的装置是用这个无辜刀匠的心脏做成的。他谎称丹羽携家眷潜逃时杀死自己的刀匠助手，取出他的心脏制成净化装置，把他留给人偶让人偶代替踏鞴砂的人赴死。而他之所以选择人偶，是因为他根本不是人类，而是人偶，就算死了也也没人会在乎。就像用来耕田的工具一样，被制造出来就是为了用来耕作，不能用了就丢弃就好了。人偶被制造出来就只是个工具，是供人使用的物品，物尽其用后就可以丢弃了。心思单纯的人偶听信了埃舍尔的谎言。人偶崩溃了，他没想到那个一直陪伴他，告诉他「我倒觉得你也是人类喔」的丹羽从来都只是把他当成利用道具。让他待在踏鞴砂教自己锻刀，根本不是让无家可归的他有个容身之处，只是为了增加人手。而现在对踏鞴砂的人来说，他只是个失去价值的工具而已。他不能接受自己想要得到的心居然用这么残忍的方式出现在自己眼前，认为被丹羽恶心到的人偶，愤而离开踏鞴砂。
然而人偶不知道的是，丹羽让他待在踏鞴砂只是为了让他有个家。他没有畏罪潜逃而是被杀害，甚至死前他正打算前往炉心赴死。踏鞴砂的所有人都把人偶当作真正的人看待，把他当成朋友甚至是家人看待。丹羽得知博士的计划，在「博士」面前咽气前，挂念的也不是自己的家人，而是那位在人偶朋友。「…哈哈…哈…咳！要将我的心脏…给他…就告诉他，长正大人…也好，我也…好，都将他…视为…我们的一份子。他不必做任何事来证明自己，人与人…不会只有利用。唯有你这样的…才会如此…」
—主线间章第三幕 倾落伽蓝博士想试验的，只是人偶在经历这样的背叛会有什么反应，而博士完成了实验。之后博士将人偶举荐给丑角，使丑角号召他加入愚人众。而博士这么做的目的是以后就能随时对人偶进行各种实验。[46]因为倾奇者先前的求援而派出的援兵赶到踏鞴砂时，当地并不是一片惨状，相反，大部分人甚至不知道发生了什么。只听机械师埃舍尔说，最高负责人丹羽自知失职携家人畏罪潜逃，倾奇者代替丹羽进入核心区，关闭了大炉。八重得知倾奇者与潜逃的丹羽本是好友，猜到他心情沉重，不愿在此时打扰他，只差人将那枚羽毛交还过去。[64]（流浪者角色PV中他离开踏鞴砂后身上带着金羽，因此他的确收到八重派人交还的金羽了[70]）
折翼的鸟雀离开踏鞴砂后，倾奇者在搭浪船来到的清籁岛[71]海边某处的小屋里再度遇到了先前自己去天守阁求援时遇过的孩童。年幼的男孩体弱多病，独自居住在漏风的旧屋里。倾奇者从破旧的木门缝隙中看到他那张总是沾着尘土的脸，胸口莫名一阵绞痛，似乎某些古老的感受再次被唤醒过来。为此，他留在木屋里照顾那个生病的孩子，为他找来瓜果与饮用水，替他擦去脸上的脏污。一连数日，孩子的父母都没回来。倾奇者后来才知道，那对夫妻也是踏鞴砂的工人。这家人本可以过上平凡幸福的生活，夫妻俩却在工作中因祟神之力沾染了怪病，时常咳血。不再归来，意味着他们或许已在某处悄无声息地死去。之后男孩被邻居收养，但他的邻居大概率也是死了。孩子的名字并不重要。他有另一个身份：倾奇者的朋友，以及家人。他们交换彼此的出生信息，约定要在那间破旧小屋里一起生活下去。作为友情的见证，倾奇者也带他到借景之馆看自己曾居住过的房间。火红的枫叶，破旧的窗棂…一切都像从前一样。他想他应该永远不会再回到这里，就像他不曾料到孩童病逝也只需要一夜。而一夜时间，根本只够倾奇者出门找些食物，取些别人不要的家具而已。即便经历了那么多变故，在那时的他心里，人之故去也不是一瞬就能完成的事。「一瞬」能留给人的只有疼痛。震惊之余，他感到无比愤怒——他又是孤单一人，那不就是被遗弃了吗？再一次、再一次、再一次！地上那具幼小的躯体蜷缩着，如同大团花瓣聚在一起，被血染红了一角。鲜红的血像极了枫叶…也像极了烈火。
那天夜里，海边燃起了熊熊火光。小男孩的死成为压垮人偶的最后一根稻草，倾奇者烧掉木屋，带上从小屋里找来的旧草帽，放弃人类的情感，踏上了漫长的旅途。他四处游荡，不知该哪里去。沿途他遇见更多人，却不再将他们之中任何一个视作同伴。[72]
