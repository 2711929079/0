# 九条阵屋

幕府军营在神无冢的驻扎之处。信仰雷电将军的鸣神子民，与信仰海祇大御神的珊瑚宫，双方之间存在着永远无法化解的矛盾。而这一矛盾，因眼狩令的施行更加激化。为迎战珊瑚宫的叛军，天领奉行率幕府军驻扎于此。与珊瑚宫军相比，幕府军人员充足，军备精良，制式统一，似是胜券在握。
