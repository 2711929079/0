# 黄金

百年前的坎瑞亚已会使用炼金技术，其中的炼金术士「黄金」莱茵多特在宫廷内有极高的声望。
她是魔女会的一员，[20]创作了许多炼金生命。有将其作品分层的习惯，目前已知黄金的作品分为：淋溶层、腐殖层、白垩层。
曾经创作一位名为杜林的魔龙，并在执行创造原初之人的计划时创造一个失败品，作为失败品，莱茵多特将他喂给魔龙杜林。
后来，她创造了“原初之人”阿贝多，与杜林一样是莱茵多特最为满意的造物。[21]除此之外，兽境猎犬、兽境幼兽、黄金王兽以及厄里那斯皆为她的造物。巴窟纳瓦为黄金失控造物，就算只剩下肉块也会聚会复苏，无法杀死，就连深渊教团都对他所知甚少，曾吞食莱茵多特部分躯体[22]。
莱因多特最初的造物莱因多特最初的造物，众深渊魔物名录中的长兄，是一头独角兽。但它没有完成自己的使命，到达哲学之彼岸[23]。
