# 坠星

疑似天钉（原文本写作晨星）降落[21]，夏宫灵囿因此深埋地底，被称为仲夏庭园。[22][23]疑似天钉的东西（原文本写作晨星）坠入此间大地后，砸出了星落湖。千风神殿周围地区因此被称为坠星山谷。[21]
