# 玻瑞亚斯回到挪德卡莱

（前见#狼与仙灵）高塔崩毁，烈风止息，曾与龙卷之魔神争夺执政权柄的王狼，自觉无法描绘人类的未来，便将昔日庇护的族群托付于新的神明（巴巴托斯），默默回到了苦寒的北境。旧主早已离去的宫殿间，狼自此阖上了睡眼[31]（后续见#玻瑞亚斯与流浪骑士）。
迁都至冬堡白皇自终北的故墟（黄金城）归来后沉默不语，于白桦林深处思索良久，最终向一度被他拒绝的人类敞开了大门。皇帝陛下以其无限的恩泽与仁慈，赦恕人类为臣民，并迁都至冬堡[8]。随着至冬堡的高炉一同建立的，乃是在冻土尽头如迷宫般复杂的宫殿，最智慧的人类与妖精们被召集至此，伟大狂想结成的胚胎亦于此萌芽[26]。
