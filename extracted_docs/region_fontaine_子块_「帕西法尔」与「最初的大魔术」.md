# 「帕西法尔」与「最初的大魔术」

曾经的小贼姐妹受科培琉司大师的剧作的影响，以剧中侠盗「帕西法尔」和蓝宝石美女「约瑟芬」为名，将盗窃的手法应用在魔术表演上，最终成为了广受枫丹上下层欢迎的知名魔术师。但是枫丹上下层的矛盾难以调和，最终在「灰河整肃」中，「帕西法尔」身亡，「约瑟芬」继承了「帕西法尔」的名号，并设计出了属于她们的「最初的大魔术」。「我是大魔术师——『伟大者』帕西法尔！」「你们将要目睹的是不曾想象的梦幻世界！」「没有贵族没有王，没有指着你们的刀剑！」「没有生来的富贵，没有无法摆脱的贫困！」[40]
「灰河渡手」与爱德华多在过去曾经被当作武器使用的趁手废弃铜管，曾经在一些人当中享有盛名或是臭名昭著。[41]雅各布的生父爱德华多•贝克与因城市优化而被迫进入下水道「灰河」生活的其他人们没有什么不同最初他只是拿着捡到的铜管驱逐鳄鱼，再后来是保护商贩店家免于强人帮派的侵扰勒索、调解灰河的「同胞」之间的纷争，为新婚者祝福。[41]除此之外，爱德华多在灰河经营酒馆，「帕西法尔」姐妹金盆洗手后的魔术表演就在他家的酒馆出道表演。[40]就这样，爱德华多顺理成章的成为了「灰河」实际上的领袖，团结领导着着枫丹的灰色地带（刺玫会的前身），他惯用的铜管被众多朋友亲切的称为「渡手」[41]。最终爱德华多因在「灰河整肃」中组织反抗被捕，流放往沙漠。
