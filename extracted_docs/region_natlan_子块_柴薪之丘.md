# 柴薪之丘

又被本地生活的部族人称为「烟垄」的矮小山麓，得到这样的两个名字，据说是因为图兰大火山之底有条地下岩浆的支脉流经此地，山谷之间因此常有烟雾升起。此处的温度对于常与燃素打交道的部族人来说都有些高了，但这或许也是离群索居的异种翼龙将之选为居所的原因。
