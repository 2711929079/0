# 原神元素反应机制详解

《原神》的元素反应是战斗系统的核心机制，七种元素之间可以产生丰富的反应效果：

1. 融化（火+冰）：
   - 火元素触发：伤害提升200%
   - 冰元素触发：伤害提升150%
   - 应用场景：冰C配火辅助或火C配冰辅助

2. 蒸发（火+水）：
   - 火元素触发：伤害提升150%
   - 水元素触发：伤害提升200%
   - 应用场景：火C配水辅助或水C配火辅助

3. 超导（冰+雷）：
   - 效果：造成冰元素伤害，降低敌人物理抗性40%
   - 应用场景：物理输出队伍

4. 感电（雷+水）：
   - 效果：持续造成雷元素伤害，可在多个敌人之间传导
   - 应用场景：群攻战斗

5. 超载（火+雷）：
   - 效果：造成爆炸伤害，可击飞敌人
   - 应用场景：需要控制敌人时

6. 冻结（水+冰）：
   - 效果：冻结敌人，使其无法行动
   - 应用场景：需要控制敌人时，可被重击击碎造成额外伤害

7. 扩散（风+其他元素）：
   - 效果：扩散其他元素效果，对周围敌人造成对应元素伤害
   - 应用场景：温迪、万叶等风元素角色

8. 激化（草+雷）：
   - 效果：触发激化状态，后续草元素和雷元素伤害提升
   - 应用场景：纳西妲、赛诺等草雷队伍

9. 绽放（草+水）：
   - 效果：产生草原核，接触火元素触发"燃烧"，接触雷元素触发"超激化"
   - 应用场景：草元素队伍

元素精通是提升元素反应伤害的关键属性，合理搭配元素队伍和利用元素反应是提升战斗力的核心策略。
