# 几十年前

斯坦利两位蒙德冒险家：汉斯亚齐博尔德与斯坦利来到了烬寂海探险。斯坦利在烬寂海时，左手和腰部被连同类都会吞食的魔物啃掉，在渊上帮助下暂时止血。原本渊上化名诺奇特，在救活斯坦利后说自己赶时间要离开了，但傲娇的他其实还是跟在斯坦利后头之后斯坦利遭到危险时，然后再换个长相并用假名「娜娜丝」再救了他[96]。斯坦利为救汉斯而死，如愿在烬寂海探索到最后一刻，汉斯愧对于友人，想要让人们一直记住他的事迹，于是回到蒙德后一直冒用斯坦利的名字来宣传探险事迹[128]。
