# 阿贝多

莱因多特教授了阿贝多炼金术知识，但师徒二人在地底找到一件名为「纳贝里士之心」的圣遗物后莱茵多特失踪，临走前留下吩咐让阿贝多到蒙德找自己的老友艾莉丝。后来她吞噬生之执政纳贝留士的心[71]。阿贝多自此被艾莉丝收养，并借助她的关系进入西风骑士团，担任首席炼金术士兼调查小队队长。[72]莱因多特吞噬纳贝留士之心后成为生之执政，且与其余影子关系好。
凯亚最后的希望为了“某个古老的阴谋”，深渊教团创始者克洛达尔·亚尔伯里奇的后人凯亚·亚尔伯里奇的父亲，[1]将其子抛弃在蒙德，让他成为坎瑞亚安插在蒙德的间谍。
“这是你的机会，你是我们最后的希望”
亲生父亲捏紧凯亚单薄的肩头，视线越过他，望向更远的地方。
地平线的另一头，是父子俩遥远的故乡坎瑞亚。
凯亚从没忘记那混杂着憎恨与希冀的眼神。[73]
