# 铁钱

「铁钱」是艾维雷勒安发行的货币。他相信，在没有界限的世界里，在无人忍受饥寒的乐园里，神明所赐予、所掌控的金钱，不该成为人与人互助的基础。「人」应当是金钱之主，而非其仆役。即使无法成为黄金之心的主人，也该让所有人平等掌控本就属于所有人的财富。由「人」赋予意义的铁钱，在他死后，很快也被他相信的「人」所遗忘，成为了无用的废铁[40]。而「最初的铁钱」，不过是在那灯光所未及的冬夜，躲避着新大人们的视线、偷偷溜出学院的二人，为扮演侠盗的游戏而刻下的代币罢了。
盗取「三宝磨」霍德望来到挪德卡莱后，利用雷德·米勒为受欺凌和侮辱的人们创造「乐园」的理想，劝诱其跟从自己[33]，成为自己的弟子，并用谎言让他替自己取得「三宝磨」[34]。流行于那夏镇的故事里，大盗雷德·米勒被围困之时，正是靠着霜月的圣女爱莉厄送给他作为定情信物的符石隐去身形，逃脱了追捕。霜月之子的追随者坚决否认这样的说法，将其斥为胡言乱语，毕竟现在的寒涌石也没有这样神奇的能力[15]。为免除三宝磨引来众人抢夺的后患，当雷德米勒与追随他的爱莉厄前往盗取三宝磨时，爱依菈默许了[42]。从对话来看，此时的爱依拉应仍屈居于娄维娅的阴影下，并对姐姐的离开有诸多埋怨。
