# 三年前·苦壑崖封印

在三年前，苦壑崖封印发生异动，执灯人前往调查时遇袭，分队的成员在深渊影响下自相残杀，损失惨重。唯二清醒的成员中，队长奥尔松强令叶洛亚带上受伤失去意识的伊瓦尔撤退求援，自己留下断后。所封印魔物「靡穷之孽」的突破最终失败。该事件后，执灯长尼基塔判断暂时无法彻底解决问题，自此设下观察哨，将苦壑崖一带划为禁区。[34]
挪德卡莱·清理首领挪德卡莱曾经被诸多「首领」控制（有宝藏猎人，可能也有其他人士），他们在无数次的密会里划定了地盘，就像某些失乡者们想要逃离的暴君与僭主一般，将这片本该属于所有人的乐园无声地据为己有，尔后派出自己的爪牙，肆意施行暴虐。奈芙尔来到挪德卡莱后清理了这些地头蛇，并在日后建立了「秘闻馆」，接手了「北大陆情报网」。[131]
