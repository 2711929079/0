# 蒙德.芬德尼尔

在冻结雪山的「长钉」降下时，风雪遍天。天降之柱崩裂为三。[21]当冻结世界的长钉蓦然降下，连这棵树也被其余波粉碎、冻结时，芬德尼尔的公主取走了最完整的枝，想为荫蔽一国的树接续生命。但在最后，嫁接的生命始终没能活下来。刀刃般冰冷的风雪，最终将月光遮蔽了…[42]
蒙德.塞西利亚古国疑似天钉的东西（原文本写作晨星）降落在坠星山谷的星落湖，塞西利亚的古国在此后覆灭。[43]建立塞西利亚花苗圃的古国覆灭后，只剩摘星崖存在这种花的踪影。夏宫灵囿深埋地底，被称为仲夏庭园。
