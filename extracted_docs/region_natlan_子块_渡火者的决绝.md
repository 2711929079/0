# 渡火者的决绝

火红的花朵，仿佛闪光灼灼的玛瑙石一般。这浴火的花，被火中通行的智者佩在胸前。在渡火的智者（渊上）生命的最末，他对人们说道：「这是受烈焰焚烧而盛开的花。若我未被化作烟灰」「热浪与黑烟中必然会有这朵花闪烁着余火的光辉」后来，人们追踪着明灭的光芒，来到了烬寂海的边缘，却发现智者已经不知所踪，而花依旧盛放在余烬之中。[138]
