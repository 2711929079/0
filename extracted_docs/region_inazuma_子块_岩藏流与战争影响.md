# 岩藏流与战争影响

岩藏流的现任掌门岩藏光造忠心于幕府，主动上交了自己的神之眼，并带着徒弟去到幕府军中效力。他的两名大弟子因反对这一举动而被逐出师门，其他弟子也因为忍受不了军中氛围而落草为寇。曾经著名的岩藏流最终因战争而解散了。绀田村曾有着繁荣的印染产业，但在战争期间因许多年轻人都被征召入伍而导致村子陷入萧条。
祟神遗恨踏鞴砂关停由于幕府军和反抗军的交战，踏鞴砂受到了影响，发生了爆炸，祟神力量外泄，工人和居民逃离。负责维护运作的枫丹工程师泽维尔为防止祟神力量外泄影响，停止了御影炉心的运作，并开启了保护罩，留下善后。[82]
