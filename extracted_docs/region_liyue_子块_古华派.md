# 古华派

一位出身沈玉谷名为古华的游侠，悟得了裁雨法、刺明法、生克法三大剑术。但此剑术需搭配神之眼使用。古华曾在练习中受伤，在养伤的日子里，一位名叫“伯玄”的好友经常烹饪羊烩鲤来拜访他，古华十分喜欢这道菜，于是用“生克”法向伯玄换取菜谱。古华死后，白玄大师创立古华团，后来演变成古华派。古华团要求门下弟子要像古华一样，用学习到的功法去行侠仗义。古华派的前三代掌门武理通达，并著有《诸武集成校笺》，方便后人传承学习。辉煌时期，「古华派」曾以枪剑闻名璃月。古华团第11 任掌门，人称 “贯索剑”的“赢妥”觉得古华功法太难掌握不便推广，于是就自创了一条走捷径的入门技巧。《诸武集成校笺》也被人留下了错误的备注，武学门派「古华派」因此式微。古华曾仗剑独行于璃月四方，四处斩妖除魔、惩奸除恶，但他并没有开山立派的想法。在古华死后，其友人伯玄学会了他的武功精华，并以他的名号建立了古华团，将古华生前所居之地建为门派祖庭：灵濛堂。受古华荫蔽的百姓将“灵濛堂”称作“王山厅”，以表恩敬。古华的武功虽然厉害，但是对于没有神之眼的人而言非常难学，第11代掌门羸旻的觉得这些功法太难掌握，不适合推广，便稍作修改，研究出一套全新的入门技巧。虽然是方便了门人学习，可是简化后的武功其威力也大打折扣，而且门人也不愿意花费几十年时间去专研武功中的精髓，导致古华团不复往日威名。古华团因势力低迷、人心流散，先是凋零成了区区百人的古华堂，再后来又衰败成了如今门可罗雀的古华派。相传，其有三大镇派秘术——枪法「刺明法」、剑法「裁雨法」，以及枪剑双绝的「生克法」。此三绝之术随代代传承不断精进，时光推移，三绝技巧渐臻化境，威力却不断削弱，寻致古华派势力低迷，人心流散…三绝更是如尘埃中的宝库，静候一位继承者[137][138][139]。
长生几百年前的璃月曾瘟疫肆虐，一名药师为了救济世人而来到沉玉谷寻找传说中的药君，但是此时的药君早己因耗尽力量而变成了一条虚弱的白蛇，她也失去了一些曾经的记忆，化名长生。长生告诉药师只要与自己签订契约，就能获得医治百病的术法：珥蛇拖龙法，但代价是会变得短寿，药师答应与长生建立契约，而与药师分享生机的长生得以存活。借助长生所授的救病之法，药师救下了许多患者，但自己也因过度透支生机而英年早逝。在临终前，药师长生与契约托付给一位自愿继承的病患，这位继承者亦如同药师生前那般悬壶济世。在他死后，长生也先后传给了许多心性至纯的医者，他们无不救济苍生，却又无不英年早逝[77]。
