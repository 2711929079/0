# 七贤僧

七贤僧为七位深得赤王信赖的下属与盟友。实际上他们大多是败给赤王的魔神，然后投靠了赤王。他们分别为公羊之王赫里沙夫、鳄之王索贝克、鹮之王图特、圣者贝努、圣者谢斯芬赫、阿赫·阿佩普和赫曼努比斯。赫曼努比斯被称为「众贤之至贤」，在赤王文明崩溃后曾来到图莱杜拉，并建立蓝宝石城，有一应传说故事流传至今。为保护其身后遗迹的信徒组建了缄默之殿，世代守卫于图莱杜拉城曾经建城之处。鳄之王和䴉之王分别被后世称为「众统帅之统帅」、「众书记之书记」。须弥历史载有鳄之王和䴉之王论法的故事。鹮之王是掌书，葬火之战前曾是是星与月的秘使（月使），与兹白等月使在月亮上共事[42]，也曾守护过图莱杜拉，发明须弥沙漠的古文字[38]。虽然星与夜的秘史是正史[16]，但其详情不可考，据图特本人夸耀說自己之前是如何在三个月亮上辗转工作，只为引导地上的愚钝凡众，如何为众生编织合理的命运之线，让地上的万事万物轮回萌发。在来到须弥，被赤王慧眼识英后，又是如何被擢升至伟大的「赤王七柱」之首的，之后还与赤王把酒言欢，与花神一同栽种绿植…[43]但佞臣能言善骗，图特尤其爱讲谎话，可信度较低（不过奈芙尔角色故事 月之轮和奔霄送玉轮第四幕提及有部分是真话，包括担任月使）。公羊之王被称为「众相之相」，是赤王最欣赏的一柱，以过人的智慧闻名后世。由于野心膨胀，曾对赤王做出僭越之举。圣者贝努被称为「至明」，圣者谢斯芬赫被称为「至圣」，根据须弥典籍记载，分别用于指代「人的灵魂」和「镇灵」。出于对人类和魔神的排斥态度，龙之王阿佩普是唯一拒绝接受称号、不曾列席的七柱。[39]鳄之王索贝克是赤王麾下的最高统帅，七柱中司掌战争的王。在小说的描述中，鳄之王从前也是一位少年，并且有着苍青色的面容，不知是否可信。图特是赤王身边负责记录的书记官。鹮之王与鳄之王两人素来争论不休，大到世界法则，小到日常琐事，一言不合就开始争辩。赫里沙夫是宰相负责协助赤王统领整个沙漠王国，拥有极大的权力。贝努是守护在赤王左右的二位圣兽之一，代表着赤王的崇高灵魂。相传贝努能够庇护人们不受风沙伤害，反之也能唤起狂躁的风沙。谢斯芬赫是守护在赤王左右的二位圣兽之一，代表着赤王的无上意志。
鳄之王与鹮之王鳄之王与鹮之王素来争论不休，而那日他们争论着事物的转变。他们也喜欢下棋，常常以此斗嘴，据图特所说鳄之王和羊之王棋品不好，最后都会怪自己出老千，不知是否可信[38]。
鳄之王寻来了身怀妙术的人。此人被留住此处的纯水精灵祝福，任何液体经过他的手，都会自行转变，直到比例恰当，化为美酒琼浆。
鹮之王则寻来了另一个异士。他误触了古老魔神的尸骸与坟冢，因此被诅咒，双手触碰之物会逐渐变成纯银——毕竟今日来看，变化黄金与摩拉乃是仅属贵金之神的伟业。
两人打赌究竟哪种不可思议之变化会最终胜出。他们也让赫曼努比斯——也就是最后一位要提到的七贤——来猜测结果。
赫曼努比斯是沙漠贤人、祭司集团的上首。因此他既是勇士也是贤者。因人类中流传着名为炼金术的学问，他又深谙此道，让他来判断最为合理。「两种伟力针锋相对，犹如利矛坚盾。」赫曼努比斯说，「他们会各自退让一步，此乃均衡之理。」最后的结果是杯子化成了纯银，含着砂砾的水变成了落着银粒的美酒。[39]
