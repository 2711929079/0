# 日月的元年

大日御舆目盲之年贤人阿布拉克他被开启了神智，他展示了从手中发出光的奇迹。先祖们以他为首领，开始建设「赫利俄斯」。[3]目明之年「赫利俄斯」——太阳的神车，终于落成。白夜到来，常夜消散。日月的纪年开始了。[3]彼时，渊下宫才因为太阳的出现而得到些许喘息。因那龙嗣亲近黑暗、忌惮光明，不再肆意妄为。龙嗣横行、敛人命如草的时日终于结束，渊下宫民变得能与其相抗了。然而，异种外患没有完全根除，这人性的腌臜就已经暴露了。[5]“他们”利用民众，开始推行大日御舆崇拜。将人心寄托于虚妄死物已是愚昧至极。到了最后…甚至推举「太阳之子」作为傀儡以统治白夜之国。[6]「太阳之子，诞辰御祝，至天马之架席，共烈日之飨…」日月的二年先祖们尝试寻找归途。地表的大战应该已经结束。但是原初的那一位，第一个王座，布下了禁令。先祖们无法找到归家之路。既然是如此，那原初的那一位，应该打败了后来的第二位吧。[3]「宇宙无始无终，曾经的大地也是这样。只不过对我们来说没有意义。承载我们的土地已经不再和无始无终的永恒相连了。」——唯一的贤人，阿倍良久这么对初代太阳之子诉说。太阳之子早已准备惩处阿倍良久，这次宣他到御前问答，不过是想加以刁难，然后将他拘禁。相传，阿倍良久被常世大神启开了智慧，因此才从不见太阳的渊下宫里掘出来了光。但是太阳之子嫉妒他的才华，把他囚禁直至他寿终。然而，太阳之子们却未曾想过，若非有他造出地下的太阳，哪会有自己呢。「…天地原如鸡卵，龙蛇本就一体。」贤人阿倍良久说完这句话，随即就被埋伏的兵士们按倒。[5]傀儡不怀好意的贵族把大日御舆说成是神的产物，逮捕了阿倍良久。阿倍良久最终因为不敬神明而被监禁，没过几年就因为忧愤过多而死去，他在死后被保留了贤者的称号。贵族们为了继续掌权，决定设立「太阳之子」作为元首，由婴儿担任，扶植傀儡政权。当太阳之子满十岁后将被献祭。大日御舆落成后，渊下宫得名「白夜国」。为了反抗太阳之子，斯巴达克与他的学生发动了反抗。但斯巴达克本人被抓捕处刑，他的学生领导的反抗集团因为沉迷龙骨花凝珠（类似毒品）而失去斗志。起事失败了。日月之十年阿倍原先的副手因写作《日月前事》而被逮捕，书里记载很多白夜国之前的隐秘历史。《日月前事》后来被奥罗巴斯亲自封禁。阿布拉克故去已久。日月之前的事情已经记录得足够。若无把一切按事实记写的胆量，哪里能成为常世大神的书记呢？[3]
