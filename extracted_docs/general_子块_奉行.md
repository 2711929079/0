# 奉行

本身是日本平安时代至江户时代（794-1868）中授予武家的官职名称之一，办公住所称为“奉行所”。
本意为动词，指执行上级命令，衍生为名词执行者。
寺社奉行、勘定奉行、町奉行是江户幕府常设的三种奉行职位，也被称为“三奉行”，其中第一个就是游戏中的“社奉行”，第三个“町奉行”尽管游戏中也有出现，但不属于游戏中的“三奉行”。
