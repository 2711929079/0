# 菫鹮

高贵优雅的鸟类，生着宽阔的双翼与修长的腿。长着堇色的美丽羽毛，喜欢沉静地漫步在溪流或稻田之中。在稻妻的古歌中称之为「里梅鸟」，曾经有着非常广泛的分布，但近来似乎越来越少见了。
