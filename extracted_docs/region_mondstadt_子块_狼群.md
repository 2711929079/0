# 狼群

兽径猎犬来到蒙德，与当地狼群展开争战，[104]最后兽径猎犬们因不敌而逃往鹤观，仅几只兽径幼兽在奔狼岭蛰伏[105]。兽径幼兽破坏奔狼领北风之狼领地周围的地脉树，导致该地有许多大型断树根，且依靠地脉才能传送的传送锚点离北风狼很远，因为更里面的地脉被破坏了。考虑到杜林也是从坎瑞亚一路跑到忍冬之树，兽径猎犬和黄金王兽跑到地脉树「栖木」所在的鹤观，鹤观雾散后它们在天下人之章第二幕跑去破坏雷樱树，因此它们可能会循着地脉行动。
北风骑士的「陨落」守护之心当猛毒之龙杜林殒命在冰封之山，湛青之龙长眠于尖塔古城时，骑士鲁斯坦却在谷地中流尽了鲜血。倒下时，他所挂念的只有——「现在在远方求学的她（罗莎林），应当平安无事吧。真想再听听她唱的歌」「至少艾伦德林与罗兰无恙。她归乡时，灾厄应该已经平息了吧」[99]之后，艾伦德林因为鲁斯坦的死而辞去大团长的位置，大团长席位空缺，骑士团陷入人手不足的困境。艾伦德林的行为在风神吟唱的诗篇中被认为是「狮牙朽坏，鹰旗不扬」[106]。
