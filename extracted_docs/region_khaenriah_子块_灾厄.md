# 灾厄

五百年前，坎瑞亚决定引入深渊力量作为新的能源。但深渊能量失控，天理只得号召七神前往覆灭坎瑞亚。受深渊影响，伊尔明开始动摇世界的根基。「预言家」维瑟弗尼尔献上预言，却因预言内容而被伊尔明囚禁于大牢，并被刺瞎双眼，暂时丧失预言能力。戴因联合「贤者」海洛塔帝、「黄金」莱茵多特、「极恶骑」苏尔特洛奇、「猎月人」雷利尔劫狱救出戴因的哥哥维瑟弗尼尔，并在之后临时起意阻拦「黑王」伊尔明继续动摇世界的根基[33]。伊尔明似乎因深渊力量肉体产生畸变[42]。然而除了戴因之外的五人受到了类似的理由深渊蛊惑与放大欲望，瓜分了那些足以毁灭世界的力量，成为了超然的存在，每个人掌控的力量都能与世界相匹敌。在此之前，五人就一直持续受到深渊的诱惑[40]。被称为「黄金」的炼金士堕落为了罪人，释放自己孕育的大量漆黑魔兽。它们流淌着与此世不容的黑血，在大陆上跳踏众生、破坏一切。它们的生命是变质的，由世界外的力量赐予的。统治地下世界坎瑞亚王国的黑日王朝覆亡，灾难突破了古国城墙在大陆上散布开来。在天地间留下伤痕累累…战火飘熄，余烬落定。灾厄发生时，五大罪人没有一人出面阻止这场灾难。整个大陆都经历了一个混乱的时代。黑暗的力量扩散行进，污染所遇到的一切。那段时间里，大地上蛮族滋生、魔兽肆虐。人们的生活空间被压缩到了城墙之内，野外充满危险。以「末光之剑」戴因斯雷布为首的宫廷卫队以及白鹄骑士积极参与对抗漆黑魔兽的战争，但还是没能让坎瑞亚免于灭亡的命运。[9]后来，末代君主伊尔明失能，安弗塔斯摄政领导「救国会议」[43]，暂时统领坎瑞亚遗民。[38]被黑日推翻的赤月王朝遗民，在此时对黑日展开复仇。[44]
守护的职责坎瑞亚灾变当日，戴因斯雷布赶往皇宫之前，嘱托哈夫丹「转告所有黑蛇骑士，不论如何，守护好坎瑞亚的人民」。之后他们受到诅咒成为失去灵魂的黯色空壳。在这些已经失去理智的黑蛇骑士心中，他们依然在为守护坎瑞亚人民而战。将遗迹视为灾难中的坎瑞亚，将那些丘丘人视为正在求助的民众…这份意志如此顽强，经历了五百年时光都未被磨灭。[4]
