# 简介

被称为「极恶骑」，是坎瑞亚「五罪人」之一，犯下了「渎空之罪」，曾持有弓「居瑟诺德」（现名「贯星」）[1]。是丝柯克的师父，与戴因斯雷布是同级同学，跟着哈登学习剑术，由于哈登的卑劣行为，曾断了腿与胳膊，后来经过不断奋斗，成为精英骑士之一。他是死囚的儿子，整日和一帮不良青年混迹街头，受过不少批评和苦难，但他始终保持乐观、自信，后来结识了身患重病的坎瑞亚贵族继承人斯克弗努格并承认了对方的天赋，同他一同切磋，10岁那年，他偷来了禁忌的研究资料，治好了斯克弗努格的病，离开提瓦特时，将吞星之鲸送给了对方；是吞星之鲸的主人，传授「极恶技」，实力深不可测，由于哈登的影响，其肢体变得残缺，性格也变得极度扭曲，为了变强不择手段，力量高于整个世界，威胁整个宇宙，丝柯克角色故事提及其创立名为寰宇劫灭的组织，是宇宙内的重大威胁，在他救下丝柯克之后，就对她采用了魔鬼式训练，后来送她去了提瓦特，临走时极恶骑表示，自己有一天会夺走她拥有的一切，试图用恐惧促进她的“进化”；与「贤者」海洛塔帝、「预言家」即戴因斯雷布之兄维瑟弗尼尔、「黄金」即魔女R莱茵多特和「猎月人」雷利尔并列为“坎瑞亚五罪人”，他们曾经与戴因一起阻止了伊尔明的阴谋，在五百年前瓜分反主的深渊力量获得比肩世界的力量，但在坎瑞亚灾变的时候，这5个人没有履行救国的责任，被戴因斯雷布视为仇人，打算向其报仇。其力量是许多生灵能想像的最高顶点，因此许多生灵为了追求更强，会幻化成他的模样。如今他穿梭于各个宇宙，能随意改变容貌，通常使用盔甲在宇宙中穿行。有一众追随者在其底下拜师。中文名苏尔特洛奇英文名Surtalogi外号极恶骑汉语CV林强日语CV竹内良太所属坎瑞亚/深渊/寰宇劫灭职业坎瑞亚「五罪人」
