# 烟绯技能详细介绍

烟绯是《原神》中的火元素四星角色，璃月的律法咨询师，称号"智明无邪"。
元素战技「丹火印」：
- 释放火元素能量，对周围敌人造成火元素伤害并施加"火元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「业火枷」：
- 创造一个持续8秒的火元素领域，领域内的敌人会受到持续火元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "律法的威严"：丹火印命中敌人后，有15%概率触发"燃烧"效果。
- "火元素增幅"：烟绯的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "业火的制裁"：元素爆发持续时间内，烟绯的火元素伤害提升15%。
