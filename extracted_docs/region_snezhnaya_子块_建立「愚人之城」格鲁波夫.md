# 建立「愚人之城」格鲁波夫

第七百九十九纪的第四年，女皇让愚人众在至冬堡附近建起一座新的城塞，名叫格鲁波夫，意思是愚人之城[8]。
设立皇都评议会第七百九十九纪的第六年，女皇设立皇都评议会，处理王公缺位产生的行政问题，并指派人类约安娜·伊万诺夫娜为初任议长[8]。
