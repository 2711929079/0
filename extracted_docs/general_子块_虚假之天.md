# 虚假之天

丑角曾派遣散兵寻找降临在蒙德的陨石，其中蕴含关于虚假之天的信息[94]。在狩夜者战争中，玛薇卡使用全力轰向空中的古斯托特，余下的力量穿过爆炸击碎了局部天穹，露出了泛着红光的虚空以及漂浮在其中的苍白星体残片。天穹破损轮廓由边缘泛红的方形构成，与「天理的维系者」所使用的力量相似。这一现象未被其他国家观测到。
