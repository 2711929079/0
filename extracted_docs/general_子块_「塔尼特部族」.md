# 「塔尼特部族」

「塔尼特部族」是赤王沙漠中最古老的部族之一，是一支信仰花神的母系氏族、流浪氏族。驻扎地位于塔尼特露营地。塔尼特部族会收养、劫掠幼童，将之培育成专门忠于长老的「猎鹰」。理念：「掌握过去的人，才有可能创造未来。」秉承此理念，塔尼特一直在追求探索沙漠的过去。[12]芭别尔接管部族的主母后，为了提高自身的威望，企图彻底控制永恒绿洲。为此不惜与愚人众合作，甚至试图除掉旅行者和婕德[13]。塔尼特部族由「九长老」分享部族的最高权力。但九长老中的八个在芭别尔的算计下被一一清除，仅剩马塞拉得以幸免。马塞拉对待「猎鹰」塔德菈（他的女儿）十分严苛。世界任务比勒琪丝的哀歌中指使女儿谋害旅行者未果，逃离了塔尼特部族。目前被婕德追杀中。世界任务仇敌仿佛众水翻腾中，塔尼特部族被婕德和旅行者联手，全部屠灭。
