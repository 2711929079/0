# 花灵与人类

五百年前，花灵和荼诃的剑士、东方来的独臂贤者（那伽朱那）、坎瑞亚的残军一起对抗过漆黑兽潮。其后，一部分花灵和它们的搭档组成了最初的矫论团。矫论团曾经用具利鼓和花灵建立连结。可是这些具利鼓逐渐遗失了。矫论团曾经用仪式帮助无法返回甘露池的花灵送回甘露花海。但是相当多的仪式都在一次意外中散失了。之后，祖尔宛封闭了甘露花海，只有花灵认可的人类可以进入。
