# 申鶴

申鹤出生于驱邪世家（重云的世家）的旁系。她5岁时，母亲因病去世，父亲爱妻心切，去往远方寻求复活亡妻的办法。一年后，申鹅的父亲从至交“明俊”那里找到了一本书，书里记载了复活亡妻的方法。申鹤的父亲回到象中，向申鹤谎称一处山洞中有惊喜，申鹤走进山羽，父亲利用魔神残渣召唤出恐怖的“仙灵”，“仙灵”指出申鹤是孤辰劫煞，申鹤的父亲便決定用申鹤的生命换回自己妻子的生命，独自离开了山洞。申鹤拿起母亲道物中的一柄驱邪匕首与“仙灵”死战。数日后，留云借风真君循着邪祟的气息找到了在战斗中占据上风卻奄奄一息的申鹤，帮助她清理了魔物后，便将她带回府中修养，而后收她沟徒。这几天里，申鹤的父亲橋然醒悟，深感愧疚的他选择了自杀，而明俊得知这系列事情后一直深感愧疚。仙家收留申鹤后，削月筑阳真君为其卜卦，卦象显示她既犯孤辰，又冲劫煞。为了让申鹤安全长大又不殃及他人，仙人们为其身上施了红绳缚魂之法[146][147]。留云借风真君将“翠钿白玉梳”送给了申鹤，并告诉她：如果想斩尽凡缘，踏入仙门，就用这把梳子梳三下头发。申鹤毫不犹豫地照办了，原本的一头青丝变为了白发。[148][149]。修行了十多年的申鹤在一天心血來湖，回到了自己的故乡，发现自己儿时居住的村庄已被遗弃，父亲也早已离世[150][151]。
明蕴镇一位名叫元鸿的矿工带着一家人在瑶光滩北部的山岭间开采矿石，白手起家建立了明蕴镇。明蕴镇曾经很繁荣，是仅次于层岩巨渊和黑岩厂的采矿中心，元鸿一直被人们认为在镇子里埋藏了一份祖传宝藏，连元鸿的三个儿子都信以真，纷纷前去寻宝并相继死于意外，元鸿也因此心力交瘁，最终病逝。在元鸿死后没过几年，明蕴镇的矿脉耗尽了，矿工们都转去层岩巨渊工作，明蕴镇也从此荒废，如今只有盗宝团会光顾此地[152]。志琼就出生于明蕴鎮，也跟随大家去往层岩巨渊[153]。
