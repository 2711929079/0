# 蒂莱尔

坎瑞亚偏远村庄的少女蒂莱尔从小就憧憬「救世主」的故事，虽然因为某个路过的，被大家认为无所不知的圣人（是女性，可能就是伊斯塔露）所述的影响，大家都说「灭顶之灾终将到来」，她仍然打算寻找救世主，求助她/他拯救自己的世界。她因伊斯塔露干涉遇到了来自未来五百年后的旅行者，为了证明自己见过他/她，接过其手中飞船休眠舱钥匙（也可选择收回不给她）[28]。
深渊坎瑞亚人引入深渊力量后，其为机关供能的速率远大于阿索斯物质。[34]以阿索斯物质作为能源的技术路线，在当时以被王朝当作过时的举动。由于旧的能源无法供应高度复杂的动力系统，以阿索斯物质功能的机关工坊陆续淘汰，直到最后，王家的机关工坊“Gnade”也被迫关闭…[35]但并非所有坎瑞亚人都将国家的希望寄托在这种无法控制且不可名状的事物上。他们认为，依靠人类的智慧，也能发掘出真正的永续能源。[34][36]但坎瑞亚末代君主伊尔明没有听取以皮耶罗为首的保守派大臣的意见，反而遵从贤者海洛塔帝的嘱咐引入深渊力量。[37]但为了不要让地脉技术彻底失传，安弗塔斯元帅请求坎瑞亚王保留部分依照旧式标准制造的机体。[36]最终，坎瑞亚王答应安弗塔斯·亚尔伯里奇[38][14]，保留被须弥人称作“遗迹巨像”的大型守望机关，而如今散落各处的耕地机因为是旧式机体，仍然使用地脉能量作为驱动能源。[39]与此同时，深渊能量引入，王国四柱迎来了前所未有的繁荣，其机关术突破了世界法则的桎梏[33]，但因为无法妥善使用而导致失控…[11]
