# 傀儡

不怀好意的贵族把大日御舆说成是神的产物，逮捕了阿倍良久。阿倍良久最终因为不敬神明而被监禁，没过几年就因为忧愤过多而死去，他在死后被保留了贤者的称号。贵族们为了继续掌权，决定设立「太阳之子」作为元首，由婴儿担任，扶植傀儡政权。当太阳之子满十岁后将被献祭。大日御舆落成后，渊下宫得名「白夜国」。为了反抗太阳之子，斯巴达克与他的学生发动了反抗。但斯巴达克本人被抓捕处刑，他的学生领导的反抗集团因为沉迷龙骨花凝珠（类似毒品）而失去斗志。起事失败了。
日月之十年阿倍原先的副手因写作《日月前事》而被逮捕，书里记载很多白夜国之前的隐秘历史。《日月前事》后来被奥罗巴斯亲自封禁。阿布拉克故去已久。日月之前的事情已经记录得足够。若无把一切按事实记写的胆量，哪里能成为常世大神的书记呢？[3]
