# 草晶蝶

空气中飘荡不定的草元素微粒聚合而成的纯粹元素生命，同蛺蝶一般悠游。闪光的草元素生物,是元素上升凝合而成的产物，以无处不在的草元素维生。传说草晶蝶是生命的种子。无论经历过多少次的枯荣，只要种子不死，大地就一定会重新绽放生机。
