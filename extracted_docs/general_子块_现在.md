# 现在

据胡桃角色故事，胡桃幼年在生死边界游离于生死之间时意识撼动了某位「不可提及名讳之神」，因此胡桃获得神之眼。疑似为若娜瓦。雷电将军颁布眼狩令，修建千手百眼神像，神像的造型酷似若娜瓦。与旅行者携手击退深渊后，玛薇卡欲把生命献给夜神来稳固纳塔地脉，并以此向死之执政支付代价。背负不死诅咒、承载了无数纳塔和坎瑞亚战士灵魂的队长提出代替玛薇卡赴死，并迫使死之执政在“收回不死诅咒”和“放任其以不死支付死亡”之间做出抉择，作为对其的报复。死之执政给出了“不死诅咒必须延续”的回答，队长坐上「源火之圣座」，如愿与夜神融为一体，改写了夜神之国的规则。
