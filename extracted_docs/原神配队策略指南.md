# 原神配队策略指南

《原神》配队需要考虑元素反应、角色定位和团队协同。
配队原则：
- 元素多样性：队伍中包含至少两种不同元素，以便触发元素反应。
- 角色定位：
  - 主C：主要输出角色，如甘雨、胡桃、优菈等。
  - 副C：次要输出角色，如行秋、香菱、万叶等。
  - 辅助：提供治疗、护盾、增益等支持，如班尼特、钟离、迪奥娜等。
  - 工具人：提供元素能量、元素附着等功能，如雷神、九条裟罗等。
经典配队组合：
- 万达国际：万叶+达达利亚+香菱+班尼特（火水反应队）。
- 胡行钟夜：胡桃+行秋+钟离+夜兰（火水反应队）。
- 神鹤万心：神里绫华+申鹤+万叶+心海（永冻队）。
- 雷国队：雷神+香菱+行秋+班尼特（雷火反应队）。
- 激化队：赛诺/提纳里+草神+雷神+钟离（草雷激化队）。
配队技巧：
- 元素共鸣：队伍中包含特定元素组合可以获得元素共鸣加成。
- 能量循环：确保队伍中有足够的元素能量产生，保证大招循环。
- 角色命座：考虑角色命座对配队的影响，如六命班尼特、六命行秋等。
