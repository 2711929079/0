# 桃都

魔神战争时期，璃月一神山上有一巨树，是魔神，名「桃都」。「桃都」背后是死者之地。若人一生罪孽深重，死后魂魄便无法越过桃都抵达彼岸。后来魔神战争到来，「桃都」被卷入战争漩涡，一时之间死地异动，幽冥颠倒，死亡开始侵袭生的领域。桃都实力强大，但为了众生自愿被炼化。后来就有了八奇们横空出世，布下大阵，共同炼化了魔神「桃都」。之后「桃都」的神骸沉入了地脉，化为边界，隔绝了璃月的生与死。[29]八奇虽各有神通，但桃都也非一般魔神，要炼化它，并非易事。因此他们使用了一秘法，名为八门七门大阵。
发动此阵，必须择一门隐去，才可断绝邪恶的出路[29]，镇压着该地的恶螭龙[91]。幽蝶仙子主持了隐去的那一门，便因此消失于世间[29]。相传桃都被炼化后，玄鸟仙子与万象风角灵官结伴云游去了，白马仙子则隐世不知所踪。八奇中无妄姥爷，他倾心于幽蝶仙子，收拾了她的衣钵，并将其传承了下来。幽蝶仙子的衣钵也传承到往生堂手中[29]。
沉玉谷仙像待魔神的混战结束后，璃月大地上海潮退去，复归和平。仙兽们从此失却了在神的战争中守护凡人的意义，便纷纷隐居起来，过着与世无争的生活。然而，有一些仙兽依旧怀念着追随岩之神的时光，依然渴念着守望璃月的岁月。仙兽虽是超凡的活物，却依旧被寿限所羁绊。因此，它们向岩王帝君请愿，将肉身化作永恒的磐岩。就这样，慈悲的岩之神允准了它们的祈求，将它们化成了永不腐朽的山岩。[16]
