# 新的执灯人

带领着一支愚人众边防特遣队的准尉擅自出击对抗深渊，拼死救下孩子们。擅离职守，背叛女皇的重罪，又因新立下的誓言而获赦免。在骸骨垒成的阶梯上，那位准尉成为新的执灯人，新的战士开始守望，直至渊海枯竭。虽然预定的计划出了些小小的纰漏，左道的贤医（博士）却不以为意，因为目的已然达成。[45]在空寂走廊，愚人众第六连队两个派系分别对月矩力展开实验，却造成无法挽回的局面，整个连队遭到月矩力与深渊力量吞噬。
三年前·苦壑崖封印在三年前，苦壑崖封印发生异动，执灯人前往调查时遇袭，分队的成员在深渊影响下自相残杀，损失惨重。唯二清醒的成员中，队长奥尔松强令叶洛亚带上受伤失去意识的伊瓦尔撤退求援，自己留下断后。所封印魔物「靡穷之孽」的突破最终失败。该事件后，执灯长尼基塔判断暂时无法彻底解决问题，自此设下观察哨，将苦壑崖一带划为禁区。[34]
