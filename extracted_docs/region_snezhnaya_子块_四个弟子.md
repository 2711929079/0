# 四个弟子

「教授」霍德望有四个弟子/仆从，根据剧情表现，其中三位可能是「众噩之蝰」拉伊莫、「靡穷之孽」（战斗副标题为“通向魔天的先导”）、「末缀之劫」莫洛佐夫，而第四位则是大盗雷德·米勒（艾维雷勒安）[34]。
霜月之子千载信仰依循古老的预言诗，黄金之城的遗嗣在荒寂肃穆的苔原上修筑起避世的圣堂。千载月光在霜白的林间悄然流淌，终北的裔胄依旧恪守着古老的祭礼与信仰，以无瑕的秘银锻造出苍色的礼冠，拣选出至洁的主祭，指引众人前行的方向，如旧日那般，向着早已逝去的神明献上祷望，向着不再回应的高天献上咏唱。唯有那传承的祷歌，唯有那千年前的祭礼，才能成为维系失乡者的徽章。[22]
