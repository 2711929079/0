# 尘世七执政

人们所说的七神，更本源的称谓是「尘世七执政」。划分尘世、各自统治七分之一，这是作为神明的责任。七神只有履行神职，才能积累神力。需要特别说明的是，在完成枫丹主线第四章第五幕后，芙卡洛斯为解决枫丹预言，将水神之座彻底摧毁，芙宁娜作为名义水神也卸任了水神一职，故在此之后，名义上的“七神”将会减员，变为“六神”。
火
首任-赫布里穆（希巴拉克）第二任-赫布里穆（赤瞳少年）中间任-赫布里穆（库穆库尔、恩德盖、柯夏尼娜、古洛哈莉、修落维兹、伊莎佳、某曾参加七神酒会的火神等）现任-赫布里穆（玛薇卡）火神由人类担任，神位与魔神名“赫布里穆”一同代代相传[26]。
水
首任-厄歌莉娅第二任、末任-芙卡洛斯（实际）/芙宁娜（表面）水神神座已被摧毁，水神之位不会再有继任者。
风
首任至今-巴巴托斯（温迪）
雷
首任-「御建鸣神主尊大御所/雷电将军」巴尔（雷电真）第二任、现任-「御建鸣神主尊大御所/雷电将军」巴尔泽布（雷电影，雷电将军）雷电真、雷电影以及由雷电影身体改造而成、在净土外代行其职责的将军人偶，三者共用“雷电将军”之名，在民众认知中雷神一直由“雷电将军”担任，未曾换代。
草
首任-「大慈树王」第二任、现任-「摩诃善法大吉祥智慧主/小吉祥草王」布耶尔（纳西妲）纳西妲通过世界树“删除”了大慈树王，大慈树王的历史因此被叠加到了小吉祥草王身上，在提瓦特人认知中草神一直由小吉祥草王担任。
冰
首任-「白沙皇/冬沙皇」（莫诺马赫·雪奈茨尼伊）[27]第二任、现任-「冰之女皇」（安娜丝塔夏·费奥多罗夫娜·雪奈茨娜娅）白沙皇或冰之女皇的魔神名应为“巴纳巴斯”，尚不明确“巴纳巴斯”指魔神战争的胜者白沙皇，还是当前的冰神冰之女皇，亦或者冰神的魔神名也存在继承关系[28]。
岩
首任至今-「岩王帝君/岩王爷」摩拉克斯（钟离）
现存魔神
风之魔神/风神-巴巴托斯、岩之魔神/岩神-摩拉克斯、雷之魔神/雷神-巴尔泽布、草之魔神/草神-布耶尔、火之魔神/火神-赫布里穆、冰之魔神/冰神-巴纳巴斯、旋涡之魔神-奥赛尔、炉灶之魔神-马科修斯（锅巴）、沉玉谷旧主、「蕈之贤主」比夫茏
已故魔神
龙卷的魔神-迭卡拉庇安、北风之王-安德留斯、盐之魔神-赫乌利亚、尘之魔神-哈艮图斯（归终）、初代雷神-巴尔、海祈大御神-奥罗巴斯、赤王-阿蒙（阿赫玛尔）、花神-娜布·玛莉卡塔、初任水神-厄歌莉娅、第二代水神-芙卡洛斯、初代草神-大慈树王、初代火神-希巴拉克、初代冰神-白沙皇、神王-雷穆斯、恩德盖、库穆库尔、柯夏尼娜、二代火神-赤瞳少年、螭、梦之魔神、擘那（桃都）、前代火神-古洛哈莉、前代火神-修落维兹、某曾参加七神酒会的火神
