# 夜莺之歌

苦壑崖一带爆发了大规模的狂猎，执灯人集结队伍准备击退魔物、调查引发狂猎的源头。旅行者受执灯长尼基塔委托，协助寻找五百年前铸灯者击败、封印魔物所用剑、灯、镜三件圣物之一——银灯芯。「银灯芯」原本是霜月之子的祭器，可以展开结界、抵御狂猎的攻击，在平息了灾厄后，铸灯者便将它交给了守护厄布拉神柱的妖精爱莉厄保管（实际并非妖精，爱莉厄及厄布拉神柱相关内容详见#爱莉厄的牺牲）。旅行者与「魇夜之莺」分队长叶洛亚在厄布拉神柱相遇，一同清除了高塔的污染，帮助爱莉厄恢复力量以取得「银灯芯」。将「银灯芯」交给旅行者后，厄布拉神柱一带的结界全部失效，深渊重新占据此地，霍德望卷土重来试图高塔逃离「世界」。爱莉厄尽全力将其困于高塔，等待旅行者等人找到将他真正置于死地的方法。离开厄布拉神柱后，旅行者与派蒙受叶洛亚邀请，成为「魇夜之莺」调查分队的成员。在「望崖营壁」，旅行者与「魇夜之莺」小队其他成员会合，展开下一阶段的作战行动：抢先夺下崖上营地，并筑起抵御狂猎的前线。队员比约恩在保卫阵地时牺牲，小队其余成员一同击败了霍德望的仆从「靡穷之孽」，并带回铸灯者用以封印妖魔的圣剑「螺旋剑」。菲林斯寻回了由执灯人代代守护「德肋庇革劳诺之箭」，当「魇夜之莺」小队归来时，执灯人们已完成了准备工作。原来麦酒大厅正是围绕「德肋庇革劳诺之箭」建造的发射器，而「螺旋剑」则是它的箭簇。叶洛亚与旅行者一同启动「德肋庇革劳诺之箭」，螺旋剑射向高塔。在闪耀的光芒中，爱莉厄以雷德·米勒的名义宣告霍德望的终结，「不死的毒蛇」霍德望迎来第三次死亡，如预言般「被月下世界的英雄们彻底埋葬」[34]。
