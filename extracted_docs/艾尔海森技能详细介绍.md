# 艾尔海森技能详细介绍

艾尔海森是《原神》中的草元素五星角色，须弥教令院的书记官，称号"诲韬诤言"。
元素战技「森罗万象」：
- 释放草元素能量，对周围敌人造成草元素伤害并施加"草元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「裁断」：
- 释放强大的草元素能量，对周围敌人造成草元素伤害，并创造一个持续8秒的草元素领域。
- 冷却时间：18秒。
- 能量消耗：60点。
固有天赋：
- "万象的祝福"：森罗万象命中敌人后，有15%概率触发"激化"效果。
- "草元素增幅"：艾尔海森的普通攻击和重击造成暴击时，恢复3点元素能量。
- "裁断的威严"：元素爆发持续时间内，艾尔海森的草元素伤害提升20%。
