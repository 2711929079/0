# 真果角鲀

体型圆润的中型鱼类，生有犄角般的软骨结构，据说它们会有这一结构开凿水下洞穴，建造栖身之所，或是驱赶试图伤害自己的生物。体表呈现出青绿色彩的角鲀是这一谱系中最常见的品种，状如枝头尚未成熟的果实，因而得名「真果」。虽说这种角鲀鱼刺极少，肉质肥美，但要是处理的方式不当，或是摘除胆囊的时间较晚，鱼肉就会像未熟果实般的酸涩。因此，「能否制作鲜美的角鲀料理」也成了衡量须弥厨师技艺的标准之一。
