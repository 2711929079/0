# 妖僧霍德望

彼时黑日已经取代了赤月，也有人已经得知了深渊的秘密，窥探星空成了禁忌的知识，却也成了少数为独眼君王服务之人的特权。年轻的学僧拉乌斯万格意外透过镜子瞥见了星空的奇景，也窥探到了深渊的秘密，他被深深吸引，这也是一切罪行的开端。逾越身份窥探秘密，拉乌斯万格被判以流放之刑。这对于出身异国的他无异于死罪，一旦来到地上，因背弃神明而受到古老的诅咒就会将他吞噬。自认已经见证过乐园胜景的拉乌斯万格走出了洞穴，准备迎接阳光的炙烤，可他却并未遭受那可怖的诅咒，也从此自诩“被命运选中之人”。或许是坎瑞亚的炼金与占星术太过先进，又或许是北国的宫廷之内早已只剩迂腐之辈，学僧拉乌斯万格摇身一变化身妖僧霍德望教授，成为了前代冰神的宠臣。沙皇早有叛逆之心，他寄希望于亥珀波瑞亚遗留下来的秘宝，希望以此创造属于月下生灵的永恒乐园。他派出妖僧作为自己的钦差，却未曾想过妖僧早已倾心于深渊，誓要将世界作为薪柴，带他走向那深邃的黑暗。为此他设下许多阴谋布局，详见至冬[19]。
黄金百年前的坎瑞亚已会使用炼金技术，其中的炼金术士「黄金」莱茵多特在宫廷内有极高的声望。
她是魔女会的一员，[20]创作了许多炼金生命。有将其作品分层的习惯，目前已知黄金的作品分为：淋溶层、腐殖层、白垩层。
曾经创作一位名为杜林的魔龙，并在执行创造原初之人的计划时创造一个失败品，作为失败品，莱茵多特将他喂给魔龙杜林。
后来，她创造了“原初之人”阿贝多，与杜林一样是莱茵多特最为满意的造物。[21]除此之外，兽境猎犬、兽境幼兽、黄金王兽以及厄里那斯皆为她的造物。巴窟纳瓦为黄金失控造物，就算只剩下肉块也会聚会复苏，无法杀死，就连深渊教团都对他所知甚少，曾吞食莱茵多特部分躯体[22]。
