# 厄里那斯之战

雷内·英戈德在自然哲学学院学习期间创立「水仙十字结社」，招揽信众集结力量以实施他的救世方案。期间雷内将自己转化为「大师」。距今四百余年前，枫丹爆发了内战，交战双方为「水仙十字结社」与枫丹廷的机构「逐影庭」，战场之一位于厄里那斯体内。隶属于逐影庭的阿兰·吉约丹和水仙十字结社的莉利丝、雅各布都参加了这场战争。最终十字结社一方溃败。战斗中双方使用各种机关交战，诱发了大爆炸。爆炸把厄里那斯的尸体炸得血肉横飞，机械零件也散落其中。其后，厄里那斯的肉片和机械零件形成了美露莘这一种族。她们与厄里那斯的意识进行交流，在其尸体内建造了「海沫村」。[46]卡特皮拉作为结社的一员，参加了厄里那斯决战。战后卡特皮拉被捕，关押在梅洛彼得堡400年，与其相关的档案资料早已佚失[59]。
玛尔菲莎与埃尔纽艾尔为追查在枫丹境内作乱的「水仙十字结社」，德怀特委托退役多年的埃马纽艾尔调查[67][34]，循着线索找到了「大师」，被其利用莉利丝的能力溶解吸收。[33]传奇决斗代理人玛尔菲莎出身于灰河或白淞镇，公义的酬报为其所用武器，年轻时曾与埃马纽艾尔交手，被其使诈偷袭而败于其手，被迫认输。信丰公义的玛尔菲莎对此耿耿于怀，本想再找他对决，却意外得知他早早退役、投身烈酒。为了亲自查明真相，她追随着剑客的足迹与创伤，一路追猎命运中的对手…尽管有重要之人（扮演帕西法尔的约瑟芬）倒在命运的决斗场上，尽管内心已残破不堪，在追踪到剑客所留下的面具后，一切沧桑仿佛都得到了报偿[33]。最终，玛尔菲莎与自己苦苦追寻的埃马纽艾尔落得同样的结局，「大师」获取了她的枪术技能。[42]
