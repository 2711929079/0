# 迷宫

森林王为了守护法留纳神机的秘密，用梦境、树篱和支流建造复杂的迷宫，森林迷宫的路途会将迷误之人带往梦里。但在数百年前，这个迷宫便不复存在了。[36][37]
赤王的筹备交易赤王找到了草之龙阿佩普，想与其做个交易，赤王已经获得了深渊知识，而深渊知识正是阿佩普对抗天理所需要的。最终他们达成共识，阿佩普允许赤王在自己的领地建立强盛的王国，但在赤王死后，他所获得的一切知识都要归阿佩普所有。[5]虽然阿佩普只是被赤王利用，且其他六贤也都知晓[38]，但因为这场交易阿佩普与赤王成为明面上的盟友，阿佩普被赤王列为“七贤僧”之一。[39]
