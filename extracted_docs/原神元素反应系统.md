# 原神元素反应系统

《原神》的核心战斗机制是元素反应系统，七种元素之间可以产生各种反应：

1. 融化：火元素 + 冰元素
   - 火元素触发：伤害提升2倍
   - 冰元素触发：伤害提升1.5倍

2. 蒸发：火元素 + 水元素
   - 火元素触发：伤害提升1.5倍
   - 水元素触发：伤害提升2倍

3. 超导：冰元素 + 雷元素
   - 造成冰元素伤害
   - 降低敌人物理抗性40%

4. 感电：雷元素 + 水元素
   - 持续造成雷元素伤害
   - 可以在多个敌人之间传导

5. 超载：火元素 + 雷元素
   - 造成爆炸伤害
   - 可以击飞敌人

6. 冻结：水元素 + 冰元素
   - 冻结敌人，使其无法行动
   - 可以被重击击碎造成额外伤害

7. 扩散：风元素与其他元素
   - 扩散其他元素的效果
   - 对周围敌人造成对应元素的伤害

8. 激化：草元素 + 雷元素
   - 触发激化状态
   - 后续草元素和雷元素伤害提升

9. 绽放：草元素 + 水元素
   - 产生草原核
   - 草原核接触火元素触发"燃烧"，接触雷元素触发"超激化"

元素反应是战斗策略的核心，合理利用元素反应可以大幅提升伤害输出。
