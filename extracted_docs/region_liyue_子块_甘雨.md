# 甘雨

魔神战争结束后，甘雨与帝君签下契约，前往月海亭担任秘书。
琥牢山琥牢山的琥珀，来自于理水叠山真君植下的花，名踱山葵。它本身大部分隐于地下，若被踩到，就会生出琥珀，将入侵者包裹在其中。理水叠山真君归隐到此处后，为了山门清静，将它种植在四周[88]。
