# 其他

美露莘一族只有雌性，通常个体较为矮小，手脚与人类不同，寿命可达数百年，其中，希格雯的外观更加接近人类。
美露莘的视觉与常人不同，并且不同美露莘的视觉亦有差异。譬如夏诺蒂拉看旅行者是能吞下天地的怪兽，派蒙是线连在天空之上的彩色小气球，而希格雯的视觉则更接近人类。
美露莘拥有独特的饮食习惯，其食物外表令人难以接受，“尝试海沫村料理”还被《蒸汽鸟报》评入“年度前一百勇敢者挑战”，排行第二十位。——但味道还是可以接受的。
枫丹法律规定，称呼美露莘只能使用人格化的“她”，而不能使用指代动物的“它”。该法律由那维莱特推动订立。
