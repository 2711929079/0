# 希巴拉克

察阿克受盗火者的敌人，即夜神之国神祇的招唤下成为大灵在盗火贤者预料之外，但所幸计划仍稳定进行[56]。曾统治纳塔的火之古龙「焰主」修库特尔假死后，龙众因执迷于守候修库特尔的躯体而日渐衰败，瓦萨克拉胡巴肯将龙的未来寄托在了人类身上，意图创造一个同时拥有龙与人血脉的存在击败火龙，开创龙与人的新时代，因此他将人类的灵魂融入了子嗣奥奇坎的躯体中，使其成为了半人半龙的特殊存在。在预言中，奥奇坎将斩杀火龙，以暴虐和狡猾成为两个世界的救主。领主第八席「花烛与风羽的司巫」认为盗火者给人类燃素是为了让他们用燃素自相残杀，同情人类的第八席不满这做法认为太过残忍。而第三席「纺秘者」则悲悯同胞，失去智识化为野兽的龙蜥，因为越来越多龙蜥被人类驯养用来当作内战的武器，于是打算借用人类灵魂打造梦境，让龙蜥陷入完美的梦中得以解脱。第八席认为人类会反抗阶级制度导致无法妥善管理，于是打算打造个封闭的保护人类免受天灾与龙族攻击的的庇护所，在其中重塑人们的价值观，教导他们前往正确道路。修库特尔清醒期间越来越少，她将此计划告知修库特尔，修库特尔同意此计划，以及第三席「纺秘者」的计划，却使得第八席迷茫，火龙王总是轻易同意任何计划，究竟哪个计划才是受到它心里真正认可的？或许自己的行为在其他龙眼中就像她看待盗火者一样「疯狂」？她对这个计划展开思辨，途中和盗火者一样发现不盲目遵守阶级制度是人类的优点，于是转而想要学习人类，为了观察人类收养了孤儿莉安歌[39]。奥奇坎曾因龙与人的双重身份被人类囚禁，但在希巴拉克的宽容下，其最终如盗火贤者计划被释放并被人类所接纳，盗火贤者计划以此让他接近人类。奥奇坎卻被希巴拉克的人格魅力所折服，并且也开始憎恨其本应归属的龙族。因为盗火贤者总是埋怨其他龙不理解自己，又总是自视甚高、孤傲自负且谜语人不告知其余龙自己的想法，领主中的第八席「花烛与风羽的司巫」看不起他[39]。盗火贤者的做作气到她连青蜜莓都没吃。库库尔坎的种种「龙奸」行为也早已引起了同族们的关注与非议。十三位至高领主与三位亲王由第二席主导联名发起了对摄政王库库尔坎的弹劾调查，认为他似乎与天外掠夺者带来的「人类」走得有些太过近了，怀疑就是他将本不该为人类所掌握的燃素力量交予了人类。而由库库尔坎亲自培育的实验体（奥奇坎）出现在了人类联邦中，半人的谋主依仗自身龙族血脉与知识协助人类猛烈打击龙族防线的这一事实，更是成为了众龙质疑库库尔坎早前所谓「实验体意外出逃」的说辞乃是谎言，炮轰他是族裔可耻叛徒的铁证。尽管库库尔坎对同族的指责不屑一顾，但弹劾案还是以9票通过0票反对4票弃权的结果通过了。发起弹劾的领主当面斥责库库尔坎贵为亲王却做出如此背叛同胞之举，根本没有资格接下摄政之权，理当接受圣裁。但让众龙没想到的是，当时身体状况已非常糟糕、极少在公众场合露面的火龙王修库特尔，竟会恢复清醒突然现身审议厅。王的意思自然无需过多揣度，原本争论不休的十三位领主和三位亲王立刻收回了他们的话语，弹劾一事也就这样不了了之了。后来在盗火者计划外出现的希巴拉克物理说服部族的初代英杰们(拥有人的灵魂与龙的躯体因此被囚禁，后来被希巴拉克救下后迷恋希巴拉克的半人半龙的谋主奥奇坎、浪荡于林间的少年尤潘基、后来分别成为了沃陆之邦和回声之子首领的阿霍布和伊葵两兄妹、深谷（日后的烟谜主）祭司的侍从马古汉、梅兹特利（流泉之众）的虹之王乌努库、龙母之女，花羽会的创始人莉安歌）一起遵循大灵的指引进入图兰大火山，它位于纳塔西南的巨大火山，被称为「圣山」。进入火山的勇者之中，有数位曾在蕴火的幽墟修行过[21]。来自梅兹特利的少年——乌努库，他遇见了一位纯水精灵伊艾涅伊菈（希涅伊菈），纯水精灵为了实现将厄歌莉娅的恩泽传播大地的理想[57]，想要驱逐水中使其无法前行的燃素，让他站出来反抗龙王。他没能成功，烈火焚尽了他的身体，[58]不过他的古名「团结」(沃摩延)传承了下来，被五百年前的「巨人」图帕克和现在的玛拉妮继承。[10]大战的最后阶段，希巴拉克在图兰大火山对战火龙王。此时被深渊侵蚀已即将死亡的修库特尔已经将全身浸入源火，仅留下一只眼睛在外，这只眼睛也称为唯一的弱点遭到希巴拉克击穿[1]。一人一龙一起从空中坠落（这个时候双方都死去了），然后希巴拉克得到古龙大权成“仿佛全身被火焰包裹的男人”从死去的火龙王的心脏中复活。击败圣山里的火龙后，火龙王死时希巴拉克获得古龙大权成为初代火神。这也是纳塔的火神历代都是人的原因。希巴拉克成为火神后获得魔神名赫布里穆，此魔神名传承给之后数数代代的火神[59]。世事终究出乎了盗火贤者的意料，最终斩杀火龙、成为君主的并非被其寄予厚望的并非奥奇坎，而是身为人类的希巴拉克，而希巴拉克吸收火龙的权能，升华为火之执政后也并未如预言般用铁腕统治纳塔，而是团结了纳塔的各个部族，一同抵抗深渊。战争结束，夺得火之大权成为“最初的人神”后，在始灾的边界盗火贤者与希巴拉克定下契约，令其重建的国度能够同时容纳人和龙，约定在未來古龙的族裔日后归来时不会进行阻拦，并且在深渊来临之际，他们要拼合「通牒金盘」，用「焰主」的血脉开启修库特尔制造的[1]「维奇洛波奇特利」，虽然开启后人类会无法挺过这场浩劫，但也能创造出超越人和龙的新生命。不过他后来没有打算履行这个契约，因为他在这之后产生了更好的计划[59]。随即在盗火者开口前，希巴拉克来到奥奇卡纳塔加冕为王，在此死之执政告知希巴拉克深渊的危机迫近[59]，纳塔将被深渊侵略。因此他利用神座的力量联系死之執政若娜瓦，向她借走力量为纳塔制定了新的规则並强化夜神之国[13]，使纳塔的凡人可以成神，而代价是获得她力量的火神必须支付「死亡」为代价。若娜瓦为了避免天理制裁要求希巴拉克保密，对外宣称是他偷走自己的力量。[13]希巴拉克制造圣火（圣火竞技场最上面那个），并使用死之执政的力量为纳塔建立各种规则，包括古名，以及透过归火圣夜巡礼对抗深渊。这些规则盗火者从未想到，因此令他十分惊讶。[15][10]希巴拉克要求若娜瓦，在深渊危机解除后让他暂时复活看看到时候的纳塔[60]。之后，夜神时刻都在借助圣火的力量对抗深渊的侵蚀。[13]他向纳塔的人们提出诫言，希望日后纳塔的人和龙能够和平共处，然后走入了圣火。走入圣火时在源火中窥见纳塔的未来—纳塔人将在对抗深渊的战争中获得胜利[59]。谋主奥奇坎捡起了希巴拉克留下的缺裂明环（火龙王的半个眼睛），成为了纳塔暂时的统治者[61]。莉安歌的龙养母正是龙族至高十三领主中的第八席「花烛与风羽的司巫」伊·卡科茨·伊齐玛瓦夏。在人类出行对抗火龙王时，曾委托傲娇的领主第二席打造弓送与讨伐火龙的莉安歌，但隐瞒弓由第二席所造。燃素锻造的技艺源于古代的龙众。在那个时代，存在一种更为精妙的、如今无法想象的锻造方式，即是直接以燃素为锻造的原料，构筑出不同的物质[62]。第二席认为人类优秀于龙族，那人类理应击败火龙王取代龙类。第八席之所以收养莉安歌，是为了分便观察人类[39]。至高领主第八席“花烛与风羽的司巫” 委派龙仆调查坎科普里尼克引发骚乱的人类的残余物，从中发现了神之眼的存在。而后与至高领主第三席“纺秘者”讨论后认力：神之眼的工作原理与涅法海姆的祭礼用具 “月之轮”类似。后来第八席还为莉安歌留下了一枚月之轮[63]。原本的议事之厅在繁荣之时，龙众在此处创造诸多秘法与机关。但在修库特尔死后，此处变成了审判同族的场所。第八席在修库特尔死后[64]被审议之厅投票处死。虽然对人类心存善意，但直到被处决前，第八席心中依然没有背叛龙众，心怀身为领主的骄傲[65]。龙丧失了力量后，为了生存不得不更加依赖环境后，固定下来的形态就是如今的纳塔龙众。图兰大火山原本是一度伟大的龙的居所，可是在龙自火中遁去之后，他们的居所就被狡猾的火山大王占据了。[66]此时龙族13位至高领主，10位死亡，9号流落在外。4号阿乔是最后仅存的领主，因不明原因被人封印在神秘腕刻中数千年。第1席，第7席，第12席，死在圣山的决战里。第6席，第10席，被人干掉了。至高领主第六席“烈怒如雷霆般撼动大地者”在与阿霍布的角力中战败，被一箭射中脖颈身亡。至高领主第十席“不凋的孤高之翼”将伊葵仿造的玉石误认成是至高领主第二席“玉石的少女”的信物，又被诱骗喝下剥夺力量的毒药，被活理进山岩中去世[67]。之后至高领主的历史被后人忘却[65]。古老的文明利用镌光的金纹驱使燃素流动的诸多法式，令巨大造物与恢弘城市得以在金纹创生的智慧中运转。据说除了固定结构的机械以外，名为熔岩游像的魔物其实也是以此方法创造的残缺生命，只不过在圣山的火焰熄灭之后，它们大多失去了目的与方向[68]。
纳塔大联盟时期帮助希巴拉克讨伐火龙王的成员如下：半人半龙的奥奇坎，其实是继承了「焰主之祝」的「盗火贤者」瓦萨克拉胡巴肯之子，他将自己儿子奥奇坎融入人类灵魂，以此达成人龙合一的新物种[69]。希巴拉克身为「人神」寿命无法长久，盗火贤者打算希巴拉克寿命将尽时让奥奇坎取代他并统治纳塔，让理想中的人龙合一者治理纳塔。希巴拉克结识时他是成为火神之前，奥奇坎一度遭受拘禁，希巴拉克解救并拉他进討伐火龙王的阵营。悬木人的祖先——尤潘基，他和希巴拉克、奥奇坎是朋友关系，他是大联盟的军械官，年轻的他十分莽撞活泼。被尤潘基逗得哈哈大笑的弓手是莉安歌，被龙养大的少女，三度射落邪龙，三度将它治愈，成为至交，称希巴拉克为「拉克」，她成立了「花羽会」。[70]双生英杰兄妹——阿霍布与伊葵，父亲因龙王一生心血而殒命，母亲因保护他们而牺牲，他们因年幼时失散，被不同的部落养大。重逢后踏上冒险，依循「力量」与「祝福」的道途。阿霍布创立了「沃陆之邦」，伊葵则创立「回声之子」[67]。伊葵之所以被称作「锻火的英雄」，也是因为她真的通晓如何锻造燃素本身。她并非龙众，只是因为没有父母赐予的名字，才依照龙的语言规则为自己取了名。萨库科见过伊葵的技艺[62]。烟谜主他们居住的深谷那时还不叫「米克特兰」，继承王名的是一位剜去双足的的少女（祭星者之望的主人），这里龙与人类立下盟约，龙保护人类(奴隶)，人类选出一位圣王指引龙入梦。少女圣主身边的仆人丁加预知到希巴拉克到来时，少女与他相见，下次相见会刺穿她的心脏以此打破旧的盟约，成为屠龙者马古汉后，建立烟谜主的新制度。[71]传说在古老的时代，在大地拒绝种子时，部族民无法向土地求得食粮而陷入了绝境。那时，祭司马古汉听从夜神指引，自河底挖来淤泥，向其中种下动物的牙齿。而后他站在淤泥中，割伤自己，用鲜血浇灌，于是那一颗颗牙齿顺着祭司的身子长了出来，变成了如今的颗粒果[72]。
