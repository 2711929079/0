# 大灵、夜神之国与深渊

「大灵」象征着守护与祝福他们的东西，它能通晓过去洞见未来，火神来自地上，「大灵」位于地下。极少数人可以和「大灵」进行精神沟通，正常手段都去不了那里，只允许通过意识进入，但也存在将人完全送进去的做法，但那样会遭到「夜神之国」的排斥，同时要面对深渊力量的侵蚀。「大灵」所在之地为「夜神之国」，介于肉体与灵魂，生命与死亡之间的，非常神秘的地方。「深渊界门」为深渊制造的「门」，魔物靠它从夜神之国冲到地面上来，想要终结入侵，必须击溃深渊界门。深渊所针对六个部落留下的影响还在，「流泉之众」的水受到污染，「回声之子」地下涌出灾厄的泥浆、「花羽会」吹起永不停歇的黑风，玛拉妮认为深渊之所以有这种充满智慧的破坏方式，是因为深渊入侵了「夜神之国」，时刻都在读取这片土地的记忆，所以变得凶险又狡猾。长期停留在「夜神之国」个体的存在会被加速吞噬，最终与海量的灵魂融为一体。[67]
