# 火龙王

火龙王修库特尔在图兰大火山处的源火诞生[1]。
图兰圣城火龙王建造图兰圣城，用源火创造纳塔龙裔。通过浸泡流炎之池，纳塔龙裔即可从其中海量的燃素里获取自己需要的智识[2]。古老的龙众曾经用自身脱落的鳞片以传递秘密的信息与谕令[3]。分火塑造库库尔坎。火龙王用燃素打造13个至高领主的意识，至高领主意识能透过机械容纳，各领主们都拥有各自的土地管理城市[1][4]。十三位至高领主，既是秘源技术的巅峰，也是火龙王的子嗣。高傲如他们，自不会接受「命之座」的禁锢[5]。
