# 黑泥

阿佩普体内的元素生命逃离受禁忌知识污染的阿佩普，寻求大慈树王的帮助，大慈树王根据阿佩普“绿洲之心”的运作方式，制作了能重新激活绿洲之心的“火种”，并答应元素生命们未来会让他们重新回到阿佩普体内。为了避免禁忌知识消失后，与其关联的记忆出现问题，一只元素生命来到层岩巨渊的天钉附近，利用天钉的净化之力将禁忌知识转化汐较为稳定的“黑泥”，以此来保存记忆[115]。
帕西法尔来自蒙德的侠盗帕西法尔逃离蒙德后，来到璃月港。初到港口时，身上仅佩着一把细剑，除此之外，只有一根小小的蓝宝石色羽毛，佩在破旧的斗篷上。后来，帕西法尔成为大副，与一位船师投身汪洋，和风暴、海兽与豪浪相搏。据说最终在与海兽的战斗中死去。曾经蓝宝石色的飞羽也被血水染红，浸润了大海的咸涩。[116]
