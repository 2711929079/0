# 草龙

草龙身为草元素生命的顶点，是须弥曾经的统治者。它曾在沙丘下的庞大空洞中修筑行宫…
沙虫曾是它的廷臣，飞蛇曾是它的友伴。但在今日，它们一并失去了高贵的神智与光彩。[2]
部族最初，各个部族与沙为伴，将血脉与大地相连。遵守血的律法，畏惧深入血脉的流散饥馑记忆。[3]
