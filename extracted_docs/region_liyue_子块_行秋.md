# 行秋

飞云商会的二少爷行秋向往武学，在几年前拜师于衰落已久的古华派门下，并且仅用四年就自学领悟了古华派的三大武学奥义，被视为是古华派振兴的希望[137]。
香菱与锅巴万民堂年少有为的厨师香菱几年前在寻找食材时，结果天降大雨。又累又饿的香菱进了一个山洞中避雨，将随身携带的辣肉窝窝头放在了山洞中神龛前的供桌上，随后睡着了。醒来后看见失去知性的马科修斯正在吃辣肉窝寓头，二者就此成为朋友，香菱为其取名为“锅巴”[190][80][191]。
