# 回火夜

在战役中，布基纳被因深渊侵蚀而发狂的伙伴山王攻击，在死前将自己的力量注入给山王。山王在此后一直希望了结自己以赎罪。在人们帮助下，「山王」决定带着罪孽坚强的活下去，因为了结自己很简单，但背负痛苦活下去却很困难，背负着过往活下去成为了山王的赎罪。[18]
盗火者缝影针旅行者帮助盗火贤者伪装的蒂图修好了小缝影针，并让困在封印中陷入永恒战斗的两位坎瑞亚士兵西弗安雅与伊利萨得以解脱。
