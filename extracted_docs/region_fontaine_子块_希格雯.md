# 希格雯

在曾经那个人类厌恶美露辛的时代里，名为希格雯的美露辛拜访了一位性格孤僻但并不排斥美露辛的老医生，向她学习医术。之后希格雯与一个小女孩成为朋友，这个小女孩身患重病，希格雯想帮助她却被她的家人拒之门外。她只好向老师求助，恰好她的老师当时在研究原始胎海水，在研究中顺带发明了一种能让美露辛变成人形的药剂。老医生想要找到拯救枫丹人的方法，于是拿自己试药，最终失败被溶解。希格雯为老师建了一座衣冠冢后，便服下了另一瓶药剂变成人形，顺利地以医生身份去治好了那个小女孩。之后希格雯前去自首，因为她知道自己违反了一条枫丹自古以来就有的法律：禁止将其他物种转化为人类。在服刑期间希格雯运用医术帮助人们，就算刑满后也一直留在梅洛彼得堡当护士长[73]。
特许券愚人众执行官“富人”认为金钱作为交易用的物品，应当由人们亲自掌控，而非由神制造。为了制造能取代摩拉的货币，富人在梅洛彼得堡发行特许券作为实验，并使用无人机“监督者”观察着梅洛彼得堡的金流。无人机不知疲倦地记录着每笔金钱的易手，每个人的积蓄与挥霍，每种价值的提升降低，每张货币在指定周期内的流通次数。其间唯一法律是囤积者的私法，而唯一制裁是贫乏或死亡。或食利掌权而支配，或被支配而劳碌至死，规则总是公平。由此，人将凭借自身所拥有的野心与财富与神并肩，而那些竞争中失去一切的弱者将被人世的洪流吞噬，不再有神的力量介入，从贫者面前粉饰富者的威势，不再有神的财富涌入，从富者脚下拯救贫者的尊严。[74]（卡特皮拉提到400年前就有特许券存在[59]）
