# 纳西妲技能详细介绍

纳西妲是《原神》中的草元素五星角色，须弥的草神，称号"智慧之神"。

元素战技「所闻遍计」：
- 效果：释放蕴种印，标记敌人并造成草元素伤害。被标记的敌人受到元素反应时，纳西妲会触发协同攻击。
- 持续时间：8秒。
- 冷却时间：10秒。
- 能量消耗：无。

元素爆发「摩耶之殿」：
- 效果：创造一个领域，领域内的敌人会被持续施加草元素。
- 元素转换效果：
  - 水元素：扩大领域范围
  - 雷元素：提升领域内角色的攻击速度
  - 火元素：触发爆炸，对领域内敌人造成草元素伤害
- 持续时间：12秒。
- 冷却时间：18秒。
- 能量需求：70点。

天赋技能：
1. 千树叠章：元素战技命中敌人时，为队伍中所有角色提供元素精通加成。
2. 明论：元素爆发领域内的角色获得额外的元素精通。

角色定位：
- 主要定位：后台输出
- 次要定位：辅助
- 核心优势：草元素附着、元素精通加成、元素反应触发

培养建议：
- 武器推荐：流浪乐章、祭礼残章、四风原典
- 圣遗物推荐：深林的记忆四件套（草元素伤害加成和减抗）
