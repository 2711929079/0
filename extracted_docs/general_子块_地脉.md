# 地脉

连接这个世界上一切事物的概念网络，不同的元素在其中奔流。地脉流动即时间流逝。若与地脉失去连接会失去庇佑变得不幸。据说，地上地下发生的一切，都记录在了地脉当中…地脉是世界树的根系，所有的记忆与知识都会顺着地脉流入世界树，但若地脉与世界树之间联系断了，记忆就会储存在与世界树断了联系的地脉内。提瓦特的灵魂最终会流入地脉。但葬火之战前的灵魂是前往月亮[50][51]。最初提瓦特就有地脉，是天理将其改造成如今能庇护提瓦特的。天理制约魔神，让魔神们口中是天理创造地脉的。所以炽烈的还魂诗后日谈中玛薇卡说天理创造地脉，但夜神却说天理改造地脉，而失落圣城与千年来者中盗火贤者也说天理改造地脉。
