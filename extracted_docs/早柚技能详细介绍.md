# 早柚技能详细介绍

早柚是《原神》中的风元素四星角色，稻妻忍术大师，称号"忍里之貉"。
元素战技「早柚的忍术」：
- 释放风元素能量，对周围敌人造成风元素伤害并击飞。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「大玉旋圆」：
- 创造一个持续8秒的风元素领域，领域内的敌人会受到持续风元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "忍术的精妙"：早柚的忍术命中敌人后，有15%概率触发"扩散"效果。
- "风元素增幅"：早柚的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "旋圆的威力"：元素爆发持续时间内，早柚的风元素伤害提升15%。
