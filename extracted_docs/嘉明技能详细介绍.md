# 嘉明技能详细介绍

嘉明是《原神》中的火元素四星角色，璃月的杂技演员，称号"星螺戏法"。
元素战技「戏法·火焰」：
- 释放火元素能量，对周围敌人造成火元素伤害并施加"火元素附着"。
- 冷却时间：8秒。
- 能量消耗：无。
元素爆发「烟花盛宴」：
- 创造一个持续10秒的火元素领域，领域内的敌人会受到持续火元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "戏法的魅力"：戏法·火焰命中敌人后，有15%概率触发"燃烧"效果。
- "火元素增幅"：嘉明的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "烟花的祝福"：元素爆发持续时间内，队伍中的角色火元素伤害提升15%。
