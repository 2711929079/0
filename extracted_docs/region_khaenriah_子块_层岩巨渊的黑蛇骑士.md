# 层岩巨渊的黑蛇骑士

层岩巨渊化作深渊魔物的的黑蛇骑士将体内的血视为「神圣之血」，天钉的力量视为不祥之物。因为深渊力量会与天空的元素力互相烟灭，所以黑蛇骑士透过深渊之血消除层岩巨渊天钉的力量[53]。
五大罪人五大罪人分别是「黄金」莱茵多特、「贤者」海洛塔帝、「极恶骑」苏尔特洛奇、戴因斯雷布的哥哥「预言家」维瑟弗尼尔、「猎月人」雷利尔。坎瑞亚被深渊入侵时，才华出众的五人连同戴因斯雷布被寄予「拯救坎瑞亚」的厚望，但五人因自己私欲滥用深渊力量反而加速坎瑞亚覆灭。之后维瑟弗尼尔成为了紫色晶石的样子，在枫丹留下了「所有人都会溶解在海里」的预言石板[52]，并向罗南谕示，使其知道天空的不公而投向深渊[54]。苏尔特洛奇离开提瓦特时，将吞星之鲸送给了斯克弗努格。身带不死诅咒的斯克弗努格拆下吞星之鲸某些部位缝入自己心脏，透过吞星之鲸的力量增强自己，但精神状况变差了[27]。收留一个徒弟丝柯克，向其传授「极恶法」，还将吞星之鲸当成宠物[55]。其的力量高于整个世界，威胁整个宇宙，有一众追随者在其底下拜师。其力量是许多生灵能想像的最高顶点，因此许多生灵为了追求更强，会幻化成他的模样。如今他穿梭于各个宇宙，能随意改变容貌，通常使用盔甲在宇宙中穿行。其创立的组织「寰宇劫」是这个宇宙的一大威胁[56][27]。戴因斯雷布在坎瑞亚覆灭后，立誓要向五大罪人复仇[52]。
