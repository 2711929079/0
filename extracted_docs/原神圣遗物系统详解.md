# 原神圣遗物系统详解

《原神》的圣遗物系统是角色养成的重要组成部分：

1. 圣遗物部位：
   - 花：固定主属性为生命值
   - 羽：固定主属性为攻击力
   - 沙：主属性可选生命值、攻击力、元素精通、元素充能效率
   - 杯：主属性可选元素伤害加成、物理伤害加成、生命值、攻击力
   - 冠：主属性可选暴击率、暴击伤害、元素精通、治疗加成

2. 圣遗物稀有度：
   - 一星到五星，五星圣遗物属性最优
   - 高稀有度圣遗物有更多的副词条

3. 圣遗物套装：
   - 2件套效果：装备2件同套装圣遗物激活
   - 4件套效果：装备4件同套装圣遗物激活
   - 主流套装：
     * 翠绿之影：风元素伤害加成和减抗
     * 深林的记忆：草元素伤害加成和减抗
     * 绝缘之旗印：元素充能效率和元素爆发伤害加成
     * 追忆之注连：元素战技伤害加成
     * 魔女的炎之牢：火元素伤害加成
     * 冰风迷途的勇士：冰元素伤害加成
     * 如雷的盛怒：雷元素伤害加成
     * 沉沦之心：水元素伤害加成
     * 逆飞的流星：护盾强效和普通攻击伤害加成

4. 圣遗物获取：
   - 秘境挑战：主要获取途径
   - 世界BOSS：掉落特定套装
   - 活动奖励：限时获取
   - 商店兑换：使用材料兑换

5. 圣遗物强化：
   - 通过消耗其他圣遗物提升等级
   - 每强化4级随机提升一条副词条
   - 强化到20级为满级

6. 圣遗物选择策略：
   - 主C角色：优先选择对应元素伤害加成、暴击率、暴击伤害
   - 辅助角色：优先选择元素充能效率、元素精通、生命值
   - 副C角色：平衡元素伤害和暴击属性
