# 浮锦、药君与灵渊

一条鲤鱼修炼成仙人浮锦，与同样被尊称为仙的白蛇药君和猊兽山主灵渊为好友，三仙共同服侍一位魔神，与此同时也和萍姥姥等仙人有过来往。浮锦并非仙力强大的仙人，无法长期离开清水活动，因此萍姥姥赠给了她一盏茶壶，相约由其友人用茶壶装着她去璃月港游玩，可惜到最后都未能实现。浮锦和药君与各种各样的人接触过后，逐渐对人类产生好感。药君化成人形，帮人们治病，浮锦在灵渊的山头上种下茶树，并将种茶技艺分享给人类。灵渊虽然无法理解两位好友对人类的喜爱，但出于和友人的情谊愿意尊重她们的选择。[24]灵渊也化为人形与人类生活，还因吃了过多的茶糕，担心自己长胖而求药君制作可以减肥的茶叶。最终在灵渊和浮锦的共同请求下，药君制作了这种茶叶[51][52]。药君在伏仙洞种下茶树，并赋予一些机关与仙像生命[53][54]。玄文兽在遥远的过去曾是灵渊的眷属[55][24][56]。
恶兽「八虬」那是璃月草创的年代，海洋还是巨怪与魔神的园地。先民在对海洋的恐惧中度日，以微薄之力与海作战。万千个春与秋中，过于巨大的海兽成了广袤海域不羁的霸主，即使与岩之主造出的石鲸厮杀也不落下风。那是被璃月人称为「八虬」的魔兽，在海渊深处未曾有匹敌。偶然自无底深渊浮上浅水，掀起巨浪如山崩般摧毁屋舍舟船。岩神于是亲手用玉石与矶岩塑了一只鸢鸟。石鸟甫一造成，便挣脱了万物桎梏的大地盘旋登上高空，如烈日投出的长枪般，直插大洋中心魔兽与岩鲸激斗的战场，巨兽应声被钉入了黑暗的海沟，不复浮出。从此璃月的人们便摆脱了海中巨兽的咆哮。[57]
