# 王座到来

提瓦特是一个星球，原始胎海孕育了这颗星球最初的生命[6]。最初，水龙王是原始胎海的心脏[7]。龙族曾是提瓦特最早诞生的生命，其中的七位龙王与其共主尼伯龙根统治着古代提瓦特。龙王包括水龙王那维莱特，火龙王修库特尔、草龙王阿佩普等（那维莱特是初代水龙的转世，初代水龙殒落后以人形重临，就是现在的那维莱特）。在鸽子衔枝之年，天上的永恒王座自世界外降临，第一王座——“原初之人”法涅斯从蛋中出生，割裂出「原初四影」作为最初的执政[8]。法涅斯向七龙王开战。经过40年的惨烈战争，七位龙王们被打败，而此时尼伯龙根因不明原因离开提瓦特去世界外旅行没有参战，法涅斯成为了新的统治者，并制造了人类，剥夺了七位龙王的一部分力量。破镜为那场战斗中法涅斯留下的武器碎片，曾有熄灭月光的力量[9]。其原由尼伯龙根所造，却被法涅斯盗取[10]。七龙王全向法涅斯俯首称臣。法涅斯剥夺七龙王的一部分力量，最初的胎海之心水龙王陨落，龙在枫丹地区的统治崩溃。原初的那一位大王开始了對天地的改造，将其改造成后来诞生的人类宜居的样子。原始的蛮荒之力与燃素，被改造成如今的「元素力」，龙族宜居的蛮荒的大地开始能种植粮食。无相元素是天空岛神系侵入改造元素力后诞生的新生命[11]。部分战败龙族逃亡暗之外海，而部分龙族如草龙阿佩普一直等待尼伯龙根归来。后来部分龙族选择与人共存，也有一部分等待着龙王的归来，等待复仇的一天。三月女神的力量自然无法与天理对抗，但天理也没有将她们视为必须铲除的异端，本质上三月女神维系看这颗星球的存续与运转，与天理的目的并无冲突[12]。此事被龙族视为三月背叛龙族，之后法涅斯透过三月打造了神之眼的前身「月之轮」，与神之眼不同的是它使用燃素，发放给人类用来让人们联系月神，与神之眼相同获取后就是卖身于命运。山川与河流落成，此时的山比天穹还高，大地比天空更广大[13]。不服从的龙群逃亡去大海和大洋并各自演化，这就是渊下宫里深海龙蜥的由来。原初的那一位和一位影子制造出了飞鸟、走兽和水鱼。它们还一起制造出了花草和树木。法涅斯依靠自己的记忆打造出世界与其规则[14]。最后它们造出了数目不可知晓的人。自此时起，人的先祖和原初的那一位立约。法涅斯对人有一套神圣的规划。人只要幸福，它便欢欣。[8]在这之后，部分龙族选择与人共存，也有一部分等待着龙王的归来，等待复仇的一天[15]随着第一王座的降临，水龙王及龙在枫丹地区的统治崩溃，水龙王同时陨落。
胎海的新心脏水龙陨落后，生之执政在原初的大海里创造了另一颗心脏，名为厄歌莉娅。[7]
