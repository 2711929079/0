# 重云

重云出生驱邪世家成为受到某人祝福，具有驱邪“纯阳之体”的方士[181]。因为具有“纯阳之体”，重云驱邪时只需要静坐在現场一段时间，便可以完成委托[182][183]。
嘉良十年前，嘉良患上了罕见的心病，白术的师父本就时日无多，还继续折损寿命来医治嘉及，江蓠不忍，带着嘉良远离师门，避而不见。为拯救嘉良的生命，江蓠利用“螭”的魔神残渣炼就了一味毒，成功保住了嘉良的生命，但代价是心智和记忆都会被魔神残渣慢慢侵蚀。嘉良和江蓠约定，等到二人的孩子“阿雩”长大，就让自己上路。[77]
