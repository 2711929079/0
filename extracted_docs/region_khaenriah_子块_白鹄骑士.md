# 白鹄骑士

传说五百年前，荼诃古国（坎瑞亚）的「白鹄骑士」曾驾驶着遗迹巨像对抗黑渊涌出的魔群。在残酷的战争后，最终残留下来的三台遗迹巨像被贤者们所封印。[45]部分成员：
西弗安雅与伊利萨五百年前两位坎瑞亚战士—西弗安雅与伊利萨和当时的「回声之子」长老陶奥鲁缔结了深厚友谊，牺牲自己帮助纳塔英雄封印深渊力量，两人自己却和深渊魔物一同被封印住，从此在封印里陷入了永远的战斗中无法解脱，但仍和其他被封印的纳塔人/龙的意识一起阻止深渊涌出「镌光铭印」。但却被后人误解为引来深渊的人[47]。
