# 勘定奉行

柊家。稻妻的财务中枢，负责国家税收和财政。职责之一，是严格审核出入稻妻者的身份，并在「离岛」对外来人员进行统一管理，出入都要「勘定奉行」多重审核。当代家主为柊慎介。由于与愚人众勾结一事败露，被羁押，由其女柊千里代行家主一职。松浦家辅佐柊家执政，在柊慎介入狱后想要通过柊千里和九条镰治结婚控制勘定奉行，其阴谋被神里绫人用计挫败。
