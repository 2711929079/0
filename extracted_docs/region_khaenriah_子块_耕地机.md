# 耕地机

「土地不是用农具去犁的，而是用铁与血去争夺的。」——基于这样的理念，「耕地机（遗迹守卫）」诞生了。耕地机的能量来源并非纯粹的元素，[9]而是一种名为阿索斯物质的地脉能量块[10][11]。驱动耕地机的能量集中于其“眼”中的核心。初代耕地机诞生后，具有强大破坏力的它不受控制，在奔狼领被北风之狼击败后于风龙废墟瘫痪。[9]之后古国研发了其他各式战争机械，包括：
古国研发的许多机关中，有许多都利用了仿生技术，比如模仿龙蜥对元素的反制力的遗迹龙兽、模仿海洋生物与蜂巢的遗迹机兵等等。[12][2]透过遗迹巨蛇，古国得以在地下建立四通八达的通道。[13]
探寻无果的远征坎瑞亚向龙脊雪山派出了许多的耕地机（遗迹守卫），试图从这个覆灭的文明探寻些许的痕迹。「即使来自天空的不祥之物，亦可为我所用……」——疑似前往探寻层岩巨渊的天钉的黑蛇骑士「为了我们的国家，我们无法放弃这从天而来的力量，但是我们失败了」——无果的远征坎瑞亚为了研究从天而来的力量（寒天之钉），派出了大量的遗迹守卫前往雪山。到达雪山的研究者们四下寻找，最终找到了当年寒天之钉从天而降分裂的碎片。在冻结雪山的「长钉」降下时，风雪遍天。天降之柱崩裂为三。他们对寒天之钉的力量加以研究，试图获取这个力量，但是他们失败了。而他们带去的机器人大多也都损坏了（游戏中，龙脊雪山区域可以看到大量损坏的遗迹守卫）。于此，无果而归。
