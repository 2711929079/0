# 渡火者的智慧

流浪烬寂海的智者（渊上）渡过能前往任何时间点的烬寂海到达许多过去的文明，换过不同的身分，期间身在四百年前某文明时身分为学徒，用赤色玛瑙打造了一尊冠冕，用以适应烈火与高温。智慧与灼热执念成就了防火的冠冕，却引来了同僚和前辈的恐惧妒忌。「这自大的年轻人，竟敢挑战烬寂海的烈焰，这是百年来未有的亵渎」「火海必将他吞噬蚕食净尽，连灰烬也被热浪扬入高空，终化做虚无」善妒的老师企图刁难他的门徒，便要他戴上冠冕，步入火海。渊上见眼前之人这副德性，知道这里不是心目中的故乡于是离去。那位导师眼见这冠冕的主人在岩浆中悠然漫步，消失在了视野之外。[137]（关于渊上的时间线，这里只是解读之一，另有其他推论）
渡火者的决绝火红的花朵，仿佛闪光灼灼的玛瑙石一般。这浴火的花，被火中通行的智者佩在胸前。在渡火的智者（渊上）生命的最末，他对人们说道：「这是受烈焰焚烧而盛开的花。若我未被化作烟灰」「热浪与黑烟中必然会有这朵花闪烁着余火的光辉」后来，人们追踪着明灭的光芒，来到了烬寂海的边缘，却发现智者已经不知所踪，而花依旧盛放在余烬之中。[138]
