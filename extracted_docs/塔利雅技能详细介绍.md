# 塔利雅技能详细介绍

塔利雅是《原神》中的草元素四星角色，须弥的学者，称号"智慧之芽"。
元素战技「智慧之术」：
- 释放草元素能量，对周围敌人造成草元素伤害并施加"草元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「智慧之光」：
- 创造一个持续8秒的草元素领域，领域内的敌人会受到持续草元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "智慧的祝福"：智慧之术命中敌人后，有15%概率触发"激化"效果。
- "草元素增幅"：塔利雅的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "智慧的守护"：元素爆发持续时间内，塔利雅的草元素伤害提升15%。
