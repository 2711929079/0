# 八门

「七星下辖又有八门，内外诸业无所不包」目前八门未在游戏中完全出现，以下仅列出已知部门和分管事项。
总务司
负责璃月一切事务的机构，主管八门的机构。
和记厅
主要负责海运物流、交通运输的调配工作，也会适当干预相关的建筑工程和路政，因此雇员较多。镖局是和记厅的下属机构。合辙驿是和记厅在遗珑埠设立的办事处。
银原厅
主管盐业。其中有原盐之魔神赫乌莉亚的遗民。
辉山厅
主管矿业，负责璃月矿区（明蕴镇、层岩巨渊等）的开采工作。工头老戴。
盛露厅
前身为层岩巨渊中以开采琉璃晶砂，制造和贸易瓷器为生的部族，归化后曾在镇压若陀龙王时立下战功，绯云坡公告板上盛露厅和天权就曾因琉璃行业复工进行谈判。现主要负责琉璃行业，古闻之章二中负责人文古迹研究的琬玉也来自盛露厅。可能兼顾人文古迹研究。行业代表若雨女士。
流金厅
与铸币发行有关。
