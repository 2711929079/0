# 辛焱

出身于贫困农家的辛焱背负着父母极高的期望，各种各样的“成见”支配了辛焱的童年。直到有一天，辛焱接触了来自枫丹的“摇滚”，自此开始用“摇滚精神”反抗其他人对她的“成见”[172][173][174]。辛焱改造了从枫丹传来的摇滚乐器，作为摇滚的践行者在璃月港进行表演。一开始辛焱搭建的临时舞台总是会在表演最热烈的时候燃烧起来，引起了璃月上层的注意，以“防止民众受伤”为由，禁止了辛焱的演出。但一位名叫 “大坤”的千岩军是辛焱的热衷粉丝，总是偷偷告诉辛焱去往千岩军巡逻的死角进行演出[175][176][177]。凝光专门修订了法典，划定了一块演出专属场地，只要申请了许可，便可以进行演出[178]。云堇非常欣赏摇滚乐，与辛焱成好友，经常去听辛焱的摇滚演出。因为大人们总是教育云堇不要去听摇滚，于是云堇与范二爷合作，捏造了一位名叫“星燕”的端庄大方的女生。云堇每次去听摇滚，范二爷就会对别人说云堇在和星燕相处[179]。
夜兰夜兰曾与一群同僚参与情报调查工作，直到一次深渊调查任务后，夜兰失去了所有的同伴。夜兰因此认识到自己的不足，开始闭关苦练，过程中获得神之眼。后来凝光找到了夜兰，并与其达成合作，夜兰就此成为凝光的特别情报官[180]。
