# 迫近

彼时的坎瑞亚，使用一种名为阿索斯物质的地脉能量为机关供能，[30]但坎瑞亚仍不满足，妄想取得新的能源，以取代地脉能量。[11]为此，他们向天空祈祷。“天空回应了招唤”，旅行者的血亲降临在坎瑞亚[31][32]。旅行者血亲原本沒有來到坎瑞亚前的记忆，直到在坎瑞亚待一阵子后记忆才逐渐浮现[28]。因为不知道自己是谁，沒有一切记忆的情况下，旅行者的血亲被迫成为坎瑞亚王储。坎瑞亚人将其视作「深渊」的代表。旅行者的血亲感受到了坎瑞亚人民的善意，将坎瑞亚视为自己的第二故乡，决定帮助他们，伊尔明将旅行者的血亲立为王储，并带着他/她一步步下到了王国的最深处。此时的戴因斯雷布，还不过是个默默无名的青年骑士[33]。旅行者的血亲降临坎瑞亚后，曾和当时身为宫廷法师的皮耶罗生活一段时间。[31]之后，旅行者的血亲成为坎瑞亚的“公主/王子”，坎瑞亚人遥望着这位金发异乡人，希望能掌控深渊之力的旅行者血亲能帮助他们开启通往新能源的大门—因为他/她就象征着深渊，象征着超乎常理的一切。[1]黑王伊尔明与高层们大力鼓吹深渊力量，旅行者的血亲甚至被称为「救世主」[28]。至此，坎瑞亚人引入深渊力量，灾厄随之降临…
蒂莱尔坎瑞亚偏远村庄的少女蒂莱尔从小就憧憬「救世主」的故事，虽然因为某个路过的，被大家认为无所不知的圣人（是女性，可能就是伊斯塔露）所述的影响，大家都说「灭顶之灾终将到来」，她仍然打算寻找救世主，求助她/他拯救自己的世界。她因伊斯塔露干涉遇到了来自未来五百年后的旅行者，为了证明自己见过他/她，接过其手中飞船休眠舱钥匙（也可选择收回不给她）[28]。
