# 阿貝多

「原初之人」实验的失败品因吸收杜林腹中之血复活，因为嫉妒完美品阿贝多打算取代他并来到人类社会生活[21]，后被阿贝多击败杀害。杜林吸收了失败品的遗体后得到他的记忆，对阿贝多也萌生同样的情绪[71]。
渊上日月前事渊上到渊下宫寻找书籍《日月前事》，他伪装成稻妻人，在渊下宫的图书馆等待旅行者，并为其提供指引，引导旅行者替其通过试炼。在旅行者通过最终试炼后，渊上想要杀死旅行者夺书，被击败而撤退[81]。
