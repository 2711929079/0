# 浅濑 响

又名「阿响」。昆布丸（雾切高岭）的意中人，两人惺惺相惜但没有结果。四五百年前的稻妻人物，有一只活了四五百年名叫[寝子]的猫，做的一手好猫饭。是[浅濑神社]的最后一代，与惟神晴之介、御舆长正曾一同求学于影向天狗，被合称[影向役者三人组]。[1]后曾在[鸣神大社]见习，与狐斋宫结交。后返回[浅濑神社]，与海贼首领[赤穗百目鬼]相交莫逆。
死于雷暴。
