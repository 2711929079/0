# 半年前

骑士团远征法尔伽魔女B让通过试炼的法尔伽看了自己命运的水占盘，法尔伽在水占盘中看到自己命运的歧路，他可以选择在蒙德击败风魔龙，成为蒙德的英雄，登上人生的巅峰，也可选择前往挪德卡莱对抗终将扩散至蒙德的狂猎，解救人们的性命。法尔伽自然选择了后者。法尔伽率领西风骑士团五分之四的人再一次离开蒙德，踏上远征之路。[151]其中包括了西风骑士团的所有骑兵[152]和西风教会的主教西蒙·佩奇（芭芭拉与琴的父亲）。西蒙离开后，教会的事务都交给卡尔文枢机卿打理。[153]临走时让副团长琴担任代理团长一职。[149]他是征服与创造传奇的骑士，而她是守护和平与自由的骑士。「骑士团就交给你了。反正这些年也是你在做团长的工作。」「放心交给我吧，大团长。」等你回来，将会见到一座更加温暖、繁荣、平和的蒙德城。[154]黑火案使团在西风骑士团的护送离开之际下遭到了某人使用黑火（黑炎之蛇）袭击，两名使者身亡。主犯（柯莱）趁乱潜入蒙德城。迪卢克因邪眼的关系遭到骑士团怀疑，但确认了其清白。[128]柯莱柯莱来自须弥，曾是「愚人众」十一席之一「博士」人体实验计划（注射魔神残骸）的实验对象。[155][156]来到蒙德后与「侦察骑士」安柏相识。柯莱告诉她，来到蒙德是为了寻找一样东西（寻仇）[155]回收「愚人众」为了回收「文件」与「种子」投放机关鸟，企图吸引魔物，破坏祭典，引发混乱。骑士团在发动之前将其全部摧毁。[156][141]
