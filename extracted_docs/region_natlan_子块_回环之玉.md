# 回环之玉

为了抵御深渊灾厄，科潘前去人迹罕至的废墟中挖掘秘源机械，并将其改造为能够帮助战士们战斗的机械。改造过程进行得十分顺利（疑似是因盗火贤者所为），直到有一天，科潘在一座庞弃古城的深处，读到了有关龙族曾创造出能够一举消灭深渊力量的“回环之玉”的织卷，织卷还提到其就在奥奇卡纳塔上空的天蛇船上。科潘将织卷上的信息分享给了其他人，自此开始有冒险者前去奥奇卡纳塔子找“回环之玉”[120]。深渊灾厄几乎摧毁花羽会，于是泰卡告别妻子，将尚未出生的孩子取名玻娜。流泉之众巡山客（山间导游与救难队）蒂图、驯兽师雷格巴、龙遗物研究协会会长科潘、此时花羽会最强战士玻娜之父泰卡、秘术师莉莉乌组队寻找能据说能清除深渊的奥奇卡纳塔秘宝回环之玉，但全员因邪龙袭击无人生还。玻娜在探索奥奇卡纳塔的过程中从统律核心解放了奥奇坎的意识，让其化作小龙「绿松石」与其一同探险，但最终也在奥奇卡纳塔牺牲，被瓦萨克拉胡巴肯取走灵魂[69][89]。
坎瑞亚小队尚未加入愚人众成为执行官，当时率领一批小队的「队长」在战役中且战且退来到纳塔，见证战役带来的巨大牺牲和深渊带来的恐惧与绝望。在纳塔，他认识了当时烟谜主首领阿伊祖，两人互相交好，队长因此学会烟谜主的法术，能将外溢的灵魂送回地脉。同时他也从阿伊祖口中得知火神之心的用法。[13]
