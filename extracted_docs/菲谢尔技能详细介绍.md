# 菲谢尔技能详细介绍

菲谢尔是《原神》中的雷元素四星角色，断罪之皇女，称号"幽夜华尔兹"。
元素战技「夜巡影猫」：
- 召唤奥兹，持续攻击敌人并造成雷元素伤害。
- 长按可以调整奥兹的位置。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「至夜幻现」：
- 释放雷元素能量，对周围敌人造成雷元素伤害，并强化奥兹的攻击。
- 冷却时间：20秒。
- 能量消耗：60点。
固有天赋：
- "断罪的皇女"：奥兹的攻击有10%概率触发"感电"效果。
- "幽夜的眷属"：菲谢尔的普通攻击和重击命中敌人后，有20%概率减少元素战技的冷却时间1秒。
- "皇女的尊严"：元素爆发持续时间内，奥兹的攻击伤害提升20%。
