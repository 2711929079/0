# 一心流

由经津流发展出的一心流，三位弟子枫原景光、丹羽长光、赤目实长，分别又发展出了不同的三个派别。一心流所追求的事人剑一体，因此锻造出的刀倾注锻刀者的心境，颇具个性。向来都是人选刀，但一心流乃刀选人。枫原一族博闻强识，作品最具真砂丸风骨。丹羽一族宅心仁厚，以烧刃见长。(烧刃指的是在日本刀上可以看到的类似波浪花纹的东西)赤目一族认为「一心」即为无冗余念想，为纯粹的目的锻刀，因此锻刀时加入祟神之活念，便能锻造出活刀（踏鞴砂事件中就是当时赤目一家将埃舍尔使用祟神之力锻刀的技术引荐给丹羽）。而刀乃杀生之物，必见血光，因此追求「斩人刀」的终极。但也因为如此，赤目一族佩刀虽极其锋利，却因长年于祟神之念共处，身心皆伤，寿命皆短。并被官家评为「不良」，而未继续成为「一心流」的总长。初代目惣领为赤目实长，但因赤目一流锻刀过于凶煞，为官家所恶，是故其惣领一职仅传三代即遭剥夺。丹羽长光的后人丹羽久秀与赤目实长的后人在踏鞴砂锻刀，其中丹羽久秀还成为了造兵司正[46]。
倾奇者人偶在初代雷神雷电真死于坎瑞亚之战、二代雷神雷电影肃清稻妻的灾厄后，影决定以某种最初源于坎瑞亚的神秘技术制造对抗磨损的人偶。她透过白色的枝条，於借景之馆[60]制造了一名人偶试做品，然而无名的人偶却因容纳雷神之心在睡梦中流下眼泪。作为容纳神之心的容器,影认为他既没有人类的坚忍不拔，亦没有器物的寡淡无情，实在过于脆弱,与永恒的理念相违背。八重神子原先建议将人偶直接毁去[61]，但雷电影不忍毁弃自己的造物，于是将人偶的机芯「神之心」取出。原先把它放在沈眠之庭門口[62]，后来把他安置在据说是古时避世的武人在大地深处借世外之景，修建的华美宅邸借景之馆[63]，并为他留下一枚金色的羽状饰品作为身份凭证。在制造之后的人偶时，影舍去了安放「心」之处的设计。[31][46][64][65]人偶在得名倾奇者之前还有个名字。但这个名字没多少人知道，就连人偶自己也忘记了。[65][66]
