# 远征

法尔伽通过魔女B的试炼，观看了对方的水占盘，发现有一场席卷提瓦特的深渊灾厄（就是「狂猎」）正在北方酝酿，此后也会染指蒙德，于是他带人前往挪德卡莱。远征期间，远征队碰到了「队长」，向他了解了一些北方深渊灾厄的内容，后来他单独前往纳塔，找玛薇卡交换对抗深渊的战术和知识。期间远征队有很多人回到了蒙德，驻扎在拜达港。现在远征队在安瓦蒂尼尔湖一带扎营，除收集深渊情报外，还对安瓦蒂尼尔湖进行测绘与侦察。根据记载，安瓦蒂尼尔湖与过去的「北风骑士」以及狼主安德留斯都有相当密切的关联（见至冬#狼与仙灵）[2]。
