# 戴因斯雷布

你与戴因斯雷布见面，得知他正在追踪深渊教团的踪迹。途中你发现血亲留下的气息…你的血亲曾在深渊教团活跃处出现过。
旅行者相见你与戴因不断深入探索着教团据点，并找到了「污秽逆位神像」与「深渊使徒」。原来深渊教团打算将奥赛尔结合「原型机之眼」和污秽逆位神像打造命运的织机，借此动摇天空岛的神座。于是戴因率先藏匿原型机之眼。期间你得知深渊教团的成员是坎瑞亚的移民，坎瑞亚在500年前因天理的制裁而覆灭，为此深渊教团决定反抗天理。而且自己曾参与坎瑞亚战役——代表你与你血亲被神灵封印的時间是500年前。你与戴因合力牵制住了「深渊使徒」，占据了上风，眼看就要胜利之时，一道熟悉的身影出现将你们击退。你难以置信地发现，从你们手中救下「深渊使徒」的竟是你苦寻已久的至亲。你得知戴因和你的血亲早已见过面，并且戴因正在对抗深渊。当你还没了解事情始末时，你的血亲离开了。戴因斯雷布追了上去，但你没能追上进入传送门的他们。
