# 赤鹫

以残骸为食的大型鸟类，赤红色的羽毛在阳光下显得有些稀疏。这些鹫鸟并非原生于荒漠中的物种，而是在猩红之王统治的古老文明消亡后才出现在沙漠之中的。同时由于其特殊的生活习性，在某些沙漠遗民看来，它们乃是带来死亡与灾厄的生物。
