# 发条机关

数百年前至今，枫丹陆陆续续出现许多新的发条机关。攻坚特化型机关与“侦察记录型”一样，是最早设计制造的新式发条机关机械之一。名称是用来纪念发明者的友人西摩尔而起的，最初设计的目的是帮助人。但后来的使用者普遍认为用它攻击人较之用人攻击人的相对优势更大。[64]区域警戒型是在枫丹科学院建立之前，自然哲学学院的相关研究中产生的试验装置。因此，尽管作为警戒与防御设备取得了相当的商业成功，大多数工程师并不将其视为严格意义的发条机关机械。[65]过去在枫丹曾经发生了巨大魔兽入侵的悲剧，在陆上大湖受到景仰的白色铁甲舰队为了护国全军覆没。深海攻击型的设计者希望如有此类事件再次发生，至少能在水中以强大火力出击的发条机关能先于将官与士兵成为第一道防线。[66]
