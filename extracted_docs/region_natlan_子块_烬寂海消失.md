# 烬寂海消失

有人在烬寂海目击到一个黑色山岳般的巨物（巴窟纳瓦）在其中移动，不久后，整个烬寂海突然消失。巴窟纳瓦历经五百年复甦，为了避免其卷土重来，只得再次封印烬寂海，使烬寂海从纳塔消失。烬寂海消失后，该地才开始出现人烟。[118]「阿夏」是特诺奇兹托克地区特有的一种幻写灵，比普通幻写灵更聪明，更亲近人，于烬寂海消失的同时出现在特诺奇兹托克地区。在特诺奇兹托克的文化中,「阿夏」这个词有两个含义，其中之一是「好灵魂」，另一个含义则是「灰烬」。五百年前，为防止污染扩散，温迪将烬寂海从时间轴上吹了出去，无法回归夜神之国的灵魂沉淀在这块土地上，与幻写灵结合，便形成了阿夏。班尼特的亲生父母便是以阿夏的形式存在着，直到班尼特来到特诺奇兹托克。纳塔局势稳定后，温迪与旅行者等人到访悠悠度假村，风火两神合力将愿意回家的灵魂接出了烬寂海[96][129]。二三十年前冒险团来到特洛奇兹托克，虽然这个冒险团没有进入烬寂海，但却误入了疑似烬寂海的区域「倒影之海」，不只一处，就好像烬寂海在世界上投下影子一样。原有很多冒险家进入过，但后来再也没有出现（直到十几年前班尼特的父母也来过，所以可知十几年前还是在的，后来不知为何消失了）。[130]
