# 金色的那菈

旅行者的血亲前往恒那兰那，帮助兰那罗解决遭到深渊侵蚀的法留纳神机的危机，与兰那罗们一同度过了快乐的无忧节，并与兰穆护昆达和兰帝裟一并打败并封印了「无留陀的使徒」。协同作战的兰那罗有兰穆护昆达，兰拉娜，兰帝迦、兰般度等第一批高端战力的兰那罗种子。她/他经常和兰那罗们说起自己血亲的事情，让一部分兰那罗知晓了家庭的概念，并也开始以兄妹相称。而他/她留下的笔记本被兰纳罗称作「森林书」。旅行者的血亲为了让须弥雨林更好的恢复，留下了神器「无郁夷摩」以及看守神器的生命之杯「阎迦由玛」（可能都来自坎瑞亚）。神器有效的恢复着雨林的生态，並设下了封印，保护好雨林的重要植物。在无忧节中，旅行者血亲为兰那罗们做了很多菜肴。为了能够在下次无忧节上能让旅行者血亲吃到自己尽心准备的至上美味，兰那罗哥哥们进行了长达五百多年的厨艺修行。无忧节后觉王树孕育出了惟耶之实。当时可能是使用的村长兰拉迦的记忆，所以兰拉迦失去了对旅行者血亲的记忆。旅行者血亲知道真像后痛哭流泪，向村长讲述了一起战斗的故事。无忧节过后，旅行者血亲与兰那罗们击败了死域核心，虽然最终击败了死域的化身，但是无法彻底根除，所以兰那罗只能进行封印。兰那罗构建了四重封印将其束缚在了他们曾经的故乡往昔的桓那兰那。虽然漆黑灾厄与旅行者血亲无关，但是她/他毕竟与坎瑞亚关系密切。她此时是带着愧疚之情的，所以在走之前并没有留下名字。金发人类临行前告诉兰那罗们，日后一定会有另一个与他/她相似的人与他们相会。因为金发人类没有留下名字，于是兰那罗们便根据其修好「法留纳神机」的功劳称呼他/她为「那菈法留纳」。[30]之后，旅行者的血亲前往甘露花海，遇见戴因斯雷布。[72]
西摩格五百年前的坎瑞亚灾厄中，前代水神厄歌莉娅死于甘露花海，身躯化作「甘露活水」。神鸟西摩格饮下「甘露活水」，将自己拥有的「灵光之力」与甘露相融合，形成灵光百种。其中具有智性的即是花灵。最初的花灵祖尔宛，与其他的花灵都参与了对深渊兽潮的战斗。花灵和戴因斯雷布、东方来的那伽朱那团的独臂贤者、那伽朱那、坎瑞亚的残军一起对抗过漆黑兽潮。其后，一部分花灵和它们的搭档组成了最初的矫论团。在那之后，戴因斯雷布与旅行者的血亲离开了。[72]
