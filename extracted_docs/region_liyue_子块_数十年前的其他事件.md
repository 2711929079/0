# 数十年前的其他事件

刻晴的祖父是璃月著名的土地业大亨，也是一位学者，酷爱搜集古籍，曾与夜兰的家族有过学术合作。夜兰的祖辈曾为七星效力[155]。三十年前，一位名叫 “若心”的戏子在“和裕茶馆”表演，成为一代名角。若心曾经常与现如今的 “范二爷”搭戏，而且总是在台上扮演一对，让范二爷对若心心生情愫，但若心倾心于一位冒险家[156][157][158]。
