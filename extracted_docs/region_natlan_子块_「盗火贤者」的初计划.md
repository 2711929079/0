# 「盗火贤者」的初计划

“葬火之战”后，重建了国度的哥哥修库特尔由于深渊侵蚀，自感时日无多，便任命弟弟库库尔坎作为摄政，让弟弟代行统治工作，同时还建造了「维奇洛波奇特利」。库库尔坎相当鄙夷盲从强者的其他龙族，且强烈反对其兄长开启「维奇洛波奇特利」。（以上内容前文已有相关的详细叙述，此处不再赘余。）库库尔坎将龙的未来寄托在了人类身上。它化名“瓦萨克拉胡巴肯”，将使用燃素的技术教给了察阿克等人类（一些纳塔人认为「盗火贤者」是人类），但人类却陷入了内部的征伐，至高领主中第八席「花烛与风羽的司巫」伊·卡科茨·伊齐玛瓦夏甚至误以为库库尔坎之所以泄露燃素技术，是有意在引诱人类自我毁灭。库库尔坎又意图创造一个同时拥有龙与人的血脉的存在，去击败修库特尔，开创龙与人的新时代。因此他以实验体为由，将人类的灵魂融入了子嗣奥奇坎的躯体中，使其成为了半人半龙的特殊存在。库库尔坎通过伪装为意外出逃的方法，将奥奇坎放入了人类社会，并谎称其已被处决。此外，圣城内的一部分龙族也同样不认可火龙王修库特尔的计划，于是库库尔坎庇护了他们，为他们找了一块在圣山之外的实验地，来进行“燃素状态转换器”的研究，这项发明可以将燃素转化为坚固的结构，从而搭建对抗深渊的防御工事。此外，库库尔坎曾暗中推动丁加（马古汉）离开他侍奉的少女“圣主”，投奔希巴拉克，以图消灭在他看来的“堕落”龙族（但最终目的并未完全达成）。
