# 错位时空

旅行者的血亲身上的深渊力量被五大罪人瓜分。蒂莱尔终于找到了「救世主」（旅行者的血亲），然而旅行者的血亲看到她手中来自五百年后的飞船休眠舱钥匙，以为旅行者醒了并掉了钥匙得蒂莱尔捡到，陷入「拯救坎瑞亚」和「救旅行者」两难选择，最后选择救旅行者，到了飞船休眠舱时却发现旅行者还没醒来。此时旅行者的血亲已来不及拯救坎瑞亚，坎瑞亚随之覆灭。蒂莱尔打算拯救坎瑞亚的努力，却导致坎瑞亚的覆灭，这就是蒂莱尔的命运。（旅行者也可以将钥匙收回，但结果是一样的，因为五百年后旅行者的血亲也会受到伊斯塔露的干涉见到蒂莱尔，为了防止收回钥匙导致涟漪效应使世界走向未知的命运使自己的计划（即自己因为被降格了，所以成立深渊教团用一整个世界与天理对抗打算获取与之平等对话的资格恢复降临者身分，和让新坎瑞亚成为自己合适的新的家乡）（有说法降格的原因是书籍裴伦埃里中教养院的神秘仪式所致，但终北的夜行诗中「仆人」重现此仪式时，仪式传来对话：降临者⋯没有权限⋯，因此可能无法使降临者降格）不能如计划中进行，于是将自己一样的另把钥匙丢给蒂莱尔。旅行者在内心纠结下顺从血亲的计划。）[28]
海洛塔蒂海洛塔蒂是深秘院院长，因为总是提出伊尔明「想要看到」的方案而受到重视以获取利益。此做法引起莱茵多特不满，认为伊尔明背地里比较重用他而损失自己利益。后来伊尔明几乎陷入疯狂与偏执，戴因与五罪人—以及坎瑞亚的大部分人都认为他是昏君[40]。
