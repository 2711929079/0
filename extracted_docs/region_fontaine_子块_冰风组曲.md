# 冰风组曲

科培琉司是自然哲学院成员之一，后成为剧作家。[34]发条机关「冰风组曲」的制作灵感来自曾在剧院仅上演一次，仅演出过前几幕的剧作《发条的歌裴莉娅》，讲述的是发明家倾心于拥有所谓「人心」的发条机关歌裴莉娅的故事。该剧在上演前备受评论界与文艺界的期待，但在初次上演的第一次幕间休息时在世界各地刻下深重伤痕的坎瑞亚灾难发生了。它所留下的故事只剩下依据当时有幸光临剧院的观众拼凑出的前几幕台词，以及残垣断壁与过期报刊上留下的广告。《发条的歌裴莉娅》的剧本并未公开、出版过。当日光荣剧院的幸存者当中，有部分观众依据记忆转写了幕间休息前，部分演出的情节与台本。[49]征求科培琉司先生遗族与上述人士的同意后，敝社将相关资料整理、综合，力求还原科培琉司先生原作的一角。虽说不过是管中窥豹，但求能抛砖引玉。那是很多年以前的事情，在当时欧庇克莱歌剧院尚未开始修建，而那间剧场也没有在被毁后被人买下并改建，并在无数次的易主后成为如今的德波大饭店。[50]
