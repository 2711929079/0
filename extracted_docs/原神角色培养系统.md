# 原神角色培养系统

《原神》的角色培养系统是游戏的核心玩法之一：

1. 角色等级：
   - 通过消耗经验书提升等级
   - 达到一定等级需要突破才能继续升级
   - 突破需要对应的元素材料和BOSS材料

2. 天赋系统：
   - 每个角色有普通攻击、元素战技、元素爆发三个天赋
   - 天赋升级需要消耗天赋书和BOSS材料
   - 天赋等级上限为10级，可使用皇冠提升到10级

3. 武器系统：
   - 武器分为单手剑、双手剑、长柄武器、弓、法器五种类型
   - 武器稀有度：三星、四星、五星
   - 武器升级需要消耗武器经验素材
   - 武器突破提升基础属性

4. 圣遗物系统：
   - 圣遗物分为花、羽、沙、杯、冠五个部位
   - 圣遗物稀有度：一星到五星
   - 圣遗物主属性：每个部位有固定的主属性类型
   - 圣遗物副词条：随机生成的额外属性
   - 圣遗物套装效果：收集特定数量的同套装圣遗物激活加成

5. 命之座系统：
   - 通过重复抽取相同角色解锁命之座
   - 每个角色有6个命之座
   - 命之座提供额外的技能效果和属性加成

6. 元素共鸣：
   - 队伍中相同元素角色数量达到2或4时触发
   - 提供各种加成效果，如元素伤害、元素抗性等

7. 培养优先级：
   - 主C角色：优先升级元素战技和元素爆发
   - 辅助角色：优先升级元素战技，元素爆发视情况而定
   - 副C角色：平衡升级各项天赋

合理的角色培养是提升战斗力的关键，需要根据角色定位和队伍需求进行针对性培养。
