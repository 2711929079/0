# 特诺切

坎瑞亚灾难爆发，但此时纳塔人几乎失去一切，人们无心团结对抗深渊。六大部族仍然争执不休，玛薇卡于是委托不属于六大部族的英雄最后的特诺奇兹托克部族人[96]特诺切联合六部族的部分勇士，前往解决灾难。玛薇卡告知特诺切自己的计划：自己将把力量注入圣火，在五百年后甦醒对抗深渊。第一位前来迎接英雄特诺切的，是善奔的万杰鲁与她聒噪的伙伴卡乌克，她黝黑的皮肤仍留存着与英雄一同冒险的证据，深深的伤疤仍清晰可见。虽说各大部族彼此争扰不休令人疲惫，但特诺切的邀请让她再焕发活力。即将为纳塔炽热的大地而投身激战，万杰鲁与族人们又有何拒绝的理由？第二位前来拥抱英雄特诺切的，是勇猛的孟尼力克与他忠诚的伙伴恩古布。「来吧，来吧！尽管诸部族已将你逐出族人的视野，即使我们总是不合」「但在孟尼力克眼里，持棒的特诺切始终是壮士中的壮士，兄弟中的兄弟」第三位被英雄特诺切寻到的，是狡诈的桑哈杰·康波雷与他难以捉摸的伙伴马汉巴。桑哈杰过去曾被「聊聊棒」说服，英雄答应让出战利品，他才终于加入战士的队伍。康波雷预见到特诺切与自己的壮烈结局，与日后被称作「烬寂海」之地将如何诞生。「然而无妨，无妨，或许以毒蛇般精明闯出名声的恶棍，理应有那样逞英雄的一天」第四位自愿追随英雄特诺切的，是年轻的布基纳与他那莽撞的伙伴康加玛托。布基纳未曾经历过特诺切的苦战，但雄伟的黑曜石棒引他走上了凶险的前途。战争在特诺切身上留下的无数伤疤，让布基纳确证了自己接下来要走的道路…变革本应是年轻人的命运，为炽热的公义流血，也好过在安逸的草榻上腐烂。
第五位被英雄特诺切说服的，是矿山的首长松伽塔与他稳重的伙伴穆胡鲁。「当年为了诸部的稳定，为避免再起争端，我曾向火神请愿投票将你放逐」「如今你却仍不放弃血战…也罢，或许这是我们这一代人最后的战斗了吧」「既然你决心一意孤行，那么我没有理由不来奉陪，但不要牵涉我的族人」第六位与英雄特诺切同行的，是他年轻时的死敌巨人图帕克，没有坐骑能承担他庞大的身躯。特诺切提着大棒来到图帕克栖居处想找他聊聊时，他正细数着身上被挑战者留下的众多伤痕。「三百多处疤痕，二十多处骨折，还有一百枚黑曜石碎片深深嵌入了皮肉，小小的珠宝一样」「两百来处轻伤，两根肋骨破碎，一只眼睛再也分不清远近，你给我留下的战利品同样丰厚」曾经的两位敌手哈哈大笑，就此联起手来。就这样，特诺切从六大部落召集齐了盟友，如明亮灿烂的野火般，与黑色的山岳相撞…[97]之后特诺切、桑哈杰与布基纳在烬寂海的战争中率先牺牲[117]，但由于特诺切失去了古名，在战后的还魂诗中无法复活。特诺切之死是因为桑哈杰对抗巴窟纳瓦时，自愿让桑哈杰在身上画下用以施放边界法术的燃素刻录然后被巴窟纳瓦吞噬，此时桑哈杰再召唤并引爆一个微型夜神空间而受到波及，桑哈杰与特诺切都因此而死。引爆的微型空间炸出烬寂海[118][96]。玛薇卡收藏了特诺切的武器「聊聊棒」作为纪念。[10]战前桑哈雷曾写过自传，名为「狡诈的康波雷」，但绝大部分内容被认为是虚构[33]。
山王「山王」康加玛托曾经在坎瑞亚灾变时随布基纳与众英杰们携手面对漆黑潮水，因此在「悬木人」部族的成员心中具备极重的分量。[119][97]在战役中，布基纳被因深渊侵蚀而发狂的山王攻击，在死前将自己的力量注入给山王。山王在此后一直希望了结自己以赎罪。「山王」在战役中受深渊污染，每当在污染侵蚀严重时处于清醒状态便会狂暴，于是悬木人部族会举办仪式让「山王」睡着，再清理牠身上的污染。[18]
