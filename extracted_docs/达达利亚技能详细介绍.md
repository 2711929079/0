# 达达利亚技能详细介绍

达达利亚是《原神》中的水元素五星角色，愚人众执行官第十一位，称号"公子"。

普通攻击「弓术」：
- 效果：发射水箭，造成水元素伤害。
- 蓄力攻击：发射强力水箭，造成高额水元素伤害。

元素战技「魔王武装·狂澜」：
- 效果：切换到近战形态，普通攻击和重击转化为水元素伤害。
- 断流机制：攻击被水元素附着的敌人时，触发断流效果。
- 持续时间：15秒。
- 冷却时间：12秒。
- 能量消耗：无。

元素爆发「极恶技·尽灭闪」：
- 效果：释放大量水刃，对周围敌人造成高额水元素伤害。
- 断流效果：触发大范围的断流·爆效果。
- 冷却时间：18秒。
- 能量需求：60点。

天赋技能：
1. 断流：攻击被水元素附着的敌人时，触发额外伤害。
2. 永无谢幕：元素爆发造成的伤害提升。

角色定位：
- 主要定位：主C输出
- 次要定位：副C
- 核心优势：灵活切换形态、高额水元素伤害、断流机制

培养建议：
- 武器推荐：冬极白星、飞雷之弦振、绝弦
- 圣遗物推荐：沉沦之心四件套（水元素伤害加成）
