# 「灰河渡手」与爱德华多

在过去曾经被当作武器使用的趁手废弃铜管，曾经在一些人当中享有盛名或是臭名昭著。[41]雅各布的生父爱德华多•贝克与因城市优化而被迫进入下水道「灰河」生活的其他人们没有什么不同最初他只是拿着捡到的铜管驱逐鳄鱼，再后来是保护商贩店家免于强人帮派的侵扰勒索、调解灰河的「同胞」之间的纷争，为新婚者祝福。[41]除此之外，爱德华多在灰河经营酒馆，「帕西法尔」姐妹金盆洗手后的魔术表演就在他家的酒馆出道表演。[40]就这样，爱德华多顺理成章的成为了「灰河」实际上的领袖，团结领导着着枫丹的灰色地带（刺玫会的前身），他惯用的铜管被众多朋友亲切的称为「渡手」[41]。最终爱德华多因在「灰河整肃」中组织反抗被捕，流放往沙漠。
蕾蒂西娅尽管贵族们将灰河视为肮脏之处，仍然有许多贵族对该地抱有好奇与向往。出生便不愁吃穿的大小姐蕾蒂西娅·兰道夫因为向往冒险，以及为了拥有和好友聚会聊天的话题来到灰河。一潜入该地就因为过于瞩目即将遭遇危机—所幸她先前认识的里德为她引路。她在灰河没有想回去的念头，和里德有了孩子。灰河整肃爆发时，蕾蒂西娅帮助人们对抗枫丹官方，失败后她、里德和孩子仍然被父亲接纳。然而事后在兰道夫家，她才回想起逐影猎人们潜入的暗道不应被外人知晓，也就是出了内鬼，而此人就是她丈夫里德，他将秘密泄露给兰道夫。事前兩人相愛時，里德曾對蕾蒂西婭說過：「蕾蒂西娅，我敬爱你高洁的灵魂。」「我们将不再会是砍向树木的斧柄。」「若有一日，我竟耽于尘世的荣华，」「那时，就由你来裁决我的命运吧…」于是，蕾蒂西娅舉起她的大斧，对丈夫做出最终的裁断[43]。兰道夫家族存有后人[44]。
