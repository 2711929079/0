# 仙灵退化

自外而来者带来了战争，亦为人类带来突破桎梏的妄想…而天的主人恐惧妄想与突破，降下修补大地的天钉，毁灭了凡人的王国…仙灵一族因为与外来者相爱惨遭遗弃，天理放逐了仙灵一族。仙灵一族失去了与天空的联系，也失去了教化的能力，从此背负可怕的诅咒，退化成只能凭借本能引导人类寻找宝藏的空壳。[41]
枫丹.梅洛皮斯古文明在最初的世代，初民们的道德足以自持，无需仰赖律法与威权的统治。天城的使者行走在大地之上。经由他们的指引，人们在自太古以来便一直延续的安宁中平静地度日，安享繁荣与丰饶。天启的统治不知延续了多少个世代，直到人们厌倦了难以捉摸的永恒。后代之人不再虔信神谕，反而希冀神明并未许诺之事，试图挣脱命运的枷锁。于是高天为之震怒，祂（祂：称上帝、耶稣或神的第三人称代词）命大海掀起巨浪，将定居者的城邦碾碎。随后大雨连降百日，潮水吞没了一切罪恶与妄想，枫丹初民的时代就此终结。[17]
