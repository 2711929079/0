# 献祭人习俗

盗火者引导人类发展，但部族还留着迂腐的原始献祭习俗，在大灵诞生后更加巩固，因为部族认为大灵是靠献祭产生的[51]。而且个部落间的人们击退龙族后却互相争斗，争夺政权[52][41]。部族的人们因为对抗龙族部族首领被12席「轻语如泉者」击杀，失去首领的他们还开始内战[39]。盗火者看不下去了，于是引退静观其变，因为根据自己的预言人类迟早会诞生一位有贤能的人击败火龙王[51]。掌握燃素使用方法的人类开始互相残杀，虽然不理解龙族如何从血液中提取生命力，但是人类依然模仿龙族的行为，通过杀戮向自己幻想出来的神明献祭。龙族原本虽然因为燃素被人类学习很愤怒，但看到人们拿着燃素自相残杀后就转而看戏了[39]。
平息鸣雷的尊者纳塔人的献祭习俗再现，在后来的烟谜主一带，平息鸣雷的勇士的亲眷皆被献祭。昔日该处一帶暴虐的雷之魔兽，曾以雷霆手腕驱役人。勇士参透万物兴衰，站出高傲地面对雷电。即使与雷之魔兽的血战，也未曾将之动摇半分。勇士冒着雷暴折却它的爪牙，最终斩下雷电之魔兽的头颅。然而亲眷已不在的他孤独终老[53]。平息鸣雷的尊者描述当时雷之魔兽的领地也存在雷之猛禽，可能就是鹤观的雷鸟卡帕奇莉或她的同胞。卡帕奇莉听过特瓦林的声音[54]，因此可能远离过清籁岛，又或者原本她就不在清籁岛，后来才迁居到清籁岛。（平息鸣雷的尊者时之沙右上方的图案长得极像烟谜主的涂鸦，像欧洛伦与厄水之祸上的图案。因此这里猜测其可能是讲述纳塔烟谜主的故事。因圣遗物故事中写到尊者血亲被献祭，因此时间线则推测位为纳塔献祭人习俗盛行时。平息鸣雷的尊者提及的雷鸟很像镜璧山上的紫色鸟涂鸦，可能就是雷鸟的形象。这个涂鸦也很像卡帕奇莉，或许就是卡帕奇莉或她的同胞。较有可能是雷鸟同胞，因为欧洛伦服装设计酷似雷的蝙蝠，而雷音权现的样貌就是雷蝙蝠，其简介描述「雷音权现只会随着雷霆降生在遗留着强大忿恨的地方，譬如说支离破碎、雷云黝黝的天云峠[55]」看来比较像雷音权现是个物种，它是个种群不是个体，而且只会在充满怨恨的地方出现，如天云峠。从这说法来看其他充满怨恨的地方也可能存在过雷音权现，而存在过雷音权限就代表存在过雷鸟，因为雷音权现是雷鸟遗恨形成的。若欧洛伦蝙蝠形象来自雷音权现，那么烟谜主很有可能曾存在雷音权限，又或者说存在过雷鸟。但以上均仅为推测。）
