# 几十年前

白朮胡桃爷爷的弟弟白附子是往生堂的二把手。两人年轻时外出游历，在沉玉谷遇到了一名药师，而后胡桃的叔公不顾胡桃爷爷的阻拦，抛弃家业，去到沉玉谷找那位药师学医。胡桃的爷爷继承了往生堂，成为第七十五代堂主。胡桃的叔公继承了药师所身负的长生的“珥蛇拖龙法”，在沉玉谷一代行医[143][144]。胡桃的叔公收了一位名叫“江蓠”的女孩为徒。几十年前，珥蛇拖龙法的传人是白附子，他也是往生堂前代老堂主的兄弟、现任堂主胡桃的叔公。幼年时期的白术的故乡遭遇一场疲疠，白术因此失去了双亲。继承了长生“珥蛇拖龙法”契约的医者白附子抑制了疾病的传播，白术从此跟随这位医者学习。江蓠比白术年长几岁，成为其师姐。白附子希望徒弟白术能够打破先祖的宿命，在师父死后，白术因不忍让长生断绝契约后死去，所以依然继承了珥蛇拖龙法，并开始追求不死之法，发誓要成为最后一任继承者，而后带着长生前往璃月港，开办了药庐“不卜庐”。白术曾在绝云间的山林里遇到了从琥珀中破封而出的七七，知道了她是僵尸后将其收留到不卜庐里，尝试研究出不死之法[145]。江蓠开始独自出去游历，期间与一位名叫“嘉良”的采药人成了家[77]。白鹤留云曾偶遇一只获得了仙力的白色野鹤，作为师父引导她修炼。50年前璃月突发旱灾，白鹤想要像师父那样拯救苍生，但那时的她尚不是真正的仙人，无法掌握仙人的化形之法，于是留云为她炼了一种「化形药」让她能化成人形，但药效只能维持十年，十年后若她没有变回鹤形，就会忘记自己作为仙兽时的一切记忆，变成非人、非妖、非仙的存在。然而白鹤在下山镇压旱灾期间与一个天衡方士相恋，因此背弃了十年后变回野鹤的诺言，写信与留云辞别后归隐于人间。以人类身份生活时，白鹤患上了严重的健忘症，生活难以自理。40年前，留云看见信后非常生气，本想去找白鹤兴师问罪，因为她背弃了诺言，而且她找的天衡方士悟性不足，驱使符箓毫无技巧可言。但留云在看到白鹤的现状以及丈夫照顾她的决心后心生怜悯，便易容成商人模样，在望舒客栈为她提供了一壶以自己血液炼制的灵药，才帮助她压制了几分化形药的副作用。当白鹤分娩时，留云还帮助她与爱人击败了追杀他们的魔神残渣。白鹤的孙女曾生了重病，留云以鹤形拜访希望医治，但白鹤的孙女坚持认为只有白鹤才是仙鹤，因此留云只得将自己全身涂成白色，驮着她去自己的洞天里医治疾病。[47]申鶴申鹤出生于驱邪世家（重云的世家）的旁系。她5岁时，母亲因病去世，父亲爱妻心切，去往远方寻求复活亡妻的办法。一年后，申鹅的父亲从至交“明俊”那里找到了一本书，书里记载了复活亡妻的方法。申鹤的父亲回到象中，向申鹤谎称一处山洞中有惊喜，申鹤走进山羽，父亲利用魔神残渣召唤出恐怖的“仙灵”，“仙灵”指出申鹤是孤辰劫煞，申鹤的父亲便決定用申鹤的生命换回自己妻子的生命，独自离开了山洞。申鹤拿起母亲道物中的一柄驱邪匕首与“仙灵”死战。数日后，留云借风真君循着邪祟的气息找到了在战斗中占据上风卻奄奄一息的申鹤，帮助她清理了魔物后，便将她带回府中修养，而后收她沟徒。这几天里，申鹤的父亲橋然醒悟，深感愧疚的他选择了自杀，而明俊得知这系列事情后一直深感愧疚。仙家收留申鹤后，削月筑阳真君为其卜卦，卦象显示她既犯孤辰，又冲劫煞。为了让申鹤安全长大又不殃及他人，仙人们为其身上施了红绳缚魂之法[146][147]。留云借风真君将“翠钿白玉梳”送给了申鹤，并告诉她：如果想斩尽凡缘，踏入仙门，就用这把梳子梳三下头发。申鹤毫不犹豫地照办了，原本的一头青丝变为了白发。[148][149]。修行了十多年的申鹤在一天心血來湖，回到了自己的故乡，发现自己儿时居住的村庄已被遗弃，父亲也早已离世[150][151]。明蕴镇一位名叫元鸿的矿工带着一家人在瑶光滩北部的山岭间开采矿石，白手起家建立了明蕴镇。明蕴镇曾经很繁荣，是仅次于层岩巨渊和黑岩厂的采矿中心，元鸿一直被人们认为在镇子里埋藏了一份祖传宝藏，连元鸿的三个儿子都信以真，纷纷前去寻宝并相继死于意外，元鸿也因此心力交瘁，最终病逝。在元鸿死后没过几年，明蕴镇的矿脉耗尽了，矿工们都转去层岩巨渊工作，明蕴镇也从此荒废，如今只有盗宝团会光顾此地[152]。志琼就出生于明蕴鎮，也跟随大家去往层岩巨渊[153]。小九九与长九爷长九爷幼年被叫九哥，与同样家中排名第九的小九九是小情侣，长九爷为了小九九写了本小说，在小九九有天边闲逛边看那本小说时，掉到水里不幸溺水，他知道这本书对九哥而言非常重要，于是紧紧抱住书直到溺毙，小情侣就此天人永别。因为家中这个变故导致小九九家举家搬迁，这是为了不让长九爷有天发现小九九死讯，而不知情的长九爷一直认为是他们家看不起自己，觉得自己没家世是穷小鬼，为了不要让小九九不与自己接触才搬家的，非常不服气，打算如当初所说让自己小说大卖，包括当初写给小九九的那本来证明自己。清策庄的老人直到如今仍在后悔自责自己没能救下小九九。后来小九九因为没保护好书的执念变成幽灵游荡在无妄坡，但只找到几页残页[154]。数十年前的其他事件刻晴的祖父是璃月著名的土地业大亨，也是一位学者，酷爱搜集古籍，曾与夜兰的家族有过学术合作。夜兰的祖辈曾为七星效力[155]。三十年前，一位名叫 “若心”的戏子在“和裕茶馆”表演，成为一代名角。若心曾经常与现如今的 “范二爷”搭戏，而且总是在台上扮演一对，让范二爷对若心心生情愫，但若心倾心于一位冒险家[156][157][158]。
