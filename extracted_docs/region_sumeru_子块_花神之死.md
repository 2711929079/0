# 花神之死

花神与赤王的计划在月夜之下决定，他们要为了凡人的未来去赌那一丝可能性。大慈树王对他们的计划感到大为震撼，与他们分道扬镳，离开三人创立的绿洲乐园。花神临死前，她像第一次花神诞日时做的那样[19]，从自己的身体里分出一丝「灵光」，交由大慈树王抚养，并警告她提防未来漆黑浪潮的来临。花神预言：灵光将在未来改换面目，将面临分裂与死亡的考验，而后将长生不死，但那是更为黑暗的道路。水神与树王都将先于灵光而陨落。她们将被遗忘，而灵光也将只剩下牺牲的记忆。[22]灵光具有强大的力量，能与深渊的力量相抗衡，但其同样拥有自己的意识。[23]花神在开启通往深渊知识的密道时，就注定牺牲，树王从与花神偕眠的那夜便以知晓此理…虽然看上去似乎有内幕，但原文本就是这么写…[22]文本中还将灵光称作是花神的孩子花神用自己生命的终结为赤王指引了通往天空与深渊一切知识的密道。神柱也自高空降下，将草甸林木埋入漫流之砂。失去了魔神的乐土风暴骤起，黄沙漫天，很快被灾祸吞噬，绿洲不复有紫红的帕蒂沙兰盛开。[13]现今的帕蒂沙兰乃是大慈树王依赖丰沃之惠重新培育的品种。[24]花神死后，赤王决心要在地上建造理想国，让一切悲伤从此消失；而大慈树王则决定散布草木与绿茵，让土地充满智识与幸福。[25]自己的死，与赤王日后的行动等，都已被花神的「神的棋盘」所预演，在花神的计划之中。尽管可能性微乎其微，她也要为了人类赌那微小的突破天空桎梏的可能性。在染成血红的月亮下，棋盘曾预演了之后的乱世[26]。
花神逝后在花神死后，赤王十分悲痛，执着于旧日的回忆，在寒天之钉上为花神修建了陵墓：永恒绿洲。[13]在花神死后，乐土「瓦利·韦杰」被狂沙席卷曾经乐土的子民在百年无神的混乱与迷茫中据藩地自守。而在这百年间，沙漠出现了许多国家。[27]在这百年间，月女城变为废墟，（秘境「蜃景祭场」便是月女城的遗址[28]）。赤王在这百年间都在研究寒天之钉。[29]
