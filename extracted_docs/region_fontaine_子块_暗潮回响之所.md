# 暗潮回响之所

刺玫会的起源之地。如同有朝一日会成为雄鹰的雏鸟破壳而出，宏大而光明的野心曾经在狭小而阴暗的地方孕育。出身白淞镇的人与灰河的住民一样，从这些地方的历史中明白了，有些事情是值得、必须，或是只能用血去争取的。
