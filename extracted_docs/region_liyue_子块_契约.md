# 契约

岩之君曾施展神迹，仿造昆吾全盛时期的才华[39]从无杂质的金色石珀中削出长刀一柄，挥剑斫去山峰的一角，以此向子民立下无上庄严的契约——离散的人，必将聚拢回归；背约的人，必然加以惩治。失去挚爱者、痛失珍宝者、蒙受不公者，将得到补偿。[40]据说当年玥珠曾是岩王爷赐给璃月人的第一颗神珠[41]。在岩王爷的指引下，璃月先民治炼矿石，用最初的货物与世间诸国展开贸易[42]。
歌尘浪市数千年前，萍姥姥仍自称「歌尘浪市真君」，喜欢热闹，喜酒好歌。她认为音乐不可能由机关自发生成，和用机关「涤尘铃」编写音乐的归终是对手，常以作曲相互较量。留云借风真君请岩王帝君出面平息了二人的矛盾，「涤尘铃」也被帝君收走，用于操办典仪。此后，歌尘浪市真君和归终经常一同在山间聚会，探讨雅乐、机关与人间事[43]。夜叉中的“伐难”与“应达” 也经常和留云借风真君、歌尘浪市真君一同享受音乐[44][45]。
