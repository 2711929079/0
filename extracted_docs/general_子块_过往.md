# 过往

你和戴因一同进入城市调查，得知了黑蛇骑士和丘丘人的身世来源：他们都是坎瑞亚移民，被天理诅咒而变成魔物。而身为曾领导黑蛇骑士的宫庭卫队队长戴因是受不死诅咒的坎瑞亚人，和你的血亲曾是旅伴，但后来两人绝裂了。
安魂曲你的血亲与深渊教团打算透过装置让受诅咒的人民恢复原状，但戴因料到此计划必定失败，还会让受诅咒的人们产生极大痛苦，于是和你破坏掉了深渊教团留下的装置，并且击败了深渊使徒。最后，戴因独自离去，你则决定继续踏上寻找血亲之路。
