# 琴技能详细介绍

琴是《原神》中的风元素五星角色，蒙德西风骑士团代理团长，称号"蒲公英骑士"。
元素战技「风压剑」：
- 释放风元素能量，对周围敌人造成风元素伤害并击飞。
- 长按可以调整方向和范围。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「蒲公英领域」：
- 创造一个持续8秒的风元素领域，领域内的敌人会受到持续风元素伤害，同时恢复队友的生命值。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "顺风而行"：队伍中的角色移动速度提升10%。
- "蒲公英的恩惠"：蒲公英领域持续时间内，每2秒恢复队友20%的生命值。
- "迅捷之风"：琴的普通攻击和重击命中敌人后，有20%概率减少元素战技的冷却时间1秒。
