# 挪德卡莱·抗击灾厄

为便于展示和阅读，挪德卡莱众人在灾厄时期的事迹分别记录，不再按时间拆开。
狂猎「狂猎」最初只是家长吓唬孩子的传说、悲鸣中的女性诡影。坎瑞亚灾变发生、「猎月人」消失后，在其身躯残骸的作用下。挪德卡莱的「狂猎」现象急剧恶化，成了深渊灾难的代名词[61]。在冻土之下，掩埋着黄金城的伟大战士。即便存在过的痕迹早已被抹去，勇士们的英魂依旧守护着黄金城的故土。这些灵魂因深渊的污染而苏醒，所有敢于入侵之人都会被他们当作猎物[62]。与其他地方相同，挪德卡莱受到深渊入侵付出惨痛的代价[6]。
