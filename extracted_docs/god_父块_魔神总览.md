# 魔神总览

## 魔神
据慈水怒涛之翼描述「天理」法涅斯与第二王座大战后，其神体碎片（原初的碎片）与宿主（如歌仙之章中提及曾是元素精灵的风神温迪）结合升华为魔神（法涅斯在葬火之战中受到重创，机能损毁，散落大量力量碎片，无力再凭借自己的绝对威权压制这个世界本来的秩序。来自原初的碎片与提瓦特大陆上的元素生命体或者魔兽融合，形成了所谓魔神权柄神格一类的东西。至此，这些元素生物或者魔兽脱胎换骨，拥有了神明般的伟力，这就是魔神。原初的那一位对人有一套神圣的规划。人只要幸福，它便欢欣。因此，不论获得了原初碎片的魔神是何模样，性情如何，都偏爱人类，会不自觉的庇护提瓦特上的人类。只是，这群神明级别的魔兽或者元素生命许多都不懂何为仁爱，不懂人类，因此会对人类行恶行却不自知，如龙卷的魔神迭卡拉庇安。）神之眼持有者即是「原神」，「原神」拥有登上天空岛的资格，并且成神（如温妮莎），但此类「神」到底是何种性质尚且未知，暂不应与魔神混为一谈。三月女神与纳塔人信仰的「夜神」与「十字路的主人」（现已确认是「空月女神」哥伦比娅[1]）也是同理。理论上魔神是获取法涅斯身体碎片升格的事物，法涅斯也曾主动将身体碎片赋予使其成为魔神，如厄歌莉娅。葬火之年后不久，由于原初法涅斯战中受到重创，机能损毁，无力再凭借自己的绝对威权压制这个世界本来的秩序。为了镇压、调服世界的不甘与愤恨，原初法涅斯与另一位后来者一同制造了「神之心」。世上应当建立起这样的秩序，人因此也只能拥有这七类念想。所有来自原初的碎片被驱使着互相吞并，造就魔神战争。魔神战争结束，提瓦特大陆上的魔神所剩无几。除去少部分逃到暗之外海，大部分不是战死就是被镇压。最终幸存的七位魔神得到神之心，且与神座绑定，加冕成为尘世七执政。武器焚曜千阳故事中，希巴拉克自称为赫布里穆，因此历代火神的魔神名是传承的。并且从魔神任务中可以得知，历代火神均不具备原初碎片，因此无属于魔神的权能与寿命，其神力主要来源于自身天赋、神座、部分死之执政的权能以及神之心。魔神能接受四影分享的权能。派蒙、空之执政—阿斯莫代、生之执政—纳贝里士、死之执政—若娜瓦和时之执政—伊斯塔露，虽然名字对应所罗门魔神名，但目前游戏剧情内尚未定性为魔神。
普尔森purson的描述：他能隐藏或是显现那些被隐蔽的宝藏；他能看见任何时间所发生的一切，并能为追随者带来强大的仆从；他能为万物带来真正的答案，但却会引诱寻知者询问那些他们无法承担的问题。相传许多书中都有提到祂原本是座天使或力天使堕天之后变成的恶魔。”
带来真理、强大的仆从，引诱询问无法承担的问题，由天使堕落，也都能符合花神的故事。花神原为天使，而天使受诅咒变成的仙灵也会显现被隐蔽的宝藏。

## 巴纳巴斯
Barnabas可能是第五柱魔神Marbas的别名Barbas中央加上na。Barnabas这个名字其实是《新约圣经》中记载的一个早期犹太人基督徒（本名约瑟），是曾与保罗一同传教的使徒。之所以会使用此名，是因为原神的宇宙观基本上参考了希伯来的宇宙观关于原神宇宙观与希伯来神话宇宙观的对应：提瓦特Teyvat本身就是希伯来语的方舟。希伯来人认为世界是巨大的圆顶（天穹）笼罩着，用来隔绝世界与世界外的水。对应原神中的虚假之空，用来隔绝吞星之鲸这样的星间巨物与禁忌知识，而吞星之鲸也是以海洋生物作为设计原型。而且世界之外也被称为暗之外海。圆顶有天窗和天门这样的设计可供出入，在日月前事中写道法涅斯封堵了诱惑的通道（引入禁忌知识的通道），因此真的有类似的通道存在。在足迹PV中，旅行者旅程的最后戴因说,「现在，踏入天地之人，你的旅途已经告终，但你仍未越过最后的门扉。」应该是指旅行者旅途告终但仍未离开提瓦特。玩家从登入介面「来到」提瓦特的方式也是进入登入介面的门扉。开场动画中旅行者双子因为坎瑞亚灾变爆发决定离开时也是来到门扉所在的天空岛。因此这个门扉可能是前往世界外或来到世界的途径，对应天门。祈愿动画角色降临的过程也很符合从天窗落下来的描述。天穹还有着支撑点的设定，被称为天柱。虽然希伯来宇宙观中只有两个支撑柱，但大部分其他有相同设定的文化中都是多个支撑柱，并且世界中心都会有个核心柱。而支撑柱可能对应地脉树，核心柱对应世界中心的世界树。原神的世界树英译Irminsul,Irminsul是日耳曼神话中的生命之树，又名Irmin-saüle,saüle意指柱。十分了解提瓦特真相的坎瑞亚也将支撑王国的四大产业称为王国四柱，并且有骑士被封与「天柱骑士」这个称号。两个世界观的大地深处刚好也是深渊。，而「冰神攫取神之心」就是贯穿原神宇宙观的重要主线，可以说主线剧情都是围绕在执行官夺取神之心的故事线进行的，因此冰神选了个契合底层宇宙观的魔神名。Barnabas被称为使徒，而因为那几位米哈游创始人是EVA的死忠粉，米哈游的剧情常常在《新世纪福音战士》（EVA)上得到重要启发，而《新世纪福音战士》中也有使徒这个重要设定。在剧中，使徒对应的英文是「Angel（天使）」，使徒的命名也大多源于圣经中登场的天使。然而使徒这个词原意是担负使命的人、传递信息的人。在基督教，是特别的宗教头衔，天主教汉译为宗徒或门徒，最早用于耶稣所亲自拣选的十二名门徒。在基督教会（Ecclesia）之后，将这个名称用在圣经圣灵拣选，有特殊使命的人，类似于先知。也就是说《新世纪福音战士》中的使徒形象来自天使，但使徒这个词却来自Barnabas这样的使徒。而原神魔神名的原型所罗门72柱魔神本质上是72位恶魔，恶魔其实本质上是堕天使，天使即EVA中使徒的原型。Barbas是第五柱魔神，而《新世纪福音战士》第五使徒是雷米尔（雷天使）。正巧，官方发布第一期——Boss篇01｜无相元素的起源中提到，无相之雷的设计很大程度受到《新世纪福音战士》中的雷米尔的启发。早期无相元素的变形也基本是参照了雷米尔的变形。又巧合的是，无相雷的代号阿莱夫也取自希伯莱字母，无相之雷的代号取自字母“א”(Aleph)。在这之后的无相元素代号都取自希伯莱字母。又与希伯来对应上了。最重要的是，冰之女皇统领的至冬原型是俄罗斯（俄国），俄罗斯帝国曾经是世界上存在最多流散希伯来民族（包括犹太人）人口最多的国家。现今俄罗斯境内的犹太人口在欧洲国家中依旧排名第一莫斯科大亲王国内的犹太人最早可以追溯到1471年。叶卡捷琳娜二世统治期间，因瓜分波兰吸纳大量犹太人，犹太人被限制在俄罗斯境内的栅栏区。后来犹太人获准离开栅栏区前往栅栏区以东地区居住。亚历山大三世时期反犹太政策变本加厉。俄罗斯帝国曾经是世界上存在最多流散犹太人人口最多的国家，但从1880年代开始，帝国境内出现了一波又一波的反犹太大屠杀，哪怕到了苏联时期仍有反犹事件。1880年至1920年间，超过200万犹太人逃离俄罗斯，大部分逃往美国和现今的以色列（以色列使用的新谢克尔纸币人物均为祖籍沙俄的犹太人）。但现今俄罗斯境内的犹太人口在欧洲国家中依旧排名第一。。Barnabas直译为黑影虽然这么说，但没找到这个翻译的依据，但确实有资料将其直译为黑影，如美国电影《黑暗阴影》：本作男主角，18世纪的一个商人、绅士，被女巫安琪莉卡诅咒变成吸血鬼（顺道一提原神的《裴伦埃里》中也有女巫安吉莉卡存在。并拥有赫雷贝里在其眼前因诅咒化为丘丘人的场景。），埋在一座坟墓里长达两百多年与世隔绝。1972年，建筑工人在施工时挖出了他，之后被唤醒的他开始寻找自己的后代、仇敌，以及曾经爱过的女孩。<br>暴雪多人FPS游戏《守望先锋》中一位女黑客的代号是“巴纳巴斯”黑影。<br>这两个翻译的依据无法查清，在使徒行传4:36中使徒们給约瑟一个别名——Barnabas，意思是「劝慰子」。称某人做「……之子」表示他具有某种显著特征。（参看申命记3:18，《圣经新世界译本》脚注。）在公元1世纪，以别名去突出人的特质是一种相当普遍的做法。（参看马可福音3：17。）这种称呼也为大众所接纳。，可能是象征冰之女皇一直在黑影中暗中反抗天理。冰之女皇的魔神名是取自看似毫不相关的所罗门七十二柱神的Barbas中间加上na，可能就是因为上述原因。

## 奥赛尔
有部分人认为奥赛尔（Osial）的名字来自所罗门七十二柱魔神第57柱欧赛（Ose）。但奥赛尔和欧赛完全没有对应之处，因此祂的魔神名Osial可能是取自能对应上的第68柱贝利亚尔（Belial），又或是Ose与Belial的结合，但并没有其他魔神命名是采同这个方式，且这样的命名完全意义不明，这里默认其原型为Belial。
「Belial」是象征着「无价值」、「无赖」、「破灭」等一切归于「恶」的意思。《死海古卷》〈光之子与暗之子之战〉称其为天之副君主、暗之军团的大统率、叛天的首谋首恶。自称原名Satanail（Osial的S可能来自于此），是所有堕天使中最危险、凶恶的一名。他在堕天后最成功的一次反扑行动是唆使犹大王国再次悖逆神，转而膜拜其邪术，以致见弃于神、被巴比伦灭亡。《哥林多后书》第六章十四节以「黑暗之子」称之，因为其身份和耶稣基督对立，经注释为撒旦的别名。与《原神》中奥赛尔作为大反派，也是第一个与主角正面对上的象征着极恶的魔神对应。暗之军团的大统率、叛天的首谋首恶，与统帅深渊教团的殿下打算将其打造机械魔神并反抗天理对应。在天使们都在为上帝尽心尽力传福音的同时，却拒绝了上帝所执行的命令，反倒是喜好以自已的方式和习惯行事，比起其他天使显得更加独立，却也更加不合群。明明是众天使之首，拥有如此荣耀的身份，却如此的叛逆，不愿诚心为上帝奉献。对应不管在现今还是魔神战争时奥赛尔都实力强大，魔神爱人的特质却似乎特别薄弱，与其他魔神不同直接大肆侵略璃月港，彷佛只想破坏与向岩神及其子民复仇一般。纪录Belial事迹的《死海之卷》让人联想到「漩涡之魔神」与「海」有关的设计元素。上帝将其打入了地狱深处，让它在地狱的浓烟和烈焰之中，承受永不止尽的折磨，并在痛苦之中思索自己所犯下的罪行与被岩神镇压在海中的奥赛尔对应。《冒险家罗尔德的日志·孤云阁》也曾记录下「夜里睡得很不安稳，梦境总是黑暗潮湿的。我彷佛梦见自己是被岩神刺穿在海底的魔怪，正扑扑簌簌地挣扎、抓挠着坚固的岩枪，每一番动作都带着极大的痛苦与憎恨…」。奥赛尔的形象可能受中国古代神话传说中共工和相柳的启发。共工是水神，被列为「四罪」之一，对应Belial象征着一切的恶；相柳是共工的臣属，九首蛇身，最后被禹所杀。
