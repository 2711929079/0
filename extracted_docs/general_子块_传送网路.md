# 传送网路

地脉的根系构成错综复杂的网路，在地脉的根系间移动能抵达另个地方。「传送/口袋锚点」与深渊使徒/咏者的传送门就是利用此原理。部分没有传送锚点但地脉完整的地方能直接进入地脉的根系移动（传送），如层岩巨渊地下的地脉树能向上通往须弥。
