# 少女连环失踪案开端

20年前瓦谢和薇涅尔在探索遗迹的时候，薇涅尔触碰到了原始胎海之水导致溶解，瓦谢根据附近寻找到的雅各布的研究笔记想要重新见到恋人，于是到处溶解少年做人体实验。[78]
香水狂热起初只是一些商家推出了几款高端香水，为它们赋上了各种标签，却没想到它们在市场上引发了异乎寻常的疯狂，价格节节攀升。有人想用它们标榜自己的身价，更多人想趁着价格飞涨囤积居奇。在人们的盲目推崇下，一瓶普通香水的售价都被抬到了数十万摩拉之巨，由此引发的纠纷、骗局更是层出不穷。好在这股风潮彻底扩大之前，沫芒宫意识到了它的危害，逮捕了许多投机与诈骗团伙，最后这场泡沫的破裂，也只影响到了一部分富贵人家。而奥古斯都就是这场狂热中风头最盛的香水，曾经卖出过一瓶千万摩拉的高价。泡沫消失后，培育奥古斯都花的花圃毁于一场大火，从此再也没人见过这种花。二十年前席尔万和卢西恩曾经与香水师艾德加合作开发奥古斯都，因造假帐和扰乱市场等罪名被捕。「奥古斯都」是靠邪眼扭曲形成的花种。负责培育奥古斯都，表面身分为须弥学者的维佳雅真名是叶莲娜，真实身份是自至冬流亡的愚人众。她本来也被逐影庭盯上了，但她在被捕前自焚身亡，烧毁了奥古斯都花的苗圃和炒作团伙的利润。她的弟弟基里亚当初只有十几岁，负责给团伙打下手，颇得维佳雅宠爱，但案发后他就失踪了。奥古斯都花含有邪眼的畸变元素力，长年累月接触会摧残人体。表示叶莲娜想继续改良花种，但众人决定靠着香水狂热尽早上市奥古斯都。案发时，席尔万本来计划除掉叶莲娜和花圃，结果她提前自杀了。席尔万和艾德加认为，叶莲娜自杀是为了她弟弟：她因为过度使用邪眼，身体不堪重负，所以与弟弟一起流亡。在枫丹，非法用邪眼培育奥古斯都花使她的身体愈加衰弱，但她一直瞒着基里亚。案发时，叶莲娜自杀并毁掉了苗圃，给她弟弟留了一大笔钱。基里亚不知道叶莲娜是自杀，认为姐姐的死是席尔万一伙为了销毁证据做的，为了替姐姐复仇，杀害了出狱后不久，真正的调香师艾德加，顶替了他的身份，然后再宣布远赴须弥，在漫长的岁月中精进调香技术并用邪眼使自己的身体老化。他知道席尔万出狱后一定会来找艾德加，于是装成他的样子等待出狱后的「姐姐的仇人」找上门。期间收艾梅利埃为徒弟。[79]
