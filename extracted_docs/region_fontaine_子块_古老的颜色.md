# 古老的颜色

雅各布需要厄里那斯的血液，为了不引起逐影猎人注意，决定从厄里那斯住民美露莘那里和平获取。他自称艺术收藏家，向美露莘玛梅赫购买以厄里那斯凝血为颜料绘制的画，但没有成功。之后，雅各布雇佣镀金旅团佣兵强买玛梅赫的画，自己作为彩蛋在附近山头监控。玛梅赫与旅行者外出时，雅各布与佣兵们闯入海沫村玛梅赫家搜寻凝血颜料未果，被玛梅赫撞见。自此，雅各布放弃和平获取凝血的计划，他变身成深罪浸礼者，激活了遗留下的法阵，逆转厄里那斯心脏的晶化过程。期间，法阵短暂打开了兽境。最终，雅各布从厄里那斯重新跳动的心脏处获得了足够多的血。雅各布在告知旅行者「世界式」预定的时刻即将到来，以深渊之力将旅行者暂时封印后离开。[46]
水仙的追迹提瓦特危机将至，雅各布要锻造「水仙十字圣剑」，并交给大师。他前往四象限圆环，逆转了四象限的「密合之约印」，使得原始胎海封印解除，又分别取走了三个象限中的「记忆」、「愿望」、「灵魂」之块，而对于最重要的「人格」之块（内部保有原来雷内切除的人格），则被他设了陷阱。安、卡特皮拉、西摩尔与旅行者则打算一同阻止雅各布的计划。旅行者在取得「人格」之块后被传送至格式塔塔顶，从天窗进入了水仙十字结社。雅各布在那里等待旅行者，想要抢夺「人格」之块，但被旅行者，卡特皮拉和西摩尔合力击败。奄奄一息的雅各布交出了其他三块锻造材料，并带领旅行者一行人将圣剑锻造完毕。他自知大限将至，想让安吸收他的愿望与记忆，但被安以「不应该保留现在罪孽深重的你。」而拒绝。在圣剑锻造完毕后，雅各布宣告约定之时已到，原始胎海封印解除，大师即将归来，而他的命运即将走到终点，随后被下落的时针砸死。雅各布逆转四象限的「密合之约印」，原始胎海封印解除，大师重新甦醒。原始胎海之水将随着大师的苏生淹没世界，而新的宇宙将由此诞生，即大师与所有人融为一体，以此对抗末日。旅行者来到格式塔底部的原始胎海，大师想要取得「水仙十字圣剑」并完成他的救世计划，便与旅行者展开对决。旅行者利用圣剑的力量将大师击败。被大师溶解的意识纷纷脱离，他的位阶被打破，丧失了对原始胎海的掌控力，雷内的人格回归，大师也退化成了紫色的浊水幻灵。在失败之后，大师认为自己之前所做的一切毫无意义，他的世界式出现了错误，少算了旅行者这个「变量」。安称自己愿意原谅大师，大师则认为自己梦想崇高没有私心，没有人能评价他，但还是为他的计划导致的代价感到抱歉。最后安对大师提议回去见一见其他人，大师同意了。大师先回水仙十字结社见到了卡特皮拉，并称赞他比几百年前更像人。卡特皮拉则冷嘲热讽了一番大师「拯救世界」的理想，并表示了对他「玩弄生命」的鄙夷。
大师又随旅行者一行人回到了安眠处，起初「玛丽安」还表现出抗拒，试图沉迷于过去，设下一系列机关试图赶走他们。在大师的指导下，安「否定」了她设下的机关，使其失去了作用，而其「抗拒的意志」被旅行者用水仙十字圣剑击碎。一行人终于抵达了「真正的安眠处」。「玛丽安」见到大师后，直呼其为雷内，大师也回复了她。大师得知了玛丽安已经与莉利丝院长融合成了纯水精灵「玛丽安」，称赞她们超越了人类与纯水精灵，是超越者的形态。而「玛丽安」虽然无法原谅纳齐森科鲁兹，但她认为雷内是另一码事。大师提议可以利用「水仙十字圣剑」将玛丽安与莉利丝分离，但被「玛丽安」以想做为自己活下去而拒绝，而这一意志使得圣剑的使命完成，在故事中的意义消弭，失去了力量。「玛丽安」还邀请大师一同旅行，但被拒绝。大师的力量消耗过多，已经无法再维持存在，他与安、卡特皮拉对话，认为自己在百年中先是用尽力量避免意志消散，再是用强烈的意志抵抗袭来的命运，才坚持了下来。而对他们来说也是一样。最终，在与二人道别后，大师在「真正的安眠处」消散。他一次也没有把名字变回「雷内」「人生，如果只是一个梦呢？」…[85]
