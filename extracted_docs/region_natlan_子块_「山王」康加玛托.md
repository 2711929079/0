# 「山王」康加玛托

匿叶龙，500年前坎瑞亚灾厄时期英雄布基纳（悬木人）的龙伙伴。布基纳曾与名叫“康加玛托”的匿叶龙赌气，因为后者嗤笑他动作痴愚。为了通过摘取山巔上母树结出的火榴果来证明白己，布基纳前往了花羽会观察龙的肢体动作，在沃陆之邦锤炼体魄，还去烟谜主敏说了自己的感知与意识。特诺切在此期间见识到了布基纳的决心，于是帮助他共同打造了一把武器硕果钩。行到母树结出的火榴果成熟之日，布基纳凭借自身积累的经验与趁手的式器迅速接近了火榴果，但突如其來的强风让布基纳陷入危险，好在一直在暗中观察布基纳的康加玛托及时出现救下了布基纳。最后，布基纳把火榴果献给了康加玛托，并得到了康加玛托的认可，二者成为了伙件。“第四位自愿追随英雄特诺切的，是年轻的布基纳与他那莽撞的伙伴康加玛托。”与布基纳一起对抗过大量深渊魔物，虽胜利，但自身也受到了严重的深渊侵蚀，在意识狂乱的状态下杀死了布基纳。此后被「悬木人」长期稳定在沉睡状态。部族纪闻「尤潘基的回火」：回火仪式中主动撞入圣火，封印了自己的意识，只剩下依靠本能行动的躯体用于试炼后来的挑战者。魔神任务「炽烈的还魂诗」：意识复苏，与基尼奇、瓦伊纳一起给旅行者送行。
