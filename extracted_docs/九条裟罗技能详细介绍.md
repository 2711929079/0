# 九条裟罗技能详细介绍

九条裟罗是《原神》中的雷元素四星角色，稻妻天领奉行的大将，称号"霆雷快雨"。
元素战技「鸦羽天狗霆雷召来」：
- 释放雷元素能量，对周围敌人造成雷元素伤害并施加"雷元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「煌煌千道镇」：
- 释放强大的雷元素能量，对周围敌人造成雷元素伤害，并创造一个持续8秒的雷元素领域。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "霆雷的祝福"：鸦羽天狗霆雷召来命中敌人后，有15%概率触发"感电"效果。
- "雷元素增幅"：九条裟罗的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "千道的威严"：元素爆发持续时间内，九条裟罗的雷元素伤害提升15%。
