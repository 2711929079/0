# 调查烬寂海

渊上伪装成稻妻人来纳塔调查烬寂海。他以帮助找回龙宝宝为条件，向人打听「回火」的消息，见到旅行者和派蒙路过而急忙离开。之后渊上和基尼奇达成了交易——由渊上提供能够吸引兽境猎犬的装置，帮助基尼奇打开兽境，而基尼奇允许渊上进行「历史研究」。渊上与阿乔二者私下又达成交易：由渊上夺取基尼奇的身体给阿乔，以此对阿乔进行研究。在「回火之夜」前一天，基尼奇通过装置成功召唤兽境后，渊上从兽境中出现。他本想夺取基尼奇的身体，但因为旅行者也在而作罢撤退[84]。之后渊上用计从茜特菈莉处打听到烬寂海的位置[85]。
渡火者的解脱渊上调查烬寂海是为了寻找自己认定的家乡，发现烬寂海能前往时间线的每个角落后前往步过烬寂海[22]。传说一种鸟，生而孤独，能在火中高歌，人民将它作为图腾崇拜，君主以之为贵。火山之地的智者佩着它的羽毛，在烈火中退隐。孤独而来的他最终也孤独而去，下落从此不明。从此以后，人们时常听到烬寂海深处传来鸣唱。不知是浴火之鸟的鸣叫，还是渡火之贤者的吟叹呢。[86]
