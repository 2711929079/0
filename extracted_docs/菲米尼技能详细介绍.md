# 菲米尼技能详细介绍

菲米尼是《原神》中的冰元素四星角色，枫丹的潜水员，称号"深海探险家"。
元素战技「冰棱穿刺」：
- 释放冰元素能量，对周围敌人造成冰元素伤害并施加"冰元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「冰棱风暴」：
- 创造一个持续8秒的冰元素领域，领域内的敌人会受到持续冰元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "冰棱的祝福"：冰棱穿刺命中敌人后，有15%概率触发"冻结"效果。
- "冰元素增幅"：菲米尼的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "风暴的守护"：元素爆发持续时间内，菲米尼的冰元素伤害提升15%。
