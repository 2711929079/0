# 目盲之年

贤人阿布拉克他被开启了神智，他展示了从手中发出光的奇迹。先祖们以他为首领，开始建设「赫利俄斯」。[3]
目明之年「赫利俄斯」——太阳的神车，终于落成。白夜到来，常夜消散。日月的纪年开始了。[3]彼时，渊下宫才因为太阳的出现而得到些许喘息。因那龙嗣亲近黑暗、忌惮光明，不再肆意妄为。龙嗣横行、敛人命如草的时日终于结束，渊下宫民变得能与其相抗了。然而，异种外患没有完全根除，这人性的腌臜就已经暴露了。[5]“他们”利用民众，开始推行大日御舆崇拜。将人心寄托于虚妄死物已是愚昧至极。到了最后…甚至推举「太阳之子」作为傀儡以统治白夜之国。[6]「太阳之子，诞辰御祝，至天马之架席，共烈日之飨…」
