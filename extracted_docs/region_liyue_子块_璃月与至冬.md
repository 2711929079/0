# 璃月与至冬

执行官「公鸡」掌控了关于根绝层岩巨渊黑泥污染的情报，于是派兵驻守层岩巨渊，协助调查并根绝层岩巨渊的污染。当时愚人众甚至被璃月人民视为英雄。在至冬和璃月关系尚好，即愚人众派兵驻守层岩巨渊之下时，「女士」曾经和璃月七星谈成了一些互利的贸易协定。[208]
夜兰与「富人」夜兰渗透了愚人众执行官“富人”在暗中布下的隐蔽商路，成功拦截了运载的货物，也因此损失了一只名为“幽奇腕阑”的祖传玉镯。货物中有一件珍稀的魔兽皮毛，本来是计划送给至冬女皇的礼物，夜兰将其作为玉镯的赔偿收走了，并制作为一件大衣[209]。
