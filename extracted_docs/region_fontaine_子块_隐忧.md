# 隐忧

自从雷蒙多和舒瓦瑟尔接管重建工作后，科学院的研究精神开始变质：先是所谓正义的国度，开始通过过度的监控机制在抹杀科学的探索精神，然后就是学术机构逐渐官僚化，更多的流程更多的人情世故，在腐化这个曾经塑造枫丹工业强国的基础。跑个申请流程就需要填写各种复杂的申请表要填写半个月，整个流程走完可能1个月都过去了。申报的经费如果花不掉，就会缩减明年的经费，所以就要最后突击性的花掉。这一切都在述说着着目前枫丹科学院的学术官僚严重情况。更疯狂的还有枫丹学术界的对赌协议。在枫丹独立研究者向科学院申请经费进行研究是需要承担一定风险的，如果实验顺利得到了成果，那么一切皆大欢喜，研究者功成名就。但是如果钱花光了却什么成果都没有，那么研究者就会被立案调查甚至被审判。
