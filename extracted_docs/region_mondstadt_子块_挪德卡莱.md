# 挪德卡莱

骑士团的远征取得了可观进展，并由此做出了一个重大决定：根据途中收集到的新线索，大团长将前往挪德卡莱，进行更深一步探寻与查证，如今西风骑士团在安瓦蒂尼尔湖附近驻扎，原因之一便是调查此处有关过去的「北风骑士」以及狼主「安德留斯」的痕迹[162]；而赤杨骑士（芙蕾德莉卡·古恩希尔德，琴的母亲）等人则领命回到荆夫港驻防[163]。托马的母亲身为骑士也跟着被调往挪德卡莱[164]。
