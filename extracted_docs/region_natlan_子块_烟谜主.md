# 烟谜主

奥奇卡纳塔遭焚后，灰烬如谜烟笼罩山麓。为了应对此危机，「烟谜主」的祭司曾在洞窟研究如何让涂鸦在雾霭中亦能被分辨的秘法，据说那就是为何烟谜主涂鸦具有别样力量的开端[92]。最初将燃素刻录与涂鸦油彩相结合的人是来自「烟谜主」部族一位离经叛道籍籍无名的祭司。但也正是这位祭司感受到了荧光角菌中的力量，收集其中的荧光素加入了涂料之中，才有了后来的「灵谜涂鸦」的诞生[93]。
比夫茏魔神战争时，魔神「蕈之贤主」比夫茏通过为人类提供蘑菇的方式，成为了人类的信仰，它以力量创造分身，赐予人类无穷食粮。但如果蘑菇吃得太多，会逐渐能听到比夫茏的声音，它也借此对人类施加精神控制。但人类在发展的过程中，比夫茏的分身被越吃越少，它的力量越来越弱，甚至一度无法维持实体。而分身之一的赞玛兰也切断了和比夫茏的精神联系。比夫茏想要恢复力量，打赢魔神战争，向人类复仇。但须弥人都已知道它创造出的蘑菇分身十分美味，分身已不可能再在须弥生长起来了。于是比夫茏控制一信仰者，来到了纳塔的火山之下，却遭遇火山爆发，被深埋地底[94]。
