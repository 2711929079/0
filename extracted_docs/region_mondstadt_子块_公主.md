# 公主

山脊守望矗立在山脊的起点，如同界碑一般守望着南与北、山岳与水泽的远古城墟。传说中它的门扉只对心中燃烧着永不熄灭的烈火的人敞开，为他们祝圣戴冠[9]。相传古代的人会深入幽谷，寻找祭坛上的古树，将秘密说给树洞听，以卸下守密的沉重负担。忘却之峡中，充满了被遗忘的秘密[10]。当芬德尼尔的祭司之女诞生在这棵白树之下，接受祝福时，苍翠山岳的国境中充满了欢欣。沙尔·芬德尼尔的幸福一定会永远存续，正如贯通大地、永不凋败的银白之树。——为山中国度撰史的乌库，当时是这么想的。曾见证过无数人与事的记事者由衷相信，公主的美貌与才德将如月光般永远皎洁…[11]
忍冬之树寒天之钉在冻结雪山的「长钉」降下时，风雪遍天。天降之柱崩裂为三。[2]
