# 2000年前

魔神战争结束魔神战争结束世上曾有诸多魔神，也曾有旷日持久的战争，直到两千年前方才终结。期间生灵涂炭，流血无数。最终，提瓦特只剩下七位胜利者[99]，七位魔神各自登上「神」之座，自此终结了「魔神战争」时代。尘世划归七神所有，他们被称作「尘世七执政」。虽然性格迥异又相隔万里，却都肩负着「引导人类」的神圣职责。他们在败者的残骸上建立国度与城邦，开始了七神的时代。[84]七神们被天理分发了神之心。被七神击败，不服七神管制的战败魔神逃往暗之外海。岩枪的核心是发着暗淡光辉的核心，那是战败魔神的力量深入岩石形成的。他们的怨恨并不针对摩拉克斯，因为他们早就在暗之外海被深渊侵蚀的已经没有自我了，不可能还残留私怨[100]。原本居住在层岩巨渊地区的山民跟随岩王帝君迁入了璃月港生活，专以制造和販卖瓷器为业，不再顾兵戈之事[15][101]。摩拉世上第一枚摩拉，是在数千年前，岩王爷亲手用神之心[27]将自己的血肉转换铸造的。后来岩王爷简单花掉了[42]。而后黄金屋建立，作全大陆唯一的铸币厂，借助岩之神的力量制造摩拉[27][102]。复甦此前魔神混战干年，桑田尽毁。璃月之民遂以商贾、巧工为业。其佼佼者联合一体，名之「七星」。故璃月港雏形既成。七星下辖又有八门，内外诸业无所不包。又有千岩之军听令七星，内抚民众，外镇妖魔。是所谓岩王帝君之治。[30]追随岩王帝君者自成部队，以「岩君」、「璃月」为号，称「千岩军」，听令于七星，护卫璃月。后璃月之经济逐年屡升，成七国商贾之心，得誉「黄金的心脏」。七星璃月正式设立“七星八门”，甘雨主动成为辅佐七星工作的秘书，并获得神之眼。千岩团改名千岩军，由七星管理。七星分别为：“天权星”、“天枢星”、“天璇星”、“天现星”“玉衡星”、“开阳星”、“瑶光星”。八门分别为：“和记厅”、“银原厅”、“辉山厅”、“盛露厅”、“流金厅”。璃月开始每年举办名为“七星请仙典仪”的迎神仪式。七星作为璃月港凡民与岩神之间的媒介，负责与岩神直接沟通，并以简明易懂的方式向市民传达神谕[103]。七星的特务开始经营“望舒客栈”。夜叉璃月在古时瘴疠丛生。有人说，当初魔神征战不休，失败者被镇在坚固的岩石之下，腐朽后化作土壤，回归了天地间诸元素的经络循环。魔神的的魂魄有的满心憎恨，不愿屈服，便凝结成为妖邪。每当妖邪躁动不安，起而发难，瘟疫、鬼怪与异变就会发生。妖邪毁坏土地，使土地荒芜，在江河与大海之中作乱，人民为之深感痛苦。所谓妖邪，其实是被击败的魔神遗留的怨念。岩王帝君召唤夜叉们消灭妖邪。夜叉是璃月的仙兽，生性凶猛恐怖，凶悍善战为了守护岩王帝君的统治而不惜杀生。夜叉中最强大的有五个：浮舍、应达、伐难、弥怒、金鹏。这五位夜叉追随岩王帝君屡次与妖邪作战，消除瘟疫与瘴气，世人称他们为「仙众夜叉」。「仙众夜叉」守护岩王帝君，平定世间各种灾祸，就这样经历了许多年。然而，尽管夜叉们拥有强大的威能，却也难免被业障所困，被魔神的遗恨污染。他们中有的陷入难以言喻的狂怒、难以形容的恐惧而发狂，有的自相残杀，最终战死，有的最终走火入魔。弥怒使用武器凭虚，最初用兵器弥补失去的力量的仙人所使用的剑，斩杀了另一位夜叉。斩杀过无数的妖魔，最终也终结了过去的同族[104]。历经千年的劫数之后，五位夜叉有三位死于非命，一位不知所踪，此外还有许多不知名的夜叉死去或遁走了，只剩下金鹏至今仍然存活。所谓金鹏，号为「金翅鹏王」，又被称作「降魔大圣」。人们不知他从何而来，也不知他最终去了哪里。每逢春夜海灯节时，望着孤云阁上空的浮光，璃月的人民会说：「这是护法夜叉在讨伐魔兽啊。」有人听见荻花洲深处传来笛声，进去找寻却不见吹笛之人，便会说:「这是夜叉在呼唤旧友归乡呢。」因此，拥有大神通者必然会经历极大的苦痛摧残。夜叉失去了所有亲友与同袍，积累了太多的罪孽业障，心中永远充满憎恨，不得止息。以旧日遗留的愤恨与怨念为仇敌，既得不到酬报，也没有解脱，如此的苦难实在是空游饿鬼的痛苦，纵使经历万劫，也不会得以结束。[105]夜叉“弥怒”为身边的朋友设计服装。这些衣物设计精美，但穿起来过于不便，所以夜又“浮舍”不卖他面子，而摩拉克斯非常喜欢，收藏了许多[106][107]。魈与望舒客栈魈在荻花洲对抗魔神残渣的力量，望舒客栈也会支援魈进行降魔。有一次，魔神残渣所带来的无穷怨恨冲击魈的心智，危急关头，是温迪的笛声压制了那念，为他争取到片刻安宁[107][108]。七神酒会两千年前巴巴托斯初临璃月，「岩王帝君」的第一反应是：这位同僚在履行职责时遇到了困难，需要自己的帮助。所以还没等巴巴托斯从风中落下，岩之神已经做好准备迎接这位邻国神明，只等他开口，自己就将倾尽所能。然而，风之神却将一瓶酒递到了他面前。「这是蒙德城的酒，你要尝尝吗？」——为了送一瓶酒而弃职责于不顾，实在荒唐。即使如此，那位风神依然不断前来造访，到璃月港四处游玩，还总向岩之神问出各种奇怪问题。这位风神的好奇心，就和他手中的酒一样无穷无尽。自那时起，那个时代的七神往往会在璃月相聚。[84]
