# 凯亚技能详细介绍

凯亚是《原神》中的冰元素四星角色，蒙德西风骑士团骑兵队长，称号"寒风剑士"。
元素战技「霜袭」：
- 释放冰棱，对路径上的敌人造成冰元素伤害，并施加"冰元素附着"。
- 冷却时间：5秒。
- 能量消耗：无。
元素爆发「凛冽轮舞」：
- 挥舞剑刃，释放冰元素能量，对周围敌人造成冰元素伤害，并产生一个持续旋转的冰元素领域。
- 冷却时间：20秒。
- 能量消耗：60点。
固有天赋：
- "霜咬"：霜袭有10%概率使敌人冻结2秒。
- "秘密剑术"：凯亚使用元素战技后，普通攻击和重击的伤害提升15%，持续8秒。
- "冰潮的祝福"：元素爆发持续时间内，队伍中的角色移动速度提升15%。
