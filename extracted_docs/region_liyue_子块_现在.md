# 现在

主线剧情最后的契约旅行者与派蒙造访璃月港，恰逢璃月七星主持请仙仪式，却在仪式上惊现岩王帝君之尸骸，来自异乡的旅行者被怀疑为刺杀帝君的凶手，得到愚人众执行官「公子」的帮助。旅行者在「公子」的介绍下与往生堂客卿钟离一同为岩王帝君的送仙仪式作准备，同时亦因帝君之死而在七星和仙人之间周旋，调和仙人与璃月人的矛盾。「公子」借机暴露真实目的，欲从仙祖法蜕中取得神之心，但在发现仙祖法蜕只是个空壳后，他认为帝君只是为了某目的假死，为了逼出岩神便以百无禁忌箓放出被摩拉克斯镇压的「漩涡之魔神」奥赛尔，旅行者、七星、千岩军和仙人们同仇敌忾，团结起来迎击奥赛尔和伺机捣乱的愚人众，最终以牺牲群玉阁为代价将奥赛尔重新封印。事件后旅行者找到钟离，却目睹钟离以岩之神的神之心与「女士」进行交易——原来钟离就是真正的摩拉克斯，在察觉到自己终将受到磨损并失去神位后，制作了自己死亡的假象来考验璃月的各方势力，最终结果超乎想象，让钟离可以放心把璃月交给人们，而神之心则是摩拉克斯用来邀请愚人众参与这次事件的筹码[27]。而钟离担任往生堂客卿的原因是因为此后往生堂堂主将负责主持攸关璃月港存亡的八门七门大阵，提前前去监督已在必要时提供帮助[213]。当晚，钟离对七星托梦坦白了实情。次日，送仙仪式正式举行，也代表璃月正式迈入人治的时代。凝光在外交上紧咬着愚人众不放，使至冬趋于弱势，并逐渐排查北国银行势力[214]。此后，璃月与至冬断交，层岩巨渊内提供给愚人众的补给站废弃。[208]七星选拔之后愚人众干涉「七星选拔」，打算扶植魁儡政权，被阻止。「富人」干涉選拔，打算在知易上任后以其名义发起青墟浦探索计划，对外声称是该地拥有外发掘的宝藏[82]。然而「富人」的最终计划是找到替代摩拉的货币，钟离提及透过神之心能把自己血肉转换为摩拉，而青墟浦为旧时某名号佚失的魔神领地，存在许多秘密[82]。因此「富人」的目的可能是透过取得青墟浦底下的魔神遗骸并以神之心转换为摩拉。然而这样铸币法也还是要依靠来自神明的神之心，其仍在尝试其他铸币法，为了原料大量购买石珀[215]。人治的时代岩王帝君退位后，部分仙人担忧将璃月交由人们治理是否为明智之举。与此同时，璃月的海域出现异常，察觉异样的「天权星」立刻下令修建群玉阁，将蛰伏在海中的奥赛尔的妻子与其最后的追随者「跋掣」吸引出来。此后人们靠自己的力量击退拔掣，仙人们才认可帝君将璃月港托付给人们的决策。[148]若陀龙王若陀龙王近来由于封印出现了一丝裂隙，若陀龙王的恶意便化身阿鸠徘徊于伏龙树旁，寻找着解除封印的时机。但另一方面，若陀龙王并未完全忘记曾与帝君签订的「守护人类」的契约。龙王最后的契约，使龙王的善念附身于璃月名匠昆吾后人昆钧身上，凭着自己的本能寻找着摩拉克斯，希望他能阻止封印的解除。封印的解除被阻止后，「昆钧」将被遗忘的与人类同行的记忆展示给阿鸠，阿鸠在遗憾与悔恨中力竭，重新被封印。若陀龙王的善念也在和钟离短暂叙旧之后力竭，就此消散。「摩拉克斯，若有缘，他日必将再会。」[114]层岩巨渊层岩巨渊旅行者一行人解开七星对层岩巨渊设下的封印。在旅行者的帮助下晶石已经回归了地下更深处的渊薮，空留冷峻的「巨柱岩宫」。之后层岩巨渊的开采恢复正常。[153]沉玉谷投珑灵渊计划将沉玉谷改造回最原始也最适合三仙居住的环境。随着灵渊改造计画的实施，沉玉谷出现了大范围的氲气，近年来翘英庄的茶叶质量也开始下降。浮锦在锦落庭察觉了地脉异常，但力量不完全的她无能为力。
之后因为旅行者的帮忙，浮锦成功实施投珑仪式，灵渊也决定让步，不再改造沉玉谷的环境。[24]兹白因真实之月重回提瓦特，虚假之月即将崩解，波及到兹白的尸神之一，唯有在虚假之月彻底崩解前聚合三灵魂，兹白才不会彻底复活无望。最后兹白成功复活[5]。
