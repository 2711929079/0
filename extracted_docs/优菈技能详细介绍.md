# 优菈技能详细介绍

优菈是《原神》中的冰元素五星角色，西风骑士团游击小队队长，称号"浪沫的旋舞"。

普通攻击「西风剑术·宗室」：
- 效果：进行五段连续挥砍，造成物理伤害。
- 特点：总倍率高，是主要输出手段之一。

元素战技「冰潮的涡旋」：
- 点按：旋转劈砍，造成冰元素伤害，并获得一层"冷酷之心"。
- 长按：消耗所有"冷酷之心"，造成更高倍率的冰元素伤害和追加攻击。
- 冷酷之心：最多叠加两层，增强技能效果。
- 冷却时间：点按6秒，长按10秒。
- 能量消耗：无。

元素爆发「凝浪之光剑」：
- 效果：释放光剑，获得"光降之剑"充能机制。
- 充能机制：通过普攻、战技和大招为光剑充能，层数越高伤害越高。
- 爆发伤害：光剑充能结束后，造成巨额物理伤害。
- 持续时间：7秒。
- 冷却时间：20秒。
- 能量需求：80点。

天赋技能：
1. 不屈的冰潮：元素战技减少敌人冰元素抗性。
2. 光降之剑：元素爆发的物理伤害提升。

角色定位：
- 主要定位：主C输出
- 次要定位：副C
- 核心优势：高额物理爆发伤害、减抗效果

培养建议：
- 武器推荐：松籁响起之时、天空之傲、螭骨剑
- 圣遗物推荐：苍白之火四件套（物理伤害加成）
