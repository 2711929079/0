# 神秘岛

流泉之众初代首领乌努库征服了许多危险地带，但都不能满足，于是踏上前往艰险的通往「神秘岛」的道路。他的伙伴们接受他的任性一同前往，克服重重困难，抵达神秘岛前。然而他的伙伴正面临危机可能丧命，看着即将抵达的神秘岛，初代首领决定放弃前往一路返回。这时他的内心得到了真正的满足，原来自己一直想得到的「神秘岛」就是他的家乡与伙伴，对他而言流泉之众就是他的「神秘岛」。[48]
悬木人原本住在祖遗庙宇的悬木人的先祖在祖遗庙宇毁灭后用绳索与钩爪将块块巨木悬挂，部族因此也有了「悬木人」这样的名字。[49]传说中，「悬木人」的勇士与一百只匿叶龙一同，用了一百根绳子，花了整整一百天，才将象征信仰的「曜石图腾柱」搬到了现在它所在的地方。[50]
