# 魔神战争时期～2000年前

魔神战争混战南出天衡，东入瑶光，西登绝云，北访轻策，诸地萧条，生灵涂炭。璃月广袤，竟难容一处安宁。[42]岩之魔神.摩拉克斯传说在神魔混战的年代里，岩神曾显露出无边杀伐之相。在诸神之间的厮杀死战中，温柔从来不属于岩峦的神主。臧否分明，从无失准;在颠覆天地的混战中，即使故友反目也不会留情。那个传说时代的岩王帝君，碣岩般冰冷的面目之上未曾浮现过任何波澜。据说待尘埃落定，才卸下不动如玄石的神情。[62]兹蹻昔年帝君将兹蹻从「詹诸吞月」帛画中解脱出来，其便助帝君征战，先后平沈玉谷，定轻策山，此为契约之故。而后两人互不相欠[5][63]兹白陨落后，唯有下尸神的身影仍偶尔掠过故士的山水，直至在魔神战争中助力钟离后，方逐渐隐于山林之中。她身上的时之痕迹（磨损）依旧对那位古老的岩神产生了莫名的影响。于是在魔神战争后，自知身有威胁的兹蹻变隐入山林，从此不现人间。只是她生性逍遥自在，又喜好人前显圣，因凡人寿限不过百年，几乎不受时之痕迹的磨损。所以她偶尔会在一些需要帮助的凡人面前现身。一边助他们脱困，一边又存着将那天塌地陷前的故事悄悄传递下去的小心思。但若是后人成书不如她所意，她大概也只会看一眼就抛去一边。毕竟她就像一匹在无人旷野中肆意奔行的野马，没什么能束缚她自由洒脱的脚步。她从兹白的陨灭中诞生，但她从来不觉得自己应戴上兹白或是任何人的缰辔。她随时都准备好潇洒快意地跳入时间的尽头，然后「砰」地一声，化为一团闪耀着金芒的尘埃[64]。在山中自在逍遥，偶有小人儿误入，还能找找他们的乐子，早已不再求些什么。但若真如此，为何她又孜孜不倦地向凡人们讲述那葬火之战前的古老故事[5]？白驹逆旅老板的祖宗为了生病的娘采药想换取微薄的药费，却摔落山谷，被兹白的下尸神兹蹻所救。兹蹻送与她一则故事，让她带给书店赚点钱。讲述故事时，凡间已过兩千年，来到现在，她来到五百年后的归离集并遇到宝儿，所幸身为时间的女儿的兹蹻将其送回去，她才得以回归离集[11]。为了感谢白马先人的帮助，白驹逆旅就此得名[5]。沉玉谷部民口中的「时间的女儿」是兹蹻。八奇炼桃都中的「金目乘黄月驹」也是兹蹻[64]。留云借风真君.魔神战争在魔神战争中追随帝君，守护璃月。当时大多以人类姿态现身于世，果敢伶俐，受人敬仰[65]。留云借风真君以人形参与魔神战争，并多次庇护受难的人群。战后隐居于奥藏山，无拘无束，多以鹤形态现世[47][66]。魔神战争當时没人在求仙問道，於是太山谷被仙人藏入水下封印起来。铜雀摩拉克斯曾为夜叉中的一位小仙赐名“铜雀”，他喜欢吃“烤螭虎鱼”。但铜雀敢终因实力不济，战死于魔神战争[67]。盐之魔神·赫乌利亚在众魔神无情的混战中，人类是过于渺小的存在。而盐之魔神却并未参与铁石心肠的竞争，而是收拢了那些战火中无助流亡的人们，带领他们在这里建起城镇，在天地翻覆的末世中带给人们慈爱与安慰，试图寻求与众魔神重归和平的可能。她聚集起了一群追随者，在如今被称为「地中之盐」的聚落中苟安。这座城市矗立了数百年，直到魔神倒下的那一天，它才随之分崩离析。温柔的魔神并非死于与神的对决，而是死于她所深爱的凡人的背叛。他是这里的第一位凡人之王，也是末代之王。尽管与族人一样，他曾深爱着盐之魔神。但她的温柔太过柔弱，以凡人的心胸，他终究无法揣测舍弃自我的神之爱。为了寻求守护与战斗的力量，为了证明温柔的不合时宜，他以手中的长剑弑杀了孤独的魔神。盐之魔神形骸溃散，盐的圣殿随着盐之魔神的倒下而崩塌，没能逃走的人也因无法承受魔神死亡时流失的力量而化为盐雕，凡人的城迎来了盐块般苦涩的结局。[59]未经此灾的盐神子民离开了她的领地，前往璃月，请求岩王帝君的庇护。他们的后代害怕赫乌利亚的魔神残渣，害怕她施予他们永世的诅咒。于是，他们冒着危险将剑折断、供奉，希望能平息她留下的怒火。但这位直到最后都没有反抗过的魔神，又怎么会对自己的子民心怀忿怒呢。[42]据说盐之魔神的遗体依旧留在这片遗迹深处，虽然早已化为盐晶，却依旧保持着被长剑刺穿那一瞬间的姿态。[59]古闻之章第一幕中钟离提及，赫乌利亚之死成为对他的告诫。赫乌利亚的死成为他在魔神战争时的告诫，告诫他战争时不可过于软弱，必要时也得在之后的魔神战争战场上下杀手，表示赫乌利亚死于魔神战争较早期。后人的传说中，这个历史也变了样，因为赫乌利亚死于魔神战争较早期，传说中古城早在魔神战争之前就已经荒废了。传说地中之盐的两棵树下埋着古城的末代之王与他的敌人，同时也是他的挚爱⋯他们在那里同归于尽，却永远无法相互触及。直到一切化为士灰，废墟上生出的树木也是这样相离相望⋯[68]银原厅银原厅中有原盐之魔神赫乌莉亚的遗民，一直认为盐神是帝君所杀[42]。绣球在璃月，新人的婚礼仪式上，新娘会将一束绣球抛向前来参加婚礼的宾客。恰好接住绣球者据说会获得一整年的好运：商人将财运亨通、穷人会得福转运，未婚男女将遇到好姻缘，而已婚夫妻则会对彼此真心相待，不再为生活争吵。关于这一习俗的起源，有人认为是受到邻国「风之国」蒙德羽球节习俗的影响，也有人认为起源于魔神战争之前，「盐之魔神」漫步璃月大地的时代。根据一则如今已面目全非的传说，她曾向自己的子民分发花球，以此降下祝福⋯或至少残酷乱世中的一点安慰。当盐之魔神重归元素循环之后，她流散的人民或许也将这个传统留给了璃月人，而擅长竞争、喜好热闹的璃月人也依自己的个性改变了这种民俗的形式[69]。渌华池千百年前渌华池本是一处园圃。传说在魔神战争的时代，一对不受家族认可的恋人曾在这里密会。然而动乱无情，男子追随岩神而去，以凡人之身投入了魔神战争之中…就像那个时代的无数凡人，从此百年杳无音讯。女子徘徊在园圃之中，等待着恋人归来。后来，鲜花被荒草取代，荒草在潮水中朽烂；当潮水终于退去，她也回归土壤时，泪水汇成了这座池塘。或许正是因为浸淫了如此深刻的相思，这里的池水才如此清澈温润吧。[59]梦之魔神被称为「悬黎」的玉石，即便在至深至暗的夜里也能放射光芒。当被雕刻成劲弓之时，据说它射出的利箭能自行寻歼一切邪恶。
群岩之主的目光始终冷彻坚定，恰如永恒闪烁的金珀之光，冰冷而沉郁。在那个动荡不已的年代里，弓弦的震颤、鸣响与尖啸代替了安逸的歌声。洞悉人心弱点的梦之魔神，曾被这张强弓所射杀。她拥有明澈的双眼与锋利的头脑，善于操控与占有臣民的心。“看吧，我的孩子。看这无畏的英雄被恐惧啃啮，无私的王者卑躬屈膝’‘看呀，我的孩子。看那立下海誓山盟的情人，如何被背叛与谎言折磨（渌华池传说里的那对）”在怨恨的缠绕间，斜倚御座的魔神品尝着欢愉的果实，浆液自指缝间溢出。无形铁链铮铮作响，深入血肉的剧痛与狂喜中，忠诚的恶犬吞咽着“美梦。不屈的骨髓承受反抗着每一次穿刺与鞭答，血脉的悸动每一刻都带来剧痛。鲜血淋漓的梦之美味美妙而易逝，勾起纯粹的憎恨，憎恨正是魔神的狂喜⋯快说呀，“然后，告诉我，快说呀，我的孩子，你吞下的“美梦”是什么滋味”孩子，可知"爱"不过是瞬逝的云露，“权力"才是"美梦”的原料’痛苦与残酷不过是强权的涟漪，破碎的'美梦'才是真正喜悦的食粮。但善于刺穿他人弱点的魔神未曾料到，她的心脏也将被帝君用悬黎千钧射出的碧玉利箭射中。
在神体崩毁的那一刻，万干幻梦的碎片随之进散，纷纷扬扬化为虚无⋯追逐钳制人心的魔神奔向自我毁灭，势有干钧的巨弓宣告了她的报应。
如此终焉本就应是理所当然的恶果，暴虐的欢愉自当迎来残忍的结局。当一切复归平静，黑夜不再有哀叹和哭泣，碧血的劲弓便被悉心封藏。

人，擅使葫芦，其二名为鉴真人，擅使铜镜。他们留下的记录很少，只知修习的是连山秘法，因此被称为「连山二贤」。震雷连山密宫就是他们留下的道场。至于三怪，因她们同在桃都山修行，所以并称「桃都三怪」。这三位不是人类，其实也被叫做「桃都三仙」。一为朱赤引火幽蝶，二为银翎翦玉玄鸟，三为金目乘黄月驹，全是身负神通、能化人形的瑞兽。[29]华清归藏密宫是归藏三隐的密宫，虽然看起来只是一方清池，下面却埋藏着足以昏乱凡人心智的宝藏[89]。相传因连山二贤的秘法的运作，山腹内产生了无穷的雷暴。在秘境震雷连山密宫门前传说偶然能听见隐隐的滚雷之声，呼唤着求索的灵魂[90]。桃都魔神战争时期，璃月一神山上有一巨树，是魔神，名「桃都」。「桃都」背后是死者之地。若人一生罪孽深重，死后魂魄便无法越过桃都抵达彼岸。后来魔神战争到来，「桃都」被卷入战争漩涡，一时之间死地异动，幽冥颠倒，死亡开始侵袭生的领域。桃都实力强大，但为了众生自愿被炼化。后来就有了八奇们横空出世，布下大阵，共同炼化了魔神「桃都」。之后「桃都」的神骸沉入了地脉，化为边界，隔绝了璃月的生与死。[29]八奇虽各有神通，但桃都也非一般魔神，要炼化它，并非易事。因此他们使用了一秘法，名为八门七门大阵。
发动此阵，必须择一门隐去，才可断绝邪恶的出路[29]，镇压着该地的恶螭龙[91]。幽蝶仙子主持了隐去的那一门，便因此消失于世间[29]。相传桃都被炼化后，玄鸟仙子与万象风角灵官结伴云游去了，白马仙子则隐世不知所踪。八奇中无妄姥爷，他倾心于幽蝶仙子，收拾了她的衣钵，并将其传承了下来。幽蝶仙子的衣钵也传承到往生堂手中[29]。沉玉谷仙像待魔神的混战结束后，璃月大地上海潮退去，复归和平。仙兽们从此失却了在神的战争中守护凡人的意义，便纷纷隐居起来，过着与世无争的生活。然而，有一些仙兽依旧怀念着追随岩之神的时光，依然渴念着守望璃月的岁月。仙兽虽是超凡的活物，却依旧被寿限所羁绊。因此，它们向岩王帝君请愿，将肉身化作永恒的磐岩。就这样，慈悲的岩之神允准了它们的祈求，将它们化成了永不腐朽的山岩。[16]鬼神据村落中的长老所言，灵濛山的山民曾信仰诸多鬼神，与山野之间精灵鸟兽中的灵长者同行，但在千年之前那场翻覆天地的混战之后，许多鬼神与精灵陪伴着曾经的住民一同消逝了。尽管如此，多情的群山还是将记忆的回声挽留在了地脉之中，偶然在某个烟雨笼罩的清晨或入夜时分反复重现[92]。灵濛山在岩王帝君镇抚山林之后，曾有一位船工进入灵濛山下错综的溪涧，迷失在傍晚潮湿的山雾中。撑竹篙渡过泛着蓝紫色幽光的水草，经过落花迷离的树丛，追随着即使在梦中也未曾见过的碧色禽鸟，船工闯入了沉睡的洞窟。借着玉石的荧光与菌类的幽光，船工恍惚看到了旧时先民的身影。他们身着古老的纱质长衣，衣角点缀着水光般纯净的玉珏和无名的芳草，仿若山中鬼神。他们排成行列，站在深潭岸上。诵唱着船工未曾听闻的歌谣：「昼晦冥冥兮独含怨，幽风携雨兮烟气濛。」「徒留灵修兮怅岁晏，独怨君子兮竟后来。」结合英文版本参考翻译：我的怨恨是如何孤独地潜伏在日暮之光里！幽邃的风裹挟着雨，浓雾如何掩盖！没有留下一个孤独的神哀叹失去的岁月——唉，唉！只因为你来得太晚了。歌谣哀伤清静，似有怅怨之意。船工再仔细望向洞窟之内泛着幽光的人影，他们开始一个接一个解下玉珏，投入漆黑的潭水之中，仿佛从未察觉到来客的存在。船工愈觉凄然阴森，便急忙撑篙返程，在一路上留下记号，返回了翘英庄。据说后来月海亭又派来测绘师，企图探寻未尝在地图上出现的神秘聚落，也有人说，曾有千岩军队伍进入深山探查可能的非法窝点，但均一无所获。遗珑埠名医蓝璟（蓝砚先人）青年时进入灵濛山探查传说中的神秘洞府，以求古代药方，归来后却再不开口提及此事。直至其人去世时，他的家人才在遗物中寻得一方砚台，据说色如清水，光似高天，然而斯人已逝，这方砚台的来历终不可知。后来，名医后人因商船经营不善而破产，这方砚台也流落民间，从此下落不明[93]。往生堂往生堂夜叉与诸魔神厮杀后，逝去的魔神留下的怨念引发了瘟疫。后来此瘟疫被往生堂初代堂主解决。魔神战争时期，诅咒、瘟疫等灾祸横行于世，而主动站出来净化空气、焚烧尸体来阻止瘟疫传播的人正是往生堂的建立者们，当时他们的工作类似于医者。千百年来，他们一直致力于维持边界的平衡，不让「生」与「死」之间产生严重偏离。像是早已被消除的诅咒和瘟疫，自然要作为秘密，让它们以普通的传言而非真相流传在民间。[94]护摩医者（往生堂创立者）持妖红的手杖，引燃沾染了邪污的事物。那些被恶业与灾殃缠上的往者，未能承受深重哀伤的逝者，将在火中化作灰之蛱蝶，自浊世的一切不幸与嗟伤中解脱。而点燃无数柴禾堆的医者，传说也最终化成了飞蝶般的烟。后来这种祭仪随着和平与岁月被遗忘。[95]赞玛兰比夫茏的分身之一赞玛兰，从须弥迁到层岩巨渊地下安居[96][97]。
