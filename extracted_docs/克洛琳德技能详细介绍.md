# 克洛琳德技能详细介绍

克洛琳德是《原神》中的雷元素五星角色，枫丹的执法官，称号"执法者"。
元素战技「雷霆制裁」：
- 释放雷元素能量，对周围敌人造成雷元素伤害并施加"雷元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「雷霆审判」：
- 释放强大的雷元素能量，对周围敌人造成雷元素伤害，并进入"狂怒"状态。
- 冷却时间：20秒。
- 能量消耗：80点。
固有天赋：
- "制裁的祝福"：雷霆制裁命中敌人后，有15%概率触发"感电"效果。
- "雷元素增幅"：克洛琳德的普通攻击和重击造成暴击时，恢复3点元素能量。
- "审判的威严"：元素爆发持续时间内，克洛琳德的雷元素伤害提升20%。
