# 500年前

言灵在传说中，世界树上倒悬的贤人海洛塔帝获得了书写符文操使言灵的知性，沿着树根建造的坎瑞亚王国最终窥见了魔天的秘密。[16]献出「力量」求取「人智」的路径，即是白树上意图窥见魔天奥秘的「倒吊人」。掌控于凡世之人手中的智性虽有诸多缺陷，却也不可被称作谬误。恰恰相反，颠倒的眼瞳中映出的，是无限明光下纯净的真实[17]。此后，坎瑞亚人能够不使用神之眼，就能透过咒语导出元素之力。在古国的灾厄降临后，部分遭受诅咒坎瑞亚人因为保留操使言灵的记忆，成为吟唱着漆黑符文的“丘丘萨满”[18]妖僧霍德望彼时黑日已经取代了赤月，也有人已经得知了深渊的秘密，窥探星空成了禁忌的知识，却也成了少数为独眼君王服务之人的特权。年轻的学僧拉乌斯万格意外透过镜子瞥见了星空的奇景，也窥探到了深渊的秘密，他被深深吸引，这也是一切罪行的开端。逾越身份窥探秘密，拉乌斯万格被判以流放之刑。这对于出身异国的他无异于死罪，一旦来到地上，因背弃神明而受到古老的诅咒就会将他吞噬。自认已经见证过乐园胜景的拉乌斯万格走出了洞穴，准备迎接阳光的炙烤，可他却并未遭受那可怖的诅咒，也从此自诩“被命运选中之人”。或许是坎瑞亚的炼金与占星术太过先进，又或许是北国的宫廷之内早已只剩迂腐之辈，学僧拉乌斯万格摇身一变化身妖僧霍德望教授，成为了前代冰神的宠臣。沙皇早有叛逆之心，他寄希望于亥珀波瑞亚遗留下来的秘宝，希望以此创造属于月下生灵的永恒乐园。他派出妖僧作为自己的钦差，却未曾想过妖僧早已倾心于深渊，誓要将世界作为薪柴，带他走向那深邃的黑暗。为此他设下许多阴谋布局，详见至冬[19]。黄金百年前的坎瑞亚已会使用炼金技术，其中的炼金术士「黄金」莱茵多特在宫廷内有极高的声望。
