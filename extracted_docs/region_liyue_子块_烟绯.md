# 烟绯

烟绯的父亲是一位仙人，曾与岩王帝君征战许久，与一位普通的商人结婚，生下了烟绯。烟绯长大后，父母便出门远游去了。烟绯在萍姥姥的照顾下生活，逐渐成长为璃月头牌法典解读专家。凝光在每年修订法典时，都会大量参考烟绯的所作所为[160][167][161]
云堇云堇出身于璃月曲艺世家（原是锻刀世家[168]），母亲继承了祖父的衣体，曾是一代名角，父亲则是一名剧作家。在耳濡日染下，云堇小小年纪便开始学戏[169]，年少成名，成为了“云瀚社”的当家，是璃月港内风头正劲的名角[170][171]。
