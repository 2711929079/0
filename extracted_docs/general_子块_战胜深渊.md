# 战胜深渊

旅行者的古名「希望」打造完成，玛薇卡与旅行者前往夜神之国直面古斯托特。古斯托特化形成火龙王修库特尔的模样与旅行者和玛薇卡决斗，期间旅行者激活的火元素属性，最终在众多的灵魂和在地面上的纳塔人与夜神和仙灵们的帮助下战胜了古斯托特。之后，玛薇卡必须向死之执政履行契约支付自己的生命作为使用死之执政力量的代价。夜神在对抗深渊的漫长过程中生命已走到尽头，她打算把生命支付给夜神，让夜神重新获得生命力，但自己也会牺牲。不过玛薇卡提供的生命力也有极限，夜神迟早会死亡，但能帮夜神之国撑一段时间，到时候再想办法。但队长代替了玛薇卡，如愿与夜神融为一体，使夜神得以不死，并改变夜神之国的规则，使其可容纳纳塔人以外人的灵魂，使自己在坎瑞亚的众多部下的灵魂拥有归宿，并以此作为向对他降下不死诅咒的死之执政的复仇。拥有不死诅咒的队长支付的生命使夜神不死。[62]
坎瑞亚的蓝图命运的织机需要重建坎瑞亚的地脉完成复国，需要坎瑞亚的所有记忆，然尔深渊教团的人因记忆磨损无法达成。于是深渊教团举全兵力攻入坎瑞亚引诱戴因斯雷布出现，再由戴因的熟视仇人哈登诱骗他回想坎瑞亚的过去，导致他的所有关于坎瑞亚的记忆被命运的织机读取，坎瑞亚地脉的蓝图完成了。此事在夜神的帮助下完成的，但夜神向旅行者的血亲谈了某个条件作为代价。坎瑞亚的地脉虽完成却无处存在，旅行者的血亲只得寻找承载坎瑞亚地脉的方法。同时，因伊斯塔露暗中操作，旅行者将自己的飞船休眠舱钥匙给了五百年前的蒂莱尔（旅行者也可以将钥匙收回，但结果是一样的，因为五百年后旅行者的血亲也会受到伊斯塔露的干涉见到蒂莱尔，为了防止收回钥匙导致涟漪效应使世界走向未知的命运使自己的计划（即自己因为被降格了，所以成立深渊教团用一整个世界与天理对抗打算获取与之平等对话的资格恢复降临者身分，和让新坎瑞亚成为自己合适的新的家乡）不能如计划中进行，于是将自己一样的另把钥匙丢给蒂莱尔。旅行者在内心纠结下顺从血亲的计划。）。经历各个事件，旅行者与血亲重逢，问到他/她做的事对我们自己而言有意义吗？旅行者的血亲会答：「从一开始，就只关乎你我」。因此，旅行者的血亲帮助坎瑞亚复国是与旅行者和自己最初旅行的目的相同的，而他们旅行最初的目的是在故国覆灭后找到适合的新的家乡（见双子的旅程），因此旅行者的血亲重建坎瑞亚的目的，大概是为了将重建后的坎瑞亚弄成适合当成自己和旅行者新的家乡的样子。在旅行者的血亲表达自己打算利用整个世界换取与天理平等对话的资格时（即自己因为被降格了，所以成立深渊教团用一整个世界与天理对抗打算获取与之平等对话的资格恢复降临者身分），天理的维系者疑似听到了此对话。之后，旅行者的血亲离开[63]。
