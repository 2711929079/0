# 蛇

不知经过了多少岁月，渊下宫里有一小童，与伙伴做了赌约。他只身一人，潜到三隅之外，避开龙嗣痕迹，想去寻那龙骨花。但是他却在洞中见到了一只未曾见过的大蛇。不知为何，小童见蛇之庞巨、诡异，并不觉恐怖，反而有亲近之感。「我乃渎身渎名之蛇神，虽有眷属百千，但所荫蔽之众已无一人。今日落入此界，与你相见，也算有缘。你虽非我民，但终是人子。有何愿望，但说无妨。」「试问，你能做我们渊下之民的神吗？」于是一人与一蛇，面对太阳之子的王权、境外的龙嗣侵攻，力挽狂澜之演义就此开幕。[5]海祇大神到来，蛇之地的名字并没有改变。一般古早绘画，无鳞蛇为“奥罗巴洛斯”，有珊瑚的蛇为“远吕羽氏”。
海祇御使与珊瑚虫为了结束龙蜥的灾害，大蛇创造了一个叫阿祇的海祇御使，她有着沟通人类和龙蜥的能力，本来是希望她能作为人类和龙蜥沟通的桥梁，结果她在年幼时就被龙蜥劫走并抚养长大，所以她以为自己是龙蜥人[7]。海祇御使是奧罗巴斯最初的眷属。奥罗巴斯使用自己的第二个眷属“珊瑚虫”转化无序的元素力，遏制了渊下宫的圣土化。人们从奥罗巴斯身上采集“血枝珊瑚”，将其植入深海龙蜥，最终选育出可以稳定提供血枝珊瑚的深海龙蜥个体。奥罗巴斯的到来结束了太阳之子制度，对龙蜥势力进行全面反击，用珊瑚王虫改造蛮荒元素力。在此期间大蛇阅读了《日月前事》的内容，意识到这是被天空秩序所封禁的历史，立即禁止了《日月前事》的传播。但不幸的，偷看禁忌的大蛇被天理叛了死刑，必须在无人怀疑的情况下自尽。
