# 蒙德.塞西利亚古国

疑似天钉的东西（原文本写作晨星）降落在坠星山谷的星落湖，塞西利亚的古国在此后覆灭。[43]建立塞西利亚花苗圃的古国覆灭后，只剩摘星崖存在这种花的踪影。夏宫灵囿深埋地底，被称为仲夏庭园。
璃月.层岩巨渊文明天钉降下后层岩巨渊不再适合居住。沉玉谷的先民自遥远的层岩巨渊迁居而来，他们在此地筑起了直通高天的祭台，留下无数恢宏的殿宇。[44]该文明古时的祭场曾因葬火之战埋入地下，又随着天钉陨落而隆起，最终又因摩拉克斯与若陀龙王对厮杀造成的破坏而显露出来。但曾经来往此间的人早已不再了[45]。
