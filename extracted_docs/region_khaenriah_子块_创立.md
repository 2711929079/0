# 创立

约四百数十年前[70]，在自称“罪人”的「预言家」维瑟弗尼尔[52]引导下，亚尔伯里奇一族成员克洛达尔·亚尔伯里奇创立了深渊教团。坎瑞亚灾变后，坎瑞亚的子民化作魔物，一部分人则成为了深渊法师。克洛达尔联合这些化作魔物但仍有理智的坎瑞亚遗民成立深渊教团，并着手执行“命运的织机”计划，希望自己的私生子卡利贝尔·亚尔伯里奇，能“在极度的悲伤中成为命运的织机”。为了计划，他盗取蒙德的一个风神像。此外，一些加入教团者会成为昭告着漆黑之音的深渊使徒与深渊咏者。百年之后，即使是深渊也无法实现克洛达尔的夙愿，再加上不死诅咒带来的磨损，导致其变得疯疯癫癫，最后不知所踪。后来，他破除了不死诅咒的影响，丧身在须弥。深渊教团成立之前，克洛达尔曾与旅行者的血亲有所接触。五百年前，克洛达尔不满意于家族为其安排的生活，在结识一蒙德女子之后得子卡利贝尔。后坎瑞亚灾变，妻子失散（实际上已经去世，被埋葬在田地里），他自己作为“纯血坎瑞亚人”蒙受不死诅咒，而其子卡利贝尔则变为丘丘人。此后他长期居住在须弥道成林一林间小屋中，试图用他多年前所读王庭禁书里提到的须弥药方使卡利贝尔恢复心智。旅行者的血亲到来后，克洛达尔与其一同制作让卡利贝尔恢复神智的解药，途中阴错阳差地误入一座圣所，并遇见一位“罪人”。在罪人的幕后操纵下，卡利贝尔恢复神智，克洛达尔则沉迷于改变卡利贝尔命运的深渊力量，变得疯疯癫癫，成立深渊教团。也是从此刻开始，旅行者的血亲决定走上通往深渊的不归路…[1]哈登加入了深渊教团，变成深罪浸礼者，在漫长时光里损失记忆。
