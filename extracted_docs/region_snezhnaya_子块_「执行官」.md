# 「执行官」

愚人众拥有七国最强的武力，「执行官」们被冰之神赐予了权柄，获得了超越凡人的力量，冰之女皇是全体愚人众执行官的唯一效忠对象[58]。
建立「愚人之城」格鲁波夫第七百九十九纪的第四年，女皇让愚人众在至冬堡附近建起一座新的城塞，名叫格鲁波夫，意思是愚人之城[8]。
