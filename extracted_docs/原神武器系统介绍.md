# 原神武器系统介绍

《原神》的武器系统种类丰富，主要包括：

1. 武器类型：
   - 单手剑：适合多种角色，攻速快，通用性强
   - 双手剑：攻击力高，破盾能力强
   - 长柄武器：攻击范围大，部分武器有特殊效果
   - 弓：远程攻击，适合输出和辅助
   - 法器：元素伤害为主，适合法师角色

2. 武器稀有度：
   - 三星：基础武器，容易获取
   - 四星：优秀武器，通过祈愿和活动获得
   - 五星：顶级武器，通过限定祈愿获得

3. 武器属性：
   - 基础攻击力：武器的主要属性
   - 副属性：额外的属性加成，如暴击率、暴击伤害、元素精通等
   - 武器特效：独特的被动技能效果

4. 武器升级：
   - 通过消耗武器经验素材提升等级
   - 等级上限受武器稀有度限制
   - 升级提升基础攻击力和解锁特效等级

5. 武器突破：
   - 达到特定等级后需要突破
   - 突破需要对应的材料
   - 突破后提升等级上限和基础属性

6. 主流武器推荐：
   - 单手剑：雾切之回光、斫峰之刃、风鹰剑
   - 双手剑：狼的末路、松籁响起之时、螭骨剑
   - 长柄武器：护摩之杖、薙草之稻光、和璞鸢
   - 弓：阿莫斯之弓、终末嗟叹之诗、天空之翼
   - 法器：四风原典、神乐之真意、天空之卷

7. 武器选择策略：
   - 主C角色优先选择高基础攻击力和暴击属性的武器
   - 辅助角色优先选择元素充能和元素精通属性的武器
   - 根据角色天赋和队伍需求选择合适的武器
