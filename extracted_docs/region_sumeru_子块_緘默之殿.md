# 緘默之殿

最初由尊主赫曼努比斯在神王阿赫玛尔死后带着他的信仰者连同长年担任祭司的提纳勒人一同到图莱杜拉城所建，职责是保护和管理赤王文明所遗留的知识，后在与居尔城的战争中因被沙漠民排斥不得不离开沙漠，进入雨林，与教令院曾建立合作。[65]
