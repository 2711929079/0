# 命运的织机完工

随着深渊教团「命运的织机」计划的推进，其操控地脉、投放记忆的能力开始增强。卡利贝尔不甘心自己在这个世界上仅仅活了八岁，他好奇，自己如果安静地活在这个世界上会经历怎样的人生。于是他操控「命运的织机」开始向维摩庄的村民们投放自己和他们一起生活的记忆。在记忆中，他为人热情、乐于助人，还擅长陪伴他人、安慰别人。他还听村民阿托莎倾诉，变成了她无比亲近的朋友，阿托莎甚至对他产生了情愫。但由于村民们在现实中找不到卡利贝尔，许多维摩庄的村民前往冒险家协会发布寻人委托。许多冒险家前来寻人，但一无所获。旅行者接到委托，调查「只存在于记忆中的人」。戴因斯雷布也追查深渊教团来到此处。二人认为这个现象与「命运的织机」有关，决定联手调查。第二天一早，戴因斯雷布的记忆中出现了一段关于他的片段。他决定用「耕地机的眼睛」引出深渊教团。在旅行者和戴因斯雷布进入虚假的「藏宝地」之前，卡利贝尔向戴因斯雷布投放了「戴因斯雷布将『眼睛』交给旅行者」的虚假记忆。之后戴因斯雷布吩咐旅行者先离开，去调查失踪者的事。戴因斯雷布原本的计划是将深渊教团引进虚假的藏宝地，并将藏匿于自己身上的「眼睛」交给旅行者，自己独自面对殿下。但是由于戴因斯雷布没有真正地将眼睛交给旅行者，再加上他在与殿下对战时仍显犹豫，第一颗「眼睛」被殿下夺走，命运的织机因此完成。旅行者和派蒙从「藏宝地」返回维摩庄的过程中，发现受深渊教团影响而躁动的丘丘人们安静了下來。二人通过推理，推测出了卡利贝尔的存在和此时所在的位置。二人来到了卡利贝尔和阿托莎在记忆中经常一起交流的那棵树下，找到了正在向阿托莎告別的卡利贝尔。见到旅行者，卡利贝尔非常高兴，将其拉入了自己的意识空间，把自己的遭遇、「命运的织机」计划的真相、自己给维摩庄村民投放记忆的目的和戴因斯雷布的失策都告诉了旅行者。卡利贝尔表示，自己无法窥见「命运的织机」计划的意图，但此刻他的目的已经达成——由他向所有丘丘人投放一段有关爱和童话的故事，安抚所有的丘丘人。旅行者认为，即使卡利贝尔不曾存在过，他也足以安慰、激励许多人了。卡利贝尔的使命完成了，他闭上眼睛，身躯在风中消散。殿下也赶到了卡利贝尔的意识空间，但没来得及见他最后一面。旅行者回到了现实的维摩庄，发现村民们的脑海中多出了卡利贝尔被父母亲接走的记忆，这正是他最后的告別。旅行者的口袋里凭空多出了一张他、殿下和卡利贝尔在意识空间的合影。在那本应不再存在的空间，卡利贝尔不知用何种手段超脱了規则，为旅行者留下了珍贵的相片。[52]
坎瑞亚的蓝图命运的织机需要重建坎瑞亚的地脉完成复国，需要坎瑞亚的所有记忆，然尔深渊教团的人因记忆磨损无法达成。于是深渊教团举全兵力攻入坎瑞亚引诱戴因斯雷布出现，再由戴因的熟视仇人哈登诱骗他回想坎瑞亚的过去，导致他的所有关于坎瑞亚的记忆被命运的织机读取，坎瑞亚地脉的蓝图完成了。此事在夜神的帮助下完成的，但夜神向深渊殿下谈了某个条件作为代价。坎瑞亚的地脉虽完成却无处存在，深渊殿下只得寻找承载坎瑞亚地脉的方法。同时，因伊斯塔露暗中操作，旅行者将自己的飞船休眠舱钥匙给了五百年前的蒂莱尔（旅行者也可以将钥匙收回，但结果是一样的，因为五百年后深渊殿下也会受到伊斯塔露的干涉见到蒂莱尔，为了防止收回钥匙导致涟漪效应使世界走向未知的命运使自己的计划（即自己被降格了，所以成立深渊教团用一整个世界与天理对抗打算获取与之平等对话的资格恢复降临者身分，和让新坎瑞亚成为自己合适的新的家乡）不能如计划中进行，于是将自己一样的另把钥匙丢给蒂莱尔。旅行者在内心纠结下顺从殿下的计划）。经历各个事件，旅行者与殿下重逢，问到他/她做的事对我们自己而言有意义吗？殿下会答：「从一开始，就只关乎你我」。因此，殿下帮助坎瑞亚复国是与旅行者和自己最初旅行的目的相同的，而他们旅行最初的目的是在故国覆灭后找到适合的新的家乡（见双子的旅程），因此旅行者的血亲重建坎瑞亚的目的，大概是为了将重建后的坎瑞亚弄成适合当成自己和旅行者新的家乡的样子。在殿下表达自己打算利用整个世界换取与天理平等对话的资格时（即自己因为被降格了，所以成立深渊教团用一整个世界与天理对抗打算获取与之平等对话的资格恢复降临者身分），天理的维系者疑似听到了此对话。之后，殿下离开[28]。
