# 灵善坊

曾接待过「影向役者三人众」的千门虚舟成为引退的天狗族长灵善坊灰心之后虚度余生的樊笼，沉入了海下[26]。惟神晴之介环游世界途中来到千门虚舟，发现老师灵善坊在此度过余生，并且发现狐斋宫来过了[54]。（狐斋宫曾带领人们躲进该地的地宫[3]）
清籁雷暴浅濑响清籁岛的浅濑响，意外遇见了一个神似雾切高岭的海盗，赤穗百目鬼。她对其十分照顾，一直当做自己的儿子来养。浅濑响也收养了一只猫妖，寝子，两人一猫就这样打理着浅濑神社。坎瑞亚灾厄之后的清籁岛民多成为海贼以维持生计，赤穗百目鬼与其旗舰清籁丸成为了清籁岛人的骄傲。
