# 新月神像与月之轮

挪德卡莱矗立着与七天神像相似的「新月神像」，为霜月之子雕刻而成，形象是「月之少女」库塔尔，即前愚人众执行官第三席「少女」哥伦比娅，现在哥伦比娅居住在银月之庭。部分新月神像被愚人众用机械装置改造。关于新月神像的更多内容见七天神像。挪德卡莱的角色佩戴类似神之眼的「月之轮」，月之轮由三月女神所造，用于赐予追随者们以便与之沟通。与燃素相同，月之轮依靠更纯净的力量运行，而非受到僭主污染的元素力。关于月之轮的更多内容见神之眼。
