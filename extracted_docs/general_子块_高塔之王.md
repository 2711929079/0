# 高塔之王

约3000年前，风王的神位存在空缺。来自不同地域的两位魔神「高塔孤王」迭卡拉庇安与「北风之王」安德留斯为了争夺神位而开战。迭卡拉庇安建造了让臣民免于风雪天灾侵害的王都（即旧蒙德，今风龙废墟）。安德留斯对高塔的孤王宣战，也无法损伤王都分毫。古城整体呈环形。在内环与外环之间，似乎所有人民都被事先安排好了各自的位置。正中央坐落着过于高大的塔楼，那里是烈风之王的宫城。高塔之上的风之君王，对自己所造的王城十分满意，他在高塔上接受所有追随者的跪拜。王都内的众人对高塔上的孤王俯身也并非出于敬仰爱戴，而是在无尽的烈风中不得不中躬下身子。迭卡拉庇安则认为他们顺服。
