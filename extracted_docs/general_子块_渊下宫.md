# 渊下宫

早在提瓦特大陆的远古时期，龙是这个世界的支配者。之后天之王座的到来，它们与七位元素龙王开战。七位龙王全部被打败，七个王国全部对天上俯首称臣，此刻天之王座开始了对这片天地的创造。山川与河流落成，大海和大洋接纳了部分对天之王座的反抗者，这就是渊下宫里深海龙蜥的由来。在这之后的又几百年里，自然万物与人类诞生。人们生活在这片土地上安居乐业，整个世界都在统一的文化之下。直到天上第二个王座的出现。随后战争又起，天倾地裂，海渊之民的先祖与他们栖居的土地落入此处，也就是如今的渊下宫。白夜之国土，以三角三隅为标。曾经这里就是人类势力与龙蜥势力拉锯的极限了。三角在白夜国时期修建有三界塔，用来调和三界。古时的名字已经失传，在海祇到来之后，才改了现在的名字。三界塔非常重要，并且不处在风与水的体系下。甚至与之相反，它们负责平衡整个白夜国的倾向，控制着白夜国的风与水。一旦三界塔出现问题，整个白夜国将有大灾。因此它们由秘法隐去，仅能由巫女与御使唤出。渊下宫不见日光，因此并无日夜之分别，取而代之的是“白夜”和“常夜”之分。渊下宫的环境与机关在“常夜”或“白夜”里会呈现出不同风貌。后来渊下宫荒废，在魔神奥罗巴斯帮助下，居民迁徙到海祇岛，成为了珊瑚宫的原住民，并起了稻妻式名字。
