# 散兵

約三百年前，在流浪了近百年后的流浪的最后，倾奇者视自己为超越者，凭借自己的意志选择了「国崩」这个名字，后来被丑角招募加入愚人众。在至冬本土，一位陌生人接待了他。那人自称为「博士」，对他的到来深表欢迎，并邀请他成为自己实验的重要参照对象，参与到伟大的研究中来。「人偶」技术最初源于坎瑞亚，他作为雷神的造物更显特殊。「博士」对这一领域充满兴趣，以他为蓝本不断研究实验数十年才终于得到日后被用于制作「切片」的技术基础。作为回报，「博士」解开了藏匿在他躯体内的封印，使他能力大增，足够与席位较低的执行官一战。可即使到了这一时期，他依然没有索要任何名字。同僚们始终称呼他为人偶，他也这样定义自己，坚信自己是不畏惧死亡、不会被消耗殆尽的人偶。奉女皇之命，他率一支部队前往深渊展开探索，在那里度过了漫长的时光。期间不止一次受伤，被「博士」修复，在伤痛中变强，又在更强大的敌人面前负伤。再后来，他带着深渊探索的成果重返至冬，获封第六席，使命也从深渊探索改为机动待命，支援愚人众在各国的秘密行动。经过「博士」的调整后恢复被封印的力量。后来，因探索深渊有功，获封第六席，并接受了新名字「散兵」。[73]在愚人众指派的探索深渊的任务终于结束后，「散兵」已经因「博士」解放了他的封印与之后的人体改造而获得足够向丹羽家复仇的力量，于是灭绝丹羽家族以示复仇。[15]。
丹羽一族滅絕丹羽一族疑似因丹羽久秀逃跑受罚（实则被博士杀害，伪造为逃跑），约三百年后，丹羽家主失踪（被散兵所害），只留一子丹羽义庆于世，最后由枫原家收养，改名枫原义庆。
