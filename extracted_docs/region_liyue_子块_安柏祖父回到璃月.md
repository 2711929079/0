# 安柏祖父回到璃月

安柏的祖父从蒙德不告而别，理所当然地被西风骑士团视为叛逃，导致其带领的侦查小队保受波及[196]。他实为回到璃月轻策庄[197]。
沈秋拾剑录行秋将自己的故事为素材，以“枕王”作为笔名撰写了《沉秋拾剑录》，但在璃月无人爱看。后来一位稻妻的商人将这本书带到了稻妻的八重堂，结果广受好评[198]。
