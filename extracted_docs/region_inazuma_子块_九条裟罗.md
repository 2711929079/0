# 九条裟罗

一名年幼的天狗少女在山林里遇到魔物袭击，奋力逃跑的过程中意外获得神之眼。在获救后，掌管天领奉行的九条孝行看中了她的潜力，于是收养了她，赐名为九条裟罗。后来九条裟罗成为幕府军中的大将[78]。
鹿野院平藏幼时的鹿野院平藏在一次祭典上结识一好友，好友经常送给他一些新奇的小玩意儿，后来平藏得知这些都是好友偷来的后与其绝交，并把之前收到的礼物全部丢掉只剩一颗曾经象征友谊的鹅卵石。一年后的祭典上，平藏鬼使神差的带着那枚鹅卵石来到祭典，见到了倒在地上的好友，好友是为了保护自己装有鹅卵石的钱袋不被小混混抢走，而被匕首刺中了胸膛。好友死后，平藏五味杂陈，誓要对抗世间所有的“罪”[79]。
