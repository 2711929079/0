# 胡桃技能详细介绍

胡桃是《原神》中的火元素五星角色，璃月往生堂第七十七代堂主。

元素战技「蝶引来生」：
- 效果：消耗当前生命值的30%，进入"彼岸蝶舞"状态，普通攻击、重击和下落攻击转化为火元素伤害。
- 伤害加成：生命值越低，伤害加成越高。
- 持续时间：9秒。
- 冷却时间：16秒。
- 能量消耗：无。

元素爆发「安神秘法」：
- 效果：释放焰蝶，对周围敌人造成火元素伤害，并根据胡桃的生命值上限回复生命值。
- 回复效果：生命值越低，回复量越高。
- 冷却时间：12秒。
- 能量需求：60点。

天赋技能：
1. 血之灶火：胡桃生命值低于50%时，获得33%暴击率加成。
2. 蝶隐之时：元素战技持续期间，重击不消耗体力。

角色定位：
- 主要定位：主C输出
- 次要定位：副C
- 核心优势：高额火元素伤害、生命值转化为伤害

培养建议：
- 武器推荐：护摩之杖、和璞鸢、决斗之枪
- 圣遗物推荐：追忆之注连四件套或魔女的炎之牢四件套
