# 甘雨技能详细介绍

甘雨是《原神》中的冰元素五星角色，璃月七星秘书，半人半仙的麒麟混血。

普通攻击「流天射术」：
- 效果：进行蓄力射击，造成冰元素伤害。
- 二段蓄力：发射冰棱，造成高额冰元素伤害并产生冰棱碎片。
- 特点：蓄力攻击伤害高，是主要输出手段。

元素战技「山泽麒迹」：
- 效果：召唤冰莲，对周围敌人造成冰元素伤害并施加冰元素附着。
- 持续时间：冰莲持续存在6秒。
- 冷却时间：12秒。
- 能量消耗：无。

元素爆发「降众天华」：
- 效果：降下冰棱，对范围内敌人造成持续冰元素伤害。
- 持续时间：15秒。
- 冷却时间：20秒。
- 能量需求：60点。

天赋技能：
1. 天地交泰：元素战技结束时，为甘雨恢复20%元素能量。
2. 唯此一心：重击伤害提升20%，并减少30%蓄力时间。

角色定位：
- 主要定位：主C输出
- 次要定位：副C
- 核心优势：远程冰元素输出、高伤害、稳定的元素附着

培养建议：
- 武器推荐：阿莫斯之弓、试作澹月、破魔之弓
- 圣遗物推荐：冰风迷途的勇士四件套（冰元素伤害加成）
