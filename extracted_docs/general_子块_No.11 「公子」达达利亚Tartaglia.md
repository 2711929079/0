# No.11 「公子」达达利亚Tartaglia

中文CV：鱼冻（李春胤，成年）/薄棠（幼年） 日文CV：木村良平详见至冬#「公子」他是最年轻的愚人众执行官，也是最危险的愚人众执行官之一。在1.1版本成为自机角色，拥有水神之眼和雷之邪眼。然而，「公子」似乎永远与他的同僚们不太合得来。作为最纯粹的战士，他在这个充满阴谋的集体中显得格格不入。在黄金屋之战与他交战至第三阶段，开启魔王武装后会显示其为愚人众十一执行官末席，在本人的语音“初次见面…”中他再次声称自己是第十一席。他在家中排行第三，家中兄弟姐妹还有安东、冬妮娅和托克（最小）。在14岁时失足跌入了深渊，偶遇丝柯克并向她拜师，在逗留于深渊的三个月中学到了丝柯克「畅行深渊的得意之术」。而深渊外的时间仅仅过了三天。在璃月为了获得岩神之心，将旅行者引荐给往生堂客卿钟离，然后设法打听到仙祖法蜕的下落，前往黄金屋，想要获得岩神之心。但被旅行者成功击败，并且推测出岩神没有去世，随后放出了奥赛尔。之后，奥赛尔被璃月众人成功镇压（岩神没有出面），而由于放出奥赛尔这一鲁莽行为，导致至冬一方在与璃月的外交中极为被动，璃月断绝了两国合作。稻妻主线结束，「散兵」取得了雷神之心后，便断绝了一切与愚人众的联络。他前往稻妻，想要找到「散兵」，可惜一无所获。在枫丹，他在向卡布里埃商会讨债时，碰巧遇上了旅行者和派蒙，于是便和旅行者、派蒙聊天。而商会成员不懂得读气氛，屡次打断聊天，他便忍无可忍选择暴打了一顿商会成员。途中他发现自己的水元素力不稳定，说出了自己最近的异状和往事，并给出猜测，然后将神之眼交给了旅行者。在枫丹危机结束后，旅行者将神之眼交给阿蕾奇诺，委托其代为归还。后被芙宁娜指控犯下少女连环失踪案，在一通审判后，证明他与本案无关，但「谕示裁定枢机」却做出了「公子」有罪的审判。他选择暴力拒捕，并试图开启魔王武装，被那维莱特直接打晕，关进了梅洛彼得堡。之后他越狱逃脱并设法进入了原始胎海中，开启魔王武装与吞星之鲸大战。丝柯克赶到后，将已经昏迷的他连同被打包的吞星之鲸一同扔进了空间裂隙里。枫丹危机解决后，他被送回至冬养伤。但他认为自己还有没做完的事情，所以跳船又回到枫丹，试图找到师父，但是没有找到。目前正在参与「严冬计划」。在挪德卡莱「博士」阳奉阴违、集齐三月权能的事件中，为「仆人」和「木偶」带去冰之女皇的口谕，「——许可执行官特殊时期专断之权。如果多托雷构成威胁，允许阿蕾奇诺与桑多涅对其实施剿灭」，在听到旅行者被抓后，想要冲进去救人，但被仆人阻止，随后返回至冬。他的命之座图案就是他的执行官代表图标，一头独角鲸鱼。
执行官互评折叠「散兵」：头脑简单，四肢也不甚发达。让他做最后一席都是在拉低执行官的整体水准。万幸，我已经不在其列了。「仆人」：平心而论，他的性格并不适合做执行官。他常以善意揣测别人，也常被别人利用。当然，他的天赋很好，仍然在不断成长，对他的评价不必急于一时。「少女」：听说他和家人的感情很好，休假的时候会举家外出，去湖边露营烧烤。桑多涅说他是执行官当中生活得最像普通人的那一个，我不了解普通人，但我觉得那样的日子听上去倒是不错。
