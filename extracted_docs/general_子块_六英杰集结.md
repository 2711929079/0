# 六英杰集结

队长见证过深渊带来的巨大牺牲，因此打算帮助纳塔。他想回避与旅行者一战的场面(基于主角是坎瑞亚殿下的血亲还有他原来的身份)，同时一直关注着火神玛薇卡计划的进展。队长找到旅行者，坦率地说到她选择拯救纳塔的这个计划风险很大，其力量是否真的能够击败深渊。队长最担心过大的责任会让玛薇卡越发无法直面惨痛的事实，最后连启动备选方案的时机都完美错过，所以队长打算使用神之心替他动手。现在队长落败不打算再去打神之心的主意，但队长拯救纳塔的目的没有改变，他找到新的办法，希望旅行者能协助。而欧洛伦也在帮助队长。夜神之国受损后，内部的灵魂会逸散。这些灵魂无法回归地脉，只能游离在外，其流失会令夜神之国进一步衰败。有的办法是让欧洛伦出生起自身残缺的灵魂吸收，作为载体进入夜神之国再送回强化地脉力量，但这个献祭计划当时失败了。欧洛伦长大后得知自己刚出生时被献祭的事情，认为「如果当时自己被献祭成功的话大家就能得救」，于是认定自己有帮助拯救纳塔的责任。在了解队长的计划后，他加入队长的行列。与旅行者的会谈因茜特菈莉介入而终止。队长了解这备选计划的方案，即透过神之心抽空整个夜神之国的灵魂与记忆，编织出一张包裹住地脉，抵抗深渊侵蚀的大网。但他想从根源解决深渊问题，他打算重构地脉，于是寻找能重构地脉的古代「秘源机关」。队长当时找来了三位连玛薇卡都不知道的龙遗物学者—阿贝雷瓦、博苏维、奎克特利(实为由盗火贤者假冒），知道了「秘源机关」的事情。重构地脉并不是修补，几乎等同于放弃现有地脉中的一切，会让纳塔人失去记忆历史与文化，甚至导致神智异常，和动用「神之心」付出的代价很相似。玛薇卡于是派人阻止，重构地脉的计划因他们的介入失败，过程中欧洛伦得到大灵的承认，成为六英杰之一。启动的「秘源机关」因其重构地脉的功能同时也唤醒了纳塔的地脉（夜神之国）的掌控者夜神。「古名」不会记录纳塔之外的经历，也就不会被记录到夜神之国中，无法得到大灵承认的记忆与经历，就无法成为「古名」的材料。为主角打造「古名」，只有夜神能决定。但人们必须通过做梦(出神)与它的意识产生接触。而非纳塔本地人的旅行者无法进行出神仪式前往夜神之国。因此这次唤醒使得回声之子工匠希诺宁直接说服夜神为旅行者打造古名。之后，深渊的本体「古斯托特」向纳塔发起总攻，期间恰斯卡受到大灵承认成为最后觉醒的六英杰。玛薇卡与六英杰动用若娜瓦的力量暂时击败古斯托特，暂时保下纳塔，并将「虚假之天」打出裂缝。之后玛薇卡打算与旅行者一同直面古斯托特，而队长也请求夜神的协助，为了与死之执政最后一战。[61]
战胜深渊旅行者的古名「希望」打造完成，玛薇卡与旅行者前往夜神之国直面古斯托特。古斯托特化形成火龙王修库特尔的模样与旅行者和玛薇卡决斗，期间旅行者激活的火元素属性，最终在众多的灵魂和在地面上的纳塔人与夜神和仙灵们的帮助下战胜了古斯托特。之后，玛薇卡必须向死之执政履行契约支付自己的生命作为使用死之执政力量的代价。夜神在对抗深渊的漫长过程中生命已走到尽头，她打算把生命支付给夜神，让夜神重新获得生命力，但自己也会牺牲。不过玛薇卡提供的生命力也有极限，夜神迟早会死亡，但能帮夜神之国撑一段时间，到时候再想办法。但队长代替了玛薇卡，如愿与夜神融为一体，使夜神得以不死，并改变夜神之国的规则，使其可容纳纳塔人以外人的灵魂，使自己在坎瑞亚的众多部下的灵魂拥有归宿，并以此作为向对他降下不死诅咒的死之执政的复仇。拥有不死诅咒的队长支付的生命使夜神不死。[62]
