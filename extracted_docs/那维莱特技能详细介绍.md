# 那维莱特技能详细介绍

那维莱特是《原神》中的水元素五星角色，枫丹的审判官，称号"水之龙王"。
元素战技「水之审判」：
- 释放水元素能量，对周围敌人造成水元素伤害并施加"水元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「水之领域」：
- 创造一个持续12秒的水元素领域，领域内的敌人会受到持续水元素伤害。
- 冷却时间：18秒。
- 能量消耗：60点。
固有天赋：
- "水之祝福"：水之审判命中敌人后，有15%概率触发"冻结"效果。
- "水元素增幅"：那维莱特的普通攻击和重击造成暴击时，恢复3点元素能量。
- "水之威严"：元素爆发持续时间内，那维莱特的水元素伤害提升20%。
