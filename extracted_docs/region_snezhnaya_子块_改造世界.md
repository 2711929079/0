# 改造世界

衔枝后四十余年，七位元素龙王被「高天的主宰」击败，七个王国全部对天上俯首称臣。衔枝的四百余年内[5]，「高天的主宰」开始了对世界彻头彻尾的改造。箱舟停驻，创造之事已毕，山川与河流相继落成，地底下、沧海里容纳了旧世界落败的族裔与王酋。蛋壳中的世界之主用虚假的帷幕隔绝真实的魔天，遮蔽了一度使统辖兽群的诸王也无法安眠的恐怖。暴烈的日光清扫了天空，但新的王座却仍为月夜的三姐妹预留了天上的位置，本质上三月女神维系看这颗星球的存续与运转，与天理的目的并无冲突[6]，或许因为在它的规划中，这个连元素的奔流也井然有序的世界依然需要潮汐。三月女神明白，仅凭她们无法从必将到来的堙灭中守护这颗孤旷的行星。在最后的寂灭到来前，无论谁在善恶的斗争中取胜，总好过同归虚无的永劫。也许她们身陷迷雾的旧主永远也无法找到解答，又或许那个答案已近在眼前。正如她们所期望的，高天的主宰厌弃黑色迷雾中的毁灭者。想必在穿越暗域的漫长旅途中，它已目睹无数陨灭的星辰。于是她们也选择了「秩序」，或者更准确地说，容纳了它。那是三月与晨星共治的时代，那是永恒依旧为永恒的时代。[2]
人类诞生天理到来四百余年后，人类受造诞生。[5]
