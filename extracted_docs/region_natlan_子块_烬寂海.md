# 烬寂海

其污染具有时间性，会侵蚀时间的每个角落。为了防止其继续侵蚀世界，巴巴托斯将其从时间轴上吹出去。没有一丝风的无边原野[1][2][3]，提瓦特四大险地之一，原位于纳塔最西边远离各部族的地界。为了找到它的位置，深渊教团曾派渊上偷看地图。坎瑞亚灾变后，烬寂海成为不毛之地。[4]传说约二三十年前，烬寂海出现巨物异动，随后整片区域消失不见。烬寂海消失后，才开始出现人烟。[4]该地存在「魔物」[5]，以及渡火者系列圣遗物故事中提及的该地存在的「凤凰」[6]。烬寂海是时空混乱的地方。存在名为「巴窟纳瓦」的可怖魔物，流浪乐团的笛手与当地人曾一同与之交战，取下牠身上一块蕴含生命力的符石[7]。目前只开放烬寂海的一角。
