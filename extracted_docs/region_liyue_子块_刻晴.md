# 刻晴

几年前，出身名门望族的刻晴在祖父过世后年纪轻轻便成为了璃月七星中主管土地建设的玉衡星[80]。
行秋飞云商会的二少爷行秋向往武学，在几年前拜师于衰落已久的古华派门下，并且仅用四年就自学领悟了古华派的三大武学奥义，被视为是古华派振兴的希望[137]。
