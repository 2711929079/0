# 早期

在很久、很久以前，曾经有一位守护鸣神岛的「狐斋宫」。传说中，她是雷神（影）大人麾下的一匹白狐，自称「斋宫」，据说会化成人形出现，也会倾听人们的愿望。 狐斋宫是白辰血脉的女儿，曾与雷神（影）同行，守护稻妻的子民，过去在一些地方贵为祭神的仙狐。
狐斋宫向来偏爱短歌之美，知一千一百一十一首，也有自如应用的风雅情趣。
在狐斋宫的一生中，她结交了众多的友人。她曾与鬼族的千代一同在御前献艺倾奇与神乐，曾与天狗的族长竞足，曾与妖狸大王斗法。她也曾化身人形，与诸多人类结为挚友。
