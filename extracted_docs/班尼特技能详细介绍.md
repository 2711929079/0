# 班尼特技能详细介绍

班尼特是《原神》中的火元素四星角色，蒙德冒险家协会的热血少年，称号"命运试金石"。

元素战技「热情过载」：
- 点按：挥出火焰剑，造成火元素伤害，冷却时间5秒。
- 长按：蓄力攻击，根据蓄力阶段造成不同程度的火元素伤害。
- 一段蓄力：冷却时间7.5秒，造成中等范围伤害。
- 二段蓄力：冷却时间10秒，造成大范围伤害并击飞敌人。
- 能量消耗：无。

元素爆发「美妙旅程」：
- 效果：创造一个领域，领域内的角色获得治疗和攻击力加成。
- 治疗效果：基于班尼特的生命值上限。
- 攻击力加成：基于班尼特的基础攻击力。
- 领域持续时间：12秒。
- 冷却时间：15秒。
- 能量需求：60点。

天赋技能：
1. 热情复燃：元素战技命中敌人时，有20%概率重置冷却时间。
2. 信念之焰：元素爆发领域内的角色获得额外的元素伤害加成。

角色定位：
- 主要定位：辅助
- 次要定位：副C
- 核心优势：群体治疗、攻击力加成、元素伤害加成

培养建议：
- 武器推荐：西风剑、祭礼剑、腐殖之剑
- 圣遗物推荐：昔日宗室之仪四件套（提升大招效果）
