# No.7 「木偶」桑多涅Sandrone

中文CV：洪海天 日文CV：本多真梨子阿蕾奇诺评价她热衷机械研究，很少露面。在「问题美露莘与答案机器人」中，对枫丹科学院已故研究员莫索博士的研究成果感兴趣。是「奇械公」阿兰·吉约丹仿照其妹妹玛丽安创造的人偶，在其65岁生日时完成，余生中的阿兰逐渐不再相信一切人的智性所不能完全掌握、不能彻底理解的东西，他构想出一个由机关与非元素能源所驱动的王国。最后他意识到水仙十字院终要有个人以正常人的身份活着，于是放下手头的研究，拿起童话书念给他的造物同时是他的好友的桑多涅，以普通人而非天才的样子离世。制造了凯瑟琳，身边的大机器人叫「普隆尼亚」，在对付猎月人时普隆尼亚疑似使用岩元素，性格傲娇。部下维李米尔（同时也是「木偶」的造物之一）企图将伊涅芙诱捕以研究古龙造物，被挫败。热衷机械研究，喜欢白天睡觉晚上工作。得知旅行者抵达挪德卡莱后，亲自在那夏镇的冒险家协会等候并亮出席位是第七席。驻扎在挪德卡莱月矩力试验设计局内，开展月矩力的相关研究，从霜月之子出逃的薇尔米娜在旅行者与雷利尔变成的血亲进入圣所后，趁乱偷走“恒月月髓”并交给木偶。在雅珂达等人秘密潜入设计局并意外打开通往下层的道路后，赶回设计局，正好遇到三人，表示对月髓没有兴趣后，随意地将其还给了菈乌玛。为了对抗「猎月人」雷利尔，给普隆尼亚进行改装，旅行者和菲林斯在星砂滩与雷利尔陷入苦战时，木偶带着同样得到月矩力强化的普隆尼亚来到现场，普隆尼亚与雷利尔大战，但失败，在千钧一发之时，哥伦比娅出手，将几人传送到了「银月之庭」。旅行者和哥伦比娅为了找阿蕾奇诺帮忙，前往设计局，向「木偶」询问阿蕾奇诺的踪迹，而后雷利尔追杀至此，「木偶」让二人离开，自己用设计局内的防卫力量抵挡雷利尔，但又一次失败了。在流浪者提及「博士」坑害同僚的事迹后立刻陷入共鸣，疯狂吐槽，并表示可以一起骂他的人没几个。在月之四主线中和阿蕾奇诺一同与「博士」决裂，与旅行者等人一同对抗他。她通过水仙十字社的资料，制定出了阻止「博士」的术式，其和普隆尼亚被「博士」杀死，核心被阿蕾奇诺送回枫丹科学院（哥伦比娅想用权能复活，但没有成功），大部分遗物则由阿蕾奇诺保管，但阿蕾奇诺将其笔记给了旅行者。其笔记表明其有重生的可能，官方发布的预告立绘中也有一个隐藏起来的第6立绘，暗示其仍然有回归并在本版本加入卡池的可能（水仙十字题解：何物徒留名字？何物遍开幽谷？何物映自身于镜水？何物象征拯救世界的孤独牺牲？）。
执行官互评折叠「公子」：好像是个埋头研究的人？我想想，那些机械是不是都跟她有关啊。总之，仅有几次的见面中，她一副想杀了我的眼神。可我完全不记得做过什么惹她生气的事。「散兵」：她除了自己的研究以外什么都不关心，性格恶劣得很。但即便做到如此极端的地步，却也只是制造了大量废品，在我仍是执行官时还低我一席，真是可怜。「仆人」：我对她没什么兴趣。她热衷于研究，很少公开露面，许多士官甚至都没有见过她。相较于另一个把自己切成很多份、满世界乱跑的人，她算是另一个极端。「少女」其一：她背后的发条会一直转吗？是永远朝同一个方向转吗？睡觉的时候，她会把发条拆下来吗？翻身要怎么办？刚见到桑多涅的时候，我就对她的发条产生了好奇。但那时和她还不熟，直接去碰似乎也不太好。所以我只要有机会就一直跟在她身后，想找到她发条转动的规律。但有一天，她突然转过身，看起来很生气地质问我为什么总要跟着她。我老老实实告诉她之后，她对我说，「你那么好奇的话，我给你背后安一个试试得了」。于是我就背着发条生活了几天，唔…的确很辛苦，桑多涅的生活真不容易。自那之后，我就更关心她了。「少女」其二：我不清楚对人偶来说，怎么样才算是「死亡」…但桑多涅真的不动了，我感受不到她的「运转」，她的发条也不动了。她是为了那个「术式」，为了让我回到这里，才变成了这样…我多么希望她这么做只是为了吓我一次，就像她曾经说过会报复我的那样…
