# 风之翼

侦查骑士靠它在天空中行动，蒙德的市民们也很喜欢用。只有拿到飞行执照，才能在蒙德城合法飞行。飞行执照先要考试合格，然后再由骑士团统一签发。使用风之翼飞行时需要遵守规章制度和飞行标准，即《飞行指南》。在风之翼测试中，实验对象的选择包括但不限于野猪、丘丘人等。
