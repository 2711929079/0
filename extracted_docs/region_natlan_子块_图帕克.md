# 图帕克

来自流泉之众的图帕克认力自己生不逢时，活在了平庸的时代，于是决定前往未知的外海。一路上，图帕克击败了许多巨兽，他将翼龙的双翅制成精帆，将沧鲸的骨头制成白身，将海蟒长牙制成回旋镖「乘浪的回旋」，将巨鱿做成下酒菜，最终他成功抵达了海的边界[110]。
万杰鲁天生就具有奇力的万杰鲁出身于沃陆之邦，一直渴望能拥有一把趁手的武器，但是普通的材料都无法承受她的力量，直到她听说了龙族昔日的圣城图兰中仍有龙族残存。万杰鲁孤身一人前往了图兰，与一只龙形的金影战斗，最终万杰鲁徒手打败它，并获行了一段龙骨。万杰鲁后来用这段龙肯打造出了一把长剑「弥坚骨」[111]。沃陆之邦有一位沉默寡言的少女名叫“菲耶希娜”，经常瑟缩在篝火旁，全神贯注地聆听万杰鲁讲迟英雄传说[112]。
