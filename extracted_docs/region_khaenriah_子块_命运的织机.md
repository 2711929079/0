# 命运的织机

为了执行命运的织机计划，深渊教团将璃月某处秘境作为据点，将污秽逆位神像倒挂，并派出大量教团成员人畜无害的深渊法师搜寻“原型机之眼”（即「世界上第一颗耕地机的眼睛」），想将原型机的眼睛置于神像的掌中，并透过它操纵魔神奥赛尔的躯体打造机械魔神。剧情里派蒙称其为终极杀人机器，机械魔神乌贼剧情里戴因对此的评价是“天马行空”
此外，教团精锐成员深渊使徒也参与此次行动，阻扰破坏计划的碍事之人，并对北风之狼施行腐殖仪式，想要他为深渊教团效力。
在旅行者和一直追猎深渊教团的戴因斯雷布的阻扰下，腐植仪式被打断，原型机之眼被戴因带走，寻找原型机之眼的深渊法师全被击败。在执行计划的深渊使徒要被逮住时，教团的殿下出手掩护他。
在与旅行者短暂交谈后，殿下透过传送门离开，将污秽的逆位神像抛下。「我们终将重逢」「但不用急,我有足够的时间来等你」「我们的时间从来都是足够的」[9]
复国的第一步为了复国，深渊教团的殿下在犹豫很久后答应深渊使徒，打算强行解除人民身上的诅咒，但解除诅咒的过程会让被诅咒的人民非常痛苦，而且成功的机率微乎其微。殿下认为，就算人民回归永恒的轮回之中，也不要不尊严地死去，并表示即使只有千分之一成功的机率，计划也要执行。然而此时戴因被深渊使徒的传送门阴差阳错地传送到了层岩巨渊深处。旅行者也因为接受委托来到层岩巨渊。此时深渊教团占领了层岩巨渊深处的建筑并打算启动净化装置，强行解除汇聚于此的坎瑞亚遗民（丘丘人）的诅咒并让他们在痛苦中死去，最终戴因的前部下——已经化为黑蛇骑士的哈夫丹阻止了这荒唐的仪式。事件解决后，戴因再度与旅行者辞别。[4]
