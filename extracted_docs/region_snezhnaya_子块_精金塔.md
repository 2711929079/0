# 精金塔

亥珀波瑞亚人曾在拂晓之星的引导下，筑起直抵天穹的精金高塔[10][11]。
霜夜灵嗣传说这些高贵的生灵受造于最初的神使，受霜月祝福而诞生，是黄金之城的侍仆。辉映于其瞳中的乃是至圣至耀的天光，在霜月之子最古老的祷歌中被称作神使无垢的垂怜，象征着纯然的源质与肇始的辉光。乐园陨落之时，它们曾与霜月的女主人立约，誓要以得赐于霜月的源光，涤净尘世的污秽，直至旧日的恩主再度归来。如今那缕残光早已熄灭，而失却了旧主的灵嗣依然漫行于月下的霜夜，守望无人铭记的誓言。霜夜灵嗣具备极强的元素亲和力，或许正是因为如此，它们才会因无法忍受自身的纯白被染上天光的七色，而开始拒斥一切过于接近它们的生灵。传说霜夜巡天灵主因着神使的偏宠而不会为天光所侵染，时至今日，即便是在霜月的追随者中，它们亦被视作已不存此世的圣兽，沿着月光的秘径隐遁到了世界之外。但正如过去的数十个百年一般，它们依然漫行于月光的影子中，漫行于两个世界的边际，巡护这被灾厄觊觎的土地，与生活在其上的人们——因为这便是它们最初向黄金城的圣徒立定的誓言。[12]。
