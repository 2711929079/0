# 腐植仪式

为了“让深渊殿下的国度重临于世”，深渊教团透过腐植仪式诱惑特瓦林，使其以蒙德为敌，成为深渊教团的兵器，并且拉拢丘丘人进犯蒙德。在旅行者的介入下，深渊教团的计划挫败。公主/王子殿下目睹旅行者与教团安插的魔物对抗后离开，并且在之后整个教团都刻意避开旅行者的行动路线。但是他们没料到整个提瓦特都是旅行者的行动路线，包括本该已经被封印的层岩巨渊底[80]
命运的织机为了执行命运的织机计划，深渊教团将璃月某处秘境作为据点，将污秽逆位神像倒挂，并派出大量教团成员人畜无害的深渊法师搜寻“原型机之眼”（即「世界上第一颗耕地机的眼睛」），想将原型机的眼睛置于神像的掌中，并透过它操纵魔神奥赛尔的躯体打造机械魔神。剧情里派蒙称其为终极杀人机器，机械魔神乌贼剧情里戴因对此的评价是“天马行空”此外，教团精锐成员深渊使徒也参与此次行动，阻扰破坏计划的碍事之人，并对北风之狼施行腐殖仪式，想要他为深渊教团效力。在旅行者和一直追猎深渊教团的戴因斯雷布的阻扰下，腐植仪式被打断，原型机之眼被戴因带走，寻找原型机之眼的深渊法师全被击败。在执行计划的深渊使徒要被逮住时，教团的殿下出手掩护他。在与旅行者短暂交谈后，殿下透过传送门离开，将污秽的逆位神像抛下。「我们终将重逢」「但不用急,我有足够的时间来等你」「我们的时间从来都是足够的」[9]
