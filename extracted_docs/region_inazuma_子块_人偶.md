# 人偶

在初代雷神雷电真死于坎瑞亚之战、二代雷神雷电影肃清稻妻的灾厄后，影决定以某种最初源于坎瑞亚的神秘技术制造对抗磨损的人偶。她透过白色的枝条，於借景之馆[60]制造了一名人偶试做品，然而无名的人偶却因容纳雷神之心在睡梦中流下眼泪。作为容纳神之心的容器,影认为他既没有人类的坚忍不拔，亦没有器物的寡淡无情，实在过于脆弱,与永恒的理念相违背。八重神子原先建议将人偶直接毁去[61]，但雷电影不忍毁弃自己的造物，于是将人偶的机芯「神之心」取出。原先把它放在沈眠之庭門口[62]，后来把他安置在据说是古时避世的武人在大地深处借世外之景，修建的华美宅邸借景之馆[63]，并为他留下一枚金色的羽状饰品作为身份凭证。在制造之后的人偶时，影舍去了安放「心」之处的设计。[31][46][64][65]人偶在得名倾奇者之前还有个名字。但这个名字没多少人知道，就连人偶自己也忘记了。[65][66]
人偶甦醒鹤观人阿釜的祖先曾误入借景之馆发现被封印力量的人偶[67]（因为此时人偶身上有关节痕迹所以看得出来是人偶[66]）。之后人偶打破身上的封印自行甦醒。[65]踏鞴砂曾经出现过一个人偶，它面貌秀丽，打扮得体，懂得掩藏身体上那些特殊的关节痕迹。若非偶尔有人提起，旁人很难看出他是个人偶。另外，人偶特有的关节痕迹似乎还会随着时间推移慢慢淡去，最终或许会消失不见，变成完全的人类模样。人偶的名字并无几人知道。只是听说有这样一个人偶曾出没于踏鞴砂地带，有人在核心地区，也有人在海滩上见过它。传说它会伫立在海边遥望一海之隔的稻妻城，那里有什么，它又在看什么，实在无从知晓。[66]之后人偶回到借景之馆，但借景之馆因不明原因被封印[63]，人偶再次被困在借景之馆内，直到桂木救下他。
