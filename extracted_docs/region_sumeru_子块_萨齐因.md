# 萨齐因

距今28年前，因论派学者萨齐因怀着理想来到沙漠，试图改变当地争乱不休的现状。但是在沙漠生活了八年后，萨齐因对世界感到了绝望，认为人性本恶，纷争是人类无法避免的命运。沉重的虚无感让萨齐因选择自尽。不过在这之前，萨齐因将自己的产业都交由教令院打理，要求教令院要用其中的三成收益作为学院争霸赛的奖金，以此吸引参赛者。而萨齐因又利用从前偶然得到的才识之冠来存入自我意识，之后的每一届学院争霸赛中他都会暗中观察，希望能寻找到有资格传承自己理念的人[91]。
