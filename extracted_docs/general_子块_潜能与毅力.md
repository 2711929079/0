# 潜能与毅力

班尼特在一次危险的冒险中，失血过多昏了过去。醒来的时候，发现了神之眼，且神之眼保护了自己。迪奥娜在一场大暴雨中，出门找到了失踪的父亲，并破例为父亲调了一杯正常配方的酒，获得了神之眼。雷泽在一个雷雨天，被深渊法师掳走。卢皮卡家人们（狼群）为营救雷泽，被全部击倒。雷泽为了保护狼群，奋力向深渊法师复仇。此时获得了神之眼。罗莎莉亚出生后不久，被屠村的盗贼捡走。在一个寒冷的夜里逃跑，被盗贼老人拦下。罗莎莉亚将这个捡走了她、教她杀人技法的老人杀掉，获得了神之眼，也获得了盗贼们的最终接纳与认可。米卡独自前往情报失效的地区绘图时，惊扰了两个深渊法师。在烈火之下，米卡仍然不忘职责与任务，获得了神之眼，从危险中逃离。北斗驾船与鱼龙在海上搏斗整整四天，第五天日出时刻成功斩杀鱼龙。此时获得神之眼。申鹤被父亲当成祭品，与魔物搏斗数日。生死攸关之时，获得了神之眼。辛焱为了练习摇滚，往音乐中加入了火花与爆炸，每天坚持练习。训练的时候，多次烫伤手指、炸飞乐器。练成之前的最后一刻，获得了神之眼。夜兰在一次任务后，闭关习武，日夜磨练，十指带血，获得神之眼。绮良良还是小猫的时候，到城市寻找早出未归的奶奶。绮良良自从被老奶奶收养，就经常听奶奶讲城市里的故事；而此刻城市夜晚发出的光，点燃了绮良良的心，“渴望栖身于那些光芒之中”。绮良良化成人形，获得神之眼，顺利找到了奶奶。神里绫华家中重大变故后，为了担负起家族责任、为兄长分忧，拾起荒废许久的剑术与诗歌。“千般锤磨，素振亦无人可当。”剑术训练成为了神里绫华日常生活的一环，从未间断，不知道多少天。终于，练成一击退敌的那一刻，获得了神之眼。早柚曾在任务中，被强敌环伺。为了安全带回情报，带着伤精疲力尽、马上要晕厥过去的早柚，心中想着：“…弱者也有弱者的生存之道。正因为弱小，才更能留意到那些微不可查的、改变命运的瞬间…”利用忍术，抓住瞬间顺利逃脱。第二天醒来发现神之眼。
