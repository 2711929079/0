# 森林王

「很多、很多个月亮以前，森林是一座巨大的迷宫…」在过去，凡有树荫的地方，都是森林王的疆域。而他在自己的领土中漫步时，仰赖丛林而生的飞禽与走兽会在他经过时低低地垂下头颅，以示尊敬与屈从。历代的王都有自己的宫殿：每位森林王继任后，森林就会依照他的形象与梦发生改变。最后的森林王的宫殿隐藏在许多树木与藤蔓构成的藩篱、水流与山崖画成的网格之后，那是月光透过树叶落在静水上，构成的珍珠的圆盘。王就在水边，独享着两倍的月光。而森林的迷宫就是这样来的。传说中，森林王令人目眩的斑纹中，描绘着迷宫的路径。每代王的宫殿交叠在一起，于是树与树之间的小径交错，溪流中断或是产生新的分支，林间的道路就变得越来越复杂。只有森林王、兰那罗，还有带着王树瑞佑的人们，能够自在地深入森林王的疆土，在树与树之间、溪流与溪流之间找到自己该走的路途。[32]森林王的权能是制造迷宫，每代森林王所制造的迷宫叠加在一起，让整个雨林变成了一个复杂无比的大迷宫。只有草神，兰那罗以及持有王树瑞佑的人类才能穿越其中。
迷宫森林王为了守护法留纳神机的秘密，用梦境、树篱和支流建造复杂的迷宫，森林迷宫的路途会将迷误之人带往梦里。但在数百年前，这个迷宫便不复存在了。[36][37]
