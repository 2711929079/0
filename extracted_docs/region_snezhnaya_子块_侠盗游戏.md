# 侠盗游戏

双亲亡故后，艾维雷勒安被一位名为米赫年科的底层军官收养，并得以进入至冬皇家军事学院就读。就读于军事学院时，艾维雷勒安常和军事学院的同学（索洛维）扮作侠盗劫富济贫，并仿照挚友家徽夜莺的形貌，共同刻下了最初的鸦印[44][40]。趁着军校组织来到挪德卡莱演练，艾维雷勒安和索洛维软硬兼施，从5岁的布莱赫处套话，将挪德卡莱总督藏匿在「金银岛」的私房钱偷走，送给了海螺帮的孩子们，并留下「最初的铁钱」作为得手的宣告（此地后续见#「伟大财宝」）。但他们不知道的是，不出三天这些钱就被总督加倍搜刮回来了[40]。彼时的「乐园」还不过是两个孩童在月下的空想，许多年以后，其中一人成为闻名诸国的大盗（详见至冬#「大盗」雷德·米勒），一人成为了最初的铸灯者（详见至冬#「铸灯者」索洛维）[45]。
建立盗宝团在官方记录里，艾维雷勒安毕业成为士官不久后，便被记录为「因故殉职」；据爱莉厄推测，是霍德望与阿尔维斯大公共同设下圈套，使艾维雷勒安丢掉军官的身份而成为大盗[34]。几年后，他用假名「雷德·米勒」出现在挪德卡莱，聚集起游离在社会秩序外的贫民——这就是最初的「盗宝团」。盗宝团的人称他为头领，相信这个不知从何而来的年轻人能为他们击碎枷锁，为所有饥寒之人筑起平等的乐园[40]。
