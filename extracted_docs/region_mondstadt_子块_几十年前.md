# 几十年前

斯坦利两位蒙德冒险家：汉斯亚齐博尔德与斯坦利来到了烬寂海探险。斯坦利在烬寂海时，左手和腰部被连同类都会吞食的魔物啃掉，在渊上帮助下暂时止血。原本渊上化名诺奇特，在救活斯坦利后说自己赶时间要离开了，但傲娇的他其实还是跟在斯坦利后头之后斯坦利遭到危险时，然后再换个长相并用假名「娜娜丝」再救了他[121]。斯坦利为救汉斯而死，如愿在烬寂海探索到最后一刻，汉斯愧对于友人，想要让人们一直记住他的事迹，于是回到蒙德后一直冒用斯坦利的名字来宣传探险事迹[122]。卡莉露与芬奇纯水精灵卡莉露与少年时期的芬奇在清泉镇相遏，彼时哭注的芬奇向卡莉露吐露真心，而卡莉露则陪在他身边聆听，久而久之，芬奇对卡莉露有了思慕之情，那时的卡莉露深知作为纯水精灵的自己与人类的命运不问，因为无法回应芬奇的心意而逃走了[116]。罗莎莉亚出生不久的罗莎莉亚所居住的小山村被一伙盗贼洗劫，罗莎莉亚也被盗贼劫走，成为奴隶。在与盗贼团的生活中，罗莎莉亚学习到了战斗技巧。在一个饥寒交迫的夜晚，罗莎莉亚不堪雪负选择逃走，盗贼团中的一位老人将她追回，并让她终结自己的生命，用这样的办法，罗莎莉亚获得了神之眼并被盗贼团接纳[123]。
