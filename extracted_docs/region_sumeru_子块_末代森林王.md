# 末代森林王

坎瑞亚灾变中，末代森林王为了保护森林死去了，森林不再构成新的宫殿，迷宫从此也消失了。或者是漆黑的群兽如同飓风让树木如同麦浪一样倒下。迷宫死去了，森林王也死去了。[32]森林王曾经的侍者，血缘与猫更近的长须虎在之后继承了虎的名号。[70]
桓那兰那桓那兰那被毁。三位古老的兰那罗封印了无留陀的化身，但封印逐渐松动。兰帝裟用石锁封印众多坎瑞亚的战争机械。此后，兰那罗们在兰拉迦的梦中建立了新的桓那兰那，鲜少外出。长老兰拉迦成为了维持梦境「桓那兰那」的「梦之树」。[30]
