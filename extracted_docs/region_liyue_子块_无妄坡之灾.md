# 无妄坡之灾

无妄坡曾有村子。十多年前在无妄坡，边界异动，胡桃的父亲用护摩秘法焚烧死气，却不慎沾染太多，当场把自己送走。大半个村子的人也没能幸免，山洪暴发，该地天气变得诡异，活下来的人也搬走了。无妄坡就变成了现在这个样子。后来，是当时也在场的胡桃的爷爷替他收了尾。他当时吸收太多死气收到影响，驾鹤西归时，也还未过花甲…失败原因是历代堂主在离开时，都会把自己的物品带走，最多留下一两件，其他全烧干净。但胡桃老爹留了一大堆东西，胡桃埋怨他失败的原因，就是舍不得。就是因为他舍不得变老的父親，还小的女兒，舍不得大家都在的世界。所以他在最后关头才会犹豫，才会畏惧，才会心神不稳，输给了死亡，连累那么多人[163]。胡桃的爷爷生前为胡桃留了一项“乾坤泰卦帽”。往生堂依遗嘱为这位第七十五代堂主举办了葬礼，并由13岁的胡桃主理[164][165][166]。
