# 荆夫港

蒙德的海港[4][5][6]，或许是果酒湖的出口，或许能通往蒙德城外小港口，位于蒙德北方，蒲公英海的北方[7]。常常需要游击队前往护卫[8]。亨利莫顿提及可以雇船。
