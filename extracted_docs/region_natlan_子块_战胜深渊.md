# 战胜深渊

旅行者的古名「希望」打造完成，玛薇卡与旅行者前往夜神之国直面古斯托特。古斯托特化形成火龙王修库特尔的模样与旅行者和玛薇卡决斗，期间旅行者激活的火元素属性，最终在众多的灵魂和在地面上的纳塔人与夜神和仙灵们的帮助下战胜了古斯托特。之后，玛薇卡必须向死之执政履行契约支付自己的生命作为使用死之执政力量的代价。夜神在对抗深渊的漫长过程中生命已走到尽头，她打算把生命支付给夜神，让夜神重新获得生命力，但自己也会牺牲。不过玛薇卡提供的生命力也有极限，夜神迟早会死亡，但能帮夜神之国撑一段时间，到时候再想办法。但队长代替了玛薇卡，如愿与夜神融为一体，使夜神得以不死，并改变夜神之国的规则，使其可容纳纳塔人以外人的灵魂，使自己在坎瑞亚的众多部下的灵魂拥有归宿，并以此作为向对他降下不死诅咒的死之执政的复仇。拥有不死诅咒的队长支付的生命使夜神不死。[134]之后在确认无害后，夜神以「某个条件」为代价同意协助旅行者血亲制作坎瑞亚地脉的蓝图[135]。在与深渊的战役结束后，巴巴托斯与玛薇卡合作救回烬寂海游荡的逝者灵魂[125]。
山王回火夜在战役中，布基纳被因深渊侵蚀而发狂的伙伴山王攻击，在死前将自己的力量注入给山王。山王在此后一直希望了结自己以赎罪。在人们帮助下，「山王」决定带着罪孽坚强的活下去，因为了结自己很简单，但背负痛苦活下去却很困难，背负着过往活下去成为了山王的赎罪。[18]
