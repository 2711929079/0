# 其二

…真实需要时间垒就，谎言却可一夕而得。
若谗谤之言滋蔓，信任的根基终将消蚀；
再坚实的盟誓，于猜忌之中也不可维系。
利用贵胄世家的倨傲，让嫌隙蔓延丛生；
便使包藏最深的那颗祸心有了可乘之机。
此即为，三宗大罪其二。…以石柱为刻，咒文为缄；
将「众噩之蝰」的罪行永录于灯影之中。
