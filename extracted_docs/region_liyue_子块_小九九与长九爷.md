# 小九九与长九爷

长九爷幼年被叫九哥，与同样家中排名第九的小九九是小情侣，长九爷为了小九九写了本小说，在小九九有天边闲逛边看那本小说时，掉到水里不幸溺水，他知道这本书对九哥而言非常重要，于是紧紧抱住书直到溺毙，小情侣就此天人永别。因为家中这个变故导致小九九家举家搬迁，这是为了不让长九爷有天发现小九九死讯，而不知情的长九爷一直认为是他们家看不起自己，觉得自己没家世是穷小鬼，为了不要让小九九不与自己接触才搬家的，非常不服气，打算如当初所说让自己小说大卖，包括当初写给小九九的那本来证明自己。清策庄的老人直到如今仍在后悔自责自己没能救下小九九。后来小九九因为没保护好书的执念变成幽灵游荡在无妄坡，但只找到几页残页[154]。
数十年前的其他事件刻晴的祖父是璃月著名的土地业大亨，也是一位学者，酷爱搜集古籍，曾与夜兰的家族有过学术合作。夜兰的祖辈曾为七星效力[155]。三十年前，一位名叫 “若心”的戏子在“和裕茶馆”表演，成为一代名角。若心曾经常与现如今的 “范二爷”搭戏，而且总是在台上扮演一对，让范二爷对若心心生情愫，但若心倾心于一位冒险家[156][157][158]。
