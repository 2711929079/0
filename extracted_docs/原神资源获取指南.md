# 原神资源获取指南

《原神》中有多种资源需要获取，以下是主要资源的获取方式：

1. 原石：
   - 祈愿抽卡的主要货币
   - 获取途径：
     * 任务奖励：主线任务、支线任务、世界任务
     * 成就奖励：完成各种成就
     * 深境螺旋：首次通关和月度重置奖励
     * 活动奖励：限时活动和版本活动
     * 每日委托：完成4个每日任务获得60原石
     * 探索奖励：开启宝箱、找到神瞳

2. 摩拉：
   - 游戏内通用货币
   - 获取途径：
     * 任务奖励
     * 秘境挑战
     * 世界BOSS
     * 每日委托
     * 出售多余物品

3. 经验书：
   - 角色升级材料
   - 获取途径：
     * 秘境：启示之花
     * 活动奖励
     * 每日委托

4. 角色突破材料：
   - 元素材料：通过对应元素BOSS获取
   - 天赋材料：通过天赋秘境获取
   - 周本材料：通过周常BOSS获取

5. 武器突破材料：
   - 武器原胚：通过锻造获得
   - 武器材料：通过武器秘境获取

6. 圣遗物：
   - 获取途径：
     * 圣遗物秘境
     * 世界BOSS
     * 活动奖励

7. 培养素材：
   - 怪物掉落：击败各种敌人获取
   - 采集：在地图上采集植物和矿物
   - 钓鱼：通过钓鱼获取特殊材料

资源规划建议：
- 优先完成每日委托，稳定获取原石和摩拉
- 合理安排体力，优先获取当前培养角色所需的材料
- 定期清理深境螺旋，获取大量原石
- 积极参与限时活动，获取稀有奖励
