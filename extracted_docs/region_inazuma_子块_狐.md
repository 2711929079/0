# 狐

狡黠而高傲的生灵，在民话传说中拥有奇特的智慧与悠远的记忆。即使有生人靠近也不屑一顾的样子，似乎是已经习惯了与人类相处。在稻妻文化中有着悠久的历史渊源。有传它们是过去的「狐斋宫大人」的亲眷中的一支。在她消失后，力量更强的「天狐」与「地狐」都化作石像等待她的回归，而这些狐狸则随着一族的灵脉变得稀薄，而不再开口，也失去了法力。
