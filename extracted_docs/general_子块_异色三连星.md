# 异色三连星

游戏中包括异色三连星·西多连科（冰铳重卫士）、异色三连星·诺莫科诺夫（雷锤前锋军）和异色三连星·瓦西里（火铳游击兵）三人。三人的现实原型分别是：伊万·米哈伊洛维奇·西多连科（Иван МихайловичСидоренко，1919-1994），苏联狙击手，苏联英雄。谢苗·丹尼洛维奇·诺莫科诺夫（Семён ДаниловичНомоконов，1900-1973），苏联狙击手。瓦西里·格里戈里耶维奇·扎伊采夫（ВасилийГригорьевич Зайцев，1915-1991），苏联狙击手，苏联英雄。
