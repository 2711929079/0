# 珊瑚宫心海技能详细介绍

珊瑚宫心海是《原神》中的水元素五星角色，稻妻海祇岛的领袖，称号"沧海的惠佑"。
元素战技「海月之誓」：
- 释放水元素能量，对周围敌人造成水元素伤害并恢复队友生命值。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「海祇之心」：
- 创造一个持续12秒的水元素领域，领域内的队友会持续恢复生命值。
- 冷却时间：18秒。
- 能量消耗：60点。
固有天赋：
- "海月的祝福"：海月之誓命中敌人后，有15%概率触发"冻结"效果。
- "水元素增幅"：珊瑚宫心海的普通攻击和重击造成暴击时，恢复2点元素能量。
- "海祇的守护"：元素爆发持续时间内，队伍中的角色获得"水元素伤害提升10%"的加成。
