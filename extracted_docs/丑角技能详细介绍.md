# 丑角技能详细介绍

丑角是《原神》中的风元素五星角色，愚人众执行官第一位，称号"丑角"。
元素战技「风之戏法」：
- 释放风元素能量，对周围敌人造成风元素伤害并击飞。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「风之领域」：
- 创造一个持续8秒的风元素领域，领域内的敌人会受到持续风元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "风之祝福"：风之戏法命中敌人后，有15%概率触发"扩散"效果。
- "风元素增幅"：丑角的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "风之威严"：元素爆发持续时间内，丑角的风元素伤害提升15%。
