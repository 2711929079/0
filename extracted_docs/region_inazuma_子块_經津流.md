# 經津流

经津第三代─经津实锻造了带着蓝海波光的美刀─波穗月白经津，波穗月白经津曾为御舆千代所配带之刀，但在五百年前与将军的真剑对决中断刃，虽送回重锻，却并没有锻成，最终这把「波穗」成了历史。而经津实的儿子─也就是经津第四代经津弘芳却没有其母的技艺。但他的义兄，经津实的养子─政芳却不同，他从养母手中获得「真砂丸」之名，与经津之艺。原本实打算让真砂丸继承经津流，但真砂丸都婉拒，随后实离世，留下真砂丸与其义弟弘芳。真砂丸因领命重铸「波穗」，最后造出了唯一有自己铭名的「波乱月白经津」，因其剑与「波穗」相似，却又有完全不同的气质，真砂丸声名大噪。但这严重影响了经津流真正的传人─其义弟弘芳的地位。随后真砂丸传授完经津流给弘芳后，随即离开了此地，四处流浪，并钻研各种锻刀流，晚年出了三位得意弟子─枫原景光、丹羽长光、赤目实长，也就是后来的「一心三传」。[11]
一心流由经津流发展出的一心流，三位弟子枫原景光、丹羽长光、赤目实长，分别又发展出了不同的三个派别。一心流所追求的事人剑一体，因此锻造出的刀倾注锻刀者的心境，颇具个性。向来都是人选刀，但一心流乃刀选人。枫原一族博闻强识，作品最具真砂丸风骨。丹羽一族宅心仁厚，以烧刃见长。(烧刃指的是在日本刀上可以看到的类似波浪花纹的东西)赤目一族认为「一心」即为无冗余念想，为纯粹的目的锻刀，因此锻刀时加入祟神之活念，便能锻造出活刀（踏鞴砂事件中就是当时赤目一家将埃舍尔使用祟神之力锻刀的技术引荐给丹羽）。而刀乃杀生之物，必见血光，因此追求「斩人刀」的终极。但也因为如此，赤目一族佩刀虽极其锋利，却因长年于祟神之念共处，身心皆伤，寿命皆短。并被官家评为「不良」，而未继续成为「一心流」的总长。初代目惣领为赤目实长，但因赤目一流锻刀过于凶煞，为官家所恶，是故其惣领一职仅传三代即遭剥夺。丹羽长光的后人丹羽久秀与赤目实长的后人在踏鞴砂锻刀，其中丹羽久秀还成为了造兵司正[46]。
