# 吉光虫

会摄取层岩巨渊地下奇石的能量，发出微光的无害生物。其发光的特性备受矿工宠爱，被视为吉兆，得名「吉光」。因为黑暗当中的细小光明总是受到珍惜的。在层岩巨渊以外掘出「怪东西」之后，它们的族群也繁茂起来。
