# 草神迭代

因为世界树的机制，大慈树王无法将自己从世界树中删除，她在逝世前折下世界树最纯净的枝杈（即小吉祥草王），作为她的轮回转世。她逝世前留下「让世界彻底遗忘我」的线索在世界树中播放，让发现线索的小吉祥草王能来到世界树将其中有关她的资料彻底清除，以杜绝深渊污染，但同时她也会被世界遗忘。树王逝世后，须弥学术界看似兴盛，却一直没能出现较大的突破。[55]因为树王之死，本带在树王身边的乌婕特之眼佚失。[19]贤者找到了新生的小吉祥草王，将其接回了须弥城。[68]
薇瑞迭瑟恩猎手薇瑞迭瑟恩是森林王收留的孩子之一，如同在晨光下的苍翠草甸与林木间穿行的野兽一般纯净，从不带有一丝恶意，也从未为生存以外的目的放出箭簇。猎人是在远离城市的地方，在森林王这样的训导中长大的…「吾等诞生在绿茵中。只要有青草与树荫，便通行无阻」「吾等与鸟兽无异。只要遵循天地的法理，便无虞生死」「遵循自然的万物，在生命尽头都会去往无边际的原野」遵循着这个信念，猎手不留下足迹，不与天地自然为敌，安抚着被箭矢穿透心脏的野兽，直到它的生命回归自然。灾厄发生，纯白骑士罗南向着漆黑前进，盲眼少年也跟着兄长[31]，不幸被魔物所伤，已是奄奄一息的状态。那天猎手循着沾染在地面的污血前行，来到自己休憩的树下，借着白枝微弱的光遇到那改变自己想法的盲眼少年。她放箭射杀了魔物，听了关于少年的请求，心中已然不再是曾经遵循自然的理念，复仇的情绪充满了猎人的内心，如野兽一般猎杀魔物。如果灾厄没有发生,如果她没有循着草叶上的污血前行，如果没有在常常小憩的树下遇到那奄奄一息的盲眼少年，那她就不会为复仇的愿望、话语中鲜活的血与火光动摇…「千万不要忘记，善良的薇瑞迭瑟恩」「千万不要忘记呀，你是属于绿茵的、树林的孩子」「不能为了争斗、为了仇恨、为了所谓荣誉而射箭」「染血者永远也不能找到，彼方无边际的苍翠猎场」「那至少，不要让这把弓被即将到来的，仇恨的厮杀玷污」「既然我（薇瑞迭瑟恩）无法抵达彼方，和师父（森林王），和从未谋面的血亲（其是孤儿流浪的林内被收留从没见过父母）相见」「至少让这把弓保持纯洁，让它能够传达我的思念和歉意」在盲眼少年的请求下，薇瑞迭瑟恩接下来自少年的弓，开始讨伐肆虐的漆黑魔物，为盲眼少年复仇。后来，她在厮杀中被黑血染身，被路过的罗南（染血骑士）误认为魔物斩杀。[69]
