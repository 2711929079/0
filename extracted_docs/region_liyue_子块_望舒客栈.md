# 望舒客栈

端坐在荻花洲南端的客栈，似乎也承担着客栈以外的使命。据说望舒客栈的楼阁是恋人共约赏月的胜地。坊间有传说道，纵然贵为仙人者，偶尔也会在此沐浴清冷的月光…名字来源：出自《楚辞·离骚》：前望舒使先驱兮，后 飞廉 使奔属。（望舒为中国神话传说中为月驾车之神，也可借指月亮。）
