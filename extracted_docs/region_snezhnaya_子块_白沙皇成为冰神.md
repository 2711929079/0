# 白沙皇成为冰神

魔神战争结束，白沙皇成为冰神，位列尘世七执政。「巴纳巴斯」应为白沙皇或冰之女皇的魔神名，游戏内尚未明确[30]。
玻瑞亚斯回到挪德卡莱（前见#狼与仙灵）高塔崩毁，烈风止息，曾与龙卷之魔神争夺执政权柄的王狼，自觉无法描绘人类的未来，便将昔日庇护的族群托付于新的神明（巴巴托斯），默默回到了苦寒的北境。旧主早已离去的宫殿间，狼自此阖上了睡眼[31]（后续见#玻瑞亚斯与流浪骑士）。
