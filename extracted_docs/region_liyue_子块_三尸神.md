# 三尸神

兹白的三尸神分别为上尸神兹距、中尸神兹踬、下尸神兹蹻。茲距受困於虛假之月，地上一年，天上时间流逝数百年[11]即便身困凝滞的虚假之月，她仍然重复地用数字记录着时刻、时时刻刻与无时无刻。这是她抵抗虚无与磨损、记任自我与故乡的方式。兹踬承载着兹白最多的记忆与智慧，也继承了那份对凡人的深爱，打算在当有人向天空举起叛旗时，于叛旗上写下自己名字。兹白陨落后，唯有下尸神兹蹻的身影仍偶尔掠过故士的山水[5]。
