# 稻妻城

稻妻最为繁华热闹的城区，稻妻人大多居住生活在此。从花见坂到町街，可顺着地势一路向上，游览当地传统的老店，品尝稻妻特色的美食。
天领奉行府也位于城中，管理民众治安与幕府军备等事务。
在稻妻城至高之处，雷电将军高居天守，凌于众生之上。
