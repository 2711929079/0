# 霄灯

霄灯由「逐月」仪式的月灯演变而来，魔神争战，天下大乱，人们为替前方厮杀的战士英灵指引家的方向，便会点起如此灯火[5]。
桃都八奇八奇分别为归藏三隐、连山二贤、桃都三怪，合称八奇。主流的说法里，归藏三隐与连山二贤都是人类。归藏三隐里有一人，名「万象风角灵官」，据传，是古璃月奇门算术祖师。他还是沉玉谷十二部民中的蓝氏先辈，也将奇门术传承了下来，蓝氏族志中有相关记载。归藏三隐中另一人，民间尊称「云来钓叟」，钓法出神入化。云来剑法，还有更早的云来古剑法，就是从「云来钓叟」的抛竿姿势演化而来。余下一隐名为「无妄童子」，也有「无妄姥爷」之称。据传是一童身隐士，擅岐黄符箓，能捉鬼逐疠，也会驱死救生。往生堂最早的一批创办者，便在「无妄童子」门下求过学。他们本是凡人，以渺小之躯维护生死之界，困难重重，所幸习得了秘法，并代代相传至今。因此往生堂祖传了一根烧火棍，据说里面藏了一种很厉害的净世祭仪。二贤其一，名为瓠真人，擅使葫芦，其二名为鉴真人，擅使铜镜。他们留下的记录很少，只知修习的是连山秘法，因此被称为「连山二贤」。震雷连山密宫就是他们留下的道场。至于三怪，因她们同在桃都山修行，所以并称「桃都三怪」。这三位不是人类，其实也被叫做「桃都三仙」。一为朱赤引火幽蝶，二为银翎翦玉玄鸟，三为金目乘黄月驹，全是身负神通、能化人形的瑞兽。[29]华清归藏密宫是归藏三隐的密宫，虽然看起来只是一方清池，下面却埋藏着足以昏乱凡人心智的宝藏[89]。相传因连山二贤的秘法的运作，山腹内产生了无穷的雷暴。在秘境震雷连山密宫门前传说偶然能听见隐隐的滚雷之声，呼唤着求索的灵魂[90]。
