# 石龙

很久以前，璃月曾经有龙的身影。并非乘风翱翔天际的龙，而是踞身于山峦之中，身躯亦如山峦般庞大的悠古石龙。传说它曾沉睡在南天门一带，与群山混若一体。苏醒时，即使是微小的动作，或许只是伸懒腰，也会导致大地轰鸣颤抖。当时的岩君为了平息地动，来到了古龙的面前。传说大地平息了很长一段时间。而岩君的身边也多了一名伙伴—若陀龙王。龙王生为地下的生物，本来没有双眼，于是帝君与他签订契约，依他的愿望带他来到地面。之后，若陀成为帝君的战友，随他征战左右。但在最后，龙与神、人或许最终还是不相容吧…龙因为磨损忘却自己的愿望和契约，视摩拉克斯为仇敌，化作恶龙。他认为帝君不过是夺取古龙力量的僭越之人，[109]认为人类攻击古龙赖以生存的地脉。他和帝君展开征战，一路从层岩巨渊到伏龙树都是当年大战的痕迹。「陨龙之梦」是曾经被帝君射穿若陀龙王的弓，[110]其为昆吾以辰晶所作，原名昆吾坠龙[111]。帝君无法彻底封印若陀，此时若陀的意识一分为二，善念的意识甘愿嚷帝君将自己封印。理水叠山真君为了谨惕后人不可靠近封印而立下写有警示的石碑。龙被镇在大地深处后，畏怖着仙家与神的怒火，曾经聆听岩石间的震颤，一同作乱的岩之龙蜥，遁入了山间地底。但随着数千年的时光流转,龙蜥再次开始骚动…若陀邪念的意识重返世间，试图破坏封印。传说，帝君在镇压着古龙的巨木旁刺下一把「无工之剑」（昆吾所作，原名昆吾蛰剑[112]）。作为封印之一，魔物或是奸佞的人皆无法触碰，只有心澄如镜的人可以靠近它。但如果传说属实的话，这把剑又为何不知所踪呢…[113][114]层岩巨渊古文明古时的祭场曾因葬火之战埋入地下，又随着天钉與天星陨落而隆起，最终又因岩神和若陀龙王厮杀造成的破坏而显露出来。但曾经来往此间的人早已不再了[25]。
陨龙之梦在山的深处沉睡着一只比群山更古老的巨龙，在他漫长的沉眠中，他梦到了马车与月宫。在他的美梦中，龙王又一次看到了他曾经跟随过的那位伙伴。“如果我们骑在你的背上越过群山是不是就可以到达月宫呢？”这就是龙王的梦一在他不为人知的沉眠中。有时候，他会在痛苦中醒来，想起了那天他说的话，想起了那天他惹怒了那个人，他们从此走上了相反的道路：“我与你们一起为这片土地带来了和平，但我以后将撕裂群山，让群山像龙牙一样锋利，它们将摧毁你建立的一切，将这座城市化为废墟。”
为什么他会说这样的话？他明明爱着这片与那个人共享着美好回忆的土地；他明明爱着那些聚集在他们羽翼下小小的人类。为什么事情会变成这样呢？有时，被困住的龙王会陷入这样的思考。但是这种想法很快就消散了，因为他想起了故事的结局：那刺穿他身体的灼热的箭，那脖子被牢牢卡住时的室息，那让他心如刀绞的，来自金色眼眸的冷漠、锋利的视线[110]。若陀龙王为最后的仙兽[4]。
