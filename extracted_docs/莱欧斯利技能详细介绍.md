# 莱欧斯利技能详细介绍

莱欧斯利是《原神》中的冰元素五星角色，枫丹的狱卒长，称号"沉默的守卫"。
元素战技「冰牢」：
- 释放冰元素能量，对周围敌人造成冰元素伤害并施加"冰元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「冰狱」：
- 创造一个持续10秒的冰元素领域，领域内的敌人会受到持续冰元素伤害。
- 冷却时间：18秒。
- 能量消耗：60点。
固有天赋：
- "冰牢的祝福"：冰牢命中敌人后，有15%概率触发"冻结"效果。
- "冰元素增幅"：莱欧斯利的普通攻击和重击造成暴击时，恢复3点元素能量。
- "冰狱的威严"：元素爆发持续时间内，莱欧斯利的冰元素伤害提升20%。
