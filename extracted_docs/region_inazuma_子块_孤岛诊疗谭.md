# 孤岛诊疗谭

旅行者来到八酝岛，帮助医生保本收集鸣草以制作药物。保本多次小剂量试药，后前往八酝岛南部沙洲，服下了全剂量试药，毒发而亡。
执望三千里旅行者来到八酝岛，被长次感动（蒙骗）收集晶化骨髓，误会解除后长次在旅行者的帮助下从离岛出发寻找母亲。
