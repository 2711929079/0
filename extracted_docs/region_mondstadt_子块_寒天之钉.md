# 寒天之钉

几千年前，寒天之钉从天而降将雪山冻结，沙尔·芬德尼尔之城也因此毁灭。在冻结雪山的「长钉」降下时，风雪遍天。天降之柱崩裂为三。具体所指是，如今恢复了能量，悬浮在龙脊雪山上空的神秘结构体。
