# 神里绫人技能详细介绍

神里绫人是《原神》中的水元素五星角色，稻妻社奉行的家主，称号"磐祭叶守"。
元素战技「神里流·镜花」：
- 释放水元素能量，对周围敌人造成水元素伤害并施加"水元素附着"。
- 冷却时间：8秒。
- 能量消耗：无。
元素爆发「神里流·水囿」：
- 创造一个持续12秒的水元素领域，领域内的敌人会受到持续水元素伤害。
- 冷却时间：18秒。
- 能量消耗：60点。
固有天赋：
- "镜花的涟漪"：神里流·镜花命中敌人后，有15%概率触发"冻结"效果。
- "水元素增幅"：神里绫人的普通攻击和重击造成暴击时，恢复3点元素能量。
- "水囿的祝福"：元素爆发持续时间内，队伍中的角色水元素伤害提升15%。
