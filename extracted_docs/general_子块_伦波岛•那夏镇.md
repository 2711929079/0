# 伦波岛•那夏镇

——探险者的狂欢乐园位于东南方无论贫富，无论出身，无论地位，这里是属于所有心怀希望之人的乐园。金钱、谎言、阴谋与真心在同样的欢声笑语下流涌，作为挪德卡莱最繁荣的商港，来到这里的人们学会的第一课往往是「保护好你的摩拉」。伦波岛是由三个岛组成的群岛，又被称为伦波三岛。那夏镇鱼龙混杂，就连愚人众也无法掌握。愚人众在当地有眼线，有些眼线甚至已安插十余年。
