# 2000年前～500年前

马斯克礁马斯克礁尖帽子山被风神「改造」为如今的马斯克礁[52]。不清楚马斯克礁名为马斯克礁的原因，可能是莱纳德姓马斯克，尖帽子峰也是莱纳德命名的[56]。透过游戏二测任务道具「马斯克的游记」可知其名来自某位马斯克家族成员，是此游记的影响使其获得此名。
