# 百年前

德波饭店光荣剧院在无数次的易主后成为如今的德波大饭店[76]。《德波大秘典》就是饭店创始人莫德斯特和萨维利在冒险过程中创作的菜谱，他们以此建立了德波餐馆，但是很快二人就分道扬镳了，莫德斯特更是英年早逝，百年之后餐馆才再度复兴、做大做强。莫德斯特在制作大蛋糕时添加了雷穆利亚秘宝幽光素，但他逐渐觉得蛋糕的味道变了，才发现秘宝会在丰富精神的同时剥夺人的味觉，他本人也在失去味觉后自杀，在遗书里向萨维利说明了一切。萨维利就回到了遗迹销毁了所有秘宝以及其相关菜谱——他宁肯自己背负骂名也要保全朋友和饭店的名声[77]。
