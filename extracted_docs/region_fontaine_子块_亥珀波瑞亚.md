# 亥珀波瑞亚

在以前，枫丹这块土地被称为「梅洛皮斯」，也可以说成是在最初统一文明之下的梅洛皮斯古文明。到后面雷穆利亚时期才有了“枫丹”这一名称。梅洛皮斯这块土地曾因「神秘的力量」而板块抬升，变成高于其他大陆的高海[1]。「高海」其实为淡水湖，即「大枫丹湖」。目前不知抬升板块的神秘力量从何而来。在最初的世代，初民们的道德足以自持，无需仰赖律法与威权的统治。天城的使者行走在大地之上。经由他们的指引，人们在自太古以来便一直延续的安宁中平静地度日，安享繁荣与丰饶。天启的统治不知延续了多少个世代，直到人们厌倦了难以捉摸的永恒。后代之人不再虔信神谕，反而希冀神明并未许诺之事，试图挣脱命运的枷锁。于是高天为之震怒，祂（祂：称上帝、耶稣或神的第三人称代词）命大海掀起巨浪，将定居者的城邦碾碎。随后大雨连降百日，潮水吞没了一切罪恶与妄想，初民的时代就此终结。待到潮水退去，大地再次显露之时，高海之上已不再有任何城市与文明。余生或新生的人们栖息于山林与河畔之间，重归蒙昧和野蛮。如同地上与水下的一切生灵般，这时的人类任由自然法则的鞭笞与驱使,在无始无终的时间中浑噩度日。「亥珀波瑞亚」轮回对应的是失乐园。至于「纳塔兰提亚」轮回，对应的则是战胜恶龙。而这一时期的枫丹人，仍旧处于蒙昧的状态。现在已被遗忘的高海下的「亥珀波瑞亚」轮回时期的枫丹初民曾供养银白古树，而金蜂西比尔就是当时「上一次轮回中的幸存者」，是守卫银树的“使者”，在漫长的时间里失去了神智与身形。[2]
萃凝晶璀璨的萃凝晶曾是高天的使者带来的馈赠，日夜澈映的微光则是白月恩泽的象征，予世人以锻造奇物的权柄。[3]
