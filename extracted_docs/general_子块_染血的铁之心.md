# 染血的铁之心

游侠骑士第一次打倒魔物，将受难的仕女救下时，他拒绝了酬劳，因此获得了少女赠与的洁白的花。「骑士仅有的报酬就是践行骑士道」「我的奖章就是这朵花，如此足矣」这朵花一直别在他的胸前，直到被鲜血反复浸染，和骑士的银盔甲一样，变得像冬天的夜晚一样黑，又和骑士的心一样，变得像淬火后的锻铁一样硬。
