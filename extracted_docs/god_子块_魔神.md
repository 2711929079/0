# 魔神

据慈水怒涛之翼描述「天理」法涅斯与第二王座大战后，其神体碎片（原初的碎片）与宿主（如歌仙之章中提及曾是元素精灵的风神温迪）结合升华为魔神（法涅斯在葬火之战中受到重创，机能损毁，散落大量力量碎片，无力再凭借自己的绝对威权压制这个世界本来的秩序。来自原初的碎片与提瓦特大陆上的元素生命体或者魔兽融合，形成了所谓魔神权柄神格一类的东西。至此，这些元素生物或者魔兽脱胎换骨，拥有了神明般的伟力，这就是魔神。原初的那一位对人有一套神圣的规划。人只要幸福，它便欢欣。因此，不论获得了原初碎片的魔神是何模样，性情如何，都偏爱人类，会不自觉的庇护提瓦特上的人类。只是，这群神明级别的魔兽或者元素生命许多都不懂何为仁爱，不懂人类，因此会对人类行恶行却不自知，如龙卷的魔神迭卡拉庇安。）神之眼持有者即是「原神」，「原神」拥有登上天空岛的资格，并且成神（如温妮莎），但此类「神」到底是何种性质尚且未知，暂不应与魔神混为一谈。三月女神与纳塔人信仰的「夜神」与「十字路的主人」（现已确认是「空月女神」哥伦比娅[1]）也是同理。理论上魔神是获取法涅斯身体碎片升格的事物，法涅斯也曾主动将身体碎片赋予使其成为魔神，如厄歌莉娅。葬火之年后不久，由于原初法涅斯战中受到重创，机能损毁，无力再凭借自己的绝对威权压制这个世界本来的秩序。为了镇压、调服世界的不甘与愤恨，原初法涅斯与另一位后来者一同制造了「神之心」。世上应当建立起这样的秩序，人因此也只能拥有这七类念想。所有来自原初的碎片被驱使着互相吞并，造就魔神战争。魔神战争结束，提瓦特大陆上的魔神所剩无几。除去少部分逃到暗之外海，大部分不是战死就是被镇压。最终幸存的七位魔神得到神之心，且与神座绑定，加冕成为尘世七执政。武器焚曜千阳故事中，希巴拉克自称为赫布里穆，因此历代火神的魔神名是传承的。并且从魔神任务中可以得知，历代火神均不具备原初碎片，因此无属于魔神的权能与寿命，其神力主要来源于自身天赋、神座、部分死之执政的权能以及神之心。魔神能接受四影分享的权能。派蒙、空之执政—阿斯莫代、生之执政—纳贝里士、死之执政—若娜瓦和时之执政—伊斯塔露，虽然名字对应所罗门魔神名，但目前游戏剧情内尚未定性为魔神。
普尔森purson的描述：他能隐藏或是显现那些被隐蔽的宝藏；他能看见任何时间所发生的一切，并能为追随者带来强大的仆从；他能为万物带来真正的答案，但却会引诱寻知者询问那些他们无法承担的问题。相传许多书中都有提到祂原本是座天使或力天使堕天之后变成的恶魔。”
带来真理、强大的仆从，引诱询问无法承担的问题，由天使堕落，也都能符合花神的故事。花神原为天使，而天使受诅咒变成的仙灵也会显现被隐蔽的宝藏。
