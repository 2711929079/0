# CL-04「众命途的隐者」阿·阿瓦乔·库穆勒坎

库胡勒阿乔，基尼齐的龙伙伴，因未知原因被封印在神秘腕刻之中数千年。阿乔与基尼齐签订了契约：阿乔听从基尼奇的指挥，并依照契约部分暂时地将力量借给基尼奇使用。而代价是基尼奇要将阿乔带去外面的世界，并在死后将身体交给阿乔使用。从此，「祝基尼奇早日完蛋」成为了阿乔最大的愿望。由于本体仍被封印，阿乔只能借燃素来显现如今的样貌。世界任务「失落圣城与千年来者」：库库尔坎称，目前仅有一位「至高领主」在世；而阿·阿瓦乔·库穆勒坎曾从事生命燃素化的相关研究，与阿乔如今的燃素身体相对应，暗示了阿乔的真实身份。
