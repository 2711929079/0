# 克林索尔

教令院调查团在完成对遗迹巨像的调查后，继续沿着巨像行进的道路抵达了「那伽朱那团」的营地，据说他们当初也是沿着这条道路抵达这里的。但在听说我们（调查团）的来意后，他们表现的似乎并不友好…经过与那些「那伽朱那团」的家伙交涉后，终于获准进入荼诃人的地下遗迹进行调查。但这一准许仍是有限的，并且需要在他们的监督下进行，他们派来了一个名为克林索尔的「巡路官」，负责照看和监视我们，他自称曾经是雨林人，但从他的名字来看应该并非出生于须弥。茶诃人掌握着我们难以想象的秘密，而这些秘密的答案可能就藏在这些巨大的遗迹当中。「那伽朱那团」对此的保守方针实在是暴殄天物，就连负责监视我们（调查团）的「巡路官」也同意这一点…[83]在遗迹中找到了一些考察人员（雷内一行人）留下的痕迹，看来早已有人（雷内一行人）先于我们进入过这里。不过他们（雷内一行人）留下的记录里无非是些疯话。以及不知所谓的呓语，并不能使我们（调查团）得到什么教益…奇怪的是，对于遗迹中发现的文献，那位「巡路官」似乎表现得比我们更感兴趣。或许能借此让他放松对我们的监管。[84]不知受何影响，调查团内部关于是否继续对另一处遗迹进行调查产生了分歧。有些成员提到，他们一直被某种从地下传来的声音困扰…在领队大人的坚持下，终于决定继续前进…克林索尔完全被遗迹中的知识所俘获，对于我们进一步的调查行为已不再有任何阻拦…[85]与此同时，克林索尔发现矫论团的长老们對他隐瞒了些什么，与调查团们同行，才终于解开了他多年来的疑惑，发现深渊的力量。且发现坎瑞亚是全体人类的祖国（第一文明/统一文明）的遗胤，因而称海杜菈为自己先祖，打算为了全体人类与其祖国使用使人超越的力量。他找到了多日来萦绕在耳边的「福音」的源头—深渊力量，化作漆黑的深渊魔物，对调查团出手已将他们灭口[86][87]。
