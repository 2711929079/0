# 巴窟纳瓦

层侵的黄金造物，曾吞食黄金的身体部位，具有某种连深渊教团也无法理解的更加超越性的深渊力量，就算只剩肉块，也会从肉块聚合成原状。烬寂海污染的源头，使其污染具有时间性，会侵蚀时间轴的每个角落[96]。火神柯夏尼娜曾率众讨伐巴窟纳瓦，并发现它能复甦，讨伐它并非一代就能结束，时日不多的她只得走入圣火，留下必须建造封印组隔烬寂海的遗言[101]，直到五百年前才有火神建造封印并由五百年来的火神逐渐加固[96]。直到最后柯夏尼娜始终保持坚强[99]。之后历代都有勇者（其中至少部分为历任火神）继承使命，已知除特洛切一行与柯夏尼娜外有至少五代人前往讨伐并留下石碑，但其中至少三代具体情况暂不可考。某二代勇士分別名为古洛哈莉与修洛维兹，皆是前代火神（最后的特诺奇兹托克人（世界任务）提及石碑是历代火神留下的，所以石碑上的名字就是各火神名）。年代较近的一代勇士伊莎佳在讨伐完魔物，生命结束前，潇洒的坐下歌唱[102][98]。该处聚集历代前往烬寂海讨伐的英杰（包括特洛切一行人）生前遗物，透过特殊法术将前往特诺切召唤的夜神空间及进一步前往烬寂海的入口隐藏[98]。曾有讨伐巴窟纳瓦的勇士在烬寂海远方依稀见到样式熟悉的建筑（可能是别处时间线的文明，或是渡过烬海的渊上暂时的落脚点），认为不可能于是前往调查，将此事刻在古盾上[103]。
乐团与烬寂海乐团的笛手曾与当地人在烬寂海与巴窟纳瓦交战，并拆下它身上某块蕴含生命力的符石[104]。
