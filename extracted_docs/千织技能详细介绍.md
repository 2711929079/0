# 千织技能详细介绍

千织是《原神》中的岩元素四星角色，稻妻的织染师，称号"织染纺光"。
元素战技「织染之术」：
- 释放岩元素能量，对周围敌人造成岩元素伤害并生成岩元素结晶。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「千丝万缕」：
- 创造一个持续8秒的岩元素领域，领域内的敌人会受到持续岩元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "织染的精妙"：织染之术命中敌人后，有15%概率触发"结晶"反应。
- "岩元素增幅"：千织的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "万缕的威力"：元素爆发持续时间内，千织的岩元素伤害提升15%。
