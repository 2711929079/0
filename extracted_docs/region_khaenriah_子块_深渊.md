# 深渊

坎瑞亚人引入深渊力量后，其为机关供能的速率远大于阿索斯物质。[34]以阿索斯物质作为能源的技术路线，在当时以被王朝当作过时的举动。由于旧的能源无法供应高度复杂的动力系统，以阿索斯物质功能的机关工坊陆续淘汰，直到最后，王家的机关工坊“Gnade”也被迫关闭…[35]
但并非所有坎瑞亚人都将国家的希望寄托在这种无法控制且不可名状的事物上。他们认为，依靠人类的智慧，也能发掘出真正的永续能源。[34][36]但坎瑞亚末代君主伊尔明没有听取以皮耶罗为首的保守派大臣的意见，反而遵从贤者海洛塔帝的嘱咐引入深渊力量。[37]但为了不要让地脉技术彻底失传，安弗塔斯元帅请求坎瑞亚王保留部分依照旧式标准制造的机体。[36]最终，坎瑞亚王答应安弗塔斯·亚尔伯里奇[38][14]，保留被须弥人称作“遗迹巨像”的大型守望机关，而如今散落各处的耕地机因为是旧式机体，仍然使用地脉能量作为驱动能源。[39]与此同时，深渊能量引入，王国四柱迎来了前所未有的繁荣，其机关术突破了世界法则的桎梏[33]，但因为无法妥善使用而导致失控…[11]
错位时空旅行者的血亲身上的深渊力量被五大罪人瓜分。蒂莱尔终于找到了「救世主」（旅行者的血亲），然而旅行者的血亲看到她手中来自五百年后的飞船休眠舱钥匙，以为旅行者醒了并掉了钥匙得蒂莱尔捡到，陷入「拯救坎瑞亚」和「救旅行者」两难选择，最后选择救旅行者，到了飞船休眠舱时却发现旅行者还没醒来。此时旅行者的血亲已来不及拯救坎瑞亚，坎瑞亚随之覆灭。蒂莱尔打算拯救坎瑞亚的努力，却导致坎瑞亚的覆灭，这就是蒂莱尔的命运。（旅行者也可以将钥匙收回，但结果是一样的，因为五百年后旅行者的血亲也会受到伊斯塔露的干涉见到蒂莱尔，为了防止收回钥匙导致涟漪效应使世界走向未知的命运使自己的计划（即自己因为被降格了，所以成立深渊教团用一整个世界与天理对抗打算获取与之平等对话的资格恢复降临者身分，和让新坎瑞亚成为自己合适的新的家乡）（有说法降格的原因是书籍裴伦埃里中教养院的神秘仪式所致，但终北的夜行诗中「仆人」重现此仪式时，仪式传来对话：降临者⋯没有权限⋯，因此可能无法使降临者降格）不能如计划中进行，于是将自己一样的另把钥匙丢给蒂莱尔。旅行者在内心纠结下顺从血亲的计划。）[28]
