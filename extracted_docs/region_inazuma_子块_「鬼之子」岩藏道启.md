# 「鬼之子」岩藏道启

母亲御舆千代的叛变，让御舆这个姓氏蒙羞。御舆道启在影向山一座小社遇到了光代，光代为其取名“岩蔽”，希望他能够走出家族的阴影。光代还让岩藏修炼剑法，两人约定未来每一年都在此地决斗。道启愈发勤奋的与天狗族长之女光代练功，并委托天目打造「薄缘满光天目」。最终练成天狗抄这一门剑法，据说其可以击落飞翔中的天狗，道启也成为九条家的剑道指南。距离约定过去了13年，岩藏在第13次决斗中领悟了剑法“天狗抄”并成功战胜了光代，小社也因此被毁。天狗一族决定归隐山林，道启无奈与光代分别，自立宗派名为岩藏流。岩藏道启持有天目流名刀「薄缘满光天目」，但于前代的时候被弟子丢入海渊之中（最终会由玩家在蛇肠之路找到[48]）。天领奉行请其训练新兵，岩藏道启晚年离开天领奉行，来到绀田村隐居，将早年委托锻造的好刀薄缘满光天目的锻造图纸，赠予帮助他的柴门一家。[49][50]
魔偶剑鬼在制造人偶的过程中，雷电影还发明了魔偶剑鬼这种副产物，这种量产型的人偶有的会作为“鸣神特使”在演武传心中测试凡人武者的实力；有的被分配到幕府军中；还有的因为失控而流落野外。经时任九条家剑道指南的道启允许集成了以秘剑「天狗抄」闻名的剑道流派初代宗主岩藏道启的记忆[51]。御舆千代在背叛雷神后逃亡，最后死于继承岩藏道启记忆的某個失控被废弃的魔偶剑鬼刀下(稻妻人推测的其死因之一）。[52][42]
