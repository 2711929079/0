# 深渊

提瓦特由光界、人界与虚无界构成，深渊力量即虚无界的力量。「深渊」力量会侵蚀光界与人界的所有，光界力能稍微抵抗深渊。在命运的终点，人会被深渊吞噬。深渊本体是「漆黑意志」，是目前已知最强大的存在之一，拥有意识（声音为男性），会学习，透过侵入读取地脉中的记忆制定侵略战术。深渊能够幻化形体，若妥善运用能成为强大能量，与元素力本质同源[95]。是未发生的命运的恨意所产生的[96]。
