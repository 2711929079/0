# 阿陀河谷

贯穿须弥的大河阿陀逻伐底常年冲刷所形成的河谷，沿河可以直抵南方的海岸。虽说是须弥最主要的水上商路，但河流两岸在巡林官的悉心保护下仍然保持着相对纯净的自然生态，生活着众多可爱的生灵。
