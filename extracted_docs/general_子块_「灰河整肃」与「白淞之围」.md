# 「灰河整肃」与「白淞之围」

枫丹的城市优化使下水道成为了贫民窟，爱德华多·贝克凭借着一根随手拣来的铜管，从一开始的击退鳄鱼，到保护商贩店家免于强人帮派的侵扰勒索、调解灰河「同胞」之间的纷争，最后成为了灰河社会的话事人[5]。出身灰河，以行骗、行窃为生出没于光荣剧院的小贼姐妹，为科培琉司戏中的游侠喜剧故事所感染，萌生了凭借魔术走上舞台、平等地为贫富众生带来表演的梦想。于是一人以戏中的蒙德侠盗「帕西法尔」为艺名，一人则扮演助手「约瑟芬」，两人的表演从爱德华多的酒馆中、灰河水渠旁，走进了光荣剧院里，帕西法尔的名字在地上城与地下城中都令人津津乐道[8]。但城市的“优化”是没有尽头的，在枫丹廷发起的灰河整肃行动中，扮演「帕西法尔」的少女死去了，「约瑟芬」赶回灰河，只见到了没能完全干涸的血。于是「约瑟芬」接下了「帕西法尔」的名字，并立志实现一个没有贵族和王者、没有生来的富贵和无法摆脱的贫困的世界，以作为属于她们的「大魔术」[8]。而爱德华多及其手下则被捕，押送沙漠流放，途中被其党羽营救，一行人伙同在秋分山活动的匪徒占领了白淞镇，挟持了负责押送的执律庭成员，试图与枫丹廷谈判。记者卡尔·英戈德主动请缨传递谈判要求，为调停争端而在枫丹廷与白淞镇间来回奔波[6]。在事态恶化前，他曾经拍下爱德华多、帕西法尔与时任白淞镇镇长雷诺·德·佩特莉可等人，以及当时还是孩子的雷内和雅各布的合影，当时的众人眼中充满着对未来的希望。[5]。卡尔是出身水仙十字院，与德怀特•拉斯克、贝瑟·埃尔顿以及埃马纽艾尔·吉约丹一同长大的孤儿[3][4]，在幼时的勇者游戏中，卡尔常常扮演恶龙贾巴沃克，而埃马纽艾尔常常扮演讨伐恶龙的英雄[2]。长大后的埃马纽艾尔加入了逐影庭，以打击犯罪为工作，「白淞之围」正是在其参与指挥下进行的，而这场围剿的进行也代表着卡尔为之努力的谈判最终没能达成。悔恨于自己所为，失望于枫丹廷的领导，埃马纽艾尔离开了逐影猎人的队伍，从此自暴自弃借酒消愁[7]。没能促成和谈的卡尔则在另一侧同样经历了这场围剿，帕西法尔用「戏法」强行将其转移到漆黑而安全的地窖，只能听到隔着砖石土地闷闷地传来的喊声、破裂声与金属碰撞声。白淞之围后，帕西法尔被捕，即将迎来审判[6][9]；爱德华多与雷诺死在了围剿中，雷内与雅各布则作为犯罪者子嗣被送入水仙十字院抚养[2]。大魔术师「帕西法尔」请求比武审判的消息连同其出人意料的罪行，在枫丹廷引起了相当的轰动，审判庭更派出传奇决斗代理人玛尔菲莎作为控方的代表参与比武审判。此时的卡尔已辞去记者的职务，作为探险者与荒野、废墟和遗迹为伍。数年后其回到十字院为成员们拍照时，透过镜头想起过去的种种经历，直到最后，没能作为记者在比武中记录帕西法尔最后一战的懊悔才将他的视野席卷——决斗的结果是帕西法尔身亡[6]。灰河与白淞镇的事由此告一段落，再往后便是坎瑞亚灾变，枫丹水域上涨，时任副院长的贝瑟在十字院淹没前将四名孩子分别托付给卡尔和埃马纽艾尔[10][7][3]。至此，埃马纽艾尔才不情愿地再度尝试与人一同生活，尝试为了儿女创造一个安全的世界。于此同时，自然哲学院的学院长德怀特·拉斯克下令封闭学院，以免血气方刚的青年外出送死。[4]故事的最后，为追查在枫丹境内作乱的「水仙十字结社」，德怀特委托退役多年的埃马纽艾尔调查，[11][4]循着线索找到了「大师」，被其利用莉利丝的能力溶解吸收（埃马纽艾尔感受到院长的怀抱）[7][2]。传奇决斗代理人玛尔菲莎年轻时曾与一位剑客（埃马纽艾尔）交手，被其使诈偷袭而败于其手，被迫认输。信丰公义的玛尔菲莎对此耿耿于怀，本想再找他对决，却意外得知他早早退役、投身烈酒。为了亲自查明真相，她追随着剑客的足迹与创伤，一路追猎命运中的对手…尽管有重要之人（扮演帕西法尔的约瑟芬）倒在命运的决斗场上，尽管内心已残破不堪，在追踪到剑客所留下的面具后，一切沧桑仿佛都得到了报偿。最终，玛尔菲莎与自己苦苦追寻的埃马纽艾尔落得同样的结局，「大师」获取了她的枪术技能[9]。
