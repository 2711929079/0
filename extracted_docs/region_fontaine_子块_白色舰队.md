# 白色舰队

过去的辉煌年代，纯白色的铁甲旗舰「斯彭西安」曾是枫丹海军的骄傲。那个时代里，装备着庞大重炮与坚固冲角的铁甲舰队乃是枫丹廷的宠儿，在光荣剧院的时代，观众无不为白色舰队清扫魔像军团的故事起立鼓掌。
坎瑞亚战争坎瑞亚战争爆发，六神听从天理的号召前往坎瑞亚对抗深渊。厄歌莉娅恍惚间意识到这一去很可能就是永别，因为只要身为原始胎海心脏的自己一天不死，预言就一天不会发生。但天理不可与之为敌，厄歌莉娅接受了自己的命运。临行前，厄歌莉娅找到芙卡洛斯，将水神的职责与拯救枫丹的任务一起继承给这个自己最信任的纯水眷属。厄歌莉娅最终死在苍漠囿土，化作甘露之海，孕育出花灵一族。她死之后，没了心脏的原始胎海开始躁动，枫丹迎来第一波涨水期。这让枫丹人们都想起了那条预言：所有人都会溶解，枫丹被水淹没，水神自己在神座上独自哭泣。芙卡洛斯潜入海底安静地思考了很久，直到身边的蚌壳都开始冒泡泡了，才想出了那个唯一的答案。命运是既定的，越是反抗反而陷得越深，可命运与现实之间有一条小小的裂缝，借此我们便可以欺骗命运，或者说——欺骗天理。芙卡洛斯将自己的「神格」从身体与精神之中分离出来，取名为芙宁娜。随后，芙卡洛斯诅咒了芙宁娜。她虽有水神之名，却无水神之实。她虽能喜怒哀乐，却不能这样做。她将被众人仰视，但却没有哪怕一丝目光是真的向她投去。她甚至没有死亡的自由。她将被迫以这歌剧院为舞台，成为歌剧院真正的女主角，为了让预言看似实现而永远扮演水神的戏份。芙卡洛斯又选择了那维莱特，一个对自身为何以人形降临毫无所知的水龙王。拯救枫丹人民说白了就是替换血液里的原始胎海，想必能做到这一点的，只会是原始胎海的真正心脏——掌握古龙大权的完全体元素龙王。法涅斯窃取了古龙的力量，使其在一任任元素执政间流转。所以想要归还水神的权柄，就必须杀死「水神」，或者说摧毁水神的神座。于是，芙卡洛斯创造了谕示裁定枢机，自己带着神之心与它合二为一，默默收集民众的信仰[45]。
