# 元素

提瓦特的力量原本是荒蛮的光界力，后来被天理改造成更温和方便人们使用的七元素力。分为风、岩、水、火、草、冰、雷。如今提瓦特的作物无法适应蛮荒的光界力，土地从元素变成荒蛮燃素的过程称为「圣土化」。月矩力是光界力的一种。
