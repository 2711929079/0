# 埃洛伊技能详细介绍

埃洛伊是《原神》中的冰元素五星角色，来自地平线系列，称号"异界的救世主"。
元素战技「冰寒射击」：
- 释放冰元素能量，对周围敌人造成冰元素伤害并施加"冰元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「冰冻领域」：
- 创造一个持续8秒的冰元素领域，领域内的敌人会受到持续冰元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "冰寒的祝福"：冰寒射击命中敌人后，有15%概率触发"冻结"效果。
- "冰元素增幅"：埃洛伊的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "冰冻的守护"：元素爆发持续时间内，埃洛伊的冰元素伤害提升15%。
