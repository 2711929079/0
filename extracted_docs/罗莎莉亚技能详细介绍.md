# 罗莎莉亚技能详细介绍

罗莎莉亚是《原神》中的冰元素四星角色，西风教会修女，称号"棘冠恩典"。
元素战技「穿刺的冰枪」：
- 投掷冰枪，对敌人造成冰元素伤害并施加"冰元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「终天的闭幕」：
- 释放冰元素能量，对周围敌人造成冰元素伤害，并创造一个持续8秒的冰元素领域。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "冰枪的裁决"：穿刺的冰枪命中敌人后，有15%概率触发"冻结"效果。
- "修女的祈祷"：罗莎莉亚的普通攻击和重击造成暴击时，恢复2点元素能量。
- "冰冷的祝福"：元素爆发持续时间内，队伍中的角色冰元素伤害提升15%。
