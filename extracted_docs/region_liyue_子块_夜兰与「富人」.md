# 夜兰与「富人」

夜兰渗透了愚人众执行官“富人”在暗中布下的隐蔽商路，成功拦截了运载的货物，也因此损失了一只名为“幽奇腕阑”的祖传玉镯。货物中有一件珍稀的魔兽皮毛，本来是计划送给至冬女皇的礼物，夜兰将其作为玉镯的赔偿收走了，并制作为一件大衣[209]。
枫原万叶北斗救下并收留了正逃离稻妻幕府通缉的枫原万叶[210][211][212]。
