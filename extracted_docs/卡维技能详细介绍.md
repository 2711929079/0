# 卡维技能详细介绍

卡维是《原神》中的草元素四星角色，须弥的建筑师，称号"天穹之镜"。
元素战技「建筑之术」：
- 释放草元素能量，对周围敌人造成草元素伤害并生成草元素结晶。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「空中楼阁」：
- 创造一个持续8秒的草元素领域，领域内的敌人会受到持续草元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "建筑的精妙"：建筑之术命中敌人后，有15%概率触发"激化"效果。
- "草元素增幅"：卡维的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "楼阁的威力"：元素爆发持续时间内，卡维的草元素伤害提升15%。
