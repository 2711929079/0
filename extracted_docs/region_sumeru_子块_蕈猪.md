# 蕈猪

生活在雨林之中的野林猪，在演化过程中与蕈类形成了奇妙的共生关系。根据学者的研究，蕈猪依靠寄生其上的蕈类作为天然的迷彩，可以有效地躲避捕食者，更容易在丛林中存活下来。不过同时也因此被当作了天然的预制食材而在餐桌上广受欢迎。
