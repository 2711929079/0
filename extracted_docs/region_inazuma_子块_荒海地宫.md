# 荒海地宫

狐斋宫带领人们躲进荒海的地宫，其中也包括御舆兄弟俩和浅濑响。[3]曾大闹宴会的御伽大王和孩子们告别后，竟也加入了守护稻妻的大军里和御舆千代一同作战，并在战中身亡[23]
「狐斋宫」「狐斋宫」原本是白辰的血脉，曾与雷神同行,守护稻妻的子民。在这一生中，我也曾化为人的形姿，与这些短寿而美丽的小小生灵同行，以不同的身份，成为许多人的挚友（文本提及浅濑响、御舆长正、惟神晴之介、弘嗣、长野原一族）。守护他们的结界，但愿不会被任何黑暗侵蚀…这一切，所有的所有，都是我如今所怀念的「所以啊，撕咬我的漆黑意志，」「现在我已经失去了所有力量，」「我的白辰之血就任您挥洒吧。」「但是，尽管处于卑微的立场，」「我仍希望您能聆听我的请求…」「如果您能看见我所珍重之物，」「那么就请您饶恕那些生灵吧。」「如果您恩准我提出一个愿望，」「就请您将我永远明亮的记忆，」「归还给我热爱的这片土地吧。」「希望以此，在您的肆虐之后，」「仍然有美好的东西能留下来…」[40]在五百年前的灾厄中,「狐斋宫」为了保护鸣神岛，而与漆黑的灾厄作战，最终被灾厄吞没了。吞食她的灾厄最终被斩落，而她的思念与记忆也回到了鸣神岛的大地中。[1]狐斋宫死后，力量更强的「天狐」与「地狐」都化作石像等待她的回归，而这些狐狸则随着一族的灵脉变得稀薄，而不再开口，也失去了法力。狐斋宫死后，八重神子接手了「宫司」一职[12]。浅濑响拿着狐斋宫生前用的烟管，失去之人烙印在梦中。惟神晴之介远走他国，制作训练对抗妖邪阴阳寮，并自己亲手打造了一个烟管，思念着某个友人。
