# 琳妮特技能详细介绍

琳妮特是《原神》中的风元素四星角色，枫丹歌剧院的魔术师助手，称号"星彩魔术师"。
元素战技「幻影突袭」：
- 释放风元素能量，对周围敌人造成风元素伤害并击飞。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「魔术表演」：
- 创造一个持续8秒的风元素领域，领域内的敌人会受到持续风元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "幻影的祝福"：幻影突袭命中敌人后，有15%概率触发"扩散"效果。
- "风元素增幅"：琳妮特的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "表演的威力"：元素爆发持续时间内，琳妮特的风元素伤害提升15%。
