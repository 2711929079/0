# 祭礼时期至葬火之战结束

［芬德尼尔时间线推论·非正文］葬火之战时期天使们因触犯禁忌（与外来者相爱）遭到天理诅咒，退化成仙灵，失去记忆、智慧与情感。[1]游戏内并没有明确交代沙尔·芬德尼尔的建立时间，以前玩家透过雪山石碑上记载的“冰雪与纷争”推测其建立于魔神战争时期。[2]但实际上魔神战争结束后，风神才吹散冰封蒙德各地的冰雪[3]，因此沙尔·芬德尼尔可能成立于魔神战争前任何时段。
随着须弥开放，关于葬火之战有了更多文本记载[1]，沙尔·芬德尼尔的时间线也需要更动：因为沙尔·芬德尼尔的人们会透过神使和天空岛沟通，并追求神使的指引，[4][5][6]因此沙尔·芬德尼尔应创立于葬火之战前，创立于神使并未遭到诅咒被贬下天空岛，人们能被准许与天空岛沟通时。
另外，因为人类使用禁忌知识，天空岛在葬火之战结束后降下许多天钉封堵禁忌知识。[1]沙尔·芬德尼尔大概率是在此时因降下的天钉覆灭。第一文明第一文明曾经有一个时代，大地上的人们还能直接听到来自天空岛的启示。神的使者走在蒙昧的人当中。彼时万物的气息渐弱，大地永冻[7]。在天使的带领下，人们创造了统一文明，即「第一文明」。第一文明大部分都是石制砖头建筑，上面有凯尔特三角的标志，风格明显，很好辨认。蒙德的第一文明建筑中，望风山地的看似零散，实际上构成了一座建筑群。南部的遗迹群一路向高地延伸，遗迹的尽头是一处秘境——仲夏庭院，古代文明建造的夏宫灵囿，是一座用于休闲观赏的庭院建筑，不难想象它曾经的繁荣景象。过去的人们向往天空，于是将城塞建立在高峰之上[8]。（那些坐落在山峰上的古老的遗迹，毫无疑问连同山峰一起在风神修改地形时随风而去了，如今只留下少量的残存）风啸山坡的建筑群在达达乌帕谷谷内，地图上山体凹陷处，较能阻挡当时的风雪。苍风高地存有一大片遗迹，由于遗迹大多分布在大山丘上。苍风高地七天神像所在的山丘无论从遗迹数量上看，还是从位置上看，都很像是古国的核心区域，因此推测这里是主城。在周围还分布着一些城镇和祭坛，有些依旧保存着原貌。塞西莉亚园圃的位置相对偏僻，远离遗迹群中心，道路也十分曲折。尽管如此，一路上都分布着与山丘上古国相同的遗迹，这说明它确实属于这一古国。秘境介绍中提到，这里曾是培育塞西莉亚花的温室，如此看来位置偏僻一点也不奇怪。蒙德古国会兴建能产生风场的机关。蒙德曾存在与古树并存的古国。古国的先人们兴建了塞西莉亚花的苗圃。在今苍风高地建立了塞西莉亚苗圃，这个苗圃是种植塞西莉亚花的温室，该地也具备祭祀职能。位于今摘星崖和千风神殿地带的文明建造了祝圣祭坛夏宫灵囿。雪山沙尔·芬德尼尔之城彼时万物的气息渐弱，大地永冻[7]，蒙德被冰雪覆盖，生存条件极为恶劣。一位名叫法鲁希的男性带领族人来到了当时的雪山地区，出乎意料的是，这里并没有被风雪覆盖，而是保持着自然的苍绿葱翠，于是他们在此建立了王城。——沙尔·芬德尼尔（Sal Vindagnyr）[2]，成为第一文明在如今名为龙脊雪山的区域的分支。其遗迹最西延伸到石门。公主山脊守望矗立在山脊的起点，如同界碑一般守望着南与北、山岳与水泽的远古城墟。传说中它的门扉只对心中燃烧着永不熄灭的烈火的人敞开，为他们祝圣戴冠[9]。相传古代的人会深入幽谷，寻找祭坛上的古树，将秘密说给树洞听，以卸下守密的沉重负担。忘却之峡中，充满了被遗忘的秘密[10]。当芬德尼尔的祭司之女诞生在这棵白树之下，接受祝福时，苍翠山岳的国境中充满了欢欣。沙尔·芬德尼尔的幸福一定会永远存续，正如贯通大地、永不凋败的银白之树。——为山中国度撰史的乌库，当时是这么想的。曾见证过无数人与事的记事者由衷相信，公主的美貌与才德将如月光般永远皎洁…[11]忍冬之树寒天之钉在冻结雪山的「长钉」降下时，风雪遍天。天降之柱崩裂为三。[2]毁灭当冻结世界的长钉蓦然降下，连这棵树也被其余波粉碎、冻结时，那位少女取走了最完整的枝，想为荫蔽一国的树接续生命。但在最后，嫁接的生命始终没能活下来。刀刃般冰冷的风雪，最终将月光遮蔽了…[11]芬德尼尔太靠近与依赖地脉树，天理砸钉子是为了钉地脉封堵禁忌知识扩散，结果芬德尼尔也给砸掉了。且忍冬之树死亡，失去忍冬之树庇护，芬德尼尔立刻变为雪山，不再适合生存。但当时天理为了及时封堵禁忌知识无法（也不想，因为第一文明使用禁忌知识，芬德尼尔则是第一文明支系，虽然没有用禁忌知识却属于第一文明，否则天理砸钉子前可以发警告）考虑这些。又或者天钉出故障才毁灭芬德尼尔的。毕竟天钉是打入地脉根系堵禁忌知识，不是砸树。其他天钉钉完地脉后都会飞回去，但雪山的不知为何不会，可能是出了故障。祭司法鲁希冻结天空的长钉降下后，过去的祭司登上了原本朝向天空的祭场，却没能听见天空的声音，也没能返回。[5]「老身本以为后代能如白树一般生生不灭。」「但白树即将枯死，『他们』的声音不再传来，老身的期待、他们于此身的期待也即将落空。」「老身将再次登上峰顶，寻求指引。下山时女儿第二幅画应该已经完成了。」「第三幅画，若是冰雪消融的图景就好了。」[6]异乡人祭司之女梦见了蔽日的漆黑之龙（以时间线来看，葬火之战前，日后葬火之战就发生，因此是尼伯龙根），是为凶异之兆。同月，异邦人伊蒙洛卡[2][12]来到王城。祭司之女将星银铸成的大剑交予异乡的勇士，风雪的啸声中，她说出的话没能传达给对方。[13]「这里的第四幅壁画为你而准备，你的形象将会永远留在这面墙上。」「为了这幅壁画，为了大家,我会一直在这里等你，祈祷你的归来…」当雪葬之都的女儿与嫁接无果的银枝条一同凋零时，命定挥舞此剑斩开冰雪的异乡人正在远方寻求答案。皎洁如月光的她，最后的思念也没能传达给远行者。……[13]「壁画恐怕无法完成了。」「我已经很久没看过晴空与绿地了。不知道应该用什么样的蓝色、什么样的绿色，才能画出父亲想要的，冰雪消融的景象。」「等他回来，一切都会恢复过去的样子吧。但是，太冷了。我已经不行了。」「对不起，爸爸。对不起，一直看着我们丑态的乌库。」「伊蒙洛卡，如果能再见你一面，就好了…」不满足于受保护的绘画少女，她向倾慕的人留下这最后的嘱咐：「假如天性中的胆怯与绝望将你压倒，令你终于不再归来，那么…」「…请你活下去。请不要与我们共同走向灭亡，湮没于冰冷的遗忘」[14]记事者乌库「愿降下这片冰雪的天空燃烧，直到世界上再也没有一丝生命。」「愿我们化作灰烬，顺风而行，追上那抛弃了公主再也不返回的异邦人。」「愿公主梦见的蔽日的黑龙，为可憎的大地涂上赤红的毒。」「我是最后一个。再也没有留守的必要了。听说何处有人正在建立无神的国度（坎瑞亚）。那么或许他们也有对抗世界的力量吧。」[12]最后记事者乌库背弃神明成为了以非人身份存活至今的丘丘霜铠王。芬德尼尔覆灭后，其残余遗民乌库成了丘丘王，而部分遗民失去过去记忆，试图向下探索，寻找已经被遗忘的东西。遗民失忆可能是和乌库一样成为丘丘人，而丘丘人挖掘的动作就是因为试图向下探索。为了找寻平息高天怒火的方式，为了通晓命运，他们试图向下探索，寻找已经被遗忘的东西，从深埋地底的古代祭场、废都中寻求启示与答案[15][16][17]。古树树洞的另一端，过去的秘密凝结成了宝藏[18]。伊蒙洛卡家族这就是他找到的答案——异乡的勇士终于结束了徒然的旅程，漆黑的污血从大剑的刃上点点滴落，沉重的双脚踏上已变得陌生的雪径。当疲惫的异乡人终于归往山国殿堂，为他洗尘者却仅有报死的空幽回音。「就连这里，也没有留下值得我守护的事物了吗…」「天上的你们，只是想要看到生灵涂炭的惨状吧。」「既然如此，就以钢铁与血的歌，给你们消遣吧。」异乡人将少女交给他的，本应斩碎风雪的星银留在了壁画之间。然后下山寻找充满纷争与战斗的地方，能让他泼洒鲜血的地方。[13]最终，伊蒙洛卡选择与栖身寒渊的冰霜之巨蛇殊死缠斗。「冰川的严寒甚至足以冻结时间」部族的人们中间，有这样的传言。与巨蛇决战的英雄对此嗤之以鼻。当勇士登上叹息之墙时，时值深夜，雪暴呼啸。勇士进入北境冰川时，暴风中已透入几缕日光。无论何种严寒暴雪，确无法令时间的洪流凝滞，即使英雄本身也随着记忆消逝。最终勇士陨落在凛寒之中,连热血也被冻结[19]。
