# 龙裔建筑

黄金城存在龙裔建造，被后世霜月之子拿来当作避难设施的地区。其中的机械能辨识进入者的基因，但已经故障[13]。
迁灭之时黄金城覆灭一夕之间，光耀的精金塔像无花果树般倾倒；从圣城到古都，极北的冰原上凡有血气的城邑，皆被幽蓝的晶钉尽行毁灭。妖精们尖啸着化为银白的冰雾，无论是那受颂的圣徒还是最初的神使，均于灾厄后不知所踪（见#狼与仙灵），因远离诸城而侥幸生还者瑟缩于风雪中，在荒废凄凉的幽暗中等待灭亡的终临。[11][1]
