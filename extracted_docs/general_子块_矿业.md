# 矿业

明蕴镇
一位名叫元鸿的矿工带着一家人在瑶光滩北部的山岭间开采矿石，白手起家建立了明蕴镇。明蕴镇曾经很繁荣，是仅次于层岩巨渊和黑岩厂的采矿中心，元鸿一直被人们认为在镇子里埋藏了一份祖传宝藏，连元鸿的三个儿子都信以真，纷纷前去寻宝并相继死于意外，元鸿也因此心力交瘁，最终病逝。在元鸿死后没过几年，明蕴镇的矿脉耗尽了，矿工们都转去层岩巨渊工作，明蕴镇也从此荒废，如今只有盗宝团会光顾此地。
黑岩厂
位于璃月南部的矿石加工厂，黄金屋南侧。曾因层岩巨渊关闭导致此矿场生产压力过大。在纷乱的烽燧染红大地的年岁，此地亦被诸部族的先民用作避难之所。存在某个覆灭古国的遗址。
层岩巨渊
璃月第二大矿区，盛产琉璃晶砂。同时也是岩王爷击败若陀龙王的古战场。曾因挖掘到天钉停工一段时间。
