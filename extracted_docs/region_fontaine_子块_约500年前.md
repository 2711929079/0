# 约500年前

城市优化随着枫丹城区的扩张，所有阴影纷纷被秩序之光驱逐。「心善」的贵族们将「肮脏」的平民「提供住所安置」在下水道区。下水道之所以能成为灰河很大程度上也得益于便携动力锯这种便利工具。灰河的居民相信，即使栖身于没有阳光与雨露的城下之城，经由绝不缺乏的勤奋与坚韧，他们或将获得居于地表的资本，或将能把自己栖身的「水之国」建设得如同地表上一般美丽。这种链锯并没有随着船坞长剑、勘探钻机等道具一同被禁止，因为在发条机关的早期生产作业中依然有着难以替代的作用。再后来在城外绝对安全的地方建设起了发条机关最大的工厂，能够熟练使用它的人就与它一同渐渐消失在了进步的洪流中。[32]久而久之，灰河的人们建立起自己的秩序，与枫丹官方达成微妙的平衡，直至「灰河整肃」到来。卡尔与埃尔纽艾尔卡尔是出身水仙十字院，与德怀特·拉斯克、贝瑟·埃尔顿以及埃马纽艾尔·吉约丹一同长大的孤儿。在幼时的勇者游戏中，卡尔常常扮演恶龙贾巴沃克，而埃马纽艾尔常常扮演讨伐恶龙的英雄。[33]长大后的埃马纽艾尔加入了逐影庭，以打击犯罪为工作，「白淞之围」正是在其参与指挥下进行的，而这场围剿的进行也代表着卡尔为之努力的谈判最终没能达成。悔恨于自己所为，失望于枫丹廷的领导，埃马纽艾尔离开了逐影猎人的队伍，从此自暴自弃借酒消愁。[33]没能促成和谈的卡尔则在另一侧同样经历了这场围剿，帕西法尔用「戏法」强行将其转移到漆黑而安全的地窖，只能听到隔着砖石土地闷闷地传来的喊声、破裂声与金属碰撞声。[33][34]贝瑟贝瑟·埃尔顿原为水仙十字院孤儿，长大后加入枫丹海军，成为海军司令，是枫丹海军「白色舰队」旗舰「斯彭西安」的舰长，浪影阔剑为其武器。「铁甲熔火帝皇」因为重甲蟹的习性，会随着成长不断寻找合适的螺和贝壳来当作甲胄，如果不阻止它，说不定它会无限生长下去。但是后来，贝瑟·埃尔顿击败了它，才终止了继续成长的进程[35]。主动退役后，贝瑟回到十字院接任副院长[36]，在抚养卡特期间烹饪水平很差，经过苦练在抚养雷内四人时已变得水平高超。[37][33]坎瑞亚灾变发生、十字院遣散前，贝瑟联系卡尔和埃马纽艾尔领养阿兰四人[38]，并将墨绿羽饰留给了雅各布。[39]自己则返回部队，重登战舰，对抗以「厄里那斯」为首的兽潮。「灰河整肃」出身灰河，以行骗、行窃为生出没于光荣剧院的小贼姐妹，为科培琉司戏中的游侠喜剧故事所感染，萌生了凭借魔术走上舞台、平等地为贫富众生带来表演的梦想。于是一人以戏中的蒙德侠盗「帕西法尔」为艺名，一人则扮演助手「约瑟芬」，两人的表演从爱德华多的酒馆中、灰河水渠旁，走进了光荣剧院里，帕西法尔的名字在地上城与地下城中都令人津津乐道[40]。但城市的“优化”是没有尽头的，在枫丹廷发起的灰河整肃行动中，扮演「帕西法尔」的少女死去了，「约瑟芬」赶回灰河，只见到了没能完全干涸的血。于是「约瑟芬」接下了「帕西法尔」的名字，并立志实现一个没有贵族和王者、没有生来的富贵和无法摆脱的贫困的世界，以作为属于她们的「大魔术」[40]枫丹高层因第二次城市「优化」对下水道「灰河」中生存的城市底层进行的武装镇压。枫丹廷的官僚决定「整理市容」，将无数人赶入地下的灰河。这批人中的雅各布的生父[41]爱德华多·贝克成为了灰河的首领，负责维持灰河秩序。然而不久后枫丹廷继续整肃灰河，已建立自己秩序与规则的人们自然不满而反抗，为首的爱德华多等人被捕，但其党羽将他救出，伙同秋分山匪徒占领白淞镇继续对抗枫丹廷，挟持了负责押送的执律庭成员，试图与枫丹廷谈判。记者卡尔·英戈德在其中斡旋试图促成和平谈判，但枫丹廷方面仍然选择了命令逐影猎人（包括埃马纽艾尔）进攻白淞镇，使得爱德华多、「帕西法尔」及雷内的生父[41]，与白淞镇的歹徒勾结的镇长雷诺·德·佩特莉可身死，反抗运动失败，史称「白淞之围」。阿兰、雷内、雅各布、玛丽安四人以犯罪者家属身份被投入水仙十字院。而「帕西法尔」妹妹约瑟芬决定以「帕西法尔」的身分在事后被捕（假冒帕西法尔）并在法庭上申请决斗，[33][42]审判庭派出传奇决斗代理人玛尔菲莎作为控方的代表参与比武审判。此时的卡尔已辞去记者的职务，作为探险者与荒野、废墟和遗迹为伍。数年后其回到十字院为成员们拍照时，透过镜头想起过去的种种经历，直到最后，没能作为记者在比武中记录帕西法尔最后一战的懊悔才将他的视野席卷——决斗的结果是帕西法尔身亡[33]。「帕西法尔」与「最初的大魔术」曾经的小贼姐妹受科培琉司大师的剧作的影响，以剧中侠盗「帕西法尔」和蓝宝石美女「约瑟芬」为名，将盗窃的手法应用在魔术表演上，最终成为了广受枫丹上下层欢迎的知名魔术师。但是枫丹上下层的矛盾难以调和，最终在「灰河整肃」中，「帕西法尔」身亡，「约瑟芬」继承了「帕西法尔」的名号，并设计出了属于她们的「最初的大魔术」。「我是大魔术师——『伟大者』帕西法尔！」「你们将要目睹的是不曾想象的梦幻世界！」「没有贵族没有王，没有指着你们的刀剑！」「没有生来的富贵，没有无法摆脱的贫困！」[40]「灰河渡手」与爱德华多在过去曾经被当作武器使用的趁手废弃铜管，曾经在一些人当中享有盛名或是臭名昭著。[41]雅各布的生父爱德华多•贝克与因城市优化而被迫进入下水道「灰河」生活的其他人们没有什么不同最初他只是拿着捡到的铜管驱逐鳄鱼，再后来是保护商贩店家免于强人帮派的侵扰勒索、调解灰河的「同胞」之间的纷争，为新婚者祝福。[41]除此之外，爱德华多在灰河经营酒馆，「帕西法尔」姐妹金盆洗手后的魔术表演就在他家的酒馆出道表演。[40]就这样，爱德华多顺理成章的成为了「灰河」实际上的领袖，团结领导着着枫丹的灰色地带（刺玫会的前身），他惯用的铜管被众多朋友亲切的称为「渡手」[41]。最终爱德华多因在「灰河整肃」中组织反抗被捕，流放往沙漠。蕾蒂西娅尽管贵族们将灰河视为肮脏之处，仍然有许多贵族对该地抱有好奇与向往。出生便不愁吃穿的大小姐蕾蒂西娅·兰道夫因为向往冒险，以及为了拥有和好友聚会聊天的话题来到灰河。一潜入该地就因为过于瞩目即将遭遇危机—所幸她先前认识的里德为她引路。她在灰河没有想回去的念头，和里德有了孩子。灰河整肃爆发时，蕾蒂西娅帮助人们对抗枫丹官方，失败后她、里德和孩子仍然被父亲接纳。然而事后在兰道夫家，她才回想起逐影猎人们潜入的暗道不应被外人知晓，也就是出了内鬼，而此人就是她丈夫里德，他将秘密泄露给兰道夫。事前兩人相愛時，里德曾對蕾蒂西婭說過：「蕾蒂西娅，我敬爱你高洁的灵魂。」「我们将不再会是砍向树木的斧柄。」「若有一日，我竟耽于尘世的荣华，」「那时，就由你来裁决我的命运吧…」于是，蕾蒂西娅舉起她的大斧，对丈夫做出最终的裁断[43]。兰道夫家族存有后人[44]。白色舰队过去的辉煌年代，纯白色的铁甲旗舰「斯彭西安」曾是枫丹海军的骄傲。那个时代里，装备着庞大重炮与坚固冲角的铁甲舰队乃是枫丹廷的宠儿，在光荣剧院的时代，观众无不为白色舰队清扫魔像军团的故事起立鼓掌。坎瑞亚战争坎瑞亚战争爆发，六神听从天理的号召前往坎瑞亚对抗深渊。厄歌莉娅恍惚间意识到这一去很可能就是永别，因为只要身为原始胎海心脏的自己一天不死，预言就一天不会发生。但天理不可与之为敌，厄歌莉娅接受了自己的命运。临行前，厄歌莉娅找到芙卡洛斯，将水神的职责与拯救枫丹的任务一起继承给这个自己最信任的纯水眷属。厄歌莉娅最终死在苍漠囿土，化作甘露之海，孕育出花灵一族。她死之后，没了心脏的原始胎海开始躁动，枫丹迎来第一波涨水期。这让枫丹人们都想起了那条预言：所有人都会溶解，枫丹被水淹没，水神自己在神座上独自哭泣。芙卡洛斯潜入海底安静地思考了很久，直到身边的蚌壳都开始冒泡泡了，才想出了那个唯一的答案。命运是既定的，越是反抗反而陷得越深，可命运与现实之间有一条小小的裂缝，借此我们便可以欺骗命运，或者说——欺骗天理。芙卡洛斯将自己的「神格」从身体与精神之中分离出来，取名为芙宁娜。随后，芙卡洛斯诅咒了芙宁娜。她虽有水神之名，却无水神之实。她虽能喜怒哀乐，却不能这样做。她将被众人仰视，但却没有哪怕一丝目光是真的向她投去。她甚至没有死亡的自由。她将被迫以这歌剧院为舞台，成为歌剧院真正的女主角，为了让预言看似实现而永远扮演水神的戏份。芙卡洛斯又选择了那维莱特，一个对自身为何以人形降临毫无所知的水龙王。拯救枫丹人民说白了就是替换血液里的原始胎海，想必能做到这一点的，只会是原始胎海的真正心脏——掌握古龙大权的完全体元素龙王。法涅斯窃取了古龙的力量，使其在一任任元素执政间流转。所以想要归还水神的权柄，就必须杀死「水神」，或者说摧毁水神的神座。于是，芙卡洛斯创造了谕示裁定枢机，自己带着神之心与它合二为一，默默收集民众的信仰[45]。第一次涨水期厄里那斯「黄金」的「孩子」因为贪玩瞒着大家来到了地面上的王国，它并不知道自己的存在对于地上王国而言是一种怎么样的灾厄。它见识到了这缤纷多彩的世界，为了不让这世界被“我（厄里那斯）”破坏掉，心甘情愿地死在了枫丹。（实际上只是肉体死亡，精神仍存活，并能与后来在其体内诞生的美露辛对话[46]）水仙十字院解体此时枫丹经历了「第一次涨水期」，水仙十字院在此时被淹没。没有什么是永恒的，水仙十字院迎来了属于它的结局。在深渊兽潮来临、十字院被迫遣散前，时任副院长贝瑟将孩子们托付给信赖的人收养。贝瑟在十字院淹没前将四名孩子分别托付给卡尔和埃马纽艾尔。至此，埃马纽艾尔才不情愿地再度尝试与人一同生活，尝试为了儿女创造一个安全的世界。于此同时，自然哲学院的学院长德怀特·拉斯克下令封闭学院，以免血气方刚的青年外出送死。海军对抗厄里那斯时，枫丹海军丧失大部分的舰队。海军司令贝瑟·埃尔顿主动退役，尔顿的大副纳撒尼尔·皮克曼最终失去了归属，余年只得彷徨在英雄虚名中。皮克曼曾经奔走多年，企图说服枫丹廷重启「斯彭西安」铁甲舰的打捞与重建。但努力无果而终，意欲重建枫丹海军的想法也被战殁者遗属以为是在沽名钓誉…[47]「斯彭西安」则一半留于「厄里那斯」腹中，一半散落在海里[48]。活下來的皮克曼其實是個逃兵，因为最终贝瑟的指令要求大副和轮机长随自己留下，其余人弃船撤离[48]。剧院曾经光荣剧院十分辉煌，枫丹人会在此地欣赏舞台剧等演出。后来第一次涨水期时，光荣剧院被淹没，连带淹没了许多其他建筑。光荣剧院毁灭后，在伊黎耶岛新修建了欧庇克莱歌剧院。「斯彭西安」与「白色舰队」的诸多传奇在光荣剧院毁灭之后的一段时间内，曾是在伊黎耶岛新修建的欧庇克莱歌剧院中热门的剧目，直到发生演出意外被禁演。其中最令人难忘的是舰队的最后一战：舞台上的十门舰炮所演绎的壮烈战斗，细节还原自贝瑟·埃尔顿麾下大副的记录。不知是究竟是皮克曼认为它不值得写下，正如他也从不谈论那次荒唐的行动，抑或是编剧认为这一细节与预期的悲壮史诗不合而将其从剧本中善意地略去：贝瑟·埃尔顿在时隔数年再度登舰面对着皮克曼等一众列队乘员时，低语道：「看来，这次终于可以对敌人开火了。」随后才如歌剧院中的演绎，笑着高声说：「小子们、姑娘们！看样子没有我你们真的不行啊，但叙旧的话晚点再说！」「让我们先把那个不知天高地厚的大块头，轰到大瀑布下边去！」[47]冰风组曲科培琉司是自然哲学院成员之一，后成为剧作家。[34]发条机关「冰风组曲」的制作灵感来自曾在剧院仅上演一次，仅演出过前几幕的剧作《发条的歌裴莉娅》，讲述的是发明家倾心于拥有所谓「人心」的发条机关歌裴莉娅的故事。该剧在上演前备受评论界与文艺界的期待，但在初次上演的第一次幕间休息时在世界各地刻下深重伤痕的坎瑞亚灾难发生了。它所留下的故事只剩下依据当时有幸光临剧院的观众拼凑出的前几幕台词，以及残垣断壁与过期报刊上留下的广告。《发条的歌裴莉娅》的剧本并未公开、出版过。当日光荣剧院的幸存者当中，有部分观众依据记忆转写了幕间休息前，部分演出的情节与台本。[49]征求科培琉司先生遗族与上述人士的同意后，敝社将相关资料整理、综合，力求还原科培琉司先生原作的一角。虽说不过是管中窥豹，但求能抛砖引玉。那是很多年以前的事情，在当时欧庇克莱歌剧院尚未开始修建，而那间剧场也没有在被毁后被人买下并改建，并在无数次的易主后成为如今的德波大饭店。[50]白淞镇在第一次涨水期前，白淞镇曾是枫丹航运的中心枢纽之一，因为涨水期导致的地形的突然变化导致船运无法继续从此处通航。[51]纯水精灵枫丹的水体被深渊污染，不再适合纯水精灵的生存。大批纯水精灵离开枫丹追寻厄歌莉娅的葬身地。它们有的向南进入须弥大赤沙海，最远出现在蒙德清泉镇。芙宁娜为了找到除了扮演水神外解除预言的方法，差遣纯水精灵到各国成为间谍，缔造间谍情报网，来探查各国有没有解决类似危机的方式，但为了避免恐慌没有告知纯水精灵自己缔造间谍网的目的。因前代水神差遣纯水精灵到各国是为了散播自己的恩泽，而如今水神居然想缔造纯水精灵间谍网，大部分纯水精灵不接受其理念，拒绝效忠新的水之神，间谍网計劃失敗[52]。之后枫丹廷派遣浊水精灵前往刺杀叛逃的纯水精灵洛蒂娅。（在「纯水之愿」中洛蒂娅对间谍网的计划极度反感，因此也有可能浊水精灵是要寻回她们，但被偏激的洛蒂娅误以为是来刺杀自己。毕竟浊水精灵去刺杀纯水精灵会水元素免疫。）[29]之后浊水的危机解除，但枫丹仍再也没有出现纯净的纯水精灵。[53]隙境原体隙境原体是巨兽厄里纳斯死后其细胞演变成的魔物。据与「护国的白骑士」同行的逐影猎人的记述，在厄里那斯的惨烈战斗结束后，曾经出现了大量的隙境原体。这些隙境原体并没有对他们采取任何攻击行为。[54]
城市优化随着枫丹城区的扩张，所有阴影纷纷被秩序之光驱逐。「心善」的贵族们将「肮脏」的平民「提供住所安置」在下水道区。下水道之所以能成为灰河很大程度上也得益于便携动力锯这种便利工具。灰河的居民相信，即使栖身于没有阳光与雨露的城下之城，经由绝不缺乏的勤奋与坚韧，他们或将获得居于地表的资本，或将能把自己栖身的「水之国」建设得如同地表上一般美丽。这种链锯并没有随着船坞长剑、勘探钻机等道具一同被禁止，因为在发条机关的早期生产作业中依然有着难以替代的作用。再后来在城外绝对安全的地方建设起了发条机关最大的工厂，能够熟练使用它的人就与它一同渐渐消失在了进步的洪流中。[32]久而久之，灰河的人们建立起自己的秩序，与枫丹官方达成微妙的平衡，直至「灰河整肃」到来。
