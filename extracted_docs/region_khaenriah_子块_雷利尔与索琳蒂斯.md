# 雷利尔与索琳蒂斯

与其他罪人不同，成为罪人前雷利尔的生活条件与普通人相当。曾在黑王伊耳明麾下充当特务，清剿、追猎赤月王朝遗民，对此工作一直存在心理障碍，无法无情的清算那些人，但为了生活被迫这么做。女友为索琳蒂斯，俩人相处甜蜜，雷利尔向她求婚，而索琳蒂斯因为还在考虑所以没有正式接受，所以不算未婚妻。当雷利尔与戴因斯雷布聚会时，在场的维瑟弗尼尔预知到雷利尔的婚姻将不幸，但没开口，却被雷利尔察觉。雷利尔隐瞒自己的工作，希望索琳蒂斯不要知道自己是杀手。而因当今国王在清剿赤月遗民索琳蒂斯也隐瞒自己的血脉是赤月王朝遗民的旁系。因为雷利尔清剿赤月遗民被称为「猎月人」。赤月王朝遗民查清了猎月人的身分，由赤月遗民乌洛告知索琳蒂斯，索琳蒂斯大为震撼。而昏聩的君王接触深渊后几乎陷入疯狂，下令屠杀越来越多赤月遗民，雷利尔心里不能接受但还是继续工作。伊尔明屠虐的目标，由赤月遗民扩大成赤月遗民旁系与相关人士，但大部分赤月遗民其实都对王朝没有威胁，很多都是孩子与老弱妇孺···但伊尔明执意清除所有「威胁」，如今猎杀名单已不限于赤月遗民和赤月遗民旁系与相关人士，雷利尔认为他打算把这地下的人全屠光才满意。他请求维瑟告诉他上次见面时维瑟预知到什么，在多次请求后维斯如实告诉他。如同预言般有天猎杀名单上，出现赤月遗民旁系的索琳蒂斯。另一方面索琳蒂斯是深秘院的研究员，疑似研究如何改变血统不成功。她想要阻止君王的暴行，甚至想要杀死造成这一切的深渊的源头旅行者的血亲。其研究虹月月髓，发现月矩力的力量能开启月之门通往过去。乌洛告诉索琳蒂斯雷利尔的身分，期望她能杀掉雷利尔，但被索琳蒂斯拒绝，她虽然不知道雷利尔最终会不会甚至杀了自己，但她绝对不能对雷利尔动手。后乌洛被雷利尔肃清。在肃清对象波及赤月遗民旁系后，索琳蒂斯将所有薪水都给与赤月王朝其他遗民一同躲起来的妹妹。他决定要拯救索琳蒂斯。在猎杀名单内看到索琳蒂斯后，雷利尔决定不能这样再继续下去，于是偷偷窝藏赤月遗民，包括索琳蒂斯妹妹，虽然他清楚自己这样迟早会被皇上发现并处决。雷利尔持续受到伪装成维瑟声音的深渊的蛊惑。某天维瑟被关入大牢，戴因和其他五人前往劫狱，当中就包括雷利尔，但他这么做是要问清维瑟怎么避免他与女友的未来，否则杀了他。并且他质问他在脑海里和他说话的目的，维瑟察觉那其实是来自深渊的声音。放出维瑟后，与戴因等人交战的坎瑞亚王使用深渊，坎瑞亚灾变发生。雷利尔赶紧去深秘院救索琳蒂斯，索琳蒂斯看到雷利尔时以为是来杀自己的，也受到深渊蛊惑的她为了不要再继续这样悲痛的活着，活在杀死恋人的阴影中，于是用月矩力开启月之门，打算回到过去杀死引入深渊力量的关键，使一切无法挽回的的旅行者的血亲，結果卻被撕裂成碎片，只余灵魂在尼伯龙根的囚笼内，变成深渊要挟雷利尔的筹码[41]。雷利尔崩溃，深渊蛊惑他雷利尔没资格拯救他，想要拯救她就得更强大的力量与意识。于是雷利尔前往瓜分深渊力量，成为最后诞生的罪人，取得力量后跳入月之门，打算找回索琳蒂斯。掉入异空间后雷利尔身体被破坏成数千片，落入各个时空中。狂猎在猎月人身躯被破坏成千块后因猎月人的残骸作用下演变成深渊灾害。猎月人因不死诅咒得以保命，后来猎月人一直收集自己身体碎片以求以完整力量重新降临以拯救索琳蒂斯[32][8][40]。魔神任务不存在的国土中雷利尔的回忆里参杂了索琳蒂斯的记忆，剧情里还引起奈芙尔起疑，因索琳蒂斯可能也变成狂猎碎片，不小心被雷利尔吸收到，所以雷利尔碎片中也有索琳蒂斯碎片。而索琳蒂斯可能被降下不算诅咒也得以已身为碎片的方式保命，不存在的国土中戴因也提及这个可能性。只不过没有像雷利尔一样搜集其他碎片。雪浪与苍林之舞剧情里说狂猎原本是悲鸣的女性诡影，后来在猎月人身躯被破坏成千块后因猎月人的残骸作用下演变成深渊灾害，此呜咽声大概就是索琳蒂斯。
灾厄五百年前，坎瑞亚决定引入深渊力量作为新的能源。但深渊能量失控，天理只得号召七神前往覆灭坎瑞亚。受深渊影响，伊尔明开始动摇世界的根基。「预言家」维瑟弗尼尔献上预言，却因预言内容而被伊尔明囚禁于大牢，并被刺瞎双眼，暂时丧失预言能力。戴因联合「贤者」海洛塔帝、「黄金」莱茵多特、「极恶骑」苏尔特洛奇、「猎月人」雷利尔劫狱救出戴因的哥哥维瑟弗尼尔，并在之后临时起意阻拦「黑王」伊尔明继续动摇世界的根基[33]。伊尔明似乎因深渊力量肉体产生畸变[42]。然而除了戴因之外的五人受到了类似的理由深渊蛊惑与放大欲望，瓜分了那些足以毁灭世界的力量，成为了超然的存在，每个人掌控的力量都能与世界相匹敌。在此之前，五人就一直持续受到深渊的诱惑[40]。被称为「黄金」的炼金士堕落为了罪人，释放自己孕育的大量漆黑魔兽。它们流淌着与此世不容的黑血，在大陆上跳踏众生、破坏一切。它们的生命是变质的，由世界外的力量赐予的。统治地下世界坎瑞亚王国的黑日王朝覆亡，灾难突破了古国城墙在大陆上散布开来。在天地间留下伤痕累累…战火飘熄，余烬落定。灾厄发生时，五大罪人没有一人出面阻止这场灾难。整个大陆都经历了一个混乱的时代。黑暗的力量扩散行进，污染所遇到的一切。那段时间里，大地上蛮族滋生、魔兽肆虐。人们的生活空间被压缩到了城墙之内，野外充满危险。以「末光之剑」戴因斯雷布为首的宫廷卫队以及白鹄骑士积极参与对抗漆黑魔兽的战争，但还是没能让坎瑞亚免于灭亡的命运。[9]后来，末代君主伊尔明失能，安弗塔斯摄政领导「救国会议」[43]，暂时统领坎瑞亚遗民。[38]被黑日推翻的赤月王朝遗民，在此时对黑日展开复仇。[44]
