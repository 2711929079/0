# 复甦

此前魔神混战干年，桑田尽毁。璃月之民遂以商贾、巧工为业。其佼佼者联合一体，名之「七星」。故璃月港雏形既成。七星下辖又有八门，内外诸业无所不包。又有千岩之军听令七星，内抚民众，外镇妖魔。是所谓岩王帝君之治。[30]追随岩王帝君者自成部队，以「岩君」、「璃月」为号，称「千岩军」，听令于七星，护卫璃月。后璃月之经济逐年屡升，成七国商贾之心，得誉「黄金的心脏」。
七星璃月正式设立“七星八门”，甘雨主动成为辅佐七星工作的秘书，并获得神之眼。千岩团改名千岩军，由七星管理。七星分别为：“天权星”、“天枢星”、“天璇星”、“天现星”“玉衡星”、“开阳星”、“瑶光星”。八门分别为：“和记厅”、“银原厅”、“辉山厅”、“盛露厅”、“流金厅”。璃月开始每年举办名为“七星请仙典仪”的迎神仪式。七星作为璃月港凡民与岩神之间的媒介，负责与岩神直接沟通，并以简明易懂的方式向市民传达神谕[103]。七星的特务开始经营“望舒客栈”。
