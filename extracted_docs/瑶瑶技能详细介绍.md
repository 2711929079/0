# 瑶瑶技能详细介绍

瑶瑶是《原神》中的草元素四星角色，须弥的采药少女，称号"仙蕊玲珑"。
元素战技「采药之术」：
- 释放草元素能量，对周围敌人造成草元素伤害并恢复队友生命值。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「仙蕊」：
- 创造一个持续8秒的草元素领域，领域内的队友会持续恢复生命值。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "采药的祝福"：采药之术命中敌人后，有15%概率触发"激化"效果。
- "草元素增幅"：瑶瑶的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "仙蕊的守护"：元素爆发持续时间内，队伍中的角色获得"草元素伤害提升10%"的加成。
