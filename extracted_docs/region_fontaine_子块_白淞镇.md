# 白淞镇

坐落于秋分山下的宁静小镇，镇民多以渔业与打捞业为生。据说在第一次涨水期前，这里曾是枫丹航运的中心枢纽之一。尽管地形的突然变化导致船运无法继续从此处通航，但小镇如今温馨安逸的景象，依然会让每一位访客心生向往。
