# 执政

执政
天上永恒的王座到来，世界为之焕然一新。然后真王，原初的那一位开始和旧世界的主人们，七位恐怖大王开战。那恐怖的大王们是龙。原初的那一位造出了自己发着光的影子。而影子的数量是四。四位执政不喜欢被任何生命提及名讳，无论普通的人类还是尘世七执政，因为影子的身份才最适合他们。其权能能够部分分予魔神。分为「生」、「死」、「空」、「时」四位执政，游戏中活动页面的指针是时之执政，纪行页面的翅膀是死之执政，祈愿页面的菱形是代表空之执政的传送门，千星奇域的奇域页面的菱形也是代表空之执政的传送门。四影构想来自已《所罗门的小钥匙》的72柱神为发想形成的日本轻小说设定「至上四柱」。
目前已知的有：「生之执政」纳贝里士
天理投下的四个影子之一，掌握着创生的力量，创造了飞鸟与人类。被其余影子称为「纳贝」、「纳贝小姐」。若生之执政外的存在创造人类，即会违反天空岛的大忌遭到神罚。圣遗物的「生之花」为其象征。原初的水之龙被击败后，由其创造了原初大海的另一颗心脏，即初代水神厄歌莉娅[117]。圣遗物「纳贝里士之心」由莱茵多特母子在世界深处寻获[118]，随后莱茵多特失踪，吞噬纳贝里士与其融为一体[103]。莱因多特吞噬纳贝里士后，纳贝里士失去对身体的主导权，但仍可以在脑海中发话。当莱因多特出现不敬与违背天理的发言时，会被纳贝里士驳回。而莱因多特也会予以尊重。目前二人呈「一体双魂」状态。黄金占据主导权时，头饰与背后螺旋状DNA光环颜色为金黄色，纳贝里士则为蓝色[119]。眼珠图案是人類的X与Y染色体。是异瞳，左眼为金色，右眼为蓝色。曾经的纳贝里士戴眼镜。背后的双螺旋中央的花是生之花。纳贝留士会以屏幕对画框的形式出现并发言。肩上有凯尔特三角结图案。在霜月之子的祷歌集中被称为「赐生之母·玛特拉赫卡」[120]。
「死之执政」若娜瓦
天理投下的四个影子之一，掌握着死亡的力量。不关心死亡的时间与形式，但会确保「死亡」的发生。甚至能够宣告「空间」的死亡，施展力量时身上的眼睛会发红光。引导夜神整合纳塔残存地脉，建立夜神之国；帮助纳塔建立了「还魂诗」等一系列规则，在希巴拉克的进一步请求下于火神神座中留下了力量。其帮助纳塔的一系列行为让天理不满，此后死之执政便自暴自弃，不再追究别人谈论其名字。背后有双翼，稻妻的千手百眼神像翅膀即为死之执政的造型。背后的翅膀与护摩之杖的相同，而护摩之杖也是用来超度死者的祭仪仪仗。腰间有凯尔特三角结图案。在霜月之子的祷歌集中被称为「冥府之母·图奥内塔尔」[121]。
「时之执政」伊斯塔露
天理投下的四个影子之一，掌握着时间的力量。她从很久以前便开始记录这个世界，将所有人、所有事、所有的相遇与别离都刻印在自己的记忆中。她对栖息在尘世的生灵一视同仁，同时这些生灵在她心中也是同等的无足轻重。在霜月之子的祷歌集中被称为「时序之母·拉伊梅莱阿」，其祷歌中提到了编写年代尚未出现的事物[122]。裙摆上有凯尔特三角结图案。
「空之执政」阿斯莫代
天理投下的四个影子之一，掌握着空间的力量。力量的象征是红色方块。四位执政中，只有其身上没有凯尔特三角结图案。双子所遇到的陌生神明[123]，自称天理的维系者，在动画中称旅行者为人之子。现已失踪去哪了呢好难猜啊，不再出席四影会议，伊斯塔露推测其已叛变，若娜瓦称若阿斯莫代叛变，那么她一定爱新的主人胜过爱自己[119]。旅行者介绍中写道「维系者正在死去」。角色形象、中日双语配音都与《崩坏3》的空之律者（女王）一致。《原神》的「再临测试」前瞻直播中，包括刘伟在内的两位主持人都暗示「陌生的神灵」与米哈游前作角色「似曾相识」，「有劳原学家挖掘」。而两者一致之处即「空间」的权能。因此被部分玩家戏称这位神灵为女王再就业。在霜月之子的祷歌集中缺失对应篇目[124]。
