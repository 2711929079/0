# 瑶瑶

香菱向萍姥姥学习枪法。出生在轻策庄的瑶瑶被萍姥姥看中，瑶瑶的父母在被劝说后，最终同意瑶瑶跟随萍姥姥去往璃月港生活。年幼的瑶瑶字还没识全就拜入萍姥姥门下，成为香菱的小师妹。萍姥姥教瑶瑶生活技能、诗书礼仪以及枪法。留云借风真君还制作了“月桂”作为防身之物，将其送给了瑶瑶[192][193][194][195]。
灵渊灵渊不忍两位友人浮锦和药君的生存环境随着人类对沉玉谷的开发而逐渐偏离自然法则，抱着即使被热爱人类的浮锦谴责也没关系的心态暗地重塑沉玉谷的地脉，慢慢地将沉玉谷改造回最原始也最适合三仙居住的环境。随着灵渊改造计画的实施，沉玉谷出现了大范围的氲气，近年来翘英庄的茶叶质量也开始下降。浮锦在锦落庭察觉了地脉异常，但力量不完全的她无能为力。[24]
