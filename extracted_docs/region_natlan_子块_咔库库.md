# 咔库库

绒翼龙幼龙，伊法的龙伙伴，也是助手、龙医生，负责和龙沟通，担当翻译。魂附龙后与咔库库对话可知其完全理解人类的语言，只是碍于龙的发声器官而只能说出那几句话。内心里把伊法、欧洛伦和旅行者当作哥们，阿乔则是半个哥们。
