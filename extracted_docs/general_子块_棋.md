# 棋

在《雪霁逢椿》PV和神里绫人传说任务中，都出现过一种特殊的棋，这种棋是来自日本的本将棋，其棋子的表面呈现五边形，以其前方尖端指向表明所属，在日本极为流行。该棋使用9×9的棋盘，每方共有20枚棋子。该棋的王称为“玉将”或“王将”走法和国际象棋的王一致，不能升级，棋子中存在行动较为自由的飞车走法和国际象棋、中国象棋的车一致，升级后称“龙王”，走法为国际象棋车+王和角行走法和国际象棋的象一致，升级后称“龙马”，走法为国际象棋象+王，而更多的是行动缓慢的金将走法为中国象棋的将加不能后退的士，不能升级、银将走法为中国象棋的士加未过河兵，升级后称“成银”，走法和金将一致以及不能后退的桂马基本走法与国际象棋的马一致，也没有蹩腿规则，但是只能向前走2步再向左/右1步，升级后称“成桂”，走法和金将一致、香车只能向前，步数不限，升级后称“成香”，走法和金将一致以及步兵走法和中国象棋未过河兵一致，升级后称“と金”，走法也和金将一致。这种棋最大的特点是绝大多数棋子都可以升级将棋子翻面，称为“成”，此操作不可逆，条件是起点和终点至少有在一个对方初始摆放棋子的三行内，如果该棋子移动后在空棋盘上已经无法移动，则必须升级，且拥有世界其他杀王棋都不具有的特殊规则——“打入”，即花1步棋将一个已经被自己吃掉的棋子按照一定的规则重返棋盘打入的棋子必须是非升级形态，一条纵线上最多只能有一个未升级步兵；不能通过打入步兵的方式将死对手（非步兵棋子不受此规则影响）；打入的棋子在空棋盘上必须可以移动。神里绫华在PV中的“王手”，其意思和中国象棋的“将军”以及国际象棋的“Check”差不多。
