# 虚空系统

虚空系统是汇集了知识的宝库。
其原型阿卡西记录出自印度史诗《摩可婆罗多》，该学说认为：宇宙中存在一种名为阿卡西记录的领域，历史上的天才（如老子、特斯拉）通过各种方法与阿卡西记录相连接，从而获得异于常人的智慧与知识，《原神》中虚空系统亦有此功效。
