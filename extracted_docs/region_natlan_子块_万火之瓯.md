# 万火之瓯

圣火竞技场的坐落之处。「龙遗物研究协会」的成员一度进行地形勘定后，断定此处的盆地绝非自然形成，而是许多年前巨物坠落至此所致。可那所谓的巨物究竟是什么东西，却没有人能得出确凿的结论。如果古老的圣火能开口，可能就可以告诉他们答案吧，但是圣火碑中的炽焰从来只是自顾自燃烧，从未回答任何人的疑问。
