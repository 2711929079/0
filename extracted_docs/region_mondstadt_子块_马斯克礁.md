# 马斯克礁

誓言岬东边的岛礁群，深境螺旋的所在地。前身为名叫「尖帽子峰」的雪山。在古时，尖帽子峰比天空更高，冒险家莱纳德曾梦想登上去山顶，再失败后，他转而研发能上山的道具，包括风之翼。风神修改地形后其变成礁岩。
