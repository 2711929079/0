# 20年前～30年前

芭别尔二三十年前，某学者带着其怀孕妻子来沙漠做研究。前往朝圣之路的路上，雇佣了一个佣兵保护他们，佣兵对这俩人颇有微词。芭别尔这个名字是母亲取的。结果学者一直没有回来，其妻子也因为难产而大出血。这佣兵「心善」见不得她这么痛苦，就送对方上路了。母亲因产道撕裂而逝世。学者勉强幸存了下来，这佣兵也是狠，担心对方交不出佣金，想用对方女儿最后讹一笔。没谈拢，于是学者也被佣兵送上路，和爱人团聚了。佣兵看上了幸存的芭别尔，将她养大，把她教导成心狠手辣的阴险女人[89]。很多年后，芭别尔成为了塔尼特部落的首领。迪希雅从前沙漠有一支名叫“神王之遗”的大型佣兵团，由许多的附属团构成，臭名昭著、坏事做尽，组织内部采取一套严酷的管理制度，加入的成员都必须要留下案底，从而便于威胁与掌控，无法轻易脱离。迪希雅的父亲库塞拉曾是神王之遗下属的“阿赫玛尔之须”的团长，但他并不愿意和总部同流合污，从不做抢劫、走私、谋杀这些非法勾当，更从来都不让女儿接触镀金旅团的黑暗规则。在迪希雅因为误解了父亲而离家出走后，库塞拉一直暗中派人保护她。后来迪希雅成长为了能够独当一面的强大佣兵，并建立了“炽光之狮”佣兵团。后来风烛残年的库塞拉决定独自前去神王之遗的总部，用一把火烧光了阿赫玛尔之须所有成员的案底，自己也死在了大火中。阿赫玛尔之须的成员虽然从此自由了，但他们都发誓要为团长报仇，在后来几年里一直都在与神王之遗对抗[90]。
