# 优菈

出身于劳伦斯家族的优菈不顾家族的反对，成为千年以来唯一一位加入西风骑士团的劳伦斯家族成员，与劳伦斯家决裂成为骑士后，其从安柏的祖父那学会了吹奏骨笛，找到了自我的意义并获得了神之眼。优菈受琴邀请加入西风骑士团，任职游击小队队长。优菈在一次追踪深渊教团危险品的行动中，于荆夫港解救来自璃月的烟绯[124][125][126]。
