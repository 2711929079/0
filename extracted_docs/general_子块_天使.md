# 天使

天使爱上了一位外来者，并在三月女神的见证下立下结合的誓约[36]。仅三十日后，灾祸发生，天使和外来者在崩裂的天地间逃亡。后来由于天使为使用禁忌知识的人类求情，天空降下惩罚，让天使散失形体成为仙灵[41]（既然天使在经历天崩地裂的灾祸后就被惩罚，那么这个“灾祸”（仙灵之乱）指的大概率就是葬火之战。）
仙灵退化自外而来者带来了战争，亦为人类带来突破桎梏的妄想…而天的主人恐惧妄想与突破，降下修补大地的天钉，毁灭了凡人的王国…仙灵一族因为与外来者相爱惨遭遗弃，天理放逐了仙灵一族。仙灵一族失去了与天空的联系，也失去了教化的能力，从此背负可怕的诅咒，退化成只能凭借本能引导人类寻找宝藏的空壳。[41]
