# 人偶

雷电影创造的人偶的创造技术是来自坎瑞亚机械工程学，由地脉树枝打造[66][41]。
拆散旅行者的血亲曾是坎瑞亚的公主/王子殿下，坎瑞亚战争发生时，她/他前往唤醒旅行者，打算两人一同离开提瓦特。然而，一位自称“天理的维系者”的神明阻拦，让双子被迫分别…[67]之后，旅行者的血亲从昏迷中率先醒来，帮助兰那罗度过漆黑灾厄。[7]之后她前往甘露花海，遇见戴因斯雷布。
当时戴因斯雷布半身已形同魔物，正在对抗漆黑兽潮。之后，旅行者血亲跟着戴因离开，踏上寻找旅行者的旅程[51]。
