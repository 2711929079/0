# 嘉良

十年前，嘉良患上了罕见的心病，白术的师父本就时日无多，还继续折损寿命来医治嘉及，江蓠不忍，带着嘉良远离师门，避而不见。为拯救嘉良的生命，江蓠利用“螭”的魔神残渣炼就了一味毒，成功保住了嘉良的生命，但代价是心智和记忆都会被魔神残渣慢慢侵蚀。嘉良和江蓠约定，等到二人的孩子“阿雩”长大，就让自己上路。[77]
嘉明嘉明的父亲是一位普通的商人，做茶叶生意，希望嘉明能继承家业。嘉明想要从事小时候最爱看的舞兽戏，但父亲认为这根本不是事业，二人意见不合但有母亲在期间做调和。直到母亲因病去世，嘉明练舞兽戏受伤，父子矛盾彻底爆发，嘉明离家出走。嘉明的舞兽戏推广事业并不顺利，为了养活自己，他在一次机遇巧合下成为了一名镖师。期间，嘉明结识了一只小猊兽，并为其取名“文仔”，还成立了“威水舞兽队”，自己做话事人[184][185][186][187]。
