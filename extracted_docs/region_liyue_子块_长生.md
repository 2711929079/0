# 长生

几百年前的璃月曾瘟疫肆虐，一名药师为了救济世人而来到沉玉谷寻找传说中的药君，但是此时的药君早己因耗尽力量而变成了一条虚弱的白蛇，她也失去了一些曾经的记忆，化名长生。长生告诉药师只要与自己签订契约，就能获得医治百病的术法：珥蛇拖龙法，但代价是会变得短寿，药师答应与长生建立契约，而与药师分享生机的长生得以存活。借助长生所授的救病之法，药师救下了许多患者，但自己也因过度透支生机而英年早逝。在临终前，药师长生与契约托付给一位自愿继承的病患，这位继承者亦如同药师生前那般悬壶济世。在他死后，长生也先后传给了许多心性至纯的医者，他们无不救济苍生，却又无不英年早逝[77]。
