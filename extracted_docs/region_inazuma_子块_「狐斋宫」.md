# 「狐斋宫」

「狐斋宫」原本是白辰的血脉，曾与雷神同行,守护稻妻的子民。在这一生中，我也曾化为人的形姿，与这些短寿而美丽的小小生灵同行，以不同的身份，成为许多人的挚友（文本提及浅濑响、御舆长正、惟神晴之介、弘嗣、长野原一族）。守护他们的结界，但愿不会被任何黑暗侵蚀…这一切，所有的所有，都是我如今所怀念的「所以啊，撕咬我的漆黑意志，」「现在我已经失去了所有力量，」「我的白辰之血就任您挥洒吧。」「但是，尽管处于卑微的立场，」「我仍希望您能聆听我的请求…」「如果您能看见我所珍重之物，」「那么就请您饶恕那些生灵吧。」「如果您恩准我提出一个愿望，」「就请您将我永远明亮的记忆，」「归还给我热爱的这片土地吧。」「希望以此，在您的肆虐之后，」「仍然有美好的东西能留下来…」[40]在五百年前的灾厄中,「狐斋宫」为了保护鸣神岛，而与漆黑的灾厄作战，最终被灾厄吞没了。吞食她的灾厄最终被斩落，而她的思念与记忆也回到了鸣神岛的大地中。[1]狐斋宫死后，力量更强的「天狐」与「地狐」都化作石像等待她的回归，而这些狐狸则随着一族的灵脉变得稀薄，而不再开口，也失去了法力。狐斋宫死后，八重神子接手了「宫司」一职[12]。浅濑响拿着狐斋宫生前用的烟管，失去之人烙印在梦中。惟神晴之介远走他国，制作训练对抗妖邪阴阳寮，并自己亲手打造了一个烟管，思念着某个友人。
御舆千代「虎千代」背上「雷之三重巴」的纹样时名为「千代」的鬼族女武者，面对漆黑的军势时，曾被虎躯蛇尾的世外之兽吞下。最后她撕开魔兽的胸腔，得以幸存。这是「虎啮的千代」之名的来由。在此后的岁月中，这一名号渐渐简化为「虎千代」。但在深邃之兽的腹中，她染上了深罪的黯色，也透过猩红的利齿看见同行者被撕碎。因深渊力量影响她陷入噩梦，在梦中，所有的一切都变成深渊幻化而成必须对抗的敌人（即「队长」口中的渊蚀综合症[41]），最终她向雷电将军拔剑;后来，她持刀之臂与尖角被斩落，如同负伤的野兽般从城中逃向林野，雷电将军向她挥的刀斩掉影向山部分山体[22];再后来似乎是被天狗，或是终末番，或是山中修行的岩藏之胤当成不认得的怪物，收拾掉了罢，因为她俊美的容貌已经因为漆黑的仇恨与负伤痛苦变得狰狞扭曲。抑或是遇见了大蛇遗骸附近的鬼面执剑人形，故此结束了命运的旅途。[42]当时许多军士也受到「深渊噩梦」的影响，食梦貘一族在当时协助清除恶梦[22]。波穗月白经津曾为千代所配带之刀，但在与将军的真剑对决中断刃，虽送回重锻，却并没有锻成，最终这把「波穗」成了历史。[11]千代沾有深渊力量的角被几只妖狸意外挖出来，几只妖狸因此受到深渊影响陷入噩梦。深渊残余力量重返稻妻，让被影响者陷入噩梦。在五百年后因为旅行者清除他们身上的深渊力量才得以解除[22]。
