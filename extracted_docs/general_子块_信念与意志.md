# 信念与意志

安柏翻看祖父留下的寓言书，明白了“勇气的力量”的时候，得到的神之眼。菲谢尔14岁生日，愿望“皇女就是菲谢尔，菲谢尔就是皇女”被承认时，乌鸦奥兹与神之眼一同出现。凯亚养父死的当晚，凯亚向迪卢克吐露真心，说明了自己的秘密，从谎言中解脱。两人大打出手时，凯亚获得了神之眼，这是全力一战的纪念。丽莎为了探索魔导，认为自己需要一枚神之眼。想到这件事的时候，神之眼就出现了。莫娜师父曾赠予莫娜一个神之眼教具（装饰品）。尽管莫娜本人一直不渴求神之眼的力量，神之眼最后还是降临在了教具里。优菈由于特殊的身份与立场，为了找到自我，向一名老侦察骑士学习。优菈从中明白了豁达与坚持，找到了只属于她的“温柔的仇恨之路”时，获得了神之眼。白术在师父墓前，下定决心要让「珥蛇拖龙法」的契约不再流传下去。此时，获得了神之眼。胡桃13岁时，为了见爷爷最后一面，独自前往生死边界。胡桃的爷爷没有未竟的心愿，胡桃自然也就不会等到爷爷。最终胡桃明白了爷爷“遵从自心”、磊落无憾的一生，返程路上获得了神之眼。七七生前最后一刻，对死亡的恐俱、对生存的渴望、与对亲人的思念，让神之眼出现。云堇第一次唱独角戏《走雪》时，戏中人在雪中的艰难跋涉和哀叹，和戏外天空中正飘着的细雪交织融为了一体。云堇顿悟到，自己要像戏中人一样呼吸、生活，打造有情世界，这才是她想要的戏、想讲述的故事。在此，获得了神之眼。枫原万叶漂泊旅行时，在他人家中借宿。晚上重整心情，不畏险阻，“以天地为屋宇，万灵为诗友，亦有无穷趣味。”第二天醒来，获得神之眼。九条裟罗年幼时力量弱小，无法单独对抗魔物。曾在战斗中被击落山崖，翅膀受损无法张开，从高处坠落。坠落中，她遗恨，“我本以为，凭我的能力，可以永远守护这片山林…”，获得了神之眼。第二天，昏倒但毫发无伤的九条裟罗被九条孝行收养。久岐忍生在巫女世家，家族代代以侍奉鸣神为荣。但“本应侍奉神明的巫女，却因离开神社而获得神之眼的垂青。”神里绫人自幼渴望成为栋梁，让家族长久繁荣，守护家人安康。某晚，父亲卧病不起，将绫人召至身前，叮嘱托付并寄望于绫人。回屋后，神之眼出现。绫人认为，需要他担起责任的时刻已经到来。托马在神里绫人劝说托马离开稻妻时，不愿抱着遗憾和愧疚回到平凡平稳的生活中。深思熟虑之后，托马回答：“在此时离开，就会失去忠义…我愿尽我所能，为少爷和小姐尽一份薄力。”此时获得神之眼。多莉家里曾买不起天价药材，去治疗多莉姐姐的重病。多莉下定决心，“要赚多多的摩拉。这样就不会再看到亲近的人露出难过的笑脸，这样就不会再因为没有摩拉而失去什么…”的理念，让她获得神之眼。莱依拉论文受到一位诃般荼的驳斥后，坚持与之「辩论」，半年后获得神之眼。
