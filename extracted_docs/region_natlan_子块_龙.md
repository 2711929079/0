# 龙

葬火之战后[12]，尼伯龙根引入的深渊之力使修库特尔醒来[7]，火龙王修库特尔重新统治纳塔，由于天理与尼伯龙根大战后元气大伤，无暇顾及纳塔龙族的卷土重来，火龙王带领族裔将纳塔的天之使者屠戮殆尽，最后一位天使融入地脉古树，成为了夜神。为帮助纳塔人，夜神整合旧地脉的残片，重构其脉络，「夜神之国」由此诞生。夜神是在死之执政若娜瓦的指引下构建「夜神之国」。她的这个做法相当出格，让天理不太满意，这之后，她自暴自弃，再被他人谈论也不再追究[13]。纳塔龙众蒙受焰主[14]（火龙王修库特尔）的恩泽享得自由，[15]当时的人类成为龙群的奴隶。
秘源机兵·寻捕械就是龙类研发的机关之一[16]，龙透过燃素驱动此机关建造城市[17]，同时也是远古的巨龙试图强化地脉的道具，将力量注入到地脉的中枢。[13]库胡勒阿乔也是此时期的遗物[18]，「库胡勒」一词有「命定」的含义，「阿乔」即「领主」，其真实身份是由古龙开发不太管圣山事情的智能单元CL-04 「众命途的隐者」阿·阿瓦乔·库穆勒坎，曾从事生命燃素化的相关研究。他只喜欢研究燃素，不喜欢叉手政治、拌嘴与冲突，因不涉事常常被其他龙族讥笑。龙族认为生命与灵魂亦是从燧石相撞的火花中拥有最初的形廓。被名为熔岩游像包裹在体内的余烬，也是那伟大时代冷灰般的残余[19]。此时的人类文明已经退化到原始部落水平。为确保对燃素的精密调整和控制，修库特尔创选了名力“秘源龙仆”的机器，并将其原型机命名为“库库尔坎”。此后，部分同族效仿此做法，也制作了秘源龙仆[20]。人们还无法执掌火焰的年代，世上各处都有对着火焰观想，以谋求其中奥秘的智者。蕴火的幽墟原本也是环形的祭场，智者们曾试图在此掌握令燃素形态变化的法仪[21]。十一席「明晨之镜」在记录中嘴了一句阿乔还有和12席吵架，且能得知她既不喜欢技术又讨厌人类。她是领主中对人类厌恶最偏激的一个。在尼伯龙根离开提瓦特后，第十一席曾随火龙王前往三月的宫殿询问尼伯龙根的下落，但三月没有回应[22]。
龙众位于大火山北山脚处的「石火坠陨处」是在过去某次猛烈喷发中从火山内部飞出坠落地面形成的秘源遗迹，实际上，这里曾是修库特尔进行「龙裔退行演化逆转可行性」研究的场所。极致的「环境适应力」与「进化潜力」曾是龙裔们引以为豪的种族天赋，然而在天钉降下改写了这片土地的法则后，这一点却反过来被天空岛利用，无数龙裔为适应巨变的环境，被迫进行演化，失却了原本的形体与智能，从高等智慧种族沦为了在野外嘶吼的野兽。修库特尔为了挽救那些已经退化的同胞，展开了对逆转的研究，但结果却是令人失望的：尽管退化龙裔仍可适应原本龙族生活的富燃素环境，但退化过程已对龙裔的可塑性造成了几乎无法恢复的严重打击，即使强行令力量与智慧重新回归，龙裔的身躯也会因无法承受而发生畸变，并在很短时间内死去。通过实验和相应尝试，修库特尔还是找到了逆转这种影响的方法，即通过基因工程与定向选育，经由良性的、多代的血统改造，重新恢复龙裔原有的形态和进化潜能，继而令退化的族裔取回失去的高等智慧，这个过程中用作筛选标记的，便是称作「焰主之祝」的特定基因表达。尽管找到了有望最终治愈退行性演化的方法，但在修库特尔看来，这等做法所需的时间实在太过漫长，付出的代价也太过巨大，退化的同胞中终究也只有极少数能够得救，绝大多数都会在此过程中遭到「淘汰」，如此做法实在过于不人（龙）道。即使退化龙裔已同野兽几乎无异，但在这位仁慈的王眼中，他们依旧是自己的子民、自己的同胞，实在做不到像对待动物一样去对待他们。因此，修库特尔最后还是选择了放弃，对相关实验报告予以废弃封存[1][23]。传说中，嵴锋龙的祖先是山一般巨大的龙兽，拥有相当漫长的寿命，沉睡时，群山在其背脊上成型，甦醒时，就会抖落身上的泥土与沈岩，唯留宝石与美玉在鳞角中熠熠生辉。「回声之子」部族中巨大的钻状物，据说是那时龙兽巨大钻角的遗留。[24]传说中，原本的匿叶龙并非现在的模样，而是更小，小到不易被观察到的程度。它们甚至可以随意游荡在风中，依附在其他巨龙的身体表面，或是如云一般群聚与游移。只是后来为了能够维持住能力退化后的形态，不得不选择现在的身姿。[25]传说中，更为远古的龙拥有更加强的环境适应度，那时海中最为巨大的龙漂浮在水面上时，甚至会让人误以为是一座浮起的岛屿。据说现在的鳍游龙就是那种巨大海龙的后裔，从其宽阔的背部，似乎也能窥得远古海兽的庞然形态。[26]绒翼龙的先祖是最接近巨大翼龙形态的龙兽，能够如云一般在天际巡游。叙事人的故事里，远古翼龙里最伟大的那只，只要张开翅膀，就可以遮蔽一整片天空。如今的绒翼龙自然不再具有那样的形态，只有藏在绒羽之中的龙爪仍如从前一般锋利。[27]原本的突角龙具有堪称庞大的形体，甚至可以用角顶着山峰进行移动，只不过行动远不如当今这样灵敏。或许如今突角龙身躯里所具备的那种力量，也是一种血脉之中的遗留[28]。
