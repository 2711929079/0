# 三界

提瓦特分为光/龙蜥界、人/人间界与虚无界。光界其代表的是原始元素力/蛮荒元素力，已提及的光界力包括燃素和月矩力，可能还包括原始胎海之水[7]。光界力（原始元素力）强的地方也是龙蜥的宜居环境，但是对人类来说就不太合适了海祈岛圣土化的本质就是原始元素力变得强势。虚无界代表力量是深渊，主要特点是有侵蚀性，与光界和人界都不兼容。从利用睦梳之匣抵抗侵蚀和驱散黑雾来看，光界对于虚无界的侵蚀有着一定的抵抗力。而人间界代表力量则是如今的七元素，七元素合一则是地脉能量/光元素。三界的力量必须三者保持平衡，有时会失衡，如深渊入侵。早在白夜国时期，人们就有三界的概念。渊下宫是三界力量的交汇点。
