# 约400年前

厄里那斯之战雷内·英戈德在自然哲学学院学习期间创立「水仙十字结社」，招揽信众集结力量以实施他的救世方案。期间雷内将自己转化为「大师」。距今四百余年前，枫丹爆发了内战，交战双方为「水仙十字结社」与枫丹廷的机构「逐影庭」，战场之一位于厄里那斯体内。隶属于逐影庭的阿兰·吉约丹和水仙十字结社的莉利丝、雅各布都参加了这场战争。最终十字结社一方溃败。战斗中双方使用各种机关交战，诱发了大爆炸。爆炸把厄里那斯的尸体炸得血肉横飞，机械零件也散落其中。其后，厄里那斯的肉片和机械零件形成了美露莘这一种族。她们与厄里那斯的意识进行交流，在其尸体内建造了「海沫村」。[46]卡特皮拉作为结社的一员，参加了厄里那斯决战。战后卡特皮拉被捕，关押在梅洛彼得堡400年，与其相关的档案资料早已佚失[59]。玛尔菲莎与埃尔纽艾尔为追查在枫丹境内作乱的「水仙十字结社」，德怀特委托退役多年的埃马纽艾尔调查[67][34]，循着线索找到了「大师」，被其利用莉利丝的能力溶解吸收。[33]传奇决斗代理人玛尔菲莎出身于灰河或白淞镇，公义的酬报为其所用武器，年轻时曾与埃马纽艾尔交手，被其使诈偷袭而败于其手，被迫认输。信丰公义的玛尔菲莎对此耿耿于怀，本想再找他对决，却意外得知他早早退役、投身烈酒。为了亲自查明真相，她追随着剑客的足迹与创伤，一路追猎命运中的对手…尽管有重要之人（扮演帕西法尔的约瑟芬）倒在命运的决斗场上，尽管内心已残破不堪，在追踪到剑客所留下的面具后，一切沧桑仿佛都得到了报偿[33]。最终，玛尔菲莎与自己苦苦追寻的埃马纽艾尔落得同样的结局，「大师」获取了她的枪术技能。[42]桑多涅与阿兰阿兰拿过很多笔记本给我，我经常涂两笔就扔掉。每到那时，他的表情无法定义，但还是给我新的。我不知道为什么他要这样。我不知道为什么我涂两笔就扔，也不知道为什么他允许我的这种行为。他说我的情感模组非常完美，我不明白。这本是新的，我不想再扔掉它。我学会记录。我开始记录。我学得更多，谈吐也变得更优雅。优雅这个词我刚学会，阿兰说不用勉强。我不认可，我做得非常好。下午聊天。几个结论：第一，阿兰创造了我；第二，他回忆着故去的妹妹创造出我，现在他感到愧疚；第三，我的学习能力高于预期。阿兰·吉约丹的妹妹名叫玛丽安•吉约丹，已去世很久。阿兰在六十五岁生日当天将我完成，他说我是一个朋友，一位家人。他很高兴生活里多了个陪伴者。阿兰未对我和他的关系下定义。显而易见，我不是玛丽安·吉约丹。他同意这一说法，他也觉得我和玛丽安。吉约丹大不相同，并希望我随心所欲地做任何事。我说我不会再扔掉笔记本时，阿兰略显失态。他应该是想「哭』，又没有真的「哭」。阿兰说如果我想，可以有个生日。我会认真考虑。他说这世上的人无法选择自己的生日，但我可以，我是自由的。阿兰的身体越来越差。医生来过很多次，他都把门关上，不让我知道。我感觉到了什么。阿兰说，那是「愤怒」和「悲伤」，他很高兴我学会了这些。但他说我还不懂得何谓「内疚」，因为「内疚」这一情感背后的社交机制太过复杂，动物和机械很难轻易学会。我觉得我不需要懂得这个。阿兰问我想要什么，我想要他健康。另外，阿兰让我看一件新发明，他叫它「普隆尼亚」。无法定义普隆尼亚是家人或亲属或朋友，普隆尼亚并不具有与我相同的情感模组。不清楚为何阿兰不将它做成与我相同的规格。阿兰称，我和它都是独一、不可复制的。我是我，而普隆尼亚仅是纯粹的机械工程体。日常里，我整理仓库，清扫房间。除错普隆尼亚基本功能。阿兰晕倒了。状况不好。医生来过，表情十分低落。阿兰的下属来探望他，带了一套崭新的茶具和一些名贵茶叶。阿兰不爱喝茶也不习惯使用太华丽的东西，茶具归我了。我种的盆栽全都生长健壮。这太简单，我不觉得有我什么功劳，条件适合时植物自己就会疯狂成长，与他人无关。我认为我也是一种植物。我不需要任何东西也在生长。今天，我帮阿兰一同完成了普隆尼亚的最终测试。阿兰最近无法落地行走了，我用轮椅推着他过去。阿兰没有太大的情绪变化。普隆尼亚运作稳定，但他仍未表现出满意。他说，普隆尼亚的本质是一个算式，用来计算未知的事物。至于那些事物究竟是什么，他也不确定。什么是未知的？那太多了。他居然真的说不出来，而且很悲伤。我不理解。阿兰问：「如何看待世界的命运？」我回答：「用探测器看。」阿兰将他以前的笔记交给我。我很高兴，我喜欢他教我的知识。世界的命运，我最近思考了这个问题。然后很快明白这是不会有答案的东西，人连自己的命运都掌握不了，还想干涉世界吗？ 简直是蠢材。阿兰说蠢材有好几个，我嗤之以鼻。接着他也笑了，不知道在笑什么。日常里，我买了新床单、枕套，还有窗帘。我的房間收拾得很漂亮。我简直太聪明了！还是太保守了。不用「简直」两个字，我就是聪明。我学得很快，阿兰也同意。阿兰的身体坚持不了太久。他坐在壁炉前讲了一些故事。最初，他是为了纪念妹妹和自己一同生活的岁月，才制造了我。他当时居然没想好要做什么，就把我做出来。后来他希望我代替他外出旅行，开开眼界。问题是，我现在是阿兰的同行，我和他都被漫长的职业规划害了！一个人不工作，才能到处旅行。阿兰说这个梦想已经落空，已经彻底完蛋了。他说他还有一个愿望，不过很早就放弃了。我追问了很久，他有点尴尬，说是小时候想过要拯救世界…我问这个需要我继承吗，他吓了一跳，告诉我不用。我做什么都行，他没有要求。阿兰问我有没有什么想做的事，我…暂时还没找到。父母对子女的出生负责，而子女对自己的前程负责。这是社会性的规则，我认为在这套规则的最后，多半是：子女对父母的晚年负责。阿兰的晚年走到了尾声。他立下遗嘱，遗产包括少量存款、普隆尼亚和一些笔记。他让我在他死后烧掉笔记，以免被盗。那么多笔记本，我看也就只有普隆尼亚来偷才偷得动。普隆尼亚是阿兰一生最后的寄托，可惜在他活着时没能看到它算出确凿的答案。他让我别放在心上，把普隆尼亚交给枫丹科学院也可以，只是對那些人感到遗憾，他们没我懂得如何维护它。我不同意。普隆尼亚归我了。不知道有什么用[68]。枫丹科学院建立厄里那斯残骸之战让不世出的天才阿兰·吉约丹同时失去了仇敌与至亲，为了拯救枫丹免于危机，他回到枫丹廷，在伊黎耶岛的北方靠山依水建立了枫丹动能工程科学研究院，简称枫丹科学院。为了纪念玛丽安，阿兰·吉约丹修建了玛丽安纪念公园，但没有她的陪伴，阿兰变得愈发激进和不近人情，不再与任何人亲近。余生中的阿兰逐渐不再相信一切人的智性所不能完全掌握、不能彻底理解的东西，他构想出一个由机关与非元素能源所驱动的王国。从枫丹科学院退休后，吉约丹谢绝了一切来自沫芒宫与枫丹廷民间组织的邀请，前往苍晶区乡下隐居，自此与社会几乎完全隔离——尽管在此之前，这位性情孤僻的老人也从未与他人有过任何深交。如果我们接受娜莉·科克伦夫人的说法，那么就本质而言，阿兰·吉约丹是一个孤独的自闭者，总是怀着一种深深的不安全感，也正是因为如此，他总是对别人抱着疏离的态度，彷佛这样就可以用假装出的迟钝来掩饰精神的极度敏感。他不懂如何纾解自己的痛苦，也不懂如何将痛苦向其他人述说，只能将它们像珍珠一样藏在心底，用血肉去琢磨—当然，吉约丹的学生们不会接受这种说法，斥其为博人耳目的胡言乱语，只是在玷污恩师的名声。这便是枫丹科学院第三任院长洛杰尔·科特斯向《蒸汽鸟报》提出诉讼的首要原因[69]⋯最后他意识到水仙十字院终要有个人以正常人的身分或着，于是放下手头的研究，拿起童话书念给他的造物同时是他的好友的桑多涅，以普通人而非天才的样子离世。阿兰没有孩子，在与雷内决裂后也没有亲近的朋友，在他逝前，曾委托桑多涅销毁研究手稿，在此之前桑多涅曾把所有手稿看了遍[70]。在他人眼里研究手稿是被不明人士偷走。枫丹海露港等处的升降梯就是以阿兰的蓝图所造[71]。桑多涅加入愚人众今天是葬礼，仪式期间下了很大的雨，真不知道天上有谁在哭，烦！雨天撑伞真的很麻烦！烦死了！阿兰下葬了，一切平稳。我出面送别了他，普隆尼亚看家。直到半夜天气才好一点。我坐在墓地陪了阿兰一下子，天上有月亮陪着我。夜里很安静。再也没有虚弱的呼吸声⋯这世上好像也只有月亮陪着我了。离开了。再见！更新日志：为普隆尼亚更新泡茶功能、送餐功能、增加查询书籍功能、修正普隆尼亚不会晾衣服的故障。对外宣称普隆尼亚是我的侍从，这样一来，人们就不会把好奇的目光投往它身上了。我得保护它，它是阿兰的秘密与愿望。至冬真⋯冷！虽然与我无关。找到新工作，待遇还不错。上级是一个很冷漠的女人⋯真的很冷漠。同事中什么样的人都有，受不了[68]。那维莱特四百余年前那维莱特就任最高审判官。就任后他在枫丹推动改革，许可美露莘一族进入枫丹城市与人类共同生活并担任公职。在此过程中他遭到了相当大的阻力，损失了两名得力下属。[72]希格雯在曾经那个人类厌恶美露辛的时代里，名为希格雯的美露辛拜访了一位性格孤僻但并不排斥美露辛的老医生，向她学习医术。之后希格雯与一个小女孩成为朋友，这个小女孩身患重病，希格雯想帮助她却被她的家人拒之门外。她只好向老师求助，恰好她的老师当时在研究原始胎海水，在研究中顺带发明了一种能让美露辛变成人形的药剂。老医生想要找到拯救枫丹人的方法，于是拿自己试药，最终失败被溶解。希格雯为老师建了一座衣冠冢后，便服下了另一瓶药剂变成人形，顺利地以医生身份去治好了那个小女孩。之后希格雯前去自首，因为她知道自己违反了一条枫丹自古以来就有的法律：禁止将其他物种转化为人类。在服刑期间希格雯运用医术帮助人们，就算刑满后也一直留在梅洛彼得堡当护士长[73]。特许券愚人众执行官“富人”认为金钱作为交易用的物品，应当由人们亲自掌控，而非由神制造。为了制造能取代摩拉的货币，富人在梅洛彼得堡发行特许券作为实验，并使用无人机“监督者”观察着梅洛彼得堡的金流。无人机不知疲倦地记录着每笔金钱的易手，每个人的积蓄与挥霍，每种价值的提升降低，每张货币在指定周期内的流通次数。其间唯一法律是囤积者的私法，而唯一制裁是贫乏或死亡。或食利掌权而支配，或被支配而劳碌至死，规则总是公平。由此，人将凭借自身所拥有的野心与财富与神并肩，而那些竞争中失去一切的弱者将被人世的洪流吞噬，不再有神的力量介入，从贫者面前粉饰富者的威势，不再有神的财富涌入，从富者脚下拯救贫者的尊严。[74]（卡特皮拉提到400年前就有特许券存在[59]）初露之源古枫丹的自然哲学家们从未记载过类似初露之源的造物，直至四百余年前，方有枫丹科学院的工程师开始对其进行零散的研究。一些学者相信，这些至纯的灵露是厄歌莉娅在离去前留给枫丹的最后礼物——但这种说法至今未能得到任何可靠证据的支持。[75]
