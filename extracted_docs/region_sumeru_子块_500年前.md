# 500年前

贤者比鲁尼五百年前有一位大博学家兼贤者比鲁尼。[66]坎瑞亚灾变征召除大慈树王以外的七神都被召集前往名为坎瑞亚的国度，而大慈树王有一项更为重要的任务——保护世界树。[67]禁忌的降临灾厄的降临伴随着禁忌知识的污染而出现。从深渊之底而来的属于世界之外的禁忌知识污染了世界树。大慈树王知道单靠自己的力量，根本无法将禁忌知识驱逐。于是她创造了统合人类智慧的装置，命名为「虚空」。大慈树王借助虚空借走了人们的梦境，来作为清除禁忌知识的能量，禁忌知识几乎从世界树中完全清除。与世界树相连的大慈树王在治理世界树的同时，被禁忌知识所污染，她身上留有禁忌知识，却与世界树相连，导致世界树连带受到污染。只有当这个存在本身被彻底遗忘时，禁忌知识才会被彻底清除……[67]草神迭代因为世界树的机制，大慈树王无法将自己从世界树中删除，她在逝世前折下世界树最纯净的枝杈（即小吉祥草王），作为她的轮回转世。她逝世前留下「让世界彻底遗忘我」的线索在世界树中播放，让发现线索的小吉祥草王能来到世界树将其中有关她的资料彻底清除，以杜绝深渊污染，但同时她也会被世界遗忘。树王逝世后，须弥学术界看似兴盛，却一直没能出现较大的突破。[55]因为树王之死，本带在树王身边的乌婕特之眼佚失。[19]贤者找到了新生的小吉祥草王，将其接回了须弥城。[68]薇瑞迭瑟恩猎手薇瑞迭瑟恩是森林王收留的孩子之一，如同在晨光下的苍翠草甸与林木间穿行的野兽一般纯净，从不带有一丝恶意，也从未为生存以外的目的放出箭簇。猎人是在远离城市的地方，在森林王这样的训导中长大的…「吾等诞生在绿茵中。只要有青草与树荫，便通行无阻」「吾等与鸟兽无异。只要遵循天地的法理，便无虞生死」「遵循自然的万物，在生命尽头都会去往无边际的原野」遵循着这个信念，猎手不留下足迹，不与天地自然为敌，安抚着被箭矢穿透心脏的野兽，直到它的生命回归自然。灾厄发生，纯白骑士罗南向着漆黑前进，盲眼少年也跟着兄长[31]，不幸被魔物所伤，已是奄奄一息的状态。那天猎手循着沾染在地面的污血前行，来到自己休憩的树下，借着白枝微弱的光遇到那改变自己想法的盲眼少年。她放箭射杀了魔物，听了关于少年的请求，心中已然不再是曾经遵循自然的理念，复仇的情绪充满了猎人的内心，如野兽一般猎杀魔物。如果灾厄没有发生,如果她没有循着草叶上的污血前行，如果没有在常常小憩的树下遇到那奄奄一息的盲眼少年，那她就不会为复仇的愿望、话语中鲜活的血与火光动摇…「千万不要忘记，善良的薇瑞迭瑟恩」「千万不要忘记呀，你是属于绿茵的、树林的孩子」「不能为了争斗、为了仇恨、为了所谓荣誉而射箭」「染血者永远也不能找到，彼方无边际的苍翠猎场」「那至少，不要让这把弓被即将到来的，仇恨的厮杀玷污」「既然我（薇瑞迭瑟恩）无法抵达彼方，和师父（森林王），和从未谋面的血亲（其是孤儿流浪的林内被收留从没见过父母）相见」「至少让这把弓保持纯洁，让它能够传达我的思念和歉意」在盲眼少年的请求下，薇瑞迭瑟恩接下来自少年的弓，开始讨伐肆虐的漆黑魔物，为盲眼少年复仇。后来，她在厮杀中被黑血染身，被路过的罗南（染血骑士）误认为魔物斩杀。[69]末代森林王坎瑞亚灾变中，末代森林王为了保护森林死去了，森林不再构成新的宫殿，迷宫从此也消失了。或者是漆黑的群兽如同飓风让树木如同麦浪一样倒下。迷宫死去了，森林王也死去了。[32]森林王曾经的侍者，血缘与猫更近的长须虎在之后继承了虎的名号。[70]桓那兰那桓那兰那被毁。三位古老的兰那罗封印了无留陀的化身，但封印逐渐松动。兰帝裟用石锁封印众多坎瑞亚的战争机械。此后，兰那罗们在兰拉迦的梦中建立了新的桓那兰那，鲜少外出。长老兰拉迦成为了维持梦境「桓那兰那」的「梦之树」。[30]无郁稠林此时无郁稠林被侵蚀世界的污秽所垢染。在经历了漫长的净化与复苏后，才重新成为坎瑞亚灾变五百年后遮天蔽日的庞大森林。[71][30]金色的那菈旅行者的血亲前往恒那兰那，帮助兰那罗解决遭到深渊侵蚀的法留纳神机的危机，与兰那罗们一同度过了快乐的无忧节，并与兰穆护昆达和兰帝裟一并打败并封印了「无留陀的使徒」。协同作战的兰那罗有兰穆护昆达，兰拉娜，兰帝迦、兰般度等第一批高端战力的兰那罗种子。她/他经常和兰那罗们说起自己血亲的事情，让一部分兰那罗知晓了家庭的概念，并也开始以兄妹相称。而他/她留下的笔记本被兰纳罗称作「森林书」。旅行者的血亲为了让须弥雨林更好的恢复，留下了神器「无郁夷摩」以及看守神器的生命之杯「阎迦由玛」（可能都来自坎瑞亚）。神器有效的恢复着雨林的生态，並设下了封印，保护好雨林的重要植物。在无忧节中，旅行者血亲为兰那罗们做了很多菜肴。为了能够在下次无忧节上能让旅行者血亲吃到自己尽心准备的至上美味，兰那罗哥哥们进行了长达五百多年的厨艺修行。无忧节后觉王树孕育出了惟耶之实。当时可能是使用的村长兰拉迦的记忆，所以兰拉迦失去了对旅行者血亲的记忆。旅行者血亲知道真像后痛哭流泪，向村长讲述了一起战斗的故事。无忧节过后，旅行者血亲与兰那罗们击败了死域核心，虽然最终击败了死域的化身，但是无法彻底根除，所以兰那罗只能进行封印。兰那罗构建了四重封印将其束缚在了他们曾经的故乡往昔的桓那兰那。虽然漆黑灾厄与旅行者血亲无关，但是她/他毕竟与坎瑞亚关系密切。她此时是带着愧疚之情的，所以在走之前并没有留下名字。金发人类临行前告诉兰那罗们，日后一定会有另一个与他/她相似的人与他们相会。因为金发人类没有留下名字，于是兰那罗们便根据其修好「法留纳神机」的功劳称呼他/她为「那菈法留纳」。[30]之后，旅行者的血亲前往甘露花海，遇见戴因斯雷布。[72]西摩格五百年前的坎瑞亚灾厄中，前代水神厄歌莉娅死于甘露花海，身躯化作「甘露活水」。神鸟西摩格饮下「甘露活水」，将自己拥有的「灵光之力」与甘露相融合，形成灵光百种。其中具有智性的即是花灵。最初的花灵祖尔宛，与其他的花灵都参与了对深渊兽潮的战斗。花灵和戴因斯雷布、东方来的那伽朱那团的独臂贤者、那伽朱那、坎瑞亚的残军一起对抗过漆黑兽潮。其后，一部分花灵和它们的搭档组成了最初的矫论团。在那之后，戴因斯雷布与旅行者的血亲离开了。[72]独臂贤者东方来的那伽朱那团的独臂贤者在对抗漆黑兽潮后前往地下考察坎瑞亚遗迹，遇见了白鸽骑士当中的海杜菈。遗迹巨像在坎瑞亚灾厄时期，坎瑞亚派遣了被后世称作「遗迹巨像」的巨型战斗机械人清剿漆黑兽潮，但行动失败，三台遗迹巨像残留于须弥。
此后，教令院将遗迹巨像的炉心封印。降诸魔山的遗迹巨像炉心被封印在迪弗旧窟。[30]树王圣体菇在须弥，森林中的居民有着向教令院贡奉树王圣体菇的传统，但这些贡品的用途却不为世人所知。据某些逸话中说，教令院每年会在净善宫进行秘密的仪式。在仪式上，贤者们会通过食用树王圣体菇的方式来纪念树王的牺牲。[73]小吉祥草王诞生后新生草神被教令院囚禁作为新生的神明，小吉祥草王并没有大慈树王那般全知全能，无法应对世界树的污染。贤者遂改造了大慈树王留在净善宫的冥想装置，让它无法从内部开启，将小吉祥草王囚禁于其中，让教令院自己治理须弥小吉祥草王的神之心则被用于维持虚空的运转。但作为草神，小吉祥草王可以通过虚空终端访问任何一个须弥人的心智，在被囚禁在净善宫期间，其通过这个手段帮助了无数须弥人。[55]梦境的消逝教令院改造了虚空终端，让其可以盗取须弥人的梦境，借此来增强虚空的算力。「须弥人不会做梦」的传言由此产生，并被认为是智慧和理性的象征。[20]由于兰那罗的能力与梦境息息相关，因此须弥成年人无法再看见，也无法再记得兰那罗。[30]人类与兰那罗渐行渐远，最后兰那罗成为童话故事。那伽朱那团又名「萨曼尼耶」，教令院称「那伽朱那团」，创立者那伽朱那。
曾经是归属于教令院的学派团体，曾留守沙漠、对抗深渊魔物，后与教令院产生分歧。如今早已凋敝。驻扎地位于荒石沧漠。
