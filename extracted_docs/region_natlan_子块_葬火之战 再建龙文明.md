# 葬火之战 再建龙文明

第一降临者法涅斯降临提瓦特，击败了尼伯龙根（元素龙王共主）以及包括火龙王修库特尔在内的七大元素龙王，摧毁了龙族文明；修库特尔被冰结于熔岩之中，陷入假死。法涅斯改造提瓦特环境，引入了人类，让天使行走人间，引导人类建立文明。很久以后，尼伯龙根从天外归来，携深渊力量对法涅斯发起了复仇的“葬火之战”，却再度被打败。火龙王修库特尔在此战中受到深渊感召，苏醒助阵，因而染上了深渊的侵蚀。但与此同时，法涅斯一方也元气大伤，纳塔一带的天理秩序遭到了尤其严重的破坏，无暇顾及纳塔龙族的卷土重来。火龙王修库特尔带领族裔将纳塔的天之使者屠戮殆尽（最后一位天使融入地脉古树，成为了夜神），在焦土上重建由龙支配的国度——即「纳塔兰提亚」，当时的人类成为龙群的奴隶。极致的「环境适应力」与「进化潜力」曾是龙裔们引以为豪的种族天赋，然而在天钉降下改写了这片土地的法则后，这一点却反过来被天空岛利用：适合龙族的富燃土地（即人类所称的「圣土化」）被天理降下天钉消除，为了适应被天理改造的环境，擅于进化的龙族中有一些发生了“退行性演化”，失却了原本的形体与智能，从高等智慧种族沦为了在野外嘶吼的野兽。
