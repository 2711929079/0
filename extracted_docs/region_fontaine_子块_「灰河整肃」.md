# 「灰河整肃」

出身灰河，以行骗、行窃为生出没于光荣剧院的小贼姐妹，为科培琉司戏中的游侠喜剧故事所感染，萌生了凭借魔术走上舞台、平等地为贫富众生带来表演的梦想。于是一人以戏中的蒙德侠盗「帕西法尔」为艺名，一人则扮演助手「约瑟芬」，两人的表演从爱德华多的酒馆中、灰河水渠旁，走进了光荣剧院里，帕西法尔的名字在地上城与地下城中都令人津津乐道[40]。但城市的“优化”是没有尽头的，在枫丹廷发起的灰河整肃行动中，扮演「帕西法尔」的少女死去了，「约瑟芬」赶回灰河，只见到了没能完全干涸的血。于是「约瑟芬」接下了「帕西法尔」的名字，并立志实现一个没有贵族和王者、没有生来的富贵和无法摆脱的贫困的世界，以作为属于她们的「大魔术」[40]枫丹高层因第二次城市「优化」对下水道「灰河」中生存的城市底层进行的武装镇压。枫丹廷的官僚决定「整理市容」，将无数人赶入地下的灰河。这批人中的雅各布的生父[41]爱德华多·贝克成为了灰河的首领，负责维持灰河秩序。然而不久后枫丹廷继续整肃灰河，已建立自己秩序与规则的人们自然不满而反抗，为首的爱德华多等人被捕，但其党羽将他救出，伙同秋分山匪徒占领白淞镇继续对抗枫丹廷，挟持了负责押送的执律庭成员，试图与枫丹廷谈判。记者卡尔·英戈德在其中斡旋试图促成和平谈判，但枫丹廷方面仍然选择了命令逐影猎人（包括埃马纽艾尔）进攻白淞镇，使得爱德华多、「帕西法尔」及雷内的生父[41]，与白淞镇的歹徒勾结的镇长雷诺·德·佩特莉可身死，反抗运动失败，史称「白淞之围」。阿兰、雷内、雅各布、玛丽安四人以犯罪者家属身份被投入水仙十字院。而「帕西法尔」妹妹约瑟芬决定以「帕西法尔」的身分在事后被捕（假冒帕西法尔）并在法庭上申请决斗，[33][42]审判庭派出传奇决斗代理人玛尔菲莎作为控方的代表参与比武审判。此时的卡尔已辞去记者的职务，作为探险者与荒野、废墟和遗迹为伍。数年后其回到十字院为成员们拍照时，透过镜头想起过去的种种经历，直到最后，没能作为记者在比武中记录帕西法尔最后一战的懊悔才将他的视野席卷——决斗的结果是帕西法尔身亡[33]。
「帕西法尔」与「最初的大魔术」曾经的小贼姐妹受科培琉司大师的剧作的影响，以剧中侠盗「帕西法尔」和蓝宝石美女「约瑟芬」为名，将盗窃的手法应用在魔术表演上，最终成为了广受枫丹上下层欢迎的知名魔术师。但是枫丹上下层的矛盾难以调和，最终在「灰河整肃」中，「帕西法尔」身亡，「约瑟芬」继承了「帕西法尔」的名号，并设计出了属于她们的「最初的大魔术」。「我是大魔术师——『伟大者』帕西法尔！」「你们将要目睹的是不曾想象的梦幻世界！」「没有贵族没有王，没有指着你们的刀剑！」「没有生来的富贵，没有无法摆脱的贫困！」[40]
