# 「雾切高岭」

天狗族长还有一个徒弟，但是一直是以小姓的身份待在身边，其名雾切高岭，天狗叫他昆布丸。许多年前他曾是鸣神岛人类中武艺最高强的一个，天狗族长见他有天赋，便放海，以赌约与其打平后，教他天狗弓术。学成之后，更是把将军赐予的名弓飞雷之弦振，送与高岭大叔。都是天狗旗下的弟子，浅濑响难免和高岭产生了接触，二者也产生了暧昧的情愫。[27]此后，将军将雾切之回光赐给当年的旗本─高岭昆布丸，后世称为雾切高岭的武人。此刀据说能挟雷光斩开山岚夜雾，因此称做雾切。[28]
妖怪轶事八重神子八重神子成为雷神眷属。被狐斋宫抚养在鸣神大社。[12]
