# 1000年前～500年前

赤王逝后为了解决禁忌知识产生的问题，赤王决定牺牲自己[55]。赤王死后，沙漠地区大乱，各类藩王轮番称霸，不停征战。[56]沙漠诸国第一王朝称霸沙漠的第一位王是“兰沙赫尔王”卡乌斯巴依，蒙面王后裔，为了称王屠杀了很多人，毁灭了许多城市，掠夺大量人民填充自己的都城，将沙漠分为七块区域，分别派人治理。分别是：“蓝宝石之城”图莱杜拉，城主呼玛云王，吸引很多四处而来的无归者聚集[38]“勇士之城”萨勒赫，城主图兰哈冈赤王与花神共创的“圆形剧场”阿伊哈努姆，多数居民为镇灵，城主加尔沙斯普。“兰沙赫尔王”卡乌斯巴依死去后，他年幼的孩子阿塔希尔即位，图兰哈冈（萨勒赫城主）摄政，图兰哈冈有叛逆之心，未称王便被呼玛云（蓝宝石之城城主）起兵讨灭。第二王朝呼玛云为国王，其妻子为第一王朝“兰沙赫尔王”之女，几年后死于镇灵叛乱，之后三十多个藩王叛乱，叛乱被加尔沙斯普（“圆形剧场”城主）平定。第三王朝加尔沙斯普平定叛乱后称王，几年后去世，都城大火七日不绝而毁灭，民众迁往蓝宝石城。撒勒布·道莱王当兰沙赫尔王的国都灭亡。蓝宝石的穹顶下，一度向暴君献上缰绳的显贵们挣脱了辔头，却又追随僭称智者的伪主，跃入了烈日荒墟间血与沙的游戏。骄傲的图莱杜拉人啊，他们早已将缄默众贤的告诫抛诸脑后，直到野心随着库列甘的名号一同飘散，才意识到叛逆的代价。当残暴的哈冈们骑着骏马踏来时，已无勇士护卫千柱的花园。沉湎于镀金之梦的人们，唯有如良畜般再度俯首于新的主人。贪索的兵士与税吏自远方到来，盘剥着早已麻木的城民。尽管自称仁善的苍青色王廷对子民的刻薄未曾有过不同，卑屈于外邦的铁蹄下，总会有人向诗化的过往寻求慰藉，在炽日不及的冷影中，低声歌颂着呼玛云王短暂的霸业。为将同胞自枷锁中解放，年轻的贵胄古达尔兹踏入了深埋于沙下的圣堂。传说那宫殿属于鹮之王，七贤僧中至为睿哲的掌书，亦是星与月的秘使。那几乎可谓无望的绝策，因救国的筹谋皆难敌过征服者灼耀如日的军势，提纳勒的智者早已远遁，那沙中伟贤恢弘的圣殿亦已湮没于铁色的砂海。然而流泉干涸的至深处，竟果真因他的祈祷，传来了神圣而古老的回声…「属血气的贵子，你渴慕烈日与砂岚的伟力，妄图用流沙再度筑起高塔」「我能将众先知与信士未曾见证的权赐给你，一如将苦咸的水赐给渴驼」「只是花与绿洲的大主人曾经留下三个谜题，赠予我所侍奉的赤土之王」「但那智慧而明睿的主宰呀，直至沉沦于黯毒的梦乡，也未将其意晓畅」「若是你能破除这令我困扰已久的迷茫，我便要将国权赐予你作为馈贶」那年轻的古达尔兹便如此回答：「司圣谕、文书、医药与智慧的圣者呀，那便请你向我述说古老的谜题」「我将以赤诚的真意奉上凡人的回答，因我的征途中容不得虚妄的阴影」于是，贤僧中的智者便开口说：「何物迭起而兴，却又接踵而亡？」「它如月轮满盈，亦如月轮溃丧，」「智者与愚者们，时常将它渴望，」「它却总如潮汐，涨落在人心房？」那年轻的古达尔兹便如此回答：「你说的是阴谋与背叛，空谈良善之义的智者袍下孳生的荏弱与疲软」「正是因为他们的软弱，宝蓝色的穹顶才会沦丧，众民才会潸然哀叹」贤僧中的智者大笑，并未裁辨年轻的贵胄给出的答案，却依然如约定那般，将「虚诈」的权柄交予少年掌间。就这样，古达尔兹的名讳渐隐于史册，取而代之的是撒勒布·道莱王的威名。昔日不容谎言的勇士以狡计织就了王权，于苍暮的虹光下投映出黄金时代的蜃景。直至最完美的谎言也无法制御贪餮的权欲，直至砂之主的构装也无法伪造显圣的幻象，鹮之主的声音再度响起，向他询问同样的问题，年老的君王伴着居尔的哀歌给出了最后的回答：「圣者呀，我能吐出的唯有虚诈，即便是我也无法将我的话语信任」「我如今所能做的唯有拔剑死战，至少不该再辜负前来救援的兄弟」「圣者呀，无论你寻求真知亦或是谎言，尽可自我尚温的血中品尝」此后王权便被异族的代伊们所篡夺，直到列邦皆被砂原之风碾作尘埃。而如同收纳如琥珀如蜜的月光那般，圣者将君王最后的真意收作答案[16][38]。第四王朝蓝宝石城现任城主古达尔兹，与妄图称王的新居尔城城主大战三日，（声称一开始不能攻克居尔城，但在所谓赤王的幻象加持下轻易取胜），而后迁入居尔城，（声称在赤王幻象祝福下）称“撒勒布·道莱王”。在胜利后的十年，民众迎来了“苍暮细雨”的繁荣的十年，就连附近的海盗（即代伊）都来称臣（此句多半为假，因为根据后续碑文可知，海盗在朝贡路上也是烧杀抢掠无恶不作，甚至将城市拜特拉哈屠城，将城主悬首示众直至腐烂，还立其女为新城主）。道莱王杀死了自己的皇子（并没有说原因，猜测是他的儿子有谋反之心，或许是引用了国外某个皇帝杀子的故事）装作悲愤，并令数千奴隶陪葬。之后自己的兄弟，子嗣与侄儿为皇位互相开战，年老的道莱王向海盗求助，海盗一边烧杀抢掠一边平定战事，最后终于向道莱王宣战，道莱王和他年纪最小的弟弟一起对抗，终不敌。直至最后，道莱王才明白自己背叛了自己的亲人与忠臣[38]。第五王朝“代伊王（海盗）”穆扎法尔死后，由其子“绿之王”祖玛鲁德称王，绿之王某天做梦梦见沙漠变为雨林会更加繁荣，于是大力种树。代伊王告诫自己的儿子要多生孩子，以免发生道莱王的悲剧，所以绿之王有九十九个孩子，四十九个女儿，五十个儿子。绿之王死后，他最有实力的两个儿子——巴达那与穆祖丁为了王位互相争斗，最后巴达那取胜。巴达那即位后逐渐贪图享乐，变成了整日与美食、美酒与游猎为伍的暴君，甚至将自己的忠臣挖去双眼。忠臣离开王国，流浪世间，成为老朽的盲眼诗人，向沙之民收集沙海的诗篇。王子法拉玛兹与当时还是舞女的玛海菈（镇灵之后）有了除掉国王的念头。玛海菈误入遗迹遇到鹮之王，回答祂的问题，图特因此给予她「被爱」的能力[38]。巴达那死后，新王法拉玛兹即位，因其年轻，王后玛海菈摄政。[57]不久便禅位于其叔父，“蓝宝石城城主”，相国瓦拉什。国舅格拉兹起兵推翻瓦拉什，复立法拉玛兹为王，自己摄政。不久后罢免法拉玛兹，自己称王，玛海菈为皇后。蓝宝石城贵族为拉瓦什复仇，推翻格拉兹，复立法拉玛兹为王，王后摄政。勇士之城城主反叛，自立为王，王后平定叛乱。之后法拉玛兹的十二个叔父全部叛乱，战争连连，民不聊生。战争平息后，法拉玛兹再度即位，王后玛海菈一心一意想要成为皇后，但不在意国王是谁，于是法拉玛兹下毒杀害玛海菈，玛海菈中蛇毒而亡。不久奴隶兵与市民里应外合推翻法拉玛兹，并将其流放雨林。[58]在这之后，沙漠子民厌倦了战争与王朝，在盲眼忠臣领导下前往沙漠边缘建立了新的定居地，即阿如村。诸国灭亡后迷宫的游人曾经有一个人类孩子辛娜梦，她迷失在了森林里。循着森林王留下的足印，她来到了老虎的花园。「嗷呜，每一种走兽、每一种飞禽，都曾在我的五脏府中巡礼一番，最后回归大地。」「我是森林的王，杀了许多，保护了许多。唯独只有你这样的小人，我还没品尝过。」老虎这么说。当然了，那孩子当时还没有忘记你们的语言，自然也听不懂王所说的话。「嗷呜。」孩子说。森林王反倒觉得很有趣。往往他一旦开口说话，人类就会躲进木头的房屋里，走兽伏倒在地上不敢动弹，禽鸟远远地飞向太阳。就连长鬓的威武大猫，也躲进树丛。「嗷呜，不懂礼数的小东西。也好，那就由我来教你森林的道理，还有森林的语言。」「从今往后，你就是我的近侍。森林的宫殿将对你敞开，林中诸兽也不能加害于你。」「记住，这个世界只是森林的梦。有一天你会从现实中醒来，来到无边无际的猎场。」「我们的猎物所前往的地方，我们总有一天也会抵达。这个道理，你永远不要忘记。」在一代的森林王老去之后，新的王将会替代他。你们、我们，森林中的生命都是这样。这个近侍与王一同老去之前，遇到了同样迷失、不踩碎任何落叶而踏入了宫殿的孩子，并将一切教给了她。她曾经收留许多迷失林中又不曾践踏野花的孩子，通晓百兽的语言，随侍森林王，为王守卫迷宫。如同这个故事传给我一样，那个孩子也将教导传给了之后的孩子。后来，有的孩子成为了森林的守护者，并将森林的语言、保护王土的责任传给更多人，也有人在面对他人的巨大痛苦时，决定告别生涯尽头的猎场，成为令恶兽胆寒的猎手。[59]森林王诞生之时，曾得到大慈树王赐予的宝冠。它最后传达给了第一个循着自己的足印，穿过迷宫的少女。[60]法拉玛兹自黄金之地而来的浪客，身心带着争斗留下的伤痕，
曾是一邦的王子，如今却迷失苍翠曲折的迷宫之中。苍老的森林王嗅到权力的血腥气息，不禁蹙眉长叹…
持白弓的猎手受召而来，猎捕那不属于森林的困兽。阴翳已然晕染开来，「死」的低语在林木迷宫中找寻着方向。追随着流亡者的脚步，诅咒自沙海蔓延，蚕食「生」的领地。穿越翠色的回廊与狭道，自陌生气息中她发觉了来者的目的…游走于记忆与野心之间，他却在凌乱的水声与鸟鸣之中迷失…「你已中了我一箭，无礼的入侵者！下一箭将寻着你的心脏而去。」「莫再流连雨林，莫侵扰孩子们的美梦。这里没有你想要的王冠！」森林中矫健的女猎手如是警告，从未有猎物曾逃过她的箭矢与锐目，但不知为何，她却将长弓微微下压几寸，刻意没有射中那迷途的人。草木为之困惑，睡梦中躲藏的孩子们为不必挥洒鲜血而长出一口气…洞晓一切睡梦的森林之王明知她的用意，发出一阵震颤巨木的低吟：「那凡人与你不同，来自污秽之地，手上沾着脏血，心中满是欺瞒与妄想。」「但森林只收留天真的梦，鲜血只为狩猎与牺牲而挥洒，容不得任何欺瞒。」「若你以为他仍有资格在森林迷宫之中寻回荣誉，就引导他折去纯白枝条…」「到那时，明月与新星将赐予他纯真的灵智，抛弃苦酒一般的回忆与欲望。」于是，她再度握紧纯白的猎弓，将流浪者驱逐出迷宫的深处…之后法拉玛兹放弃野心，被雨林收留，折下雨林的枝条打造属于自己的剑。[61][62]西福斯第五王朝覆灭前，玛海菈在被蛇毒毒死死前貌似察觉了什么，让仍是幼子的西福斯赶紧逃跑，后来他沦为了奴隶市场的商品，被盲眼的忠臣买走，收为徒弟。最后学成的西福斯成为英雄回到了沙漠，守护着“蓝宝石之城”图莱杜拉。而另一边法拉玛兹流放雨林后，逐渐沦为一名佣兵，法拉玛兹受到了敌王的委托杀死了西福斯，最后的城市“蓝宝石之城”图莱杜拉在没有西福斯的庇护后迅速衰落消失。图莱杜拉灭亡后留有后人[38]。最后失国王子法拉玛兹的一生杀死了自己的父亲与国家。[63][64]緘默之殿最初由尊主赫曼努比斯在神王阿赫玛尔死后带着他的信仰者连同长年担任祭司的提纳勒人一同到图莱杜拉城所建，职责是保护和管理赤王文明所遗留的知识，后在与居尔城的战争中因被沙漠民排斥不得不离开沙漠，进入雨林，与教令院曾建立合作。[65]
