# 兹白

琅玕国有一神使，即兹白，佑得此地风调雨顺，引导并庇佑人们。天使赠予人们玉石，并指引他们用其筑起通天的祭台以聆听来自高天的指引。原是天使的兹白后来被选上成为月使，在月亮上与图特等月使共事。后来兹白为庇护琅玕国民硬接天钉而陨落，伊斯塔露在她死去的前一刻，凝固了她破碎的灵魂本源，并分别置于蓝眼上尸神兹踞，红眼中尸神兹踬和金眼下尸神兹蹻体内，后旅行者和钟离联手将其灵魂完成三合一，助其复活，目前住在「白驹逆旅」[22]。
