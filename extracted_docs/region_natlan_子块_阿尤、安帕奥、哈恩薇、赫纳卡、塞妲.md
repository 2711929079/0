# 阿尤、安帕奥、哈恩薇、赫纳卡、塞妲

嵴锋龙。阿尤，卡齐娜的龙伙伴。赫纳卡&塞妲，阿尤的爷爷奶奶。安帕奥&哈恩薇，阿尤的父母。魔神任务第五章第一幕：深渊入侵时，阿尤的爷爷奶奶不幸战死，阿尤一度因此十分消沉，食用了一些针对「渊蚀综合症」的清幻剂后有所好转。后来见到了未能回归夜神之国的爷爷奶奶的灵魂，终于恢复了活力。魔神任务第五章第五幕：阿尤的爷爷奶奶在深渊入侵的战斗中不幸牺牲，灵魂未能回归夜神之国，但为了能继续守护阿尤，不愿让茜特菈莉帮助它们返回夜神之国。而是以身陪伴阿尤直到灵魂消散。
