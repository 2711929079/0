# 「法罗赫之子」

团长为法罗赫老爹。其雇主为教令院学者熙蔓·法罗赫扎德，为法罗赫老爹的女儿，小时候和兰那罗一同玩耍，长大了执着于搜寻、捕获兰那罗。[3][4]卢妮雅，团中的大姐，沉溺于七圣召唤，导致旅团经费常常不足。[5]世界任务正法矩书中，受雇于教令院有钱老头，卢妮雅带领手下米农、齐里寻找须罗蕈，碰上了来给兰纳真寻找须罗蕈的旅行者，并在旅行者获取须罗蕈的途中使绊。最终卢妮雅利用旅行者，获得了须罗蕈。旅行者把卢妮雅一伙打跑，夺回了须罗蕈。在千壑沙地·巨人峡谷的遗迹中探索，仅获得一些雕刻着古老文字的石板。本想贩卖石板给教令院学者，但因庆功宴醉酒误事，石板被不小心破坏。计划落空后试图制作假石板，或许是手艺不精，能被一眼看出真假，石板最终被遗留在原地。[6]
