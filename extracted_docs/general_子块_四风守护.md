# 四风守护

为了守护这来之不易的一切，狮牙骑士温妮莎建立了西风骑士团；巨龙特瓦林为巴巴托斯不羁的风与歌所沉醉，降落大地，向他效忠；北风骑士在蒙德城结束流浪。「南风之狮」温妮莎、「西风之鹰」西风骑士团、「东风之龙」特瓦林和北风骑士，这便是蒙德初代四风守护 。在生命的最后，温妮莎来到了风起地，她永别了衷心守护的城邦，留下传奇与一株树苗。这株树苗在千风的眷顾中，在无数的日月照耀下，成长为参天巨木。离开蒙德的温妮莎登上天空岛，成为神灵，此后千年里，一直守望着蒙德。
