# 多莉技能详细介绍

多莉是《原神》中的雷元素四星角色，须弥的商人，称号"千朵玫瑰带来的黎明"。
元素战技「神奇魔法」：
- 释放雷元素能量，对周围敌人造成雷元素伤害并施加"雷元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「聚宝盆」：
- 创造一个持续8秒的雷元素领域，领域内的敌人会受到持续雷元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "魔法的祝福"：神奇魔法命中敌人后，有15%概率触发"感电"效果。
- "雷元素增幅"：多莉的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "聚宝的守护"：元素爆发持续时间内，多莉的雷元素伤害提升15%。
