# 索赫蕾之死

在一次考察时，赞迪克正在研究遗迹机关，而索赫蕾因为好奇而跟踪了他，撞见赞迪克研究遗迹机关之事。赞迪克立刻将对自己有好感的索赫蕾灭口（掐住她的脖子直到她窒息），并果断将其死因伪装成被长鬓虎袭击。在与索赫蕾的冲突中，赞迪克误触机关导致遗迹巨像对考察队成员乱开炮，造成无数死伤。…死者…陀裟多索赫蕾…多处外伤…撕裂伤…内脏挫伤…大出血……但致命伤在颈部…舌骨断裂…机械性窒息…难以断定确切死因…''—解剖记录[78]赞迪克谎称是自己随机应变关闭了因故障自动启动的机关，还借此坚持要将机关拖回教令院进行解题研究和反向重建测试。虽然因私自行动被莎娜玛呵斥一顿，但他也被当成了拯救整个考察队的英雄[79]，名声传遍整个教令院。以自己名声，赞迪克得以招开学术演说，阐扬自己的观点，因为其涉及人体改造和对神明的不敬，其见解被斥为歪理邪说，并被放逐出教令院。然而教令院对索赫蕾的尸体开展验尸工作，发现她真正的死因是人为造成的机械性窒息，[78]赞迪克被列为杀人嫌疑犯，因此还被教令院永久除名。之后失踪的赞迪克前往调查魔麟病与「法留纳传说」[80]。
活力之家被放逐到沙漠后，赞迪克前往沙漠的魔麟病医院活力之家，化名（在名单上主治医师名字是被涂黑的四格）成为医治病人的主治医生，对病人进行惨无人道的医治（人体实验）。赞迪克的研究很简单，魔麟病是体内元素力异常造成的，就将拥有浓厚元素力的魔神残渣注入人体，体内元素与魔神残渣互相反应，就能根绝魔麟病了。简单来说，就是将正常体质的人被注入后会立刻死亡或陷入癫狂（且还带有魔神意识，大部分是魔神的怨念）的魔神残渣注入人体。
例如病人体内雷元素过于浓厚引发魔麟病，就注入火元素的魔神残渣引发超载，把体內元素力反应掉，魔麟病就根除了。但大部分的病患都因为体内激烈的元素反应被炸死了，活下來的也被赞迪克折磨到肉体不成人形且精神失常。一位名为阿巴斯的病患被折磨的精神失常，为了逃离活力之家每日每夜地凿通道，最终逃脱。…「我能听到，沙漠里的毒风，艾文悼念哈基姆时愤怒的眼泪，神的话语，自始至终，我全都听得到。」「先是哈基姆，然后是艾文，接着是小阿尔伯兹，他们离开，融入黄沙，只剩我还在煎熬。但我破破烂烂的心和身体，再也忍受不下去了。」「庇护我们所有沙漠民的神，比沙暴还雄伟，比日光更辉煌，求你停下这场考验吧。」「求你解救我，躲开恶医的刀子，躲开让人浑身疼痛的药剂，让我回到家乡的田地里去。求你，求求你…」「…敲啊，阿巴斯，使劲地敲。只要敲对地方，你就能敲碎这些泥墙，再也不用吃草吃蜥蜴了。」「敲啊，阿巴斯。不要害怕，你能出去。不要害怕，你能出去。不要害怕，你能出去…」「神啊，如果你仍然愿意承认信徒阿巴斯是你的子民，请向我显圣吧，再对我说一次话，我还不想溶入黄沙…」—活力之家外部 阿巴斯刻下的文字[81]
