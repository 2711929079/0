# 博士技能详细介绍

博士是《原神》中的雷元素五星角色，愚人众执行官第二位，称号"博士"。
元素战技「雷之实验」：
- 释放雷元素能量，对周围敌人造成雷元素伤害并施加"雷元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「雷之领域」：
- 释放强大的雷元素能量，对周围敌人造成雷元素伤害，并创造一个持续8秒的雷元素领域。
- 冷却时间：20秒。
- 能量消耗：80点。
固有天赋：
- "雷之祝福"：雷之实验命中敌人后，有15%概率触发"感电"效果。
- "雷元素增幅"：博士的普通攻击和重击造成暴击时，恢复3点元素能量。
- "实验的威严"：元素爆发持续时间内，博士的雷元素伤害提升20%。
