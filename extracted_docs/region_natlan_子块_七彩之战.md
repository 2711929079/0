# 七彩之战

深渊入侵，战争中维奇琳透过得意秘术制作迷宫困住众多深渊魔物并获得胜利，被称为七彩之战，此战在纳塔众多深渊入侵的惨烈状态中排行很前面。「我们（纳塔人）的敌人，正是失去了「记忆」，失去了所爱，失去了「心」，才最终堕入深渊的吧？而拥有「记忆」的我们（纳塔人），也拥有它们所不具备的力量。」为了能让纳塔人拿起这柄强大的武器，维奇琳找到一种绝佳的媒介，那便是「色彩」。越是被珍视的记忆，色彩也越鲜明。而记忆将要被淡忘的时候，总是先开始褪色。因此，色彩的秘术最能适配记忆的力量。怀有「记忆」之人，将在鲜艳色彩的指引下穿越这座迷宫。而没有「记忆」的敌人，则永远无法跨过面前的壁障，受困于此。维奇琳过世前，将打造迷宫的秘术传给茜特菈莉，但茜特菈莉自己并不知晓。她的一部分意识留在七彩之战战场的迷宫。维奇琳过世后，她的徒子徒孙们不得不修改了考核指标，现在变成在茜特菈莉面前坚持足够长的时间就可以。之后七彩之战相关资料佚失，无人能考其真实性[127]。
