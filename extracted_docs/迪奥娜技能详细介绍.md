# 迪奥娜技能详细介绍

迪奥娜是《原神》中的冰元素四星角色，蒙德猫尾酒馆的调酒师，称号"猫尾特调"。

元素战技「猫爪冻冻」：
- 效果：释放猫爪，造成冰元素伤害并生成护盾。
- 护盾特性：吸收量基于迪奥娜的生命值上限。
- 长按效果：提升护盾强度和持续时间。
- 冷却时间：12秒。
- 能量消耗：无。

元素爆发「最烈特调」：
- 效果：创造一个酒雾领域，对敌人造成冰元素伤害并治疗领域内的角色。
- 治疗效果：基于迪奥娜的生命值上限。
- 持续时间：8秒。
- 冷却时间：15秒。
- 能量需求：60点。

天赋技能：
1. 猫爪冰墙：元素战技的护盾可以抵挡一定次数的攻击。
2. 醉韵：元素爆发领域内的角色获得冰元素伤害加成。

角色定位：
- 主要定位：辅助
- 次要定位：副C
- 核心优势：护盾保护、群体治疗、冰元素附着

培养建议：
- 武器推荐：祭礼弓、西风猎弓、绝弦
- 圣遗物推荐：宗室之仪四件套或少女的祈愿四件套
