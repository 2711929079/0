# 魔力器官

神之眼（Vision）
「导出魔力的外置器官」「神之眼」是这世上极少数受神认可者才拥有的外置魔力器官。他们通过「神之眼」来引导元素力。每一位「神之眼」的拥有者，都是有资格成神的人，因此被称作「原神」，拥有登上天空岛的资格。前身为月之轮，虽然不知神之眼是否是对月之轮优化改造而成，但月之轮确实是先于神之眼出现的外置魔力器官。月之轮是天理透过三月的力量打造而成，发放给人类用以联系三月，与神之眼不同的是使用燃素而非元素力，但与神之眼相同，获取即是卖身于命运[4]。因不明原因，雷神之眼停发了，有假说是目前提瓦特电力运用不发达，天理为防止人类学会电力，见人类快学会电力了于是停发雷神之眼及时止损。高天之上是神的领域。「神之眼」的持有者在地上行走，当他们离世时，受选者将升入天空。
相传通往天空岛之人，在神之眼中——在幻视中，看到的天梯也是如此螺旋的形态。我们走向宇宙，或者下到深渊。并没有差别，因为都是未知的漆黑。
神之心(Gnosis)
「神之心」是更高位的元素控制中枢,是七神之座的象征。透过第三降临者的遗骨打造而成。七神不需要「神之眼」这种初级器官，作为替代，神灵的魔力器官与天空岛共鸣相连…也就是「神之心」了。目前冰之女皇因不明原因收集神之心。
