# 纯水精灵与梅洛彼得堡

为了实现厄歌莉娅的理念将水神的温柔与恩泽散布于大陆，纯水精灵流往各国[29]。其间厄歌莉娅为一些犯下罪行而良心未泯的犯人提供了赎罪地，让流放者为其看守原始胎海。这个区域最后形成了梅洛彼得堡。[30]
