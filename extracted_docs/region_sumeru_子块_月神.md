# 月神

三月女神每到夜晚，就离开珍珠色的宫殿，行走在沙漠里。月莲在她们的脚边开放。[1]
草龙草龙身为草元素生命的顶点，是须弥曾经的统治者。它曾在沙丘下的庞大空洞中修筑行宫…
沙虫曾是它的廷臣，飞蛇曾是它的友伴。但在今日，它们一并失去了高贵的神智与光彩。[2]
