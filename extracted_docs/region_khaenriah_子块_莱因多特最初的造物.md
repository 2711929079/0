# 莱因多特最初的造物

莱因多特最初的造物，众深渊魔物名录中的长兄，是一头独角兽。但它没有完成自己的使命，到达哲学之彼岸[23]。
深秘院负责科研的机构，业务包括占星、研究赤月与深渊等，院长为「贤者」海洛塔帝，副院长为艾弗提尔·维特[24][25][14]。
“至高的王上啊，深秘院向您呈上预测的结果；双星早已被大地的引力捕获…经历了漫长的归回周期，如今它们的轨迹将要再度交错。”[26]
