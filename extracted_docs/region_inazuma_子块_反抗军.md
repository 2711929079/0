# 反抗军

在珊瑚宫的组织下，失去神之眼和害怕失去神之眼的人成立了反抗军。万叶的友人反对眼狩令，在御前决斗中不敌九条裟罗，被雷电将军的无想之一刀惩戒而亡。万叶为了不让好友的神之眼被砌进神像而闯入御前夺走神之眼因此被幕府通缉。社奉行因认为在三百年前未能阻止五传覆灭对枫原家有亏欠，于是帮助万叶逃离稻妻。之后万叶被暂时安置在璃月的南十字船队。
岩藏流与战争影响岩藏流的现任掌门岩藏光造忠心于幕府，主动上交了自己的神之眼，并带着徒弟去到幕府军中效力。他的两名大弟子因反对这一举动而被逐出师门，其他弟子也因为忍受不了军中氛围而落草为寇。曾经著名的岩藏流最终因战争而解散了。绀田村曾有着繁荣的印染产业，但在战争期间因许多年轻人都被征召入伍而导致村子陷入萧条。
