# 北斗技能详细介绍

北斗是《原神》中的雷元素四星角色，南十字船队船长，称号"无冕的龙王"。
元素战技「捉浪」：
- 蓄力攻击，根据蓄力阶段造成不同程度的雷元素伤害。
- 一段蓄力：造成中等范围伤害。
- 二段蓄力：造成大范围伤害并击飞敌人。
- 冷却时间：6秒。
- 能量消耗：无。
元素爆发「万雷天引」：
- 释放雷元素能量，对周围敌人造成雷元素伤害，并创造一个持续10秒的雷元素领域。
- 冷却时间：20秒。
- 能量消耗：80点。
固有天赋：
- "流浪者的经验"：捉浪命中敌人后，有15%概率触发"感电"效果。
- "雷浪的回响"：元素爆发持续时间内，北斗的普通攻击和重击造成雷元素伤害。
- "南十字的荣耀"：队伍中的角色获得"雷元素伤害提升10%"的加成。
