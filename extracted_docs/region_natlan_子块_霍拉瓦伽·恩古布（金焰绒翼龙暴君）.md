# 霍拉瓦伽·恩古布（金焰绒翼龙暴君）

绒翼龙，500年前坎瑞亚灾厄时期英雄孟尼利克（花羽会）的龙伙伴。“第二位前来拥抱英雄特诺切的，是勇猛的孟尼力克与他忠诚的伙伴恩古布。”继承了「焰主」修库特尔的血脉「焰主之祝」，因此孟尼力克曾与其一同踏上火山的圣路来帮助它摆脱终将发狂的命运，可惜失败了。如今的恩古布已彻底沦为一条发狂的、拥有超然形体的异种绒翼龙。
