# 1年前

迪卢克回归蒙德迪卢克的孤独游历持续了三年时间，途中为了探明父亲「邪眼」的来源多次捣毁愚人众据点，引来多位执行官亲自出面。迪卢克险些丧命，最终被奈芙尔引导的挪德卡莱地下情报组织救下。据对方所说迪卢克一直是情报组织的关注焦点，后来迪卢克也加入了情报组织。三年后，已成青年的迪卢克回到了蒙德城。[149]
