# 占星术

繁星昭示世人的宿命，凡人仰望天空却不解其意。教令院的明论派就是研究占星术的派系。枫丹人也会对占星术感兴趣，因此莫娜在蒸气鸟报上还有关于占星的特别专栏。
水占式占星术
提瓦特的星空非常奇妙，似乎隐藏了世间一切答案[92]。星盘是研究水占式占星术的重要道具，而莫娜的星盘来自枫丹。
