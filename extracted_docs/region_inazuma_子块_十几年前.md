# 十几年前

九条裟罗一名年幼的天狗少女在山林里遇到魔物袭击，奋力逃跑的过程中意外获得神之眼。在获救后，掌管天领奉行的九条孝行看中了她的潜力，于是收养了她，赐名为九条裟罗。后来九条裟罗成为幕府军中的大将[78]。鹿野院平藏幼时的鹿野院平藏在一次祭典上结识一好友，好友经常送给他一些新奇的小玩意儿，后来平藏得知这些都是好友偷来的后与其绝交，并把之前收到的礼物全部丢掉只剩一颗曾经象征友谊的鹅卵石。一年后的祭典上，平藏鬼使神差的带着那枚鹅卵石来到祭典，见到了倒在地上的好友，好友是为了保护自己装有鹅卵石的钱袋不被小混混抢走，而被匕首刺中了胸膛。好友死后，平藏五味杂陈，誓要对抗世间所有的“罪”[79]。荒泷一斗儿时的荒泷一斗与其父母因为鬼族身份，被村民诬陷是村里恶性事件的罪魁祸首，被迫背井离乡。后来，父母因病去世，荒泷一斗流落街头时被一名老奶奶「鬼婆婆」收留[14]。千织小时候的千织贪玩误入神里家的院子，与神里绫华相遇，两人玩了好久，最后千织被人发现，被提出门[80]。托马蒙德出身的托马担心父亲喝不到蒙德的酒，带着蒲公英酒独自乘船前往稻妻，结果遭遇巨浪，自身不幸落入水中昏迷不醒，最终漂到了稻妻的海滩上。醒来后的他一无所有，也没有找到父亲，被神里家收留，成为家政官[81]。
