# 基尼奇技能详细介绍

基尼奇是《原神》中的岩元素五星角色，纳塔的角色，称号"岩之守护者"。
元素战技「岩石屏障」：
- 释放岩元素能量，对周围敌人造成岩元素伤害并生成岩元素结晶。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「岩石领域」：
- 创造一个持续8秒的岩元素领域，领域内的敌人会受到持续岩元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "岩石的祝福"：岩石屏障命中敌人后，有15%概率触发"结晶"反应。
- "岩元素增幅"：基尼奇的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "岩石的守护"：元素爆发持续时间内，基尼奇的岩元素伤害提升15%。
