# 天钉

寒天之钉
在冻结雪山的「长钉」降下时，风雪遍天。天降之柱崩裂为三。具体所指是，如今恢复了能量，悬浮在龙脊雪山上空的神秘结构体。作用是为了封堵地脉中的深渊能量、禁忌知识，能将其净化为淤泥目前地图上存在寒天之钉的位置：龙脊雪山、层岩巨渊地下矿区、夜神之国其他曾落下天钉处：鹤观、永恒绿洲、须弥雨林某处、星落湖、亥珀波瑞亚坎瑞亚曾经为了天钉的力量派人搜集天钉
