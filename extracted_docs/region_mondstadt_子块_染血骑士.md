# 染血骑士

纯白骑士罗南师从鲁斯坦，追寻着绝对的正义。曾救下一位仕女，并与仕女交换了鲜花，仕女深深爱上这位和他只有一面之缘的骑士，但这是一个不会有结果的爱情：因为罗南又转过头落实他的「正义」去了[108]。因其实力强大比肩魔神与其理念，曾被众人认为在鲁斯坦之后会接替鲁斯坦成为最强的北风骑士，唯一在成为北风骑士前就被王狼认可[29]。罗南一生都只想着落实正义，却从未想过自己心中的正义也有可能成为「不公」。为了追寻正义，他斩遍无数魔物，因为染上魔物的血身姿也歪曲成魔物之形。在路途中，来自须弥且同样身染黑血的薇瑞迭瑟恩被他误认为魔物斩杀。[109]直至最后，那追寻黑暗塔的罗兰公子，已来到塔前。[110]鲁斯坦为了不要让偏执的罗兰步步走向深渊，打算将所思写成信交给罗南，但在此之前就身亡，此信成了遗言。艾伦德林接替一直寻找罗南但已为时已晚[29]。后续见至冬#挪德卡莱·猎魔骑士。
忍冬之树梦杜林做了一个很长、很长的梦…梦见他与大家走了很远、很远的路，来到了歌声飘舞，绿草芬芳的土地。梦见他与这里善良的人们一同欢歌，与宝石一般美丽的巨龙在空中舞蹈。梦醒了。睁开眼时，是风雪嘶吼的山岳上空。翠绿恬静的大地已经被火与血染红，而苍色诗人的琴声几乎被咆哮淹没。而那宝石般美丽的巨龙如同爱人般，将利齿没入他的脖颈。「再见了，妈妈，我的旅途结束了」「沉睡在白银般闪耀的雪中也不错」「再见了，美丽的诗人、美丽的龙」「如果我们能在不同的时间、地点」「相遇、欢唱、共舞，那该有多好」垂死的他由衷地想。「在我的血管当中鼓动的深刻祝福」「我所诞生的漆黑宇宙的美丽图景」「就由你们继承吧」[111]
