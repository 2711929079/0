# 西福斯

第五王朝覆灭前，玛海菈在被蛇毒毒死死前貌似察觉了什么，让仍是幼子的西福斯赶紧逃跑，后来他沦为了奴隶市场的商品，被盲眼的忠臣买走，收为徒弟。最后学成的西福斯成为英雄回到了沙漠，守护着“蓝宝石之城”图莱杜拉。而另一边法拉玛兹流放雨林后，逐渐沦为一名佣兵，法拉玛兹受到了敌王的委托杀死了西福斯，最后的城市“蓝宝石之城”图莱杜拉在没有西福斯的庇护后迅速衰落消失。图莱杜拉灭亡后留有后人[38]。最后失国王子法拉玛兹的一生杀死了自己的父亲与国家。[63][64]
緘默之殿最初由尊主赫曼努比斯在神王阿赫玛尔死后带着他的信仰者连同长年担任祭司的提纳勒人一同到图莱杜拉城所建，职责是保护和管理赤王文明所遗留的知识，后在与居尔城的战争中因被沙漠民排斥不得不离开沙漠，进入雨林，与教令院曾建立合作。[65]
