# 原神武器系统详解

《原神》武器系统丰富多样，不同武器适合不同角色和玩法。
武器类型：
- 单手剑：适合近战角色，如刻晴、琴、行秋等。
- 双手剑：高攻击力，适合重云、诺艾尔、迪卢克等。
- 弓：远程攻击，适合甘雨、温迪、宵宫等。
- 法器：法术攻击，适合丽莎、莫娜、可莉等。
- 长柄武器：长距离攻击，适合香菱、钟离、胡桃等。
武器稀有度：
- 一星：基础武器，无特殊效果。
- 二星：普通武器，基础属性较低。
- 三星：常见武器，有基本效果。
- 四星：优秀武器，有强力效果。
- 五星：顶级武器，特效强大。
武器选择原则：
- 主C角色：优先选择高基础攻击力和暴击相关属性的武器。
- 辅助角色：优先选择元素充能效率和元素伤害加成的武器。
- 武器特效：根据角色技能机制选择合适的武器特效。
武器升级：
- 使用武器经验素材提升武器等级。
- 武器突破提升武器等级上限和基础属性。
