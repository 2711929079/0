# 阿蕾奇诺技能详细介绍

阿蕾奇诺是《原神》中的火元素五星角色，愚人众执行官第四位，称号"仆人"。
元素战技「火焰鞭挞」：
- 释放火元素能量，对周围敌人造成火元素伤害并施加"火元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「火焰盛宴」：
- 释放强大的火元素能量，对周围敌人造成火元素伤害，并进入"狂怒"状态。
- 冷却时间：20秒。
- 能量消耗：80点。
固有天赋：
- "火焰的祝福"：火焰鞭挞命中敌人后，有15%概率触发"燃烧"效果。
- "火元素增幅"：阿蕾奇诺的普通攻击和重击造成暴击时，恢复3点元素能量。
- "盛宴的威严"：元素爆发持续时间内，阿蕾奇诺的火元素伤害提升20%。
