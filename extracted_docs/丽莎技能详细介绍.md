# 丽莎技能详细介绍

丽莎是《原神》中的雷元素四星角色，蒙德西风骑士团图书管理员，称号"蔷薇魔女"。
元素战技「苍雷」：
- 点按：释放一道雷元素射线，对目标造成雷元素伤害。
- 长按：释放三道雷元素射线，攻击范围内的敌人并造成雷元素伤害。
- 冷却时间：6秒（点按）/10秒（长按）。
- 能量消耗：无。
元素爆发「蔷薇的雷光」：
- 释放一个持续8秒的雷元素领域，领域内的敌人会持续受到雷元素伤害。
- 冷却时间：20秒。
- 能量消耗：60点。
固有天赋：
- "静电场力"：苍雷命中敌人后，有20%概率触发"感电"效果，造成额外伤害。
- "药剂通识"：合成药剂时，有20%概率获得双倍产出。
- "蔷薇的雷光"：元素爆发持续时间内，丽莎的普通攻击和重击会触发"连锁闪电"。
