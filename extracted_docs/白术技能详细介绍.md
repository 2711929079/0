# 白术技能详细介绍

白术是《原神》中的草元素五星角色，璃月不卜庐的医师，称号"悬壶济世"。
元素战技「本草经」：
- 释放草元素能量，对周围敌人造成草元素伤害并恢复队友生命值。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「长生箓」：
- 创造一个持续10秒的草元素领域，领域内的队友会持续恢复生命值。
- 冷却时间：18秒。
- 能量消耗：60点。
固有天赋：
- "本草的智慧"：本草经命中敌人后，有15%概率触发"激化"效果。
- "草元素增幅"：白术的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "长生的祝福"：元素爆发持续时间内，队伍中的角色草元素伤害提升15%。
