# 妖刀的结局

妖刀终于来到稻妻，与雷电五传后人枫原万叶相识。最后妖刀解开心结，明白了造出它的刀匠生前真正的愿望，于是希望枫原万叶能够参考自己身上种种的技术，重振一心传的荣光。之后，他的意识消失，只留下自己的名字—笼钓瓶一心。万叶也尝试复甦一心传。[74]
