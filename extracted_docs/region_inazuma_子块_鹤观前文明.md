# 鹤观前文明

鹤观文明信仰着三月女神，与天空岛的使者有联系。他们非常了解三月女神，除了称他们为月亮，还将他们称为赫卡忒[5]。他们会用壁画与石根文字纪录历史。葬火之战后被降下天钉，导致鹤观天空被净化，但也被迷雾笼罩。后来该文明覆灭。[3]
清籁文明覆灭的古文明，是第一文明支系。沉眠之庭是在清籁岛还没有如今的名字，没有被染上雷色之前就静静沉眠在此间的庭苑[6]。清籁文明的特殊机关较多，譬如特别的立方体机关，与数个踩格子机关，格子上刻有凯尔特三角结。
