# 人偶甦醒

鹤观人阿釜的祖先曾误入借景之馆发现被封印力量的人偶[67]（因为此时人偶身上有关节痕迹所以看得出来是人偶[66]）。之后人偶打破身上的封印自行甦醒。[65]踏鞴砂曾经出现过一个人偶，它面貌秀丽，打扮得体，懂得掩藏身体上那些特殊的关节痕迹。若非偶尔有人提起，旁人很难看出他是个人偶。另外，人偶特有的关节痕迹似乎还会随着时间推移慢慢淡去，最终或许会消失不见，变成完全的人类模样。人偶的名字并无几人知道。只是听说有这样一个人偶曾出没于踏鞴砂地带，有人在核心地区，也有人在海滩上见过它。传说它会伫立在海边遥望一海之隔的稻妻城，那里有什么，它又在看什么，实在无从知晓。[66]之后人偶回到借景之馆，但借景之馆因不明原因被封印[63]，人偶再次被困在借景之馆内，直到桂木救下他。
人偶与踏鞴砂踏鞴砂的寄骑武士兼御舆长正的副手桂木沿着采集晶化骨髓的人开辟的山洞误入借景之馆中，发现了內心早已抑郁发狂[63]的昏迷的人偶。此时人偶已在漫长岁月中散失关节痕迹[66]，因此桂木最初认为他是打扮奇怪的人，之后才知道他是人偶。心性善良的他将纯良如白纸的人偶带回了踏鞴砂。此后借景之馆又再次因不明原因被封印（玩家需要用神居岛蜂炮轰炸山体才能开出新的入口）。桂木由人偶的将军金饰得知了他将军使者的身份，并敏锐地察觉到这可能给不谙世事的人偶找来祸患。于是他告诉人偶，要他藏好金饰，绝对不能轻易暴露自己的身份。而对自己的上司御舆长正，桂木则称倾奇者是自己在名椎滩找到的，但依然如实告知丹羽久秀在借景之馆找到的。「这金饰是将军大人所赐身份之证，但你行走世间时，若非万不得已，切不可将自己的身份透露给他人。」人偶来到踏鞴砂后，御舆长正将其成称为倾奇者，指衣着鲜丽，行为特别的人。再怎么与众不同，它也象征他是人，是踏鞴砂的一分子。其时的踏鞴砂，在丹羽久秀的领导下正发展得欣欣向荣，而御舆长正是踏鞴砂的第三顺位负责人，性格古板但刚正不阿，为了洗刷家族污名而疲于奔命。在这个温暖的大家庭中与桂木(御舆长正的副手，一名忠心耿耿的武人，早年为御舆长正所救，宣誓愿为其赴汤蹈火，大可替死，小可随侍。）、丹羽久秀（枫原家的祖宗，雷电五传一心传的一员，为一心传丹羽家继承者，其家族与赤目、枫原并称为一心三作。为人谦和，头脑聪慧，是管理一方土地民生的优秀人才。）、御舆长正、阿望、金次郎、宫崎兼雄（丹羽的副手，出身不详，主要在锻造及人员管理方面辅助丹羽进行管理，个性温良友善，在当地有着不少朋友，与御舆长正交情亦佳。）等好友一同度过的时光里，倾奇者逐渐学会了穿衣、吃饭、舞剑、锻刀和冶炼等技术，像常人一样生活。为了洗刷污名，御舆长正疲于奔命，长正的一名下属金次郎认为长正大人铸刀对他的心境颇有好处，于是建忆长正透过锻刀陶冶性情。为此，长正购买玉钢锭若干，与宫崎、桂木彻夜讨论锻冶心得。一日，踏鞴砂终于锻成长卷一把，号曰「大踏鞴长正」。名刀锻成之日，倾奇者与其他工人一同献上欢庆之舞。当时金次郎也在场。当时长正的下属阿望为『大踏鞴长正』之美所服，为之绘制图画。[46][64][65][68]
