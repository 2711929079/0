# 璃月.层岩巨渊文明

天钉降下后层岩巨渊不再适合居住。沉玉谷的先民自遥远的层岩巨渊迁居而来，他们在此地筑起了直通高天的祭台，留下无数恢宏的殿宇。[44]该文明古时的祭场曾因葬火之战埋入地下，又随着天钉陨落而隆起，最终又因摩拉克斯与若陀龙王对厮杀造成的破坏而显露出来。但曾经来往此间的人早已不再了[45]。
稻妻.樱之庭恆月破碎，海洋升起，远古的樱之庭与其他岛屿的联系因此断绝了。[13]
