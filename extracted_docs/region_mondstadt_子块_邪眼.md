# 邪眼

至少四年前，「愚人众」十一席之一「博士」为了实验计划给予了克利普斯「邪眼」。近年因风神远去，乌萨（官方漫画序章风之歌一分镜中有两头乌萨，因此其不只一头，可能为龙群，没被剿灭完毕很正常）崛起。
克利普斯之死(4月30日)迪卢克·莱艮芬德迪卢克十八岁成人礼那天[128]，「愚人众」依靠机关鸟诱发魔物暴动，迪卢克护送的车队在森林中遭到巨大魔物（魔龙乌萨）袭击。[135]事出突然，来不及联络西风骑士团。魔物又空前强大，令年轻的骑兵队长束手无策。[136]迪卢克与凯亚配合从未失手，从明处和暗处守护着蒙德的安全。那是凯亚第一次也是唯一一次失手。纵使快马加鞭，他赶到时，局势也已经无法挽回。他与迪卢克共同的「父亲」——不曾被神认可的父亲，无法当选骑士的父亲，竟然操纵着来历不明的不详力量击退魔物，而后又遭到「邪眼」的反噬。[137]为使父亲解脱，迪卢克·莱艮芬德亲手杀死了他。[128]
