# 现存仙人

削月筑阳真君，居于庆云顶，在4.4海灯节中化名“侯章”。理水叠山真君，居于琥牢山，古文记载其为理水叠山之才，内明外知，在4.4海灯节中化名“接笏”。留云借风真君（闲云），居于奥藏山，古文记载其性不增减，心无净垢，无为无法，是逍遥仙。留云借风真君善用机关之术，庆云顶上的云中仙居就为其所造。现化名为闲云并居住在璃月港。歌尘浪市真君（萍姥姥），唯一一位以人身守护在璃月港的仙人，又名“萍姥姥”。救苦度厄真君·七七销虹霁雨真君·阿圆寻瑰纳琦真君·阿嘟金鹏大将·魈，唯一存活的五大夜叉，目前镇守于望舒客栈白马仙人·兹白，居于白驹逆旅浮锦，鲤鱼仙人药君，被沈玉谷旧主斩成数段银翎翦玉玄鸟，八奇桃都三怪之一
