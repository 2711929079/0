# 乌库

「愿降下这片冰雪的天空燃烧，直到世界上再也没有一丝生命。」「愿我们化作灰烬，顺风而行，追上那抛弃了公主再也不返回的异邦人。」「愿公主梦见的蔽日的黑龙，为可憎的大地涂上赤红的毒。」「我是最后一个。再也没有留守的必要了。听说何处有人正在建立无神的国度（坎瑞亚）。那么或许他们也有对抗世界的力量吧。」[12]最后记事者乌库背弃神明成为了以非人身份存活至今的丘丘霜铠王。芬德尼尔覆灭后，其残余遗民乌库成了丘丘王，而部分遗民失去过去记忆，试图向下探索，寻找已经被遗忘的东西。遗民失忆可能是和乌库一样成为丘丘人，而丘丘人挖掘的动作就是因为试图向下探索。为了找寻平息高天怒火的方式，为了通晓命运，他们试图向下探索，寻找已经被遗忘的东西，从深埋地底的古代祭场、废都中寻求启示与答案[15][16][17]。古树树洞的另一端，过去的秘密凝结成了宝藏[18]。
伊蒙洛卡家族这就是他找到的答案——异乡的勇士终于结束了徒然的旅程，漆黑的污血从大剑的刃上点点滴落，沉重的双脚踏上已变得陌生的雪径。当疲惫的异乡人终于归往山国殿堂，为他洗尘者却仅有报死的空幽回音。「就连这里，也没有留下值得我守护的事物了吗…」「天上的你们，只是想要看到生灵涂炭的惨状吧。」「既然如此，就以钢铁与血的歌，给你们消遣吧。」异乡人将少女交给他的，本应斩碎风雪的星银留在了壁画之间。然后下山寻找充满纷争与战斗的地方，能让他泼洒鲜血的地方。[13]最终，伊蒙洛卡选择与栖身寒渊的冰霜之巨蛇殊死缠斗。「冰川的严寒甚至足以冻结时间」部族的人们中间，有这样的传言。与巨蛇决战的英雄对此嗤之以鼻。当勇士登上叹息之墙时，时值深夜，雪暴呼啸。勇士进入北境冰川时，暴风中已透入几缕日光。无论何种严寒暴雪，确无法令时间的洪流凝滞，即使英雄本身也随着记忆消逝。最终勇士陨落在凛寒之中,连热血也被冻结[19]。
（当时曾试图援助芬德尼尔的伊蒙洛卡家族的后人在之后成为蒙德御三家贵族之一）[20]
