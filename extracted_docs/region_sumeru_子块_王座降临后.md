# 王座降临后

「龙王」尼伯龙根从世界外取得漆黑之力，但在与天理的战争中战败殒命。龙王殒命后，草龙仍想要逆转局面，前往尚未破碎的世界边缘寻求更多「禁忌知识」，被招致而来用来封堵禁忌知识[4]的判罚之钉阻止。[5]判罚之钉使须弥从雨林完全变成沙漠。[5]
无名古国雨林里曾有个美丽且伟大，为新生的人子施以祝福的场所，也有直达高空的高塔[6]。后来天钉降下，平地变成山脉，古都也埋入地中[7]。此时有顶塔是躲避风沙而建起的小小祭场[8]。
