# 「盗火贤者」的再计划

在预言中，奥奇坎将斩杀火龙，成为世界的主宰。但奥奇坎没有成为人类的王者，反倒成为了希巴拉克的跟班。最终斩杀火龙、成为君主的是身为人类的希巴拉克。
在始灾的边界，库库尔坎与盗火贤者库库尔坎定下契约，令其重建的国度能够同时容纳人和龙，约定在未来古龙的族裔日后归来时不会进行阻拦；并且在深渊来临、万不得已之际，人与龙要拼合各自持有的一半「通牒金盘」，用「焰主」的血脉开启修库特尔制造的「维奇洛波奇特利」（“灭世兵器”）。虽然开启后人类会无法挺过这场浩劫，但也能创造出超越人和龙的新生命。兄弟阋墙，手握完整「通牒金盘」的哥哥至死都没有开启「维奇洛波奇特利」。库库尔坎开始怀疑自己当初阻止「维奇洛波奇特利」启动的抉择是否正确，希望寻找其他龙众，看看它们是否会对「维奇洛波奇特利」做出同样的抉择。但奥奇坎得知，一旦这个计划执行，世界将不存在人类的位置，因此加深了对龙的仇恨，决定反抗父亲库库尔坎的计划。为反抗父亲，奥奇坎统治纳塔后，将「通牒金盘」藏在天蛇船上，使父亲无法取得。库库尔坎前来找他取得金盘，并劝说奥奇坎启动「维奇洛波奇特利」，被奥奇坎拒绝；得知金盘被藏在天蛇船上，库库尔坎怒不可遏。奥奇坎被推翻后，其残躯被深渊污染，化为在奥奇卡纳塔上空盘旋的邪龙。为了让人类取得天蛇船上的通牒金牌，库库尔坎利用死去人类的灵魂，化作不同人的样貌，散布奥奇卡纳塔有秘宝「回还之玉」的消息，吸引部族的勇者进入奥奇卡纳塔的遗迹内冒险。结果因邪龙袭击，这些人无一生还。几千年的时间过去，纳塔应对深渊的办法仍然是依赖神灵和英雄，这令库库尔坎失望；而「维奇洛波奇特利」也即将失效，库库尔坎寻找能够抉择「维奇洛波奇特利」的龙的心情日益迫切。它在纳塔散布了“若要消除「焰主之祝」，就要踏上前往火山的朝圣路，让龙重新被大灵和夜神之国接纳”的流言，还挑选了一只带有「焰主之祝」的幼嵴锋龙，希望有人能带它走一遍前往火山的朝圣路，并决定是否开启「维奇洛波奇特利」。世界任务「在应许与遗忘之间」：库库尔坎把旅行者带入一个神秘空间，通过传音让旅行者收养了「小家伙」。然后又伪装成「龙遗物研究协会」研究员曼科找到旅行者，说「小家伙」角上的印记是「焰主之祝」，可以让它拥有不可思议的力量；但现在纳塔龙的身躯不足以承担这样的伟力，它成年后可能会变为凶狠的恶龙，建议将它交给「回声之子」处理，或者踏上前往火山的朝圣路，让「小家伙」重新被大灵和夜神之国接纳。旅行者决定踏上试炼，帮助它摆脱不幸的命运。世界任务「将夜晚放还于夜晚」：库库尔坎伪装成了悬木人禁地的守卫雷格巴。「小家伙」不小心挖出了悬木人与夜神之国交流所用的黑曜石像。为了将其归还，几人一起深入了悬木人的「禁地」。世界任务「群山的影子」：库库尔坎伪装成回声之子叙事人兼看守蒂图，与「小家伙」和旅行者一起修复了用于封印深渊的「缝影针」，并来到「缝影针」下方洞窟，阻止深渊涌出「镌光铭印」。世界任务「流灰之国的迷旅人」：库库尔坎伪装成寻找失踪父亲的冒险家玻娜，与「小家伙」、旅行者和小龙「绿松石」（奥奇坎意识所化）一起在奥奇卡纳塔和天蛇船探险寻找「回还之玉」。经过一番探索，众人登上天蛇船，取得了「回还之玉」（「通牒金盘」）。而真正的玻娜的记忆突然复苏，意识到这一切实际上都是库库尔坎的阴谋。随后邪龙（奥奇坎的肉体）冲出来抢走了「通牒金盘」，旅行者和派蒙前去追击邪龙。（到此为止的该部分内容，会在下文有详细叙述，此处略写）在旅行者击败奥奇坎的残躯后，库库尔坎现出真身与「小家伙」对话，宣扬了它的进化理论，并让「小家伙」带着誓约的信物，前往图兰大火山，在那里与他再见。世界任务「失落圣城与千年来者」：在圣山内部，库库尔坎操纵了几只秘源龙仆，告诉「小家伙」与旅行者火山内部的一些基本信息，并诱导「小家伙」和旅行者前往圣山之心，启动剩下两枚「象拟聚星」，解除封印，开启中央天轴。旅行者一行经过冒险，成功前往了「圣山之心」。在「圣山之心」，库库尔坎放出了最后的考验——深古秘源机龙。旅行者被机龙封印，「小家伙」面对险境，力量爆发。库库尔坎出现，引导了「小家伙」的力量，让其短暂增长智慧，并进化为嵴峰龙。库库尔坎带着「小家伙」来到了「维奇洛波奇特利」的启动装置，说当装置启动后，「源火的大权」就能回归，「小家伙」将成为「新世界」的塑造主，亦或是新的火龙王。库库尔坎让「小家伙」选择是否启动，「小家伙」选择不启动装置。库库尔坎问为什么，「小家伙」说，自己仍然相信人类，不会让旅行者和派蒙冒险，他们怎么看待自己不重要，但他们对于自己来说很重要。库库尔坎说，凭借自己的火，才让「小家伙」拥有智慧，质问「小家伙」该如何向旅行者和派蒙解释这一切。「小家伙」不以为然，它创造出返回的道路，救下了旅行者和派蒙，与他们一起击败了秘源机龙。「维奇洛波奇特利」关闭后彻底失效，无法再启动了。库库尔坎独自站在装置前，在生命的最后时刻，它感慨自己终于理解了修库特尔的心情，但仍然不认同其理念。
库库尔坎的遗言折叠我现在也…终于理解你的感受了，我伟大的兄长修库特尔啊。在生命的终日到来之前…真的会那样渴求看到一件事的结果，或者得到一个明确的的答案，甚至愿意为此不择手段。…如果是你的话，你一定会斥责我，为什么每次想做的事和说出口的话语总有差别。
如果我想让一只龙真正开启那个系统，就该从小在圣城中培养，让它仇恨这个世界，为什么还要送到人类的身边？或许是因为…我在内心的深处，还是想要选择相信吧。但是你看，太好了，它做出的是和我、和奥奇坎当初做出的一样的选择。…我的君王，我的兄长，在你诞生之日令我诞生的主人，你曾说在最后的时刻我会理解你的愿景。呵呵，但唯有这点，我不敢苟同啊…
