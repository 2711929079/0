# 毒蝎

生活在砂岩之下的大型节肢动物，擅长利用毒液捕猎。因为在受到攻击时会分泌出含有剧毒的液体，因此在很久之前就被诸多阴谋家和炼金术士们盯上，成为了各类毒药的常用原料。果然比起毒蝎，还是人心要更加可怕。
