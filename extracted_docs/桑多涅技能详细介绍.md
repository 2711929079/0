# 桑多涅技能详细介绍

桑多涅是《原神》中的岩元素五星角色，愚人众执行官第七位，称号"木偶"。
元素战技「岩之傀儡」：
- 释放岩元素能量，对周围敌人造成岩元素伤害并生成岩元素结晶。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「傀儡盛宴」：
- 创造一个持续10秒的岩元素领域，领域内的敌人会受到持续岩元素伤害。
- 冷却时间：18秒。
- 能量消耗：60点。
固有天赋：
- "岩之祝福"：岩之傀儡命中敌人后，有15%概率触发"结晶"反应。
- "岩元素增幅"：桑多涅的普通攻击和重击造成暴击时，恢复3点元素能量。
- "傀儡的威严"：元素爆发持续时间内，桑多涅的岩元素伤害提升20%。
