# 岩晶蝶

空气中飘荡不定的岩元素微粒聚合而成的纯粹元素生命，同蛺蝶一般悠游。闪光的岩元素生物,是元素上升凝合而成的产物，以无处不在的岩元素维生。在难以尽览的万古中，即使是磐岩也会做梦。据说岩晶凝成的飞蝶正是嵯峨山石之梦。
