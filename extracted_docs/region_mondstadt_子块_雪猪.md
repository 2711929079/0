# 雪猪

寒冷地带生活的野林猪，比温暖地带的同类拥有更厚实的皮毛。根据记载，这种猪倔强地生活在风霜中，在缺少天敌的环境中茁壮发展着自身的族群。传说这种雪猪可以非常长寿，而活得足够长的雪猪会在漫长的生存斗争中长得格外巨大。但这一物种已经在数百年前灭绝了——人们原本是这么认为的，但随着对龙脊雪山探索的深入，冒险家似乎发现了新的生态样本…
