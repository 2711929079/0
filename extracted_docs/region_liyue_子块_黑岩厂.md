# 黑岩厂

位于璃月南部的矿石加工厂，黄金屋南侧[1]。曾因层岩巨渊关闭导致此矿场生产压力过大[2]。在纷乱的烽燧染红大地的年岁，此地亦被诸部族的先民用作避难之所。存在某个覆灭古国的遗址。[3]
