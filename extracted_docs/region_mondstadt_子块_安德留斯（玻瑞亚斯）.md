# 安德留斯（玻瑞亚斯）

（玻瑞亚斯成爲魔神「安德留斯」前的經歷見狼与仙灵）曾经的众神皆背负爱人的责任，一直与迭卡拉庇安争斗的「北风之王」安德留斯认为自己是讨厌人类的奔狼之王，自觉无法去描绘人类的幸福生活，因此没有资格成为尘世的风之王，而放弃与巴巴托斯争夺，回到自己诞生的挪德卡莱[29]。
新神的诞生巴巴托斯旧的神像崩毁，新神诞生。风神巴巴托斯体会到了指间流淌的力量。他用这力量所做的第一件事，便是以少年的模样重塑自身形体。——因为只有用人类的形体，才能弹好少年钟爱的竖琴。他拂动琴弦，用神风吹散冰雪，劈开山峦。让新的蒙德成为自由之地吧，成为无人称王的国度。「他也会希望生活在这样的地方吧。」[3]风神还将智慧与力量悉心传授。他建造风车，教会人们加工粮食、酿造美酒。同时身为游戏与乐曲之神，他还留下了数不清的歌谣与庆典，让民众能够度过快乐的时光。[47]
