# 原神角色培养指南

《原神》角色培养需要合理分配资源，提升角色战斗力。
角色培养步骤：
1. 等级提升：使用经验书提升角色等级，突破后解锁新的等级上限。
2. 天赋升级：提升元素战技和元素爆发的等级，增强技能效果。
3. 武器选择：根据角色定位选择合适的武器，提升武器等级和突破。
4. 圣遗物搭配：选择合适的圣遗物套装，提升角色属性。
5. 命座提升：通过祈愿获得角色命座，提升角色强度。
角色培养优先级：
- 主C角色：优先提升等级、武器、圣遗物和核心天赋。
- 辅助角色：优先提升元素爆发等级和必要的天赋。
- 工具人角色：根据需求提升特定天赋和武器。
培养材料规划：
- 提前规划突破材料和天赋材料的获取。
- 合理安排副本挑战时间，确保材料获取效率。
- 关注角色复刻时间，提前准备培养材料。
