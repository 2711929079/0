# 璃沙郊-层岩巨渊古城塞

后来，大地上无数魔神与君王在为天定的王座彼此争战，星空与深渊为之失色，悲剧与恶行阻遏了山岩与流水的呼吸。自天空而来的落星不堪其扰，不顾层岩巨渊的挽留，向高天遁逃而去。自天而降的美玉回归星天之后，留下了深深的天坑。人们在其中建起坚固的城市与要塞，偏安于陨星的遗赠之中闭关自守。[16]
反抗魔神战争开始前夕，药君与沉玉谷的魇神交谈，产生了不好的预感，于是将自己的药方封藏在现如今的伏仙洞，并将自己的一部分仙力置入彩蝶中，以此延长他们的寿命。这样一来，未来的人就可以在彩蝶的指引下找到药方[52]。同时沉玉谷妖祟蠢动，于是人们建立了赤璋城垣，来抵御妖祟之患[24][56]。魔神战争期间，沈玉谷旧主因力量上无法匹敌摩拉克斯而发狂，竟打算决了碧水河，水淹沉玉谷，三仙决定反抗主人，灵渊及其眷属将沉玉谷的居民驱赶到三仙早已准备好的避难所中，药君以蛇之身躯拖住魔神，好让浮锦能够通过投珑仪式疏通水路防治洪水，最终浮锦成功完成了投珑仪式，在灵濛山，向着河道投下玉决，这些年因为举行仪式而积攒下来的仙力被释放，大地开口吞下洪水，落入水中的玉玦也迅速变大堵住河道。消耗了自身的大半仙力将上涨的水流疏通，保护了沉玉谷。在庇护所中的人们，为了纪念浮锦和药君，将两位的形象画成壁画，留在了洞窟中。[24]结果药君被斩成数段，只能维持蛇的形态，失去记忆化名长生，和人缔结契约谋生；浮锦的力量也损失甚重，数千年来只能在锦落庭修养。[24][77]帝君使用以由纯净的翠玉精琢细雕而成的仪礼宝剑磐岩结绿斩向沈玉谷旧主，未能送出的赠礼，未能诉说的有意，亦成了斩向旧友的利刃[78]。之后，帝君收服沈玉谷地区[24]。此后少有人再见到山主猊兽。经过此后数百年的变化，猊兽才成为人们如今传说中的形象[79]。沉玉谷的魔神引发了滔天洪水，自碧水河上游席卷而下，顷刻间将无数房屋和田野毁灭殆尽，人们只能搬回璃月港生活，荻花洲就此荒废。如今只剩下被洪水淹没的古老遗迹，还在水泽之下无声地守望这里。曾经的荻花洲并不是水泽，而是广袤的肥沃田野，古时的人们在此耕种劳作，生活富足。后来洪水席卷这片土地，淹没了曾经的田野与遗迹。大水毁灭了归离集，居民被迫南下迁回璃月港[50]。青墟浦也被沈玉谷旧主引发的大水淹没[59]。
