# 泉水精灵

纯水精灵卡莉露来到了清泉镇，让这个小镇有了泉水精灵的传说[116]。
庙宇废弃在坎瑞亚的灾变结束后，蒙德回复到灾变前的长时间和平状态，由于蒙德城的发展，人们的信仰也发生了变化，四风守护的信仰逐渐佚失，对应的庙宇也被荒弃。[106]
