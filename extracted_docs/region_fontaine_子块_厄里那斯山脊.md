# 厄里那斯山脊

位于枫丹南方的苍茂山脊，风景优美却罕有人迹，美露莘们聚居的「海沫村」就隐藏在这里的群山与湖泊之下。据说高耸的山岭曾是名为「厄里那斯」的巨兽之骸，时光以风与土掩埋了它原本的模样，将它变为如今无数生灵的家园。
