# 摩拉

摩拉——通行大陆的钱。世界通用的共同语言，谁人都能理解的贵金属。
摩拉乃是以岩之神的血液铸成[62]，世界上第一枚摩拉由摩拉克斯亲手铸造，如今的摩拉也必须借助岩神之心的力量铸成。
历史上的第一枚摩拉
众所周知，摩拉是一种触媒。直到今天，也流传着利用摩拉在神秘学中的功效，来强化武器的奇术。世上第一枚摩拉蕴含着最强的力量。岩王爷用这枚摩拉作为触媒，强化了一刀一剑。得到刀就可以成为璃月至尊，拔出剑就可以加冕璃月之王。花掉了。
黄金屋
黄金屋是全大陆唯一的铸币厂，制作摩拉时必须借助「岩神之心」的力量。
