# 辛焱技能详细介绍

辛焱是《原神》中的火元素四星角色，璃月的摇滚歌手，称号"燥热旋律"。
元素战技「回旋曲·燧火」：
- 释放火元素能量，对周围敌人造成火元素伤害并施加"火元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「狂想的旋火轮」：
- 创造一个持续8秒的火元素领域，领域内的敌人会受到持续火元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "炽热的旋律"：回旋曲·燧火命中敌人后，有15%概率触发"燃烧"效果。
- "火元素增幅"：辛焱的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "狂想的烈焰"：元素爆发持续时间内，辛焱的火元素伤害提升15%。
