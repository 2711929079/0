# 希诺宁技能详细介绍

希诺宁是《原神》中的岩元素四星角色，枫丹的建筑师，称号"石匠大师"。
元素战技「岩石切割」：
- 释放岩元素能量，对周围敌人造成岩元素伤害并生成岩元素结晶。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「石墙」：
- 创造一个持续8秒的岩元素领域，领域内的敌人会受到持续岩元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "切割的祝福"：岩石切割命中敌人后，有15%概率触发"结晶"反应。
- "岩元素增幅"：希诺宁的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "石墙的守护"：元素爆发持续时间内，希诺宁的岩元素伤害提升15%。
