# 栖木普、柯娅、奇梅

绒翼龙。栖木普，恰斯卡，恰斯卡的养母，柯娅的母亲。柯娅，栖木普的孩子，恰斯卡的“姐姐”。奇梅，柯娅的孩子。部族纪闻「花之归尘，羽之将坠」：栖木普十分尊重养女与亲女儿的选择，是个慈爱的母亲。柯娅曾是恰斯卡最好的玩伴，但因不理解恰斯卡为何回归人类社会而与其闹翻，大闹一番后躲了起来。后来因为孩子奇梅被阿尔帕的党羽绑架而与人类数独发生冲突，即将被阿尔帕的党羽抓捕时被恰斯卡所救。和恰斯卡一同登上阿尔帕的空中实验基地解救奇梅，并从爆炸中保护了恰斯卡，被带回花羽会养伤照料也理解了恰斯卡为什么选择回归人类社会，和她完全和解。
