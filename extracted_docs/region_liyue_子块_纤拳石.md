# 纤拳石

传说帝君在天衡山的险路布下纤拳石这种顽石，阻挡来犯之敌。某位魔神麾下的大将率军经过，见路中石，曰「纤纤而不若吾之拳」，勒令部下继续前行。不料车轮无法碾碎这种顽石，失控倾覆，这位大将坠落山间，狼狈败走。而他的狂言成就了这种顽石「纤拳石」的雅称[85]。
移霄导天真君据说，移霄导天真君的角吸收了岩王帝君岩之力的神髓，曾是整个璃月最坚硬的东西。可就在魔神战争中，劲力迸出，山岳倾倒，他为了不波及到山脚的村庄，主动让朋友将他的鹿角砍下…他将那鲜血淋漓的鹿角作为支点，撑起了整个山岳。而他自己也坚持战斗到自己流尽最后一滴血…如今，曾被他撑起的「天衡山」依然巍然屹立，他死战之中流出的鲜血则汇聚而成「碧水河」[86]。
