# No.2 「博士」多托雷Il Dottore

中文CV：吴磊 日文CV：关俊彦详见至冬#「博士」在间章·第三幕「倾落伽蓝」的尾声为流浪者赋予新的名字时，若输入“赞迪克”为新名字，流浪者会回应“给他起愚人众相关名字”时的反应「难道你觉得这个玩笑很幽默吗？」，暗示其本名是赞迪克。在月之三版本由流浪者台词「多托雷…那时候叫『赞迪克』」实锤名字。详见须弥#赞迪克。曾经去过赤王遗迹，赤王七柱之一的「鹮之王」图特看出其心术不正（虽然极为聪慧，但极度多疑、自负，似乎也没什么行事的「边界」，和他走得太近不会有好下场），在决定送他剩下的第三份礼物时犹豫过，不过他拒绝了，回答了「不需要」。于第三章·第三幕「迷梦与空幻与欺骗」在游戏中初次登场，并亮出席位是第二席。掌握切片技术，其切片最初根据「散兵」的运行原理制造而成[3]。曾经做过多次人体实验，是一名道德败坏之徒，但实力极强，派蒙说他单凭武力也完全能够坐稳第二席，一心想着超越命运成神。在第三章·第五幕「虚空鼓动，劫火高扬」中，摧毁了自己的所有其他切片，只留下了最为自私且强大的切片并带着雷、草神之心回到了至冬。在5.3的魔神任务第五章·第五幕「炽烈的还魂诗」中，玛薇卡提到，他已经有一部分手下抵达纳塔，目前盘踞在挪德卡莱研究古月遗骸。对月髓虎视眈眈，通过欺骗「亥珀波瑞亚」虚影，获得有关月亮的知识以及亥珀波瑞亚的部分技术，以此制作人造霜月月髓。在空月之歌中，由于提瓦特的命运被注定，而身为降临者的旅行者能对其产生影响，因此「博士」夺取月髓的行动也因为天理制定的命运不可能成功。但有旅行者插手此事，一直干涉月髓的命运，将其影响，如「博士」的计划，旅行者的所有行动都形成构成蝴蝶效应的「变数」层层堆砌，最终打破了命运。为了堆砌足够的变数，也为了让旅行者一行人能顺利找到奔月的仪式的遗迹以执行下一步计划，他扰动提瓦特边界引来魔女会，而魔女会以为这是深渊影响，还和旅行者见了面，影响旅行者的行动。他深知被世界逐渐排斥的哥伦比娅在猎月人一战后肯定即将消失，只得奔月，而奔月的仪式需要摆放两个月髓，于是「博士」制造出月矩力浓度高，比月亮更像月亮的装置让人在地下秘境只凭月矩力误以为那里就是月亮。哥伦比娅中计透过奔月的仪式传送到「博士」的囚笼中，因为力量不够无法破除囚笼。接着「博士」将两个月髓用月矩力从仪式传送门也吸引过来。如今「博士」已经用一连串阴谋集齐了虹月与恒月的月髓，再加上人造霜月月髓，获得了三月权能，能够暂停时间，只有不受法则拘束的降临者能不被影响。为了超越自己命运束缚，达成不被世界允许的实验。空月之歌第六幕「散于晨雾的月芒」中，哥伦比娅为了不让自己的力量为「博士」所用跳入月之门消散，生死未卜，「博士」失算但也已获得两个月髓的力量。在夺取月髓后「博士」凝滞时间，让来阻击他的旅行者一行人冻结，但不受提瓦特法则影响的旅行者杀了上去。然而「博士」的实力碾压旅行者，旅行者被「博士」轻易击败。「博士」因此成为挪德卡莱全民公敌。空月之歌第七幕「如果在冬夜，一个旅人」中，因「博士」激怒了挪德卡莱本地人（叮铃哐啷蛋卷工坊、秘闻馆、霜月之子和执灯人）、西风骑士团、愚人众执行官「仆人」、「木偶」、「公子」以及魔女会A、N两位魔女，各势力联手对抗「博士」，并打算透过旅行者触发的变数打造利于自己的命运。此时，「博士」缺失的月髓尚未制造完毕，在与其战斗的过程中可见冰元素攻击和水元素浮游机关。在空月之歌第八幕「真实之月」中，「博士」已经成为僭月之伪神，玩弄阴谋想要说服旅行者，但对方始终不为所动。在各势力与「博士」在决战中，「博士」使用了水、冰、雷元素，最终被回归的哥伦比娅击败。本幕剧情结束后会有一个「博士」同款特效的闪动，暗示其有可能存活，即生死不明，其下一个目标很有可能是世界树。
执行官互评折叠「公子」：听说他把不同年龄段的自己切片做成了义体，负责不同的任务。没办法，我的同僚都是些不太正常的人。反过来想想，要是某一天，我遇见了自己的义体…哈，肯定得和对方决出真正能活下去的那一个。「散兵」：真遗憾，小吉祥草王让他消除了那么多的切片，剥夺了我将他一片一片千刀万剐的快乐。「仆人」：我刚接手壁炉之家，他就向我提出了许多合作计划，让我将「淘汰的孩子」交给他去做实验，他则向我共享实验成果。听说他跟前代「仆人」合作得很愉快，如果不是同为执行官，我很想送他去见前代「仆人」。「少女」：曾有一段时间，「博士」很照顾我，他说对待我就像对待自己的妹妹一样。后来我发现，越是在他心中亲近的人，他利用起来便越是自如。「既然是一家人，应该要学着忍让吧？」我看得出他由衷那么想，而我不想那么做。这种需要只会令我回想起在第一个家时，石台下的那些贪婪目光。
