# 灵渊

灵渊不忍两位友人浮锦和药君的生存环境随着人类对沉玉谷的开发而逐渐偏离自然法则，抱着即使被热爱人类的浮锦谴责也没关系的心态暗地重塑沉玉谷的地脉，慢慢地将沉玉谷改造回最原始也最适合三仙居住的环境。随着灵渊改造计画的实施，沉玉谷出现了大范围的氲气，近年来翘英庄的茶叶质量也开始下降。浮锦在锦落庭察觉了地脉异常，但力量不完全的她无能为力。[24]
安柏祖父回到璃月安柏的祖父从蒙德不告而别，理所当然地被西风骑士团视为叛逃，导致其带领的侦查小队保受波及[196]。他实为回到璃月轻策庄[197]。
