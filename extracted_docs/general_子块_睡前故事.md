# 睡前故事

深渊教团夺走原型机之眼，命运的织机得以完成。卡利贝尔的意识消散。期间你得知了引导克洛达尔·亚尔伯里奇创立深渊教团的「罪人」的身分是戴因的哥哥维瑟弗尼尔，与其他「罪人」的消息。
纳塔的危机（主线剧情）距离纳塔毁灭 仅剩一年玛薇卡将自己的意识与力量存入圣火中并在500年后复苏，原本计划是6位继承古名的英雄得到大灵承认，然后她再苏醒直接与深渊对战。但因深渊的干扰计划出错，玛薇卡醒来时还有数人没被承认，但为了不影响这些人的状态她无法直接将此事直接告知他们，只能暗中引导。因为提瓦特的命运早已注定，玛薇卡安排了不受提瓦特命运影响的降临者（旅行者）参与计划。为此，玛薇卡委托希诺宁为旅行者打造古名，这样旅行者将能借助还魂诗复活。深渊长久以來侵入夜神之国，读取其中的记忆，变得十分狡诈，透过针对性的手法侵略纳塔各处。庇佑纳塔的夜神之国几乎被侵蚀殆尽。此时「队长」来到纳塔与玛薇卡对质，而烟谜主的一位成员欧洛伦也成为叛徒，为「队长」效力。内忧外患同时袭来，此时纳塔的地脉已将全面深渊化，距离毁灭仅剩不足一年的时间。[60]
