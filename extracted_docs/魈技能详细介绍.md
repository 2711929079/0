# 魈技能详细介绍

魈是《原神》中的风元素五星角色，璃月夜叉大将，称号"降魔大圣"。

普通攻击「卷积微尘」：
- 效果：进行六段连续攻击，造成物理伤害。
- 特点：普通攻击伤害一般，主要依靠元素爆发输出。

元素战技「风轮两立」：
- 效果：极速突进，对路径上的敌人造成风元素伤害。
- 充能机制：最多可存储2次使用次数（一命后增加为3次）。
- 冷却时间：10秒。
- 能量消耗：无。

元素爆发「靖妖傩舞」：
- 效果：进入"靖妖傩舞"状态，普通攻击、重击和下落攻击转化为风元素伤害，并获得伤害加成。
- 持续时间：15秒。
- 冷却时间：18秒。
- 能量需求：70点。

天赋技能：
1. 坏劫·国土碾：元素爆发期间，每2秒损失当前生命值的7%。
2. 神通·空中自在法：元素爆发期间，下落攻击伤害提升30%。

角色定位：
- 主要定位：主C输出
- 次要定位：副C
- 核心优势：风元素输出、高机动性、爆发伤害

培养建议：
- 武器推荐：和璞鸢、护摩之杖、决斗之枪
- 圣遗物推荐：翠绿之影四件套（风元素伤害加成和减抗）
