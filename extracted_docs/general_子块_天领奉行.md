# 天领奉行

九条家，纹章为钴纹。
稻妻的治安主管，负责执行政令，在锁国令期间拥有颁发「通行凭证」的权限。管辖着除雷电将军直属的「奥诘众」和神里绫人指挥的「终末番」以外全部的幕府武装。数百年前奉将军命成立「火消队」，负责宣传消防安全及扑灭火灾。雷电将军的代行者，忠诚且无情。目前已经结束与珊瑚宫反抗军的交战。家主为九条孝行，由于与愚人众勾结一事败露，被羁押，由其次子九条镰治代行家主一职。其长子为九条政仁，是幕府军大将，于神无冢任兵职。九条裟罗，九条家的养女，也是幕府军大将，为「眼狩令」的执行者，带领九条军不断回收「神之眼」，主线结束后负责天领奉行的具体事务。奉行所，下辖机构，负责侦破案件和关押犯人。鹿野院平藏是其中一员。鹰司家辅佐九条家执政，在九条孝行入狱后想要夺取家主，被赶来的影挫败。
