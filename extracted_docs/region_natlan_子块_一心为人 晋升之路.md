# 一心为人 晋升之路

曾统治纳塔的火龙王修库特尔假死后，龙众因执迷于守候修库特尔的躯体而日渐衰败，库库尔坎将龙的未来寄托在了人类身上，意图创造一个同时拥有龙与人血脉的存在，去击败火龙王，开创龙与人共存的新时代。因此他将人类的灵魂融入了子嗣奥奇坎的躯体中，使其成为了半人半龙的特殊存在。在预言中，奥奇坎将斩杀火龙，以暴虐和狡猾成为两个世界的救主。奥奇坎曾因龙与人的双重身份被人类囚禁，但在希巴拉克的宽容下，其最终（如盗火贤者计划中的那样）被释放并被人类所接纳。但奥奇坎却被希巴拉克的人格魅力所折服，成为希巴拉克的追随者，开始憎恨上他本应归属的龙族。最终斩杀火龙、成为君主的并非被寄予厚望的奥奇坎，而是身为人类的希巴拉克。希巴拉克吸收了火龙的权能，升华为火之执政后，也并未如预言般的那样用铁腕统治纳塔，而是团结了纳塔的各个部族，一同抵抗深渊。讨伐火龙王的战争结束，希巴拉克夺得火之大权。库库尔坎与其定下契约，在深渊来临之际人和龙要拼合「通牒金盘」，开启「维奇洛波奇特利」，虽然开启后人类会无法挺过这场浩劫，但也能创造出超越人和龙的新生命。奥奇坎得知这个计划执行后，世界将不存在人类的位置，因此加深了对龙的仇恨，决定反抗父亲的计划。希巴拉克回归源火之前，曾特意交代奥奇坎等人在他死后继承守护纳塔、建立人龙共存世界的职责，希望日后纳塔的人龙能够和平共处。希巴拉克进入源火后，奥奇坎捡起了希巴拉克留下的缺裂明环（火龙王的半个眼睛，即半个「通牒金盘」），成为了纳塔暂时的统治者。然而奥奇坎并未完全理解希巴拉克对他的期许，极度迷恋希巴拉克的奥奇坎，因所爱之神离去的痛苦、拥有人的灵魂却是龙的肉身带来的生理问题、不认为除希巴拉克外还有任何人配成为火之神的执念、还有自身的性格问题，陷入了暴虐和谵妄之中。「半人」谋主奥奇坎代执一切权柄的时代到来，他在几十年后成为了暴君，建立了直通天际的庞大烬城（禁城）奥奇卡纳塔。奥奇坎知晓库库尔坎的计划只是在利用人类，以「追捕邪龙」的名义下达了追杀龙众的命令，以表抗议。
