# 「白沙皇」莫诺马赫·雪奈茨尼伊

初代冰神被称为「白沙皇」（冬沙皇），高居苍星的王座[26]，带领妖精们统治了整个北陆。白皇最终殒落于浊秽的黑浪（#白沙皇之死）[22]，如今妖精们虽然已经没落，却仍旧以自己的方式影响着这个国度。[27]
苍星王廷·波里菲尔：执掌苍星王廷财政的大臣·菲利波夫：妖精一族中最擅长变化者·阿尔维斯：矮灵大公·瑟里维克：狼形兽怪大公，（可能）出卖兽怪联盟的情报，在战后归顺苍星王廷，详见#兽怪联盟[8]。
