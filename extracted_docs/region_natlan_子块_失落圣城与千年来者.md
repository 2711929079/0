# 失落圣城与千年来者

身负「焰主之祝」的幼嵴锋龙「小家伙」与旅行者成功进入了圣山。在圣山内部，见到了一只被「盗火贤者」操纵的秘源龙仆，得知火山内部原先是龙族的城市——初始圣城、源火之地图兰，但已毁于战争。龙仆希望「小家伙」和旅行者能前往圣山之心，启动剩下两枚「象拟聚星」，解除封印，开启中央天轴。旅行者随即启动机关。最终，旅行者一行人击溃了前来侵蚀的深渊魔物，启动并稳定第一枚「象拟聚星」，开启中央天轴。而在为其充能，浸入燃素池的过程中，旅行者读取了其中的远古记忆，了解了圣城图兰的历史，并发现火龙王修库特尔曾打算启动「维奇洛波奇特利（天火之殇）」，从而毁灭、净化一切，而另一条名为库库尔坎的龙认为它简直是疯了，对方则认为库库尔坎迟早会走上和它一样的道路。随后，一行人破除第二枚「象拟聚星」的封绝，读取了记忆，发现库库尔坎是火龙王第一个创造出的龙裔，也将自己视为火龙王的兄弟。随后，旅行者启动天轴，前往了古城深处的龙居地，在获得并解锁前往圣山之心的「许可」，并清除深渊侵蚀之后，成功前往了「圣山之心」。在「圣山之心」，一行人先是清除此地的深渊污染，然后装载通牒金盘，进入「维奇洛波奇特利」的光芒之中。「盗火贤者」也放出了最后的考验——深古秘源机龙。旅行者被机龙封印，「小家伙」面对险境，力量爆发。此时「盗火贤者」再次出现，正确引导「小家伙」的力量，让其短暂增长智慧，并进化为嵴锋龙。贤者表示：将要踏上最终之路的其实只有「小家伙」自己，而「维奇洛波奇特利」其实是能将整个纳塔燃素化的机关。「小家伙」也认出库库尔坎其实就是「盗火贤者」的真名。「小家伙」和「盗火贤者」来到「维奇洛波奇特利」的启动装置。贤者说当装置启动后，「源火的大权」就能够回归，「小家伙」将成为「新世界」塑造主，或是新火龙王。它让「小家伙」选择是否启动，「小家伙」选择不启动。贤者问为什么，它说自己仍然相信人类，不会让旅行者和派蒙冒险，他们怎么看待自己不重要，但他们对于自己来说很重要。贤者说凭借自己的火，才让「小家伙」拥有智慧，而它差点得到，却又失去整个世界一事该如何向旅行者和派蒙解释。「小家伙」不以为意，它创造出返回的道路，救下了旅行者和派蒙，与他们一起击败了秘源机龙[1]。
渊上渡火者的解脱渊上调查烬寂海是为了寻找自己认定的家乡，他加入深渊教团，但深渊教团并非他的故乡。发现烬寂海能前往时间线的每个角落后前往步过烬寂海[96]。传说一种鸟，生而孤独，能在火中高歌，人民将它作为图腾崇拜，君主以之为贵。火山之地的智者佩着它的羽毛，在烈火中退隐。孤独而来的他最终也孤独而去，下落从此不明。从此以后，人们时常听到烬寂海深处传来鸣唱。不知是浴火之鸟的鸣叫，还是渡火之贤者的吟叹呢。[136]
