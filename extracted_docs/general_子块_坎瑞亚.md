# 坎瑞亚

一群不再信仰神的人们在须弥地底建立无神的国度坎瑞亚。其国民不只纯血的贵族，背弃了神的各国人民来到此处也会被接纳为国民。[46]当时，古老尊荣的氏族统治地底，那时的人们相信高天之上的赤月之骸即是万象的主宰，于是渴盼超越之人筑起无数恢宏的高塔，祷求早已逝去的赤月为他们带来拯救。建立赤月王朝。到后来，有一群学者（科学家）发现赤月并无所述功效：「若是厄月之骸早已将你锚定于死，旧日血仇又有何意义可寻」，于是有人信仰崩塌，后来，赤月王朝被推翻。
坎瑞亚有着先进的科技与庞大的野心[47]。王国护卫队成立后，队员透过名为“至真之术”的剑法守护宫廷与王国[48]。国花为因提瓦特，花期很短，但是一旦离开坎瑞亚的国土，变回无坚不摧，永不枯萎。直到重新回到坎瑞亚的土地之上，曾遍布坎瑞亚的宫廷[27]。（记事者之匣中提到沙尔芬德尼尔覆灭时，一群人建立了无神的国度，指的应该就是坎瑞亚。[49]森林书任务中提到一位学者的名言“诸位，坎瑞亚数千年的历史在看着我们！”[50][51]可得知坎瑞亚有数千年的历史。据记事者之匣的描述坎瑞亚的成立时间大概率是葬火之战期间。而坎瑞亚有数千年历史代表葬火之战发生于数千年前，与前文推断的六千年前符合。）之后坎瑞亚发明了各种战争兵器，包括「耕地机」[52]，并且前往各地搜集寒天之钉碎片。
