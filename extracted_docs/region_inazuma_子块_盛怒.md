# 盛怒

在清籁岛天云峠上的发狂雷鸟被雷神视为作乱的妖物消灭。雷神用来将她射落的箭改变了清籁岛的部分地形。其残留的力量最终成为了纯粹愤怒的化身（雷音权现），浅濑神社也为了压制其力量而设[3]。雷音权现只会随着雷霆降生在遗留着强大忿恨的地方，譬如说支离破碎、雷云黝黝的天云峠[20]。后来清籁岛因浅濑神社的发展，逐步有人定居，形成了越石村。而这户村庄里混入了一家外乡人，其便是来自鹤观的阿釜的祖先。[3]越石村充其量也只是个小渔村，也因为远离鸣神岛，并没有发展得很繁荣。
