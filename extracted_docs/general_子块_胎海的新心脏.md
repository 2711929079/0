# 胎海的新心脏

水龙陨落后，生之执政在原初的大海里创造了另一颗心脏，名为厄歌莉娅。[7]
文明制度七位龙王全部被打败，七个王国全部对天上俯首称臣，此刻天之王座开始了对这片天地的创造。山川与河流落成，大海和大洋接纳了部分对天之王座的反抗者，这就是渊下宫里深海龙蜥的由来。在这之后的又几百年里，自然万物与人类诞生。人们耕耘，第一次收获。人们开掘，第一次收获贵金。人们聚集，第一次写就诗歌。如果有饥馑，天上就落下食物与甘霖。如果有贫瘠，那大地就会生出矿藏。如果有忧郁蔓延，那么高天就会以声音回应。唯一的禁止之事，就是引入禁忌知识。但是禁忌知识的通道已经被封堵。人们生活在这片土地上安居乐业，整个世界都在类似的文化之下。[8]此时大陆上有许多文明，广义上都算第一文明的分支。这些文明大多都拥有相同文字、崇拜天空岛与其使者、使用地脉的力量等特征。它们的遗迹风格相似，提瓦特各处都可见的元素方碑、传送锚点、浪船锚点便是当时文明使用的机关。地上各处都曾有人竖起几乎能触达天顶的高塔[16]。法涅斯为第一文明建立了一个制度，第一文明需要信仰天空岛，人们能通过天使得到高天的启示天空岛会为人们带来知识，引导人们建立文明，庇佑人们免于贫瘠但第一文明不能擅自使用可能毁灭世界的“禁忌的知识”。[8]在最初的世代，初民们的道德足以自持，无需仰赖律法与威权的统治。天使行走在大地之上。经由他们的指引，人们在自太古以来便一直延续的安宁中平静地度日，安享繁荣与丰饶[17]。
