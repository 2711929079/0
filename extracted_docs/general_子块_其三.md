# 其三

…最不可僭越的，是生命奔赴终末的秩序；
这本不是凡常之人能够妄加剽夺的权柄。
将他人推上那无归的高塔，截夺其命途；
再宽仁的审判，也绝不会恕宥此等恶行。
然而堕罪者又岂会惧骇于未安魂的诘问？
只因深渊也无法掩盖自高他坠落的银辉。
此即为，三宗大罪其三。…拉伊莫所行的诸多罪恶，
由月光与幽灯共同见证。
以石柱为刻，咒文为缄，
将「众噩之蝰」的罪行铭刻于礁屿之上。
