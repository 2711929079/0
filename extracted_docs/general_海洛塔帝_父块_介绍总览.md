# 介绍总览

## 介绍
被称为「贤者」，是坎瑞亚「五罪人」之一，犯下了「渎理之罪」，曾持有双手剑「索达古瑟」（现名「赫烈」）[1]。最初于苍白之火中提及，才学过人（高于「丑角」），建议坎瑞亚国王伊尔明引入深渊力量，受到伊尔明的青睐，目前没有正式出现；他是深秘院院长，有御赐的「贤者」与首席御前法师之名，也是王国四柱相关最受青睐的人物。因为总是提出伊尔明「想要看到」的方案而受到重视以获取利益。此做法引起莱茵多特不满，认为伊尔明背地里比较重用他而损失自己利益。
