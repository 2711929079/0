# 赤念果

在恶劣的沙漠环境中也能存活的植物所结的果子，赤红的果实在黄沙映衬下显得格外动人。拥有极其鲜艳的色彩，经过简单处理后便可制成染料。在沙漠住民的生活中相当重要，据说镀金旅团烈火般的旗帜也是由其所染就。红色的织物随着旅团的足迹被带往四方，无论身在何地，每当看到这些飘拂的绸缎，便似乎可以感受到故乡炙热的风。
