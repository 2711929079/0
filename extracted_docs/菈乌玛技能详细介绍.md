# 菈乌玛技能详细介绍

菈乌玛是《原神》中的水元素四星角色，须弥的角色，称号"水之舞者"。
元素战技「水之舞」：
- 释放水元素能量，对周围敌人造成水元素伤害并施加"水元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「水之领域」：
- 创造一个持续8秒的水元素领域，领域内的敌人会受到持续水元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "水之祝福"：水之舞命中敌人后，有15%概率触发"冻结"效果。
- "水元素增幅"：菈乌玛的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "水之守护"：元素爆发持续时间内，菈乌玛的水元素伤害提升15%。
