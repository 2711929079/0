# 兹蹻

昔年帝君将兹蹻从「詹诸吞月」帛画中解脱出来，其便助帝君征战，先后平沈玉谷，定轻策山，此为契约之故。而后两人互不相欠[5][63]兹白陨落后，唯有下尸神的身影仍偶尔掠过故士的山水，直至在魔神战争中助力钟离后，方逐渐隐于山林之中。她身上的时之痕迹（磨损）依旧对那位古老的岩神产生了莫名的影响。于是在魔神战争后，自知身有威胁的兹蹻变隐入山林，从此不现人间。只是她生性逍遥自在，又喜好人前显圣，因凡人寿限不过百年，几乎不受时之痕迹的磨损。所以她偶尔会在一些需要帮助的凡人面前现身。一边助他们脱困，一边又存着将那天塌地陷前的故事悄悄传递下去的小心思。但若是后人成书不如她所意，她大概也只会看一眼就抛去一边。毕竟她就像一匹在无人旷野中肆意奔行的野马，没什么能束缚她自由洒脱的脚步。她从兹白的陨灭中诞生，但她从来不觉得自己应戴上兹白或是任何人的缰辔。她随时都准备好潇洒快意地跳入时间的尽头，然后「砰」地一声，化为一团闪耀着金芒的尘埃[64]。在山中自在逍遥，偶有小人儿误入，还能找找他们的乐子，早已不再求些什么。但若真如此，为何她又孜孜不倦地向凡人们讲述那葬火之战前的古老故事[5]？白驹逆旅老板的祖宗为了生病的娘采药想换取微薄的药费，却摔落山谷，被兹白的下尸神兹蹻所救。兹蹻送与她一则故事，让她带给书店赚点钱。讲述故事时，凡间已过兩千年，来到现在，她来到五百年后的归离集并遇到宝儿，所幸身为时间的女儿的兹蹻将其送回去，她才得以回归离集[11]。为了感谢白马先人的帮助，白驹逆旅就此得名[5]。沉玉谷部民口中的「时间的女儿」是兹蹻。八奇炼桃都中的「金目乘黄月驹」也是兹蹻[64]。
留云借风真君.魔神战争在魔神战争中追随帝君，守护璃月。当时大多以人类姿态现身于世，果敢伶俐，受人敬仰[65]。留云借风真君以人形参与魔神战争，并多次庇护受难的人群。战后隐居于奥藏山，无拘无束，多以鹤形态现世[47][66]。魔神战争當时没人在求仙問道，於是太山谷被仙人藏入水下封印起来。
