# 漆黑物语

五百年前，有这样的三个人，他们为了共同的目的来到影向山找到了影响天狗向他求学，他们这三人被称为「影向役者三人组」，最终因为「灾厄」分道扬镳[1]。他们中的一个人认识了名为【昆布丸】的少年，并爱上了他她与他们也因此认识了她，而她最后献出了生命。最后的结局如何呢？活了多少？死了多少？我们就不得而知了。在这个故事里，牵扯到了太多、太多的人，多到数都数不清的人，而如今……他们又有多少人留下了后代呢？
