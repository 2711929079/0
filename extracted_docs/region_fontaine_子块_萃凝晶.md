# 萃凝晶

澄澈明净的晶石，矿芯具备良好的韧性与塑性。在古老的传说中，璀璨的棱晶曾是高天的使者带来的馈赠，日夜澈映的微光则是白月恩泽的象征，予世人以锻造奇物的权柄。传说早已无从考证，但现代的工匠们依然会利用这种明净而坚韧的晶石，打造各种各样的饰品与武器。
