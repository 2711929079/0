# No.3 「少女」哥伦比娅Colombina

中文CV：杨梦露 日文CV：Lynn在达达利亚的语音“关于「少女」…”中提及其为第三席。在月之一版本魔神任务中正式登场，月之四版本正式成为自机角色，具有三相月临之力，表现为水。在空月之歌第五幕中得到正式名字「哥伦比娅·希珀塞莱尼娅」。希珀塞莱尼娅是自己取的，意为“在月下诞生的”。本身是正儿八经的月神，实力极强，但由于霜月在外，自身力量被提瓦特排斥，处于不断衰落中，一度失去实体，有着极强的月矩力操控能力，拥有自己的一系列剧情——伴月纪闻。五百年前，她作为新的月神诞生后被迫成为霜月之子信仰的「月神」，担任着他们的信仰。后来霜月之子向神的贪婪索取让库塔尔颇为烦心失望且无能为力，于是离开挪德卡莱加入愚人众，成为第三席执行官。后因不满愚人众的索取而离开，因此至冬方面下达了「苍星圣敕」要求愚人众把她抓回来，但大家因为有雷利尔和「博士」的威胁而无法顾及。决战结束后，「苍星圣敕」被女皇撤销。旅行者在夜晚独自散步时，被月灵「努昂诺塔」引至银月之庭，在那里，旅行者见到了正在唱歌的「少女」。她气质空灵，对外界漠不关心。但当提及愚人众时，「少女」立刻展现出了巨大的敌意，花朵瞬间变红。期间她向旅行者透露，天空中能看见的月亮是会骗人的，真正重要的东西是看不见的，暗示了提瓦特天空的虚假。为了更多地了解她，旅行者与派蒙带着食物再次拜访。「少女」讲述了自己与「霜月之子」的过往。她坦言，自己真正的「序章」丢失在了无法触及的月球上，并表明了自己现在的愿望——回到月亮上去。期间其力量出现衰弱的迹象，她解释这是第二次「离家出走」的代价。在旅行者为飞船迫降的记忆感到困惑时，「少女」展现了其作为「月神」的能力。她轻松地帮助旅行者看到了更为清晰的记忆片段，这一举动揭示了一个惊人的事实：血亲是在飞船外部找到旅行者的，尽管在此之前从未有人离开过飞船。面对雷利尔的威胁，「少女」教导旅行者隐藏月髓的气息。对于雷利尔，她坦言未能洞悉其计划，并指出对方的力量中蕴含着一股近乎疯狂的「执念」。后来雷利尔袭击「少女」，众人不敌，在关键时刻，「少女」和旅行者及时构建联系，将众人传送走，而后「少女」提出自己引诱雷利尔掩护其他人的计划，但被众人否决，随后大家一起从狂猎大军中突围，在关键时刻，法尔伽赶来击败了狂猎，随后大家一起前往安全屋，而后众人分兵。雷利尔被「少女」击败并被戴因放逐后，由于自身力量透支，被提瓦特排斥的反应进一步恶化，一度失去实体，她意识到自己时日无多，决定回那夏镇见大家最后一面，意外听见了同样焦急的众人正在寻找拯救「少女」的计划，决定跟在旅行者身后，并完整见证了计划的实施。最后，旅行者、菈乌玛和阿蕾奇诺在亥珀波瑞亚遗迹内通过三月月髓的力量，在「少女」哥伦比娅的意识也彻底消散前阻止了悲剧。哥伦比娅决定在祈月之夜回到月球上，那夏镇为此举行了盛大的庆典为哥伦比娅送别，众人全部来到现场。但「博士」实施自己的阴谋诡计，捕获了她。哥伦比娅为了不让自己力量为「博士」所用，破坏「博士」阴谋，选择跳入月之门。「月亮倒影」中的时间与现实逆向而行，哥伦比娅数次穿越镜面回到过去的现实，以「神秘帮手」的形式参与完成了历史，并以月矩力写下线索、用部分灵魂创造「努昂诺塔」，以便身处未来的同伴们知晓自己的情况。她帮助困于这片空间无法相见的雷利尔和索琳蒂丝传达了心意，最后在空间的尽头遇到了三位三月女神。原来「月亮倒影」实为葬火之战初，尼伯龙根囚禁三月女神的监牢。姐姐们将权能全部转交予哥伦比娅助她完成计划，自此，哥伦比娅·希珀塞莱尼娅便是唯一的「三月女神」。她将身体化为月光洒向挪德卡莱，成为月矩力融入大地；灵魂留在「月亮倒影」，指引身处未来的伙伴与自己；「三月」的权能则藏进霜月，等待着与灵魂、身体的重聚。在与「博士」的决战中，尼可、法尔伽和阿蕾奇诺合力将「霜月」拉入虚假之天，哥伦比娅与月亮一同重临提瓦特，完成身体、灵魂和权能三合一，完全接管战场，随后夺回了虹月月髓。然后旅行者、菈乌玛、菲林斯和奈芙尔合力向「博士」发起了新一轮攻势并打下恒月月髓，最终哥伦比娅将「博士」击败。战斗结束后，哥伦比娅将恒月与虹月的月髓分别交予菈乌玛和阿蕾奇诺保管，而月矩力重新还给了挪德卡莱，庇护着这个由她自己所选择的「家」，霜月留在了虚假之天内，哥伦比娅再也不会如过去一般失去力量。哥伦比娅带旅行者回到其被「天理」藏于霜月之上的飞船，并为旅行者解除了世界给予的部分限制。等挪德卡莱安定下来，哥伦比娅也将踏上旅行，探访旅行者走过的地方。她的命之座图案有她的执行官代表图标，一只鸽子，但也有自己的形象。关于哥伦比娅的更多信息，见至冬#挪德卡莱·空月之歌（主线剧情）、三月#「新月·哥伦比娅·希珀塞莱尼娅」。
