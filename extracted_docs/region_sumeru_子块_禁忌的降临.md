# 禁忌的降临

灾厄的降临伴随着禁忌知识的污染而出现。从深渊之底而来的属于世界之外的禁忌知识污染了世界树。大慈树王知道单靠自己的力量，根本无法将禁忌知识驱逐。于是她创造了统合人类智慧的装置，命名为「虚空」。大慈树王借助虚空借走了人们的梦境，来作为清除禁忌知识的能量，禁忌知识几乎从世界树中完全清除。与世界树相连的大慈树王在治理世界树的同时，被禁忌知识所污染，她身上留有禁忌知识，却与世界树相连，导致世界树连带受到污染。只有当这个存在本身被彻底遗忘时，禁忌知识才会被彻底清除……[67]
草神迭代因为世界树的机制，大慈树王无法将自己从世界树中删除，她在逝世前折下世界树最纯净的枝杈（即小吉祥草王），作为她的轮回转世。她逝世前留下「让世界彻底遗忘我」的线索在世界树中播放，让发现线索的小吉祥草王能来到世界树将其中有关她的资料彻底清除，以杜绝深渊污染，但同时她也会被世界遗忘。树王逝世后，须弥学术界看似兴盛，却一直没能出现较大的突破。[55]因为树王之死，本带在树王身边的乌婕特之眼佚失。[19]贤者找到了新生的小吉祥草王，将其接回了须弥城。[68]
