# 近20年来

少女连环失踪案开端20年前瓦谢和薇涅尔在探索遗迹的时候，薇涅尔触碰到了原始胎海之水导致溶解，瓦谢根据附近寻找到的雅各布的研究笔记想要重新见到恋人，于是到处溶解少年做人体实验。[78]香水狂热起初只是一些商家推出了几款高端香水，为它们赋上了各种标签，却没想到它们在市场上引发了异乎寻常的疯狂，价格节节攀升。有人想用它们标榜自己的身价，更多人想趁着价格飞涨囤积居奇。在人们的盲目推崇下，一瓶普通香水的售价都被抬到了数十万摩拉之巨，由此引发的纠纷、骗局更是层出不穷。好在这股风潮彻底扩大之前，沫芒宫意识到了它的危害，逮捕了许多投机与诈骗团伙，最后这场泡沫的破裂，也只影响到了一部分富贵人家。而奥古斯都就是这场狂热中风头最盛的香水，曾经卖出过一瓶千万摩拉的高价。泡沫消失后，培育奥古斯都花的花圃毁于一场大火，从此再也没人见过这种花。二十年前席尔万和卢西恩曾经与香水师艾德加合作开发奥古斯都，因造假帐和扰乱市场等罪名被捕。「奥古斯都」是靠邪眼扭曲形成的花种。负责培育奥古斯都，表面身分为须弥学者的维佳雅真名是叶莲娜，真实身份是自至冬流亡的愚人众。她本来也被逐影庭盯上了，但她在被捕前自焚身亡，烧毁了奥古斯都花的苗圃和炒作团伙的利润。她的弟弟基里亚当初只有十几岁，负责给团伙打下手，颇得维佳雅宠爱，但案发后他就失踪了。奥古斯都花含有邪眼的畸变元素力，长年累月接触会摧残人体。表示叶莲娜想继续改良花种，但众人决定靠着香水狂热尽早上市奥古斯都。案发时，席尔万本来计划除掉叶莲娜和花圃，结果她提前自杀了。席尔万和艾德加认为，叶莲娜自杀是为了她弟弟：她因为过度使用邪眼，身体不堪重负，所以与弟弟一起流亡。在枫丹，非法用邪眼培育奥古斯都花使她的身体愈加衰弱，但她一直瞒着基里亚。案发时，叶莲娜自杀并毁掉了苗圃，给她弟弟留了一大笔钱。基里亚不知道叶莲娜是自杀，认为姐姐的死是席尔万一伙为了销毁证据做的，为了替姐姐复仇，杀害了出狱后不久，真正的调香师艾德加，顶替了他的身份，然后再宣布远赴须弥，在漫长的岁月中精进调香技术并用邪眼使自己的身体老化。他知道席尔万出狱后一定会来找艾德加，于是装成他的样子等待出狱后的「姐姐的仇人」找上门。期间收艾梅利埃为徒弟。[79]怪盗貂十余年前，枫丹廷区出现了一名代号「貂」的怪盗。不同于劫富济贫的经典怪盗形象，「貂」来者不拒，以不分贵贱偷窃珍视之物而臭名远扬。十年前，魔术师塞萨尔表演高空逃脱魔术失败，坠落而死。人们在他的遗物中找到了大量脏物，将其认定为怪盗貂并结案。其后怪盗貂也销声匿迹。[80]刺玫会近年来，卡雷斯在白淞镇组建了刺玫会。在刺玫会的帮助下，枫丹修建了大规模的水道沟通全境。作为感谢，三条水道以卡雷斯及其妻女的名字命名。[81]枫丹科学院动用大型工程用机关挖通“白淞隧道”，解决了水道建造过程中的一大难题。[82]建造特化机关是在建筑施工、考古发掘上同样有不可估量价值的大型发条机关，其名来自修建克莱门汀线时卡雷斯雇佣的工程师。这名工程师在卡雷斯的资助下为打通白淞隧道建造了颇具传奇性的“黄金乐船”巨型工程机械，并在竣工后据此设计了这类结构相似，但尺寸缩小的发条机关[83]曾经有修建第四条巡轨船航线，连通柔灯港与枫丹廷的计划。但这一计划随着最有力执行者之死而无限期停摆了。[84]卡雷斯之死三年前，卡雷斯被指控枪杀雅克，要求以决斗自证清白。在那维莱特的见证下，卡雷斯最终死在与官方指定的决斗代理人克洛琳德的决斗中。刺玫会也因此名声大损，陷入经济危机[78]。克洛琳德的师父是逐影猎人佩特洛妮拉，此人也是卡雷斯的旧识，基于这个关系，年幼的克洛琳德与娜维娅曾是很要好的朋友。但自从发生了三年前的事情后，两人虽然不算是结仇，但也关系生变，很少见面说话了。
