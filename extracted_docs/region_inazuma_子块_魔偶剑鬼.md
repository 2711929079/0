# 魔偶剑鬼

在制造人偶的过程中，雷电影还发明了魔偶剑鬼这种副产物，这种量产型的人偶有的会作为“鸣神特使”在演武传心中测试凡人武者的实力；有的被分配到幕府军中；还有的因为失控而流落野外。经时任九条家剑道指南的道启允许集成了以秘剑「天狗抄」闻名的剑道流派初代宗主岩藏道启的记忆[51]。御舆千代在背叛雷神后逃亡，最后死于继承岩藏道启记忆的某個失控被废弃的魔偶剑鬼刀下(稻妻人推测的其死因之一）。[52][42]
惟神晴之介天狗族长认为自己没有尽职尽责，引咎而去，其女光代紧急替代位置。远在璃月修习仙法的晴之介见深渊灾厄侵袭璃月，便担忧稻妻的近况，于是回到稻妻，得知了这个惊人的消息。他拿出了狐斋宫临走前赠予他的烟斗，一时间不敢相信。那个时候，五百藏终于察觉到不对劲了，便出来找狐斋宫，大闹镇守之森。动静太大，惊扰了幕府，便请晴之介前去平息，于是五百藏和妖狸都被封印了起来。天狗族长归隐，狐斋宫也死了，浅濑响等的那个人再也回不来了，她没有理由待在这里了，便回到了清籁岛。狐斋宫虽然死了，但是过于思念这片土地，最终其强大的思念汇聚成了花散里。花散里遵循着狐斋宫的遗志，依然进行着神樱大祓的相关工作。晴之介在供奉狐斋宫的庙宇里处理后事，见了花散里，以为狐斋宫还活着，结果不是。这让他更加思念狐斋宫了，他决定要让狐斋宫放心得离开，便筹备阴阳寮相关的事。
光代继任天狗族长后，私下里也会继续教御舆道启剑艺，且与其对练。[49]
