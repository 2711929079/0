# 托马

蒙德出身的托马担心父亲喝不到蒙德的酒，带着蒲公英酒独自乘船前往稻妻，结果遭遇巨浪，自身不幸落入水中昏迷不醒，最终漂到了稻妻的海滩上。醒来后的他一无所有，也没有找到父亲，被神里家收留，成为家政官[81]。
