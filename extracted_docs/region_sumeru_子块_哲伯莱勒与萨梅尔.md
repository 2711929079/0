# 哲伯莱勒与萨梅尔

二十多年前，萨梅尔和哲伯莱勒是塔尼特部落的成员。与信仰花神的芭别尔不同，萨梅尔更崇拜赤王，在偶然中他得到了一块和赤王遗迹有关的钥石，于是组建了一支探险队，哲伯莱勒也在其中。芭别尔担心他们真的找到赤王的神迹从而威胁自己的统治地位，于是暗中策反了探险队其中一名成员塔拉法。塔法拉一路暗杀了除萨梅尔和哲伯莱勒外的所有探险队成员，最后被反杀。萨梅尔和哲伯莱勒事后叛逃了塔尼特部落，在沙漠中建立了一个名为图特摩斯的小部落。萨梅尔做事疯狂不择手段，哲伯莱勒与他有着理念分歧，但也无心抗议或脱离，只是浑浑噩噩地度日。名为优菲欣迪的教令院学者曾带着赤沙石板进入沙漠考察，萨梅尔听闻此事后便命令哲伯莱勒去劫持这位学者，想要利用她来破解赤王遗迹的谜题。但哲伯莱勒却爱上了优菲，并和她诞下了一个名叫婕德的女儿。几年后，哲伯莱勒打算带着家人离开沙漠。而萨梅尔却杀害了优菲，试图以此让哲伯莱勒回心转意。哲伯莱勒没有向萨梅尔复仇，而是在埋葬了妻子后就带着女儿离开沙漠，因为不想让女儿背负仇恨与痛苦，所以也就没有告诉她这些事情[92]。
卡维父亲卡维的父亲为了满足儿子的期望于是参加了学院争霸赛，虽然没有得到冠军，但他受到了萨齐因的影响想，为了反驳萨齐因对人性的观点于是毅然前往沙漠，他曾在阿如村为人们做了许多好事，但最终却为了拯救一支陷入流沙中的商队的不幸遇难。卡维也一直对此感到愧疚，认为是自己间接害死了父亲[91]。
