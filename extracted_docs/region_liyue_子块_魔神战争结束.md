# 魔神战争结束

世上曾有诸多魔神，也曾有旷日持久的战争，直到两千年前方才终结。期间生灵涂炭，流血无数。最终，提瓦特只剩下七位胜利者[99]，七位魔神各自登上「神」之座，自此终结了「魔神战争」时代。尘世划归七神所有，他们被称作「尘世七执政」。虽然性格迥异又相隔万里，却都肩负着「引导人类」的神圣职责。他们在败者的残骸上建立国度与城邦，开始了七神的时代。[84]七神们被天理分发了神之心。被七神击败，不服七神管制的战败魔神逃往暗之外海。岩枪的核心是发着暗淡光辉的核心，那是战败魔神的力量深入岩石形成的。他们的怨恨并不针对摩拉克斯，因为他们早就在暗之外海被深渊侵蚀的已经没有自我了，不可能还残留私怨[100]。原本居住在层岩巨渊地区的山民跟随岩王帝君迁入了璃月港生活，专以制造和販卖瓷器为业，不再顾兵戈之事[15][101]。
摩拉世上第一枚摩拉，是在数千年前，岩王爷亲手用神之心[27]将自己的血肉转换铸造的。后来岩王爷简单花掉了[42]。而后黄金屋建立，作全大陆唯一的铸币厂，借助岩之神的力量制造摩拉[27][102]。
