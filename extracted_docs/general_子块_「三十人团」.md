# 「三十人团」

须弥最古老的镀金旅团之一，因成立初期由三十个人组成而得名。负责须弥城的防务工作，也会承接居民们的委托。驻扎地位于聚砂厅。[7]同时，作为须弥最有威望的镀金旅团团体，三十人团也常负责协调各旅团之间的关系，调停各种矛盾。[8]
