# 日月前事

渊上到渊下宫寻找书籍《日月前事》，他伪装成稻妻人，在渊下宫的图书馆等待旅行者，并为其提供指引，引导旅行者替其通过试炼。在旅行者通过最终试炼后，渊上想要杀死旅行者夺书，被击败而撤退[81]。
三界路飨祭渊上与渊下宫的深海龙蜥达成同盟：用黑雾笼罩整个渊下宫，方便龙蜥精锐潜下大日御舆，拯救被囚禁的同族。而龙蜥作为回报，许可深渊教团在它们领地上通行。深渊教团与龙蜥改造了三座高塔，使渊下宫被黑暗笼罩，但深渊黑雾对龙蜥来说也是剧毒。感到背叛的深海龙蜥便让收养的「海祇御使」后代阿祇找到旅行者帮忙消除黑雾。
在三座塔都处理完了之后，渊上驱动深渊力量，使大日御舆内部出现了翻涌的黑暗。但渊上被随后赶来的旅行者击败。被打败后，渊上对旅行者说阿祇是由深海龙蜥进化而来的「龙蜥人」，并让旅行者引导光路去驱散大日御舆的黑暗。撤退前，他还把在大日御舆中找到的一块挂坠碎片交给了旅行者[82]。之后渊上将与旅行者交手的报告交给了深渊殿下[83]。
