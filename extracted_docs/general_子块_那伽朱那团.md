# 那伽朱那团

那伽朱那团是很久以前教令院的一个旁支学派。据说他们主张「为了守护知识的传承应当不惜身命啊」、「争战才是提瓦特秩序」之类的奇谈怪论。总之神神叨叨的，早就和其他学派分道扬镳，不在同一根枝桠上。虽然他们现在好像还存在于沙漠深处，不过一般人要么根本没听说过，要么当做荒诞不经的传说。
如今名唤「矫论团」的那伽朱那团驻扎于铁穆山，现任领导人为那先朱那
