# 现在

纳塔的危机（主线剧情）距离纳塔毁灭 仅剩一年玛薇卡将自己的意识与力量存入圣火中并在500年后复苏，原本计划是6位继承古名的英雄得到大灵承认，然后她再苏醒直接与深渊对战。但因深渊的干扰计划出错，玛薇卡醒来时还有人没被承认，但为了不影响那些人的状态她无法直接将此事直接告知他们，只能暗中引导。因为提瓦特的命运早已注定，玛薇卡安排了不受提瓦特命运影响的降临者（旅行者）参与计划。为此，玛薇卡委托希诺宁为旅行者打造古名，这样旅行者将能借助还魂诗复活。深渊長久以來侵入夜神之国，读取其中的记忆，变得十分狡诈，透过针对性的手法侵略纳塔各处。庇佑纳塔的夜神之国几乎被侵蚀殆尽。此时「队长」来到纳塔与玛薇卡对质，而烟谜主的一位成员欧洛伦也成为叛徒，为「队长」效力。内忧外患同时袭来，此时纳塔的地脉已将全面深渊化，距离毁灭仅剩不足一年的时间。[10]六英杰集结队长见证过深渊带来的巨大牺牲，因此打算帮助纳塔。他想回避与旅行者一战的场面(基于主角是坎瑞亚殿下的血亲还有他原来的身份)，同时一直关注着火神玛薇卡计划的进展。队长找到旅行者，坦率地说到她选择拯救纳塔的这个计划风险很大，其力量是否真的能够击败深渊。队长最担心过大的责任会让玛薇卡越发无法直面惨痛的事实，最后连启动备选方案的时机都完美错过，所以队长打算使用神之心替他动手。现在队长落败不打算再去打神之心的主意，但队长拯救纳塔的目的没有改变，他找到新的办法，希望旅行者能协助。而欧洛伦也在帮助队长。夜神之国受损后，内部的灵魂会逸散。这些灵魂无法回归地脉，只能游离在外，其流失会令夜神之国进一步衰败。有的办法是让欧洛伦出生起自身残缺的灵魂吸收，作为载体进入夜神之国再送回强化地脉力量，但这个献祭计划当时失败了。欧洛伦长大后得知自己刚出生时被献祭的事情，认为「如果当时自己被献祭成功的话大家就能得救」，于是认定自己有帮助拯救纳塔的责任。在了解队长的计划后，他加入队长的行列。与旅行者的会谈因茜特菈莉介入而终止。队长了解这备选计划的方案，即透过神之心抽空整个夜神之国的灵魂与记忆，编织出一张包裹住地脉，抵抗深渊侵蚀的大网。但他想从根源解决深渊问题，他打算重构地脉，于是寻找能重构地脉的古代「秘源机关」。队长当时找来了三位连玛薇卡都不知道的龙遗物学者——阿贝雷瓦、博苏维、奎克特利(实为由盗火贤者假冒），知道了「秘源机关」的事情。重构地脉并不是修补，几乎等同于放弃现有地脉中的一切，会让纳塔人失去记忆历史与文化，甚至导致神智异常，和动用「神之心」付出的代价很相似。玛薇卡于是派人阻止，重构地脉的计划因他们的介入失败，过程中欧洛伦得到大灵的承认，成为六英杰之一。启动的「秘源机关」因其重构地脉的功能同时也唤醒了纳塔的地脉（夜神之国）的掌控者夜神。「古名」不会记录纳塔之外的经历，也就不会被记录到夜神之国中，无法得到大灵承认的记忆与经历，就无法成为「古名」的材料。为主角打造「古名」，只有夜神能决定。但人们必须通过做梦(出神)与它的意识产生接触。而非纳塔本地人的旅行者无法进行出神仪式前往夜神之国。因此这次唤醒使得回声之子工匠希诺宁直接说服夜神为旅行者打造古名。之后，深渊的本体「古斯托特」向纳塔发起总攻，期间恰斯卡受到大灵承认成为最后觉醒的六英杰。玛薇卡与六英杰动用若娜瓦的力量暂时击败古斯托特，暂时保下纳塔，并将「虚假之天」打出裂缝。之后玛薇卡打算与旅行者一同直面古斯托特，而队长也请求夜神的协助，为了与死之执政最后一战。[13]战胜深渊旅行者的古名「希望」打造完成，玛薇卡与旅行者前往夜神之国直面古斯托特。古斯托特化形成火龙王修库特尔的模样与旅行者和玛薇卡决斗，期间旅行者激活的火元素属性，最终在众多的灵魂和在地面上的纳塔人与夜神和仙灵们的帮助下战胜了古斯托特。之后，玛薇卡必须向死之执政履行契约支付自己的生命作为使用死之执政力量的代价。夜神在对抗深渊的漫长过程中生命已走到尽头，她打算把生命支付给夜神，让夜神重新获得生命力，但自己也会牺牲。不过玛薇卡提供的生命力也有极限，夜神迟早会死亡，但能帮夜神之国撑一段时间，到时候再想办法。但队长代替了玛薇卡，如愿与夜神融为一体，使夜神得以不死，并改变夜神之国的规则，使其可容纳纳塔人以外人的灵魂，使自己在坎瑞亚的众多部下的灵魂拥有归宿，并以此作为向对他降下不死诅咒的死之执政的复仇。拥有不死诅咒的队长支付的生命使夜神不死。[134]之后在确认无害后，夜神以「某个条件」为代价同意协助旅行者血亲制作坎瑞亚地脉的蓝图[135]。在与深渊的战役结束后，巴巴托斯与玛薇卡合作救回烬寂海游荡的逝者灵魂[125]。山王回火夜在战役中，布基纳被因深渊侵蚀而发狂的伙伴山王攻击，在死前将自己的力量注入给山王。山王在此后一直希望了结自己以赎罪。在人们帮助下，「山王」决定带着罪孽坚强的活下去，因为了结自己很简单，但背负痛苦活下去却很困难，背负着过往活下去成为了山王的赎罪。[18]盗火者缝影针旅行者帮助盗火贤者伪装的蒂图修好了小缝影针，并让困在封印中陷入永恒战斗的两位坎瑞亚士兵西弗安雅与伊利萨得以解脱。流灰之国的迷旅人旅行者、派蒙和他们的龙伙伴，身负「焰主之祝」的幼嵴锋龙「小家伙」来到了奥奇卡纳塔，并遭到了邪龙与众多深渊魔物的袭击，「绿松石」和被瓦萨克拉胡巴肯取得灵魂，又放出来作为陷阱的玻娜解救了他们，并与他们一同踏上寻找「回还之玉」的通路，途中击退了一次邪龙。经过一番探索，众人终于登上天蛇船，并取得了「回还之玉」，发现「回还之玉」就是「通牒金盘」，玻娜的记忆突然出现，意识到这一切实际上都是瓦萨克拉胡巴肯的阴谋，随后邪龙冲出来抢走了「通牒金盘」，旅行者和派蒙追上邪龙后用「绿松石」的力量将其击坠到地下。「绿松石」在意识之中对旅行者表示感谢，随后回归了邪龙体内，为自己的躯体排解了深渊的侵蚀。奥奇坎向旅行者和派蒙坦白了自己的真实身份，得知旅行者和派蒙打算带着「通牒金盘」去火山的朝圣路为「小家伙」解除「焰主之祝」的诅咒，奥奇坎表示「通牒金盘」很重要也很危险，但因瓦萨克拉胡巴肯的干预，他已忘了该如何正确处理金盘，一切答案只能到火山去寻找了。奥奇坎向旅行者分享了自己的记忆，随后陷入了永恒的沉睡。[69]为我敞开心扉旅行者协助被奥奇坎利用的龙族旧日的统律核心·L型自律监管机阵统合处理单元「伊·奎库叙·茨博隆·库·雷尔（伊蕾尔）」解脱。[80]盗火者的算计这么多年纳塔这么安静，是因图兰火山因深渊侵蚀出了问题要他亲手维系不能干预纳塔，而旅行者一来他发现了旅行者独特的体质就故意安排一条受「焰主之祝」的谎称长不大的小龙逼旅行者去火山朝圣顺道铲除深渊（就是玩家打的黄金王兽），途中也多次化成各种人如曼科、蒂图来引导旅行者前进。他打算引导旅行者搜齐通牒晶盘再引到图兰大火山，以开启对付深渊的兵器维奇洛波奇特利、彻底清除深渊[69]。而夜神之国的大灵则化成猫短暂将旅行者拉入夜神之国，警告旅行者不要相信所谓「前往圣山的朝圣路」，实则是要通知旅行者别完全相信盗火者，他对人类怀有敌意，帮助人类也是为了自己目的。失落圣城与千年来者身负「焰主之祝」的幼嵴锋龙「小家伙」与旅行者成功进入了圣山。在圣山内部，见到了一只被「盗火贤者」操纵的秘源龙仆，得知火山内部原先是龙族的城市——初始圣城、源火之地图兰，但已毁于战争。龙仆希望「小家伙」和旅行者能前往圣山之心，启动剩下两枚「象拟聚星」，解除封印，开启中央天轴。旅行者随即启动机关。最终，旅行者一行人击溃了前来侵蚀的深渊魔物，启动并稳定第一枚「象拟聚星」，开启中央天轴。而在为其充能，浸入燃素池的过程中，旅行者读取了其中的远古记忆，了解了圣城图兰的历史，并发现火龙王修库特尔曾打算启动「维奇洛波奇特利（天火之殇）」，从而毁灭、净化一切，而另一条名为库库尔坎的龙认为它简直是疯了，对方则认为库库尔坎迟早会走上和它一样的道路。随后，一行人破除第二枚「象拟聚星」的封绝，读取了记忆，发现库库尔坎是火龙王第一个创造出的龙裔，也将自己视为火龙王的兄弟。随后，旅行者启动天轴，前往了古城深处的龙居地，在获得并解锁前往圣山之心的「许可」，并清除深渊侵蚀之后，成功前往了「圣山之心」。在「圣山之心」，一行人先是清除此地的深渊污染，然后装载通牒金盘，进入「维奇洛波奇特利」的光芒之中。「盗火贤者」也放出了最后的考验——深古秘源机龙。旅行者被机龙封印，「小家伙」面对险境，力量爆发。此时「盗火贤者」再次出现，正确引导「小家伙」的力量，让其短暂增长智慧，并进化为嵴锋龙。贤者表示：将要踏上最终之路的其实只有「小家伙」自己，而「维奇洛波奇特利」其实是能将整个纳塔燃素化的机关。「小家伙」也认出库库尔坎其实就是「盗火贤者」的真名。「小家伙」和「盗火贤者」来到「维奇洛波奇特利」的启动装置。贤者说当装置启动后，「源火的大权」就能够回归，「小家伙」将成为「新世界」塑造主，或是新火龙王。它让「小家伙」选择是否启动，「小家伙」选择不启动。贤者问为什么，它说自己仍然相信人类，不会让旅行者和派蒙冒险，他们怎么看待自己不重要，但他们对于自己来说很重要。贤者说凭借自己的火，才让「小家伙」拥有智慧，而它差点得到，却又失去整个世界一事该如何向旅行者和派蒙解释。「小家伙」不以为意，它创造出返回的道路，救下了旅行者和派蒙，与他们一起击败了秘源机龙[1]。渊上渡火者的解脱渊上调查烬寂海是为了寻找自己认定的家乡，他加入深渊教团，但深渊教团并非他的故乡。发现烬寂海能前往时间线的每个角落后前往步过烬寂海[96]。传说一种鸟，生而孤独，能在火中高歌，人民将它作为图腾崇拜，君主以之为贵。火山之地的智者佩着它的羽毛，在烈火中退隐。孤独而来的他最终也孤独而去，下落从此不明。从此以后，人们时常听到烬寂海深处传来鸣唱。不知是浴火之鸟的鸣叫，还是渡火之贤者的吟叹呢。[136]渡火者的智慧流浪烬寂海的智者（渊上）渡过能前往任何时间点的烬寂海到达许多过去的文明，换过不同的身分，期间身在四百年前某文明时身分为学徒，用赤色玛瑙打造了一尊冠冕，用以适应烈火与高温。智慧与灼热执念成就了防火的冠冕，却引来了同僚和前辈的恐惧妒忌。「这自大的年轻人，竟敢挑战烬寂海的烈焰，这是百年来未有的亵渎」「火海必将他吞噬蚕食净尽，连灰烬也被热浪扬入高空，终化做虚无」善妒的老师企图刁难他的门徒，便要他戴上冠冕，步入火海。渊上见眼前之人这副德性，知道这里不是心目中的故乡于是离去。那位导师眼见这冠冕的主人在岩浆中悠然漫步，消失在了视野之外。[137]（关于渊上的时间线，这里只是解读之一，另有其他推论）渡火者的决绝火红的花朵，仿佛闪光灼灼的玛瑙石一般。这浴火的花，被火中通行的智者佩在胸前。在渡火的智者（渊上）生命的最末，他对人们说道：「这是受烈焰焚烧而盛开的花。若我未被化作烟灰」「热浪与黑烟中必然会有这朵花闪烁着余火的光辉」后来，人们追踪着明灭的光芒，来到了烬寂海的边缘，却发现智者已经不知所踪，而花依旧盛放在余烬之中。[138]
