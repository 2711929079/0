# 命之座

名称效果流光的星火焰烈之槛或灼火之槛存在期间，当前场上角色造成的伤害提升6%；若当前场上角色处于夜魂加持状态，造成的伤害还会提升9%。对抗「古斯托特」化形的蚀灭的源焰之主时，焰烈之槛或灼火之槛存在期间，旅行者的攻击力提高40%，受到的伤害降低40%，且抗打断能力提高。长明的烛火施放元素战技流火剑后的12秒内，队伍中附近的角色对敌人触发火元素相关反应后，将为旅行者恢复14点夜魂值，每次施放元素战技流火剑至多通过这种方式恢复28点夜魂值。对抗「古斯托特」化形的蚀灭的源焰之主时，处于夜魂加持状态下时，旅行者的元素类型为火元素的攻击命中敌人后，该敌人的火元素抗性降低20%，持续6秒，此效果至多叠加2层，每层独立计算持续时间。燧传的烽火元素战技流火剑的技能等级提高3级。至多提升至15级。对抗「古斯托特」化形的蚀灭的源焰之主时，旅行者获得40%火元素伤害加成。燎灼的烈火施放元素爆发灼火燎原后，旅行者获得20%火元素伤害加成，持续9秒。对抗「古斯托特」化形的蚀灭的源焰之主时，施放元素爆发灼火燎原后，会为队伍中附近的所有角色恢复生命值，回复量相当于其生命值上限的40%，并在接下来的10秒内，每秒为旅行者恢复4点元素能量。不灭的薪火元素爆发灼火燎原的技能等级提高3级。至多提升至15级。对抗「古斯托特」化形的蚀灭的源焰之主时，旅行者的暴击率提升20%，暴击伤害提升40%。永燃的圣火处于夜魂加持状态下时，旅行者的普通攻击、重击与下落攻击将转为具有夜魂性质且无法被附魔覆盖的火元素伤害，且这些攻击的暴击伤害提升40%。对抗「古斯托特」化形的蚀灭的源焰之主时，处于夜魂加持状态下时，旅行者的移动速度提升20%，体力消耗降低30%，且进行普通攻击或重击时，还会向前方挥出流火，对路径上的敌人造成相当于旅行者攻击力96%的具有夜魂性质的火元素伤害。
