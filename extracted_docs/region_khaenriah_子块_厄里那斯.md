# 厄里那斯

厄里那斯与“哥哥”们，因为贪玩瞒着大家来到了地面上的王国，对地上的一切充满好奇，但他并不知道自己的存在对于地上王国而言是一种怎么样的灾厄。他见识到了这缤纷多彩的世界，为了不让这世界被自己破坏掉，心甘情愿地死在了枫丹。身为漆黑的造物，他随时随地能够复苏，但他却因为拒绝给周围带来不幸，而陷入长眠…
黄金王兽王兽并没有名字，因为本身不过是“黄金”无心的造物。[62]因为嫉妒与羡慕地上的狼群，它们幻化成群狼的模样，妄想取而代之。[63]坎瑞亚覆灭后，黄金王兽曾引领黑狼之群，攻击奔狼领。[64]在被北风之狼击败后，牠们逃至鹤观，只有数只兽境幼兽仍蛰伏于该处，撕裂世界的空隙，以打开能从鹤观通往奔狼领的通道，让“王兽”再次降临…[63][62][65]
