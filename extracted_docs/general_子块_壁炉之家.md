# 壁炉之家

执行官「仆人」领导的特务机构，收养全提瓦特的孤儿。根据孤儿的性别，他/她们会获得通用的姓氏，其中男性是雪奈茨维奇（英Snezhevich/俄Снежевич），女性是雪奈茨芙娜（英Snezhevna/俄Снежневна）这两个词的词根都是Снег（雪），意思是“雪的儿子/女儿”。任何企图违抗组织规矩的人，包括打算脱离组织者，都将付出生命的代价。但是在阿蕾奇诺管理的时候，不再是直接处决，而是改为喝下名为「瓶中之火」的药剂，消除其在壁炉之家生活的记忆，然后将其逐出「壁炉之家」，以普通平民的身份生活。[4]成员：林尼、琳妮特、菲米尼
