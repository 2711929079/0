# 荻花洲

受碧水河的流向影响形成的冲积平原和浅滩。此间是沙沙作响的荻草与蛙声合鸣之地，也是无数古代侠客的最后安息之所。西北角的瀑布在旅客中有着一定的名气。水草丰润,物产丰富,是轻策山周边的一方沃土。
