# 泡泡桔

生长在温暖、湿润、光照充足之处的果实，圆圆的外表下，偷偷藏着甜甜的甘鲜。据说古枫丹的哲学家们曾经进行过一场旷日持久的辩论，争议泡泡桔的颜色究竟更接近日出还是日落——究竟是将它称为「日出果」还是「日落果」。最终，相持不下的双方终于发现，「日落果」已有其名，于是双方不得已达成妥协，用形状而非颜色来命名这种甜甜的水果。
