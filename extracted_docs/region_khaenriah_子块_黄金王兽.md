# 黄金王兽

王兽并没有名字，因为本身不过是“黄金”无心的造物。[62]因为嫉妒与羡慕地上的狼群，它们幻化成群狼的模样，妄想取而代之。[63]坎瑞亚覆灭后，黄金王兽曾引领黑狼之群，攻击奔狼领。[64]在被北风之狼击败后，牠们逃至鹤观，只有数只兽境幼兽仍蛰伏于该处，撕裂世界的空隙，以打开能从鹤观通往奔狼领的通道，让“王兽”再次降临…[63][62][65]
人偶雷电影创造的人偶的创造技术是来自坎瑞亚机械工程学，由地脉树枝打造[66][41]。
