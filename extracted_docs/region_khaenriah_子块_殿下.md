# 殿下

最近，[77]旅行者的血亲继承[1]深渊教团，成为教团成员口中的公主/王子殿下，即深渊教团的统领者。如今这位殿下统领深渊教团的终极目标，包括反抗天理、淹没神座以及复国。此外除了内疚，她执行这些目标时别有原因，其中包括打算「利用整个世界，换取与天理平等对话的资格（即自己因为被降格了，所以成立深渊教团用一整个世界与天理对抗打算获取与之平等对话的资格恢复降临者身分）」，和让新坎瑞亚成为自己合适的新的家乡[28]。同时，戴因为了阻止曾经的旅伴踏上征讨深渊教团的征程[4][78][79]，途中有遇过丝柯克，并质问其剑术是跟谁学的[69]。
腐植仪式为了“让深渊殿下的国度重临于世”，深渊教团透过腐植仪式诱惑特瓦林，使其以蒙德为敌，成为深渊教团的兵器，并且拉拢丘丘人进犯蒙德。在旅行者的介入下，深渊教团的计划挫败。公主/王子殿下目睹旅行者与教团安插的魔物对抗后离开，并且在之后整个教团都刻意避开旅行者的行动路线。但是他们没料到整个提瓦特都是旅行者的行动路线，包括本该已经被封印的层岩巨渊底[80]
