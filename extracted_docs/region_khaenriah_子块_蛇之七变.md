# 蛇之七变

「蛇之七变」最初是坎瑞亚深秘院开发出来的小法术，本来的名字是「混沌渗透」，用来嘲弄大地上的元素力使用者。术士们可以用深渊力量诱发出类似元素力的变化。其原理类似于用深渊渗透现实，制造特定性质的负压，来诱发元素力的流入。坎瑞亚本就以此为幻戏，只在表演与游行里使用。极恶骑本是武人，根本不通这东西。但随着对深渊力量的理解渗入，他对这项技术可以说是一点就通。寻常人研习一种元素变化做表演之用，当卖艺糊口的把戏都需要数月的功夫，极恶骑初次尝试就可以七种元素次第变换了[29]。
迫近彼时的坎瑞亚，使用一种名为阿索斯物质的地脉能量为机关供能，[30]但坎瑞亚仍不满足，妄想取得新的能源，以取代地脉能量。[11]为此，他们向天空祈祷。“天空回应了招唤”，旅行者的血亲降临在坎瑞亚[31][32]。旅行者血亲原本沒有來到坎瑞亚前的记忆，直到在坎瑞亚待一阵子后记忆才逐渐浮现[28]。因为不知道自己是谁，沒有一切记忆的情况下，旅行者的血亲被迫成为坎瑞亚王储。坎瑞亚人将其视作「深渊」的代表。旅行者的血亲感受到了坎瑞亚人民的善意，将坎瑞亚视为自己的第二故乡，决定帮助他们，伊尔明将旅行者的血亲立为王储，并带着他/她一步步下到了王国的最深处。此时的戴因斯雷布，还不过是个默默无名的青年骑士[33]。旅行者的血亲降临坎瑞亚后，曾和当时身为宫廷法师的皮耶罗生活一段时间。[31]之后，旅行者的血亲成为坎瑞亚的“公主/王子”，坎瑞亚人遥望着这位金发异乡人，希望能掌控深渊之力的旅行者血亲能帮助他们开启通往新能源的大门—因为他/她就象征着深渊，象征着超乎常理的一切。[1]黑王伊尔明与高层们大力鼓吹深渊力量，旅行者的血亲甚至被称为「救世主」[28]。至此，坎瑞亚人引入深渊力量，灾厄随之降临…
