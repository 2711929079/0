# 遗迹守卫

学者和歌者认为，地脉之中流淌着世间的情感与记忆。[59]因此，依靠地脉能量驱动的遗迹守卫虽然没有自我意识，但在名为坎瑞亚的故乡覆灭之后，它们在百年的游荡之后，仍然会来到遗迹的断垣残壁间停下，并用爆炸性的武力守护这些遗迹。因为唯有在这些遗迹旁，它们才能得到些许慰藉[9]
杜林坎瑞亚覆灭后，莱茵多特释放大量漆黑魔物，杜林也是其中之一。它听闻风之歌者的琴声前往蒙德，与人们一同歌舞，欢唱。但实际上，那个琴声是只在它脑中回响的声音，是将它引往蒙德破坏的诱饵，它以为自己在和人们共舞，但实际上却造成了无法挽回的灾厄。最终，它从梦中惊醒，在白雪之中，看着风之歌者指引蓝宝石般的龙，将利牙刺入自己的喉咙…「再见了，妈妈，我的旅途结束了」「沉睡在白银般闪耀的雪中也不错」「再见了，美丽的诗人、美丽的龙」「如果我们能在不同的时间、地点」「相遇、欢唱、共舞，那该有多好」垂死的他由衷地想。「在我的血管当中鼓动的深刻祝福」「我所诞生的漆黑宇宙的美丽图景」「就由你们继承吧」[60]后来，杜林长眠于龙脊雪山，吞下漆黑之血的特瓦林陷入沉睡。古树吸收了杜林的漆黑之血，继承了它的意志…魔女M因不满杜林的结局而在童话故事中创造小杜林。[61]
