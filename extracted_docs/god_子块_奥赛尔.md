# 奥赛尔

有部分人认为奥赛尔（Osial）的名字来自所罗门七十二柱魔神第57柱欧赛（Ose）。但奥赛尔和欧赛完全没有对应之处，因此祂的魔神名Osial可能是取自能对应上的第68柱贝利亚尔（Belial），又或是Ose与Belial的结合，但并没有其他魔神命名是采同这个方式，且这样的命名完全意义不明，这里默认其原型为Belial。
「Belial」是象征着「无价值」、「无赖」、「破灭」等一切归于「恶」的意思。《死海古卷》〈光之子与暗之子之战〉称其为天之副君主、暗之军团的大统率、叛天的首谋首恶。自称原名Satanail（Osial的S可能来自于此），是所有堕天使中最危险、凶恶的一名。他在堕天后最成功的一次反扑行动是唆使犹大王国再次悖逆神，转而膜拜其邪术，以致见弃于神、被巴比伦灭亡。《哥林多后书》第六章十四节以「黑暗之子」称之，因为其身份和耶稣基督对立，经注释为撒旦的别名。与《原神》中奥赛尔作为大反派，也是第一个与主角正面对上的象征着极恶的魔神对应。暗之军团的大统率、叛天的首谋首恶，与统帅深渊教团的殿下打算将其打造机械魔神并反抗天理对应。在天使们都在为上帝尽心尽力传福音的同时，却拒绝了上帝所执行的命令，反倒是喜好以自已的方式和习惯行事，比起其他天使显得更加独立，却也更加不合群。明明是众天使之首，拥有如此荣耀的身份，却如此的叛逆，不愿诚心为上帝奉献。对应不管在现今还是魔神战争时奥赛尔都实力强大，魔神爱人的特质却似乎特别薄弱，与其他魔神不同直接大肆侵略璃月港，彷佛只想破坏与向岩神及其子民复仇一般。纪录Belial事迹的《死海之卷》让人联想到「漩涡之魔神」与「海」有关的设计元素。上帝将其打入了地狱深处，让它在地狱的浓烟和烈焰之中，承受永不止尽的折磨，并在痛苦之中思索自己所犯下的罪行与被岩神镇压在海中的奥赛尔对应。《冒险家罗尔德的日志·孤云阁》也曾记录下「夜里睡得很不安稳，梦境总是黑暗潮湿的。我彷佛梦见自己是被岩神刺穿在海底的魔怪，正扑扑簌簌地挣扎、抓挠着坚固的岩枪，每一番动作都带着极大的痛苦与憎恨…」。奥赛尔的形象可能受中国古代神话传说中共工和相柳的启发。共工是水神，被列为「四罪」之一，对应Belial象征着一切的恶；相柳是共工的臣属，九首蛇身，最后被禹所杀。
