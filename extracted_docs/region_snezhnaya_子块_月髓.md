# 月髓

「月髓」是月神死后力量凝结而成的东西，本质是一种力量核心。「恒月」月髓由霜月之子作为圣物保存至今，详见至冬#霜月之子；「虹月」月髓如火焰般燃烧不灭，由坎瑞亚人保存并研究，详见至冬#坎瑞亚；未曾破碎的「霜月」则孕育着新的月神。
「月矩力」与「月神的眷属」月光（详见至冬#挪德卡莱·空月之歌（主线剧情））洒向挪德卡莱的大地，月矩力开始影响这片大地上的生物。只要是在月矩力下长大的动物，都可以算「月神的眷属」[24]。
