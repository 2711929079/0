# 喜多院

喜多院是一个精通枪术的家族，被任命为八蕴守，守护八蕴岛的祟神之力。[43]500年前，喜多院家族的最后一代—喜多院秀家，也是当时的八蕴守。当时他即将与将军一同参与坎瑞亚的远征，身为巫女的母亲希望儿子能在战场上归来，于是把用妖怪蜃气楼遗骸制成的宝物交给他。
然而战场上，友人的死让秀家彻底崩溃，他逃离战场，但在之后的日子里他十分自责，一直无法原谅自己成为逃兵的事情。后来秀家在生命最后来到流行蜃境，希望蜃境中的纯水精灵伊迪娅能帮助他重返战场。随着那场战争的景象呈现在眼前，他才终于释怀，写下最后的遗书后结束生命。[44]
鹤观漆黑灾厄发生后，阿瑠陷入循环的灵魂看到丘丘人、遗迹守卫、兽径猎犬明显变多了，鹈猿也几乎灭绝[3]。
