# 审判庭

审判庭，即「欧庇克莱歌剧院」。「欧庇克莱歌剧院」位于伊黎耶岛，是岛上最为宏伟的建筑。一般枫丹人口中的歌剧院便是指「欧庇克莱歌剧院」。歌剧院除了承办各类文艺演出以外，还有一项更为重要的作用，就是作为「审判庭」，见证枫丹所有案件的审理。前身为光荣剧院，但在五百年前深渊入侵时的涨水期而毁掉。如今审判已经成了崇尚「正义」的枫丹文化的重大象征，在枫丹，审判被当作演出，其与表演的界线并不清晰。审判时，由审判官与预示裁定枢机共同审判，最终结果以预示裁定枢机的结果为准，因其象征着神明的裁判。预示裁定枢机会听从证言最初判决，如今其原本的作用已失去，改为审判官全程掌控审判。如今枫丹最高审判官为那维莱特，地位崇高。
