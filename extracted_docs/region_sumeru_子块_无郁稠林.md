# 无郁稠林

守护须弥北方的茂密丛林，其中生长着其他地方很难见到的巨大花束。过去这里曾被侵蚀世界的污秽所垢染。在经历了漫长的净化与复苏后，才重新成为今日遮天蔽日的庞大森林。荧光闪烁的巨蕈之林巨蘑的伞盖构筑幽光的天幕，遮日蔽月，似是谁人培育的乐土，亦或静待生机的园圃。
