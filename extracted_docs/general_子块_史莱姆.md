# 史莱姆

并不起眼的凝胶状元素生物，拥有最基础的智能。但是放着不管的话，有时候也能引发小小的灾害…？史莱姆可以说是一种对人类来说很宝贵的生物，它的流体所形成的史莱姆凝液是各地元素工坊中最常见的材料，而史莱姆清虽然对皮肤有害，但是所蕴含的元素力更高，也是更为高级的材料，史莱姆原浆可以说是不完全的史莱姆，因为它静置时会自主的进行不规则的运动。
