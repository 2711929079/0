# 酒会

在那个酒会上，狐斋宫还认识了新任天狗族长灵善坊，和面容皎美的鬼族少女御舆千代。也是在这次酒会上，一个自称御伽金刚狮子大王的壮汉，提着全名為「赤角石见金石溃金涂金啮御狮子杵」的刀闯入会场，自信满满得说要除妖灭鬼。御舆千代也在兴头上，便同他摔跤。即使千代在他面前显得十分娇小，最后这位大王还是摔得人仰马翻，害羞得溜了出去。[23]会场上洋溢着欢乐的气氛。酒会中，狐斋宫让妖狸头子五百藏偷雷电将军御苑里的鸣草，自己陪在影将军身边。五百藏被逮住后，狐斋宫一顿求情，不仅免了五百藏的罪，还赐予他保生司正一职，守护镇守之森。酒会上，雷电将军请御舆千代献上一舞，狐斋宫与其一同舞蹈。结束后，天狗族长又邀请狐斋宫比比谁跑的更快，结果是打成平手。狐斋宫自己也清楚，天狗族长是让着自己。酒会结束后，狐斋宫回到鸣神大社，面见了浅濑响。狐斋宫把自己能教的东西都授予浅濑响后，便将其引荐给天狗族长。
浅濑响与昆布丸浅濑响清籁岛因浅濑神社的发展，逐步有人定居，形成了越石村。越石村充其量也只是个小渔村，也因为远离鸣神岛，并没有发展得很繁荣。一直到约500年前，浅濑神社的宫司浅濑响，决心发展家乡的神社，便前往鸣神大社进学。浅濑响一度师从神通广大的八重神子，学习打理神社事宜。那时的她，不过是从小渔村（清籁岛的越石村）来到鸣神的幼稚巫女。比茶筅还要愚钝。也还未曾褪去孩童的任性与好奇，对斋宫大人优雅难懂的话语。总是抱着天真的怀疑。她曾在秋夜的坂道上，和斋宫大人同赏蝉鸣与月光。那时的她还不过是一个乡下巫女，年幼而无比倔强。像一只叽叭喳喳的团雀般，聒噪着坚持自己的见解，望着狐狸大人浅笑的面庞出神，却未听懂她的话语。[25]
