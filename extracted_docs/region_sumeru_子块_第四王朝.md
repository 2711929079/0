# 第四王朝

蓝宝石城现任城主古达尔兹，与妄图称王的新居尔城城主大战三日，（声称一开始不能攻克居尔城，但在所谓赤王的幻象加持下轻易取胜），而后迁入居尔城，（声称在赤王幻象祝福下）称“撒勒布·道莱王”。在胜利后的十年，民众迎来了“苍暮细雨”的繁荣的十年，就连附近的海盗（即代伊）都来称臣（此句多半为假，因为根据后续碑文可知，海盗在朝贡路上也是烧杀抢掠无恶不作，甚至将城市拜特拉哈屠城，将城主悬首示众直至腐烂，还立其女为新城主）。道莱王杀死了自己的皇子（并没有说原因，猜测是他的儿子有谋反之心，或许是引用了国外某个皇帝杀子的故事）装作悲愤，并令数千奴隶陪葬。之后自己的兄弟，子嗣与侄儿为皇位互相开战，年老的道莱王向海盗求助，海盗一边烧杀抢掠一边平定战事，最后终于向道莱王宣战，道莱王和他年纪最小的弟弟一起对抗，终不敌。直至最后，道莱王才明白自己背叛了自己的亲人与忠臣[38]。
第五王朝“代伊王（海盗）”穆扎法尔死后，由其子“绿之王”祖玛鲁德称王，绿之王某天做梦梦见沙漠变为雨林会更加繁荣，于是大力种树。代伊王告诫自己的儿子要多生孩子，以免发生道莱王的悲剧，所以绿之王有九十九个孩子，四十九个女儿，五十个儿子。绿之王死后，他最有实力的两个儿子——巴达那与穆祖丁为了王位互相争斗，最后巴达那取胜。巴达那即位后逐渐贪图享乐，变成了整日与美食、美酒与游猎为伍的暴君，甚至将自己的忠臣挖去双眼。忠臣离开王国，流浪世间，成为老朽的盲眼诗人，向沙之民收集沙海的诗篇。王子法拉玛兹与当时还是舞女的玛海菈（镇灵之后）有了除掉国王的念头。玛海菈误入遗迹遇到鹮之王，回答祂的问题，图特因此给予她「被爱」的能力[38]。巴达那死后，新王法拉玛兹即位，因其年轻，王后玛海菈摄政。[57]不久便禅位于其叔父，“蓝宝石城城主”，相国瓦拉什。国舅格拉兹起兵推翻瓦拉什，复立法拉玛兹为王，自己摄政。不久后罢免法拉玛兹，自己称王，玛海菈为皇后。蓝宝石城贵族为拉瓦什复仇，推翻格拉兹，复立法拉玛兹为王，王后摄政。勇士之城城主反叛，自立为王，王后平定叛乱。之后法拉玛兹的十二个叔父全部叛乱，战争连连，民不聊生。战争平息后，法拉玛兹再度即位，王后玛海菈一心一意想要成为皇后，但不在意国王是谁，于是法拉玛兹下毒杀害玛海菈，玛海菈中蛇毒而亡。不久奴隶兵与市民里应外合推翻法拉玛兹，并将其流放雨林。[58]在这之后，沙漠子民厌倦了战争与王朝，在盲眼忠臣领导下前往沙漠边缘建立了新的定居地，即阿如村。
