# 望风山地

高耸的交错山地，曾经是蒙德应对海上风暴的前哨。时至今日，风暴早已不再，剩下的只有空置的哨塔和游荡其间的怪物还有一位永久改变了望风山地地形将其变成平地的小太阳可莉。
