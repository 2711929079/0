# 布基纳

来自特诺奇兹托克部族的特诺切与来自悬木人的讯使布基纳是好友。布基纳曾与名叫“康加玛托”的匿叶龙赌气，因为后者嗤笑他动作痴愚。为了通过摘取山巔上母树结出的火榴果来证明白己，布基纳前往了花羽会观察龙的肢体动作，在沃陆之邦锤炼体魄，还去烟谜主敏说了自己的感知与意识。特诺切在此期间见识到了布基纳的决心，于是帮助他共同打造了一把武器硕果钩。行到母树结出的火榴果成熟之日，布基纳凭借自身积累的经验与趁手的式器迅速接近了火榴果，但突如其來的强风让布基纳陷入危险，好在一直在暗中观察布基纳的康加玛托及时出现救下了布基纳。最后，布基纳把火榴果献给了康加玛托，并得到了康加玛托的认可，二者成为了伙件[109]。
图帕克来自流泉之众的图帕克认力自己生不逢时，活在了平庸的时代，于是决定前往未知的外海。一路上，图帕克击败了许多巨兽，他将翼龙的双翅制成精帆，将沧鲸的骨头制成白身，将海蟒长牙制成回旋镖「乘浪的回旋」，将巨鱿做成下酒菜，最终他成功抵达了海的边界[110]。
