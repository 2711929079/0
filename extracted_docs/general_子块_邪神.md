# 邪神

暗之外海
提瓦特沿海的诸国把超越七神庇护的外部称之为「暗之外海」，据说许多战败的魔神与龙族不愿意活在新的七神秩序之下，于是逃到了远岛，化作了邪神。魔神奥罗巴斯曾经在魔神战争期间便一度逃往了暗之外海，化作了邪神，并在外海获得了馈赠——蕴含超常力量的珊瑚枝，随后再度回归提瓦特大陆。与稻妻比邻[29]。海祇岛大将海御前据说在大蛇东侵失败后前往了暗之外海[30]。在暗之外海会因为深渊侵蚀失去自我[31]。
