# 提纳里技能详细介绍

提纳里是《原神》中的草元素五星角色，须弥道成林的巡林官，称号"浅蔚轻行"。
元素战技「识果种法」：
- 释放草元素能量，对周围敌人造成草元素伤害并施加"草元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「荒泷一斗」：
- 创造一个持续10秒的草元素领域，领域内的敌人会受到持续草元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "识果的祝福"：识果种法命中敌人后，有15%概率触发"激化"效果。
- "草元素增幅"：提纳里的普通攻击和重击造成暴击时，恢复2点元素能量。
- "巡林的守护"：元素爆发持续时间内，提纳里的草元素伤害提升15%。
