# “暴君”暴政 终遭讨伐

当时跟随希巴拉克和奥奇坎进入大火山的英雄们，都已成为各部族领袖的众英杰。这些英雄中，只有阿霍布（沃陆之邦英雄）因父母皆为龙族所杀，对龙族深恶痛绝，响应了奥奇坎的号召，带领大量沃陆之邦战士到回声之子的领地，要求其胞妹伊葵交出“藏匿山间的凶兽”。伊葵不愿无辜的龙众因奥奇坎的无理命令受到牵连，因此与其兄争执。伊葵的手稿存世极少。包括，几乎全部被她本人焚毁在了工坊里。伊葵点燃了自己的工坊，随后逼迫兄长与她一同自尽，以免他在历史上留下与暴君奥奇坎一样的恶名（烧毁工坊的目的是为了避免自己掌握的锻火的技艺、以及其他记载的研究成果落入奥奇坎手中。这一举措也致使伊葵的手稿存世极少、技术失传）。阿霍布和伊葵死后，奥奇坎假意宽恕龙众，让失去首领的沃陆之邦和回声之子暂时将权柄交给联盟，等到两大部族选出新的领袖后再归还旗帜，实则在暗中记下包庇龙众之人的名字，待到合适的时机将他们暗杀。莉安歌（花羽会首领）的儿子支持奥奇坎的屠龙之计，又料定其母亲必然会反对奥奇坎的计划，因此向奥奇坎谏言，在莉安歌反叛之前，对莉安歌先下手为强。奥奇坎在奥奇卡纳塔设宴邀请莉安歌前来，实则让几百个兵士一同围攻莉安歌，察觉到奥奇坎阴谋的莉安歌先是一箭射杀出卖自己的逆子，随后又与数量庞大的奥奇坎部下拼杀，最终寡不敌众，力竭而亡。直到最后，莉安歌都将奥奇坎视为误入歧途的友人，而非敌人。奥奇坎解决了最后的心腹大患后，彻底开始了暴戾统治的时代。得知龙城地下藏有古龙的庞大遗产后，奥奇坎立刻以兴修水利之名广抓壮丁，强迫他们“自愿”抽干地下遗迹的积水，不少工人在这次工程中丧命。奥奇坎见到了龙族旧日的统律核心·L型自律监管机阵统合处理单元「伊·奎库叙·茨博隆·库·雷尔」（CL-09，即伊蕾尔）。伊蕾尔称呼奥奇坎为命定之王，向他展示了龙族的历史，以及龙族所具备的各种尖端科技，希望能够规劝奥奇坎回归龙众，但奥奇坎在看过龙族过往具备的强大力量后，反而加深了认为“龙族危险”、“无法与龙族和解”的想法。因此奥奇坎利用焰主血脉的权能，剥夺了伊蕾尔的权柄，篡夺了秘源技术的控制权，让伊蕾尔在不死去的情况下为秘源技术持续供给能量。奥奇坎坚信，希巴拉克才是唯一的人类之神，认为过多的统治者会导致国家像北海的蛮族（枫丹）和砂中的列邦（须弥）一样走向内斗分裂，纳塔只能有一位王，一位神。奥奇坎建造了天蛇船，在天蛇船上留下铭文要求世人不要信仰龙与夜神，字里行间无不在赞颂希巴拉克的伟大。挖掘古龙秘宝的深井遭到深渊的完全侵蚀，为了不让深渊力量外溢，奥奇坎用圣火将地下遗迹连同工作的工人们完全焚烧，几乎所有工人都惨死在了深井中，除了一个名为曼科的工匠。曼科得到了统律核心的馈赠，获得了大量来自古龙的知识，最终成功逃出生天。后曼科加入了赤瞳少年反抗暴君的行列。深渊的外溢终究难以抑制，为了不让深渊波及到由希巴拉克一手建立的国家，奥奇坎将被深渊侵蚀的奥奇卡纳塔完全封锁；又将通牒金牌藏在天蛇船上，使库库尔坎无法取得；最后自己吞下深渊的源头，将意识储存在了统律核心之中。临终前，奥奇坎在天蛇船上留下最后的铭文，斥责盗火贤者库库尔坎的残忍和天空的无情，认为唯一可行的只有希巴拉克的路。库库尔坎前来找他要金盘，但得知金盘被藏在天蛇船上后的库库尔坎怒不可遏。

折叠
…留给我们的唯有一条路，那就是希巴拉克的道路…
此时，赤瞳的少年因为包庇龙族而被逐出花羽会，受一位老妇人的启发站出来，他历经试炼，最终模仿希巴拉克用黄金匕首刺穿了自己的心脏，其灵魂进入了夜神之国，并且获得了夜神之国的加持，再次复活归来，成为新任火神。赤瞳少年率领反抗军攻入了奥奇卡纳塔，大败奥奇坎的军队。暴君奥奇坎在这场战役的最后，于赤瞳少年面前摘下面具，口中喷出金色的火焰，将自己烧成了熔化的金属，坠入了深渊的漆黑泥沼之內（实际上是由于深渊的入侵，奥奇坎分离了自己的意识，而肉体和灵魂则化作了紫色巨龙，之后此龙时常盘旋于奥奇卡纳塔遗址，袭击任何经过此地的人）。奥奇坎暴政结束后，各部落取回自己的权柄，纳塔开始稳定的发展，直到五百年前的坎瑞亚灾变毁于一旦。
