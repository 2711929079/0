# 结盟

传说中，赤王「阿赫玛尔」是天空遗落的子嗣，而如今是沙漠地区的王者。[14]在远古时代的须弥，花神先是遇到了赤王。赤王与花神共同为镇灵修建了圆形剧场之城「阿伊·哈努姆」，在古镇灵语意为「月女城」。[15]花神曾经留下三个谜题，赠予赤王，但赤王直至沉沦于黯毒的梦乡，也未将其意晓畅[16]。在听说须弥有着一位智慧的王女后，花神带着礼物与谜题去拜访了大慈树王。最后，在树王顺利回答了所有的谜题后，二位女王便缔下了坚实的盟约。欢宴属于花神，权威属于赤王，生命属于树王。仿若白银的月亮、黄金的太阳与翡翠的绿洲，三位神王立下誓言，结成同盟挚友。[17]在他们三人的努力下，须弥真的诞生了一个安乐和平，没有灾祸，统一无忧的乐园「瓦力·韦杰」。有关于三王共同治理时期须弥的描述：「在那些年代，月光曾将他们的幸福讲述给夜莺与蔷薇，」「而她们也为之惊惶、为之羞涩，以致唱不出回应的歌。」[17]赤王、花神与大慈树王也时常举办宴会。传说在烈日君王夜夜不休的宴席上，枣椰所制成的蜜饯便是第一道开胃的美食。[18]有顶塔实际上是建构地上乐园时大慈树王暂居的地方[8]。
首次花神诞日在第一次的花神诞日，赤王与花神为大慈树王准备了惊喜。他们沿途散播糖果作为线索，引大慈树王前往宴会之地。好奇的大慈树王跟着来到一处湖边。忽闻一声琴音，原来是赤王开始拨弦奏乐，而花神则跳起美丽的舞蹈。他们共同庆祝大慈树王的生日，并献出礼物—花神献上了灵光，赤王赠出能纪录过去的宝石「乌婕特之眼」，用以纪录三人相处的时光。乌婕特之眼力量特殊，只有赤王后裔才能与之共鸣。[19]
