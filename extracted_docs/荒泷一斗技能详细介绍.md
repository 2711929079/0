# 荒泷一斗技能详细介绍

荒泷一斗是《原神》中的岩元素五星角色，稻妻荒泷派的头领，称号"花坂豪快"。
元素战技「荒泷逆刺」：
- 释放岩元素能量，对周围敌人造成岩元素伤害并击飞。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「魔像攻击」：
- 释放强大的岩元素能量，对周围敌人造成岩元素伤害，并进入"狂怒"状态。
- 冷却时间：20秒。
- 能量消耗：80点。
固有天赋：
- "逆刺的威力"：荒泷逆刺命中敌人后，有15%概率触发"结晶"反应。
- "岩元素增幅"：荒泷一斗的普通攻击和重击造成暴击时，恢复3点元素能量。
- "魔像的威严"：元素爆发持续时间内，荒泷一斗的岩元素伤害提升20%。
