# 雷鸟

雷鸟因不明原因定居在鹤观，雷鸟拥有一个「她」取的名字，但是雷鸟不需要名字，也没人敢呼唤雷鸟。后来这个名字她自己也不记得了（之后她才被阿瑠命名为卡帕奇莉）[3]有说法是天理想让她成为魔神于是赐予她魔神名，但她因忘记自己的魔神名因此没有成为魔神。
雾海天钉落下后[5]，迷雾瞬间笼罩鹤观，鹤观后文明在此时成立，但远不比先前覆灭的鹤观前文明先进。鹤观后文明的人们看着雷电之鸢在迷雾中的穿行，拾起她掉落的羽毛，在伸手不见五指的浓雾中，传递信息，于是人们认为，这迷雾是雷电之鸢降下，为了保护他们。鹤观后文明的人会补捉与饲养鹈猿为食。[3]在雾气环绕鹤观的时代，岛外有一支部族，名唤作夷摩美氏，部族的主母大狐白辰一脉，是一个狐狸样的女人[9]。
