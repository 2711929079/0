# 赤穗百目鬼

「快下去。女人在船上很碍手碍脚！」唤作赤穗百目鬼的贼人说，背过身去。但听了这话，巫女却忍不住微笑起来。如果当初教我射术的人没有随将军出征的话，那我们的儿子，如今应当是左卫门的年纪了，而我或许就应当姓高岭，或者由他姓浅濑了…左卫门的口吻，和故意背向我的模样，简直和他当年提着刀离开的时候一样。那么这次，我绝对不会让这个人死去，哪怕与「雷之三重巴」之旗为敌也好…「扬帆之刻到了，铦与刀都已磨利。」「就让那些官兵看看，清籁的骨气！」[55]
清籁雷暴天领奉行开始准备集中清剿海盗。这让浅濑响倍感不安，如果海盗被抓住，一定会人头落地的。但是，她只是一个巫女而已，能做什么呢？只能破坏封印释放魔枭的遗恨招来闪电击沉奉行的船队。但这么做对不起祖先，对不起狐斋宫，也对不起天狗族长。她在动手前回到了鸣神岛，回忆着过去在这里发生的一切，在内心表示歉意后，本打算离开的她。发现了一个魔物向她走来，她认出了那是被执念驱使，为了完成当年赌约的雾切高岭。搭弓射箭，一气呵成，雾切高岭早就死了，是为了稻妻战死的，留下这样的历史就可以了。[25]收拾完一切后，将自己制作的弓留在了海岸，浅濑响踏上了赤穗百目鬼的船回到清籁岛。早年曾偷学的关于家乡封印的法术，没想到最后还是派上了用场。封印解除，怨念吸引来了雷音权现，引发雷暴，浅濑响死于雷暴中，雷电肆虐在海上，击沉了奉行的船队，但是赤穗百目鬼的船越石丸也被击沈，只好从海贼蛇目处取得一艘船逃跑，漂流到雾海群岛，被雾海群岛第一批住民救下，在与海民的共同努力下成功回到稻妻[56]。几年后，赤穗百目鬼从雾海群岛回来，已经找不到浅濑响了，于是金盆洗手，环游世界。[55][57]因为雷暴影响，越石村废弃。在幕府的统一安排下，清籁岛上的人基本都迁到了鸣神岛上。[58]在稻妻的海螺屋阿敬处可拿到破魔之弓图纸，图纸获得是剧情的一环，如在柴门家拿到天目影打刀图纸。因此其可能是浅籁响将图纸给赤穗百目鬼，百目鬼在约四百年前遭遇海难漂流到雾海群岛遇到那里受困的海民。之后海民与百目鬼成功逃出群岛来到稻妻，海民在稻妻定居，百目鬼打算游历世界，因此将图纸交给海民，而阿敬就是海民后代，据金苹果群岛回声海螺记载，海民与他们祖先善使海螺捕鱼，所以其后代可能也会玩海螺。
