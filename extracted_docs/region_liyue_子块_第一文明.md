# 第一文明

法涅斯占领提瓦特前。层岩巨渊地下矿区曾是海洋地型，在法涅斯与龙的大战中地型变动层岩巨渊成为陆地，许多原先在此生存的生物灭绝成为化石[1]。天使带领人们建立文明，即第一文明。第一文明的建筑是石砖堆砌，上面有凯尔特三角结，样貌十分容易辨认。在碧水原南部，龙脊雪山西边，琼玑野西边的那片区域，与孤云阁区域，存在许多第一文明建筑。（由此可知孤云阁的大部分虽然是岩神长枪，但孤云阁原本也是有岛屿，不是全由长枪构成）在奥藏山北部与碧水河相接的地区，也存在零星的第一文明建筑，与地灵龛、秘境-华池岩岫共同组成了遗迹群。奥藏山山脚、庆云顶山脚、靖世九柱、太山谷、天遒谷附近也有第一文明建筑。纯水精灵的场地也是第一文明建筑，可能是大型观景庭园。仙众们诞生于天地间拟荡的元素力，是至纯的元素生物[2][3]。岩王帝君是最初的仙人[4]，最初是「岩主天星」[5]（疑似为陨星）。第一文明人民也会求道修仙。太山府在古代第一文明人用于对求道修仙之人的试炼[6]（太山府也是第一文明风格建筑）
层岩巨渊与沈玉谷在层岩巨渊地底，第一文明的分支建立倒挂的地下城市，并拥有净化深渊的圣水的力量。[7]部分人类在现如今层岩巨渊地表建立起古国「琅玕」，依山逐岭建立居所群落。地下的文明与琅玕国是不同的文明[8]。「炸春鲜」是琅玕国的菜肴[9]。天使兹白受时间执政选为月使，获得月之轮[10]，在月亮上与䴉之王图特等月使共事[5]。琅玕国有一神使，即兹白，是该地社稷神，佑得此地风调雨顺，引导并庇佑人们[11]。天使（兹白）赠予人们玉石，并指引他们用其筑起通天的祭台以聆听来自高天的指引[8][12]。在白马仙人显灵的仪式「逐月」中，人们会念祭辞并放月灯，祈求白马仙人庇佑[5]。后来葬火之战后层岩巨渊有天星（可能与「岩主天星[5]」为同一物）坠落，此天星实为晨星之神的高车。随着天星坠地，直面冲击的荒原化作宏大深邃的巨渊，美玉金石从中生长而出，取之不竭，因而成就了璃月此后千年的采矿产业。天星的坠落为层岩巨渊带来了名为“琉璃晶砂”的矿物。部分居住在层岩巨渊的人民选择留下来开采这种矿石，以及制作陶瓷维生[13][14][15]。传说天星坠落之时，曾有一块碎片迸溅而出，正落在璃沙郊北部的群岩之间[16]。昔日兹白以月使身份降临人间，曾结识了一位神异之人。他为人类之姿，面目不凡，却能腾云驾雾，搬山填海，震岩碎玉。可身具如此神通，却又并非天使一族，令兹白十分惊异。一日，二人相约在山间切磋仙法，时候到了，不见其人，却听着天边隆隆。兹白循声而去，便见着云间有三道雷同的身影，正彼此相伐。刀兵碰撞间，她隐隐听得，云中传来「岩主天星」的攻喝声。少顷，战斗结束，身影又化三为一，落在兹白面前，正是那位友人。兹白对此分外好奇，那人也不藏私，慷慨相授，原来此为一古技，名曰：斩三尸。所谓三尸，意为修行者心中的浊念与恶欲，斩去三尸，得见澄净自我。但兹白身为高天与三月的使者，心中从未有过妄念，无论如何尝试，也不能斩出三尸神。那友人也并不在意，只是留下一句「天心无定，白云苍狗」便淡然离去了。那时她还不知，在此后指引人类的漫长路途中，她会如何潜移默化地染上人类的情绪。而在更遥远的未来，当她亲眼目睹天罚落在她所珍视的国度上时，愤懑，惮慑，怫郁，会如何像附骨之蛆般啃噬她的神魂。她一定会在那个时刻斩出自己的三尸神，那便是她割舍自我、蛰伏人世，留作向高天复仇的后手[17]。在历经数代人时间的的传承发展后，地上的人们建立了规模性的采矿人群落，但以开采矿产维持生息的地上人却不知道，在矿坑之下所暗藏的葬火前纪元文明，正酝酿着不为常人所知的深远罪恶[8]，接触与使用禁忌知识。而天空岛也因此震怒而撤回了使者[18]，並对这个忤逆的前纪元遗留文明施以天罚灭绝[19]。天钉降下，层岩巨渊地表因此弥漫大雾[20]。而判罚之钉的降下也没有宽宥凡人，地上毫不知情的采矿人部族，也因为地下遗民作孽而被连带波及遭殃。从而为了避祸被迫背井离乡另寻出路，这些迁居的采矿人族群就是所谓“沉玉谷先民”的前身，对于这段历史他们也以壁画和传说的形式保留了下来。天钉降下后层岩巨渊不再适合居住。居住在层岩巨渊的人们在兹白带领下迁移到遥远的沉玉谷，在沉玉谷筑起了直通高天的祭台（赤望台）[21]，留下无数恢宏的殿宇[8]。沉玉谷得名于当地的传说：沉玉谷的山上有一块玉，为了不落入妖邪之手，这块玉被扔进了水中。兹白因为保护琅玕国民硬接天钉违反天意而陨落。兹白的灵魂在陨落前被时之执政定格，以切割灵魂的仪式（割尸神）[22]化作三缕本源，封存于三尸神体内。若灵魂重聚，其或将再度跨越时光的缝隙，重返人间。兹白生前喜爱凡人，三尸神虽性格各异，却皆与这人间烟火有着千丝万缕的宿缘。部分天钉在降下并解决该地深渊后会飞回天空岛（如鹤观[23]），层岩巨渊因深渊力量须天钉常年封堵，天钉一直留在层岩巨渊地下矿区。山主灵渊一直存在于沉玉谷地区，后来与仙人浮锦、仙人药君结识。浮锦化作人形为沉玉谷先民种下第一棵茶树，并逐渐改变沉玉谷水土，让其变得更加适合人类生存。而药君也化作人形，为人类治病。沉玉谷的人们试图通过将玉石投入河中的仅式重新与天空询通，但是失败。后来投𤦻仪式成为节庆仪式，每次举行仪式都能为玉石积蓄仙力，投珑仪式由浮锦主持。之後一位魔神来到沉玉谷，灵渊、浮锦、药君开始为其效力，共同宁扩沉玉谷的地区[24]。层岩巨渊古文明古时的祭场曾因葬火之战埋入地下，又随着天钉陨落而隆起[25]。
当时虚假之月存在，并遮挡住真实之月，此现象在沈玉谷的传说中名为「詹诸吞月」[5]。
