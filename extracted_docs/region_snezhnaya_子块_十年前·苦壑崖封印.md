# 十年前·苦壑崖封印

十年前霍德望的仆从「靡穷之孽」尝试突破封印，最终失败[34]
讨伐黑灾起始之地自铸灯者在女皇的御前立下永誓以来，已过去了数百年之久。「乐园」中的人们相信，只要灯塔不曾熄灭，他们便能得享自由与平安。百年未见的狂潮开始肆虐，集结的军号响彻山野，执灯人战士们在高塔下列队，等待着南下伦波三岛的号令。然而在北方，灿如极昼的大军（愚人众）也已进至城下。执灯人将要讨伐的是黑灾起始之地，至冬无法承担失败的代价。若执灯人失败，就将由愚人众负责善后。[45]
