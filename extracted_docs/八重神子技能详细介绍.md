# 八重神子技能详细介绍

八重神子是《原神》中的雷元素五星角色，稻妻鸣神大社的宫司，称号"浮世笑百姿"。
元素战技「野干役咒·杀生樱」：
- 放置杀生樱，持续攻击敌人并造成雷元素伤害。
- 最多可以放置3个杀生樱。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「天狐霆雷」：
- 释放强大的雷元素能量，对周围敌人造成雷元素伤害，并强化杀生樱的攻击。
- 冷却时间：20秒。
- 能量消耗：80点。
固有天赋：
- "杀生樱的祝福"：杀生樱的攻击有15%概率触发"感电"效果。
- "雷元素增幅"：八重神子的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "天狐的威严"：元素爆发持续时间内，杀生樱的攻击伤害提升20%。
