# 千手百眼天下人间

旅行者在社奉行的委托下暗中反抗眼狩令，而条件是社奉行会将旅行者引荐给将军，使旅者和将军得以见面。在第一百颗神之眼献祭仪式上旅行者为了救人情急之下进入雷电影的一心净土使八重神子看到了事情的转机。但旅行者也因此被通緝，投向海祇島的反抗軍。愚人众第八执行官「女士」向反抗军提供邪眼，毒害了大批有志之士。旅行者为阻断邪眼制造前往邪眼工厂，被当时看管工厂的「散兵」击败命悬一线，八重神子用神之心与其交换旅者的性命[90]。旅行者与八重神子一道，揭露了九条家与柊家与愚人众勾结的阴谋，气昏了天领奉行九条孝行，策反了九条裟罗。旅行者揭露了「女士」在稻妻的各种罪行，并在御前决斗中将其击败，「女士」最终被雷电将军斩杀。反抗军和万叶在社奉行的带领下[91]冲进天守阁，旅行者唤起万千人的心愿，击败雷电影。后者因被人民的愿望感动而不久后废除眼狩令[92]。
战后此前被收缴的神之眼悉数奉还。天领奉行九条孝行、勘定奉行柊慎介被捕入狱。散兵持雷神之心与愚人众失联，不知所踪。愚人众內奸包括内森据反抗军声称全被肃清。[93]九条镰治被鹰司家绑架，被雷电影营救后成为家主名代。九条家趋于没落。珊瑚宫与幕府议和，战事暂时平息，但双方各留一定分寸。愚人众执行官末席「公子」来到稻妻，调查散兵失联一案，遇到意外开启的诀箓阴阳寮。
