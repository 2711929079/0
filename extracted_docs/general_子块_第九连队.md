# 第九连队

原型取自苏联第40集团军第345独立近卫空降团第9连为掩护运输线安全，于1988年1月在阿富汗3234高地上成功防守400余名武装分子和巴基斯坦特种部队士兵混编部队的残酷血战（第9连39人有6人阵亡，28人受伤，战后所有人被授予/追授红旗勋章与红星勋章，另有两人追授苏联英雄，而对方损失200-300人），这一事迹后改编为俄罗斯电影《第九连》（9 рота）。
