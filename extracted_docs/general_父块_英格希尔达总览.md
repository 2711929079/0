# 英格希尔达总览

## 英格希尔达
坎瑞亚的「白鹄骑士」，绰号「幽谷之女」，是降诸魔山遗迹巨像的操作员之一，在对抗漆黑兽潮的战斗中失踪，被坎瑞亚以骑士规格安葬，实际上受到不死诅咒变成了黑蛇骑士，号称「失乡之剑」。

## 希尔德里克
坎瑞亚的「白鹄骑士」，绰号「胡子爸爸」，是降诸魔山遗迹巨像的操作员之一，因重伤导致多器官衰竭而亡。

## 海杜菈
坎瑞亚的「白鹄骑士」，绰号「金殿卫士」，是降诸魔山遗迹巨像的操作员之一，曾经背叛了骑士团，使得一台机器报废，骑士元帅安弗塔斯失去了左眼，后被处决，但考虑其是坎瑞亚最后幸存者之一，死后仍以骑士规格下葬。
