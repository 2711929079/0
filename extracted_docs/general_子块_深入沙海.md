# 深入沙海

你按照小吉祥草王给出的指引前往沙漠地区，并且与诸多伙伴结成了反对教令院的同盟。意外之下，你进入了赤王遗迹，得知了赤王与大慈树王之间某段不为人知的往事。
水落石出最终，你彻底厘清了教令院贤者们的阴谋——运用「虚空」的知识以及「神之心」，人为制造新的须弥之神。为了击垮贤者，拯救神明，你和伙伴们在阿如村集合，开始寻找一切可行的对策。
