# 薨陨

后诸魔神谋位，大乱遽起。虽诸仙人奋战力保，然战事杂乱，归离原深受混战之苦，尘神归终亦薨于其中。故岩王帝君迁其民入天衡之南，归离原自此荒废，不复往日之繁盛。[10]归终仙逝时，其躯体被五大夜叉封印。应歌尘浪市真君之求，摩拉克斯转涤尘铃于其保管。留云借风真君循归终之遗愿，迭进「归终机」，以此缅怀[4]。许多年，帝君都没能解开归终留下的这把尘世之锁（锁实为昆吾所造，原名昆吾落星，其中魔神的力量与记忆有如呼吸脉动若隐若现。稍加引导，便能释放其中伟力[11]）。随着岁月变迁，如今野生的琉璃百合也几乎不见踪迹了。[1]归终在魔神战争中仙逝后，歌尘浪市真君向岩王帝君请求将「涤尘铃」交给自己保管，获允。她不再爱结伴，转而独踞山顶，抚琴而思。在此过程中，她凑巧救下了枫丹人德沃沙克的祖先。不知过了多少年，她终于编出一首满意的曲调来，但似乎只弹过一次。其后，歌尘浪市真君自己也化作老妇外形，更名「萍姥姥」，封枪收铃，隐居尘世[4]。
