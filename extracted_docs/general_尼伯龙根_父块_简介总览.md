# 简介总览

## 简介
与提瓦特一同诞生的始源之龙，提瓦特龙族的共主，曾与七位龙王共同统治原初的提瓦特，拥有极其强大的力量。其在得知天外而来的“旅者”所告知的“湮灭之潮”后，铸造三月，代其施行提瓦特周流运作的权能后，只身前往世界之外探索。后取得深渊之力返回提瓦特并成为第二降临者，发现提瓦特被「天理」法涅斯霸占而对其发动葬火之战。对三月管理权被天理夺去十分愤怒，囚禁投靠天理的月神，将三月改造为对抗天理的天基武器。后被天理击败并抹除存在，部分龙族继承其遗志继续在天理的规则下搞小动作。
