# 平冠鹱

栖息在枫丹的大型海鸟。因其白色平直的羽冠，而得名平冠鹱。作为一种以捕鱼为生的鸟类，平冠鹱有着极佳的鱼群追捕能力。故而有文件说，熟练的渔民不会驱赶在船只附近捕食的平冠鹱，反而会选择与它们和谐共生。
