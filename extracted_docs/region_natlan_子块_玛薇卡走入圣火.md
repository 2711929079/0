# 玛薇卡走入圣火

玛薇卡带领松伽塔、万杰鲁、孟尼利克与图帕克展开最后一场对抗深渊的战役。在烬寂海之战最后一天，孟尼利克与深渊魔物同归于尽，松伽塔替万杰鲁挡下攻击负伤，最终两人双双牺牲。战胜深渊后，图帕克将负伤的玛薇卡背回圣火前，并因伤重力竭逝世。玛薇卡在发现图帕克没了气息后，按计划将自己的生命存入圣火之中。在圣火中，玛薇卡依然会受磨损的影响，险些忘记自己的使命与计划，幸好最终依然撑了过去。[117]之后玛薇卡的妹妹成为建筑师兼艺术家，支援部落振兴。其设计手稿和书签被保留在悬木人部族[10]。
