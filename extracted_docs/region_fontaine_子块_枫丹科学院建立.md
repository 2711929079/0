# 枫丹科学院建立

厄里那斯残骸之战让不世出的天才阿兰·吉约丹同时失去了仇敌与至亲，为了拯救枫丹免于危机，他回到枫丹廷，在伊黎耶岛的北方靠山依水建立了枫丹动能工程科学研究院，简称枫丹科学院。为了纪念玛丽安，阿兰·吉约丹修建了玛丽安纪念公园，但没有她的陪伴，阿兰变得愈发激进和不近人情，不再与任何人亲近。余生中的阿兰逐渐不再相信一切人的智性所不能完全掌握、不能彻底理解的东西，他构想出一个由机关与非元素能源所驱动的王国。从枫丹科学院退休后，吉约丹谢绝了一切来自沫芒宫与枫丹廷民间组织的邀请，前往苍晶区乡下隐居，自此与社会几乎完全隔离——尽管在此之前，这位性情孤僻的老人也从未与他人有过任何深交。如果我们接受娜莉·科克伦夫人的说法，那么就本质而言，阿兰·吉约丹是一个孤独的自闭者，总是怀着一种深深的不安全感，也正是因为如此，他总是对别人抱着疏离的态度，彷佛这样就可以用假装出的迟钝来掩饰精神的极度敏感。他不懂如何纾解自己的痛苦，也不懂如何将痛苦向其他人述说，只能将它们像珍珠一样藏在心底，用血肉去琢磨—当然，吉约丹的学生们不会接受这种说法，斥其为博人耳目的胡言乱语，只是在玷污恩师的名声。这便是枫丹科学院第三任院长洛杰尔·科特斯向《蒸汽鸟报》提出诉讼的首要原因[69]⋯最后他意识到水仙十字院终要有个人以正常人的身分或着，于是放下手头的研究，拿起童话书念给他的造物同时是他的好友的桑多涅，以普通人而非天才的样子离世。阿兰没有孩子，在与雷内决裂后也没有亲近的朋友，在他逝前，曾委托桑多涅销毁研究手稿，在此之前桑多涅曾把所有手稿看了遍[70]。在他人眼里研究手稿是被不明人士偷走。枫丹海露港等处的升降梯就是以阿兰的蓝图所造[71]。
桑多涅加入愚人众今天是葬礼，仪式期间下了很大的雨，真不知道天上有谁在哭，烦！雨天撑伞真的很麻烦！烦死了！阿兰下葬了，一切平稳。我出面送别了他，普隆尼亚看家。直到半夜天气才好一点。我坐在墓地陪了阿兰一下子，天上有月亮陪着我。夜里很安静。再也没有虚弱的呼吸声⋯这世上好像也只有月亮陪着我了。离开了。再见！更新日志：为普隆尼亚更新泡茶功能、送餐功能、增加查询书籍功能、修正普隆尼亚不会晾衣服的故障。对外宣称普隆尼亚是我的侍从，这样一来，人们就不会把好奇的目光投往它身上了。我得保护它，它是阿兰的秘密与愿望。至冬真⋯冷！虽然与我无关。找到新工作，待遇还不错。上级是一个很冷漠的女人⋯真的很冷漠。同事中什么样的人都有，受不了[68]。
