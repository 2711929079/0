# 名椎滩

稻妻古语称,「名椎」二字乃是「被神灵之手慈祥抚摸」的含义。但自千余年来,这片浅滩却见证了太多大战厮杀。名椎滩位于八酝岛与神无冢之间，是战事密集之处。浅滩之上，除了偶然落下的堇鹮之外，遍布着残旗、断箭等战争的痕迹，与破败船帆、碎裂甲板等废弃海船的残骸。因战乱的缘故，海贼与浪人常在此处游荡，甚至连罕见的血斛也妖冶盛开…
