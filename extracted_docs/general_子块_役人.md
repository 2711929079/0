# 役人

愚人众当中精锐的特务。自幼就被选中的人，经长久年月的教化与训练，在无数次的汰换后才能成为所谓的「役人」。她们是愚人众的阵列中最优秀的士兵，有着出色的身手与不渝的忠诚心。役人在愚人众中并没有常规的军种编制。依据命令与任务的内容，她们既可以与士兵同进退，也能独自或者与埋在各地的「钉子」一同，进行谍报、反侦察、护卫、抹杀等工作。为了那些更为特殊的任务，役人往往也需要掌握多种多样的技能乃至才艺。深境螺旋中的役人，名字是随机的。怪物：愚人众·风役人、愚人众·霜役人
