# 近年来到现在

莱因多特阿贝多莱因多特教授了阿贝多炼金术知识，但师徒二人在地底找到一件名为「纳贝里士之心」的圣遗物后莱茵多特失踪，临走前留下吩咐让阿贝多到蒙德找自己的老友艾莉丝。后来她吞噬生之执政纳贝留士的心[71]。阿贝多自此被艾莉丝收养，并借助她的关系进入西风骑士团，担任首席炼金术士兼调查小队队长。[72]莱因多特吞噬纳贝留士之心后成为生之执政，且与其余影子关系好。凯亚最后的希望为了“某个古老的阴谋”，深渊教团创始者克洛达尔·亚尔伯里奇的后人凯亚·亚尔伯里奇的父亲，[1]将其子抛弃在蒙德，让他成为坎瑞亚安插在蒙德的间谍。
“这是你的机会，你是我们最后的希望”
亲生父亲捏紧凯亚单薄的肩头，视线越过他，望向更远的地方。
地平线的另一头，是父子俩遥远的故乡坎瑞亚。

在三座塔都处理完了之后，渊上驱动深渊力量，使大日御舆内部出现了翻涌的黑暗。但渊上被随后赶来的旅行者击败。被打败后，渊上对旅行者说阿祇是由深海龙蜥进化而来的「龙蜥人」，并让旅行者引导光路去驱散大日御舆的黑暗。撤退前，他还把在大日御舆中找到的一块挂坠碎片交给了旅行者[82]。之后渊上将与旅行者交手的报告交给了深渊殿下[83]。调查烬寂海渊上伪装成稻妻人来纳塔调查烬寂海。他以帮助找回龙宝宝为条件，向人打听「回火」的消息，见到旅行者和派蒙路过而急忙离开。之后渊上和基尼奇达成了交易——由渊上提供能够吸引兽境猎犬的装置，帮助基尼奇打开兽境，而基尼奇允许渊上进行「历史研究」。渊上与阿乔二者私下又达成交易：由渊上夺取基尼奇的身体给阿乔，以此对阿乔进行研究。在「回火之夜」前一天，基尼奇通过装置成功召唤兽境后，渊上从兽境中出现。他本想夺取基尼奇的身体，但因为旅行者也在而作罢撤退[84]。之后渊上用计从茜特菈莉处打听到烬寂海的位置[85]。渡火者的解脱渊上调查烬寂海是为了寻找自己认定的家乡，发现烬寂海能前往时间线的每个角落后前往步过烬寂海[22]。传说一种鸟，生而孤独，能在火中高歌，人民将它作为图腾崇拜，君主以之为贵。火山之地的智者佩着它的羽毛，在烈火中退隐。孤独而来的他最终也孤独而去，下落从此不明。从此以后，人们时常听到烬寂海深处传来鸣唱。不知是浴火之鸟的鸣叫，还是渡火之贤者的吟叹呢。[86]渡火者的智慧流浪烬寂海的智者（渊上）渡过能前往任何时间点的烬寂海到达许多过去的文明，换过不同的身分，期间身在四百年前某文明时身分为学徒，用赤色玛瑙打造了一尊冠冕，用以适应烈火与高温。智慧与灼热执念成就了防火的冠冕，却引来了同僚和前辈的恐惧妒忌。「这自大的年轻人，竟敢挑战烬寂海的烈焰，这是百年来未有的亵渎」「火海必将他吞噬蚕食净尽，连灰烬也被热浪扬入高空，终化做虚无」善妒的老师企图刁难他的门徒，便要他戴上冠冕，步入火海。渊上见眼前之人这副德性，知道这里不是心目中的故乡于是离去。那位導師眼见这冠冕的主人在岩浆中悠然漫步，消失在了视野之外。[87]（仅为关于渊上的时间线解读之一，有其他解读）渡火者的决绝火红的花朵，仿佛闪光灼灼的玛瑙石一般。这浴火的花，被火中通行的智者佩在胸前。在渡火的智者（渊上）生命的最末，他对人们说道：「这是受烈焰焚烧而盛开的花。若我未被化作烟灰」「热浪与黑烟中必然会有这朵花闪烁着余火的光辉」后来，人们追踪着明灭的光芒，来到了烬寂海的边缘，却发现智者已经不知所踪，而花依旧盛放在余烬之中。[88]杜林悖理阿贝多将用炼金术把小杜林变成人类，从而能够令其身上纯粹的部分抵消杜林身上原始、邪恶的部分，替代即将复活的毒龙杜林在世界上降生，规避杜林复活的命运[71]。丝柯克斯克弗努格斯克弗努格，是苏尔特洛奇的朋友和追随者，将吞星之鲸的肉块缝进心脏后，获得了其权能，但力量过大使其丧失理智，纯靠「不死诅咒」维持身体。斯克弗努格的理智很快就会再次消失，他求丝柯克给他一个解脱，但丝柯克选择将其意志与力量全部打碎，而不死诅咒会重聚其身形，使其回到过去纯净的样子，只不过时间很长。丝柯克与过去的自己和解[27]。猎月人雷利尔归来「猎月人」雷利尔阴谋透过狂猎与月矩力集齐所有身躯，后被旅行者一行人扰乱，最后只能以十分之一的不完整力量归来，并成为挪德卡莱与齐聚该地的人们的共同敌人[32]。最后被大伙合力驱逐[40]。哥伦比娅进入月之门后，遇到了他，他已经彻底看开了，表示自己也是被「博士」利用的，并且请求对方向索琳蒂丝带个话，尔后帮助她清除了深渊制造的阻碍。
莱因多特阿贝多莱因多特教授了阿贝多炼金术知识，但师徒二人在地底找到一件名为「纳贝里士之心」的圣遗物后莱茵多特失踪，临走前留下吩咐让阿贝多到蒙德找自己的老友艾莉丝。后来她吞噬生之执政纳贝留士的心[71]。阿贝多自此被艾莉丝收养，并借助她的关系进入西风骑士团，担任首席炼金术士兼调查小队队长。[72]莱因多特吞噬纳贝留士之心后成为生之执政，且与其余影子关系好。
