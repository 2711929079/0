# 成神

雷穆利亚文明灭亡后，胎海的心脏厄歌莉娅获授了原初那一位的碎片，也拥有了魔神之格以及迟来的神圣使命（迟来的神圣使命:说明厄歌莉娅当水神的时间比较晚，同时间其他国度已经有了尘世执政），成为水神。厄歌莉娅弥合了诸族之间的纷争，围绕着奔涌的泉水，人们建立起新的城邦，开启了律法的统治，直至今日。[5]七神秩序建立后，枫丹地区由水神统治，信仰「正义」。被称作「甘露之主」的水神厄歌莉娅统治枫丹地区期间，修建了自然哲学学院和水仙十字院。
水仙十字院枫丹决定将孤儿与犯罪者子嗣教养院正式编入公共福利体制，基于其位址所处的前世文明遗迹，命名为水仙十字院。其院长由性情温顺的纯水精灵担任，执行副院长一名，由品德与执行能力突出的人担任。[27]此后水仙十字院向枫丹社会各界输送了许多高级人才。其中包括但不限于：海军司令贝瑟·埃尔顿、自然哲学学院院长德怀特•拉斯克、逐影猎人埃马纽艾尔·吉约丹、知名记者卡尔·英戈德[28]。
