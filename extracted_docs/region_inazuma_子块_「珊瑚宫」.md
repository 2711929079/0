# 「珊瑚宫」

海祇岛民信仰魔神奥罗巴斯，因为当年正是这位形似巨龙的蛇神担起了引导他们的责任。所以海祇岛的人们心怀感激，视奥罗巴斯为护佑他们的神灵。大蛇终为大御所殿下斩杀，薨于八酝岛。自此以后，珊瑚宫遣使降服，尊稻妻幕府为大宗主也。[17]此战以后，曚云神社便遭废弃，珊瑚宫家的海祇岛现人神巫女率领海祇之民建设了现在的珊瑚宫，作为对大蛇的纪念。
梦貘梦貘原本是璃月梦之魔神的眷属，后来梦之魔神败于魔神战争后逃至稻妻成为稻妻的妖怪[19]。
