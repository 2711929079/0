# 冰晶蝶

空气中飘荡不定的冰元素微粒聚合而成的纯粹元素生命，同蛺蝶一般悠游。闪光的冰元素生物,是元素上升凝合而成的产物，以飞散在霜雾中的冰元素维生。尽管无比脆弱，却能在风雪的簇拥下起舞。尽管本身无害，却昭示着霜雪降临，万物沉寂。你问为什么?大概是因为雪山太冷了吧。
