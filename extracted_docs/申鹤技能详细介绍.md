# 申鹤技能详细介绍

申鹤是《原神》中的冰元素五星角色，璃月的护佑者，称号"孤辰茕怀"。
元素战技「冰翎」：
- 释放冰元素能量，对周围敌人造成冰元素伤害并施加"冰元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「寒天冻地」：
- 释放强大的冰元素能量，对周围敌人造成冰元素伤害，并创造一个持续12秒的冰元素领域。
- 冷却时间：20秒。
- 能量消耗：80点。
固有天赋：
- "冰翎的祝福"：冰翎命中敌人后，有20%概率触发"冻结"效果。
- "冰元素增幅"：申鹤的普通攻击和重击造成暴击时，恢复3点元素能量。
- "寒天的守护"：元素爆发持续时间内，队伍中的角色冰元素伤害提升20%。
