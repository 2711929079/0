# 沙尔·芬德尼尔之城

彼时万物的气息渐弱，大地永冻[7]，蒙德被冰雪覆盖，生存条件极为恶劣。一位名叫法鲁希的男性带领族人来到了当时的雪山地区，出乎意料的是，这里并没有被风雪覆盖，而是保持着自然的苍绿葱翠，于是他们在此建立了王城。——沙尔·芬德尼尔（Sal Vindagnyr）[2]，成为第一文明在如今名为龙脊雪山的区域的分支。其遗迹最西延伸到石门。
公主山脊守望矗立在山脊的起点，如同界碑一般守望着南与北、山岳与水泽的远古城墟。传说中它的门扉只对心中燃烧着永不熄灭的烈火的人敞开，为他们祝圣戴冠[9]。相传古代的人会深入幽谷，寻找祭坛上的古树，将秘密说给树洞听，以卸下守密的沉重负担。忘却之峡中，充满了被遗忘的秘密[10]。当芬德尼尔的祭司之女诞生在这棵白树之下，接受祝福时，苍翠山岳的国境中充满了欢欣。沙尔·芬德尼尔的幸福一定会永远存续，正如贯通大地、永不凋败的银白之树。——为山中国度撰史的乌库，当时是这么想的。曾见证过无数人与事的记事者由衷相信，公主的美貌与才德将如月光般永远皎洁…[11]
