# 讨伐奥奇坎

希巴拉克走入圣火中，极度迷恋希巴拉克的奥奇坎因为所爱之神离去的痛苦、拥有人的灵魂却是龙的肉身带来的生理问题、不认为除希巴拉克外还有任何人配成为火之神的执念、还有自身的性格问题，陷入了暴虐和谵妄之中。谋主「半人」奥奇坎代执一切权柄的时代到来，他在几十年后成为了暴君建立了烬城(禁城)奥奇卡纳塔。因为拥有人类灵魂并迷恋人神希巴拉克的奥奇坎知晓盗火贤者的计划只是在利用人类，于是要求铲除剩下的龙表示抗议。已成为各部族领袖的众英杰中，只有阿霍布因父母皆为龙族所杀，对龙族深恶痛绝而响应了奥奇坎的号召，带领大量沃陆之邦战士到回声之子的领地，要求其胞妹伊葵交出「藏匿山间的凶兽」，伊葵不愿无辜的龙众因奥奇坎的无理命令受到牵连，因此与其兄争执。伊葵的手稿存世极少。包括锻火的技艺在内，几乎全部被她本人焚毁在了工坊里。她点燃了自己的工坊，随后逼迫兄长与她一同自尽，以免他在历史上留下与暴君奥奇坎一样的恶名[62]。点燃自己的工坊的目的也是为了避免自己的技术落入奥奇坎手中[65]。阿霍布和伊葵死后，奥奇坎假意宽恕龙众，让失去首领的沃陆之邦和回声之子暂时将权柄交给联盟，等到两大部族选出新的领袖后再归还旗帜，实则暗中记下包庇龙众之人的名字，待到合适的时机将他们暗杀。莉安歌的儿子支持奥奇坎的屠龙之计，又料定其母亲必然会反对奥奇坎的计划，因此向奥奇坎谏言，在莉安歌出行之后成功反叛之前对莉安歌先下手为强。奥奇坎在奥奇卡纳塔设宴邀请莉安歌前来，实则让几百个兵士一同围攻莉安歌，察觉到奥奇坎阴谋的莉安歌先是一箭射杀出卖自己的逆子，随后又与数量庞大的奥奇坎部下拚杀，最终寡不敌众，战至力竭的她将最后一箭射向了自己[70][67]。直到最后，莉安歌都将奥奇坎视为误入歧途的友人而非敌人[65]。曾经与莉安歌交好的邪龙再次开始作恶，试图夺回自己的土地。莉安歌的女儿设计将邪龙骗进一处洞穴，利用装满液态燃素的罐子点燃了邪龙，并将其射杀。圣王奥奇坎派出曾经是莉安歌女儿的青梅竹马的少年前去接收花羽会的旗帜，少年想与与莉安歌的女儿结婚，以此来接管花羽会。但是莉安歌的女儿没有顺从安排，而是将少年引入仓库，并点燃了提前布置好的陷阱，与少年同归于尽[73]。奥奇坎解决了最后的心腹大患后，奥奇坎不再畏忌昔日的情谊和约束，将崇信龙族的村落斥力叛徒，彻底开始了暴戾统治的时代。丁加·马古汉响应了号召，攻破了龙族设下迷雾结，并在这次行动中履行了少女曾经与自己的约定，用奥奇坎的金刃刺入少女的心脏，死去的少女化为了夜神之国新的大灵。丁加·马古汉故意放跑了龙族，事后向奥奇坎请罪，谎称龙族逃跑是因为自己年老体疫，于是奥奇坎准许了丁加·马古汉告老还乡的请求。在这之后，丁加·马古汉建立了名沟米克特兰（烟谜主）的部族[71]。得知龙城地下藏有古龙的庞大遗产后，奥奇坎立刻以兴修水利之名广抓壮丁，强迫他们「自愿」抽干地下遗迹的积水，不少工人在这次工程中丧命。联盟中只有花羽会坚决搏命反抗这个命令。随着排空积水工作的进行，地底下暗藏的深渊力量逐渐开始影响劳工们的健康，部分劳工开始出现幻觉、闻到臭味、感到不安，施工进度因此进展缓慢。为了鼓励劳工们尽快完成工作，监工申请了额外谷物配给，以及优化了轮班制度。奥奇坎认力劳工出现的问题是因力其对燃素铭刻的了解不足，而出现迷信行为。于是奥奇坎遣散了大部分劳工，只保留了一支23人的小队[74][75][76]。原本负责水利工程的匠人声称自己身体不适，退出了工程队，于是匠人曼科代替其加入[77]。负责水利工程的劳工因受深渊力量影响，情况越来越糟，一些劳工被指责在消极怠工，作为惩罚被切断了食物配给。小队原本的23人只剩下15人。奥奇坎依然认为情况可控。在包括受科在内的工匠的努力下，水利工程取得阶段性进展。但在抽水机启动后，深渊力量一夜之间吞噬了整个深井。了控制局势，奥奇坎借给了龙武士烈火，派其烧死了所有感染者，将现场称为受诅咒之地，并将其封锁。曼科在大火中逃到了囚禁第九席的雕像下，一滴水意外滴入他的眼睛，曼科得到统律之心的馈赠，看见龙族的许多智慧，以及一条能逃出生天的道路，充满冲击性的画面让曼科拼命逃向那条路，最终行以幸存。水利工程事件后，民间开始出现恐慌情绪，有人发疯似的大喊联盟将要灭亡，有人认为深渊力量的出现是奥奇坎禁止人们向夜神之国献祭而招致的惩罚。还有人认为龙族是造成这一切的罪魁祸首，于是一些人开始袭击关押龙众的夜岛，但都被密力奇赶跑。曼科等到自己的烧伤完全愈合后，逃离了奥奇卡纳塔[75][77][78][74]。在奥奇坎纳塔的文本中，曼科的故乡被抹去作“（…）”，而在之前联盟告示中的“（…）”部族已被证实是特诺奇兹托克，可猜测曼科也是特诺奇茲托克人。奥奇坎见到了龙族旧日的统律核心·L型自律监管机阵统合处理单元「伊·奎库叙·茨博隆·库·雷尔（伊蕾尔）」（CL-09[79]），伊蕾尔称呼奥奇坎为命定之王，向他展示了龙族的历史，以及龙族所具备的各种尖端科技，希望能够规劝奥奇坎回归龙众，但奥奇坎在看过龙族过往具备的强大力量后，反而加深了认为龙族危险，无法与龙族和解的想法，因此利用焰主血脉的权能剥夺了伊蕾尔的权柄，篡夺了秘源技术的控制权，让伊蕾尔在不死去的情况下为秘源技术持续供给能量[80]。被囚禁在雕像内的第九席依然不愿意力奧奇坎效力，于是奥奇坎决定建立只属于人的统律之心。奥奇坎计划将自己作力龙的部分变力奴隶，将其囚禁在制造出来的统律之心内，控制城内的秘源机关[76][4]。第九席的不服从让奥奇坎对龙族彻底失望，于是他下令囚禁所有的龙族，并将他们送往名叫 “夜岛”的地方集中管理。管理人名叫密力奇，对待龙族很友善[76][81][82][83]。奥奇坎坚信希巴拉克才是唯一的人类之神，认为过多的统治者会导致国家像北海的蛮族和砂中的列邦一样走向内斗分裂，纳塔只能有一位王，一位神。奥奇坎建造了天蛇船，在天蛇船上留下铭文要求世人不要信仰龙与夜神，字里行间无不在赞颂希巴拉克的伟大[76]。深渊的外溢终究难以抑制，为了不让深渊波及到由希巴拉克一手建立的国家，奥奇坎将被深渊侵蚀的奥奇卡纳塔完全封锁，又将通牒金牌藏在天蛇船上使瓦萨克拉胡巴肯无法取得，最后自己吞下深渊的源头，将意识储存在了统律核心之中。临终前，奥奇坎在天蛇船上留下最后的铭文，斥责瓦萨克拉胡巴肯的残忍和天空的无情，认为唯一可行的只有希巴拉克的路，瓦萨克拉胡巴肯前来找他取得金盘，得知金盘被藏在天蛇船上后怒不可遏。盗火贤者与奥奇坎见面，说他注定失败，要他交出通牒晶盘，但被拒绝[69]。盗火贤者找到了一位来自烟谜主，曾经探索古龙帝国，而后被奥奇坎迫害的工匠曼科，唆使他反对奥奇坎。此时，赤瞳的少年因为包庇龙族而被逐出花羽会，受一位老妇人的启发站出来，他历经试炼，最终模仿希巴拉克用黄金匕首刺穿了自己的心脏，其灵魂进入了夜神之国，并且获得了夜神之国的加持，再次复活归来，成为新任火神[84]。赤瞳少年归来讨伐奥奇坎，回声之子的代行人，矿山的女儿萨库科(他父亲因包庇龙取走了性命，爱慕赤瞳少年)、孪生兄弟（能说会道的哥哥阿塔瓦尔帕，与曾被奥奇坎奴役的弟弟瓦斯卡）、此时沉默寡言，原本对反抗奥奇坎一事已经心灰意冷，接下大联盟军士官闲职的的尤潘基（回火古名的开创者），和曼科加入队伍[85]，他们打败了奥奇坎。战争前尤潘基从禁城偷走回火，在战争中将回火扔向奥奇坎的士兵，击败大量敌人[86]。最后他们的结局，曼科死于怪物的屠刀之下、赤瞳少年效仿初代火神进入圣火中、尤潘基在逃出禁城时听到奥奇坎的呼唤回头时被回火烧死[86]、阿塔瓦尔帕亲眼目睹弟弟瓦斯卡死于敌手哭到失声、阿塔瓦尔帕和萨库科从战争幸存下来(许多年后两人先后逝去)。暴君奥奇坎在这场战役的最后于赤瞳少年面前摘下面具，口中喷出金色的火焰，将自己烧成了熔化的金属，坠入了深渊的漆黑泥沼之内。实际上是由于深渊的入侵，奥奇坎分离了自己的意识，而肉体和灵魂则化作了紫色巨龙。之后此龙时常盘旋于奥奇卡纳塔遗址，袭击任何经过此地的人[69]。奥奇坎暴政结束后，各部落取回自己的权柄，纳塔开始稳定的发展，直到五百年前的坎瑞亚灾变毁于一旦。[87]名为紫英的植株原本应该拥有更为深邃亮丽的色彩，只不过，这种曾一度受到烬城居民喜爱的花朵，在整座烬城都被侵蚀时，也难以逃离被幽影沾染、以至于最后被神的怒焰焚毁的结局[88]。曼科曾与萨库科相约为她打造羽饰，而萨库科也许诺送他松石宝冠回礼，但曼科在大战中牺牲；赤瞳少年不久就回归圣火 ，并嘱咐萨库科在圣火中抹去他的名字。萨库科自曼科牺牲后，一直在收集曼科打造的首饰，不管是否是赝品。可是就连阿帕瓦尔帕也先她离开了，萨库科在这个熟悉的时代里，跌跌撞撞地活着像个陌生人。“在没有你们在的时代，实在是太无趣一些”有一天，萨库科带着刻着曼科名字的饰品，走入夜幕中，再也没有回来……由于切实存在的暴戾行径，以及对深渊真相的部分隐瞒，奥奇坎被后世的纳塔人视为暴君，饱受唾骂。为了取得天蛇船上的通牒金牌，瓦萨克拉胡巴肯利用死去人类的灵魂化作不同人的样貌散布奥奇卡纳塔有秘宝「回还之玉」的消息，吸引部族的勇者进入奥奇卡纳塔的遗迹内冒险，结果因邪龙的袭击无一生还，包括五百年前的蒂图、驯兽师雷格巴、一名巡山客、科潘、玻娜之父泰卡、秘术师莉莉乌。[69][89]
暝视龙诞生朽败的烬城与僭主谵妄的狂梦一同坠入日轮般耀眼的炽炎，赤瞳的英杰唤来神的怒火，将那黑灾侵蚀的圣座焚作飞烟。自屠龙者马古汉将旧日的主宰逐出深谷，已经过去数十年，昔时崇信群龙的村落中，新生的世代早已遗忘与龙的誓言。正因如此，当年轻的祭司瓦加杜遵循屠龙者马古汉的遗愿，将他葬入无名的坟茔、带着他的弓踏入瘴雾缭绕的荒山时，全然不知老者在临终的时刻，对自己述说的话语是何意义。何谓「再立盟约」，何谓「重建和平」，瓦加杜全无头绪，只当是年老的祭司要让他将生前珍藏的弓归于凶险的夜域，安抚那些因古老的献祭仪式被荒置多年而大发烈怒的魂灵。拨开谜烟的帷幔，踏上三叠石阶，在幽域边缘，浮现在他面前的，接踵而来的幻象。恍惚间，无数相异的形貌宛如虹彩般流转蹁跹，瓦加杜慌忙跪倒在地，乞求夜域的诸圣饶恕他无意的冒渎，怎料雾气散尽的时刻，眼前并非他想象中的死者，而是翼展如天虹垂落般的魁伟巨龙。「命理锚定于星象的无魂者，汝不必为我的形貌心生怯惧，」「汝的来意我早已知晓，纵然汝自身仍为愚懵的迷雾遮蔽。」「那自称屠龙者的祭司差遣汝来此，是为立定后世的盟约，」「古民的贵血从未沾染他的刀刃，此般诚意足抵千言万语。」「无妨。我等的王上早已殒殁，我亦倦于养护梦中的同族，」「我愿将我知晓的秘仪悉数授予汝等，令汝等的部族繁盛，」「不必向堕落于幽世的使者献上血祭，亦可聆见夜的低语。」「为此，我只会向汝等索求两件事，作为盟约立定的条件：」「其一，要让汝等的后嗣，如良友般善待我曾庇护的古民，」「时刻将它们的劝诫聆听，莫让誓言沦为大联盟般的笑柄。」「其二，要让汝等的后嗣，将我的名讳从一切织卷中剜除，」「免得那自诩贤者的蠢物，对我等的盟约妄加可悲的报复。」「汝无需思索，应允便是。庇护汝的大灵亦曾是我的奴仆，」「早在汝尚未降生的时日，她便向我述说过这僭越的盟约。」恍若初醒之人无从分辨梦与真实，懵懂的瓦加杜颔首应允，顷刻间，鎏金的殿堂与魁伟的巨龙皆如谜烟般散尽。正疑惑于方才目睹的异象是否只是因一时疲惫产生的幻梦，身侧柔软如绒羽、清冷如夜风的触感却让青年不由得回头————那是自夜神之国走出的，第一只真正的暝视龙[90][91]。
