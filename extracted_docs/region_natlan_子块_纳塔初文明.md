# 纳塔初文明

在「原初的那一位」法涅斯降临之前，纳塔由火龙王统治。当时，元素龙们的技术水平已经非常高，他们利用分离自「源火」的「燃素」驱动了无数先进机械，这些机械至今仍遗留在纳塔。在穷尽了对提瓦特的探索后，他们把目光投向了提瓦特之外的星海[6]。但法涅斯的降临使得一切美梦化为了泡影。天空岛降临提瓦特后，击败了火龙王在内的七龙王。火龙修库特尔曾被天理冰封在熔岩之中陷入假死[7]。打败龙族后，法涅斯并没有对龙族赶尽杀绝，它派天空岛的使者借助银白古树的力量，在纳塔的土地上布下了一个半圆屏障，并在其庇护之下，引领人类创造了远古统一的文明。在银白古树的屏障之外，龙族们和火龙王在地底国度苟且偷生[8]。创世之战后修库特尔和纳塔龙众躲地底了，所以天理收不回火龙大权（推测）。
纳塔兰提亚时期诸部族的子嗣尚且不明晓如何用织卷记述过往的时日，古老而高傲的贵胄曾在这里（火山）筑起恢宏的圣都，脾睨那些渺小赢弱的奴仆们称作「纳塔兰提亚」的国度。自诩强大的广袤帝国在战火中殒落，往日神圣威赫的都城，如今也早已沦作幽闭的藏骸所。[9]
