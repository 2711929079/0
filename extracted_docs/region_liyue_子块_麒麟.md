# 麒麟

三千年前，甘雨回应岩之神摩拉克斯的召唤，成为他在魔神战争中的助力。麒麟是仙兽中的仁兽，饮必甘露，食必嘉禾。不履生虫，不折生草，不群居，不旅行，不入阱陷，悠闲则循循入也，动则有容仪，是温柔而优雅的一族。而当海中巨兽为非作恶、足下的土地遭遇危机时，和平便不再成为生活的选项。[61]
