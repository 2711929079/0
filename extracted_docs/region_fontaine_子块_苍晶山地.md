# 苍晶山地

与枫丹廷隔水相望的山原，旧日的遗迹散落其间。据说在古老的年岁里，厄歌莉娅的信徒们曾将这里视为欢游与郊筵之地，咏颂如苍晶般澄澈的爱与梦景。如今，他们的歌声早已散入微茫的月明，唯有清冷的风依然像往昔那般吹拂过薄荷丛生的高地。
