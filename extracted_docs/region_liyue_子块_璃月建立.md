# 璃月建立

名为摩拉克斯的魔神与仙人签订契约并和众人在如今被称为归离原的地方建立起了名为「璃月」的城市。——「契约既成，食言者当受食岩之罚。」
契约岩之君曾施展神迹，仿造昆吾全盛时期的才华[39]从无杂质的金色石珀中削出长刀一柄，挥剑斫去山峰的一角，以此向子民立下无上庄严的契约——离散的人，必将聚拢回归；背约的人，必然加以惩治。失去挚爱者、痛失珍宝者、蒙受不公者，将得到补偿。[40]据说当年玥珠曾是岩王爷赐给璃月人的第一颗神珠[41]。在岩王爷的指引下，璃月先民治炼矿石，用最初的货物与世间诸国展开贸易[42]。
