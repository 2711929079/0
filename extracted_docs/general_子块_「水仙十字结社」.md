# 「水仙十字结社」

雷内·德·佩特莉可与雅各布为“白淞之围”中犯罪者的子嗣，被送入水仙十字院养育，与阿兰、玛丽安为童年玩伴。坎瑞亚灾变前被副院长托付给卡尔·英戈德收养，随养父四处冒险。与养父在须弥的甘露花海冒险时，研究了灵光与深渊力量的性质，雷内发现甘露花海圣莲的构成与自己和雅各布有一定相似之处，但与养父不同。雷内想要进一步采样研究，但被那伽朱那团拒绝。探索途中，雅各布接触深渊力量陷入虚弱和昏迷，雷内研究后决定让雅各布继续吸收，雅各布恢复，但已受深渊力量改造变得无需进食[17]。雷内通过坎瑞亚遗留资料推导出了「世界式」，得知世界会在第二次涨水期毁灭。之后返回枫丹探索，阅读前文明的书籍时得知前人把关于文明灭亡与新生的算式称为「法图纳」，但「世界式」所推演的结果是文明毁灭后不会再有新的文明诞生，除非考虑引入系统之外的「变量」。此后雷内又在枫丹发现了前文明的「金色剧团」遗留的魔法术式「密合之约印」。二人溜进厄里那斯残骸体内调查，雷内认为其中的力量可以增强雅各布，但害怕伤害到雅各布而不忍心让其吸收，雅各布自主吸收了厄里那斯的血液，仅有呕吐的轻微反应。继续进食厄里那斯血肉增强雅各布力量后，二人尝试开启了「通道」，出现了造型与狗狗有相似之处的怪兽。雷内假定「通道」连接了某个充满了此类怪兽的地方，并将其称为「兽境」，将出现的这类怪兽命名为「兽境猎犬」。于此同时，已经加入自然哲学院的阿兰在养父埃玛纽艾尔的监督下，和玛丽安一起在「发生过战斗的地方」寻找灵感，四人意外重逢。[18]雷内没有提雅各布的事情，因为他不信任曾参与过白淞之围的埃玛纽艾尔。[19]经由阿兰向学院长推荐，雷内与雅各布来到自然哲学学院，加入阿兰的实验室。雷内向阿兰说明了先前的末日预言，希望能改变研究方向，但「世界式」与阿兰的计算结果不符，阿兰不相信雷内和雅各布看到过的东西。[20]雷内和雅各布只得拉拢学院的其他人，并整合他们的力量与资源，使用揭示之书向别人展示自己所见的末日图景。与阿兰兄妹和卡特在佩特莉可田野调查时，雷内发现遗迹与「金色剧团」有关，并找到了一张残破的地图。得知了卡特的病情，雷内意识到不能继续与阿兰沉迷机械研究，和雅各布向学院长要了新的实验室，并私下进行深渊力量的研究，期间研究并命名了「深赤之石」，研究了层岩巨渊的黑泥。
雷内为了提供手下组织和归属感，用「水仙十字院」命名「学会」为「水仙十字结社」以委托信徒办事。结社成员将雷内称为「大师」。将雅各布称为“英戈德先生”。[11][21]雅各布提议让卡特也接触深渊力量，二人向卡特展示了雅各布的变化，说明了其中风险，卡特答应了，但融合没能像雅各布那样顺利，卡特持续劣化，成为了某种难以言说的东西。…暂时保存下来。阻止进一步的组织崩溃。雅各布现在也不会看到他就流眼泪了…但是偶然还会发出卡特的声音。这样的时候才最糟糕。尝试过把大概是发声器官的地方先切除了，但会在一日到三日内再生…雷内与阿兰因卡特之事决裂，雷内下定决心一定要找到能够将卡特救回来的方法。与雅各布重返佩特莉可以及之前的遗迹考察，找到了有关原始胎海之水的资料，发现古人曾经成功“回归起源”，雷内由此找到了借助深渊力量以外第二条应对末日的道路。雷内找到一处胎海水的决口，决定在上方建设「自体自身之塔」（又称“格式塔”）。雅各布指示结社成员主动向沫芒宫报备建设工程，利用结社内的贵族成员影响力作掩饰，并让逐影猎人负责安保工作，打消枫丹廷的疑虑。二人回到十字院旧址看望院长，雅各布将副院长的墨绿羽饰留在了此处[14]，雷内发现十字院旧址很可能是古代黄金剧团的设施。雷内研究了古代术式中「四象限圆环」和「流涌之树」等内容，考察了多个封印点和四要素的位置，并对四象限进行了改造，取得能够割舍人格的「水仙十字圣剑」，为最终的术式做准备。雷内重返十字院，将怀表（「宝物」）留在此处，并说服院长前往格式塔，利用其能溶解和吸收意识的能力。在最后的仪式中，雷内割舍了自己的人格，并将自己溶解，计划“借由水再次诞生”。然而计划失败了，雅各布保存了雷内的意识。在此之后，「大师」又吸收了三名追查结社的逐影庭探员（其中第一位是埃马纽艾尔·吉约丹[7][2]，第二位是玛尔菲莎[9]），从他们的记忆中发现枫丹廷官方已经对结社的存在，并得知雷内疑似已经研究出可以移植意识的机关。加入逐影庭的玛丽安带着阿兰所研发的原型机西摩尔[22][23]秘密追查水仙十字结社，与雅各布等社员在厄里那斯体内展开了大战，迸发的碎片零件等划伤了厄里那斯的体内，诞生了美露莘这一种族，其与生俱来的“信物”便是当时大战的残留物。最终雅各布存活，继续践行深渊力量救世的道路；玛丽安重伤，意识被莉莉丝溶解融合。随后，同化的「玛丽安」回到「安眠处」住下，捏造出对应着曾经伙伴们的水形幻灵，演绎着童话。[24][25]厄里那斯残骸之战让不世出的天才阿兰·吉约丹同时失去了仇敌与至亲，为了拯救枫丹免于危机，他回到枫丹廷，在伊黎耶岛的北方靠山依水建立了枫丹动能工程科学研究院，简称枫丹科学院。为了纪念玛丽安，阿兰·吉约丹修建了玛丽安纪念公园，但没有她的陪伴，阿兰变得愈发激进和不近人情，不再与任何人亲近。余生中的阿兰逐渐不再相信一切人的智性所不能完全掌握、不能彻底理解的东西，他构想出一个由机关与非元素能源所驱动的王国。从枫丹科学院退休后，吉约丹谢绝了一切来自沫芒宫与枫丹廷民间组织的邀请，前往苍晶区乡下隐居，自此与社会几乎完全隔离——尽管在此之前，这位性情孤僻的老人也从未与他人有过任何深交。如果我们接受娜莉·科克伦夫人的说法，那么就本质而言，阿兰·吉约丹是一个孤独的自闭者，总是怀着一种深深的不安全感，也正是因为如此，他总是对别人抱着疏离的态度，彷佛这样就可以用假装出的迟钝来掩饰精神的极度敏感。他不懂如何纾解自己的痛苦，也不懂如何将痛苦向其他人述说，只能将它们像珍珠一样藏在心底，用血肉去琢磨—当然，吉约丹的学生们不会接受这种说法，斥其为博人耳目的胡言乱语，只是在玷污恩师的名声。这便是枫丹科学院第三任院长洛杰尔·科特斯向《蒸汽鸟报》提出诉讼的首要原因[26]⋯最后他意识到水仙十字院终要有个人以正常人的身份活着，于是放下手头的研究，拿起童话书念给他的造物同时是他的好友的桑多涅，以普通人而非天才的样子离世。阿兰没有孩子，在与雷内决裂后也没有亲近的朋友，在他逝前，曾委托桑多涅销毁研究手稿，在此之前桑多涅曾把所有手稿看了遍[27]。在他人眼里研究手稿是被不明人士偷走。枫丹海露港等处的升降梯就是以阿兰的蓝图所造[28]。
