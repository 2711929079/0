# 基捷城建立

基捷——全至冬的第一座城市。在莫诺马赫·雪奈茨尼伊皇帝的领导下，从第四百一十七纪的第四年，也就是建城的那一年，一直到迁都至冬堡，这座仿照妖灵先祖的习俗修筑的帝都，都是统管冻土的权力中心。百年的时间里，由莫诺马赫皇帝领导、以基捷城为中心的同盟（苍星王廷），在经济上和政体架构上迅速发展，呈现出一种全新的面貌[8][25]。
第428纪（约3090年前）白沙皇统一各氏族第四百二十八纪的第三年，即莫诺马赫·雪奈茨尼伊皇帝在冻土上修筑起帝都基捷的第七十六年，皇帝派使者去风仙的领地，制止风仙们向穆斯季斯莱沃王纳贡。第四百二十八纪的第六年，穆斯季斯莱沃王谴责苍星的王廷破坏了先前议定的和平协议，要求皇帝向兽怪联盟缴纳贡物以维护和平。皇帝拒绝了他们的最后通牒，双方随后做好了战争准备。第四百二十八纪的第七年，穆斯季斯莱沃王发兵攻打基捷，率领着穆斯季斯莱沃洛氏族、尼约默罗德氏族、达洛格莱德氏族、切纳雷卡氏族，以及数目众多的仆从氏族。他们的军队在基捷山麓、冰湖与平原总共交战三次，皇帝杀死了穆斯季斯莱沃王以及跟随他的诸多王公，瑟里维克大公归顺。出于慈悲与仁爱，皇帝寻回了他们的尸体，把他们送回各自的氏族领地安葬。皇帝说：「他们不是王，不是诞生自神圣的血脉，不是应当为王的，而我是苍星的继承者，是全体妖灵的庇护主。」余下的妖灵氏族都来觐见皇帝，这样，他就拥有了统治他们的权力，并从此自称为白沙皇。据传，在最后一场战役中，穆斯季斯莱沃王出于绝望，尝试利用了阴影世界的力量以拖延苍星王廷的进军。尽管这并未能扭转战局，但依然对皇帝日后的决策产生了一定影响[8]。
