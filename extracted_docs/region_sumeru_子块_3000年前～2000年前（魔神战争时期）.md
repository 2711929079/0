# 3000年前～2000年前（魔神战争时期）

比夫茏魔神战争时，魔神「蕈之贤主」比夫茏通过为人类提供蘑菇的方式，成为了人类的信仰，它以力量创造分身，赐予人类无穷食粮。但如果蘑菇吃得太多，会逐渐能听到比夫茏的声音，它也借此对人类施加精神控制。但人类在发展的过程中，比夫茏的分身被越吃越少，它的力量越来越弱，甚至一度无法维持实体。而分身之一的赞玛兰也切断和比夫茏的精神联系。比夫茏想要恢复力量，打赢魔神战争，向人类复仇。但须弥人都已知道它创造出的蘑菇分身十分美味，分身已不可能再在须弥生长起来了。于是比夫茏控制一信仰者，来到纳塔的火山之下，却遭遇火山爆发，被深埋地底[52]。
