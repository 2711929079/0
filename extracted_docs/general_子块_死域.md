# 死域

死域（无留陀）有些区域会出现的异变现象，已经存在了上千年，是世界树的疾病的体现…。不仅会使植物枯萎，也会夺走动物和人类的生命。只有神之眼——也就是能驾驭元素力的人，才能在一定时间内抵御死域的侵蚀。死域的存在似乎会对魔鳞病患者造成影响。死域是被禁忌知识污染的元素生命死亡形成。
