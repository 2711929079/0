# 沈秋拾剑录

行秋将自己的故事为素材，以“枕王”作为笔名撰写了《沉秋拾剑录》，但在璃月无人爱看。后来一位稻妻的商人将这本书带到了稻妻的八重堂，结果广受好评[198]。
海山北斗曾多次挑战名力“海山”的冥海巨兽，但都以失败告终。后来北斗再一次率领船队挑战海山，战斗持续了整整四天，最终北斗给予了海山终结一击，将其头颅彻底砍断，宣告胜利[199]。
