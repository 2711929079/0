# 御伽大王

在那个时代有一个自称「御伽金刚狮子大王」的怪人，他拿着爱刀「赤角石见金石溃金涂金啮御狮子杵」在鸣神岛四处作乱─也不能说作乱，只是自称可以打倒各种怪力乱神的妖物。御伽大王确实有气力，七个人无法把他推出土俵。小孩想采堇瓜，一脚把堇瓜树踢倒。也曾带着小孩直接闯入官家赏红叶宴会中，「御伽金刚狮子大王来退治恶鬼了!!!」如此大喊着，最后被一个娇小的鬼人(御舆千代）丢出去。「之前对月痛饮，患了风寒而已!」他这样狡辩着，随后大笑了起来。大王并没有背负将军旗帜的资格，真不知是哪来的自信…「总有一天我会把她的犄角拆下带回来，就尽管跟着狐狸的使者躲起来吧，等老子回来再陪你们摔角!」他嚷嚷着，小孩们只想得到那娇小鬼人把带着怒吼冲过来的大王，轻哼一声便把大王高高抛出去的场景…[23]（御伽众是日本战国时代至江户时代初期的闲职。通常都是由僧侣、没落失势的大名、以及出家或退休的武士担任，雇主若无闲暇时则几乎一整天都没事做，以日本职场文化而言是会被同僚看不起的低贱职位，仅有极少数的御伽众能因自身才学或特别受喜爱而晋身为雇主的核心幕僚，其余多被当附庸风雅杀时间的"消耗品"而已。）妖怪「绪方四鬼」曾被五百藏与御伽大王封印进画里。这妖怪有自创刀术「绪方四鬼抄」[24]。
酒会在那个酒会上，狐斋宫还认识了新任天狗族长灵善坊，和面容皎美的鬼族少女御舆千代。也是在这次酒会上，一个自称御伽金刚狮子大王的壮汉，提着全名為「赤角石见金石溃金涂金啮御狮子杵」的刀闯入会场，自信满满得说要除妖灭鬼。御舆千代也在兴头上，便同他摔跤。即使千代在他面前显得十分娇小，最后这位大王还是摔得人仰马翻，害羞得溜了出去。[23]会场上洋溢着欢乐的气氛。酒会中，狐斋宫让妖狸头子五百藏偷雷电将军御苑里的鸣草，自己陪在影将军身边。五百藏被逮住后，狐斋宫一顿求情，不仅免了五百藏的罪，还赐予他保生司正一职，守护镇守之森。酒会上，雷电将军请御舆千代献上一舞，狐斋宫与其一同舞蹈。结束后，天狗族长又邀请狐斋宫比比谁跑的更快，结果是打成平手。狐斋宫自己也清楚，天狗族长是让着自己。酒会结束后，狐斋宫回到鸣神大社，面见了浅濑响。狐斋宫把自己能教的东西都授予浅濑响后，便将其引荐给天狗族长。
