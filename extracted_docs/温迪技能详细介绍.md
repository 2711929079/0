# 温迪技能详细介绍

温迪是《原神》中的风元素五星角色，蒙德的风神，称号"风色诗人"。

元素战技「高天之歌」：
- 点按：释放风元素能量，对前方敌人造成风元素伤害。
- 长按：创造一个风场，可以将角色和敌人吸附到中心，造成持续的风元素伤害。
- 冷却时间：点按6秒，长按12秒。
- 能量消耗：点按3点，长按5点。

元素爆发「风神之诗」：
- 效果：创造一个巨大的龙卷风风场，将敌人卷入并持续造成风元素伤害。
- 染色机制：风场可以吸收水火冰雷四种元素，产生相应的元素伤害。
- 持续时间：8秒。
- 冷却时间：15秒。
- 能量需求：60点。

天赋技能：
1. 顺风而行：队伍中所有角色获得10%风元素伤害加成。
2. 风中的歌：使用元素战技或元素爆发时，有20%概率重置冷却时间。

角色定位：
- 主要定位：聚怪辅助
- 次要定位：副C输出
- 核心优势：强大的聚怪能力、元素染色、减抗效果

培养建议：
- 武器推荐：天空之翼、终末嗟叹之诗、祭礼弓
- 圣遗物推荐：翠绿之影四件套（风元素伤害加成和减抗）
