# 第二王朝

呼玛云为国王，其妻子为第一王朝“兰沙赫尔王”之女，几年后死于镇灵叛乱，之后三十多个藩王叛乱，叛乱被加尔沙斯普（“圆形剧场”城主）平定。
第三王朝加尔沙斯普平定叛乱后称王，几年后去世，都城大火七日不绝而毁灭，民众迁往蓝宝石城。
