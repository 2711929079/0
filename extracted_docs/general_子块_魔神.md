# 魔神

原初之人法涅斯与龙王尼伯龙根战斗时碎裂的身体碎片，依附到元素生命/魔兽身上使其成为魔神。魔神已经透过魔神战争确立出象征七种元素的魔神，即尘世七执政。魔神是“不灭”的，准确来说，所有强大的生灵，自诞生起便在影响这个世界，即便死后影响也不会完全消失，魔神残渣便是其不灭的一种形式。众神皆背负爱人的责任。
