# 隙境原体

隙境原体是巨兽厄里纳斯死后其细胞演变成的魔物。据与「护国的白骑士」同行的逐影猎人的记述，在厄里那斯的惨烈战斗结束后，曾经出现了大量的隙境原体。这些隙境原体并没有对他们采取任何攻击行为。[54]
