# 第一文明

曾经有一个时代，大地上的人们还能直接听到来自天空岛的启示。神的使者走在蒙昧的人当中。彼时万物的气息渐弱，大地永冻[7]。在天使的带领下，人们创造了统一文明，即「第一文明」。第一文明大部分都是石制砖头建筑，上面有凯尔特三角的标志，风格明显，很好辨认。蒙德的第一文明建筑中，望风山地的看似零散，实际上构成了一座建筑群。南部的遗迹群一路向高地延伸，遗迹的尽头是一处秘境——仲夏庭院，古代文明建造的夏宫灵囿，是一座用于休闲观赏的庭院建筑，不难想象它曾经的繁荣景象。过去的人们向往天空，于是将城塞建立在高峰之上[8]。（那些坐落在山峰上的古老的遗迹，毫无疑问连同山峰一起在风神修改地形时随风而去了，如今只留下少量的残存）风啸山坡的建筑群在达达乌帕谷谷内，地图上山体凹陷处，较能阻挡当时的风雪。苍风高地存有一大片遗迹，由于遗迹大多分布在大山丘上。苍风高地七天神像所在的山丘无论从遗迹数量上看，还是从位置上看，都很像是古国的核心区域，因此推测这里是主城。在周围还分布着一些城镇和祭坛，有些依旧保存着原貌。塞西莉亚园圃的位置相对偏僻，远离遗迹群中心，道路也十分曲折。尽管如此，一路上都分布着与山丘上古国相同的遗迹，这说明它确实属于这一古国。秘境介绍中提到，这里曾是培育塞西莉亚花的温室，如此看来位置偏僻一点也不奇怪。蒙德古国会兴建能产生风场的机关。蒙德曾存在与古树并存的古国。古国的先人们兴建了塞西莉亚花的苗圃。在今苍风高地建立了塞西莉亚苗圃，这个苗圃是种植塞西莉亚花的温室，该地也具备祭祀职能。位于今摘星崖和千风神殿地带的文明建造了祝圣祭坛夏宫灵囿。
雪山沙尔·芬德尼尔之城彼时万物的气息渐弱，大地永冻[7]，蒙德被冰雪覆盖，生存条件极为恶劣。一位名叫法鲁希的男性带领族人来到了当时的雪山地区，出乎意料的是，这里并没有被风雪覆盖，而是保持着自然的苍绿葱翠，于是他们在此建立了王城。——沙尔·芬德尼尔（Sal Vindagnyr）[2]，成为第一文明在如今名为龙脊雪山的区域的分支。其遗迹最西延伸到石门。
