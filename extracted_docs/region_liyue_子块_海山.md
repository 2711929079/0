# 海山

北斗曾多次挑战名力“海山”的冥海巨兽，但都以失败告终。后来北斗再一次率领船队挑战海山，战斗持续了整整四天，最终北斗给予了海山终结一击，将其头颅彻底砍断，宣告胜利[199]。
烟绯与优菈一艘装载着深渊教团危险品的商船要从荆夫港运到璃月港。烟绯作为跟随商船的顾问发现货物有鬼，就擅自进行了调查，结果引火上身，还好优菈等人及时赶到，救下了烟绯，并在其帮助下将深渊教团成员一网打尽[200][201][202]。
