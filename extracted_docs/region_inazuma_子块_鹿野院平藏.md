# 鹿野院平藏

幼时的鹿野院平藏在一次祭典上结识一好友，好友经常送给他一些新奇的小玩意儿，后来平藏得知这些都是好友偷来的后与其绝交，并把之前收到的礼物全部丢掉只剩一颗曾经象征友谊的鹅卵石。一年后的祭典上，平藏鬼使神差的带着那枚鹅卵石来到祭典，见到了倒在地上的好友，好友是为了保护自己装有鹅卵石的钱袋不被小混混抢走，而被匕首刺中了胸膛。好友死后，平藏五味杂陈，誓要对抗世间所有的“罪”[79]。
荒泷一斗儿时的荒泷一斗与其父母因为鬼族身份，被村民诬陷是村里恶性事件的罪魁祸首，被迫背井离乡。后来，父母因病去世，荒泷一斗流落街头时被一名老奶奶「鬼婆婆」收留[14]。
