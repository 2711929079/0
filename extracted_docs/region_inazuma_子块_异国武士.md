# 异国武士

在当今「锁国令」颁布之前，稻妻亦颇有一些异国武士剑豪活跃。[88]
锁国令愚人众蛊惑天领奉行九条孝行和勘定奉行柊慎介，向雷电将军建言颁布锁国令和眼狩令。社奉行发觉异常，但上奏公文被另外两奉行扣下。[31]出入稻妻境内，都会严格把关，稻妻外海被雷神意志产生的暴风雨包围，想要出入稻妻是非常困难的事情。外来的人想要进入稻妻只能来到「离岛」这个区域，且登上离岛就困难重重。[89]
