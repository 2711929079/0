# 西弗安雅与伊利萨

五百年前两位坎瑞亚战士—西弗安雅与伊利萨和当时的「回声之子」长老陶奥鲁缔结了深厚友谊，牺牲自己帮助纳塔英雄封印深渊力量，两人自己却和深渊魔物一同被封印住，从此在封印里陷入了永远的战斗中无法解脱，但仍和其他被封印的纳塔人/龙的意识一起阻止深渊涌出「镌光铭印」。但却被后人误解为引来深渊的人[47]。
毁灭诅咒灾厄发生后，除草神以外的六神与天理降临坎瑞亚[48]，对这个无神的国度降下了审判。自此天理命令死之执政对纯血坎瑞亚人降下「不死诅咒」[49]，只能就这样眼睁睁看着自己的灵魂和身体慢慢磨损，变得腐朽。死之执政降下诅咒时与「丑角」对视[50]。宫廷卫队的黑蛇骑士们随着岁月的磨损只有少数灵魂尚存，多数变成了失去灵魂的黯色空壳。[4]投靠坎瑞亚的其他魔神血脉因遭遇荒野间的诅咒而变成魔物。[1]哈登受到不死诅咒，蒂莱尔也变成丘丘人（即第五章第六幕你存在的时空中旅行者最初在山洞遇到那个，她成为丘丘人后循着与旅行者的记忆游荡到纳塔，在与旅行者相逢的地方待了500年），维瑟弗尼尔在最后一刻似乎因悲悯自己的弟弟给予他黄金指环，戴因斯雷布因为持有哥哥维瑟弗尼尔的戒指得以透过定期重置身体状态避免诅咒影响[28][51]。（莱因多特、亚尔伯里奇、坎瑞亚许多历史如双子降临、戴因斯雷布的原型之一、尼伯龙根、黄金戒指，都是来自《尼伯龙根的指环》。《尼伯龙根的指环》中亚尔伯里奇偷走莱因多特的黄金打造戒指，若《原神》按《尼伯龙根的指环》发展，那么维瑟弗尼尔大概率是亚尔伯里奇一族，偷了黄金打造成戒指。在剧情中也提到戴因与凯亚长得像，那时听凯瑟琳描述旅行者开玩笑说是凯亚的远房亲戚[9]。此外，凯亚的父亲或许是维瑟弗尼尔，因为凯亚身上没有不死诅咒，凯亚父亲能生下没有不死诅咒的他就代表他也打破的了诅咒，而五大罪人已经成为超越世界的力量因此能做到[52]。）灾厄中妖僧霍格望被深渊力量吞噬[19]，但可能没有真的死亡。
