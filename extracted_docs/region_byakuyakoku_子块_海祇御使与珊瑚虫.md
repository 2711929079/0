# 海祇御使与珊瑚虫

为了结束龙蜥的灾害，大蛇创造了一个叫阿祇的海祇御使，她有着沟通人类和龙蜥的能力，本来是希望她能作为人类和龙蜥沟通的桥梁，结果她在年幼时就被龙蜥劫走并抚养长大，所以她以为自己是龙蜥人[7]。海祇御使是奧罗巴斯最初的眷属。奥罗巴斯使用自己的第二个眷属“珊瑚虫”转化无序的元素力，遏制了渊下宫的圣土化。人们从奥罗巴斯身上采集“血枝珊瑚”，将其植入深海龙蜥，最终选育出可以稳定提供血枝珊瑚的深海龙蜥个体。奥罗巴斯的到来结束了太阳之子制度，对龙蜥势力进行全面反击，用珊瑚王虫改造蛮荒元素力。在此期间大蛇阅读了《日月前事》的内容，意识到这是被天空秩序所封禁的历史，立即禁止了《日月前事》的传播。但不幸的，偷看禁忌的大蛇被天理叛了死刑，必须在无人怀疑的情况下自尽。
魔神「奥罗巴斯」失去了一切、逃向暗之外海的魔神在海渊中，见到了什么都没有的弃民。于是它决定留下来，成为他们的「远吕羽氏尊」、「海衹大御神」。传说大蛇神曾经折下身上所有的珊瑚枝，让蜷缩在黑暗中的孩子们拥有照亮周遭的光明。又有一说它用折下的珊瑚枝架起了登高的阶梯，让它的孩子们再次回到了地面上，见到阳光。这种珊瑚在提瓦特的大地上是见不到的，是大蛇闯入「暗之外海」时，获得的礼赠。因此对于深海的大蛇而言，身负珊瑚枝象征着力量；而失去珊瑚枝时，力量会相应地折损。换言之，折断的珊瑚枝中也蕴含着超常的力量。[8]奥罗巴斯知道：若想在天理的势力，在陆地上生活，就必须根除《日月前事》中记载的历史。于是大力推广鸣神文化，来取代渊下宫的原生文化。
