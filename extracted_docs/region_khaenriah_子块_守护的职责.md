# 守护的职责

坎瑞亚灾变当日，戴因斯雷布赶往皇宫之前，嘱托哈夫丹「转告所有黑蛇骑士，不论如何，守护好坎瑞亚的人民」。之后他们受到诅咒成为失去灵魂的黯色空壳。在这些已经失去理智的黑蛇骑士心中，他们依然在为守护坎瑞亚人民而战。将遗迹视为灾难中的坎瑞亚，将那些丘丘人视为正在求助的民众…这份意志如此顽强，经历了五百年时光都未被磨灭。[4]
白鹄骑士传说五百年前，荼诃古国（坎瑞亚）的「白鹄骑士」曾驾驶着遗迹巨像对抗黑渊涌出的魔群。在残酷的战争后，最终残留下来的三台遗迹巨像被贤者们所封印。[45]部分成员：骑士元帅安弗塔斯·亚尔伯里奇：在海杜菈背叛事件中失去左眼，白鹄骑士的最后一人。在降诸魔山附近的遗迹守卫核心中留下讯息：WE SCHWANENRITTERS HAVE FOUGHT TO THE LAST ONE.（我们白鹄骑士已战斗到只剩最后一人）「金殿卫士」海杜菈：最后的同胞，因背叛行为，造成一台机械不可逆损伤报废，被骑士元帅安弗塔斯处决。「幽谷之女」英格希尔达：在对抗漆黑兽潮的战斗中失踪，依旧存活但因诅咒变为怪物黑蛇骑士，位于降诸魔山的遗迹巨像内。「胡子爸爸」希尔德里克：因重伤致多器官衰竭而去世。[46]
