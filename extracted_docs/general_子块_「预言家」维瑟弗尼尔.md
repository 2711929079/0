# 「预言家」维瑟弗尼尔

被称为「预言家」，是坎瑞亚「五罪人」之一，犯下了「渎时之罪」，曾持有法器「格莱普尼尔」（现名「圣缚」）[105]。戴因斯雷布的哥哥，在坎瑞亚灾变时将指环交给戴因才使其一定程度上能减缓诅咒的效果[106]。因告知伊尔明某个预言使其深陷囚牢，另外四位罪人与戴因一同劫狱，并临时打算防止伊尔明继续动摇世界根基，但五位罪人皆和伊尔明一同禁不住力量的诱惑使用深渊力量。在枫丹留下了「所有人都会溶解在海里」的预言石板，向罗南谕示，使其知道天空的不公而投向深渊[107]，化为紫色水晶的样子引导克洛达尔与旅行者的血亲分别创立与继承深渊教团[108][109]，目前还没有正式出现。CV：俞安琪（中，后改为曹真）/堀内贤雄（日）
