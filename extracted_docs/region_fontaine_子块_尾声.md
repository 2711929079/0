# 尾声

那维莱特将水神之心以「外交赠予」的方式送给「仆人」。芙宁娜离开沫芒宫，居住于枫丹城内别墅中。后不久以普通人的身份获得水神之眼。
水仙之末古老的颜色雅各布需要厄里那斯的血液，为了不引起逐影猎人注意，决定从厄里那斯住民美露莘那里和平获取。他自称艺术收藏家，向美露莘玛梅赫购买以厄里那斯凝血为颜料绘制的画，但没有成功。之后，雅各布雇佣镀金旅团佣兵强买玛梅赫的画，自己作为彩蛋在附近山头监控。玛梅赫与旅行者外出时，雅各布与佣兵们闯入海沫村玛梅赫家搜寻凝血颜料未果，被玛梅赫撞见。自此，雅各布放弃和平获取凝血的计划，他变身成深罪浸礼者，激活了遗留下的法阵，逆转厄里那斯心脏的晶化过程。期间，法阵短暂打开了兽境。最终，雅各布从厄里那斯重新跳动的心脏处获得了足够多的血。雅各布在告知旅行者「世界式」预定的时刻即将到来，以深渊之力将旅行者暂时封印后离开。[46]
