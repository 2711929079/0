# 炎之魔女

罗莎琳留学归来时，恋人鲁斯坦因对抗杜林逝世，而此时她信仰的神明甚至未曾出手相助（实为来不及）。罗莎琳燃尽生命之火，决定焚毁世间所有造成灾难的魔物。
那位少女曾经爱唱的小曲，也在她返乡后，变了样子。「蒲公英随着晨间的风远行」「秋日的风带回收获的芬芳」「但无论怎样的风」「也不能再为我带来你的注视了」泪水与歌声都枯竭后，少女决定挥霍生命之火，洗净世界的歪曲…[99]少女的时间就此结束，炎之魔女的破灭之时就此开始，直到世间一切魔物和世间一切魔物造成的痛苦被烧尽。[102]
染血骑士纯白骑士罗南师从鲁斯坦，追寻着绝对的正义。曾救下一位仕女，并与仕女交换了鲜花，仕女深深爱上这位和他只有一面之缘的骑士，但这是一个不会有结果的爱情：因为罗南又转过头落实他的「正义」去了[108]。因其实力强大比肩魔神与其理念，曾被众人认为在鲁斯坦之后会接替鲁斯坦成为最强的北风骑士，唯一在成为北风骑士前就被王狼认可[29]。罗南一生都只想着落实正义，却从未想过自己心中的正义也有可能成为「不公」。为了追寻正义，他斩遍无数魔物，因为染上魔物的血身姿也歪曲成魔物之形。在路途中，来自须弥且同样身染黑血的薇瑞迭瑟恩被他误认为魔物斩杀。[109]直至最后，那追寻黑暗塔的罗兰公子，已来到塔前。[110]鲁斯坦为了不要让偏执的罗兰步步走向深渊，打算将所思写成信交给罗南，但在此之前就身亡，此信成了遗言。艾伦德林接替一直寻找罗南但已为时已晚[29]。后续见至冬#挪德卡莱·猎魔骑士。
