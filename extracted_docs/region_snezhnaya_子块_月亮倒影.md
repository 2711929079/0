# 月亮倒影

「高天的主宰」对一些生灵追寻力量的行为有着近乎偏执的敌意甚至恐惧，以至于会降下天钉惩罚地上生灵（见至冬#迁灭之时及其他国家编年史），卡侬尽其所能帮助受天钉惩罚之人逃离灾害，但依旧有太多生灵死在「高天的主宰」手下。就在女神们决定采取某些措施时[3]，被漆黑的世界染蚀的尼伯龙根回归，与天理爆发了战争。三月女神内部也出现了分歧，有神明认为应该追随尼伯龙根，毕竟月亮就是由他升起的，但也有神明认为被卷入争斗的话，世界会更加趋近破碎，后果难以估量，更何况尼伯龙根身上已经沾染了危险的深渊之力。尼伯龙根发现了三位女神的不忠，将她们囚禁在了「月亮倒影」之中的「月宫」内[6]。「月亮倒影」的时间自未来向过去流淌，其终点便是创建的这一刻，即「月宫」所在，无论如何移动，女神们终难逃离此处。三月女神与她们未来的妹妹哥伦比娅·希珀塞莱尼娅相遇，将权能尽数加于其身，自此哥伦比娅即为唯一的「三月女神」。她依姐姐们的愿望将她们送回现实，此后将自己的灵魂留在「月亮倒影」之中，肉体化作月光洒向大地、以月矩力形式影响着挪德卡莱的生物，三月权能则被其藏进霜月之中（哥伦比娅相关内容详见至冬#挪德卡莱·空月之歌（主线剧情）、三月#「新月·哥伦比娅·希珀塞莱尼娅」）[3]。
空月之死囚禁三位女神后，尼伯龙根将月亮改造成了对抗天理的武器。虽然天理曾和三月女神有过和平约定，但眼下月亮对提瓦特的威胁远远超过了可容忍的程度；亦有说法认为天理不知道月神被囚禁的内情，认为是月神率先撕破了和平的誓言[6]。最终，天理击碎了恒月与虹月，将霜月扔到了天外。「恒月」粉碎成数块悬浮于虚假之天外，部分坠落于地面；「虹月」粉碎坠落后为延续生命而与深渊结合，弥合自身残片，成为坎瑞亚人所崇拜的「赤月」。在天之围桓破碎的那日，玉盘坠陨，诸龙众皆为曾背叛了他们的月亮之死而残忍欢呼[4]。当「恒月」陨落，天地为之翻覆；当「虹月」破碎，赤影隐入渊海；当「霜月」停转，列邦归于沉沦[22]。传说明月的高车破碎时，沾染神血的车軎殒入月下的尘世，化作如今分布于挪德卡莱的虹色的宝玉「虹滴晶」。从那以后，大地上的人们不再为死去的神明奉上信仰，唯有那些知晓夜的君王们为何而死的幸存者们依然在徒劳地歌唱着那些早已被遗忘的名字[23]。
