# 嘉明

嘉明的父亲是一位普通的商人，做茶叶生意，希望嘉明能继承家业。嘉明想要从事小时候最爱看的舞兽戏，但父亲认为这根本不是事业，二人意见不合但有母亲在期间做调和。直到母亲因病去世，嘉明练舞兽戏受伤，父子矛盾彻底爆发，嘉明离家出走。嘉明的舞兽戏推广事业并不顺利，为了养活自己，他在一次机遇巧合下成为了一名镖师。期间，嘉明结识了一只小猊兽，并为其取名“文仔”，还成立了“威水舞兽队”，自己做话事人[184][185][186][187]。
蓝砚年纪轻轻的蓝砚已然成为沉玉谷手工艺工会中最厉害的藤编师，因为出身于擅长使用奇门术法的蓝氏，蓝砚还掌握一些基础的术法[188][189]。
