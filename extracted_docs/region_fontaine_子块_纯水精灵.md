# 纯水精灵

枫丹的水体被深渊污染，不再适合纯水精灵的生存。大批纯水精灵离开枫丹追寻厄歌莉娅的葬身地。它们有的向南进入须弥大赤沙海，最远出现在蒙德清泉镇。芙宁娜为了找到除了扮演水神外解除预言的方法，差遣纯水精灵到各国成为间谍，缔造间谍情报网，来探查各国有没有解决类似危机的方式，但为了避免恐慌没有告知纯水精灵自己缔造间谍网的目的。因前代水神差遣纯水精灵到各国是为了散播自己的恩泽，而如今水神居然想缔造纯水精灵间谍网，大部分纯水精灵不接受其理念，拒绝效忠新的水之神，间谍网計劃失敗[52]。之后枫丹廷派遣浊水精灵前往刺杀叛逃的纯水精灵洛蒂娅。（在「纯水之愿」中洛蒂娅对间谍网的计划极度反感，因此也有可能浊水精灵是要寻回她们，但被偏激的洛蒂娅误以为是来刺杀自己。毕竟浊水精灵去刺杀纯水精灵会水元素免疫。）[29]之后浊水的危机解除，但枫丹仍再也没有出现纯净的纯水精灵。[53]
隙境原体隙境原体是巨兽厄里纳斯死后其细胞演变成的魔物。据与「护国的白骑士」同行的逐影猎人的记述，在厄里那斯的惨烈战斗结束后，曾经出现了大量的隙境原体。这些隙境原体并没有对他们采取任何攻击行为。[54]
