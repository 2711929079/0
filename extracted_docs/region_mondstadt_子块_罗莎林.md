# 罗莎林

罗莎琳曾是一位在喷泉边唱着诗歌的纯真少女。鲁斯坦常常来此听她的歌，唯有这么做才能缓解自己的忧愁。后来他们深深爱上彼此，但尚未表达爱意前，罗莎琳便前往须弥教令院留学。离别前，鲁斯坦送她一个沙漏[102]。
