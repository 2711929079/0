# 高塔之王的陨落

约2600年前，旧蒙德民众以风之花为号，爆发了追求「自由」的战争，温迪与群众一同在反抗的硝烟里见识了乖僻的君王之殁。高塔崩落，无人再登王座。君王曾为臣子们提供没有苦寒的城市。直到最后，他都以为众臣爱他，一如他爱他们。女猎手阿莫斯将箭矢射向了乖僻的君王，而阿莫斯也被君王聚拢的烈风撕碎。
