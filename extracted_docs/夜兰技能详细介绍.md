# 夜兰技能详细介绍

夜兰是《原神》中的水元素五星角色，璃月总务司的神秘职员，称号"兰生幽谷"。
元素战技「萦络纵命索」：
- 释放水元素能量，对敌人造成水元素伤害并施加"水元素附着"。
- 长按可以调整方向和范围。
- 冷却时间：8秒。
- 能量消耗：无。
元素爆发「破局矢」：
- 释放强大的水元素能量，对周围敌人造成水元素伤害，并创造一个持续10秒的水元素领域。
- 冷却时间：18秒。
- 能量消耗：80点。
固有天赋：
- "幽谷的静谧"：萦络纵命索命中敌人后，有20%概率触发"冻结"效果。
- "水元素增幅"：夜兰的普通攻击和重击造成暴击时，恢复3点元素能量。
- "兰生的祝福"：元素爆发持续时间内，队伍中的角色水元素伤害提升20%。
