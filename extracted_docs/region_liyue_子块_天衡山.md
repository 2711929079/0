# 天衡山

璃月港西边的山脉，也是抵御外敌的一道天然保护屏障。有记载以来，作为璃月的壁垒经历了数不清的大小战争，因此也可以看到不少古城墙、堡垒和防御工事的遗迹。传说中璃月人最早的采矿作业亦在此处进行。
