# 吹沙角鲀

体型圆润的中型鱼类，生有犄角般的软骨结构，据说它们会有这一结构开凿水下洞穴，建造栖身之所，或是驱赶试图伤害自己的生物。这种角鲀有着黄沙般的色泽，其貌不扬，却是相当重要的品种。学者们根据这种角鲀的生理结构，推断出了整个角鲀谱系的生态特点——这种角鲀有着同族中最强的力量和最坚硬的犄角，扮演着「水下工人」的角色。由于长期在河床与湖底活动，与弥漫的河沙与湖沙为伴，它们也进化出了相应的保护色。
