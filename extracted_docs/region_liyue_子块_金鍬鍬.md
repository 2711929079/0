# 金鍬鍬

传说中能够吐出宝玉的水陆两栖动物，多分布在水源清澈、苇草高耸的地方，譬如璃月的荻花州。传说金色鍬鍬是河流的宠儿。或许也正是因为受着河流温柔的爱抚摇荡，它们才总是保持着天真的笑容吧。会将无法吸收、消化的元素、砂砾、矿物质在体内成闪闪发亮的珠子，并在遇见遇到天敌时吐出，以迷惑掠食者。
