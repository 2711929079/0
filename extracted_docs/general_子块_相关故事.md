# 相关故事

花神是原初之人所造的仙灵一族的后裔。[2]在葬火之年 外来之人带来了战争，也带来了人们对于突破天空的妄想。而天的主人恐惧妄想与突破，降下修补大地的天钉，毁灭了凡人的王国。[3]天理愤怒于仙灵一族对人类的庇护。于是降下了诅咒，让仙灵一族失去了于天空的联系，也失去了教化的能力。花神没有因为诅咒而退化，但因为无法回到天空只能流浪。传说，花的女主人曾经在荒芜的大地上流浪了七十二个夜晚…脚跟被无情的砂砾磨破，伤口流出清泉，化作了无边的溪水。接着，溪流灌溉出绿色的园圃，从中生长出夜一般蓝的睡莲…睡莲是镇灵的母亲，镇灵生自迷醉的梦乡与失却的苦涩记忆。最初的镇灵感念创造之恩情，挽留花神留在鲜花盛放的园圃。在她停驻脚步的地方，月夜般美好的紫色花朵盛放开来——其名为「帕蒂莎兰」。[4]在远古时代的须弥，花神先是遇到了赤王。在听说须弥有着一位智慧的王女后，带着礼物与谜题去拜访了大慈树王。最后，在树王回答了所有的谜题后，二位女王便缔下了坚实的盟约。与一直在她们身旁的赤王三人勠力同心，在魔神互相厮杀的年代共享权利，决心共同引领子民走向永恒的繁荣。[2]在他们三人的努力下，真的诞生了一个安乐和平，没有灾祸，统一无忧的乐园。[5]然后，就算是在这样一个美好的绿洲乐园中，花神仍然是时常叹息，赤王询问，她也只是缄口不言。最后，在一个月光皎洁的夜晚，向赤王诉说了仙灵种族曾经的往事。在讲完故事后，花王对赤王发出了警告，不要追随四重影子的主人，不要窥视天空与深渊的秘密。否则，正如判罚之钉（寒天之钉）昭示的那般，接踵而至的，乃是灾难与苦涩的结局。可是赤王并没有认同伴侣的警告，在心里许下了僭越的愿望，把自己的诉求说与花神。那夜赤王所言及之事，最古老的镇灵也宁愿闭口不言，就连最智慧的神祇（大慈树王）也为之震悚。正如花王所预料的，这位沙漠的王有着叛逆的狂想，但在他的话语与梦想中，花神看到了超越世界荒谬规则的可能性。赤王拒绝了神座降下的恩赐，出于自我意志开拓了新的道路。就算花神所展示的未来如此惨淡可怖，赤王也并未退缩半步。花神认为赤王对于无忧梦乡的追寻终会破灭，所寻之物不过捕风，但她发觉可贵的叛逆在神的野心中燃烧。这样的理念与伟大尝试所蕴藏的不只是谎言，更是属于凡人未来的希望。花神默许了友人的愚行。她向赤王允诺：「我将为你守秘，是因为我对你与那智慧之主怀着同等深切的心意。」「我将为你搭建桥梁，你将满足你的狂想，但不要恐惧幽蓝的晶钉…」「我将引导那深邃的知识，即使我早已警告过，你必定会失去许多…」「尽管如此，请牢记我的教训，牢记天降使者们曾遭遇的酷烈惩罚…」「请牢记，若受造的世界仍然存在希望，希望一定在碌碌凡人身上。」两人的计划在月夜之下决定，他们要为了凡人的未来去赌那一丝可能性。即使花神知道这件事注定惨淡的结局。即使赤王知道自己的心爱之人会因此消失在眼前。赤王还是选择了这个谎言，带领自己的信徒们走向必然的灭亡。以自身为桥梁，以绿洲为代价，花神用自己生命的终结为赤王指引了通往天空与深渊一切知识的密道。[4]神柱自高空降下，将草甸林木埋入漫流之砂。[5]失去一位魔神的乐土风暴骤起，黄沙漫天，很快被灾祸吞噬…赤王从蔽天的风沙中归来，而花的女主人却再也不见了影踪。最后，为了纪念花神，赤王在沙漠的深处，为自己的挚爱修建了永恒绿洲。[4]
