# 梅洛皮斯古文明

天启的统治不知延续了多少个世代，直到人们厌倦了难以捉摸的永恒。后代之人不再虔信神谕，反而希冀神明并未许诺之事，试图挣脱命运的枷锁。于是高天为之震怒，祂（祂：称上帝、耶稣或神的第三人称代词）命大海掀起巨浪，将定居者的城邦碾碎。随后大雨连降百日，潮水吞没了一切罪恶与妄想，枫丹初民的时代就此终结。[46]
