# 玛尔菲莎与埃尔纽艾尔

为追查在枫丹境内作乱的「水仙十字结社」，德怀特委托退役多年的埃马纽艾尔调查[67][34]，循着线索找到了「大师」，被其利用莉利丝的能力溶解吸收。[33]传奇决斗代理人玛尔菲莎出身于灰河或白淞镇，公义的酬报为其所用武器，年轻时曾与埃马纽艾尔交手，被其使诈偷袭而败于其手，被迫认输。信丰公义的玛尔菲莎对此耿耿于怀，本想再找他对决，却意外得知他早早退役、投身烈酒。为了亲自查明真相，她追随着剑客的足迹与创伤，一路追猎命运中的对手…尽管有重要之人（扮演帕西法尔的约瑟芬）倒在命运的决斗场上，尽管内心已残破不堪，在追踪到剑客所留下的面具后，一切沧桑仿佛都得到了报偿[33]。最终，玛尔菲莎与自己苦苦追寻的埃马纽艾尔落得同样的结局，「大师」获取了她的枪术技能。[42]
桑多涅与阿兰阿兰拿过很多笔记本给我，我经常涂两笔就扔掉。每到那时，他的表情无法定义，但还是给我新的。我不知道为什么他要这样。我不知道为什么我涂两笔就扔，也不知道为什么他允许我的这种行为。他说我的情感模组非常完美，我不明白。这本是新的，我不想再扔掉它。我学会记录。我开始记录。我学得更多，谈吐也变得更优雅。优雅这个词我刚学会，阿兰说不用勉强。我不认可，我做得非常好。下午聊天。几个结论：第一，阿兰创造了我；第二，他回忆着故去的妹妹创造出我，现在他感到愧疚；第三，我的学习能力高于预期。阿兰·吉约丹的妹妹名叫玛丽安•吉约丹，已去世很久。阿兰在六十五岁生日当天将我完成，他说我是一个朋友，一位家人。他很高兴生活里多了个陪伴者。阿兰未对我和他的关系下定义。显而易见，我不是玛丽安·吉约丹。他同意这一说法，他也觉得我和玛丽安。吉约丹大不相同，并希望我随心所欲地做任何事。我说我不会再扔掉笔记本时，阿兰略显失态。他应该是想「哭』，又没有真的「哭」。阿兰说如果我想，可以有个生日。我会认真考虑。他说这世上的人无法选择自己的生日，但我可以，我是自由的。阿兰的身体越来越差。医生来过很多次，他都把门关上，不让我知道。我感觉到了什么。阿兰说，那是「愤怒」和「悲伤」，他很高兴我学会了这些。但他说我还不懂得何谓「内疚」，因为「内疚」这一情感背后的社交机制太过复杂，动物和机械很难轻易学会。我觉得我不需要懂得这个。阿兰问我想要什么，我想要他健康。另外，阿兰让我看一件新发明，他叫它「普隆尼亚」。无法定义普隆尼亚是家人或亲属或朋友，普隆尼亚并不具有与我相同的情感模组。不清楚为何阿兰不将它做成与我相同的规格。阿兰称，我和它都是独一、不可复制的。我是我，而普隆尼亚仅是纯粹的机械工程体。日常里，我整理仓库，清扫房间。除错普隆尼亚基本功能。阿兰晕倒了。状况不好。医生来过，表情十分低落。阿兰的下属来探望他，带了一套崭新的茶具和一些名贵茶叶。阿兰不爱喝茶也不习惯使用太华丽的东西，茶具归我了。我种的盆栽全都生长健壮。这太简单，我不觉得有我什么功劳，条件适合时植物自己就会疯狂成长，与他人无关。我认为我也是一种植物。我不需要任何东西也在生长。今天，我帮阿兰一同完成了普隆尼亚的最终测试。阿兰最近无法落地行走了，我用轮椅推着他过去。阿兰没有太大的情绪变化。普隆尼亚运作稳定，但他仍未表现出满意。他说，普隆尼亚的本质是一个算式，用来计算未知的事物。至于那些事物究竟是什么，他也不确定。什么是未知的？那太多了。他居然真的说不出来，而且很悲伤。我不理解。阿兰问：「如何看待世界的命运？」我回答：「用探测器看。」阿兰将他以前的笔记交给我。我很高兴，我喜欢他教我的知识。世界的命运，我最近思考了这个问题。然后很快明白这是不会有答案的东西，人连自己的命运都掌握不了，还想干涉世界吗？ 简直是蠢材。阿兰说蠢材有好几个，我嗤之以鼻。接着他也笑了，不知道在笑什么。日常里，我买了新床单、枕套，还有窗帘。我的房間收拾得很漂亮。我简直太聪明了！还是太保守了。不用「简直」两个字，我就是聪明。我学得很快，阿兰也同意。阿兰的身体坚持不了太久。他坐在壁炉前讲了一些故事。最初，他是为了纪念妹妹和自己一同生活的岁月，才制造了我。他当时居然没想好要做什么，就把我做出来。后来他希望我代替他外出旅行，开开眼界。问题是，我现在是阿兰的同行，我和他都被漫长的职业规划害了！一个人不工作，才能到处旅行。阿兰说这个梦想已经落空，已经彻底完蛋了。他说他还有一个愿望，不过很早就放弃了。我追问了很久，他有点尴尬，说是小时候想过要拯救世界…我问这个需要我继承吗，他吓了一跳，告诉我不用。我做什么都行，他没有要求。阿兰问我有没有什么想做的事，我…暂时还没找到。父母对子女的出生负责，而子女对自己的前程负责。这是社会性的规则，我认为在这套规则的最后，多半是：子女对父母的晚年负责。阿兰的晚年走到了尾声。他立下遗嘱，遗产包括少量存款、普隆尼亚和一些笔记。他让我在他死后烧掉笔记，以免被盗。那么多笔记本，我看也就只有普隆尼亚来偷才偷得动。普隆尼亚是阿兰一生最后的寄托，可惜在他活着时没能看到它算出确凿的答案。他让我别放在心上，把普隆尼亚交给枫丹科学院也可以，只是對那些人感到遗憾，他们没我懂得如何维护它。我不同意。普隆尼亚归我了。不知道有什么用[68]。
