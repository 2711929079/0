# 纳博尔与安杜

纳塔的部族之间关系十分复杂，经常出现宣战、和谈、结盟、背叛，关系变化极快。伊法有一位祖先名叫“纳博尔”，他想要救治白己生病的龙伙件“安杜”，翻阅了许多资料，最终查到了救治方法：海风草和圣火结品。部族间的争锋可以产生圣火结品，于是纳博尔凭自己多年行医所积累的人情，说服了各族首领举办久违的荣花演武。纳博尔委托白己的哥哥兼花羽会首领“伊马雷”前去参加荣花演武，自己则去子找海风草。伊马雷从纳博尔口中得知了圣火结品还可以增强战士们的力量，于是在荣花演武中背叛了其他部族，友谊赛最终变成了真正的流血冲突。安杜目睹了一切，它认为圣火结晶会蛊惑人心，因为要救自己，所以人们才会被圣火结晶蛊惑，反目成仇，于是安杜选择了离开。最终伊马雷私吞了全部的圣火结晶，并全部拿去做实验。纳博尔归来后得知了一场，愤怒地离开了部族，找到了在野外的安杜，与其一直在山丘生活。纳博尔曾尝试请求各部族分一点圣火结晶给自己，但由于伊马雷的背叛，没人愿意帮助他。安杜死后，纳博尔将自己救治各种龙的医术和这段历史记录在织物中，将其埋并在安杜旁边[105]。
