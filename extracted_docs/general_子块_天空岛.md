# 天空岛

神居住之所，是天理秩序的象征，提瓦特的凡人曾经能直接听到来自天空岛的启示，在沉玉谷正上方。神之眼持有者拥有登上天空岛成为神灵的资格[115][116]。神灵透过「神之心」的魔力器官与天空岛共鸣相连[116]。他们和提瓦特世界中的其他的神都与天空岛有联系，但对此话题保持沉默。温迪不断回避话题，每当被问到时候都会隐秘地回答，有人提起天空岛，他也明显变得不舒服[115]。漂浮的「岛屿」似乎由几个陆块组成，中央有一块基础岩，周围环绕着约七至八个较小的岩石。这些外层岩石由一个环形结构相连，该环形结构由从中央扩展出来的多个桥而保持在适当的位置。所有这些基础岩石，似乎都在过去的某个时候，从提瓦特被撕出来一样，它们的表面与崎岖、凹凸不平的底面有很大分别。虽然在外层岩石表面可以看到一些建筑，但主要焦点在中央的基础岩，有两个由大理石或类似的白色石头制成的结构。第一个是位于中央基础边缘的尖顶结构，第二个是更突出的多层尖顶，其顶端有一个类似星盘的巨型金色结构。此外，中央基础岩的结构似乎深深入地，岩石的底部露出一个颠倒的圆顶形。即使是从远距离看，天空岛的整体外观仍然壮观，但明亮的大理石建筑中也可以看到一些失修和磨损的痕迹，以及尖顶和桥梁上有明显的碎块。
