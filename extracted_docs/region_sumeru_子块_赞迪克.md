# 赞迪克

赞迪克幼年时因为某些原因，被故乡的人挥动棍棒与草叉将他赶走（圣遗物贤医之羽文案，文本里没写原因）。后来他前往教令院求学。（因此赞迪克其实不一定是须弥出生，「博士」将须弥称为「故土」可能只是因为自己在须弥求学过）赞迪克曾与黎多范谛学院的学长，图莱杜拉的后人，一同误入了沙漠的遗迹，并遇到图特。其中学长受伤，赞迪克救治他只是为了不耽误调查。赞迪克因自负多疑拒绝回答图特的问题与获得恩赐，并鄙视图特。图特看出其心术不正（虽然极为聪慧，但极度多疑、自负，似乎也没什么行事的「边界」，和他走得太近不会有好下场），在决定送他剩下的第三份礼物时犹豫过，不过他拒绝了，回答了「不需要」。图特为了避免赞迪克抛下同行的人，胡诌说这秘境需要图莱杜拉的人跟着才能出去[38]。一次，教令院组织了探索沙漠遗迹巨像（独眼大宝）的科研考察，考察队由一位德高望重，名为莎娜玛的贤者和六大学派的诸多年轻才俊组成。在考察当中，莎娜玛发现独眼大宝的武器处于卸载状态，而且武器都损坏了。因此她只使用一种名为「须陀流心」的东西封印入口避免他人进入武器库，然而同行的赞迪克却偷偷地将那位贤者的须陀流心给拆了，在取笑这位贤者的机关十分可笑后还在临走前擅自修好了武器系统并让遗迹巨像装配，导致之后遗迹巨像会对路过的人乱开炮。在这之前，赞迪克就曾无数次私自违反教令院禁令研究坎瑞亚遗址。[74][75]…依靠从废弃机体搜集来的零件，似乎可以修复这台巨像的主武器系统，不过为避免引起其他队员注意，:只能暂且记录其尺寸结构，留待日后再行实验……被贤者称为「须陀流心」的东西，其核心看来不过是元素扰动造成的淤积产物，将之作为封印核心，实属可笑…—来自荼诃落谷 赞迪克的笔记[76]之后莎娜玛再度组成探索队，探索降诸魔山的森林。探索队中包括了生论派的陀娑多索赫蕾。索赫蕾选择了与赞迪克组成探索队去探索丛林。赞迪克虽是妙论派，但却能向索赫蕾论证各种动植物生态与特征，所以索赫蕾对他颇具好感。她不知道赞迪克木讷的表面都是赞迪克为了掩盖自己对研究的过度狂热的伪装。…与赞迪克一起探索了这片丛林，这里的生物种群十分完备，尽管在其他地方逐渐滋生的「死域」也对此地的生态环境造成了影响，但尚且不必担忧…在水域中发现了几种全新的微小甲壳生物，命名为「纳西妲德虫」好了——「草神的恩赐」，嘿嘿。…这里遗留的古代机械群，深深吸引了赞迪克的注意……赞迪克很年轻，年轻又英俊。但可惜性格太古板，其学科又与我们生论派并不相通，即使一同探索也没什么共同语言可言，真是苦恼……与赞迪克讨论了本地的动植物特征，进行了演化模型的猜想，我们都很开心。约定了今晚的野餐…—来自散落在阿陀河谷的索赫蕾的笔记。[77]而赞迪克在考察当中偷偷背着考察队研究坎瑞亚机关。…回收了许多部件，看似这些庞大机械的备件，如果能够深研其工作原理与制造工艺……将零件把玩再三，险些耽误正经工作。这些精妙的机械零件，其风格独特巧妙，不似众神统治下的人工所能为之。待我拆解开来，将尺寸形制逐一记录……但首先，这个秘密决不能泄露给其他队员…—赞迪克的笔记。[76]
索赫蕾之死在一次考察时，赞迪克正在研究遗迹机关，而索赫蕾因为好奇而跟踪了他，撞见赞迪克研究遗迹机关之事。赞迪克立刻将对自己有好感的索赫蕾灭口（掐住她的脖子直到她窒息），并果断将其死因伪装成被长鬓虎袭击。在与索赫蕾的冲突中，赞迪克误触机关导致遗迹巨像对考察队成员乱开炮，造成无数死伤。…死者…陀裟多索赫蕾…多处外伤…撕裂伤…内脏挫伤…大出血……但致命伤在颈部…舌骨断裂…机械性窒息…难以断定确切死因…''—解剖记录[78]赞迪克谎称是自己随机应变关闭了因故障自动启动的机关，还借此坚持要将机关拖回教令院进行解题研究和反向重建测试。虽然因私自行动被莎娜玛呵斥一顿，但他也被当成了拯救整个考察队的英雄[79]，名声传遍整个教令院。以自己名声，赞迪克得以招开学术演说，阐扬自己的观点，因为其涉及人体改造和对神明的不敬，其见解被斥为歪理邪说，并被放逐出教令院。然而教令院对索赫蕾的尸体开展验尸工作，发现她真正的死因是人为造成的机械性窒息，[78]赞迪克被列为杀人嫌疑犯，因此还被教令院永久除名。之后失踪的赞迪克前往调查魔麟病与「法留纳传说」[80]。
