# 战后

此前被收缴的神之眼悉数奉还。天领奉行九条孝行、勘定奉行柊慎介被捕入狱。散兵持雷神之心与愚人众失联，不知所踪。愚人众內奸包括内森据反抗军声称全被肃清。[93]九条镰治被鹰司家绑架，被雷电影营救后成为家主名代。九条家趋于没落。珊瑚宫与幕府议和，战事暂时平息，但双方各留一定分寸。愚人众执行官末席「公子」来到稻妻，调查散兵失联一案，遇到意外开启的诀箓阴阳寮。
雷樱雷电影解除眼狩令的决定令将军产生怀疑，为了判断雷电影是否受到磨损，前来与其对质。在与将军一番切磋后，雷电影得到了雷樱的种子，将其种在500年前的稻妻，保护稻妻在漆黑灾厄时免受侵袭。最后，雷电影与真做了最后的道别，真的意识就此消散。[34]
