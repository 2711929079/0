# 枣椰

沙漠绿洲中的常绿乔木结出的果实。坚硬果皮下的果肉肉质绵厚，甘甜可口，对沙海子民而言是重要的食物来源。或许是因为沙漠地区特殊的地理条件影响，含糖量极高，无论是单独制成果脯还是做成各种糖果都很好吃。据说枣椰种植的历史相当悠久，在那些关于古老国度的歌谣中也常常会提到它们。传说在烈日君王夜夜不休的宴席上，枣椰所制成的蜜饯便是第一道开胃的美食。
