# 席位排序与卡巴拉生命树

生命之树（希伯来语：עֵץ חַיִּים，罗马化：ʿēṣ ḥayyim），是一种在犹太教使用的神秘符号，属于犹太教哲学传统卡巴拉的其中一部份思想。生命之树用来描述通往上帝的路径，以及上帝创造世界的方式。愚人众席位排列与卡巴拉生命树的质点排列部分符合。如在16世纪的鲁利安体系的卡巴拉中，出现了质点Da'at。Da'at所在之处，生命之树的十个质点合为一体，因此通常不被描绘。生命之树的十个质点合为一体对应「丑角」统括其余执行官，不被描绘对应其为统括官似乎不列席。第一质点Kether王冠对应「队长」最后坐在曾经希巴拉克的王座源火之圣座，第二质点Chochmah智慧对应「博士」，第四质点Chesed慈悲对应「仆人」身为情报头子虽然手上不干净但是个不错、慈悲的人。第六质点Tiphareth美丽对应华馆套描述「散兵」是绝美的人偶。第八质点Hod荣光对应「队长」描述「女士」「尽管手段玷污了荣耀」。第九质点Yesod基础对应摩拉（资金）是做任何事的基础，即「富人」的理念。其余执行官与质点的定义因为其余执行官的剧情身世尚未明了必须等待官方更新。
