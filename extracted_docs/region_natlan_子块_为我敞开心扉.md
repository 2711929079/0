# 为我敞开心扉

旅行者协助被奥奇坎利用的龙族旧日的统律核心·L型自律监管机阵统合处理单元「伊·奎库叙·茨博隆·库·雷尔（伊蕾尔）」解脱。[80]
盗火者的算计这么多年纳塔这么安静，是因图兰火山因深渊侵蚀出了问题要他亲手维系不能干预纳塔，而旅行者一来他发现了旅行者独特的体质就故意安排一条受「焰主之祝」的谎称长不大的小龙逼旅行者去火山朝圣顺道铲除深渊（就是玩家打的黄金王兽），途中也多次化成各种人如曼科、蒂图来引导旅行者前进。他打算引导旅行者搜齐通牒晶盘再引到图兰大火山，以开启对付深渊的兵器维奇洛波奇特利、彻底清除深渊[69]。而夜神之国的大灵则化成猫短暂将旅行者拉入夜神之国，警告旅行者不要相信所谓「前往圣山的朝圣路」，实则是要通知旅行者别完全相信盗火者，他对人类怀有敌意，帮助人类也是为了自己目的。
