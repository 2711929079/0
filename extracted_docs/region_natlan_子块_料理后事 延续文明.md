# 料理后事 延续文明

由于深渊的侵蚀，修库特尔以对龙群的暴虐统治来维系权威。自感时日无多的他，抓紧自己时不时清醒的有限的时间（最后恶化得与假死几乎无异）做了以下一些工作：一、任命弟弟库库尔坎作为摄政，在修库特尔休眠时代行统治工作；其下还有其他三位亲王，以及十三位至高领主。二、组织实验，研究逆转龙裔的退行性演化。结果发现，退化过程已对龙裔的可塑性造成了几乎无法恢复的严重打击，即使强行令力量与智慧重新回归，龙裔的身躯也会因无法承受而发生畸变，并在很短时间内死去。最终修库特尔找到了有望治愈的方法：通过基因工程与定向选育，经由良性的、多代的血统改造，重新恢复龙裔原有的形态和进化潜能，继而令退化的族裔取回失去的高等智慧。这个过程中用作筛选标记的，便是称作「焰主之祝」的特定基因表达。多代筛选，时间漫长，代价巨大，仁慈的王做不到像对待动物一样去对待子嗣与同胞。因此，修库特尔最后还是选择了叫停，对相关实验报告予以废弃封存。三、致力于防御天理与深渊，研制了能对抗深渊的武器「象拟聚星」；为龙族同胞打造全新的出路——建造了「维奇洛波奇特利」，一旦开启可将纳塔全境燃素化，乃至蔓延到整个提瓦特，一切都会被烧却，之后再通过燃素来记录、复现信息，源火之大权亦将回归燃素化的龙族生命。为控制「维奇洛波奇特利」，修库特尔改造了自己的一枚瞳孔来作为控制中枢。然而库库尔坎认为，兄长为族裔规划的「维奇洛波奇特利」道路过于可怕和狭窄，因此对此计划颇有微词。意见相左的两兄弟渐有决裂之势，不欢而散。四、开发了「秘源龙仆」，并把他亲手制作的第一台原型机赋予了「库库尔坎」的名字——与他最亲近，却不同心的兄弟（库库尔坎至死都不知道这件事）。弟弟库库尔坎相当鄙夷盲从强者的其他龙族，强烈反对兄长开启的计划，将龙的未来寄托在了人类身上：他将使用燃素的技术教给人类，并创造“奥奇坎”——同时拥有龙与人血脉的存在，希望他击败修库特尔，意图开创龙与人的新时代。哥哥修库特尔知道弟弟的所作所为，也劝告弟弟不应像虫豸一样看待自己的同胞；但修库特尔也仍相当维护他的这位兄弟，在库库尔坎遭众亲王和领主弹劾时，修库特尔少有的拖着行将就木的身体出面否决。修库特尔认为，如果库库尔坎的计划成功了，他布下的棋子来到自己面前，也是一件有趣的事，期待看到库库尔坎的道路能企及何处。
