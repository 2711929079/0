# 宵宫技能详细介绍

宵宫是《原神》中的火元素五星角色，稻妻鸣神大社的巫女，称号"琉焰华舞"。
元素战技「焰硝庭火」：
- 释放火元素能量，对周围敌人造成火元素伤害并施加"火元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「琉金云间草」：
- 创造一个持续10秒的火元素领域，领域内的敌人会受到持续火元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "琉焰的祝福"：焰硝庭火命中敌人后，有15%概率触发"燃烧"效果。
- "火元素增幅"：宵宫的普通攻击和重击造成暴击时，恢复2点元素能量。
- "琉金的守护"：元素爆发持续时间内，宵宫的火元素伤害提升15%。
