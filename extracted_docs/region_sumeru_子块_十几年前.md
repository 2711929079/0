# 十几年前

柯莱柯莱生而患有魔鳞病，父母将其托付给自称能拯救她的传教士。传教士实为愚人众，柯莱被注入魔神残渣后被愚人众废弃。柯莱的母亲在发现传教士不怀好意后被愚人众「处理掉」了存活下来的柯莱在提瓦特各地流浪[93]。赛诺、居勒什与缄默之殿很多年前，素论派学者居勒什在沙漠中遇险，被当时的缄默之殿首领巴穆恩所救，并在他的邀请下参与了复兴赫曼努比斯力量的计划。曾经赫曼努比斯为信众留下了两片蕴含智慧与神力的“跋灵”，但已经很久都没人能够继承这份力量，导致缄默之殿衰落。巴穆恩决定用人工手段将跋灵植入人体，而经过多次实验后发现孩童比成人更适合作为凭依体。大约十几二十年前，赛诺的父母为了财富而将年幼的他卖给了缄默之殿，他与巴穆恩的养孙赛索斯一同被选为跋灵凭依的实验者。以凡人之身承受神明碎片注定会带来强烈的痛苦，居勒什于心不忍，于是背叛了缄默之殿，将继承了其中一份跋灵的赛诺带到教令院里照养。在居勒什的悉心教导下，赛诺逐渐学会控制跋灵之力，从素论派毕业后成为了大风纪官[65]。这一时期里丽莎来到教令院学习，她曾是赛诺的师姐。提纳里、赛诺、艾尔海森、卡维是同一届学生，彼此间互有交情。
