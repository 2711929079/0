# 赤月王朝时期

坎瑞亚分成赤月与黑日两个朝代。在赤月王朝時期，照亮地下王国的是一轮赤月，而不是后来的黑日。赤月王朝时期，炼金术与驯兽骑士乃是中流砥柱。驯兽骑士的工作及为统率包括黑狼在内的炼金产物。[14]
教养院在赤月王朝期间，因为宇宙与海洋有许多相似之处，坎瑞亚人通常将宇宙称呼为海洋。因为王国位置特殊，总有世界之外的事物漏入其中。王国的武备会消灭其中的灾祸。同时，有些来自异世界的孩子在自己的世界毁灭后，也有可能流浪至坎瑞亚。贤人中的一位（书中提到这位贤人可能是外地人与少有的为公务离开王国而返回的人）对坎瑞亚王进言，说道自己曾从一个异界的孩子那里听闻异界的故事：曾有异世界民相信神明来自宇宙。每次发现流落的异世界难民，他们都会用最高的礼遇相待。只因他们认为神明会假借难民的形态巡查人间。期待着自所谓的宇宙而来到王国的神——或者说降临者 （堪比世界的外来者），他们建立了收容这种孩童的机构「教养院」。在那之后，王国当中的孤儿或者是来自外界的流浪儿也会被接纳。直到坎瑞亚王朝覆灭，王国的教养院都一直没能等到超越之人到来，但教养院中走出了许多奇人异士，他们中很多人都成为了王国伟大的骑士。[14]
