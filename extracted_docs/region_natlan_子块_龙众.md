# 龙众

位于大火山北山脚处的「石火坠陨处」是在过去某次猛烈喷发中从火山内部飞出坠落地面形成的秘源遗迹，实际上，这里曾是修库特尔进行「龙裔退行演化逆转可行性」研究的场所。极致的「环境适应力」与「进化潜力」曾是龙裔们引以为豪的种族天赋，然而在天钉降下改写了这片土地的法则后，这一点却反过来被天空岛利用，无数龙裔为适应巨变的环境，被迫进行演化，失却了原本的形体与智能，从高等智慧种族沦为了在野外嘶吼的野兽。修库特尔为了挽救那些已经退化的同胞，展开了对逆转的研究，但结果却是令人失望的：尽管退化龙裔仍可适应原本龙族生活的富燃素环境，但退化过程已对龙裔的可塑性造成了几乎无法恢复的严重打击，即使强行令力量与智慧重新回归，龙裔的身躯也会因无法承受而发生畸变，并在很短时间内死去。通过实验和相应尝试，修库特尔还是找到了逆转这种影响的方法，即通过基因工程与定向选育，经由良性的、多代的血统改造，重新恢复龙裔原有的形态和进化潜能，继而令退化的族裔取回失去的高等智慧，这个过程中用作筛选标记的，便是称作「焰主之祝」的特定基因表达。尽管找到了有望最终治愈退行性演化的方法，但在修库特尔看来，这等做法所需的时间实在太过漫长，付出的代价也太过巨大，退化的同胞中终究也只有极少数能够得救，绝大多数都会在此过程中遭到「淘汰」，如此做法实在过于不人（龙）道。即使退化龙裔已同野兽几乎无异，但在这位仁慈的王眼中，他们依旧是自己的子民、自己的同胞，实在做不到像对待动物一样去对待他们。因此，修库特尔最后还是选择了放弃，对相关实验报告予以废弃封存[1][23]。传说中，嵴锋龙的祖先是山一般巨大的龙兽，拥有相当漫长的寿命，沉睡时，群山在其背脊上成型，甦醒时，就会抖落身上的泥土与沈岩，唯留宝石与美玉在鳞角中熠熠生辉。「回声之子」部族中巨大的钻状物，据说是那时龙兽巨大钻角的遗留。[24]传说中，原本的匿叶龙并非现在的模样，而是更小，小到不易被观察到的程度。它们甚至可以随意游荡在风中，依附在其他巨龙的身体表面，或是如云一般群聚与游移。只是后来为了能够维持住能力退化后的形态，不得不选择现在的身姿。[25]传说中，更为远古的龙拥有更加强的环境适应度，那时海中最为巨大的龙漂浮在水面上时，甚至会让人误以为是一座浮起的岛屿。据说现在的鳍游龙就是那种巨大海龙的后裔，从其宽阔的背部，似乎也能窥得远古海兽的庞然形态。[26]绒翼龙的先祖是最接近巨大翼龙形态的龙兽，能够如云一般在天际巡游。叙事人的故事里，远古翼龙里最伟大的那只，只要张开翅膀，就可以遮蔽一整片天空。如今的绒翼龙自然不再具有那样的形态，只有藏在绒羽之中的龙爪仍如从前一般锋利。[27]原本的突角龙具有堪称庞大的形体，甚至可以用角顶着山峰进行移动，只不过行动远不如当今这样灵敏。或许如今突角龙身躯里所具备的那种力量，也是一种血脉之中的遗留[28]。
献祭习俗在原有的文明如破碎的玉石般失落,自天上来,引导地上文明的使者被醒来的巨龙吞噬殆尽之后，步入荒蛮的人失去了重新认识世界的能力。轮耆于是他们将一切所得都归于最为原始的信仰,认为承载一切的大地自有其神秘，于是,所获的猎物被作为祭品上贡的时代开始了。[29]当时的因过于原始的元素导致大地无法耕种，再无人可向土地祈求令口腹满足的食量。那时，古老的先民于祖遗庙宇，开始向夜神寻求启示。[30][31]那时候纳塔人信仰一位「十字路的主人」，“十字路的主人”是在过去为六部族降下睡梦或死亡的恩赐，引导人前往冥土，让魂灵融入黑色大河的存在，是夜之国度的主人，是“巫医爸爸”，也是“梦境妈妈”（现已确认「十字路的主人」指「三月女神」哥伦比娅[32]，本段部分描述为民俗传说）。传说他在黑色大河的尽头有一座酿酒小屋，喝了他酿的蘑菇酒就能从梦境中知晓任何人的命运[33]。传说他会指引那些不幸而未能光荣战死的人的魂灵，是个把恐怖当游戏的恶劣家伙，经常制造诡异的噩梦来玩弄被他注意到的生者[34]。十字路的主人曾「预知到漆黑兽潮」而拒绝祭祀。此时六个部落的土地因此无法耕种，部分人重新成为蛮族。当时曾有人信仰龙族，位于「歇息崖」与「曲藤地」，还兴建龙神像，被祖遗庙宇的部族人征服[35]。
