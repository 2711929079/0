# 陆鳗鳗

体型修长的两栖鱼类，有着在浅滩、沙洲的陆地上获取热量的特殊习性。陆鳗鳗肉质鲜美，烤制后味道独特。在稻妻，是人人偏爱的美味食材。陆鳗鳗的幼鱼通体透明，长成之后才会慢慢显现出漂亮的朱色耳鳍与赤黄斑纹。除了用作高级食材外，也经常被当作观赏性的鱼类。
