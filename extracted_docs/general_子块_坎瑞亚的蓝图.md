# 坎瑞亚的蓝图

命运的织机需要重建坎瑞亚的地脉完成复国，需要坎瑞亚的所有记忆，然尔深渊教团的人因记忆磨损无法达成。于是深渊教团举全兵力攻入坎瑞亚引诱戴因斯雷布出现，再由戴因的熟视仇人哈登诱骗他回想坎瑞亚的过去，导致他的所有关于坎瑞亚的记忆被命运的织机读取，坎瑞亚地脉的蓝图完成了。此事在夜神的帮助下完成的，但夜神向旅行者的血亲谈了某个条件作为代价。坎瑞亚的地脉虽完成却无处存在，旅行者的血亲只得寻找承载坎瑞亚地脉的方法。同时，因伊斯塔露暗中操作，旅行者将自己的飞船休眠舱钥匙给了五百年前的蒂莱尔（旅行者也可以将钥匙收回，但结果是一样的，因为五百年后旅行者的血亲也会受到伊斯塔露的干涉见到蒂莱尔，为了防止收回钥匙导致涟漪效应使世界走向未知的命运使自己的计划（即自己因为被降格了，所以成立深渊教团用一整个世界与天理对抗打算获取与之平等对话的资格恢复降临者身分，和让新坎瑞亚成为自己合适的新的家乡）不能如计划中进行，于是将自己一样的另把钥匙丢给蒂莱尔。旅行者在内心纠结下顺从血亲的计划。）。经历各个事件，旅行者与血亲重逢，问到他/她做的事对我们自己而言有意义吗？旅行者的血亲会答：「从一开始，就只关乎你我」。因此，旅行者的血亲帮助坎瑞亚复国是与旅行者和自己最初旅行的目的相同的，而他们旅行最初的目的是在故国覆灭后找到适合的新的家乡（见双子的旅程），因此旅行者的血亲重建坎瑞亚的目的，大概是为了将重建后的坎瑞亚弄成适合当成自己和旅行者新的家乡的样子。在旅行者的血亲表达自己打算利用整个世界换取与天理平等对话的资格时（即自己因为被降格了，所以成立深渊教团用一整个世界与天理对抗打算获取与之平等对话的资格恢复降临者身分），天理的维系者疑似听到了此对话。之后，旅行者的血亲离开[63]。
挪德卡莱你来到挪德卡莱，挫败猎月人的复活计划，途中发现自己真正的血亲从未离开飞船。此外，飞船似乎被不明人士隐藏起来[64]。因为提瓦特的命运被注定，只有身为降临者的旅行者能影响，因此「博士」想夺取月髓也因为天理制定的命运不可能成功。但在旅行者插手此事后，一直深入干涉月髓的命运，将其影响，在如「博士」计划下，旅行者的所有行动都形成构成蝴蝶效应的「变数」层层堆砌，最终打破了命运。为了堆砌足够的变数，也为了让旅行者一行人能顺利找到奔月的仪式的遗迹已执行下一步计划，他扰动提瓦特边界引来魔女会，而魔女会以为这是深渊影响，还和旅行者见了面，影响旅行者的行动。而他深知被世界逐渐排斥的「少女」在猎月人一战后肯定即将消失，只得奔月，而奔月的仪式需要摆放两个月髓，于是「博士」制造出月矩力浓度高，比月亮更像月亮的装置让人在地下秘境只凭月矩力误以为那里就是月亮，「少女」中计透过奔月的仪式传送到「博士」的囚笼中，因为力量不够无法破除囚笼，接着「博士」将两个月髓用月矩力从仪式传送门也吸引过来。如今「博士」已经用一连串计谋集齐了虹月与恒月的月髓，获得了三月生死时空的权能，能够暂停时间，只有不受法则拘束的降临者能不被影响，为了是超越自己命运束缚，达成不被世界允许的实验。「少女」为了不让自己的力量为「博士」所用跳入月之门消散生死未卜，「博士」失算但也已获得二个月髓的力量。在夺取月髓后「博士」凝滞时间。让来追杀他的旅行者一行人冻结，但不受提瓦特法则影响的旅行者杀了上去，但「博士」的实力碾压旅行者，且「博士」以计划好，先前奔月仪式之需旅行者不吃不喝了三天后暴饮暴食在忙了这么久后在祈月之夜上过节也没有休息会影响自己身体，被「博士」轻易击败。如今挪德卡莱所有人齐聚对抗「博士」，并打算透过旅行者触发的变数打造利于自己的命运[65]最终击败了「博士」，发现隐藏飞船的是天理，飞船曾在很久以前略过提瓦特[66]。
