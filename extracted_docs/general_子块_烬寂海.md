# 烬寂海

一丝风也没有的，由灰烬构成的无边原野。许多经验丰富的冒险家的梦想断送在其中，许多不曾止歇的旅者也为之止步。时空错乱的地带，黄金的失控造物巴窟纳瓦盘踞其中。其污染具有时间性，会侵蚀时间的每个角落。为了防止其继续侵蚀世界，巴巴托斯将其从时间轴上吹出去[71]。
