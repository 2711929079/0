# 鬼

彼时鬼族分汤青鬼、赤鬼两派，赤鬼与人类亲近，青鬼比校低强、固执。青鬼时不时会和人类发生矛盾，让人类觉行鬼族不好相处，导致赤鬼无法融入人类社会。后来赤鬼和青鬼各自两方的首领达成协议：让青鬼扮演恶人，赤鬼帮助人类赶走肯鬼，从而歌入人类，但是赤鬼要可护鬼族的“骄做”。之后自我牺牲的青鬼一族因为被人类讨伐逐渐没落[14]。大门手荒泷是稻妻古代著名的四大武者之一，和胤之岩藏、雾切高岭、长蛇喜多院齐名，他以豪迈又傻氣的性格聞名，广受孩童喜爱。
「五歌仙」传说在古代稻妻，曾经活跃着五位能歌善赋的诗人，世人称他们为「五歌仙」。[15]据说他五歌仙是稻妻古代的传说人物，因为能歌善赋而得到「五歌仙」的美名。据说他们的作品深得将军大人喜爱，每年五人将新作集结成册后，都会派出一人前往天守阁，将诗集献给将军品鉴。因此，流传的五歌仙诗歌作品都是四首互有联系的诗歌为一组，其中每首由一位歌仙所作。诡异的是，在成套的五歌仙诗集中，从来没有出现过黑主所作的篇目。为了纪念五歌仙，后人为他们创作了无数的故事，并发展出了祭典「光华容彩祭」。但随着时间流逝，五歌仙与「容彩祭」皆被世人淡忘（直到锁国令解除才再度举办）。五歌仙分别名为「翠光」、「葵之翁」、「赤人」、「墨染」与「黑主」。五人真名现已不可考，一种流传较广的说法是：因为五歌仙故事曾被大量搬上舞台，为方便观众辨认角色，便以绿、蓝、红、白与黑五色戏服加以区分，久而久之，观众便以角色的代表色为歌仙命名。不过也同样找到了「翠光因其草庵名『翠光堂』而得名」、「赤人因喜爱在作品上加盖朱印而得名」之类的说法。[16]
