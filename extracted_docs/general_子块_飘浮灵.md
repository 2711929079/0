# 飘浮灵

高浓度的元素孕育而生的魔物,能够飘浮在空中。
飘浮灵在翅叶闭合时会拟态成某种植物果实的样子。由此看来,它们花瓣一样的翅叶应该是一种拟态器官。因此在一些古老的传说中，它们被当作天上巨木结出的果实,被认为有着神奇的药用价值。
