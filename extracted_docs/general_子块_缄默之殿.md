# 缄默之殿

最初由尊主赫曼努比斯在神王阿赫玛尔死后带着他的信仰者连同长年担任祭司的提纳勒人（提纳里是提纳勒人的后代）一同到图莱杜拉城所建，职责是保护和管理赤王文明所遗留的知识，后在与居尔城的战争中因被沙漠民排斥不得不离开沙漠，进入雨林，与教令院曾建立合作。随着时代变更，贤者一度变得迂腐愚昧，缄默之殿无法信任当时的教令院，合作关系结束，大约在四百年前又回到沙漠。由于合作关系中断不利于教令院的名望，所以他们在教令院内建了一个假缄默之殿来掩盖真相。缄默之殿原首领为巴穆恩，在金狼之章第二幕「众贤之贤」完成后，巴穆恩病逝，在赛诺的协调下，首领由巴穆恩之孙赛索斯接任，并将重新与教令院建立合作关系。
