# 人物背景

•背景考据：从圣遗物苍白之火中的无垢之花、炽烈的炎之魔女、终末嗟叹之诗得知，「女士」应该是蒙德已逝的西风骑士团暗面、西风剑术创始人、副团长幼狼骑士鲁斯坦的少女恋人。在毒龙杜林袭击蒙德中，鲁斯坦在与魔物厮杀的战火中牺牲，从须弥教令院学成归来的少女走向了战场，不知在战场厮杀了多久，最终化为了炎之魔女。直到生命枯竭，火焰熄灭为止。在燃尽之际，接受了愚人众「丑角」的邀请，得到了冰之女皇的力量，成为了愚人众执行官之一——「女士」。
