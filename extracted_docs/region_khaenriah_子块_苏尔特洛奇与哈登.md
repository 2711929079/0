# 苏尔特洛奇与哈登

苏尔特洛奇是死囚的儿子，整日和一帮不良青年混迹街头，受过不少批评和苦难，但他始终保持乐观、自信，后来结识了身患重病的坎瑞亚贵族继承人斯克弗努格并承认了对方的天赋，同他一同切磋，成为朋友。10岁那年，他偷来了禁忌的研究资料，治好了斯克弗努格的病[27]。身为教师的哈登传授剑术，在教导戴因斯雷布黑蛇剑术时却为了维系自己尊严而对其不公的体罚。「极恶骑」苏尔特洛奇也因同样原因变的肢体残缺，在哈登的残害下变得性格扭曲。此事一直被戴因所记恨[28]。
蛇之七变「蛇之七变」最初是坎瑞亚深秘院开发出来的小法术，本来的名字是「混沌渗透」，用来嘲弄大地上的元素力使用者。术士们可以用深渊力量诱发出类似元素力的变化。其原理类似于用深渊渗透现实，制造特定性质的负压，来诱发元素力的流入。坎瑞亚本就以此为幻戏，只在表演与游行里使用。极恶骑本是武人，根本不通这东西。但随着对深渊力量的理解渗入，他对这项技术可以说是一点就通。寻常人研习一种元素变化做表演之用，当卖艺糊口的把戏都需要数月的功夫，极恶骑初次尝试就可以七种元素次第变换了[29]。
