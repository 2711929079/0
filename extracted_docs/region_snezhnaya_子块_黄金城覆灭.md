# 黄金城覆灭

一夕之间，光耀的精金塔像无花果树般倾倒；从圣城到古都，极北的冰原上凡有血气的城邑，皆被幽蓝的晶钉尽行毁灭。妖精们尖啸着化为银白的冰雾，无论是那受颂的圣徒还是最初的神使，均于灾厄后不知所踪（见#狼与仙灵），因远离诸城而侥幸生还者瑟缩于风雪中，在荒废凄凉的幽暗中等待灭亡的终临。[11][1]
霜月之子成立亥珀波瑞亚的遗胤在挪德卡莱筑起了月庭，唤回了失去的月光[14]。长夜的绝望中，唯有霜月的女主人为凡人的苦厄垂泪，出于哀怜，也出于更为隐秘的心愿，她回应了幸存者的祷唤。她用银白的辉光纺出丝线，指引流离的遗民走出了霜原、逃向南方[10]，自此亥珀波瑞亚高傲的后嗣开始以「霜月之子」自称——无论究竟是出于什么样的本意，那都是赐予众人新生的神明。在古老的传说中，寒涌石上的空洞曾是霜月指引妖灵们避难的居所，亦是通向幽域的门扉[15]。传说，咏月使的角冠，乃是霜月于迁灭之时降下的恩典：银树摧折后，再也无法寻得白枝的祭司们哀求得到新的礼冠，月的女主人便怜悯他们，用无瑕的月光织就与生俱来的圣冕。[11]
