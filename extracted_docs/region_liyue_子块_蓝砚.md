# 蓝砚

年纪轻轻的蓝砚已然成为沉玉谷手工艺工会中最厉害的藤编师，因为出身于擅长使用奇门术法的蓝氏，蓝砚还掌握一些基础的术法[188][189]。
刻晴几年前，出身名门望族的刻晴在祖父过世后年纪轻轻便成为了璃月七星中主管土地建设的玉衡星[80]。
