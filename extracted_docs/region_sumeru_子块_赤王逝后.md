# 赤王逝后

为了解决禁忌知识产生的问题，赤王决定牺牲自己[55]。赤王死后，沙漠地区大乱，各类藩王轮番称霸，不停征战。[56]
沙漠诸国第一王朝称霸沙漠的第一位王是“兰沙赫尔王”卡乌斯巴依，蒙面王后裔，为了称王屠杀了很多人，毁灭了许多城市，掠夺大量人民填充自己的都城，将沙漠分为七块区域，分别派人治理。分别是：“蓝宝石之城”图莱杜拉，城主呼玛云王，吸引很多四处而来的无归者聚集[38]“勇士之城”萨勒赫，城主图兰哈冈赤王与花神共创的“圆形剧场”阿伊哈努姆，多数居民为镇灵，城主加尔沙斯普。“兰沙赫尔王”卡乌斯巴依死去后，他年幼的孩子阿塔希尔即位，图兰哈冈（萨勒赫城主）摄政，图兰哈冈有叛逆之心，未称王便被呼玛云（蓝宝石之城城主）起兵讨灭。
