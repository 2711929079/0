# 深海鳗鳗

体型修长的两栖鱼类，通常出没于深海中。生物学而言是陆鳗鳗的异种，会通过共生的方式，随其他相比之下更加庞大的生物行动。之所以出现在渊下宫，或许是因为深海龙蜥、鳐，抑或是古代的大蛇将它们带到了这片陌生的土地上吧。
