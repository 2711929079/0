# 白朮

胡桃爷爷的弟弟白附子是往生堂的二把手。两人年轻时外出游历，在沉玉谷遇到了一名药师，而后胡桃的叔公不顾胡桃爷爷的阻拦，抛弃家业，去到沉玉谷找那位药师学医。胡桃的爷爷继承了往生堂，成为第七十五代堂主。胡桃的叔公继承了药师所身负的长生的“珥蛇拖龙法”，在沉玉谷一代行医[143][144]。胡桃的叔公收了一位名叫“江蓠”的女孩为徒。几十年前，珥蛇拖龙法的传人是白附子，他也是往生堂前代老堂主的兄弟、现任堂主胡桃的叔公。幼年时期的白术的故乡遭遇一场疲疠，白术因此失去了双亲。继承了长生“珥蛇拖龙法”契约的医者白附子抑制了疾病的传播，白术从此跟随这位医者学习。江蓠比白术年长几岁，成为其师姐。白附子希望徒弟白术能够打破先祖的宿命，在师父死后，白术因不忍让长生断绝契约后死去，所以依然继承了珥蛇拖龙法，并开始追求不死之法，发誓要成为最后一任继承者，而后带着长生前往璃月港，开办了药庐“不卜庐”。白术曾在绝云间的山林里遇到了从琥珀中破封而出的七七，知道了她是僵尸后将其收留到不卜庐里，尝试研究出不死之法[145]。江蓠开始独自出去游历，期间与一位名叫“嘉良”的采药人成了家[77]。
白鹤留云曾偶遇一只获得了仙力的白色野鹤，作为师父引导她修炼。50年前璃月突发旱灾，白鹤想要像师父那样拯救苍生，但那时的她尚不是真正的仙人，无法掌握仙人的化形之法，于是留云为她炼了一种「化形药」让她能化成人形，但药效只能维持十年，十年后若她没有变回鹤形，就会忘记自己作为仙兽时的一切记忆，变成非人、非妖、非仙的存在。然而白鹤在下山镇压旱灾期间与一个天衡方士相恋，因此背弃了十年后变回野鹤的诺言，写信与留云辞别后归隐于人间。以人类身份生活时，白鹤患上了严重的健忘症，生活难以自理。40年前，留云看见信后非常生气，本想去找白鹤兴师问罪，因为她背弃了诺言，而且她找的天衡方士悟性不足，驱使符箓毫无技巧可言。但留云在看到白鹤的现状以及丈夫照顾她的决心后心生怜悯，便易容成商人模样，在望舒客栈为她提供了一壶以自己血液炼制的灵药，才帮助她压制了几分化形药的副作用。当白鹤分娩时，留云还帮助她与爱人击败了追杀他们的魔神残渣。白鹤的孙女曾生了重病，留云以鹤形拜访希望医治，但白鹤的孙女坚持认为只有白鹤才是仙鹤，因此留云只得将自己全身涂成白色，驮着她去自己的洞天里医治疾病。[47]
