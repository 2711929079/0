# 第五王朝

“代伊王（海盗）”穆扎法尔死后，由其子“绿之王”祖玛鲁德称王，绿之王某天做梦梦见沙漠变为雨林会更加繁荣，于是大力种树。代伊王告诫自己的儿子要多生孩子，以免发生道莱王的悲剧，所以绿之王有九十九个孩子，四十九个女儿，五十个儿子。绿之王死后，他最有实力的两个儿子——巴达那与穆祖丁为了王位互相争斗，最后巴达那取胜。巴达那即位后逐渐贪图享乐，变成了整日与美食、美酒与游猎为伍的暴君，甚至将自己的忠臣挖去双眼。忠臣离开王国，流浪世间，成为老朽的盲眼诗人，向沙之民收集沙海的诗篇。王子法拉玛兹与当时还是舞女的玛海菈（镇灵之后）有了除掉国王的念头。玛海菈误入遗迹遇到鹮之王，回答祂的问题，图特因此给予她「被爱」的能力[38]。巴达那死后，新王法拉玛兹即位，因其年轻，王后玛海菈摄政。[57]不久便禅位于其叔父，“蓝宝石城城主”，相国瓦拉什。国舅格拉兹起兵推翻瓦拉什，复立法拉玛兹为王，自己摄政。不久后罢免法拉玛兹，自己称王，玛海菈为皇后。蓝宝石城贵族为拉瓦什复仇，推翻格拉兹，复立法拉玛兹为王，王后摄政。勇士之城城主反叛，自立为王，王后平定叛乱。之后法拉玛兹的十二个叔父全部叛乱，战争连连，民不聊生。战争平息后，法拉玛兹再度即位，王后玛海菈一心一意想要成为皇后，但不在意国王是谁，于是法拉玛兹下毒杀害玛海菈，玛海菈中蛇毒而亡。不久奴隶兵与市民里应外合推翻法拉玛兹，并将其流放雨林。[58]在这之后，沙漠子民厌倦了战争与王朝，在盲眼忠臣领导下前往沙漠边缘建立了新的定居地，即阿如村。
诸国灭亡后迷宫的游人曾经有一个人类孩子辛娜梦，她迷失在了森林里。循着森林王留下的足印，她来到了老虎的花园。「嗷呜，每一种走兽、每一种飞禽，都曾在我的五脏府中巡礼一番，最后回归大地。」「我是森林的王，杀了许多，保护了许多。唯独只有你这样的小人，我还没品尝过。」老虎这么说。当然了，那孩子当时还没有忘记你们的语言，自然也听不懂王所说的话。「嗷呜。」孩子说。森林王反倒觉得很有趣。往往他一旦开口说话，人类就会躲进木头的房屋里，走兽伏倒在地上不敢动弹，禽鸟远远地飞向太阳。就连长鬓的威武大猫，也躲进树丛。「嗷呜，不懂礼数的小东西。也好，那就由我来教你森林的道理，还有森林的语言。」「从今往后，你就是我的近侍。森林的宫殿将对你敞开，林中诸兽也不能加害于你。」「记住，这个世界只是森林的梦。有一天你会从现实中醒来，来到无边无际的猎场。」「我们的猎物所前往的地方，我们总有一天也会抵达。这个道理，你永远不要忘记。」在一代的森林王老去之后，新的王将会替代他。你们、我们，森林中的生命都是这样。这个近侍与王一同老去之前，遇到了同样迷失、不踩碎任何落叶而踏入了宫殿的孩子，并将一切教给了她。她曾经收留许多迷失林中又不曾践踏野花的孩子，通晓百兽的语言，随侍森林王，为王守卫迷宫。如同这个故事传给我一样，那个孩子也将教导传给了之后的孩子。后来，有的孩子成为了森林的守护者，并将森林的语言、保护王土的责任传给更多人，也有人在面对他人的巨大痛苦时，决定告别生涯尽头的猎场，成为令恶兽胆寒的猎手。[59]森林王诞生之时，曾得到大慈树王赐予的宝冠。它最后传达给了第一个循着自己的足印，穿过迷宫的少女。[60]
