# 纳塔大联盟时期

帮助希巴拉克讨伐火龙王的成员如下：半人半龙的奥奇坎，其实是继承了「焰主之祝」的「盗火贤者」瓦萨克拉胡巴肯之子，他将自己儿子奥奇坎融入人类灵魂，以此达成人龙合一的新物种[69]。希巴拉克身为「人神」寿命无法长久，盗火贤者打算希巴拉克寿命将尽时让奥奇坎取代他并统治纳塔，让理想中的人龙合一者治理纳塔。希巴拉克结识时他是成为火神之前，奥奇坎一度遭受拘禁，希巴拉克解救并拉他进討伐火龙王的阵营。悬木人的祖先——尤潘基，他和希巴拉克、奥奇坎是朋友关系，他是大联盟的军械官，年轻的他十分莽撞活泼。被尤潘基逗得哈哈大笑的弓手是莉安歌，被龙养大的少女，三度射落邪龙，三度将它治愈，成为至交，称希巴拉克为「拉克」，她成立了「花羽会」。[70]双生英杰兄妹——阿霍布与伊葵，父亲因龙王一生心血而殒命，母亲因保护他们而牺牲，他们因年幼时失散，被不同的部落养大。重逢后踏上冒险，依循「力量」与「祝福」的道途。阿霍布创立了「沃陆之邦」，伊葵则创立「回声之子」[67]。伊葵之所以被称作「锻火的英雄」，也是因为她真的通晓如何锻造燃素本身。她并非龙众，只是因为没有父母赐予的名字，才依照龙的语言规则为自己取了名。萨库科见过伊葵的技艺[62]。烟谜主他们居住的深谷那时还不叫「米克特兰」，继承王名的是一位剜去双足的的少女（祭星者之望的主人），这里龙与人类立下盟约，龙保护人类(奴隶)，人类选出一位圣王指引龙入梦。少女圣主身边的仆人丁加预知到希巴拉克到来时，少女与他相见，下次相见会刺穿她的心脏以此打破旧的盟约，成为屠龙者马古汉后，建立烟谜主的新制度。[71]传说在古老的时代，在大地拒绝种子时，部族民无法向土地求得食粮而陷入了绝境。那时，祭司马古汉听从夜神指引，自河底挖来淤泥，向其中种下动物的牙齿。而后他站在淤泥中，割伤自己，用鲜血浇灌，于是那一颗颗牙齿顺着祭司的身子长了出来，变成了如今的颗粒果[72]。
明晨之镜第十一席明晨之镜在龙族败北后，远本属于龙族的故乡失去了，于是陷入替故乡向人类报仇，或是寻找新的归宿的纠结中，在矛盾与对自己的诘问中，意识一分为二，分别是怀恨人类的「领主」与想寻找新故乡的「悖谬」。领主对「悖谬」下了诅咒，「悖谬」只要想要寻找故乡就会遭受诅咒折磨，于是悖谬只能持续丢失记忆使自己不要产生寻找归宿的渴望，以此避免痛苦。因为第八席的机关权限位于第八席养女莉安歌手中，领主欺骗莉恩歌让其删去悖谬却反被封印。「花烛与风羽的司巫」曾试图将自己的领地转变为完全封闭的避难所，以此建立她理想中的国度。她将这计划命名为「希泊巴」——那个时代人类传说中的冥府。然而「希泊巴」在完成后从未被投入使用，直至她被处决的数十年后，被她莉安歌用作囚禁第十一席上千年的牢笼[5]。伊葵将月之轮嵌入悖谬的核心。后来悖谬在莉安歌鼓励下扛着诅咒踏上寻找归宿的路途。而莉安歌也启程讨伐奥奇坎。古时的纳塔人曾种植过一种名为香荚兰的作物，用于调味与医用。它们原生于深林的幽暗处，既没有粗壮的枝干，也没有发达的根系，只能攀附于其他树木，靠树冠间隙里细碎的阳光默默生长。多么弱小的植物啊，就和人类一样。年幼的女儿曾如此惋叹。但依附于强者，也是一种生存之道。稍长的儿子却如此思忖。不，即使只凭藤蔓，也可以绞杀大树。只要它们像蛛网一样虬结，像锁链一样紧扣。母亲用利刃剖开成熟的荚果，籽粒伴着芳香洒落而出。只要它们永不忘却自己的孤傲：即便生于潮湿的腐殖土，也要以藤的筋骨，触碰天空的高度。在伊涅芙临行之际，莉安歌曾折下一株香荚兰，饰于其构件之间，作为铭刻的纪念[5]。历史上记载第十一席被莉安歌与伊葵击杀；与莉安歌为了不让秘源技术落入奥奇坎手中封印第八席的实验室，皆非事实[65]。
