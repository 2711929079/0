# 诺艾尔技能详细介绍

诺艾尔是《原神》中的岩元素四星角色，西风骑士团女仆，称号"未授勋之花"。
元素战技「护心铠」：
- 生成一个岩元素护盾，吸收伤害并提升防御力。
- 长按可以提升护盾强度和持续时间。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「大扫除」：
- 挥舞大剑，对周围敌人造成岩元素伤害，并恢复护盾强度。
- 冷却时间：20秒。
- 能量消耗：60点。
固有天赋：
- "女仆的自我修养"：护心铠持续时间内，诺艾尔的防御力提升20%。
- "勤劳的女仆"：元素爆发持续时间内，诺艾尔的普通攻击和重击造成岩元素伤害。
- "守护的决心"：护盾破裂时，恢复诺艾尔20%的生命值。
