# 陨龙之梦

在山的深处沉睡着一只比群山更古老的巨龙，在他漫长的沉眠中，他梦到了马车与月宫。在他的美梦中，龙王又一次看到了他曾经跟随过的那位伙伴。“如果我们骑在你的背上越过群山是不是就可以到达月宫呢？”这就是龙王的梦一在他不为人知的沉眠中。有时候，他会在痛苦中醒来，想起了那天他说的话，想起了那天他惹怒了那个人，他们从此走上了相反的道路：“我与你们一起为这片土地带来了和平，但我以后将撕裂群山，让群山像龙牙一样锋利，它们将摧毁你建立的一切，将这座城市化为废墟。”
为什么他会说这样的话？他明明爱着这片与那个人共享着美好回忆的土地；他明明爱着那些聚集在他们羽翼下小小的人类。为什么事情会变成这样呢？有时，被困住的龙王会陷入这样的思考。但是这种想法很快就消散了，因为他想起了故事的结局：那刺穿他身体的灼热的箭，那脖子被牢牢卡住时的室息，那让他心如刀绞的，来自金色眼眸的冷漠、锋利的视线[110]。若陀龙王为最后的仙兽[4]。
盛露厅盛露厅前身为层岩巨渊中以开采琉璃晶砂，制造和贸易瓷器为生的部族，归化后曾在镇压若陀龙王时立下战功。
