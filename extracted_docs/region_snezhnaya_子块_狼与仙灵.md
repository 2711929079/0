# 狼与仙灵

背叛的使者被剥去名讳与形骸，而她的同族自此亦被烙下诅咒：若是胆敢再凝视他人的眼瞳，将本应属于众生的怜爱献予一人，那得赐于高天的瑰美形体便要溃散成风，心智也要殒落于尘中，直至那堕落者的残躯蜷成仙灵，在永世徘徊中吞咽记忆的残影。[1]受罚的天使居于殿宇中，有许多微小的灵在她周围行走，形状缺短、光芒凄凉。她不认识它们，但心中却无比哀伤。天使忘却了自己的名和所做的事，心中徒有悲哀苦楚，只得以弹琴歌唱排解。但有一事常警醒着她：“你当等待在此地，不可越过这殿的门槛，也不可去往外边”。野狼自北方而来，闯进天使的居所。天使见它听话可爱，便为它取名玻瑞亚斯（意为「北风」），并叫它和它的宗族与自己同住（可参考民间传说醉客轶事#醉客轶事·第三卷）。日久，玻瑞亚斯开启灵智、吐出人言，称天使为主母。它携宗族向天使辞行，要向南方（蒙德）去——那地的民贫穷卑微，城邑却广大坚固，它要叫他们得着富足。（玻瑞亚斯便是日后魔神战争时期的北风魔神安德留斯，“开启灵智吐出人言”或许是天使所启发，又或许正是成为魔神的表现。）（后续见#玻瑞亚斯回到挪德卡莱）最后一次念起玻瑞亚斯的名字时，天使终于记起了自己曾经的国度，亥珀波瑞亚。「我要回到我的国，等待我镣铐在星外的星；在灭亡的空无中歌唱，胜过在虚假的天光下为奴。我心所爱的啊，你为我筑起这苍白的殿宇，我却要违抗你的命令，回到我们不存在的国土，等待你迎来自由的那一日，正如你也同样赐予了我自由…无论要过去多少岁月，无论晨星是否熄灭[18][19]…」穿过古老宫阙的废墟，在荒原的尽头是安瓦蒂尼尔湖畔静谧的圣林，据说能够折断林中金枝的人，便能寻找到白日不可见的仙灵的王廷。那是掩藏在天光无法照临的暗影之域，被时与空所遗忘的往日国度，曾经的主人早已离去，只留下久疏打理的庭院，与铅华剥尽的厅堂。然而本应无人的宫殿里，却不知何故，传出了某位少女无声的哀歌。最后的歌者，最初的仙灵，弹奏着终曲，坐在天使的厅堂。一度闯入这里的狼早已离去，只留下未竟的梦追逐着幻影[20]。（现在）西风骑士团在安瓦蒂尼尔湖附近驻扎，原因之一便是调查此处有关狼主安德留斯的痕迹[21]。旅行者为仙灵们修复了庭院，见证了数千年前关于狼与仙灵的故事[18]。
