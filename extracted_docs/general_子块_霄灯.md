# 霄灯

由浮生石片、灯芯绒材、灯纸纤维制作而成。在海灯节的夜里，人们会放出「霄灯」和「明霄灯」，升上夜空。节日里，人们手中放飞的霄灯叫做「霄灯」。而全璃月人共同建造的巨灯，就是「明霄灯」。市民们在为「明霄灯」的建设捐赠「浮生石」时不少人会把自己的愿望刻在「浮生石」上，以此祈福。那些灯的寓意,是「愿薪火相传，美德不灭」。璃月的人们相信夜海的明灯会引导过往的英魂归乡。霄灯由「逐月」仪式的月灯演变而来，魔神争战，天下大乱，人们为替前方厮杀的战士英灵指引家的方向，便会点起如此灯火。
