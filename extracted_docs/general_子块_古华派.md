# 古华派

那古华团门庭凋落，便成了古华堂，古华堂又日落西山,于是成了今日的古华派。数百年来，武学门派「古华派」式微。辉煌时期，「古华派」曾以枪剑闻名璃月。相传，其有三大镇派秘术——枪法「刺明法」、剑法「裁雨法」，以及枪剑双绝的「生克法」。此三绝之术随代代传承不断精进，时光推移，三绝技巧渐臻化境，威力却不断削弱，导致古华派势力低迷，人心流散…三绝更是如尘埃中的宝库，静候一位继承者。
