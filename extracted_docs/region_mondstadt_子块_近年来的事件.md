# 近年来的事件

琴加入西风骑士团，成为一支小队的队长[143]。芭芭拉加入西风教会，成为祈礼牧师。刚成为牧师后不久，在一次照顾高烧不退的孩子后获得神之眼。艾莉丝送给芭芭拉一本偶像杂志，芭芭拉因此开始学习唱歌和跳舞[144]。迪奧娜加入猫尾酒馆。罗莎莉亚所在的盗贼团被法尔伽带领的西风骑士团剿灭，法尔伽将罗莎莉亚带回蒙德，并让其加入了西风教会[123]。15岁的琴获授“蒲公英骑士”之名，并成为骑士团副团长[143]。艾莉丝在蒙德地区游历，编写《提瓦特游览指南》的蒙德部分[145]。
