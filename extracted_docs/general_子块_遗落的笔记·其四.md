# 遗落的笔记·其四

某月某日…惨烈的战斗…拒绝了当地人的援助，守着这座灯塔孤军作战，终于只剩下我一人…啊，还有这只跟随着我们一起从蒙德出发的狼…大人循着魔物的踪迹追猎而去，我不知道他是否还会回来…但无论如何，满身血污的我已无法跟上他的脚步了…这条骑士道的终点，唯有他自己，和这些循着血腥味而来的漫天鸦群可以见证了…我可不想把身体留给它们…来自故乡的狼啊，就当是送我最后一程吧…
