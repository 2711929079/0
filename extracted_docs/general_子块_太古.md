# 太古

原初原初之人遥远时空的故事里。巨人盘古的血液化作江河。普鲁沙的身体被切割滋养了宇宙众生。伊米尔的脑髓成为天空云海[1]。记述者与始源之龙尼伯龙根与提瓦特一同诞生，成为主宰大地的原初生命，彼时世间尚无生死界线，意识在无数存在中流转[2]。不朽身躯沉眠于遥远的构装天盘、灵智在虚空中流荡，测绘众多文明、沉思星海终极的答案的记述者，注意到某个不起眼的小世界，以及伴随世界诞生、被职责禁锢在行星地表的始源之龙（尼伯龙根）。本不应与原始文明进行交流的她展现了谕告，邀请始源之龙将注定毁灭的世界弃置于此，与其一同踏上旅途。龙拒绝了邀请，发誓将引领众生一同行往群星，并与旅者立下重逢的誓言。[3]空月升起企图追寻世外的始源之龙用指爪细捻白垠的游空之土，升起空月，以代其施行周流运作的权能包括生死时空，自己则前往世界之外。光焰的主人（火龙王）曾率部下造访居月的意志们在三个月面的玉座，那时王位已然空荡，天穹之上投下阴霾，各龙主任意而行。光焰的主人曾希冀从她们那求得踏入星间迷雾的王的踪迹，但答复他的，唯有三座宫殿之内，化作了三重的沉默不语。自那以后，即使是万里的雪原化作焦土，焰山被极寒冰封，她们也未曾再开口，或不能。[4][5]。王座到来王座到来提瓦特是一个星球，原始胎海孕育了这颗星球最初的生命[6]。最初，水龙王是原始胎海的心脏[7]。龙族曾是提瓦特最早诞生的生命，其中的七位龙王与其共主尼伯龙根统治着古代提瓦特。龙王包括水龙王那维莱特，火龙王修库特尔、草龙王阿佩普等（那维莱特是初代水龙的转世，初代水龙殒落后以人形重临，就是现在的那维莱特）。在鸽子衔枝之年，天上的永恒王座自世界外降临，第一王座——“原初之人”法涅斯从蛋中出生，割裂出「原初四影」作为最初的执政[8]。法涅斯向七龙王开战。经过40年的惨烈战争，七位龙王们被打败，而此时尼伯龙根因不明原因离开提瓦特去世界外旅行没有参战，法涅斯成为了新的统治者，并制造了人类，剥夺了七位龙王的一部分力量。破镜为那场战斗中法涅斯留下的武器碎片，曾有熄灭月光的力量[9]。其原由尼伯龙根所造，却被法涅斯盗取[10]。七龙王全向法涅斯俯首称臣。法涅斯剥夺七龙王的一部分力量，最初的胎海之心水龙王陨落，龙在枫丹地区的统治崩溃。原初的那一位大王开始了對天地的改造，将其改造成后来诞生的人类宜居的样子。原始的蛮荒之力与燃素，被改造成如今的「元素力」，龙族宜居的蛮荒的大地开始能种植粮食。无相元素是天空岛神系侵入改造元素力后诞生的新生命[11]。部分战败龙族逃亡暗之外海，而部分龙族如草龙阿佩普一直等待尼伯龙根归来。后来部分龙族选择与人共存，也有一部分等待着龙王的归来，等待复仇的一天。三月女神的力量自然无法与天理对抗，但天理也没有将她们视为必须铲除的异端，本质上三月女神维系看这颗星球的存续与运转，与天理的目的并无冲突[12]。此事被龙族视为三月背叛龙族，之后法涅斯透过三月打造了神之眼的前身「月之轮」，与神之眼不同的是它使用燃素，发放给人类用来让人们联系月神，与神之眼相同获取后就是卖身于命运。山川与河流落成，此时的山比天穹还高，大地比天空更广大[13]。不服从的龙群逃亡去大海和大洋并各自演化，这就是渊下宫里深海龙蜥的由来。原初的那一位和一位影子制造出了飞鸟、走兽和水鱼。它们还一起制造出了花草和树木。法涅斯依靠自己的记忆打造出世界与其规则[14]。最后它们造出了数目不可知晓的人。自此时起，人的先祖和原初的那一位立约。法涅斯对人有一套神圣的规划。人只要幸福，它便欢欣。[8]在这之后，部分龙族选择与人共存，也有一部分等待着龙王的归来，等待复仇的一天[15]随着第一王座的降临，水龙王及龙在枫丹地区的统治崩溃，水龙王同时陨落。胎海的新心脏水龙陨落后，生之执政在原初的大海里创造了另一颗心脏，名为厄歌莉娅。[7]文明制度七位龙王全部被打败，七个王国全部对天上俯首称臣，此刻天之王座开始了对这片天地的创造。山川与河流落成，大海和大洋接纳了部分对天之王座的反抗者，这就是渊下宫里深海龙蜥的由来。在这之后的又几百年里，自然万物与人类诞生。人们耕耘，第一次收获。人们开掘，第一次收获贵金。人们聚集，第一次写就诗歌。如果有饥馑，天上就落下食物与甘霖。如果有贫瘠，那大地就会生出矿藏。如果有忧郁蔓延，那么高天就会以声音回应。唯一的禁止之事，就是引入禁忌知识。但是禁忌知识的通道已经被封堵。人们生活在这片土地上安居乐业，整个世界都在类似的文化之下。[8]此时大陆上有许多文明，广义上都算第一文明的分支。这些文明大多都拥有相同文字、崇拜天空岛与其使者、使用地脉的力量等特征。它们的遗迹风格相似，提瓦特各处都可见的元素方碑、传送锚点、浪船锚点便是当时文明使用的机关。地上各处都曾有人竖起几乎能触达天顶的高塔[16]。法涅斯为第一文明建立了一个制度，第一文明需要信仰天空岛，人们能通过天使得到
