# 七七技能详细介绍

七七是《原神》中的冰元素五星角色，璃月往生堂的僵尸少女，称号"冻冻回魂夜"。
元素战技「冰寒轮舞」：
- 释放冰元素能量，对周围敌人造成冰元素伤害并恢复队友生命值。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「寒病鬼差」：
- 创造一个持续12秒的冰元素领域，领域内的队友会持续恢复生命值。
- 冷却时间：20秒。
- 能量消耗：60点。
固有天赋：
- "不卜庐的药童"：元素战技和元素爆发的治疗效果提升15%。
- "冰元素增幅"：七七的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "冰寒的祝福"：元素爆发持续时间内，队伍中的角色获得"冰元素伤害提升10%"的加成。
