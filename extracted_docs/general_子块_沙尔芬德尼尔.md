# 沙尔芬德尼尔

蒙德大地被冰雪覆盖，生存条件极其恶劣。一位名叫法鲁希的男性带领族人来到了当时的雪山地区，出乎意料的是，这里并没有被风雪覆盖，而是保持着自然的苍绿葱翠，于是他们在此建立了王城沙尔·芬德尼尔[32]。在冻结雪山的「长钉」降下时，风雪遍天。天降之柱崩裂为三。当冻结世界的长钉蓦然降下，连这棵树也被其余波粉碎、冻结时，芬德尼尔的公主取走了最完整的枝，想为荫蔽一国的树接续生命。但在最后，嫁接的生命始终没能活下来。刀刃般冰冷的风雪，最终将月光遮蔽了…[33]
