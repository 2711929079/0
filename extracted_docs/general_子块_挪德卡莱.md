# 挪德卡莱

你来到挪德卡莱，挫败猎月人的复活计划，途中发现自己真正的血亲从未离开飞船。此外，飞船似乎被不明人士隐藏起来[64]。因为提瓦特的命运被注定，只有身为降临者的旅行者能影响，因此「博士」想夺取月髓也因为天理制定的命运不可能成功。但在旅行者插手此事后，一直深入干涉月髓的命运，将其影响，在如「博士」计划下，旅行者的所有行动都形成构成蝴蝶效应的「变数」层层堆砌，最终打破了命运。为了堆砌足够的变数，也为了让旅行者一行人能顺利找到奔月的仪式的遗迹已执行下一步计划，他扰动提瓦特边界引来魔女会，而魔女会以为这是深渊影响，还和旅行者见了面，影响旅行者的行动。而他深知被世界逐渐排斥的「少女」在猎月人一战后肯定即将消失，只得奔月，而奔月的仪式需要摆放两个月髓，于是「博士」制造出月矩力浓度高，比月亮更像月亮的装置让人在地下秘境只凭月矩力误以为那里就是月亮，「少女」中计透过奔月的仪式传送到「博士」的囚笼中，因为力量不够无法破除囚笼，接着「博士」将两个月髓用月矩力从仪式传送门也吸引过来。如今「博士」已经用一连串计谋集齐了虹月与恒月的月髓，获得了三月生死时空的权能，能够暂停时间，只有不受法则拘束的降临者能不被影响，为了是超越自己命运束缚，达成不被世界允许的实验。「少女」为了不让自己的力量为「博士」所用跳入月之门消散生死未卜，「博士」失算但也已获得二个月髓的力量。在夺取月髓后「博士」凝滞时间。让来追杀他的旅行者一行人冻结，但不受提瓦特法则影响的旅行者杀了上去，但「博士」的实力碾压旅行者，且「博士」以计划好，先前奔月仪式之需旅行者不吃不喝了三天后暴饮暴食在忙了这么久后在祈月之夜上过节也没有休息会影响自己身体，被「博士」轻易击败。如今挪德卡莱所有人齐聚对抗「博士」，并打算透过旅行者触发的变数打造利于自己的命运[65]最终击败了「博士」，发现隐藏飞船的是天理，飞船曾在很久以前略过提瓦特[66]。
