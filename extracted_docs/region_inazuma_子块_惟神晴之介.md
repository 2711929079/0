# 惟神晴之介

天狗族长认为自己没有尽职尽责，引咎而去，其女光代紧急替代位置。远在璃月修习仙法的晴之介见深渊灾厄侵袭璃月，便担忧稻妻的近况，于是回到稻妻，得知了这个惊人的消息。他拿出了狐斋宫临走前赠予他的烟斗，一时间不敢相信。那个时候，五百藏终于察觉到不对劲了，便出来找狐斋宫，大闹镇守之森。动静太大，惊扰了幕府，便请晴之介前去平息，于是五百藏和妖狸都被封印了起来。天狗族长归隐，狐斋宫也死了，浅濑响等的那个人再也回不来了，她没有理由待在这里了，便回到了清籁岛。狐斋宫虽然死了，但是过于思念这片土地，最终其强大的思念汇聚成了花散里。花散里遵循着狐斋宫的遗志，依然进行着神樱大祓的相关工作。晴之介在供奉狐斋宫的庙宇里处理后事，见了花散里，以为狐斋宫还活着，结果不是。这让他更加思念狐斋宫了，他决定要让狐斋宫放心得离开，便筹备阴阳寮相关的事。
光代继任天狗族长后，私下里也会继续教御舆道启剑艺，且与其对练。[49]
诀箓阴阳寮惟神晴之介为锻炼能够对抗魔物的武士，利用式法创立了诀箓阴阳寮和寮司式大将，在此中不断修行。[32]不知为何晴之介的符法流落到贼寇手中，成为海乱鬼为武器附魔的手段。[53]
