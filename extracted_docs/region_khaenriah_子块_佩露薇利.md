# 佩露薇利

赤月王朝遗民佩露薇利被壁炉之家收养，现成为愚人众执行官第四席[76]。
深渊教团的行动殿下最近，[77]旅行者的血亲继承[1]深渊教团，成为教团成员口中的公主/王子殿下，即深渊教团的统领者。如今这位殿下统领深渊教团的终极目标，包括反抗天理、淹没神座以及复国。此外除了内疚，她执行这些目标时别有原因，其中包括打算「利用整个世界，换取与天理平等对话的资格（即自己因为被降格了，所以成立深渊教团用一整个世界与天理对抗打算获取与之平等对话的资格恢复降临者身分）」，和让新坎瑞亚成为自己合适的新的家乡[28]。同时，戴因为了阻止曾经的旅伴踏上征讨深渊教团的征程[4][78][79]，途中有遇过丝柯克，并质问其剑术是跟谁学的[69]。
