# 风神与魔龙

特瓦林在毒血与深渊法师的影响下迷失、堕落。再度觉醒时，没有听到好友的琴声，而是深渊法师的低语。它诅咒自己的宿主，没有在最关键的时刻陪伴到自己身边。特瓦林逐渐化为魔龙而不再是曾今的风龙。甚至沦落到了被骑士团围剿的地步。你与风神温迪、骑士团团长琴、迪卢克等人一同前往龙巢净化特瓦林，阻止他继续狂暴。
神之心解决龙灾后回到蒙德城，风神的神之心在此时被愚人众十一执行官之一「女士」夺去你从温迪口中得知，「神之心」是神明的外置魔力器官，而至冬国的冰之神因不明原因正带领十一执行官搜集神之心。你得知神之眼是凡人的魔力器官，是神明给予凡人的认可，持有神之眼者等同于有登上天空岛成神的资格。之后你前往璃月，为了从下一位神明口中问出血亲的线索参加请仙典仪。
