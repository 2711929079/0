# 流沙无名古国

曾有一个无名的古国，是第一文明支系。古国的愚王在万古的流沙之上梦想筑造供养银白古树的高庭[7]。
樱之庭遥远的古代，高山比天空更高，大地比穹顶更大。直到葬火之战明镜（恒月）破碎，海洋升起，稻妻的低洼处被淹没，远古的岛屿樱之庭与其他岛屿的联系因此断绝了。[8]
