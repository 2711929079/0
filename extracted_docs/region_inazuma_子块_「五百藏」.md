# 「五百藏」

当天地变得漆黑时，「狐斋宫」担心五百藏为了守护森林，会挺身与绝对无法抗衡的强敌战斗，并因此死去。于是，就故意用妖狸擅长的「捉迷藏」挑战他…「五百藏，一定一定…无论发生什么事，都是我引诱你的法术。千万不要被骗、不要现身。」「那当然，臭狐狸。这次吾辈一定会胜过你，让你输得心服口服。」等了一段时间后，五百藏感觉不对，就到处大闹。为了引出「狐斋宫」，甚至把镇物给偷走了。他不知道这是「神樱大祓」需要的物什，只是以为这把梳子对于「狐斋宫」而言很重要。后来，就因此被惟神封在了石头中。[1]
雷樱树据记载，几百年前，雷樱差一点都枯萎了。当时魔物肆虐。连大海也变成了黑色。雷樱很努力、很努力地吸收邪秽的气息，最后几乎全都枯死了。[1]
