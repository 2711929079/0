# 雪中君

祖祖辈辈都生活在雪山地区的珍稀棘鱼。或许正因如此，其性格意外的冷静，远不如同类那样凶狠暴躁。这种棘鱼对寒冷有着非同寻常的依恋，像父母之于孩子。虽然它们的确能在温暖水域存活，但远离寒冷似乎会让它们遭受某种心理上的重创，进而郁郁寡欢，停止繁衍。「不建议捕捉它们。」——《野钓手册·雪中君篇》
