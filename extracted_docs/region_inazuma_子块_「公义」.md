# 「公义」

幕府军武士研次和公义自幼一起长大，一同参军并遇到了照顾有加的前辈战友松平。一段时间后，研次和公义纷纷晋升为旗本。在一次战役中，公义捡到了那位前辈的书信，得知前辈加入反抗军。公义因此对信念产生了怀疑，准备叛逃到珊瑚宫军下。研次因此而怒。后一场战役中，公义和研次战场相遇却不得不拔刀相向，公义战死，研次因好友在自己面前死去而悲伤逃离，不久后发疯失忆，流浪在踏鞴砂，只记得⌈公义⌋这个名字。实际上，踏鞴砂自称「公义」的正是失忆的研次，可与在九条阵屋的松平对话确认。
