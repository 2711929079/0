# 花神诞日

传说,之所以这个日子会叫做「花神诞日」,其实最早是「花神祝诞」的意思。在很久很久以前，有一次树王大人过生日,她的朋友们举办了宴席为她祝寿。宴席上,几位神明大人都喝醉了，其中一位便乘兴弹奏起了乐器，于是树王大人唱歌，花神大人就跳起舞来。在花神大人起舞时,她踏过的草地上，长出了无数美丽的帕蒂沙兰…明亮的紫红色花朵筑成了灿烂的舞台,神明们都说:「啊，若是时间能永驻此刻就好了。」其实这是个很古老的节日，最早是为大慈树王大人祝寿的。树王大人逝去后它被沿用下来，时间就改到了小吉祥草王大人的生日。[20]
雷穆斯名为雷穆斯的神祇曾是一位绿洲的歌者，在赤王的威光下丧失故国。因为不服赤王的统治而流亡到枫丹。[21]
