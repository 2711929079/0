# 式大将

式大将是惟神晴之介所制造的纸人式神，同时也是「诀箓阴阳寮」的寮司。曾被晴之介仿造出大量拓本作为量产式神投入使用。彼时在「诀箓阴阳寮」培育武士需要消耗大量的精力和拓本，而雷电将军又在那个时代以更强的力量消灭了妖魔们，「诀箓阴阳寮」几乎失去了存在的意义，式大将在晴之介落寞时开导了晴之介，告诉他“意义不该由他人赋予”，使得晴之介真正释怀，与式大将道别后离开了「诀箓阴阳寮」。斩杀的对象不复存在，需守护之人已求得平安。惟神晴之介再次启程环游世界去寻找新的意义。[1]式大将盼望着晴之介归来，因此一直在「诀箓阴阳寮」内守候，直到与「诀箓阴阳寮」一同陷入沉睡并在这个过程中失去记忆[32]。
灵善坊曾接待过「影向役者三人众」的千门虚舟成为引退的天狗族长灵善坊灰心之后虚度余生的樊笼，沉入了海下[26]。惟神晴之介环游世界途中来到千门虚舟，发现老师灵善坊在此度过余生，并且发现狐斋宫来过了[54]。（狐斋宫曾带领人们躲进该地的地宫[3]）
