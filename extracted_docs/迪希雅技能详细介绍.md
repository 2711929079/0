# 迪希雅技能详细介绍

迪希雅是《原神》中的火元素五星角色，须弥的佣兵，称号"炽鬃之狮"。
元素战技「炎狮踢」：
- 释放火元素能量，对周围敌人造成火元素伤害并施加"火元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「炎狮狂舞」：
- 释放强大的火元素能量，对周围敌人造成火元素伤害，并进入"狂怒"状态。
- 冷却时间：20秒。
- 能量消耗：80点。
固有天赋：
- "炎狮的祝福"：炎狮踢命中敌人后，有15%概率触发"燃烧"效果。
- "火元素增幅"：迪希雅的普通攻击和重击造成暴击时，恢复3点元素能量。
- "狂舞的威严"：元素爆发持续时间内，迪希雅的火元素伤害提升20%。
