# 梦境的消逝

教令院改造了虚空终端，让其可以盗取须弥人的梦境，借此来增强虚空的算力。「须弥人不会做梦」的传言由此产生，并被认为是智慧和理性的象征。[20]由于兰那罗的能力与梦境息息相关，因此须弥成年人无法再看见，也无法再记得兰那罗。[30]人类与兰那罗渐行渐远，最后兰那罗成为童话故事。
那伽朱那团又名「萨曼尼耶」，教令院称「那伽朱那团」，创立者那伽朱那。
曾经是归属于教令院的学派团体，曾留守沙漠、对抗深渊魔物，后与教令院产生分歧。如今早已凋敝。驻扎地位于荒石沧漠。
信仰神鸟「西摩格」牺牲后化为的「灵光」之力。[72]
