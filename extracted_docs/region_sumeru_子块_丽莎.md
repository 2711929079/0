# 丽莎

丽莎在须弥求学时被称为二百年不遇的天才学者，后来产生研究时若有神之眼很方便的念头时获得神之眼。丽莎发现使用神之眼与知识的代价会使自己折寿，于是离开教令院前往西风骑士团图书馆。原本被法尔伽邀请当小队长，但被拒绝。第八小队队长尼姆芙不服刚回蒙德的丽莎为何能被团长亲自引荐，与是和她展开魔法对决打成平手，尼姆芙也因此对丽莎感到佩服[94]。
