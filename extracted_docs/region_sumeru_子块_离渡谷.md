# 离渡谷

在须弥古时的教典中，「离渡」象征着追寻智慧之人唯有渡过无知无明的苦海，才能最终顿见彼岸的正理之光。如今穿越雨林之人用它来形容这片圣树脚下的谷地，或许是为了表达自己终于抵达智彗之城的喜悦心情吧。
