# 炉灶之魔神

「炉灶之魔神」马科修斯诞生于石块碰撞击出的火星。他司掌炉火与民生，是一位喜爱人类的魔神。千年前古人扩建城市，到平原上建立聚落，取名「归离集」。灶神惜护民生，化出分身潜入千家万户帮着生火做饭，助人团圆[50][44]。
浮锦、药君与灵渊一条鲤鱼修炼成仙人浮锦，与同样被尊称为仙的白蛇药君和猊兽山主灵渊为好友，三仙共同服侍一位魔神，与此同时也和萍姥姥等仙人有过来往。浮锦并非仙力强大的仙人，无法长期离开清水活动，因此萍姥姥赠给了她一盏茶壶，相约由其友人用茶壶装着她去璃月港游玩，可惜到最后都未能实现。浮锦和药君与各种各样的人接触过后，逐渐对人类产生好感。药君化成人形，帮人们治病，浮锦在灵渊的山头上种下茶树，并将种茶技艺分享给人类。灵渊虽然无法理解两位好友对人类的喜爱，但出于和友人的情谊愿意尊重她们的选择。[24]灵渊也化为人形与人类生活，还因吃了过多的茶糕，担心自己长胖而求药君制作可以减肥的茶叶。最终在灵渊和浮锦的共同请求下，药君制作了这种茶叶[51][52]。药君在伏仙洞种下茶树，并赋予一些机关与仙像生命[53][54]。玄文兽在遥远的过去曾是灵渊的眷属[55][24][56]。
