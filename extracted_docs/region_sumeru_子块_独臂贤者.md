# 独臂贤者

东方来的那伽朱那团的独臂贤者在对抗漆黑兽潮后前往地下考察坎瑞亚遗迹，遇见了白鸽骑士当中的海杜菈。
遗迹巨像在坎瑞亚灾厄时期，坎瑞亚派遣了被后世称作「遗迹巨像」的巨型战斗机械人清剿漆黑兽潮，但行动失败，三台遗迹巨像残留于须弥。
此后，教令院将遗迹巨像的炉心封印。降诸魔山的遗迹巨像炉心被封印在迪弗旧窟。[30]
