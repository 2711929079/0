# 盗火者

（记载有关盗火者事迹的书籍「哈南帕查记事」的简介中提到，经专业的历史学者考证，其内容与真实的历史毫无关系。[36]）因为火龙王修库特尔被深渊意识侵蚀，它开启了对龙群的暴虐的统治来维系权威。后来修库特尔因深渊力量只得再次假死（类似冬眠），龙众因执迷于守候修库特尔的躯体而日渐衰败。被深渊侵蚀的火龙王陷入有时清醒有时失智的状态，时日不多的他于是制定维奇洛波奇特利计划。对于修库特尔的计划，龙裔们无不盛赞于王的智慧和伟大，歌颂着王为族裔打造的未来之路，至高领主们亦无一反对，全票赞成计划启动的决议。
在深渊侵蚀的症状日渐加深、能够保持清明的时间越来越短暂后，修库特尔便亲自授予了库库尔坎统领圣城的摄政大权，可以说是给予了这位自己心爱的弟弟无比的信任。为了控制维奇洛波奇特利，修库特尔改造了自己的一枚瞳孔来作为控制中枢。然而在库库尔坎眼中，兄长为族裔所规划的道路实在是过于可怕和狭窄，那时的他认为，这种计划无异于一场豪赌，若失败等同于让整个族裔成为兄长葬礼的陪葬；即使获得了成功，新世界中诞生的所谓「生命」也失去了生命本该具有的意义，所有生命的伪像不过是巨大火炬上飘渺的焰影，失去实体，终止了进化，抹消了可能性，这种结局和「永恒的死亡」又有什么分别呢？库库尔坎对此计划颇有微词，两兄弟不欢而散，但他也秦驴技穷了，于是一边尝试是否能将人龙血脉融合一边执行该计划[1]。修库特尔休眠时由库库尔坎摄政。在龙群之中这位贤人瓦萨克拉胡巴肯，他预见了未来人类将击败火龙王，于是从烬城盗取火种交给纳塔初民，教导人们使用燃素。[15]当时天空岛派遣来引导人类的天使都被龙吃掉了，[37][29]这也是纳塔除了在夜神之国外都没有仙灵的原因，因为仙灵就是受诅咒退化的天使。[38]当时人们没有使者（天使）指导，被龙群逼退到山洞中，贤人看到后对人类顽强的生存技巧十分惊讶。[37]盗火者将偷出的圣火给人类，教他们生存之道。他用燃素创造「浮土静界」哈南帕查（就是流泉之众上的空岛）作为试炼，让第一个登上空岛的人学会使用燃素的语言。修库特尔同意此计划，它太过温柔，所有疯狂计划都会同意弟控（不是）[39]。后来浮土静界被盗火贤者隐去[40]。当然盗火者身为龙族，并非单纯地是想帮助人类，而是想让人龙两族血脉合一，并达成人龙共治。燃素被盗火者拿去给人类遭到研究燃素的至高领主第四席（后来的阿乔）的埋怨，认为这亵渎了自己的研究。第八席让他吃青蜜莓冷静下来。因为第八席对人类心存善意，他原以为燃素是第八席给人类的，后来在向第八席抱怨时察觉到真正主谋与盗火者的计划。最初接受圣火的人名为察阿克，他带领部族除掉了部族附近的恶龙。之后他不敌岁月摧残而死，死前受到了夜神之国的神祇招唤，成为第一个大灵。[15]察阿克曾透过燃素制造浮岛「虹灵的净土」，在其中将燃素使用方式传授给其他人[41]。很久以前大灵显化身的造型与如今也有很大的差别，是更粗糙，更接近黑曜石像的姿态。可是随着织卷记录下了大灵显化身的轮廓后，就很少再有变化了[42]。传说盗火者知晓人世间一切问题的答案，但却有两件事情就连他也无法解答，那便是生者的末路与亡者的归途。因为掌管冥府（夜神之国）的乃是夜域的神祇（夜神），她在夜神仍是天使时是盗火者的大敌，倘若他偷出的火种重新连通夜神之国，那夜域的神祇就可能会败坏他久虑的计划。[43]圣城内的一部分龙族也同样不认可火龙王修库特尔的计划，于是库库尔坎为他们找了一块在圣山之外的实验地，来进行“燃素状态转换器”的研究，这项发明可以将燃素转化为坚固的结构，从而搭建对抗深渊的防御工事[44][45]。
玛阿威纳塔曾是高傲的巨龙们自由驰骋的家园。在那时，大地上还没有人类的诸部族，龙也远比现在更为高大，更高傲。就像是现在的人类那样，龙们自己也分成了不同的部族，各自由古老的族母统治。那时，大地上还没有热腾腾的温泉，也没有像宝石一样平静晶莹的湖泊，在那时，在这巨龙漫游的赤色大地上，曾有过一只与众不同的幼龙。高傲的龙向来蔑视其余的生灵，无论是来自高天的使者，还是渺小的凡人，古老的战争没能抹去它们的尊荣，恰恰相反，苦难的火锻出了它们的坚忍与骄傲。然而这只幼龙却不同。它沉醉于聆听幻写灵的歌声，悄悄观察人类的行径。它的名字叫做玛阿威，在如今已经无人知晓的古老语言中意为纽带。年幼的玛阿威从未经历过古老的战争年代，也并不懂得龙族的德行。对于它而言，凉爽的月夜与红色山崖温暖的阴影，才是更值得享受的美景。「圣龙的宫殿中怎能容下这般怠惰？稚嫩的幼崽必须要寻回它高贵的野心与怒火」龙的族母如此断言，责令年幼的玛阿威离开部族，像「真正的龙」般成长，让内心变得刚硬，像黑曜石般坚忍骄傲，以追逐残酷而高远的野心…就这样，玛阿威走向了那无垠的赤色荒原。就像真正的龙那样，它要经历烈日与荒芜的试炼，寻找高贵的野心与怒火，培养高傲的心性与德行。昔日陪伴着它享受天真梦境的幻写灵也追随其后，与并不孤独的玛阿威一同踏上了寻找怒火的道路。玛阿威向北方漫步，去往那无人踏足的大地终极，与伙伴一同来到了高踞王座上的盗火贤龙面前。「年幼的孩子啊，你来到我的面前，有何目的？」温和的贤龙这样询问玛阿威，它的心中无比惊奇。「我本以为同天空的争战已令我的同族心肠刚硬，为仇怨所充塞，令它们无法再接受我给出的建议，但没想到今天，却有如此年轻的同族来到了这里。请告诉我，你想要什么？我或许可以破例满足你。」于是，玛阿威将自己的经历原原本本地向贤龙分享，玛阿威述说自己昔日珍视的幻梦，梦中凉爽的月夜，玛阿威述说红色山崖温暖的阴影，与幻写灵的歌声。玛阿威述说自己被族母逐出部族，寻找难寻的怒火。「年幼的孩子啊——你是如此天真，却也如此狂妄，你来到我的面前，却想要索取我也无法得到的东西。离开这里吧。你试图寻找的怒火，我这里是没有的。」贤龙关上了它的黑曜石大门，不再回答玛阿威的话。就这样，玛阿威失望而归。下山时，一直陪伴着它的幻写灵也因为远离燃素丰盈的土地而失去了光与温度，从而变成了灰烬般的苍白色，歌声也越来越遥远，越来越模糊了。玛阿威失落又伤心，急切地想要寻找拯救伙伴的方法。然而贤龙的黑曜石大门已经为它紧闭，不肯再看自己年幼的同族一眼。「…将我散落在硫磺的池中，将我埋葬在冰冷的潭里…」玛阿威听到了幻写灵的声音。幻写灵是不会说话的，但玛阿威并不知道这一点。于是，它决定满足伙伴的愿望。不知漫步了多久，不知识破了多少蜃景的诱骗，不知经历了多少命悬一线的危途，玛阿威来到了遍布硫磺池的热泉之地，在此时，这片地域尚且不适合生物栖息，即使是靠近也会被硫磺毒气灼伤，被滚烫的酸水吞噬。然而神奇的是，随着玛阿威将伙伴的身体撒入硫磺池中，滚烫的酸水便化作了清泉。就这样，年幼的玛阿威创造了纳塔大地上最早的温泉。玛阿威在温泉之地定居下来，四肢利爪化为流线型的美丽鱼鳍，干瘦的躯体变得圆润，成为了最初的鳍游龙。当然，玛阿威并没有忘记它的伙伴，不，它怎能忘记随它一起冒险的好朋友呢？玛阿威将它所听到的、所学会的歌，全部传授给了清泉中新生的人类。没错，这些人类正是流泉之众的祖先。而梅兹特利部族所传承的歌，正是源自那个古老年代的遗响。[46]
