# 丹羽一族滅絕

丹羽一族疑似因丹羽久秀逃跑受罚（实则被博士杀害，伪造为逃跑），约三百年后，丹羽家主失踪（被散兵所害），只留一子丹羽义庆于世，最后由枫原家收养，改名枫原义庆。
雷电五传没落在御神刀委托事件中，主管刀剑事务等传统工艺的社奉行家主─神里，委托一心流进行锻刀事务，却因散兵涂改锻刀图纸而锻刀失败，刀匠赤目兼长等人受愚人众谎骗畏罪潜逃，而神里家主与尚年轻的义庆一同追逐刀匠行迹来到海边，却遭遇倾奇者袭击，神里家主受重伤，而义庆也受伤，所幸致命一击批在斗笠上才没有毙命。倾奇者凑到义庆身旁准备下手时看到他的脸，发现他与丹羽久秀样貌相似，于是质问他与丹羽有和瓜葛，得知此人为丹羽后人，便放过此行人。散兵放过一行人和灭绝雷电五传的原因各执其词，一说是因为恨丹羽，但无法解释见到丹羽后人停手，一说是恨踏鞴砂的人，但无法解释不对御舆长正后人出手。或是只是愚人众任务，或是只是因为恨刀匠，其中种种。临走前告知义庆转告雷电影他是国崩，来对雷电影宣战。此后散兵一直以对雷电影的恨与夺得他认为属于自己的雷神之心来填补自己内心空洞为支持自己的动力。此时被散兵放过的五传刀匠仅剩天目流、一心流的赤目一家（在至冬继续锻刀）和枫原一家。之后在散兵被属下问道为何覆灭五传，又为何突然收手时，他总是模仿「博士」常挂在嘴边的「这只是关于人性的小心实验」瞒过去[65]。散兵通过一连串手段使雷电五传没落，连执掌社奉行的神里家，也因督导下属不力而备受牵连。将军降下最终裁决之前，久未涉足政事的宫司大人突然向将军进言，赶在风雨飘摇之际保下了神里家。由此，神里家虽元气大伤，却终是免去罢黜之虞。
随后数年，外界对宫司此举多有猜测：一说社奉行与鸣神大社关系密切，此行是为了在扶植忠诚于自身的代言者。——然而，鸣神大社本就自成一派，更未见宫司平日有意涉政。帮助社奉行，收益小于付出，未必是明智之举。一说宫司怀疑事有蹊跷，如今受牵连者甚广，社奉行若再生变故，恐令稻妻大局不稳。——此话乍看有理，细想却站不住脚。需知，豪门起落只是人间再平常不过的戏码。即便神里失势，社奉行也会有新当家。又有一说，神子在风波初定之后，曾与当时的神里家家主进行过密谈。——但年迈重伤的神里家主已如风中残烛，又有什么能力左右大局？至此，坊间种种猜测均无结果。神子所想所为，皆没有答案。人们不知道的是，神子当日给出的嘱咐，如同家族规章一般留在了神里府中。「神里家能从此次事件中幸存，还是将军宽容。从今往后，切莫忘了将军的恩典。」这句话成为了因缘之种，预示着社奉行未来的位置。他日，若风暴降临稻妻，即便与另两家奉行针锋相对，社奉行神里家也要牢记恩惠，恪守「将军」的永恒之道。棋盘上，宫司走出了无憾的一着。[12]虽然散兵并未取众人性命，但神里家家主因担心枫原家遭株连，劝告枫原义庆不要公开此事。此后，楓原萬葉的曾祖父枫原义庆放弃「一心传」，转而研习盆景。逝世前大多的盆栽都被他送出去，只有一个种植着枯死的树木的盆栽留在家中，不允许丢弃。内藏一封家书，关于当年义庆从未与外人提及的五传覆灭的真相。万叶的祖父枫原久通想重振「一心传」，因一直参照之前的图谱锻造而一直失败，与其父义庆闹翻，一气之下离开稻妻，实际上义庆对久通发火的原因是久通一直按着图纸锻造却没发现图纸是被篡改过的，因此认为或许一心传既然无法察觉真相，那么没落或许就是天命。久通未能见到父亲的最后一面，因愧疚而没丢弃其父的古老盆栽，后来也制作了自己的盆景，希望孙子别学自己，更多去看看这个世界。「身在人间，心向山水，万叶家世代传承皆如此。这个家早晚要交到你的手中，到时候，莫忘了本心。枫原家的刀剑，绝非为了伤人所造。」之后万叶的父亲枫原景春也试图以御神刀图谱复现「一心传」，然终未能参破其秘。身为家主难以游历，书中所见入云之山，刻入盆中。告知万叶：「家庭固然重要，但人却也不能拘泥于此。如果有一天，你厌倦了生活，就丢下这一切出去走走。切记，不要让生活束缚了你。」之后枫原久通逝世，万叶因反抗眼狩令被通缉流亡海外。[15][59]
