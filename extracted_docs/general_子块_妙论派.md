# 妙论派

依托于刹诃伐罗学院，因为研究与制造驱动机关而产生。由于制作机关要涉及到各方面的知识，所以是个相当综合的学派。其标志为白色。包含建筑学、机关术等学科。曾经风靡过对机械生命的研究，但由于为了探究自然生命与机械生命的区别，狂热的学者对动物进行无节制的活体解剖，所以被教令院下令禁止
