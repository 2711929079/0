# 娜维娅技能详细介绍

娜维娅是《原神》中的岩元素五星角色，枫丹的地质学家，称号"璀璨夺目的宝石"。
元素战技「宝石冲击」：
- 释放岩元素能量，对周围敌人造成岩元素伤害并生成岩元素结晶。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「宝石盛宴」：
- 释放强大的岩元素能量，对周围敌人造成岩元素伤害，并创造一个持续10秒的岩元素领域。
- 冷却时间：18秒。
- 能量消耗：60点。
固有天赋：
- "宝石的祝福"：宝石冲击命中敌人后，有15%概率触发"结晶"反应。
- "岩元素增幅"：娜维娅的普通攻击和重击造成暴击时，恢复3点元素能量。
- "盛宴的威严"：元素爆发持续时间内，娜维娅的岩元素伤害提升20%。
