# 摘星崖

同样作为地势高、风景优美的地方，摘星崖以夜景出名。塞西莉亚花的生长地，清冷而风急[3]，也是诸多旅行者前往坠星山谷东面未被地图绘出的岛屿的出发地。
三个约会圣地之一。
