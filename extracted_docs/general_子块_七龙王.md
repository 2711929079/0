# 七龙王

法涅斯占领提瓦特前统御大地的七位元素龙王。后来与法涅斯大战后凋零，古龙大权被夺走，有的死亡，有的背地里继续反抗天理。因为游戏内对龙王的定义为「元素生命的顶点」，因此其形态不一定像龙，如那维莱特。龙王们的共主为尼伯龙根。
