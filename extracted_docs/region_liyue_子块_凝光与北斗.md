# 凝光与北斗

大约二十多年前，幼时的凝光还是一位默默无闻的贩夫走卒，经过许多年的摸爬滚打，她成为了如今璃月首屈一指的大商人，更是璃月七星中掌管律法的天权星。利用经商多年所积累下来的显赫财富，凝光建造了举世闻名的群玉阁[159]，每年对现有法典查缺补漏[160][161]。同样是二十多年前，年幼的北斗是吃百家饭长大的孤儿，曾被望舒客栈附近的一个小渔村收留，后又被人们当成是灾星而赶走。经历了多年打拼后，北斗创立了赫赫有名的南十字船队，甚至有过斩杀海兽的壮举。北斗是天枢星凝光最信任的友人之一，时常互相合作，而两人在幼时也有过一些因缘[162]。
