# 古月遗骸

在与深渊的战斗中，玛薇卡曾将「虚假之天」轰开破洞，露出虚空中的白色碎块「古月遗骸」，此番景象只在纳塔境内可以看到。根据正在大陆上游历考察的龙遗物研究协会研究员穆托瓦的说法，在纳塔西部山地（图兰大火山）存在着与古月遗骸高度相似的古物，估计也是坠落的碎片[21]。纳塔四散的「星星碎片」似乎也是古月遗骸的一部分[22]。愚人众执行官「博士」为探究「古月遗骸」而将挪德卡莱作为据点，派遣手下进入纳塔[23]。愚人众将天上的「古月遗骸」拉了下来，两片最大的「古月遗骸」落到了挪德卡莱，一片落到了希汐岛，一片落到了帕哈岛。「愚人众」在帕哈岛的那块陨石上建立起月矩力试验设计局[20]。
