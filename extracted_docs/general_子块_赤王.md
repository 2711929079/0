# 赤王

远古时期的须弥，有三位神明共同统治着这个地区。而赤王是沙漠地区的王者。在十分久远的过去，赤王与花神就已经相遇，并且一同为镇灵建造了「月女城」。[1][2]大慈树王、花神、赤王三人一同建造了乐土「瓦力·韦杰」。有关于三王共同治理时期须弥的描述：「在那些年代，月光曾将他们的幸福讲述给夜莺与蔷薇，」「而她们也为之惊惶、为之羞涩，以致唱不出回应的歌。」[3]赤王在听到外来者所带来的战争与灾难以及仙灵一族的遭遇后，无视了花神的警告，在心里许下僭越的愿望后，把自己的欲求诉诸花神。花神虽然认为赤王的想法不过是捕风而已，但是她仍然决定为了人类的未来去赌那一丝可能性，最终花神牺牲了自己，为赤王打开了联通天空与深渊一切知识的通道，而代价是花神的生命在花神死后，赤王十分悲痛，执着于旧日的回忆，在寒天之钉上为花神修建了陵墓：永恒绿洲。[1]在花神死后，赤王与树王分道扬镳。赤王决心在地上建造理想国，让一切悲伤从此消失；而大慈树王决定散播草木与绿茵，让土地充满智识与幸福。[4]在花神死后，乐土「瓦利·韦杰」被狂沙席卷，一直到百年之后赤王与大慈树王回归决定重建绿洲乐园为止，曾经乐土的子民在百年无神的混乱与迷茫中据藩地自守。而在这百年间，沙漠出现了许多国家。[5]在这百年间，月女城变为废墟，（秘境「蜃景祭场」便是月女城的遗址）大慈树王在雨林中研究法留纳神机。而根据现有资料，我们并不能明确得知赤王在这百年间做了什么，不过我们可以推测赤王或许是在研究寒天之钉。[6]等到百年之后，赤王与树王归来想要重建绿洲乐园，但两人的道路却依然大相径庭。赤王想要通过晶石与花神的教诲去完善原本脆弱可悲不完美的世界，创造一片无忧乐土，也就是创造一个新的世界。“首先创造了日月。如此便有了白昼与黑夜。她曾以我已经忘却的语言描述三轮明月的夜空，那月轮之数便应当为三。但愿世界的影子苏醒时，她们能在大地上投下珍珠色的微光。人们便能循着夜幕中沙丘镀银的轮廓寻得宿命的终点。接下来创造重量。如此砂土便沉下，形成了大地。而没有重量的便是天空。我规定：应当仰赖土地，但是梦想深空。重量不应当过大。否则土地便会捆缚人的双脚，使人走不远，不能向四方八方开拓；使人飞不高，不能向未来探求。接着再次设立七贤僧，由他们治理地、水与诸星描绘的轨迹。即便天球只是幻造之物，仰望星月时常有神话诞生。在原本的世界，藩篱曾被撕毁，黯色的毒曾渗入大地。为了愈疗那个脆弱可悲不完美的世界，长钉降下，贯穿地壳。但是，我所设定的规律要更优美、更精密，所以没有必要。也不应当有追随她的人为此枉死，不应有诗文因此失落。”[7]而这所谓的无忧乐土，就是赤王陵中的阿如。在任务中，我们可以得到一个名为“阿如的倒影的记录”的物品，通过阅读可以得知，在阿如建造的最后，似乎发生了意外，所以也许这个所谓的无忧乐土并未建造成功。这也是为什么我们最后通过试炼却并未到达阿如。此前赤王一直深深沉浸在失去花神的痛苦中，渴望建造新世界的同时又痴迷于过往，而三佞臣又在不断进言，最后赤王听信了谗言，将自己困于迷宫深处，寻求深黑的禁忌知识。[8]不过，在研究完禁忌知识之后，他也终于明白了——他真正要找回的不是众生的乐园，而是三人再度坐在一起，于乐园中论道的景象。于是赤王对【阿如】的蓝图进行了变更。摘除七之轮转，因为深秘的叙事会被阻断。摘除恐惧与哀伤，因此要消除生死的隔阂。摘除日月与重量，因为不应有时空的藩篱。摘除规定、裁决、施恩的原初之理，这样她便不会为同族所受的责罚而害怕。摘除飞禽、走兽、游鱼，龙与人，其数为七的僧王，这样谁都无法窃取智慧。[7]赤王认为，这就是他所追寻的乐园。也就是我们所熟知的黄金梦乡。在黄金梦乡竣工后，赤王认为他所带来的禁忌知识不该留在这个世界上，于是最终决定牺牲自己，与大慈树王联合清除了禁忌知识。
