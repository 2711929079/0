# 诀箓阴阳寮

惟神晴之介为锻炼能够对抗魔物的武士，利用式法创立了诀箓阴阳寮和寮司式大将，在此中不断修行。[32]不知为何晴之介的符法流落到贼寇手中，成为海乱鬼为武器附魔的手段。[53]
式大将式大将是惟神晴之介所制造的纸人式神，同时也是「诀箓阴阳寮」的寮司。曾被晴之介仿造出大量拓本作为量产式神投入使用。彼时在「诀箓阴阳寮」培育武士需要消耗大量的精力和拓本，而雷电将军又在那个时代以更强的力量消灭了妖魔们，「诀箓阴阳寮」几乎失去了存在的意义，式大将在晴之介落寞时开导了晴之介，告诉他“意义不该由他人赋予”，使得晴之介真正释怀，与式大将道别后离开了「诀箓阴阳寮」。斩杀的对象不复存在，需守护之人已求得平安。惟神晴之介再次启程环游世界去寻找新的意义。[1]式大将盼望着晴之介归来，因此一直在「诀箓阴阳寮」内守候，直到与「诀箓阴阳寮」一同陷入沉睡并在这个过程中失去记忆[32]。
