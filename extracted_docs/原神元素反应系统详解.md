# 原神元素反应系统详解

《原神》的核心战斗机制是元素反应系统，不同元素组合可以产生强大的效果。
元素反应类型：
- 蒸发（火+水）：伤害提升1.5倍（火触发）或2倍（水触发）。
- 融化（火+冰）：伤害提升1.5倍（火触发）或2倍（冰触发）。
- 感电（雷+水）：持续造成雷元素伤害，对多个敌人有效。
- 冻结（冰+水）：冻结敌人，使其无法行动。
- 超载（火+雷）：产生爆炸，造成火元素范围伤害。
- 超导（冰+雷）：降低敌人物理抗性40%，持续12秒。
- 碎冰（冰+岩）：对冻结的敌人造成物理伤害。
- 扩散（风+其他元素）：将元素扩散到周围敌人，造成对应元素伤害。
- 结晶（岩+其他元素）：产生元素结晶，拾取后生成对应元素护盾。
- 激化（雷+草）：提高后续雷元素和草元素伤害。
- 绽放（水+草）：产生草元素种子，接触火元素或雷元素后爆炸。
- 燃烧（火+草）：持续造成火元素伤害。
