# 现在

恒常乐土旅行者乘死兆星号抵达离岛，受社奉行大小姐神里绫华的委托对抗眼狩令。千手百眼天下人间旅行者在社奉行的委托下暗中反抗眼狩令，而条件是社奉行会将旅行者引荐给将军，使旅者和将军得以见面。在第一百颗神之眼献祭仪式上旅行者为了救人情急之下进入雷电影的一心净土使八重神子看到了事情的转机。但旅行者也因此被通緝，投向海祇島的反抗軍。愚人众第八执行官「女士」向反抗军提供邪眼，毒害了大批有志之士。旅行者为阻断邪眼制造前往邪眼工厂，被当时看管工厂的「散兵」击败命悬一线，八重神子用神之心与其交换旅者的性命[90]。旅行者与八重神子一道，揭露了九条家与柊家与愚人众勾结的阴谋，气昏了天领奉行九条孝行，策反了九条裟罗。旅行者揭露了「女士」在稻妻的各种罪行，并在御前决斗中将其击败，「女士」最终被雷电将军斩杀。反抗军和万叶在社奉行的带领下[91]冲进天守阁，旅行者唤起万千人的心愿，击败雷电影。后者因被人民的愿望感动而不久后废除眼狩令[92]。战后此前被收缴的神之眼悉数奉还。天领奉行九条孝行、勘定奉行柊慎介被捕入狱。散兵持雷神之心与愚人众失联，不知所踪。愚人众內奸包括内森据反抗军声称全被肃清。[93]九条镰治被鹰司家绑架，被雷电影营救后成为家主名代。九条家趋于没落。珊瑚宫与幕府议和，战事暂时平息，但双方各留一定分寸。愚人众执行官末席「公子」来到稻妻，调查散兵失联一案，遇到意外开启的诀箓阴阳寮。雷樱雷电影解除眼狩令的决定令将军产生怀疑，为了判断雷电影是否受到磨损，前来与其对质。在与将军一番切磋后，雷电影得到了雷樱的种子，将其种在500年前的稻妻，保护稻妻在漆黑灾厄时免受侵袭。最后，雷电影与真做了最后的道别，真的意识就此消散。[34]神樱大祓在命定之人的帮助下完成神樱大祓后，花散里被神樱净化。岩藏流岩藏流的末路旅行者将岩藏流灭门，只有岩藏光造因为不在道馆而幸存。可以在九条阵屋东北方找到他旅行者与他比武，将其斩杀，后回到白狐之野，斩杀了岩藏流最后的两个武士。踏鞴物语旅行者与泽维尔展开了为期一周的御影炉心净化工作，挫败了愚人众的阴谋。最终清除了神无冢地区的祟神。幕府派遣工程师阳介前往九条阵屋主持秩序。[82]神无冢原住民叶名山薰回到故乡，以种花方式向世人传达故乡的净化。远吕羽氏遗事旅行者接受工人梶的委托，修复4处镇物，八酝岛的雷雨天气停止，并斩杀了发疯的鹫津。孤岛诊疗谭旅行者来到八酝岛，帮助医生保本收集鸣草以制作药物。保本多次小剂量试药，后前往八酝岛南部沙洲，服下了全剂量试药，毒发而亡。执望三千里旅行者来到八酝岛，被长次感动（蒙骗）收集晶化骨髓，误会解除后长次在旅行者的帮助下从离岛出发寻找母亲。武士的宿命旅行者救下了受伤的稻叶久藏，后受稻叶久藏委托，将名椎滩的海乱鬼清剿长次没人照顾了。稻叶久藏终与海贼战死于八酝岛。清籁逐雷记旅行者修复了清籁岛四处镇石，解放了雷音权现。寝子是只猫浅濑响的宠物猫，委托旅行者修整了浅濑神社，使其重焕生机。雾海纪行旅行者接受作家墨田的委托，来到鹤观调查异常，结束了鹤观持续数千年的轮回。雷电五传真相雷电五传后人枫原万叶与旅行者等人在神里绫人的安排下透过枫原义庆的家书得知雷电五传没落的真相。[15]妖刀的结局妖刀终于来到稻妻，与雷电五传后人枫原万叶相识。最后妖刀解开心结，明白了造出它的刀匠生前真正的愿望，于是希望枫原万叶能够参考自己身上种种的技术，重振一心传的荣光。之后，他的意识消失，只留下自己的名字—笼钓瓶一心。万叶也尝试复甦一心传。[74]蜃楼玉匣现人神巫女珊瑚宫心海取回500年前喜多院秀家留下，原属于珊瑚宫的宝物蜃楼玉匣。旅行者也透过伊迪娅了解了500年前秀家的故事。[44]
