# 卡莉露与芬奇

纯水精灵卡莉露与少年时期的芬奇在清泉镇相遏，彼时哭注的芬奇向卡莉露吐露真心，而卡莉露则陪在他身边聆听，久而久之，芬奇对卡莉露有了思慕之情，那时的卡莉露深知作为纯水精灵的自己与人类的命运不问，因为无法回应芬奇的心意而逃走了[116]。
罗莎莉亚出生不久的罗莎莉亚所居住的小山村被一伙盗贼洗劫，罗莎莉亚也被盗贼劫走，成为奴隶。在与盗贼团的生活中，罗莎莉亚学习到了战斗技巧。在一个饥寒交迫的夜晚，罗莎莉亚不堪雪负选择逃走，盗贼团中的一位老人将她追回，并让她终结自己的生命，用这样的办法，罗莎莉亚获得了神之眼并被盗贼团接纳[123]。
