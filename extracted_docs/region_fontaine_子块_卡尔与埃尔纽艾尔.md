# 卡尔与埃尔纽艾尔

卡尔是出身水仙十字院，与德怀特·拉斯克、贝瑟·埃尔顿以及埃马纽艾尔·吉约丹一同长大的孤儿。在幼时的勇者游戏中，卡尔常常扮演恶龙贾巴沃克，而埃马纽艾尔常常扮演讨伐恶龙的英雄。[33]长大后的埃马纽艾尔加入了逐影庭，以打击犯罪为工作，「白淞之围」正是在其参与指挥下进行的，而这场围剿的进行也代表着卡尔为之努力的谈判最终没能达成。悔恨于自己所为，失望于枫丹廷的领导，埃马纽艾尔离开了逐影猎人的队伍，从此自暴自弃借酒消愁。[33]没能促成和谈的卡尔则在另一侧同样经历了这场围剿，帕西法尔用「戏法」强行将其转移到漆黑而安全的地窖，只能听到隔着砖石土地闷闷地传来的喊声、破裂声与金属碰撞声。[33][34]
贝瑟贝瑟·埃尔顿原为水仙十字院孤儿，长大后加入枫丹海军，成为海军司令，是枫丹海军「白色舰队」旗舰「斯彭西安」的舰长，浪影阔剑为其武器。「铁甲熔火帝皇」因为重甲蟹的习性，会随着成长不断寻找合适的螺和贝壳来当作甲胄，如果不阻止它，说不定它会无限生长下去。但是后来，贝瑟·埃尔顿击败了它，才终止了继续成长的进程[35]。主动退役后，贝瑟回到十字院接任副院长[36]，在抚养卡特期间烹饪水平很差，经过苦练在抚养雷内四人时已变得水平高超。[37][33]坎瑞亚灾变发生、十字院遣散前，贝瑟联系卡尔和埃马纽艾尔领养阿兰四人[38]，并将墨绿羽饰留给了雅各布。[39]自己则返回部队，重登战舰，对抗以「厄里那斯」为首的兽潮。
