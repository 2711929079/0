# 挪德卡莱·清理首领

挪德卡莱曾经被诸多「首领」控制（有宝藏猎人，可能也有其他人士），他们在无数次的密会里划定了地盘，就像某些失乡者们想要逃离的暴君与僭主一般，将这片本该属于所有人的乐园无声地据为己有，尔后派出自己的爪牙，肆意施行暴虐。奈芙尔来到挪德卡莱后清理了这些地头蛇，并在日后建立了「秘闻馆」，接手了「北大陆情报网」。[131]
