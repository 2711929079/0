# 执行官代号

愚人众十一执行官的代号取于意大利假面喜剧(或意大利即兴喜剧【"Commedia dell'Arte"】)自十五世纪中（1550年-1750年）起，诞生于意大利文艺复兴时期的「意大利即兴喜剧」剧团，风靡欧洲近两百年。最初的艺术喜剧多在户外进行演出，意大利的街头巷尾和集市附近经常能看到临时搭建起来的简陋舞台。由于没有复杂的舞台布景，演员们需要依靠自身的肢体动作以及五花八门的舞台道具。街头艺人们走街串巷，在一个地方表演完后，将整个舞台装进一架马车赶往下一个地方。[5]十一位执行官取于十一种即兴喜剧的面具，名称即是上文中的英文名称。
