# No.8 「女士」席诺拉La Signora

中文CV：子音 日文CV：庄子裕衣详见蒙德#罗莎琳五百年前的炎之魔女，在他的爱人鲁斯坦死于漆黑的灾厄之后，便燃尽自己的生命之火，化身为被世人称为「炎之魔女」的可怕存在。某一天，女皇看到了她，并赋予她冰之邪眼。做事极其趾高气扬，自负而狂妄。于序章·第三幕「巨龙与自由之歌」在游戏中初次登场，并亮出席位是第八席。在《原神》2.1新版本「韶光抚月，天下人间」先行展示页-新强敌介绍中显示其代表图标为飞蛾。在蒙德时，用掏心的方式直接抢走了温迪的风神之心，并将其一脚踹飞，在璃月，按照钟离与女皇事先签订的「契约」，获得了岩神之心并回到了至冬；在稻妻一通嘲讽成功激怒旅行者后，旅行者提出与其进行御前决斗，在战斗中再次成为驱役火元素的炎之魔女，但被旅行者打败，被雷电将军一刀斩杀，原本的庆功宴直接成了吃席，已经基本上没有复活可能。击败她后，可以获得其面具，即戴丧面具。
执行官互评折叠「公子」：我和她合不来，你是知道的。如今也不必评价她本人了吧。对于执行官而言，死期并不掌握在自己手上…不过，放心，我是无论如何，都会拼尽全力活下去的。「散兵」：她的灰烬里什么都没有。我对失去未来的人不感兴趣。「仆人」：她与皮耶罗是我最先认识的两位执行官。她第一次来壁炉之家做客，一副趾高气昂的做派，不受孩子们喜欢。那之后她每次来访都会带些礼物，堂而皇之地表示「不喜欢我的人没份」。为了礼物，孩子们很快学会了逢场作戏，她也照单全收。我猜测她很享受孩子们围在身旁的感觉，这或许源于她内心的孤独…对于她的牺牲，我深感遗憾，愿她能与死去的爱人重逢。「少女」：如果羽化意味着催动翅膀扑向火焰，那么维持茧的形态是否才是留存生命的更好选择？罗莎琳仿佛从没考虑过可以那么选，又或者她早已为自己选定了结局。她尽情爱，尽情恨，尽情舞…然后将哀歌留给别人来唱。
