# 雷穆利亚征服高海

后来，雷穆斯王与他不朽的舰队征服了高海上的全部岛屿，一座座城邦臣服于雷穆斯的威权，就连海渊之下的巨龙也向他臣服。雷穆利亚拥有了整片高海，无数的城市与宫殿，鲜花与美酒。[9]自初民的时代终结以来，从未有这样美好的日子，永恒的繁荣似乎即将达成。[2]
安提库港当航船沿着御道穿越擎海之柱，驶入御船法图纳号停泊的巨港，首先映入眼帘的便是一座参天的高塔。雷穆利亚的高塔并非为了聆听高天的谕示而建，而是为了给高海上诸岛之间往来的航船提供指引。据说，高塔建立在现实与梦境的交汇之处。即便水手们因海妖的诱惑而陷入沉眠之中，也能循着钟声穿破迷雾，找到通往雷穆利亚的航线。[2]
