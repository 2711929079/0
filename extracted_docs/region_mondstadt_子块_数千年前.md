# 数千年前

特瓦林挟带着风元素的风龙王在高天诞生了。它缓缓降下，对世间一切充满了好奇。[25]旧蒙德建立最近被称为风龙废墟的地方，曾经是烈风的魔神「迭卡拉庇安」建造的王都。[26]王都的结构整体呈环形，可能分为三层，每层对应的职能不同，正中是一个高大的塔楼，是迭卡拉庇安的宫殿，最外可拓展到今风龙废墟南部通路与蒙德-璃月道路的路口，除了南部通路外，西北和东北也有疑似对外通道。旧蒙德的建筑风格与现在的蒙德有明显不同，常用一种末端卷起的十字Cercelée样式的纹章。无果的远征雪山之国沙尔·芬德尼尔覆灭后，坎瑞亚为了寒天之钉的力量派遣了包括耕地机在内的队伍前去雪山探索，大批耕地机因极端的低温失去行动能力，探索行动失败[27]。
