# 首次花神诞日

在第一次的花神诞日，赤王与花神为大慈树王准备了惊喜。他们沿途散播糖果作为线索，引大慈树王前往宴会之地。好奇的大慈树王跟着来到一处湖边。忽闻一声琴音，原来是赤王开始拨弦奏乐，而花神则跳起美丽的舞蹈。他们共同庆祝大慈树王的生日，并献出礼物—花神献上了灵光，赤王赠出能纪录过去的宝石「乌婕特之眼」，用以纪录三人相处的时光。乌婕特之眼力量特殊，只有赤王后裔才能与之共鸣。[19]
花神诞日传说,之所以这个日子会叫做「花神诞日」,其实最早是「花神祝诞」的意思。在很久很久以前，有一次树王大人过生日,她的朋友们举办了宴席为她祝寿。宴席上,几位神明大人都喝醉了，其中一位便乘兴弹奏起了乐器，于是树王大人唱歌，花神大人就跳起舞来。在花神大人起舞时,她踏过的草地上，长出了无数美丽的帕蒂沙兰…明亮的紫红色花朵筑成了灿烂的舞台,神明们都说:「啊，若是时间能永驻此刻就好了。」其实这是个很古老的节日，最早是为大慈树王大人祝寿的。树王大人逝去后它被沿用下来，时间就改到了小吉祥草王大人的生日。[20]
