# ［芬德尼尔时间线推论·非正文］

葬火之战时期天使们因触犯禁忌（与外来者相爱）遭到天理诅咒，退化成仙灵，失去记忆、智慧与情感。[1]游戏内并没有明确交代沙尔·芬德尼尔的建立时间，以前玩家透过雪山石碑上记载的“冰雪与纷争”推测其建立于魔神战争时期。[2]但实际上魔神战争结束后，风神才吹散冰封蒙德各地的冰雪[3]，因此沙尔·芬德尼尔可能成立于魔神战争前任何时段。
随着须弥开放，关于葬火之战有了更多文本记载[1]，沙尔·芬德尼尔的时间线也需要更动：因为沙尔·芬德尼尔的人们会透过神使和天空岛沟通，并追求神使的指引，[4][5][6]因此沙尔·芬德尼尔应创立于葬火之战前，创立于神使并未遭到诅咒被贬下天空岛，人们能被准许与天空岛沟通时。
另外，因为人类使用禁忌知识，天空岛在葬火之战结束后降下许多天钉封堵禁忌知识。[1]沙尔·芬德尼尔大概率是在此时因降下的天钉覆灭。
第一文明第一文明曾经有一个时代，大地上的人们还能直接听到来自天空岛的启示。神的使者走在蒙昧的人当中。彼时万物的气息渐弱，大地永冻[7]。在天使的带领下，人们创造了统一文明，即「第一文明」。第一文明大部分都是石制砖头建筑，上面有凯尔特三角的标志，风格明显，很好辨认。蒙德的第一文明建筑中，望风山地的看似零散，实际上构成了一座建筑群。南部的遗迹群一路向高地延伸，遗迹的尽头是一处秘境——仲夏庭院，古代文明建造的夏宫灵囿，是一座用于休闲观赏的庭院建筑，不难想象它曾经的繁荣景象。过去的人们向往天空，于是将城塞建立在高峰之上[8]。（那些坐落在山峰上的古老的遗迹，毫无疑问连同山峰一起在风神修改地形时随风而去了，如今只留下少量的残存）风啸山坡的建筑群在达达乌帕谷谷内，地图上山体凹陷处，较能阻挡当时的风雪。苍风高地存有一大片遗迹，由于遗迹大多分布在大山丘上。苍风高地七天神像所在的山丘无论从遗迹数量上看，还是从位置上看，都很像是古国的核心区域，因此推测这里是主城。在周围还分布着一些城镇和祭坛，有些依旧保存着原貌。塞西莉亚园圃的位置相对偏僻，远离遗迹群中心，道路也十分曲折。尽管如此，一路上都分布着与山丘上古国相同的遗迹，这说明它确实属于这一古国。秘境介绍中提到，这里曾是培育塞西莉亚花的温室，如此看来位置偏僻一点也不奇怪。蒙德古国会兴建能产生风场的机关。蒙德曾存在与古树并存的古国。古国的先人们兴建了塞西莉亚花的苗圃。在今苍风高地建立了塞西莉亚苗圃，这个苗圃是种植塞西莉亚花的温室，该地也具备祭祀职能。位于今摘星崖和千风神殿地带的文明建造了祝圣祭坛夏宫灵囿。
