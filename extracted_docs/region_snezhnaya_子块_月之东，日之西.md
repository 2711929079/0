# 月之东，日之西

雷德·米勒曾在「月之东，日之西」留下「伟大财宝」，需要集齐五件信物「识者之证」才能开启[160]。旅行者于挪德卡莱各处的冒险中集齐了其中四件（详见#幸运儿号、#阿特拉号、#涅朵奇卡、#「末日庇护所」），在派蒙的提议下来到皮达米拉城寻找最后一件「识者之证」的线索，偶遇前来探亲的托尔芬（见#幸运儿号）。原来最后一件「识者之证」正是由他的祖母海捷德持有。海捷德是盗宝团中的一位老前辈，人称「黑枪太婆」，她沉浸于盗宝团旧日的辉煌中，不满于组织松散、混乱的现状，希望能由合适的人继承「伟大财宝」和盗宝团首领之位，喊出「让死人埋葬死人吧，然后你们站起来，用高傲者的财富，筑起饥贫之人的乐园」的口号，带领盗宝团恢复雷德·米勒在世时的荣光。旅行者通过了海捷德的试炼，又在托尔芬的帮助下敷衍应下海捷德复兴盗宝团的要求，得到了忆识者的戏言。五枚「识者之证」归位，通往「伟大财宝」的道路打开。原来「伟大财宝」便是大盗留在此地的，足以让（五百年前的）北陆经济为之撼动的无数铁钱。机关启动，铁钱受磁石牵引向上飞去，露出下方的航船。握住船舵的那一刻，来自艾维雷勒安和索洛维的回忆涌入旅行者和派蒙脑中，回过神来已回到了宝库的入口处，并得到了艾维雷勒安和索洛维留在此处的最初的铁钱。通过藏宝处遗留的日记和托尔芬的讲述，「大盗」雷德·米勒与「伟大财宝」的真相终于展现在了旅行者的面前[40]。
夜莺之歌苦壑崖一带爆发了大规模的狂猎，执灯人集结队伍准备击退魔物、调查引发狂猎的源头。旅行者受执灯长尼基塔委托，协助寻找五百年前铸灯者击败、封印魔物所用剑、灯、镜三件圣物之一——银灯芯。「银灯芯」原本是霜月之子的祭器，可以展开结界、抵御狂猎的攻击，在平息了灾厄后，铸灯者便将它交给了守护厄布拉神柱的妖精爱莉厄保管（实际并非妖精，爱莉厄及厄布拉神柱相关内容详见#爱莉厄的牺牲）。旅行者与「魇夜之莺」分队长叶洛亚在厄布拉神柱相遇，一同清除了高塔的污染，帮助爱莉厄恢复力量以取得「银灯芯」。将「银灯芯」交给旅行者后，厄布拉神柱一带的结界全部失效，深渊重新占据此地，霍德望卷土重来试图高塔逃离「世界」。爱莉厄尽全力将其困于高塔，等待旅行者等人找到将他真正置于死地的方法。离开厄布拉神柱后，旅行者与派蒙受叶洛亚邀请，成为「魇夜之莺」调查分队的成员。在「望崖营壁」，旅行者与「魇夜之莺」小队其他成员会合，展开下一阶段的作战行动：抢先夺下崖上营地，并筑起抵御狂猎的前线。队员比约恩在保卫阵地时牺牲，小队其余成员一同击败了霍德望的仆从「靡穷之孽」，并带回铸灯者用以封印妖魔的圣剑「螺旋剑」。菲林斯寻回了由执灯人代代守护「德肋庇革劳诺之箭」，当「魇夜之莺」小队归来时，执灯人们已完成了准备工作。原来麦酒大厅正是围绕「德肋庇革劳诺之箭」建造的发射器，而「螺旋剑」则是它的箭簇。叶洛亚与旅行者一同启动「德肋庇革劳诺之箭」，螺旋剑射向高塔。在闪耀的光芒中，爱莉厄以雷德·米勒的名义宣告霍德望的终结，「不死的毒蛇」霍德望迎来第三次死亡，如预言般「被月下世界的英雄们彻底埋葬」[34]。
