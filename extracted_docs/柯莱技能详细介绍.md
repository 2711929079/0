# 柯莱技能详细介绍

柯莱是《原神》中的草元素四星角色，须弥道成林的见习巡林员，称号"萃念初蘖"。
元素战技「飞叶箭」：
- 释放草元素能量，对周围敌人造成草元素伤害并施加"草元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「萦绊之草」：
- 创造一个持续8秒的草元素领域，领域内的敌人会受到持续草元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "飞叶的祝福"：飞叶箭命中敌人后，有15%概率触发"激化"效果。
- "草元素增幅"：柯莱的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "萦绊的守护"：元素爆发持续时间内，柯莱的草元素伤害提升15%。
