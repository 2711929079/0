# 阿特拉号

接着阿特拉号却误入刻拉蒂在的海域，就连海鸟也会因其中迷雾受困而死。…快到那片臭名昭著的海域了，风向有些不对，涌浪也越来越厉害……得让甲板上的伙计们打起精神来。虽然已经向海中投入了烈酒和面包，格瑞蒂尔还是不放心，又奉上了几枚摩拉，甚至还有不知从哪儿弄来的铁钱，希望我们能顺利通过……在船尾的甲板上发现一只长翎鹮，已经死了，这不是…之后「盾女」洛维萨在船上率先消失，之后陆续有人失踪，包括哈尔克，船上陷入恐慌，之后整艘船沉没[16]。之后传说中有人看到海上漂流的阿特拉号（幽灵船），据霜月之子所说是因为海里的诅咒[17]。有寻宝猎人去阿特拉号船骸寻宝却被幽灵袭击，都是因为该地的一些海盗亡魂作祟[15]。
