# 梦之魔神

被称为「悬黎」的玉石，即便在至深至暗的夜里也能放射光芒。当被雕刻成劲弓之时，据说它射出的利箭能自行寻歼一切邪恶。
群岩之主的目光始终冷彻坚定，恰如永恒闪烁的金珀之光，冰冷而沉郁。在那个动荡不已的年代里，弓弦的震颤、鸣响与尖啸代替了安逸的歌声。洞悉人心弱点的梦之魔神，曾被这张强弓所射杀。她拥有明澈的双眼与锋利的头脑，善于操控与占有臣民的心。“看吧，我的孩子。看这无畏的英雄被恐惧啃啮，无私的王者卑躬屈膝’‘看呀，我的孩子。看那立下海誓山盟的情人，如何被背叛与谎言折磨（渌华池传说里的那对）”在怨恨的缠绕间，斜倚御座的魔神品尝着欢愉的果实，浆液自指缝间溢出。无形铁链铮铮作响，深入血肉的剧痛与狂喜中，忠诚的恶犬吞咽着“美梦。不屈的骨髓承受反抗着每一次穿刺与鞭答，血脉的悸动每一刻都带来剧痛。鲜血淋漓的梦之美味美妙而易逝，勾起纯粹的憎恨，憎恨正是魔神的狂喜⋯快说呀，“然后，告诉我，快说呀，我的孩子，你吞下的“美梦”是什么滋味”孩子，可知"爱"不过是瞬逝的云露，“权力"才是"美梦”的原料’痛苦与残酷不过是强权的涟漪，破碎的'美梦'才是真正喜悦的食粮。但善于刺穿他人弱点的魔神未曾料到，她的心脏也将被帝君用悬黎千钧射出的碧玉利箭射中。
在神体崩毁的那一刻，万干幻梦的碎片随之进散，纷纷扬扬化为虚无⋯追逐钳制人心的魔神奔向自我毁灭，势有干钧的巨弓宣告了她的报应。
如此终焉本就应是理所当然的恶果，暴虐的欢愉自当迎来残忍的结局。当一切复归平静，黑夜不再有哀叹和哭泣，碧血的劲弓便被悉心封藏。
隐忍地等待着下一位暴君的孳生，以夜幕无法遮掩的翠光射穿其心脏[70]。（魔神的天性是爱人，所以说这魔神三观得扭曲成啥样。有说法认为其就是发疯的沈玉谷旧主。）梦貘原本是璃月梦之魔神的眷属，后来梦之魔神败于魔神战争后逃至稻妻成为稻妻的妖怪[71]。
魈「魈」并非是这位夜叉的真名，而是帝君出于安全考量为他起的假名。曾经的魈年少无知，被梦之魔神[70]抓住弱点拘为座下大魔，听凭指示做下大量残忍血腥之事。他造了诸多杀业，踩碎诸多理想，还被要求吞下败者的美梦，痛苦万分却又身不由己。终于，在魔神的战场上，岩之神摩拉克斯与夜叉的主人相会了。「岩王帝君」解放了夜叉，并赐给他「魈」这一名字。「在异邦的传奇故事中，魈之一字代表着遭遇苦难、饱受淬链的鬼怪。你也经历诸多，以后就用这个名字吧。」为报答岩之神的恩情，魈接下了守护璃月的千年苦役。为魔神效力的往事洗去了魈的温柔与无邪。如今，他心中只剩杀戮技巧与累世业障。 战斗，是他唯一还能为人们做的事[72][73][74]。此后魈独自镇守望舒客栈千余年。[20]
