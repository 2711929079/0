# 摩拉

世上第一枚摩拉，是在数千年前，岩王爷亲手用神之心[27]将自己的血肉转换铸造的。后来岩王爷简单花掉了[42]。而后黄金屋建立，作全大陆唯一的铸币厂，借助岩之神的力量制造摩拉[27][102]。
复甦此前魔神混战干年，桑田尽毁。璃月之民遂以商贾、巧工为业。其佼佼者联合一体，名之「七星」。故璃月港雏形既成。七星下辖又有八门，内外诸业无所不包。又有千岩之军听令七星，内抚民众，外镇妖魔。是所谓岩王帝君之治。[30]追随岩王帝君者自成部队，以「岩君」、「璃月」为号，称「千岩军」，听令于七星，护卫璃月。后璃月之经济逐年屡升，成七国商贾之心，得誉「黄金的心脏」。
