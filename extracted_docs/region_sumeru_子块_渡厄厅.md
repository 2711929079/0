# 渡厄厅

无底的大厅。
传说，在沙漠的深处，古老的神明试图将神话中的幽冥与天国一一再现，让那些有罪之人堕入幽冥，受他眷顾的子民居于乐土。
无底的厅堂宛如幽冥位于深邃地下，有罪之人或是不敬赤王者将在此迎来命运的终结，而通过试炼者则将飞越不可视的河流，继续前行。
