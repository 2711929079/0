# 阿如

在花神死后，乐土「瓦利·韦杰」被狂沙席卷，一直到百年之后赤王与大慈树王回归决定重建绿洲乐园为止。赤王与树王归来想要重建绿洲乐园，但两人的道路却依然大相径庭。赤王想要通过晶石与花神的教诲去完善原本脆弱可悲不完美的世界，创造一片无忧乐土，也就是创造一个新的世界。赤王对其设想是：首先创造了日月。如此便有了白昼与黑夜。她曾以我已经忘却的语言描述三轮明月的夜空，那月轮之数便应当为三。但愿世界的影子苏醒时，她们能在大地上投下珍珠色的微光。人们便能循着夜幕中沙丘镀银的轮廓寻得宿命的终点。接下来创造重量。如此砂土便沉下，形成了大地。而没有重量的便是天空。我规定：应当仰赖土地，但是梦想深空。重量不应当过大。否则土地便会捆缚人的双脚，使人走不远，不能向四方八方开拓；使人飞不高，不能向未来探求。接着再次设立七贤僧，由他们治理地、水与诸星描绘的轨迹。即便天球只是幻造之物，仰望星月时常常有神话诞生。在原本的世界，藩篱曾被撕毁，黯色的毒曾渗入大地。为了愈疗那个脆弱可悲不完美的世界，长钉降下，贯穿地壳。但是，我所设定的规律要更优美、更精密，所以没有必要。也不应当有追随她的人为此枉死，不应有诗文因此失落。然后应当隔绝毒药传来的兽径，因为服毒是比天空更深的罪。但是低语是多么甜美。其中述说的智慧又是多么鲜明…新的世界里悄悄吹起了风。珍珠色的月光、琥珀色的余晖、草的浪潮与水的根系渐渐不再沉默，吟唱她留下的诗文。[4]而这所谓的无忧乐土，就是赤王陵中的阿如。在阿如建造的最后发生了意外，所以这个所谓的无忧乐土并未建造成功。[41]
七贤僧七贤僧为七位深得赤王信赖的下属与盟友。实际上他们大多是败给赤王的魔神，然后投靠了赤王。他们分别为公羊之王赫里沙夫、鳄之王索贝克、鹮之王图特、圣者贝努、圣者谢斯芬赫、阿赫·阿佩普和赫曼努比斯。赫曼努比斯被称为「众贤之至贤」，在赤王文明崩溃后曾来到图莱杜拉，并建立蓝宝石城，有一应传说故事流传至今。为保护其身后遗迹的信徒组建了缄默之殿，世代守卫于图莱杜拉城曾经建城之处。鳄之王和䴉之王分别被后世称为「众统帅之统帅」、「众书记之书记」。须弥历史载有鳄之王和䴉之王论法的故事。鹮之王是掌书，葬火之战前曾是是星与月的秘使（月使），与兹白等月使在月亮上共事[42]，也曾守护过图莱杜拉，发明须弥沙漠的古文字[38]。虽然星与夜的秘史是正史[16]，但其详情不可考，据图特本人夸耀說自己之前是如何在三个月亮上辗转工作，只为引导地上的愚钝凡众，如何为众生编织合理的命运之线，让地上的万事万物轮回萌发。在来到须弥，被赤王慧眼识英后，又是如何被擢升至伟大的「赤王七柱」之首的，之后还与赤王把酒言欢，与花神一同栽种绿植…[43]但佞臣能言善骗，图特尤其爱讲谎话，可信度较低（不过奈芙尔角色故事 月之轮和奔霄送玉轮第四幕提及有部分是真话，包括担任月使）。公羊之王被称为「众相之相」，是赤王最欣赏的一柱，以过人的智慧闻名后世。由于野心膨胀，曾对赤王做出僭越之举。圣者贝努被称为「至明」，圣者谢斯芬赫被称为「至圣」，根据须弥典籍记载，分别用于指代「人的灵魂」和「镇灵」。出于对人类和魔神的排斥态度，龙之王阿佩普是唯一拒绝接受称号、不曾列席的七柱。[39]鳄之王索贝克是赤王麾下的最高统帅，七柱中司掌战争的王。在小说的描述中，鳄之王从前也是一位少年，并且有着苍青色的面容，不知是否可信。图特是赤王身边负责记录的书记官。鹮之王与鳄之王两人素来争论不休，大到世界法则，小到日常琐事，一言不合就开始争辩。赫里沙夫是宰相负责协助赤王统领整个沙漠王国，拥有极大的权力。贝努是守护在赤王左右的二位圣兽之一，代表着赤王的崇高灵魂。相传贝努能够庇护人们不受风沙伤害，反之也能唤起狂躁的风沙。谢斯芬赫是守护在赤王左右的二位圣兽之一，代表着赤王的无上意志。
