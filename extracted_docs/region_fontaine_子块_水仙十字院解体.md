# 水仙十字院解体

此时枫丹经历了「第一次涨水期」，水仙十字院在此时被淹没。没有什么是永恒的，水仙十字院迎来了属于它的结局。在深渊兽潮来临、十字院被迫遣散前，时任副院长贝瑟将孩子们托付给信赖的人收养。贝瑟在十字院淹没前将四名孩子分别托付给卡尔和埃马纽艾尔。至此，埃马纽艾尔才不情愿地再度尝试与人一同生活，尝试为了儿女创造一个安全的世界。于此同时，自然哲学院的学院长德怀特·拉斯克下令封闭学院，以免血气方刚的青年外出送死。
海军对抗厄里那斯时，枫丹海军丧失大部分的舰队。海军司令贝瑟·埃尔顿主动退役，尔顿的大副纳撒尼尔·皮克曼最终失去了归属，余年只得彷徨在英雄虚名中。皮克曼曾经奔走多年，企图说服枫丹廷重启「斯彭西安」铁甲舰的打捞与重建。但努力无果而终，意欲重建枫丹海军的想法也被战殁者遗属以为是在沽名钓誉…[47]「斯彭西安」则一半留于「厄里那斯」腹中，一半散落在海里[48]。活下來的皮克曼其實是個逃兵，因为最终贝瑟的指令要求大副和轮机长随自己留下，其余人弃船撤离[48]。
