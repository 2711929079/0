# 丑角

未能阻止君王引入深渊力量，未能阻止故国覆灭的皮耶罗，在灾变后遇到理解他伤痛的冰之女皇，两人共同创立“愚人众”。既然沾染的同胞的血无法洗净，那索性成为嘲笑命运的「丑角」。既然我的才学无法与「贤者」比拟，没能获得上一位王者的青睐，也未能阻止他们撕开深罪的纱笼，招来神怒、毁灭与痴裕的狂潮，那就索性成为笨拙的「愚人」，为理解我的伤痛的「陛下」效忠…我是名为「丑角」的皮耶罗，请听我说：骄傲的愚人同士们，心怀怒火与永恒的寒冬吧。既然我们彼此都见识过世界定理的荒谬与冷漠，那就一同戴上嗤笑世界的面具，去改写天理吧。[37]
坎瑞亚小队瑟雷恩拥有机械制的心脏，被列名为天柱骑士（由「列名」可知天柱骑士不只一人）。尚未加入愚人众成为执行官，当时率领一批坎瑞亚小队的坎瑞亚人「天柱骑士」瑟雷恩在战役中且战且退来到纳塔，见证战役带来的巨大牺牲和深渊带来的恐惧与绝望。在纳塔，他认识了当时烟谜主首领阿伊祖，两人互相交好，瑟雷恩因此学会烟谜主的法术，能将外溢的灵魂送回地脉。同时他也从阿伊祖口中得知火神之心的用法[57]。当时因为深渊力量影响，小队员罹患「渊蚀综合症」陷入幻觉与疯狂，甚至自相残杀，瑟雷恩的副官古瑟雷德将自己当作实验体研发能克制病症的清幻剂，却因此牺牲[58]。虽然故国已经覆灭，瑟雷恩仍然为异乡而战。因夜神之国遭破坏，逝去之人的灵魂无家可归，他将逝去人们的灵魂容纳在自己的机械心脏内。五百年来，日复一日的承受体内灵魂的哀鸣，导致其从未入眠。瑟雷恩被降下不死诅咒，肉体在漫长时间中腐烂于是戴上面具。在加入愚人众后曾见到旅行者的血亲与戴因斯雷布一同旅行。[57]
