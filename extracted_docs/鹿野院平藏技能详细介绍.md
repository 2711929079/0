# 鹿野院平藏技能详细介绍

鹿野院平藏是《原神》中的风元素四星角色，稻妻天领奉行的侦探，称号"心朝乂安"。
元素战技「风风轮」：
- 释放风元素能量，对周围敌人造成风元素伤害并击飞。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「万风归宗」：
- 创造一个持续8秒的风元素领域，领域内的敌人会受到持续风元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "风风轮的精妙"：风风轮命中敌人后，有15%概率触发"扩散"效果。
- "风元素增幅"：鹿野院平藏的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "归宗的威力"：元素爆发持续时间内，鹿野院平藏的风元素伤害提升15%。
