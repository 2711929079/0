# 降下天钉

葬火之战后，天空岛降下了修补大地的天钉，以此阻绝禁忌知识在地脉中的扩散。很多透过地脉维生的第一文明分支，也因此灭亡[41]。如今世界各处的地灵龛是大地上矗立着的远古神龛，随着那个文明（第一文明）的覆灭，也将自己封锁了。在幽深的秘境中寻得的密钥能破除封印，或许只是利用过去的气息，让神龛忘记一切尽毁之事吧。
天使天使爱上了一位外来者，并在三月女神的见证下立下结合的誓约[36]。仅三十日后，灾祸发生，天使和外来者在崩裂的天地间逃亡。后来由于天使为使用禁忌知识的人类求情，天空降下惩罚，让天使散失形体成为仙灵[41]（既然天使在经历天崩地裂的灾祸后就被惩罚，那么这个“灾祸”（仙灵之乱）指的大概率就是葬火之战。）
