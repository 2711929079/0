20 | 花神/娜布·玛莉卡塔 | 跋尔松 |  | 已逝 |  | 矫论团的那先朱那称花灵为“跋尔松”的眷灵。如果花灵被称为“跋尔松”的眷属，那跋尔松有可能是花神的神名，因为花灵是花、草、水三神共育的神鸟分裂的灵光。普尔森purson的描述：他能隐藏或是显现那些被隐蔽的宝藏；他能看见任何时间所发生的一切，并能为追随者带来强大的仆从；他能为万物带来真正的答案，但却会引诱寻知者询问那些他们无法承担的问题。相传许多书中都有提到祂原本是座天使或力天使堕天之后变成的恶魔。”
带来真理、强大的仆从，引诱询问无法承担的问题，由天使堕落，也都能符合花神的故事。花神原为天使，而天使受诅咒变成的仙灵也会显现被隐蔽的宝藏。

55 | 海祇大御神 | 奥罗巴斯 |  | 已逝 | 被影斩杀，祟神被二度镇压世界任务白夜国晨昏记 | 通晓古今，散播着神圣的真相和创世之理，对于神与天地创造等事情相当精通，对应奥罗巴斯曾偷看纪录古今历史真相的禁书日月前事。平抑仇恨，对应奥罗巴斯帮助白夜国民对抗摆脱深海龙蜥的威胁终止人们与龙蜥的冲突，以及献祭自己化解天理的敌意。五十五魔神奥罗巴斯的名字是Orobas，但原神中的奥罗巴斯名字是orobashi，与“大蛇”的罗马音orochi有对应关系；而orobashi作罗马音时对应的汉语又是“远吕羽”，遂有“远吕羽氏”这个称呼。设计成蛇形而不是72柱魔神里原本的马形，相信是参考了和「大蛇/遠呂智（Orochi），如八俣远吕智即八岐大蛇」读音相近。另外读音和Orobas相近的Ouroboros（衔尾蛇）也是条蛇。白夜国地理水文考中提及在奥罗巴斯来到白夜国前，白夜国民曾崇拜幻想出的蛇奥罗巴洛斯，即Ouroboros（衔尾蛇），且无鳞蛇为「奥罗巴洛斯」，有珊瑚的蛇为「远吕羽氏」，因此奥罗巴斯被白夜国民称为远吕羽氏。第二十九柱魔神伊斯塔露右手持一毒蛇，原神中伊思塔露与白夜国历史高度相关，虽然后来奥罗巴斯来到白夜国之事与伊思塔露无关，但或许此毒蛇能对应奥罗巴斯。
