# 玛薇卡

玛薇卡在500年前原本是一个普通的纳塔人，为悬木人部族的成员。幼时，她曾经向回声之子的首领松伽塔学习技艺，声称自己要当火神。松伽塔本不想收她为徒，但她在几十米远的地方射中了松伽塔家中的玉米片，松伽塔大为惊奇，于是收她为徒。同为悬木人的布基纳获得古名「马力卜」时向玛薇卡炫耀，玛薇卡表示自己会有比他更厉害的古名。后来通过在「归火圣夜巡礼」中脱颖而出，升华为火之魔神。上任前几天，她的朋友、来自沃陆之邦的万杰鲁在摔跤比赛中难得胜过了她，感慨以后没时间和她摔跤了。迄今为止，玛薇卡从未有过败绩，被认为是历来最强大的火神。[10]特诺切屡次挑起各个部族的冲突，于是经回声之子的首领松伽塔请愿，玛薇卡组织投票将其放逐。[97]
坎瑞亚灾厄灾厄因为坎瑞亚并不存在稳固的地脉，纳塔成为深渊侵蚀最严重的国家。在灾厄过后，纳塔人花费数百年的时间才把深渊造成的灾厄逐渐解决，但这只是表象而已。纳塔的地脉早已被侵蚀殆尽，被彻底深渊化只需500年时间。之后的纳塔不断遭遇深渊入侵，纳塔人为了守护纳塔也变得较不常前往其他国家。[10]娜娜丝梨原本是特诺奇兹托克的特产，后来在五百年前灭绝了[113]。传说伟大的祭司将手中的镜子放在了地上，于是形成了镜璧山，以抵挡漆黑的奔流[114]。维护镜璧山的涂鸦需要代代相传的努力，据说曾有睡在巨大涂鸦画下的人于睡梦中梦见巨鸟张开翅膀的模样[115]。镜璧山上的紫色鸟涂鸦，可能就是雷鸟的形象。这个涂鸦也很像卡帕奇莉，或许就是迁居清籁岛前的卡帕奇莉或她的同胞。
