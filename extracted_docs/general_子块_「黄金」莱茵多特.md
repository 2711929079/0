# 「黄金」莱茵多特

被称为「黄金」，是坎瑞亚首席炼金术师，「五罪人」之一，犯下了「渎生之罪」，曾持有单手剑「洛格瑟姆」（现名「蛇噬」）[102]。制造了许多的魔物，这些魔物曾经给提瓦特大陆带来了很大的灾难，如今这些魔物仍然有一部分留在提瓦特之中。莱茵多特是代号为R的魔女会元老，阿贝多的母亲（制造者）及师父，教授了阿贝多炼金术知识。与阿贝多在地底寻获圣遗物「纳贝里士之心」后失踪，留下字条和推荐信让其前往蒙德投奔自己的老友艾莉丝，并完成最后的课题：向自己展示世界的真相，与世界的意义。此后莱茵多特吞噬「纳贝里士之心」与生之执政「纳贝里士」融为一体[103]；阿贝多则被同为魔女会成员的艾莉丝收养，并借助她的关系进入西风骑士团，担任首席炼金术师兼调查小队队长。[104]中文名莱茵多特英文名Rhinedottir外号黄金、魔女R汉语CV范楚绒/狄菲菲第一个配音仅在3.5版本剧情PV「魔女的茶会」使用，第二个配音从5.7版本剧情PV「神的局限性」开始使用。日语CV田中敦子（已故）/山口由里子英语CV艾维·杜普勒韩语CV소연所属坎瑞亚/四影/魔女会职业坎瑞亚「五罪人」/生之执政/魔女「R」
