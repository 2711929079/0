# 天遒谷

根据一些典籍上的记载，天遒谷那边的遗迹，在遥远的过去曾经是繁荣的集镇。 之所以变成了现在的样子，似乎是和某位古老龙王的动乱有关。虽然那场动乱没什么详细的记载，但据传说，最后是由帝君与仙家一起将它囚禁在了洞天的樊笼中。而它的眷属则遁逃至了地下，陷入深眠，等待着那位古老龙王复苏的那一天。
