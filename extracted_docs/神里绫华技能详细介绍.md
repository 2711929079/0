# 神里绫华技能详细介绍

神里绫华是《原神》中的冰元素五星角色，稻妻神里家的大小姐，称号"白鹭霜华"。

普通攻击「寒天宣命」：
- 效果：进行五段连续攻击，造成冰元素伤害。
- 重击：消耗体力，造成高额冰元素伤害。

元素战技「冰华绽放」：
- 效果：以自身为中心，造成冰元素范围伤害，并击飞敌人。
- 冷却时间：10秒。
- 能量消耗：无。

元素爆发「霜雪千年」：
- 效果：释放冰风暴，向前方移动并造成持续冰元素伤害。
- 伤害特点：多段伤害，总倍率高，是主要输出手段。
- 持续时间：5秒。
- 冷却时间：20秒。
- 能量需求：80点。

天赋技能：
1. 寒雪兆丰年：元素爆发期间，普通攻击和重击伤害提升。
2. 冰护：受到冰元素伤害时，获得伤害减免。

角色定位：
- 主要定位：主C输出
- 次要定位：副C
- 核心优势：高额冰元素爆发伤害、冻结控制

培养建议：
- 武器推荐：雾切之回光、天目影打刀、黑剑
- 圣遗物推荐：冰风迷途的勇士四件套（冰元素伤害加成）
