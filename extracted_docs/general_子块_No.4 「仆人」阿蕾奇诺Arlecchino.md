# No.4 「仆人」阿蕾奇诺Arlecchino

中文CV：黄莺 日文CV：森奈奈子详见至冬#壁炉之家本名佩露薇利。其前任为库嘉维娜，拥有水之邪眼，自称“母亲”，但性格残忍，漠视生命（强制孩子们互相进行生死决斗，以此决出其中最强的「王」，就算是侥幸未死的败者也会被送给博士去进行人体实验，这也导致阿蕾奇诺自己对其极其痛恨），后来被佩露薇利杀死。佩露薇利也因此入狱。后被女皇赦免，成为了现任「仆人」，改名“阿蕾奇诺”，自称「父亲」，并做出了诸多改革，取消了部分不人道规则。在稻妻隐藏世界任务特别的御神签中得知，一部分愚人众士兵来自「仆人」阿蕾奇诺负责的孤儿院，该系列任务的主角柳达希卡便来自该孤儿院。任务剧情中柳达希卡的上司“老师”坚持要求柳达希卡协助针对海祇岛的破坏行动，并表示由该行动造成的孤儿就是孤儿院的生源。随着4.0枫丹地图更新，世界任务特别的御神签完成后，百代会更名为千世出现在离岛的港口处（眼睛还是瞎了），千世的描述中，「壁炉之家」曾经爆发过严重的冲突，「仆人」这一席位在那场冲突中发生了交替，阿蕾奇诺在这次事件干掉了前任「仆人」成为了执行官。在4.1版本魔神任务中正式登场，并亮出席位是第四席。4.6版本成为自机角色，拥有火神之眼和火之邪眼（周本建模后脖颈处有邪眼），具有赤月王朝血脉，拥有虹月之力和深渊之力。枫丹魔神任务中，试图袭击芙宁娜并获得水神之心，把对方吓得求饶并且留下了极为严重的心理阴影，由此判断出芙宁娜不是水神，并放弃了袭击计划，也因为此，她在接下来的谈判中占据主动权。危机爆发时，积极参与拯救枫丹计划，并带领自己的人马帮助刺玫会带领白淞镇镇民前去避难，危机结束后，那维莱特将水神的神之心外交赠予了她。在传说任务净炼火之章第一幕炉火熄灭之际中，看破了旅行者、派蒙以及林尼众人隐瞒克雷薇的计划，要求旅行者在必要的时候，做出最合适的选择，帮一次壁炉之家。然后她把壁炉之家内的“叛徒”们以及克雷薇带到了决斗场，意图对其处刑。期间林尼等人提出挑战她，得到了同意，旅行者也做出了“最合适的选择”一同挑战。由于绝对的实力差距，众人被她击败。但她知晓林尼等人的决心，并早已计划换一种方式对叛徒们进行处刑——让他们喝下「瓶中之火」，消除他们在壁炉之家长大的记忆后给他们安排新的身份和新的生活，使他们成为枫丹普通公民，其中只有林尼等三人没有喝下这一药剂。在空月之歌中，旅行者和哥伦比娅离开设计局后，又一次陷入险境。她前往挪德卡莱帮助对付雷利尔，使用周本形态击退了狂猎。三人而后一同前往银月之庭取回虹月月髓，随后虹月月髓存储于她体内，力量因此有些许增强。她在察觉「博士」阴谋之后回到至冬，传达冰神希望将其召回，得到了同意。但实际上「博士」可能只有一个切片回去或者直接阳奉阴违。如今她决定与「博士」正式决裂，与旅行者等人一同对抗他。月之四剧情中，参与了至冬人员疏散工作，并与法尔伽、尼可一起将霜月拉进提瓦特的天空，将术式推进到100%，助力哥伦比娅重新降临。她的命之座图案就是她的执行官代表图标，一只攥住蜡烛的手。
执行官互评折叠「公子」：我不讨厌有自己打算的人。毕竟我加入愚人众，也是为了获得更多争斗的机会。但这个人，我不喜欢。如果背叛能够获得收益，她会毫不留情地向女皇大人挥剑。她的眼里只有疯狂。「散兵」：道貌岸然的伪君子。她对人展露「优雅」与「亲切」都只是为了更好地「控制」罢了。而直面过她疯狂一面的人，大多已经…呼~。「少女」：壁炉之家孩子们很多，有时会有些吵闹。我听见他们都叫阿蕾奇诺「父亲」，以为这是在壁炉之家时需要遵守的规矩。于是我在阿蕾奇诺经过时，也那样叫了一声。然后我才知道，原来壁炉之家还能那么安静。
