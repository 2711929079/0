# 人类诞生

天理到来四百余年后，人类受造诞生。[5]
月之轮那时，高天所降下的律法尚未结集，人理的边界尚未划定。神的子民散布在新造的园囿与山谷，其数目不可胜数。天空的使者行走在蒙昧的人群当中，播散繁荣与智慧。最后，神所许诺的繁荣启迪了智识，智识引发了怀疑。为了寻求答案，人们选出了祭司；因为恐惧答案，祭司膏立了君王。神的使者为凡人的愚妄而震怒，面对他们的疑问，御使们缄默不言。于是人们转而望向天空，在日光不及的黑夜中为三轮皓月献上祭礼。夜空之上，梦与诗歌的女儿们（三月女神）无时无刻不在观察着尘世的变迁。在阿兰若国的酣息中，在七十层的阶梯下，在银璨璨的舫舻边。直到她们听见，这些可怜小人儿的心房，也会随着月震而悸动。夜幕为大地垂下了眼眸，正如银星曾为龙众的天船亮起航灯。她们将疯狂与灵感带回月下，那是「秩序」无法给出回答。当晨星再度升起，自山巅归来的主祭带回了一枚小小的轮符。这枚「月之轮」，便是三月与众民订下的新约[2]。
