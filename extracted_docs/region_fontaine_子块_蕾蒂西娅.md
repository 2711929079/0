# 蕾蒂西娅

尽管贵族们将灰河视为肮脏之处，仍然有许多贵族对该地抱有好奇与向往。出生便不愁吃穿的大小姐蕾蒂西娅·兰道夫因为向往冒险，以及为了拥有和好友聚会聊天的话题来到灰河。一潜入该地就因为过于瞩目即将遭遇危机—所幸她先前认识的里德为她引路。她在灰河没有想回去的念头，和里德有了孩子。灰河整肃爆发时，蕾蒂西娅帮助人们对抗枫丹官方，失败后她、里德和孩子仍然被父亲接纳。然而事后在兰道夫家，她才回想起逐影猎人们潜入的暗道不应被外人知晓，也就是出了内鬼，而此人就是她丈夫里德，他将秘密泄露给兰道夫。事前兩人相愛時，里德曾對蕾蒂西婭說過：「蕾蒂西娅，我敬爱你高洁的灵魂。」「我们将不再会是砍向树木的斧柄。」「若有一日，我竟耽于尘世的荣华，」「那时，就由你来裁决我的命运吧…」于是，蕾蒂西娅舉起她的大斧，对丈夫做出最终的裁断[43]。兰道夫家族存有后人[44]。
白色舰队过去的辉煌年代，纯白色的铁甲旗舰「斯彭西安」曾是枫丹海军的骄傲。那个时代里，装备着庞大重炮与坚固冲角的铁甲舰队乃是枫丹廷的宠儿，在光荣剧院的时代，观众无不为白色舰队清扫魔像军团的故事起立鼓掌。
