# 伊黎耶岛西

自第一次涨水期以来，欧庇克莱歌剧院便伫立在被视作祝圣之地的伊黎耶岛高处，俯瞰着众水之上的大城，也昭示着新时代的律政与公义。不过，即便是在新的秩序中，一些古老的传统也依然得以留存。时至今日，新婚的恋人们依然会在露景泉前投下硬币，祈愿幸福美满的生活。
