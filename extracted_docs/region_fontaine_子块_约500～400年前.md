# 约500～400年前

水仙十字四人组雷内·德·佩特莉可与雅各布为“白淞之围”中犯罪者的子嗣，被送入水仙十字院养育，与阿兰、玛丽安为童年玩伴。坎瑞亚灾变前被副院长托付给卡尔·英戈德收养，随养父四处冒险。卡特·谢尔比乌斯是水仙十字院孤儿，阿兰四人的前辈，
阿兰·吉约丹也是水仙十字院孤儿，养父为埃马纽艾尔·吉约丹。妹妹玛丽安·吉约丹也是水仙十字院孤儿。雅各布·英戈德是水仙十字院孤儿，原名雅各布·贝克，生父为爱德华多·贝克[41]，养父为卡尔·英戈德。雷内·英戈德是水仙十字院孤儿，原名雷内·德·佩特莉可，生父为雷诺·德·佩特莉可[41]，养父为卡尔·英戈德。与养父在须弥的甘露花海冒险时，研究了灵光与深渊力量的性质，雷内发现甘露花海圣莲的构成与自己和雅各布有一定相似之处，但与养父不同。雷内想要进一步采样研究，但被那伽朱那团拒绝。探索途中，雅各布接触深渊力量陷入虚弱和昏迷，雷内研究后决定让雅各布继续吸收，雅各布恢复，但已受深渊力量改造变得无需进食。[55]雷内通过坎瑞亚遗留资料推导出了「世界式」，得知世界会在第二次涨水期毁灭。之后返回枫丹探索，阅读前文明的书籍时得知前人把关于文明灭亡与新生的算式称为「法图纳」，但「世界式」所推演的结果是文明毁灭后不会再有新的文明诞生，除非考虑引入系统之外的「变量」。此后雷内又在枫丹发现了前文明的「金色剧团」遗留的魔法术式「密合之约印」。二人溜进厄里那斯残骸体内调查，雷内认为其中的力量可以增强雅各布，但害怕伤害到雅各布而不忍心让其吸收，雅各布自主吸收了厄里那斯的血液，仅有呕吐的轻微反应。继续进食厄里那斯血肉增强雅各布力量后，二人尝试开启了「通道」，出现了造型与狗狗有相似之处的怪兽。雷内假定「通道」连接了某个充满了此类怪兽的地方，并将其称为「兽境」，将出现的这类怪兽命名为「兽境猎犬」。于此同时，已经加入自然哲学院的阿兰在养父埃玛纽艾尔的监督下，和玛丽安一起在「发生过战斗的地方」寻找灵感，四人意外重逢。[56]雷内没有提雅各布的事情，因为他不信任曾参与过白淞之围的埃玛纽艾尔。[57]经由阿兰向学院长推荐，雷内与雅各布来到自然哲学学院，加入阿兰的实验室。雷内向阿兰说明了先前的末日预言，希望能改变研究方向，但「世界式」与阿兰的计算结果不符，阿兰不相信雷内和雅各布看到过的东西。[58]雷内和雅各布只得拉拢学院的其他人，并整合他们的力量与资源，使用揭示之书向别人展示自己所见的末日图景。与阿兰兄妹和卡特在佩特莉可田野调查时，雷内发现遗迹与「金色剧团」有关，并找到了一张残破的地图。得知了卡特的病情，雷内意识到不能继续与阿兰沉迷机械研究，和雅各布向学院长要了新的实验室，并私下进行深渊力量的研究，期间研究并命名了「深赤之石」，研究了层岩巨渊的黑泥。雷内为了提供手下组织和归属感，用「水仙十字院」命名「学会」为「水仙十字结社」以委托信徒办事。结社成员将雷内称为「大师」。将雅各布称为“英戈德先生”[59]。雅各布提议让卡特也接触深渊力量，二人向卡特展示了雅各布的变化，说明了其中风险，卡特答应了，但融合没能像雅各布那样顺利，卡特持续劣化，成为了某种难以言说的东西。雷内与阿兰因卡特之事决裂，雷内下定决心一定要找到能够将卡特救回来的方法。与雅各布重返佩特莉可以及之前的遗迹考察，找到了有关原始胎海之水的资料，发现古人曾经成功“回归起源”，雷内由此找到了借助深渊力量以外第二条应对末日的道路。雅各布将卡特剩余的残骸物放入丘丘风行游侠的身体中，又用「水仙十字圣剑」的救赎之力，为其赋予记忆与智慧，卡特皮拉从而诞生，他在结社内也被称为「幻光蝶蛾之蛹」。雷内找到一处胎海水的决口，决定在上方建设「自体自身之塔」（又称“格式塔”）。雅各布指示结社成员主动向沫芒宫报备建设工程，利用结社内的贵族成员影响力作掩饰，并让逐影猎人负责安保工作，打消枫丹廷的疑虑。二人回到十字院旧址看望院长，雅各布将副院长的墨绿羽饰留在了此处，[39]雷内发现十字院旧址很可能是古代黄金剧团的设施。雷内研究了古代术式中「四象限圆环」和「流涌之树」等内容，考察了多个封印点和四要素的位置，并对四象限进行了改造，取得能够割舍人格的「水仙十字圣剑」，为最终的术式做准备。雷内重返十字院，将怀表（「宝物」）留在此处，并说服院长前往格式塔，利用其能溶解和吸收意识的能力。在最后的仪式中，雷内割舍了自己的人格，并将自己溶解，计划“借由水再次诞生”。然而计划失败了，雅各布保存了雷内的意识。在此之后，「大师」又吸收了三名追查结社的逐影庭探员（其中第一位是埃马纽艾尔·吉约丹[33]，第二位是玛尔菲莎[42]），从他们的记忆中发现枫丹廷官方已经对结社的存在，并得知雷内疑似已经研究出可以移植意识的器官。加入逐影庭的玛丽安带着阿兰所研发的原型机西摩尔[60][61]秘密追查水仙十字结社，与雅各布等社员在厄里那斯体内展开了大战，迸发的碎片零件等划伤了厄里那斯的体内，诞生了美露莘这一种族，其与生俱来的“信物”便是当时大战的残留物。最终雅各布存活，继续践行深渊力量救世的道路；玛丽安被原始胎海之水溶解同化，回到「安眠处」住下，捏造出对应着曾经伙伴们的水形幻灵，演绎着童话。[62][63]发条机关数百年前至今，枫丹陆陆续续出现许多新的发条机关。攻坚特化型机关与“侦察记录型”一样，是最早设计制造的新式发条机关机械之一。名称是用来纪念发明者的友人西摩尔而起的，最初设计的目的是帮助人。但后来的使用者普遍认为用它攻击人较之用人攻击人的相对优势更大。[64]区域警戒型是在枫丹科学院建立之前，自然哲学学院的相关研究中产生的试验装置。因此，尽管作为警戒与防御设备取得了相当的商业成功，大多数工程师并不将其视为严格意义的发条机关机械。[65]过去在枫丹曾经发生了巨大魔兽入侵的悲剧，在陆上大湖受到景仰的白色铁甲舰队为了护国全军覆没。深海攻击型的设计者希望如有此类事件再次发生，至少能在水中以强大火力出击的发条机关能先于将官与士兵成为第一道防线。[66]
