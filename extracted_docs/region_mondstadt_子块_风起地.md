# 风起地

群风环抱的谷地，一棵巨大的橡树「温妮莎之树」矗立其中。橡树的树荫庇护着树下的七天神像不受风雨，正如曾经的温妮莎庇护着蒙德一般。蒙德唯一的大平原，推测是风神巴巴托斯以这里为起点，改良了蒙德的地形。三个约会圣地之一。
