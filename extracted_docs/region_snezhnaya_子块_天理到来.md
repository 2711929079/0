# 天理到来

一个又一个千年过去，月的三姐妹始终没有等来她们的旧主，直到那一日，身披羽翼的第一降临者（三月女神称其为「高天的主宰」，「天理」为葬火之战后世人们所称）踏着晨星而来。那并非预言中将要熄灭群星的大敌，似乎也不打算掠走维系世界存续的泉源，因此三月女神并不排斥其到来，双方相安无事。「高天的主宰」逐渐开始对世界显露出改造与支配的欲望，它的规则中存在一种龙王不曾教给她们的「秩序」，「恒月」艾莉亚和「霜月」卡侬想要知道这种秩序对提瓦特来说是好是坏，而「虹月」桑娜妲对此持反对意见[3]，代行星球意志的三位守护者不知所从，争论不休。但无论臣服抑或反叛，战斗还是死亡，万军之王都将无情碾过阻碍它的一切。这是它为人之子选定的新世界，依循它的规划大地与天空皆会焕然一新。在白日的征服告终之前，夜的侍者唯有静观其变。[2]
改造世界衔枝后四十余年，七位元素龙王被「高天的主宰」击败，七个王国全部对天上俯首称臣。衔枝的四百余年内[5]，「高天的主宰」开始了对世界彻头彻尾的改造。箱舟停驻，创造之事已毕，山川与河流相继落成，地底下、沧海里容纳了旧世界落败的族裔与王酋。蛋壳中的世界之主用虚假的帷幕隔绝真实的魔天，遮蔽了一度使统辖兽群的诸王也无法安眠的恐怖。暴烈的日光清扫了天空，但新的王座却仍为月夜的三姐妹预留了天上的位置，本质上三月女神维系看这颗星球的存续与运转，与天理的目的并无冲突[6]，或许因为在它的规划中，这个连元素的奔流也井然有序的世界依然需要潮汐。三月女神明白，仅凭她们无法从必将到来的堙灭中守护这颗孤旷的行星。在最后的寂灭到来前，无论谁在善恶的斗争中取胜，总好过同归虚无的永劫。也许她们身陷迷雾的旧主永远也无法找到解答，又或许那个答案已近在眼前。正如她们所期望的，高天的主宰厌弃黑色迷雾中的毁灭者。想必在穿越暗域的漫长旅途中，它已目睹无数陨灭的星辰。于是她们也选择了「秩序」，或者更准确地说，容纳了它。那是三月与晨星共治的时代，那是永恒依旧为永恒的时代。[2]
