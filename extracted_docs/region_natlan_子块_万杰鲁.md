# 万杰鲁

天生就具有奇力的万杰鲁出身于沃陆之邦，一直渴望能拥有一把趁手的武器，但是普通的材料都无法承受她的力量，直到她听说了龙族昔日的圣城图兰中仍有龙族残存。万杰鲁孤身一人前往了图兰，与一只龙形的金影战斗，最终万杰鲁徒手打败它，并获行了一段龙骨。万杰鲁后来用这段龙肯打造出了一把长剑「弥坚骨」[111]。沃陆之邦有一位沉默寡言的少女名叫“菲耶希娜”，经常瑟缩在篝火旁，全神贯注地聆听万杰鲁讲迟英雄传说[112]。
玛薇卡玛薇卡在500年前原本是一个普通的纳塔人，为悬木人部族的成员。幼时，她曾经向回声之子的首领松伽塔学习技艺，声称自己要当火神。松伽塔本不想收她为徒，但她在几十米远的地方射中了松伽塔家中的玉米片，松伽塔大为惊奇，于是收她为徒。同为悬木人的布基纳获得古名「马力卜」时向玛薇卡炫耀，玛薇卡表示自己会有比他更厉害的古名。后来通过在「归火圣夜巡礼」中脱颖而出，升华为火之魔神。上任前几天，她的朋友、来自沃陆之邦的万杰鲁在摔跤比赛中难得胜过了她，感慨以后没时间和她摔跤了。迄今为止，玛薇卡从未有过败绩，被认为是历来最强大的火神。[10]特诺切屡次挑起各个部族的冲突，于是经回声之子的首领松伽塔请愿，玛薇卡组织投票将其放逐。[97]
