# 介绍总览

## 介绍
即原初之人法涅斯，第一位降临者，击败七龙王后改建世界，为了统领生死时空的权能分出四个影子，将原始的蛮荒燃素改造成较温和的七元素力，为了防止星间巨物的入侵，透过虚假之空隔绝提瓦特与宇宙，创造人类与飞鸟与制度，透过天使指引人类建立文明。在古龙衰落后，世界的新主人法涅斯接管了三月，并透过三月的力量打造神之眼的前身「月之轮」。三月女神起了分歧，有的认为应该效忠创造她们的旧主尼伯龙根，有的认为不应该追随沾染可怕深渊的牠。月之轮一事被龙族视为三月女神的背叛，尼伯龙根归来后，对三月管理权被夺十分愤怒，囚禁月神，将三月改造为对抗天理的天基武器。尼伯龙根向天理发动葬火之战，在葬火之战中，天理用「长剑」击破两个月亮，两个碎成碎片，另一个被丢出虚假之空外，只剩霜月苍白的尸身。葬火之战时，其战争中掉落的神体碎片化为魔神。在五百年前開始可能是因為喪失過多力量而陷入沈睡，就連七執政神座被毀也無法喚醒，在此之前将双子的飞船藏匿于霜月上。他的四个影子即四影执政。法涅斯的影子是女性，那么其可能也是女性，但「丑角」在神的局限性PV中称天理为「他」，考虑到希腊神话中法涅斯有着双重性别，《原神》中的或许也是如此。但在真实之月中，三月女神称天理为「它」。
