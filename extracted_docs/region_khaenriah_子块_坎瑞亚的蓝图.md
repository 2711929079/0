# 坎瑞亚的蓝图

命运的织机需要重建坎瑞亚的地脉完成复国，需要坎瑞亚的所有记忆，然尔深渊教团的人因记忆磨损无法达成。于是深渊教团举全兵力攻入坎瑞亚引诱戴因斯雷布出现，再由戴因的熟视仇人哈登诱骗他回想坎瑞亚的过去，导致他的所有关于坎瑞亚的记忆被命运的织机读取，坎瑞亚地脉的蓝图完成了。此事在夜神的帮助下完成的，但夜神向深渊殿下谈了某个条件作为代价。坎瑞亚的地脉虽完成却无处存在，深渊殿下只得寻找承载坎瑞亚地脉的方法。同时，因伊斯塔露暗中操作，旅行者将自己的飞船休眠舱钥匙给了五百年前的蒂莱尔（旅行者也可以将钥匙收回，但结果是一样的，因为五百年后深渊殿下也会受到伊斯塔露的干涉见到蒂莱尔，为了防止收回钥匙导致涟漪效应使世界走向未知的命运使自己的计划（即自己被降格了，所以成立深渊教团用一整个世界与天理对抗打算获取与之平等对话的资格恢复降临者身分，和让新坎瑞亚成为自己合适的新的家乡）不能如计划中进行，于是将自己一样的另把钥匙丢给蒂莱尔。旅行者在内心纠结下顺从殿下的计划）。经历各个事件，旅行者与殿下重逢，问到他/她做的事对我们自己而言有意义吗？殿下会答：「从一开始，就只关乎你我」。因此，殿下帮助坎瑞亚复国是与旅行者和自己最初旅行的目的相同的，而他们旅行最初的目的是在故国覆灭后找到适合的新的家乡（见双子的旅程），因此旅行者的血亲重建坎瑞亚的目的，大概是为了将重建后的坎瑞亚弄成适合当成自己和旅行者新的家乡的样子。在殿下表达自己打算利用整个世界换取与天理平等对话的资格时（即自己因为被降格了，所以成立深渊教团用一整个世界与天理对抗打算获取与之平等对话的资格恢复降临者身分），天理的维系者疑似听到了此对话。之后，殿下离开[28]。
补充：后续坎瑞亚剧情发展之猜测以下基于新剧情补充的猜测，非正文。在你所在的时空中，深渊殿下取得坎瑞亚的地脉，但因为地脉数目被天理固定，如今的提瓦特无处安放，她/他必须寻找能安放地脉的地点。须注意如今提瓦特无处安放，因此能安放的地方只有过去或未来的提瓦特。再假想深渊殿下找到了安放地脉的地点，但其实是伊斯塔露暗中操作让他/她到坎瑞亚建国前的提瓦特，他/她看到此处能安放地脉没有多想殊不知这是千百年前的提瓦特就安放了地脉，于是「新」坎瑞亚出现了。因此，「新」坎瑞亚和「旧」坎瑞亚其实是相同的，「旧」的坎瑞亚其实就是千百年后旅行者的血亲穿越回千百年前安放的「新」坎瑞亚。所以说其实「旧」坎瑞亚是现在的旅行者的血亲无意中「建国」的。因此提瓦特陷入一个循环：五百年前坎瑞亚覆灭，旅行者血亲成立教团，回到千百年前安放坎瑞亚地脉，坎瑞亚于千百年前建国，五百年前坎瑞亚覆灭，旅行者血亲成立教团⋯如此循环。为何伊斯塔露要做这样的事呢？降临之剑中提到宇宙法则是生者必灭，世界注定毁灭。丝柯克语言提到万物终有终点，时间也不例外。深廊终曲提到提瓦特终将毁灭。如果时间线继续走下去，提瓦特注定会毁灭。正如雷电将军介绍—「薙除俗世芜杂的执妄，只为扭转生灭轮回的羁缠。」，雷电影的做法是让时间线停止，所以强求稻妻停止发展，扭转宇宙生灭的命运，但明显行不通。因此更好的方式，是让提瓦特陷入「循环」—因时间线是环形不是线形就不会通往毁灭。因为世界终将毁灭，这应该是天理唯一能想到保住世界的方式。但如今我们看伊斯塔露都没有这样操作，或许其实是早就操作完毕了，即干涉旅行者血亲的行动导致提瓦特时间线陷入循环。否则，天下人之章第二幕中透过操作时间让稻妻种下雷樱树使其在坎瑞亚灾变时没有覆灭的伊斯塔露为何要让坎瑞亚毁灭呢？让旅行者的血亲及时吸走深渊力量拯救坎瑞亚，这样深渊就不会扩散就不会危及提瓦特了不是吗？日月前事中提及，伊斯塔露是唯一听取白夜国民的祈祷帮助他们的，因此她应该是爱人的，应当不会做出毁灭坎瑞亚的举动吧。卡利贝尔中提及双子希望在世界某处找到开满因提瓦特花的花海，而双子旅行的目的是为了找到新的家乡，因此要找的因提瓦特花的花海是希望最后被他们当成家乡的地方也能有该花的花海。而旧坎瑞亚也有这种花，似乎太巧了点。或许是深渊殿下重建新坎瑞亚的时候自己添加的，而新坎瑞亚即为旧坎瑞亚。动画短片未行之路的双子旅行的记忆闪回（双子未曾于提瓦特旅行过，飞船只曾经略过提瓦特），妹妹的头上有因提瓦特花，但双子旅行时她没到坎瑞亚，理应不会带着坎瑞亚国花（不过这些画面也可能是想像）。除非因提瓦特不是坎瑞亚独有。而既然妹妹会带着因提瓦特，其可能是来自自己的家乡。还是那句话—旧坎瑞亚也有这种花，似乎太巧了点。佐证：祭冰礼冠文本—「在循环的终点，大地会再度迎来春生。因此，永恒是环形的。」、戴因斯雷布立绘—「原初逆转了毁灭，天空之岛焚烧了大地之国。白垩追逐着黄金，赤月对黑日复仇。未来拯救过去，年长者与年幼者同血相残。——这是世间回环的命运」。其命之座蛇环座也象征无限轮回、祸津御建鸣神命介绍—「既然命运注定是一场轮回」、第二章第四幕旅行者血亲所言—「与其活的毫无尊严，不如早日回到「循环」之中」、第三章第二幕纳西妲所言—「这片大陆的历史，或许也是一个漫长的花神诞祭」、草七天神像介绍—「古老的智慧之国历经劫波，正如神像一般在冥思中俯瞰着轮回流转。」、枯萎的苦舍桓介绍—「记忆总会随着创造而轮回复生」、被怜爱的少女文本—「怀钟指针在一日又一日地轮回着， 怀钟主人的思慕与回忆也是如此」、烈日威权的残响介绍—「朝暮轮回的红日与无始无终的时光」、足迹PV—「在无始无终的轮回里」，皆表明提瓦特已陷入轮回中。如今坎瑞亚地脉已完成，旅行者血亲在找地方安放，「荒泷生命摇滚虹色大巡回」中八重神子与雷电影的对话，温迪口中的漆黑大敌逼近蒙德，表明最近提瓦特将陷入大危机，与深渊殿下完成坎瑞亚的地脉的时间或许太巧了点。而且夜神虽然有开条件，但居然同意旅行者血亲的计划，在剧情中旅行者和茜特菈莉也对此表示惊讶并产生疑问。或许是因为如今提瓦特寿命要走向尽头，伊斯塔露与夜神才筹划让旅行者血亲替他们完成轮回。且让坎瑞亚地脉能完成的必须要素「夜神之国」也是在死之执政的「破例帮助」下形成的，「夜神」也是前天空岛的天使。如今旅行者的血亲失去降临者的身分，可以被此世命运束缚，天空岛只需对她/他加个「到千百年前安放坎瑞亚地脉导致轮回」的命运，也就一定会实现了。因此双子降临时天理的维系者没有阻拦，两人要离开时才阻拦并封印，且旅行者的血亲先苏醒并和戴因旅行，让她/他有足够时间成立深渊教团复国，或许都是天理的维系者设定好的，毕竟自己设的封印应该能设定解封的时间。当你存在的时空中旅行者的血亲表达自己打算利用整个世界换取与天理平等对话的资格时（即自己以为被降格了，所以成立深渊教团用一整个世界与天理对抗打算获取与之平等对话的资格恢复降临者身分），天理的维系者疑似听到了此对话就消失，或许是因为她在偷听确认旅行者的血亲的抉择，确认她最后把钥匙交给蒂莱尔让时间线一致没有改变，与计划没有偏遗时就离去了。实际上不是降临者的旅行者血亲的命运可以制定，因此其的选择都是可以控制的，这里天理的维系者或许是为了保险起见，又或者是在现场制定旅行者血亲的下一步抉择。此外，戴因斯雷布立绘维瑟弗尼尔说「原初逆转了毁灭，天空之岛焚烧了大地之国。白垩追逐着黄金，赤月对黑日复仇。未来拯救过去，年长者与年幼者同血相残。——这是世间回环的命运」因此他是知道提瓦特是轮回的，甚至这次轮回还是其计划的—因为旅行者血亲创建深渊教团是其诱导的[1]，而且目前还未公布其这么做的目的。毕竟他自己所在的提瓦特灭亡了对他自己也没有好处。蒂莱尔的命运被天理制定为必将间接导致坎瑞亚毁灭可能也是为了此计划。而旅行者未来的目的应该是打破轮回，既然提瓦特时间线形的会通往毁灭，就暂时将整个提瓦特的「时间线重置」，毕竟现在的提瓦特已经够不堪了，他可能将所有人信息保留再重置世界让世界归于起点，再依自己的意思重新创作提瓦特。失落圣城与千年来者也提及能将提瓦特所有人信息燃素化，在新的燃素构成的世界里大家变成信息（意识）生命，即维奇洛波奇特利计划。这样因为时间线重置了所以就能避免近期毁灭。等下次提瓦特又要毁灭时再依上次重置时创作的提瓦特的不足再重置然后创作更好的就好了，这就是戴因斯雷布立绘提到的「未来拯救过去」、旅行者角色故事提及的「维系者正在死去，创造者尚未到来」、足迹PV的「然后就去重新纺织一切的命运吧」、合成台介绍的「你会创作出世界吗？」、雷电将军语音「如果是你，一定能拯救我们所有人的」、月谕圣牌之二十一「世界」提及旅行者将以自己的记忆打造世界。因为创作新提瓦特的条件是关于现在提瓦特的记忆，因此古闻之章钟离才会要旅行者记住一切，雷电将军语音才说「如果是你，一定能拯救我们所有人的。只要你记住我，我就能永远的活着。」而天空岛应该是将旅行者作为备用计划，天理的维系者才解除封印让他/她醒来，并让派蒙（失落圣城与千年来者中盗火贤者称其为「从天上来的使者」）暗中监视并保护他/她的旅程（但派蒙应该失去记忆不知情只被天理设定要跟着旅行者旅行并保护他）。而冰之女皇到最终目的应该也是创造新世界，冬夜愚戏PV丑角提及「你的灵柩将成为整个旧世界」、哀叙冰玉提及「那么，替我燃烧旧世界吧」、「仆人」介绍—「依然微茫的炉火将焚尽旧世界的一切」、冬夜愚戏中博士看着世界树遭焚，似乎都暗示他们要重置如今的世界创造新世界。这也是天空岛明明有能力却没有一举灭了愚人众的原因，因为他们也是备用计划。而很多玩家猜测女皇收集神之心可能是要复活第三降临者，复活其的目的应该是让他重建新世界，因为旅行者能够重建，同为降临者的第三降临者或许也能；另外，依据终北祷歌集，第三降临者或为女皇的恋人。而第一降临者天理陷入沉睡[48]没法达成了。此外自从罪人舞步弦后剧情一直围绕着悖论展开，如芙卡洛斯欺骗命运、炽烈的还魂诗中以悖论挑战命运的队长、悖理杜林顶替魔龙杜林的命运。因此原神的剧情核心可能就是「悖论」，提瓦特的生灵为了卡天理规则的漏洞制造悖论，而天理也为了反抗宇宙生者必灭的法则制造悖论。
