# 黑日

黑日取代赤月，照亮坎瑞亚的大地。黑日王朝正式到来。此时坎瑞亚人崇拜一个“黑日”，因此被称为黑日王朝。赤月王朝时期，炼金术与驯兽骑士乃是中流砥柱。但是在黑日王朝时期，因机械工学兴起而衰落了。[14]黑日取代赤月后，仍有人想为死掉的月亮招魂。
