# 「斯彭西安」

贝瑟·埃尔顿原为水仙十字院孤儿，长大后加入枫丹海军，成为海军司令，是枫丹海军「白色舰队」旗舰「斯彭西安」的舰长，浪影阔剑为其武器。主动退役后，贝瑟回到十字院接任副院长[12]，在抚养卡特期间烹饪水平很差，经过苦练在抚养雷内四人时已变得水平高超。[13][3]坎瑞亚灾变发生、十字院遣散前，贝瑟联系卡尔和埃马纽艾尔领养阿兰四人[10]，并将墨绿羽饰留给了雅各布[14]。自己则返回部队，重登战舰，对抗以「厄里那斯」为首的兽潮。最终，舰队全部摧毁，大副纳撒尼尔·皮克曼存活了下来可能为逃兵，因为最终贝瑟的指令要求大副和轮机长随自己留下，其余人弃船撤离[15]，彷徨在英雄虚名中，为说服枫丹廷重启「斯彭西安」铁甲舰的打捞重建、重建枫丹海军而奔走[16]。「斯彭西安」则一半留于「厄里那斯」腹中，一半散落在海里。[15]
