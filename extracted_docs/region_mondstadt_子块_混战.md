# 混战

分出自己部分灵魂的技巧（即「斩三尸」[28]）在魔神中少之又少，而安德留斯正好精于此[29]。三千年前，风王的神位存在空缺，来自不同地域的两位魔神「高塔孤王」迭卡拉庇安与「北风之王」安德留斯为了争夺神位而开战[30]。安德留斯对高塔的孤王宣战，却也无法损伤王都分毫。[31]而王都内的众人对高塔上的孤王俯身也并非出于敬仰爱戴，而是被它的烈风吹得无法起身。[32]据说在过去，孤王会禁止一些和弦与曲调，因为敏锐的人能从行诗人与歌手的音乐中感受到叛逆的信号，歌曲与唱诗确实也曾被用作抗争者联络的方式[33]。风墙外都是狂风暴雪，无法生存，孤王的子民为了求生只能被迫于高塔内依靠高塔孤王。
