# 水仙十字院

枫丹决定将孤儿与犯罪者子嗣教养院正式编入公共福利体制，基于其位址所处的前世文明遗迹，命名为水仙十字院。其院长由性情温顺的纯水精灵担任，执行副院长一名，由品德与执行能力突出的人担任。[27]此后水仙十字院向枫丹社会各界输送了许多高级人才。其中包括但不限于：海军司令贝瑟·埃尔顿、自然哲学学院院长德怀特•拉斯克、逐影猎人埃马纽艾尔·吉约丹、知名记者卡尔·英戈德[28]。
纯水精灵与梅洛彼得堡为了实现厄歌莉娅的理念将水神的温柔与恩泽散布于大陆，纯水精灵流往各国[29]。其间厄歌莉娅为一些犯下罪行而良心未泯的犯人提供了赎罪地，让流放者为其看守原始胎海。这个区域最后形成了梅洛彼得堡。[30]
