# 迦毗鸠师与兰罗希陀

“妙论派学术天才迦毗鸠师，不到三十岁就被称为诃般荼。”以他的名字，教令院设立了“迦毗鸠师奖”；得奖学者的要求为：研究课题有突破性，创新能力强，个人发展潜力大，年龄在四十岁以下。该奖由六学院的贤者亲自审阅每年的研究报告，并统一投票出奖项得主；得奖学者均在日后的须弥学术界卓有建树。此人於四百年前左右在外出考察的时候发生了意外消失无踪，而且重要的研究成果也不知所踪了。迦毗鸠师也能看见兰那罗，并和名为兰罗希陀的兰那罗同行。兰罗希陀喜欢谜语和机关，会透过谜语和机关留下他们旅途的线索。迦毗鸠师的重量级研究成果之一包括《关于混沌炉心在非稳定态下的安全性提升的理论与实践——以降诸魔山之古国坎瑞亚遗迹巨像为例》。兰罗希陀在把他的学术研究报告藏起来后便也不知所终。迦毗鸠师研究机关时的离去让教令院从此对机械技术的研究与研发推广制定严格限制，妙论派也因此没落[30]。
缄默之殿与教令院随着时代变更，贤者一度变得迂腐愚昧，缄默之殿无法信任当时的教令院，合作关系结束，大约在四百年前又回到沙漠。由于合作关系中断不利于教令院的名望，所以他们在教令院内建了一个假缄默之殿来掩盖真相。[65]
