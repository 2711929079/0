# 香菱技能详细介绍

香菱是《原神》中的火元素四星角色，璃月万民堂的厨师，称号"长柄武器大师"。

元素战技「锅巴出击」：
- 效果：召唤锅巴，向前喷火造成火元素伤害。
- 特点：自动瞄准最近的敌人，造成多次伤害。
- 冷却时间：12秒。
- 能量消耗：无。

元素爆发「旋火轮」：
- 效果：释放一个旋转的火轮，持续造成火元素伤害。
- 特性：后台持续挂火，具有锁面板机制（释放时锁定属性）。
- 持续时间：8秒。
- 冷却时间：15秒。
- 能量需求：80点。

天赋技能：
1. 炊金馔玉：元素战技和元素爆发造成的伤害提升。
2. 烈火烹油：元素爆发期间，获得元素精通加成。

角色定位：
- 主要定位：副C
- 次要定位：辅助
- 核心优势：后台挂火、锁面板机制、元素反应触发

培养建议：
- 武器推荐：渔获、决斗之枪、黑缨枪
- 圣遗物推荐：绝缘之旗印四件套（充能效率和大招伤害加成）
