# 鸣神大社

稻妻最大的神社，位于影向山顶，供奉着御建鸣神主尊大御所大人。心怀愿望的人，可以一步步脚踏实地地登上山顶；或者若是被雷元素垂怜，还可乘雷极一路向上，直达被巨大的神樱树所笼罩的神社。神樱常开不败，一如鸣神永恒。穿过重重鸟居，找巫女求上一签，祈求鸣神的保佑吧。
