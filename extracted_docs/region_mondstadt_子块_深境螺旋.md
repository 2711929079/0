# 深境螺旋

由早已覆灭的远古文明，修建的巨大地下古都。究竟是出于何种目的而修建，如今已经不可知。其结构被分为连结世界的「深境回廊」与通往未知的「渊月螺旋」，故此统称为「深境螺旋」。据说，其中埋葬着早已被这个世界遗忘的秘密与宝藏…觊觎着其中宝藏与知识的人与魔物，占据了这座螺旋塔。战胜各层各关的挑战，剿灭其中的敌人，赢得「渊星」，就能获得隐藏在其中的，某种伟大意志的嘉许吧…
