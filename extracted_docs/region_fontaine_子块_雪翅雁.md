# 雪翅雁

生活在池塘、湖泊等水体的大型游禽，因为其高贵美丽的身姿而广受世人喜爱。由于其如雪一般的双翅，以及忠诚于家庭的习性，诗人们不约而同地将雪翅雁视作纯洁与高贵的象征。在枫丹的文艺圈中，也有着将卓越的诗人、歌手或舞蹈家比作行内「雪翅雁」的传统。
