# 原神深境螺旋攻略

深境螺旋（又称深渊）是《原神》中的高难度挑战副本，位于蒙德地区。

基本信息：
- 深渊共有12层，每层分为上下两间。
- 每间有时间限制，需要在规定时间内击败敌人。
- 每层有不同的元素环境和敌人配置。

奖励机制：
- 首次通关奖励：每层首次通关可获得大量原石。
- 月度重置：每月1日和16日重置，可重新挑战获得星数奖励。
- 星数奖励：每间最多可获得3颗星，根据剩余时间和角色存活情况评定。

攻略要点：
1. 队伍配置：
   - 建议配置双元素队伍，覆盖主要元素反应。
   - 主C+副C+辅助+治疗的经典阵容。
   - 根据敌人类型调整队伍，如应对高物理抗性敌人时使用元素伤害队伍。

2. 元素反应：
   - 利用元素反应提升伤害，如融化、蒸发、激化等。
   - 针对不同元素护盾选择克制元素进行破盾。

3. 角色选择：
   - 强力主C：胡桃、甘雨、魈、雷电将军等。
   - 优秀辅助：钟离（护盾）、温迪（聚怪）、雷电将军（充能）、纳西妲（草元素）。
   - 治疗角色：琴、芭芭拉、班尼特、七七。

4. 武器和圣遗物：
   - 确保角色装备合适的武器和圣遗物。
   - 优先提升主C的武器和圣遗物等级。

5. 战斗技巧：
   - 利用地形和障碍物躲避敌人攻击。
   - 合理使用元素战技和元素爆发的冷却时间。
   - 注意元素能量的管理，确保大招及时释放。

6. 常用阵容：
   - 万达国际：达达利亚+万叶+香菱+班尼特
   - 胡行钟：胡桃+行秋+钟离+第四个角色
   - 雷国：雷电将军+香菱+班尼特+行秋
   - 神鹤万心：神里绫华+申鹤+万叶+心海

深境螺旋是检验角色养成和战斗策略的重要场所，通过不断挑战可以提升游戏技巧和角色强度。
