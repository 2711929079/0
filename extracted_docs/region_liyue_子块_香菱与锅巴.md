# 香菱与锅巴

万民堂年少有为的厨师香菱几年前在寻找食材时，结果天降大雨。又累又饿的香菱进了一个山洞中避雨，将随身携带的辣肉窝窝头放在了山洞中神龛前的供桌上，随后睡着了。醒来后看见失去知性的马科修斯正在吃辣肉窝寓头，二者就此成为朋友，香菱为其取名为“锅巴”[190][80][191]。
瑶瑶香菱向萍姥姥学习枪法。出生在轻策庄的瑶瑶被萍姥姥看中，瑶瑶的父母在被劝说后，最终同意瑶瑶跟随萍姥姥去往璃月港生活。年幼的瑶瑶字还没识全就拜入萍姥姥门下，成为香菱的小师妹。萍姥姥教瑶瑶生活技能、诗书礼仪以及枪法。留云借风真君还制作了“月桂”作为防身之物，将其送给了瑶瑶[192][193][194][195]。
