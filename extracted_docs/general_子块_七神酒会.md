# 七神酒会

两千年前巴巴托斯初临璃月，「岩王帝君」的第一反应是：这位同僚在履行职责时遇到了困难，需要自己的帮助。所以还没等巴巴托斯从风中落下，岩之神已经做好准备迎接这位邻国神明，只等他开口，自己就将倾尽所能。然而，风之神却将一瓶酒递到了他面前。「这是蒙德城的酒，你要尝尝吗？」——为了送一瓶酒而弃职责于不顾，实在荒唐。即使如此，那位风神依然不断前来造访，到璃月港四处游玩，还总向岩之神问出各种奇怪问题。这位风神的好奇心，就和他手中的酒一样无穷无尽。自那时起，那个时代的七神往往会在璃月相聚。
