# 原神战斗系统详解

《原神》战斗系统基于元素反应机制，具有深度和策略性。
战斗基础：
- 角色切换：可以在4个角色之间自由切换，每个角色拥有不同的元素属性和技能。
- 元素属性：游戏中有七种元素属性：火、水、雷、冰、风、岩、草。
- 普通攻击：每个角色都有独特的普通攻击连击。
- 重击：消耗体力进行强力攻击。
- 元素战技：角色的主要技能，冷却时间较短。
- 元素爆发：强力技能，需要积累元素能量才能释放。
战斗策略：
- 元素反应：通过不同元素的组合触发各种反应，如蒸发、融化、感电等。
- 能量管理：合理使用元素战技积累能量，为元素爆发做准备。
- 角色切换：根据战斗需要灵活切换角色，发挥每个角色的优势。
- 躲避机制：利用闪避和冲刺躲避敌人攻击，减少伤害。
- 护盾保护：使用岩元素角色或冰元素角色生成护盾，保护队伍。
