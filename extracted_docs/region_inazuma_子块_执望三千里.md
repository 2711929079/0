# 执望三千里

旅行者来到八酝岛，被长次感动（蒙骗）收集晶化骨髓，误会解除后长次在旅行者的帮助下从离岛出发寻找母亲。
武士的宿命旅行者救下了受伤的稻叶久藏，后受稻叶久藏委托，将名椎滩的海乱鬼清剿长次没人照顾了。稻叶久藏终与海贼战死于八酝岛。
