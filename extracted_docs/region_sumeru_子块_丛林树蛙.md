# 丛林树蛙

常在湿地出没的小小青蛙。潮湿地带十分常见的小动物，身上会分泌奇特的粘液，具有多种药物用途。树蛙适应丛林中的生活。据说有些树蛙可以在丛林间滑翔。根据研究，其原理与风之翼有不少相似之处。
