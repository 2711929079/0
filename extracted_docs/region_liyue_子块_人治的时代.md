# 人治的时代

岩王帝君退位后，部分仙人担忧将璃月交由人们治理是否为明智之举。与此同时，璃月的海域出现异常，察觉异样的「天权星」立刻下令修建群玉阁，将蛰伏在海中的奥赛尔的妻子与其最后的追随者「跋掣」吸引出来。此后人们靠自己的力量击退拔掣，仙人们才认可帝君将璃月港托付给人们的决策。[148]
若陀龙王若陀龙王近来由于封印出现了一丝裂隙，若陀龙王的恶意便化身阿鸠徘徊于伏龙树旁，寻找着解除封印的时机。但另一方面，若陀龙王并未完全忘记曾与帝君签订的「守护人类」的契约。龙王最后的契约，使龙王的善念附身于璃月名匠昆吾后人昆钧身上，凭着自己的本能寻找着摩拉克斯，希望他能阻止封印的解除。封印的解除被阻止后，「昆钧」将被遗忘的与人类同行的记忆展示给阿鸠，阿鸠在遗憾与悔恨中力竭，重新被封印。若陀龙王的善念也在和钟离短暂叙旧之后力竭，就此消散。「摩拉克斯，若有缘，他日必将再会。」[114]
