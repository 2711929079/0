# 悬木人

原本住在祖遗庙宇的悬木人的先祖在祖遗庙宇毁灭后用绳索与钩爪将块块巨木悬挂，部族因此也有了「悬木人」这样的名字。[49]传说中，「悬木人」的勇士与一百只匿叶龙一同，用了一百根绳子，花了整整一百天，才将象征信仰的「曜石图腾柱」搬到了现在它所在的地方。[50]
献祭人习俗盗火者引导人类发展，但部族还留着迂腐的原始献祭习俗，在大灵诞生后更加巩固，因为部族认为大灵是靠献祭产生的[51]。而且个部落间的人们击退龙族后却互相争斗，争夺政权[52][41]。部族的人们因为对抗龙族部族首领被12席「轻语如泉者」击杀，失去首领的他们还开始内战[39]。盗火者看不下去了，于是引退静观其变，因为根据自己的预言人类迟早会诞生一位有贤能的人击败火龙王[51]。掌握燃素使用方法的人类开始互相残杀，虽然不理解龙族如何从血液中提取生命力，但是人类依然模仿龙族的行为，通过杀戮向自己幻想出来的神明献祭。龙族原本虽然因为燃素被人类学习很愤怒，但看到人们拿着燃素自相残杀后就转而看戏了[39]。
