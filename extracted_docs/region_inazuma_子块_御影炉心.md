# 御影炉心

前几年，为了扩大生产的规模，天领奉行从枫丹请了一支顾问团队，建造了能充分利用「祟神」力量的「御影炉心」。「御影炉心」是以枫丹技术建造，以八酝岛运来的「祟神」核心驱动的超级冶炼炉。虽然鸣神大社也多次提醒他们，不能对「祟神」掉以轻心。但他们还是打定了主意，依然推进着建造计划。「御影炉心」投产后，枫丹顾问团的成员大部分都回国了，只有泽维尔还留在稻妻。[82]
近年来年轻的珊瑚宫心海成为海祇岛现人神巫女，同时担任军师[83]。一位名叫五郎的海祇岛普通士兵在一次战事中立下奇功，被珊瑚宫心海提拔力 “國鳗二番队”队长，并在未来逐步成为海祇岛大将[84]。鹿野院平藏因与珊瑚理念不和，退出了自己名为“万端珊瑚平藏侦探社”的侦探事务所，加入天领奉行[79]。长野原宵宫接任店长后所举办的第一场盛典就令人叹为观止，因此被称力“夏祭的女王”[85]。梦见月瑞希最终来到了求学的最后一站——纳塔的流泉之众，在那里她结识了玛拉妮，萌生出建设“娱乐集团”来帮助人们维持心理健康的想法。于是，梦见月瑞希成为了稻妻“秋沙钱汤”的大股东，并开始布局其他娱示项目[86]。
出身巫女世家的久岐忍放弃成力巫女，离家出走。初入社会的她考取了许多证书。久岐忍还曾去往璃月学习律法，烟绯曾当过她的讲师，在毕业前夕回到稻妻[87]。
