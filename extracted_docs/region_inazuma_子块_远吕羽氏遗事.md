# 远吕羽氏遗事

旅行者接受工人梶的委托，修复4处镇物，八酝岛的雷雨天气停止，并斩杀了发疯的鹫津。
孤岛诊疗谭旅行者来到八酝岛，帮助医生保本收集鸣草以制作药物。保本多次小剂量试药，后前往八酝岛南部沙洲，服下了全剂量试药，毒发而亡。
