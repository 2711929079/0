# 烟绯与优菈

一艘装载着深渊教团危险品的商船要从荆夫港运到璃月港。烟绯作为跟随商船的顾问发现货物有鬼，就擅自进行了调查，结果引火上身，还好优菈等人及时赶到，救下了烟绯，并在其帮助下将深渊教团成员一网打尽[200][201][202]。
久岐忍久岐忍来到璃月的“同文学塾”进修法学，期间烟绯还替久岐忍的老师代过课。久岐忍在毕业前夕回到稻妻探亲，结果正好碰上稻妻锁国令颁布，无法回璃月拿毕业证[203][204][20]。
