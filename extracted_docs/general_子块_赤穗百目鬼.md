# 赤穗百目鬼

稻妻的海盗，浅濑响看着他从小长到大，食量很大。别称（或者是别名？）：左卫门盘踞在清籁岛的稻妻海盗，拥有着十数艘船，以清籁的不死之鬼之名驰骋。但这样的他，经常来到浅濑神社，帮助浅濑响修剪树枝，浅濑响和他关系要好，经常和他谈论自己的理想（想要去看看这个世界）与曾经的友人（惟神和御舆长正）。在漆黑灾厄结束后的几十年后，与幕府产生了正面冲突，于是幕府派来舰队，想要将其覆灭，浅濑响看着他的背过去的模样，像极了当年的高岭（昆布丸）。于是，浅濑响不忍心让他死去，等百目鬼他们出海之后，随即解开了清籁岛镇守的封印，雷鸟的怨念从清籁岛中心爆发[1]，不一会到达战场，肆虐八方，导致双方元气大伤，幕府被迫退军进行整顿，而他们（百目鬼）从此下落不明。他们来到了一个非常远的地方，连同魔偶机关一起。是新蒙德初诞时期风神利用神力改变地形，将多余部分丢进海里，最终与某片未知海域的岛屿一同形成了一个群岛，名为——「雾海群岛」又名「金苹果群岛」。这是一片由迷雾围绕的群岛，迷雾视乎终年不散，而在迷雾中航行不管怎么走都会回到这片群岛上来。在经过蛇目海岛上的居民的努力下，他们发现了迷雾的规律，最终走出了这片群岛，而他也再次扬帆起航，回到了这篇他熟悉的土地，映入他眼帘的却是一片废墟……熟悉之人都已不在，不管是蛇目还是老巫女（浅濑响），只有寝子还在这片土地上。之后他又再次扬帆起航，去见识世界的尽头。[2]
