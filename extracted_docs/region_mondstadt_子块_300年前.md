# 300年前

雪猪雪猪因肉质鲜美遭到过度猎杀几近灭绝，只在雪山地区有少数存活。[118]一位名为欧拉夫的清泉镇猎人遭遇急冻树而被冰封了300年。据他所说，当时清泉镇将食材烤过后就吃了，没有过多料理方式。[119]欧拉夫是迪奥娜先人，擁有凯茨莱茵家族的血统，因此拥有猫耳与极佳狩猎天赋。兽人的诞生则是因为提瓦特人兽没有生殖隔离。
