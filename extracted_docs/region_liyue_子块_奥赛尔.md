# 奥赛尔

漩涡之魔神奥赛尔掀起滔天洪水大肆侵略璃月港，被摩拉克斯击败，扔下、击倒他的长枪成为如今的孤云阁岛屿。之后被封印在岩枪下的奥赛尔，在岩枪下痛苦挣扎，每个动作都充满极端的痛苦与憎恨[59]，试图从封印中逃脱。孤云阁耸立的岩柱，曾经都是摩拉克斯为了镇伏敌手降下的岩枪。久而久之，魔神之体渗入了岩柱，使之拥有了圣性。岩柱偶尔会剥落一些散发着怨恨和不样的碎屑[87]。
甘雨魔神战争结束后，甘雨与帝君签下契约，前往月海亭担任秘书。
