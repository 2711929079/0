# 绮良良技能详细介绍

绮良良是《原神》中的草元素四星角色，稻妻的快递员，称号"檐宇猫游"。
元素战技「猫爪快递」：
- 释放草元素能量，对周围敌人造成草元素伤害并施加"草元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「猫尾包裹」：
- 创造一个持续8秒的草元素领域，领域内的敌人会受到持续草元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "猫爪的祝福"：猫爪快递命中敌人后，有15%概率触发"激化"效果。
- "草元素增幅"：绮良良的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "包裹的守护"：元素爆发持续时间内，绮良良的草元素伤害提升15%。
