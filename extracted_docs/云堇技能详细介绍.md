# 云堇技能详细介绍

云堇是《原神》中的岩元素四星角色，璃月云翰社的当家花旦，称号"红毹婵娟"。
元素战技「点绛唇」：
- 释放岩元素能量，对周围敌人造成岩元素伤害并生成岩元素结晶。
- 冷却时间：8秒。
- 能量消耗：无。
元素爆发「破阵乐」：
- 创造一个持续12秒的岩元素领域，领域内的敌人会受到持续岩元素伤害。
- 冷却时间：18秒。
- 能量消耗：60点。
固有天赋：
- "戏曲的魅力"：点绛唇命中敌人后，有15%概率触发"结晶"反应。
- "岩元素增幅"：云堇的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "破阵的鼓舞"：元素爆发持续时间内，队伍中的角色防御力提升20%。
