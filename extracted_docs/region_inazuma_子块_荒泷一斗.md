# 荒泷一斗

儿时的荒泷一斗与其父母因为鬼族身份，被村民诬陷是村里恶性事件的罪魁祸首，被迫背井离乡。后来，父母因病去世，荒泷一斗流落街头时被一名老奶奶「鬼婆婆」收留[14]。
千织小时候的千织贪玩误入神里家的院子，与神里绫华相遇，两人玩了好久，最后千织被人发现，被提出门[80]。
