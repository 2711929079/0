# 枫丹科学院

科学院是由阿兰·吉约丹创立的研究机械与能源的研究所，全称是「枫丹动能工程科学研究院」。比之须弥的教令院，是性质大相径庭的机构。没有对枫丹人进行行政管理的冗余责任，也不承担教育义务，研究的方向更为集中。科学院旧址已被炸掉，现在的新科学院官僚化严重。
