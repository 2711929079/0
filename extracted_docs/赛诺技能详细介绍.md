# 赛诺技能详细介绍

赛诺是《原神》中的雷元素五星角色，须弥教令院的大风纪官，称号"缄秘的裁遣"。
元素战技「圣仪·煆煌随狼行」：
- 释放雷元素能量，对周围敌人造成雷元素伤害并施加"雷元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「圣仪·百裂灭尽」：
- 释放强大的雷元素能量，对周围敌人造成雷元素伤害，并进入"狂怒"状态。
- 冷却时间：20秒。
- 能量消耗：80点。
固有天赋：
- "裁遣的祝福"：圣仪·煆煌随狼行命中敌人后，有15%概率触发"感电"效果。
- "雷元素增幅"：赛诺的普通攻击和重击造成暴击时，恢复3点元素能量。
- "百裂的威严"：元素爆发持续时间内，赛诺的雷元素伤害提升20%。
