# 五大罪人

五大罪人分别是「黄金」莱茵多特、「贤者」海洛塔帝、「极恶骑」苏尔特洛奇、戴因斯雷布的哥哥「预言家」维瑟弗尼尔、「猎月人」雷利尔。坎瑞亚被深渊入侵时，才华出众的五人连同戴因斯雷布被寄予「拯救坎瑞亚」的厚望，但五人因自己私欲滥用深渊力量反而加速坎瑞亚覆灭。
