# 大赤沙海

须弥西南部沙漠区域的统称，是赤王的领地，包含了上风蚀地、下风蚀地与列柱沙原三个区域。大赤沙海有赤王利用禁忌的知识建造的宏伟壮观的遗迹。防沙壁以西，曾经也是一位神明的居所，也许在那时，沙漠也曾经是众生的乐园，智慧汇聚之所。而不是如同今日这般，举目所见，除却风暴以外，只有让人徒生感叹的陈迹。
