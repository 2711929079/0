# 卡雷斯之死

三年前，卡雷斯被指控枪杀雅克，要求以决斗自证清白。在那维莱特的见证下，卡雷斯最终死在与官方指定的决斗代理人克洛琳德的决斗中。刺玫会也因此名声大损，陷入经济危机[78]。克洛琳德的师父是逐影猎人佩特洛妮拉，此人也是卡雷斯的旧识，基于这个关系，年幼的克洛琳德与娜维娅曾是很要好的朋友。但自从发生了三年前的事情后，两人虽然不算是结仇，但也关系生变，很少见面说话了。
