# 最后的契约

旅行者与派蒙造访璃月港，恰逢璃月七星主持请仙仪式，却在仪式上惊现岩王帝君之尸骸，来自异乡的旅行者被怀疑为刺杀帝君的凶手，得到愚人众执行官「公子」的帮助。旅行者在「公子」的介绍下与往生堂客卿钟离一同为岩王帝君的送仙仪式作准备，同时亦因帝君之死而在七星和仙人之间周旋，调和仙人与璃月人的矛盾。「公子」借机暴露真实目的，欲从仙祖法蜕中取得神之心，但在发现仙祖法蜕只是个空壳后，他认为帝君只是为了某目的假死，为了逼出岩神便以百无禁忌箓放出被摩拉克斯镇压的「漩涡之魔神」奥赛尔，旅行者、七星、千岩军和仙人们同仇敌忾，团结起来迎击奥赛尔和伺机捣乱的愚人众，最终以牺牲群玉阁为代价将奥赛尔重新封印。事件后旅行者找到钟离，却目睹钟离以岩之神的神之心与「女士」进行交易——原来钟离就是真正的摩拉克斯，在察觉到自己终将受到磨损并失去神位后，制作了自己死亡的假象来考验璃月的各方势力，最终结果超乎想象，让钟离可以放心把璃月交给人们，而神之心则是摩拉克斯用来邀请愚人众参与这次事件的筹码[27]。而钟离担任往生堂客卿的原因是因为此后往生堂堂主将负责主持攸关璃月港存亡的八门七门大阵，提前前去监督已在必要时提供帮助[213]。当晚，钟离对七星托梦坦白了实情。次日，送仙仪式正式举行，也代表璃月正式迈入人治的时代。凝光在外交上紧咬着愚人众不放，使至冬趋于弱势，并逐渐排查北国银行势力[214]。此后，璃月与至冬断交，层岩巨渊内提供给愚人众的补给站废弃。[208]
七星选拔之后愚人众干涉「七星选拔」，打算扶植魁儡政权，被阻止。「富人」干涉選拔，打算在知易上任后以其名义发起青墟浦探索计划，对外声称是该地拥有外发掘的宝藏[82]。然而「富人」的最终计划是找到替代摩拉的货币，钟离提及透过神之心能把自己血肉转换为摩拉，而青墟浦为旧时某名号佚失的魔神领地，存在许多秘密[82]。因此「富人」的目的可能是透过取得青墟浦底下的魔神遗骸并以神之心转换为摩拉。然而这样铸币法也还是要依靠来自神明的神之心，其仍在尝试其他铸币法，为了原料大量购买石珀[215]。
