# 莫娜技能详细介绍

莫娜是《原神》中的水元素五星角色，占星术士，称号"星天水镜"。
元素战技「水中幻愿」：
- 创造一个水元素虚影，对敌人造成水元素伤害并施加"水元素附着"。
- 长按可以提升伤害和范围。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「星命定轨」：
- 释放强大的水元素能量，对周围敌人造成水元素伤害，并创造一个持续8秒的水元素领域。
- 冷却时间：20秒。
- 能量消耗：70点。
固有天赋：
- "星轨的涟漪"：水中幻愿命中敌人后，有15%概率触发"冻结"效果。
- "占卜的启示"：莫娜的普通攻击和重击造成暴击时，恢复2点元素能量。
- "水镜的祝福"：元素爆发持续时间内，队伍中的角色水元素伤害提升15%。
