# 深秘院

负责科研的机构，业务包括占星、研究赤月与深渊等，院长为「贤者」海洛塔帝，副院长为艾弗提尔·维特[24][25][14]。
“至高的王上啊，深秘院向您呈上预测的结果；双星早已被大地的引力捕获…经历了漫长的归回周期，如今它们的轨迹将要再度交错。”[26]
苏尔特洛奇与哈登苏尔特洛奇是死囚的儿子，整日和一帮不良青年混迹街头，受过不少批评和苦难，但他始终保持乐观、自信，后来结识了身患重病的坎瑞亚贵族继承人斯克弗努格并承认了对方的天赋，同他一同切磋，成为朋友。10岁那年，他偷来了禁忌的研究资料，治好了斯克弗努格的病[27]。身为教师的哈登传授剑术，在教导戴因斯雷布黑蛇剑术时却为了维系自己尊严而对其不公的体罚。「极恶骑」苏尔特洛奇也因同样原因变的肢体残缺，在哈登的残害下变得性格扭曲。此事一直被戴因所记恨[28]。
