# 降生源火 建立初文明

火龙王修库特尔在「圣山」的源火降生后，改变圣山内部构造，建立了初始圣城「图兰」，使之成为贮藏一切回忆与智慧的场所。之后，修库特尔开始利用源火塑形，创造了纳塔龙裔；其中，第一位被创造并被授予源火的一缕分火的，就是库库尔坎。火龙王用燃素打造了13个至高领主的意识，至高领主的意识能通过机械容纳，各领主们都拥有各自的土地管理城市。13位至高领主，既是秘源技术的巅峰，也是火龙王的子嗣。龙族发展了辉煌的文明，他们利用分离自源火的燃素，驱动了无数先进机械（这些机械至今仍遗留在纳塔）。在穷尽了对提瓦特的探索后，他们把目光投向了提瓦特之外的星海，甚至开始建设太空电梯。
