# 遗迹巨像

在坎瑞亚灾厄时期，坎瑞亚派遣了被后世称作「遗迹巨像」的巨型战斗机械人清剿漆黑兽潮，但行动失败，三台遗迹巨像残留于须弥。
此后，教令院将遗迹巨像的炉心封印。降诸魔山的遗迹巨像炉心被封印在迪弗旧窟。[30]
树王圣体菇在须弥，森林中的居民有着向教令院贡奉树王圣体菇的传统，但这些贡品的用途却不为世人所知。据某些逸话中说，教令院每年会在净善宫进行秘密的仪式。在仪式上，贤者们会通过食用树王圣体菇的方式来纪念树王的牺牲。[73]
