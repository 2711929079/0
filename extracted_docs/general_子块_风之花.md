# 风之花

「风之花」是风花节的象征，代表着自由与风之精神的花卉哦。人们对「风之花」的认知大不相同。「风之花」的定义确实还有争议，但蒙德人普遍认为,「风之花」不是蒲公英就是风车菊。在庆典为风神献上的「风之花」向来由风花节之星决定。「风之花」这个词语，最初只是旧蒙德时代，人民之间相互联络、共约反抗的暗语。那时人们常说，风越大，「风之花」的根基就越稳固，花朵也绽放得越鲜艳。它是向往自由的灵魂、是追逐风向的勇气…一切美好、值得被祝福的事物，都可以是「风之花」。
祝福之花、敬爱之花、深爱之花。每个人都会有属于自己的「风之花」，每个人都有自由定义它的权利。
