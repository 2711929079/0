# 主线剧情开始之后

忍冬之树新生有人消解了冰封的桎梏，随着某人的以绯红的精髓浇灌，让汲取了「绯红」的古树再度发出新芽…[157]原本早应当死去的白树，回想起了过去，汇集了一切力量，结出了果实…将我曾荫蔽的人们，将曾讴歌我的祭司，将常常在我身畔绘画的那位美丽的少女，不曾拥有的幸福，结成绯红的冰冷果实。让能向恶毒的世界降下公正「报偿」者，能带着无人之过的「苦果」，施展正义。[11]「我等必将复起，古木已发新枝」[158]龙灾龙灾（主线）现在的蒙德正面临双重危机。
其一为由非人之物组成的深渊教团，牠们对特瓦林实行腐殖仪式，诱惑特瓦林，使其成为对付蒙德的武器。因风魔龙作乱，骑士团毁弃其的庙宇。
其二为试图在龙灾期间趁乱夺取蒙德塔防的冬国士兵—愚人众。
在能够净化深渊之力的旅行者来到蒙德后，风神才能联合琴、迪卢克、旅行者等人解决龙灾。
最后，愚人众执行官“女士”现身夺取风神的神之心[106]。风神不在乎神之心的归属与持有者，它对风神没用，「女士」在西风教会前强取神之心，风神认为交战可能会波及西风教会甚至整个蒙德，于是将神之心让给她。然而「女士」却因为轻易得到二个神之心低估魔神的力量，导致被雷电将军斩杀[159]。龙灾后尽管近年来蒙德遭遇了许多事件，也一度陷入困难境地，但在西风骑士团治理蒙德的第二个五百年，风神治下的地区依然高唱着自由之歌，欢迎所有远道而来的可敬旅人，以包容的心态面对文化的融合与变迁。400年前，马斯克家族撰写《风之国土的人文习俗考察》，通译《风土人情志》，是蒙德的地情文化的重要文献。马斯克家族在丘丘人语言和习俗方面，以及人类奇异血统研究上也取得了一些突破。蒙德酒业的大发展亦出现了材料各异、种类繁多的酒精以及非酒精饮料，受到人们欢迎，改变着蒙德人的饮食习惯。猫尾酒馆调酒师迪奥娜，为蒙德酒文化带来了新的变数。阿贝多和艾莉丝带来了新的文化种子，阿贝多让蒙德人摆脱了炼金术“只够用来拼接物件碎块”的偏见，重新认识到炼金术的大用途，艾莉丝则引导芭芭拉成为蒙德人气偶像，在传统的教会文化之外萌生了偶像文化，用歌声治愈心灵的理念传遍蒙德城，也为古典诗琴演奏民谣的蒙德音乐增添独特新旋律。斯坦利被困在烬寂海的灵魂被温迪救回，为了拯救更多困在该处的灵魂，在玛薇卡对抗深渊结束后，温迪与玛薇卡合作[36]。阿贝多阿贝多「原初之人」实验的失败品因吸收杜林腹中之血复活，因为嫉妒完美品阿贝多打算取代他并来到人类社会生活[160]，后被阿贝多击败杀害。杜林吸收了失败品的遗体后得到他的记忆，对阿贝多也萌生同样的情绪[161]。纳塔主线后挪德卡莱骑士团的远征取得了可观进展，并由此做出了一个重大决定：根据途中收集到的新线索，大团长将前往挪德卡莱，进行更深一步探寻与查证，如今西风骑士团在安瓦蒂尼尔湖附近驻扎，原因之一便是调查此处有关过去的「北风骑士」以及狼主「安德留斯」的痕迹[162]；而赤杨骑士（芙蕾德莉卡·古恩希尔德，琴的母亲）等人则领命回到荆夫港驻防[163]。托马的母亲身为骑士也跟着被调往挪德卡莱[164]。悖理阿贝多将用炼金术把小杜林变成人类，从而能够令其身上纯粹的部分抵消杜林身上原始、邪恶的部分，替代即将复活的毒龙杜林在世界上降生，规避杜林复活的命运[161]。風神恢復力量为了对抗即将逼近蒙德的漆黑大敌，巴巴托斯取回自己存放在时之执政祭坛的部分力量[37]。狂猎入侵雷利尔被击败后，现在深渊模仿出的染血骑士罗南成为狂猎的新统领，统领「狂猎」入侵蒙德，后被击败，法尔伽将艾伦德林与鲁斯坦的话（信）带给这位罗南的「倒影」[29]。
