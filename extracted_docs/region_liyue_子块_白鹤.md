# 白鹤

留云曾偶遇一只获得了仙力的白色野鹤，作为师父引导她修炼。50年前璃月突发旱灾，白鹤想要像师父那样拯救苍生，但那时的她尚不是真正的仙人，无法掌握仙人的化形之法，于是留云为她炼了一种「化形药」让她能化成人形，但药效只能维持十年，十年后若她没有变回鹤形，就会忘记自己作为仙兽时的一切记忆，变成非人、非妖、非仙的存在。然而白鹤在下山镇压旱灾期间与一个天衡方士相恋，因此背弃了十年后变回野鹤的诺言，写信与留云辞别后归隐于人间。以人类身份生活时，白鹤患上了严重的健忘症，生活难以自理。40年前，留云看见信后非常生气，本想去找白鹤兴师问罪，因为她背弃了诺言，而且她找的天衡方士悟性不足，驱使符箓毫无技巧可言。但留云在看到白鹤的现状以及丈夫照顾她的决心后心生怜悯，便易容成商人模样，在望舒客栈为她提供了一壶以自己血液炼制的灵药，才帮助她压制了几分化形药的副作用。当白鹤分娩时，留云还帮助她与爱人击败了追杀他们的魔神残渣。白鹤的孙女曾生了重病，留云以鹤形拜访希望医治，但白鹤的孙女坚持认为只有白鹤才是仙鹤，因此留云只得将自己全身涂成白色，驮着她去自己的洞天里医治疾病。[47]
申鶴申鹤出生于驱邪世家（重云的世家）的旁系。她5岁时，母亲因病去世，父亲爱妻心切，去往远方寻求复活亡妻的办法。一年后，申鹅的父亲从至交“明俊”那里找到了一本书，书里记载了复活亡妻的方法。申鹤的父亲回到象中，向申鹤谎称一处山洞中有惊喜，申鹤走进山羽，父亲利用魔神残渣召唤出恐怖的“仙灵”，“仙灵”指出申鹤是孤辰劫煞，申鹤的父亲便決定用申鹤的生命换回自己妻子的生命，独自离开了山洞。申鹤拿起母亲道物中的一柄驱邪匕首与“仙灵”死战。数日后，留云借风真君循着邪祟的气息找到了在战斗中占据上风卻奄奄一息的申鹤，帮助她清理了魔物后，便将她带回府中修养，而后收她沟徒。这几天里，申鹤的父亲橋然醒悟，深感愧疚的他选择了自杀，而明俊得知这系列事情后一直深感愧疚。仙家收留申鹤后，削月筑阳真君为其卜卦，卦象显示她既犯孤辰，又冲劫煞。为了让申鹤安全长大又不殃及他人，仙人们为其身上施了红绳缚魂之法[146][147]。留云借风真君将“翠钿白玉梳”送给了申鹤，并告诉她：如果想斩尽凡缘，踏入仙门，就用这把梳子梳三下头发。申鹤毫不犹豫地照办了，原本的一头青丝变为了白发。[148][149]。修行了十多年的申鹤在一天心血來湖，回到了自己的故乡，发现自己儿时居住的村庄已被遗弃，父亲也早已离世[150][151]。
