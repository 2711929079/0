# 封印深渊

峡谷部族长老陶奥鲁将深渊封印住，而封印阵的制做方法是夜神之国的神祇提供的，在封印中极重要的镌光铭印的使用方法是盗火贤者瓦萨克拉胡巴肯提供的。夜域神祇与盗火者本是敌人，但为了对抗共同的敌人——深渊也还是达成合作。五百年前两位坎瑞亚战士——西弗安雅与伊利萨牺牲自己帮助纳塔英雄封印深渊力量，两人自己却和深渊魔物一同被封印住，从此在封印里陷入了永远的战斗中无法解脱。[121]众岩之里是五百年前「回声之子」部族的勇士们最终封印深渊的地方。固态燃素构建的地面和深渊剧烈冲突，使这里的地面起伏不定。[122]
烬寂海烬寂海因桑哈杰的法术[118]而形成，诞生于坎瑞亚灾变后。[97]它是个没有任何风的辽阔原野，另冒险者闻之丧胆的险地[123][124]。因为巴窟纳瓦的存在，烬寂海的污染具有时间性，为了防止烬寂海的对世界的侵蚀，巴巴托斯将其从时间轴上整个吹出去，但也造就了烬寂海的灵魂无法回归地脉，被困在该处只得永远在烬寂海游荡[125][96]。
