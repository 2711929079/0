# 异乡人

祭司之女梦见了蔽日的漆黑之龙（以时间线来看，葬火之战前，日后葬火之战就发生，因此是尼伯龙根），是为凶异之兆。同月，异邦人伊蒙洛卡[2][12]来到王城。祭司之女将星银铸成的大剑交予异乡的勇士，风雪的啸声中，她说出的话没能传达给对方。[13]「这里的第四幅壁画为你而准备，你的形象将会永远留在这面墙上。」「为了这幅壁画，为了大家,我会一直在这里等你，祈祷你的归来…」当雪葬之都的女儿与嫁接无果的银枝条一同凋零时，命定挥舞此剑斩开冰雪的异乡人正在远方寻求答案。皎洁如月光的她，最后的思念也没能传达给远行者。……[13]「壁画恐怕无法完成了。」「我已经很久没看过晴空与绿地了。不知道应该用什么样的蓝色、什么样的绿色，才能画出父亲想要的，冰雪消融的景象。」「等他回来，一切都会恢复过去的样子吧。但是，太冷了。我已经不行了。」「对不起，爸爸。对不起，一直看着我们丑态的乌库。」「伊蒙洛卡，如果能再见你一面，就好了…」不满足于受保护的绘画少女，她向倾慕的人留下这最后的嘱咐：「假如天性中的胆怯与绝望将你压倒，令你终于不再归来，那么…」「…请你活下去。请不要与我们共同走向灭亡，湮没于冰冷的遗忘」[14]
记事者乌库「愿降下这片冰雪的天空燃烧，直到世界上再也没有一丝生命。」「愿我们化作灰烬，顺风而行，追上那抛弃了公主再也不返回的异邦人。」「愿公主梦见的蔽日的黑龙，为可憎的大地涂上赤红的毒。」「我是最后一个。再也没有留守的必要了。听说何处有人正在建立无神的国度（坎瑞亚）。那么或许他们也有对抗世界的力量吧。」[12]最后记事者乌库背弃神明成为了以非人身份存活至今的丘丘霜铠王。芬德尼尔覆灭后，其残余遗民乌库成了丘丘王，而部分遗民失去过去记忆，试图向下探索，寻找已经被遗忘的东西。遗民失忆可能是和乌库一样成为丘丘人，而丘丘人挖掘的动作就是因为试图向下探索。为了找寻平息高天怒火的方式，为了通晓命运，他们试图向下探索，寻找已经被遗忘的东西，从深埋地底的古代祭场、废都中寻求启示与答案[15][16][17]。古树树洞的另一端，过去的秘密凝结成了宝藏[18]。
