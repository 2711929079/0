# 枫原万叶技能详细介绍

枫原万叶是《原神》中的风元素五星角色，稻妻的浪人武士，称号"红叶逐荒波"。

元素战技「千早振」：
- 点按：快速拔刀，对前方敌人造成风元素伤害。
- 长按：腾空而起，击飞并牵引周围敌人，造成风元素伤害。
- 下落攻击：通过元素战技触发的下落攻击转化为风元素伤害，并产生小型风穴牵引敌人。
- 冷却时间：点按6秒，长按10秒。
- 能量消耗：无。

元素爆发「万叶之一刀」：
- 效果：释放巨大的风元素刀刃，对范围内敌人造成风元素伤害，并创造一个领域。
- 领域效果：持续扩散其他元素，造成对应元素的伤害。
- 持续时间：8秒。
- 冷却时间：15秒。
- 能量需求：60点。

天赋技能：
1. 风物之诗咏：元素战技和元素爆发造成的伤害提升，提升幅度基于元素精通。
2. 流风秋野：拾取元素微粒或晶球时，获得元素精通加成。

角色定位：
- 主要定位：辅助
- 次要定位：副C
- 核心优势：元素扩散、减抗、聚怪、元素精通加成

培养建议：
- 武器推荐：苍古自由之誓、祭礼剑、铁蜂刺
- 圣遗物推荐：翠绿之影四件套（风元素伤害加成和减抗）
