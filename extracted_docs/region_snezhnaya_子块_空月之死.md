# 空月之死

囚禁三位女神后，尼伯龙根将月亮改造成了对抗天理的武器。虽然天理曾和三月女神有过和平约定，但眼下月亮对提瓦特的威胁远远超过了可容忍的程度；亦有说法认为天理不知道月神被囚禁的内情，认为是月神率先撕破了和平的誓言[6]。最终，天理击碎了恒月与虹月，将霜月扔到了天外。「恒月」粉碎成数块悬浮于虚假之天外，部分坠落于地面；「虹月」粉碎坠落后为延续生命而与深渊结合，弥合自身残片，成为坎瑞亚人所崇拜的「赤月」。在天之围桓破碎的那日，玉盘坠陨，诸龙众皆为曾背叛了他们的月亮之死而残忍欢呼[4]。当「恒月」陨落，天地为之翻覆；当「虹月」破碎，赤影隐入渊海；当「霜月」停转，列邦归于沉沦[22]。传说明月的高车破碎时，沾染神血的车軎殒入月下的尘世，化作如今分布于挪德卡莱的虹色的宝玉「虹滴晶」。从那以后，大地上的人们不再为死去的神明奉上信仰，唯有那些知晓夜的君王们为何而死的幸存者们依然在徒劳地歌唱着那些早已被遗忘的名字[23]。
月髓「月髓」是月神死后力量凝结而成的东西，本质是一种力量核心。「恒月」月髓由霜月之子作为圣物保存至今，详见至冬#霜月之子；「虹月」月髓如火焰般燃烧不灭，由坎瑞亚人保存并研究，详见至冬#坎瑞亚；未曾破碎的「霜月」则孕育着新的月神。
