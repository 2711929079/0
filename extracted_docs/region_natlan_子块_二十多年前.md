# 二十多年前

欧洛伦有一个弃婴因天生灵魂残缺而遭到遗弃，直到被烟谜主收养，取名为欧洛伦。烟谜主的祭司们曾打算对年幼的欧洛伦进行一种仪式，利用他灵魂残缺的体质，将许多四散的灵魂收容到他体内再送往夜神之国，从而强化地脉力量。但这会导致欧洛伦死亡。不过仪式失败了，欧洛伦也因此活了下来，被茜特菈莉照顾[131]。恰斯卡恰斯卡年幼时因身患怪病被抛弃野外，所幸一只名为栖木普的绒翼龙收养了她，后来恰斯卡回到了人类社会，被花羽会的葵可一家收养[13]。
