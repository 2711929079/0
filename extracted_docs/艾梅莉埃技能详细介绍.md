# 艾梅莉埃技能详细介绍

艾梅莉埃是《原神》中的水元素四星角色，枫丹的音乐家，称号"旋律守护者"。
元素战技「水之旋律」：
- 释放水元素能量，对周围敌人造成水元素伤害并施加"水元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「旋律盛宴」：
- 创造一个持续8秒的水元素领域，领域内的敌人会受到持续水元素伤害。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "旋律的祝福"：水之旋律命中敌人后，有15%概率触发"冻结"效果。
- "水元素增幅"：艾梅莉埃的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "盛宴的守护"：元素爆发持续时间内，艾梅莉埃的水元素伤害提升15%。
