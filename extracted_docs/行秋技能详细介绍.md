# 行秋技能详细介绍

行秋是《原神》中的水元素四星角色，璃月飞云商会的二少爷，称号"飞云之上"。

元素战技「画雨笼山」：
- 效果：释放两段水元素伤害，同时在周身生成雨帘剑。
- 雨帘剑效果：提供伤害减免，攻击敌人时恢复元素能量。
- 冷却时间：12秒。
- 能量消耗：无。

元素爆发「裁雨留虹」：
- 效果：召唤剑雨，对敌人造成持续水元素伤害。
- 特性：后台持续挂水，不占用角色站场时间。
- 持续时间：15秒。
- 冷却时间：18秒。
- 能量需求：80点。

天赋技能：
1. 灵光乍现：雨帘剑存在期间，普通攻击有概率触发额外水元素伤害。
2. 富贵荣华：队伍中其他角色使用元素爆发时，行秋获得元素能量恢复。

角色定位：
- 主要定位：副C
- 次要定位：辅助
- 核心优势：后台挂水、伤害减免、能量恢复

培养建议：
- 武器推荐：祭礼剑、绿剑、黑剑
- 圣遗物推荐：绝缘之旗印四件套（充能效率和大招伤害加成）
