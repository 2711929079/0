# 回环之玉 躯体净化

由于实实在在的暴行，以及对深渊真相的部分隐瞒，奥奇坎被后世的纳塔人视为“暴君”，饱受唾骂。为了取得天蛇船上的通牒金牌，库库尔坎散布奥奇卡纳塔有秘宝「回还之玉」的消息，吸引各部族勇者进入奥奇卡纳塔的遗迹内冒险，结果因邪龙的袭击无一生还。世界任务「流灰之国的迷旅人」：冒险家玻娜的父亲曾经在奥奇卡纳塔遗迹冒险牺牲。玻娜追随父亲，在探索奥奇卡纳塔的过程中，从统律核心种解放了奥奇坎的意识，让其化作小龙「绿松石」与她一同探险（但最终玻娜被库库尔坎取走灵魂，也牺牲在奥奇卡纳塔）。旅行者和「小家伙」来到了奥奇卡纳塔，寻找「回还之玉」，但遭到了邪龙与众多深渊魔物的袭击。「绿松石」和（库库尔坎伪装的）玻娜解救了他们，并与他们一同踏上寻找「回还之玉」的通路。途中击退了一次邪龙。众人登上天蛇船，取得了「回还之玉」，发现「回还之玉」就是「通牒金盘」。此时，玻娜的记忆突然出现，意识到这一切实际上都是库库尔坎的阴谋。随后邪龙冲出来抢走了「通牒金盘」，旅行者和派蒙追上邪龙后，用「绿松石」的力量将其击坠。「绿松石」在意识之中对旅行者表示感谢，随后回归了邪龙体内，为自己的躯体排解了深渊的侵蚀。奥奇坎向旅行者和派蒙坦白了自己的真实身份，得知旅行者和派蒙打算带着「通牒金盘」去火山的朝圣路，为「小家伙」解除「焰主之祝」的诅咒，奥奇坎表示「通牒金盘」很重要也很危险；但因库库尔坎的干预，他已忘了该如何正确处理金盘，“一切答案只能到火山去寻找了”。奥奇坎向旅行者分享了自己的记忆，随后陷入了永恒的沉睡。魔神任务第五章第五幕「炽烈的还魂诗」：在旅行者深入夜神之国对抗古斯托特时，「绿松石」形态的奥奇坎灵魂会与玻娜灵魂一同出现，帮助旅行者对抗深渊，并献上祝福。
