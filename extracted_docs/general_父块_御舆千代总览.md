# 御舆千代总览

## 御舆千代
高傲凛然、面容如月的鬼族少女，夙愿是用炽热沸腾的血气克服生死之运，为日渐稀薄的同族以战鬼之名，立下不朽的功绩。背上「雷之三重巴」的纹样时名为「千代」的鬼族女武者，面对漆黑的军势时，曾被虎躯蛇尾的世外之兽吞下。最后她撕开魔兽的胸腔，得以幸存。这是「虎啮的千代」之名的来由。在此后的岁月中，这一名号渐渐简化为「虎千代」。但在深邃之兽的腹中，她染上了深罪的黯色，也透过猩红的利齿看见同行者被撕碎。因深渊力量影响她陷入噩梦，在梦中，所有的一切都变成深渊幻化而成必须对抗的敌人（即「队长」口中的渊蚀综合症[1]），她最终向御建鸣神主尊拔剑;后来，她持刀之臂与尖角被斩落，如同负伤的野兽般从城中逃向林野，雷电将军向她挥的刀斩掉影向山部分山体[2];再后来似乎是被天狗，或是终末番，或是山中修行的岩藏之胤当成不认得的怪物，收拾掉了罢，因为她俊美的容貌已经因为漆黑的仇恨与负伤痛苦变得狰狞扭曲。抑或是遇见了大蛇遗骸附近的鬼面执剑人形，故此结束了命运的旅途。波穗月白经津曾为千代所配带之刀，但在与将军的真剑对决中断刃，虽送回重锻，却并没有锻成，最终这把「波穗」成了历史。[3]千代沾有深渊力量的角被几只妖狸意外挖出来，几只妖狸因此受到深渊影响陷入噩梦。深渊残余力量重返稻妻，让被影响者陷入噩梦。在五百年后因为旅行者清除他们身上的深渊力量才得以解除[2]。有一长子御舆道启（后改姓为岩藏），有一养子御舆长正。
