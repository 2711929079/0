# 班尼特

班尼特父母分别来自沃陆之邦与烟谜主，在特诺奇兹托克时误入倒影之海遇到深渊魔物，一行人保护仍是婴儿的班尼特到最后一刻，最后化为阿夏[36]。还是婴儿的班尼特被一名老冒险家在绝境中检到，老冒险象将他送到冒险家协会蒙德分会后就去世了。之后，班尼特由冒险家协会的一些无委无子的老冒险家共同抚养长大[130]。
