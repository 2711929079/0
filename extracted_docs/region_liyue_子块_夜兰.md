# 夜兰

夜兰曾与一群同僚参与情报调查工作，直到一次深渊调查任务后，夜兰失去了所有的同伴。夜兰因此认识到自己的不足，开始闭关苦练，过程中获得神之眼。后来凝光找到了夜兰，并与其达成合作，夜兰就此成为凝光的特别情报官[180]。
重云重云出生驱邪世家成为受到某人祝福，具有驱邪“纯阳之体”的方士[181]。因为具有“纯阳之体”，重云驱邪时只需要静坐在現场一段时间，便可以完成委托[182][183]。
