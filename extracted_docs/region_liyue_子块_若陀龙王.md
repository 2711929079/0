# 若陀龙王

近来由于封印出现了一丝裂隙，若陀龙王的恶意便化身阿鸠徘徊于伏龙树旁，寻找着解除封印的时机。但另一方面，若陀龙王并未完全忘记曾与帝君签订的「守护人类」的契约。龙王最后的契约，使龙王的善念附身于璃月名匠昆吾后人昆钧身上，凭着自己的本能寻找着摩拉克斯，希望他能阻止封印的解除。封印的解除被阻止后，「昆钧」将被遗忘的与人类同行的记忆展示给阿鸠，阿鸠在遗憾与悔恨中力竭，重新被封印。若陀龙王的善念也在和钟离短暂叙旧之后力竭，就此消散。「摩拉克斯，若有缘，他日必将再会。」[114]
