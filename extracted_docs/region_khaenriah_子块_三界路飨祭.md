# 三界路飨祭

渊上与渊下宫的深海龙蜥达成同盟：用黑雾笼罩整个渊下宫，方便龙蜥精锐潜下大日御舆，拯救被囚禁的同族。而龙蜥作为回报，许可深渊教团在它们领地上通行。深渊教团与龙蜥改造了三座高塔，使渊下宫被黑暗笼罩，但深渊黑雾对龙蜥来说也是剧毒。感到背叛的深海龙蜥便让收养的「海祇御使」后代阿祇找到旅行者帮忙消除黑雾。
在三座塔都处理完了之后，渊上驱动深渊力量，使大日御舆内部出现了翻涌的黑暗。但渊上被随后赶来的旅行者击败。被打败后，渊上对旅行者说阿祇是由深海龙蜥进化而来的「龙蜥人」，并让旅行者引导光路去驱散大日御舆的黑暗。撤退前，他还把在大日御舆中找到的一块挂坠碎片交给了旅行者[82]。之后渊上将与旅行者交手的报告交给了深渊殿下[83]。
调查烬寂海渊上伪装成稻妻人来纳塔调查烬寂海。他以帮助找回龙宝宝为条件，向人打听「回火」的消息，见到旅行者和派蒙路过而急忙离开。之后渊上和基尼奇达成了交易——由渊上提供能够吸引兽境猎犬的装置，帮助基尼奇打开兽境，而基尼奇允许渊上进行「历史研究」。渊上与阿乔二者私下又达成交易：由渊上夺取基尼奇的身体给阿乔，以此对阿乔进行研究。在「回火之夜」前一天，基尼奇通过装置成功召唤兽境后，渊上从兽境中出现。他本想夺取基尼奇的身体，但因为旅行者也在而作罢撤退[84]。之后渊上用计从茜特菈莉处打听到烬寂海的位置[85]。
