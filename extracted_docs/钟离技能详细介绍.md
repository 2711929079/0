# 钟离技能详细介绍

钟离是《原神》中的五星岩元素角色，被誉为提瓦特大陆最有安全感的男人。他的技能体系围绕岩元素和护盾展开，是游戏中最强的护盾角色之一。

元素战技「地心」详解：
- 点按效果：在角色面前创造一根岩脊，对周围敌人造成岩元素伤害。岩脊会持续存在，并与其他岩元素创造物产生共鸣效果。
- 长按效果：除了点按的效果外，还会生成玉璋护盾。护盾的吸收量基于钟离的生命值上限，能为全队提供强大的保护。
- 冷却时间：点按为3秒，长按为6秒。
- 岩脊特性：最多同时存在一个（一命后可存在两个），可以用来阻挡敌人前进或作为攀爬支点。

元素爆发「天星」详解：
- 效果：召唤巨大的星岩从天而降，对范围内敌人造成高额岩元素伤害。
- 额外效果：降低敌人20%的岩元素抗性和物理抗性，持续10秒。
- 能量需求：40点能量。
- 冷却时间：20秒。
- 伤害特点：伤害值基于钟离的生命值上限，生命值越高，伤害越高。

天赋技能：
1. 晶石命理：锻造武器时返还15%矿石，非常实用的天赋。
2. 悬岩宸断：角色提升5%护盾强效，可叠加5层，进一步增强护盾能力。

角色定位：
- 主要定位：全能护盾辅助
- 次要定位：副C输出
- 核心优势：强大的护盾、减抗能力、岩造物共鸣

培养建议：
- 血牛流（推荐）：堆生命值，最大化护盾效果
- 武神流：堆攻击力，主打输出
- 武器推荐：千岩长枪、和璞鸢、护摩之杖
- 圣遗物推荐：千岩牢固四件套
