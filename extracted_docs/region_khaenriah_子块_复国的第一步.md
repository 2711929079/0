# 复国的第一步

为了复国，深渊教团的殿下在犹豫很久后答应深渊使徒，打算强行解除人民身上的诅咒，但解除诅咒的过程会让被诅咒的人民非常痛苦，而且成功的机率微乎其微。殿下认为，就算人民回归永恒的轮回之中，也不要不尊严地死去，并表示即使只有千分之一成功的机率，计划也要执行。然而此时戴因被深渊使徒的传送门阴差阳错地传送到了层岩巨渊深处。旅行者也因为接受委托来到层岩巨渊。此时深渊教团占领了层岩巨渊深处的建筑并打算启动净化装置，强行解除汇聚于此的坎瑞亚遗民（丘丘人）的诅咒并让他们在痛苦中死去，最终戴因的前部下——已经化为黑蛇骑士的哈夫丹阻止了这荒唐的仪式。事件解决后，戴因再度与旅行者辞别。[4]
命运的织机完工随着深渊教团「命运的织机」计划的推进，其操控地脉、投放记忆的能力开始增强。卡利贝尔不甘心自己在这个世界上仅仅活了八岁，他好奇，自己如果安静地活在这个世界上会经历怎样的人生。于是他操控「命运的织机」开始向维摩庄的村民们投放自己和他们一起生活的记忆。在记忆中，他为人热情、乐于助人，还擅长陪伴他人、安慰别人。他还听村民阿托莎倾诉，变成了她无比亲近的朋友，阿托莎甚至对他产生了情愫。但由于村民们在现实中找不到卡利贝尔，许多维摩庄的村民前往冒险家协会发布寻人委托。许多冒险家前来寻人，但一无所获。旅行者接到委托，调查「只存在于记忆中的人」。戴因斯雷布也追查深渊教团来到此处。二人认为这个现象与「命运的织机」有关，决定联手调查。第二天一早，戴因斯雷布的记忆中出现了一段关于他的片段。他决定用「耕地机的眼睛」引出深渊教团。在旅行者和戴因斯雷布进入虚假的「藏宝地」之前，卡利贝尔向戴因斯雷布投放了「戴因斯雷布将『眼睛』交给旅行者」的虚假记忆。之后戴因斯雷布吩咐旅行者先离开，去调查失踪者的事。戴因斯雷布原本的计划是将深渊教团引进虚假的藏宝地，并将藏匿于自己身上的「眼睛」交给旅行者，自己独自面对殿下。但是由于戴因斯雷布没有真正地将眼睛交给旅行者，再加上他在与殿下对战时仍显犹豫，第一颗「眼睛」被殿下夺走，命运的织机因此完成。旅行者和派蒙从「藏宝地」返回维摩庄的过程中，发现受深渊教团影响而躁动的丘丘人们安静了下來。二人通过推理，推测出了卡利贝尔的存在和此时所在的位置。二人来到了卡利贝尔和阿托莎在记忆中经常一起交流的那棵树下，找到了正在向阿托莎告別的卡利贝尔。见到旅行者，卡利贝尔非常高兴，将其拉入了自己的意识空间，把自己的遭遇、「命运的织机」计划的真相、自己给维摩庄村民投放记忆的目的和戴因斯雷布的失策都告诉了旅行者。卡利贝尔表示，自己无法窥见「命运的织机」计划的意图，但此刻他的目的已经达成——由他向所有丘丘人投放一段有关爱和童话的故事，安抚所有的丘丘人。旅行者认为，即使卡利贝尔不曾存在过，他也足以安慰、激励许多人了。卡利贝尔的使命完成了，他闭上眼睛，身躯在风中消散。殿下也赶到了卡利贝尔的意识空间，但没来得及见他最后一面。旅行者回到了现实的维摩庄，发现村民们的脑海中多出了卡利贝尔被父母亲接走的记忆，这正是他最后的告別。旅行者的口袋里凭空多出了一张他、殿下和卡利贝尔在意识空间的合影。在那本应不再存在的空间，卡利贝尔不知用何种手段超脱了規则，为旅行者留下了珍贵的相片。[52]
