# 奥罗巴斯

海祈大御神奥罗巴斯前往挑战岩神，但不敌而逃往稻妻。
千岩团「千岩牢固，重嶂不移。干城戎甲，靖妖闲邪。」千年前在魔神级的战争中活跃的千岩团便是如今千岩军的前身[83]。千岩团来到沉玉谷的赤璋城垣一带抵御妖祟的袭击，期间受到灵渊及其眷属的帮助[24][56]。
