# 梦想升空

有一日，盗火贤龙与一个人类打赌：「无有燃素的力量，人类如何可以驾驭天空？」如果拥有谜烟的智慧，冥想到深处，额头与头顶隐藏的力量会使得身体漂浮起来——可是被问的人并无这种智慧。「这有何难？」男人升腾起腾燎的火堆，然后用缝合的皮革被火焰裹挟着风送上了天。⋯⋯最后他摔死了。[47]
神秘岛流泉之众初代首领乌努库征服了许多危险地带，但都不能满足，于是踏上前往艰险的通往「神秘岛」的道路。他的伙伴们接受他的任性一同前往，克服重重困难，抵达神秘岛前。然而他的伙伴正面临危机可能丧命，看着即将抵达的神秘岛，初代首领决定放弃前往一路返回。这时他的内心得到了真正的满足，原来自己一直想得到的「神秘岛」就是他的家乡与伙伴，对他而言流泉之众就是他的「神秘岛」。[48]
