# 仙像

待魔神的混战结束后，璃月大地上海潮退去，复归和平。仙兽们从此失却了在神的战争中守护凡人的意义，便纷纷隐居起来，过着与世无争的生活。然而，有一些仙兽依旧怀念着追随岩之神的时光，依然渴念着守望璃月的岁月。仙兽虽是超凡的活物，却依旧被寿限所羁绊。因此，它们向岩王帝君请愿，将肉身化作永恒的磐岩。就这样，慈悲的岩之神允准了它们的祈求，将它们化成了永不腐朽的山岩。[16]
鬼神据村落中的长老所言，灵濛山的山民曾信仰诸多鬼神，与山野之间精灵鸟兽中的灵长者同行，但在千年之前那场翻覆天地的混战之后，许多鬼神与精灵陪伴着曾经的住民一同消逝了。尽管如此，多情的群山还是将记忆的回声挽留在了地脉之中，偶然在某个烟雨笼罩的清晨或入夜时分反复重现[92]。
