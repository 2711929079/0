# 灾厄时期

狐斋宫布置了守护鸣神岛的结界，将绀田村附近结界的钥匙交给了绀田家。
狐斋宫担心五百藏为了守护森林，会挺身与绝对无法抗衡的强敌战斗，并因此死去。她以斗法为名，欺骗五百藏躲藏起来。
在离别时，她向友人雷电影提出箴言「不被蒙蔽、不受动摇，一直走在您所坚信的道路上。」
直到最终，在灾厄中，狐斋宫为了保护鸣神岛，而与漆黑的灾厄作战，最终被灾厄吞没了。
在她消失后，她在鸣神岛的眷属「天狐」与「地狐」都化作石像等待她的回归，而普通狐狸则随着一族的灵脉变得稀薄，而不再开口，也失去了法力。
因狐斋宫的记忆非常强大，所以即使是漆黑的灾厄，也没能将其完全消解。灾厄被击败后，它本身作为炽烈的污秽，流入了大地。于是，继承狐斋宫记忆的花散里诞生了。
