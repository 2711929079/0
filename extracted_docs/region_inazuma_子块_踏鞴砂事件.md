# 踏鞴砂事件

赤目一家为了提升锻造效率，将枫丹工匠埃舍尔和他的技术引荐给丹羽。此重大方案社奉行需将此上报将军（锻刀业归社奉行管），而雷电将军也同意踏鞴砂采用埃舍尔带来的新技术（仅为推测，因流浪者角色经历参考《波希米亚狂想曲》，而其中一句歌词直翻为「巴尔泽布放了一个恶魔在我身边」，恶魔明显就是实为博士的埃舍尔，因此埃舍尔来到踏鞴砂可能是也被欺骗的人偶将军同意让他去的。且埃舍尔的原型来自荷兰视错觉知名画师莫里茨·科内利斯·埃舍尔的父亲，为「水利工程师」，后来受天皇批准来到日本援助水利工作。与《原神〉内埃舍尔相同，〈原神〉埃舍尔表面身分也是工程师，枫丹原型是英法等欧洲国家，荷兰也位于欧洲，两者去过日本和以日本为原型的稻妻。既然其原型是被天皇批准来的，那《原神》的埃舍尔很可能也是将军批准来的）。埃舍尔的技术虽然大大提升生产效率，但也引发祟神之力肆虐，造成一连串灾难，最终无人能进入踏鞴砂的核心区域。丹羽与长正为了避免恐慌压下消息，并派人外出求援，第三个外出的是造兵司佑宫崎兼雄，但他也在求援的过程中身亡[69]（可能是愚人众下的手）。倾奇者随后亮出将军的金饰自愿再去求援，打算透过金饰让幕府派兵增援。倾奇者被众人寄与厚望，但御舆长正却将他囚禁，说到「这枚金饰是将军大人所赐身份之证，但你既非人类亦非器物，在下只能这样处置你，还请你不要怨恨！」倾奇者将此理解为「他们被恐惧所困缚视自己为可憎恨之物」（虽然长正这么说，但大概率是长正见求援的人都没归来，不想再无谓的牺牲任何一个同伴），后来倾奇者被桂木放走。倾奇者知道丹羽需要雷电将军的帮助，却不知这一年雷神已用自己为材料制造出完美的人偶，将管理职责托付给了「永恒的守护者」。他乘一艘小舟，顶着海上雷暴雨，不顾安危来到天守阁前求见雷电将军。在前往搭船去稻妻城天守阁的路上，他被一名孩童当作流浪者叫住。「流浪者，流浪者，你要去哪里啊？」他是踏鞴砂工匠的孩子，虽然生了病，却仍有清澈的双眼。倾奇者告诉孩子，自己必须去稻妻城。「可现在好大的雨，他们说之前离开的人也都没有回来！」倾奇者张了张嘴，最后只好对孩子微笑，[65]和孩子告别后匆匆离去。倾奇者来到天守阁，但真正的雷电将军早已身处一心净土。倾奇者屡次遭到拒绝，情急之下取出那枚金色羽毛示众，转而请求觐见八重神子。八重神子忙于辅佐永恒的守护者，甫一听说此事便匆忙赶来，也只来得及与急火攻心的倾奇者见上一面。她承诺马上待人处理此事，失去所有耐心的倾奇者却置若罔闻，将金羽丢在地上，带着幕府放弃了踏鞴砂的绝望之情返回来处。等他再次踏上这座岛屿，先前叫住他的孩子已经不见影踪。丹羽自告奋勇要用埃舍尔开发的吸收污秽的力量进入炉心，但被埃舍尔杀害。原来埃舍尔是由「博士」假冒，受「丑角」命令前来制造踏鞴砂事件，以嵌入稻妻的运转。之后他诱使人偶离开踏鞴砂，并谎称丹羽畏罪潜逃，将一切都推卸给该地负责人。长正本要以死谢罪，但桂木自愿代替他送死，长正虽然不舍，但为了振兴家族的大业，他以渎职为由用大踏鞴长正将桂木身子斩成两半，随后将此刀丢入火炉中。阿望实在不平、不忍，将全然熔毁的刀取出……严重烧伤，于当天夜里死亡。之后，金次郎将长卷与阿望的图画藏于兵库中。大踏鞴长正因此得名「桂木斩长正」。撤离时，原本应当将兵库钥匙分为三片，一片交予目付大人、一片交予造兵司正大人、一片留在踏鞴砂中，以防贼人侵入。然而实在太过匆忙，金次郎无法找到目付大人与造兵司正大人，于是将三片钥匙藏在踏鞴砂内三处宝箱中。当时在踏鞴砂当工人的阿釜祖先不满御舆长正的作为离开踏鞴砂，前往鸣神岛。釜一家会将自己家族的历史透过口耳相传传递下去。[3]御舆长正此后回到天领奉行默默工作，其后代也秉承他的意志做个工作狂。[46][64][65][68]
御舆长正御舆长正在天领奉行默默工作，有了儿子，并且活到了寿终正寝[46]。生前着《稻妻名物帐》一书。
