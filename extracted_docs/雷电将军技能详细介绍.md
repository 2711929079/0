# 雷电将军技能详细介绍

雷电将军是《原神》中的雷元素五星角色，稻妻的雷神，称号"永恒的守护者"。

元素战技「神变·恶曜开眼」：
- 效果：释放「雷罚恶曜之眼」，队伍中所有角色攻击命中敌人时触发协同雷元素伤害。
- 额外效果：提升队伍中所有角色的元素爆发伤害，提升幅度基于元素爆发的能量需求。
- 持续时间：25秒。
- 冷却时间：10秒。
- 能量消耗：无。

元素爆发「奥义·梦想真说」：
- 效果：开启「梦想一心」状态，普通攻击、重击和下落攻击转化为雷元素伤害，且不消耗体力。
- 额外效果：获得护盾，护盾吸收量基于雷电将军的生命值上限。
- 持续时间：7秒。
- 冷却时间：20秒。
- 能量需求：80点。

天赋技能：
1. 愿力系统：通过元素战技积累愿力，提升元素爆发的伤害。
2. 殊胜之御体：元素爆发期间，普通攻击命中敌人时恢复元素能量。

角色定位：
- 主要定位：充能辅助
- 次要定位：主C输出
- 核心优势：全队充能、提升大招伤害、高额输出

培养建议：
- 武器推荐：薙草之稻光、渔获、西风长枪
- 圣遗物推荐：绝缘之旗印四件套（充能效率和元素爆发伤害加成）
