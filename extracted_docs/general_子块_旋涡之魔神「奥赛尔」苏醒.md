# 旋涡之魔神「奥赛尔」苏醒

执行官「公子」想得到岩神之心，于是前往藏匿岩神尸体的黄金屋，发现岩神假死后放出封印在孤云阁的魔神奥赛尔，打算逼岩神现身阻止。但在旅行者、七星、仙人的共同努力之下，将其重新封印。
终结一切契约的「契约」岩神打算退位，让人们治理璃月，但担心人们无力治理，为了考验自己的人民于是策划一切，让人民对抗复苏的魔神。为了达成此目的，岩神与冰神签订契约，愚人众将促成「考验」的完成。在契约的最后，岩神将其神之心交给「女士」
