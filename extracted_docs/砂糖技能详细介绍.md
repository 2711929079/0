# 砂糖技能详细介绍

砂糖是《原神》中的风元素四星角色，西风骑士团炼金术士，称号"无害甜度"。
元素战技「风灵作成·陆」：
- 创造一个风元素领域，吸引敌人并造成风元素伤害。
- 冷却时间：12秒。
- 能量消耗：无。
元素爆发「禁·风灵作成·柒伍同构贰型」：
- 释放强大的风元素能量，对周围敌人造成风元素伤害，并创造一个持续10秒的风元素领域。
- 冷却时间：15秒。
- 能量消耗：60点。
固有天赋：
- "炼金的学徒"：砂糖的元素战技和元素爆发可以将敌人转化为"小风车"，造成额外伤害。
- "风元素增幅"：砂糖的普通攻击和重击命中敌人后，有15%概率减少元素战技的冷却时间1秒。
- "元素转化"：元素爆发持续时间内，砂糖可以将敌人的元素转化为风元素。
