# 可莉技能详细介绍

可莉是《原神》中的火元素五星角色，西风骑士团火花骑士，称号"逃跑的太阳"。
元素战技「蹦蹦炸弹」：
- 投掷一个炸弹，落地后爆炸造成火元素范围伤害。
- 炸弹可以弹跳，多次弹跳后爆炸。
- 冷却时间：8秒。
- 能量消耗：无。
元素爆发「轰轰火花」：
- 释放一个巨大的炸弹，造成大范围火元素伤害，并分裂成多个小炸弹。
- 冷却时间：20秒。
- 能量消耗：60点。
固有天赋：
- "火花骑士的信条"：蹦蹦炸弹和轰轰火花的伤害提升15%。
- "炸弹玩偶"：可莉的普通攻击命中敌人后，有20%概率触发"爆炸"效果。
- "灼热的回忆"：元素爆发持续时间内，可莉的移动速度提升20%。
