# 流灰之国的迷旅人

旅行者、派蒙和他们的龙伙伴，身负「焰主之祝」的幼嵴锋龙「小家伙」来到了奥奇卡纳塔，并遭到了邪龙与众多深渊魔物的袭击，「绿松石」和被瓦萨克拉胡巴肯取得灵魂，又放出来作为陷阱的玻娜解救了他们，并与他们一同踏上寻找「回还之玉」的通路，途中击退了一次邪龙。经过一番探索，众人终于登上天蛇船，并取得了「回还之玉」，发现「回还之玉」就是「通牒金盘」，玻娜的记忆突然出现，意识到这一切实际上都是瓦萨克拉胡巴肯的阴谋，随后邪龙冲出来抢走了「通牒金盘」，旅行者和派蒙追上邪龙后用「绿松石」的力量将其击坠到地下。「绿松石」在意识之中对旅行者表示感谢，随后回归了邪龙体内，为自己的躯体排解了深渊的侵蚀。奥奇坎向旅行者和派蒙坦白了自己的真实身份，得知旅行者和派蒙打算带着「通牒金盘」去火山的朝圣路为「小家伙」解除「焰主之祝」的诅咒，奥奇坎表示「通牒金盘」很重要也很危险，但因瓦萨克拉胡巴肯的干预，他已忘了该如何正确处理金盘，一切答案只能到火山去寻找了。奥奇坎向旅行者分享了自己的记忆，随后陷入了永恒的沉睡。[69]
