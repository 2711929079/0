# 阿釜

鹤观人阿釜的祖先在清籁雷暴后离开越石村，前往踏鞴砂成为工人。[3]
雷电五传經津流经津第三代─经津实锻造了带着蓝海波光的美刀─波穗月白经津，波穗月白经津曾为御舆千代所配带之刀，但在五百年前与将军的真剑对决中断刃，虽送回重锻，却并没有锻成，最终这把「波穗」成了历史。而经津实的儿子─也就是经津第四代经津弘芳却没有其母的技艺。但他的义兄，经津实的养子─政芳却不同，他从养母手中获得「真砂丸」之名，与经津之艺。原本实打算让真砂丸继承经津流，但真砂丸都婉拒，随后实离世，留下真砂丸与其义弟弘芳。真砂丸因领命重铸「波穗」，最后造出了唯一有自己铭名的「波乱月白经津」，因其剑与「波穗」相似，却又有完全不同的气质，真砂丸声名大噪。但这严重影响了经津流真正的传人─其义弟弘芳的地位。随后真砂丸传授完经津流给弘芳后，随即离开了此地，四处流浪，并钻研各种锻刀流，晚年出了三位得意弟子─枫原景光、丹羽长光、赤目实长，也就是后来的「一心三传」。[11]
