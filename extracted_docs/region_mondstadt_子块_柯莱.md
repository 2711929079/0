# 柯莱

柯莱来自须弥，曾是「愚人众」十一席之一「博士」人体实验计划（注射魔神残骸）的实验对象。[155][156]来到蒙德后与「侦察骑士」安柏相识。柯莱告诉她，来到蒙德是为了寻找一样东西（寻仇）[155]
回收「愚人众」为了回收「文件」与「种子」投放机关鸟，企图吸引魔物，破坏祭典，引发混乱。骑士团在发动之前将其全部摧毁。[156][141]
