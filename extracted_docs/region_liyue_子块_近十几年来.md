# 近十几年来

烟绯烟绯的父亲是一位仙人，曾与岩王帝君征战许久，与一位普通的商人结婚，生下了烟绯。烟绯长大后，父母便出门远游去了。烟绯在萍姥姥的照顾下生活，逐渐成长为璃月头牌法典解读专家。凝光在每年修订法典时，都会大量参考烟绯的所作所为[160][167][161]云堇云堇出身于璃月曲艺世家（原是锻刀世家[168]），母亲继承了祖父的衣体，曾是一代名角，父亲则是一名剧作家。在耳濡日染下，云堇小小年纪便开始学戏[169]，年少成名，成为了“云瀚社”的当家，是璃月港内风头正劲的名角[170][171]。辛焱出身于贫困农家的辛焱背负着父母极高的期望，各种各样的“成见”支配了辛焱的童年。直到有一天，辛焱接触了来自枫丹的“摇滚”，自此开始用“摇滚精神”反抗其他人对她的“成见”[172][173][174]。辛焱改造了从枫丹传来的摇滚乐器，作为摇滚的践行者在璃月港进行表演。一开始辛焱搭建的临时舞台总是会在表演最热烈的时候燃烧起来，引起了璃月上层的注意，以“防止民众受伤”为由，禁止了辛焱的演出。但一位名叫 “大坤”的千岩军是辛焱的热衷粉丝，总是偷偷告诉辛焱去往千岩军巡逻的死角进行演出[175][176][177]。凝光专门修订了法典，划定了一块演出专属场地，只要申请了许可，便可以进行演出[178]。云堇非常欣赏摇滚乐，与辛焱成好友，经常去听辛焱的摇滚演出。因为大人们总是教育云堇不要去听摇滚，于是云堇与范二爷合作，捏造了一位名叫“星燕”的端庄大方的女生。云堇每次去听摇滚，范二爷就会对别人说云堇在和星燕相处[179]。夜兰夜兰曾与一群同僚参与情报调查工作，直到一次深渊调查任务后，夜兰失去了所有的同伴。夜兰因此认识到自己的不足，开始闭关苦练，过程中获得神之眼。后来凝光找到了夜兰，并与其达成合作，夜兰就此成为凝光的特别情报官[180]。重云重云出生驱邪世家成为受到某人祝福，具有驱邪“纯阳之体”的方士[181]。因为具有“纯阳之体”，重云驱邪时只需要静坐在現场一段时间，便可以完成委托[182][183]。嘉良十年前，嘉良患上了罕见的心病，白术的师父本就时日无多，还继续折损寿命来医治嘉及，江蓠不忍，带着嘉良远离师门，避而不见。为拯救嘉良的生命，江蓠利用“螭”的魔神残渣炼就了一味毒，成功保住了嘉良的生命，但代价是心智和记忆都会被魔神残渣慢慢侵蚀。嘉良和江蓠约定，等到二人的孩子“阿雩”长大，就让自己上路。[77]嘉明嘉明的父亲是一位普通的商人，做茶叶生意，希望嘉明能继承家业。嘉明想要从事小时候最爱看的舞兽戏，但父亲认为这根本不是事业，二人意见不合但有母亲在期间做调和。直到母亲因病去世，嘉明练舞兽戏受伤，父子矛盾彻底爆发，嘉明离家出走。嘉明的舞兽戏推广事业并不顺利，为了养活自己，他在一次机遇巧合下成为了一名镖师。期间，嘉明结识了一只小猊兽，并为其取名“文仔”，还成立了“威水舞兽队”，自己做话事人[184][185][186][187]。蓝砚年纪轻轻的蓝砚已然成为沉玉谷手工艺工会中最厉害的藤编师，因为出身于擅长使用奇门术法的蓝氏，蓝砚还掌握一些基础的术法[188][189]。刻晴几年前，出身名门望族的刻晴在祖父过世后年纪轻轻便成为了璃月七星中主管土地建设的玉衡星[80]。行秋飞云商会的二少爷行秋向往武学，在几年前拜师于衰落已久的古华派门下，并且仅用四年就自学领悟了古华派的三大武学奥义，被视为是古华派振兴的希望[137]。香菱与锅巴万民堂年少有为的厨师香菱几年前在寻找食材时，结果天降大雨。又累又饿的香菱进了一个山洞中避雨，将随身携带的辣肉窝窝头放在了山洞中神龛前的供桌上，随后睡着了。醒来后看见失去知性的马科修斯正在吃辣肉窝寓头，二者就此成为朋友，香菱为其取名为“锅巴”[190][80][191]。瑶瑶香菱向萍姥姥学习枪法。出生在轻策庄的瑶瑶被萍姥姥看中，瑶瑶的父母在被劝说后，最终同意瑶瑶跟随萍姥姥去往璃月港生活。年幼的瑶瑶字还没识全就拜入萍姥姥门下，成为香菱的小师妹。萍姥姥教瑶瑶生活技能、诗书礼仪以及枪法。留云借风真君还制作了“月桂”作为防身之物，将其送给了瑶瑶[192][193][194][195]。灵渊灵渊不忍两位友人浮锦和药君的生存环境随着人类对沉玉谷的开发而逐渐偏离自然法则，抱着即使被热爱人类的浮锦谴责也没关系的心态暗地重塑沉玉谷的地脉，慢慢地将沉玉谷改造回最原始也最适合三仙居住的环境。随着灵渊改造计画的实施，沉玉谷出现了大范围的氲气，近年来翘英庄的茶叶质量也开始下降。浮锦在锦落庭察觉了地脉异常，但力量不完全的她无能为力。[24]安柏祖父回到璃月安柏的祖父从蒙德不告而别，理所当然地被西风骑士团视为叛逃，导致其带领的侦查小队保受波及[196]。他实为回到璃月轻策庄[197]。沈秋拾剑录行秋将自己的故事为素材，以“枕王”作为笔名撰写了《沉秋拾剑录》，但在璃月无人爱看。后来一位稻妻的商人将这本书带到了稻妻的八重堂，结果广受好评[198]。海山北斗曾多次挑战名力“海山”的冥海巨兽，但都以失败告终。后来北斗再一次率领船队挑战海山，战斗持续了整整四天，最终北斗给予了海山终结一击，将其头颅彻底砍断，宣告胜利[199]。烟绯与优菈一艘装载着深渊教团危险品的商船要从荆夫港运到璃月港。烟绯作为跟随商船的顾问发现货物有鬼，就擅自进行了调查，结果引火上身，还好优菈等人及时赶到，救下了烟绯，并在其帮助下将深渊教团成员一网打尽[200][201][202]。久岐忍久岐忍来到璃月的“同文学塾”进修法学，期间烟绯还替久岐忍的老师代过课。久岐忍在毕业前夕回到稻妻探亲，结果正好碰上稻妻锁国令颁布，无法回璃月拿毕业证[203][204][20]。层岩巨渊兩年前，由于晶石影响，七星决定透过碧鉴封锁层岩巨渊，但实际上层岩巨渊的开采并没有彻底终止，部分领有特许执照的矿工仍在该地开采以备不时之需[205]。自从经历了大灾厄后，层岩巨渊地下最深处的天钉封印便发生了松动，导致深渊污染渗透而出，而数年前深渊教团盯上了巨渊之底里的某种秘密，暗中进行破坏，导致层岩巨渊发生了各种异变：地下出现了蕴含污染的黑泥，还出现了一种被称为流明石的神秘矿石；许多在地下工作的矿工产生了严重精神问题；还有许多丘丘人诡异地往巨渊之底聚集[7]。璃月高层不得不遣散工人，封锁层岩巨渊，暂停当地的矿业开采。与此同时，愚人众执行官公鸡获得了一些有关层岩地下污泥的情报，于是派出了一支军团「第九联队」前往璃月，协助调查层岩地下的异变。当时璃月和至冬关系尚好，这一批派往层岩地下的愚人众先遣团在出征时还得到过人们的祝福[206]。层岩巨渊大部分区域的关闭使得部份矿工失去工作，有许多矿工因而成为盗贼。因层岩巨渊关停，黑岩厂生产压力大增导致工人无法负荷[207]。层岩巨渊被封闭后，志琼决定成为一名冒险家[153]。璃月与至冬执行官「公鸡」掌控了关于根绝层岩巨渊黑泥污染的情报，于是派兵驻守层岩巨渊，协助调查并根绝层岩巨渊的污染。当时愚人众甚至被璃月人民视为英雄。在至冬和璃月关系尚好，即愚人众派兵驻守层岩巨渊之下时，「女士」曾经和璃月七星谈成了一些互利的贸易协定。[208]夜兰与「富人」夜兰渗透了愚人众执行官“富人”在暗中布下的隐蔽商路，成功拦截了运载的货物，也因此损失了一只名为“幽奇腕阑”的祖传玉镯。货物中有一件珍稀的魔兽皮毛，本来是计划送给至冬女皇的礼物，夜兰将其作为玉镯的赔偿收走了，并制作为一件大衣[209]。枫原万叶北斗救下并收留了正逃离稻妻幕府通缉的枫原万叶[210][211][212]。
