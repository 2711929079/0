# 迪希雅

从前沙漠有一支名叫“神王之遗”的大型佣兵团，由许多的附属团构成，臭名昭著、坏事做尽，组织内部采取一套严酷的管理制度，加入的成员都必须要留下案底，从而便于威胁与掌控，无法轻易脱离。迪希雅的父亲库塞拉曾是神王之遗下属的“阿赫玛尔之须”的团长，但他并不愿意和总部同流合污，从不做抢劫、走私、谋杀这些非法勾当，更从来都不让女儿接触镀金旅团的黑暗规则。在迪希雅因为误解了父亲而离家出走后，库塞拉一直暗中派人保护她。后来迪希雅成长为了能够独当一面的强大佣兵，并建立了“炽光之狮”佣兵团。后来风烛残年的库塞拉决定独自前去神王之遗的总部，用一把火烧光了阿赫玛尔之须所有成员的案底，自己也死在了大火中。阿赫玛尔之须的成员虽然从此自由了，但他们都发誓要为团长报仇，在后来几年里一直都在与神王之遗对抗[90]。
