# 西摩格

五百年前的坎瑞亚灾厄中，前代水神厄歌莉娅死于甘露花海，身躯化作「甘露活水」。神鸟西摩格饮下「甘露活水」，将自己拥有的「灵光之力」与甘露相融合，形成灵光百种。其中具有智性的即是花灵。最初的花灵祖尔宛，与其他的花灵都参与了对深渊兽潮的战斗。花灵和戴因斯雷布、东方来的那伽朱那团的独臂贤者、那伽朱那、坎瑞亚的残军一起对抗过漆黑兽潮。其后，一部分花灵和它们的搭档组成了最初的矫论团。在那之后，戴因斯雷布与旅行者的血亲离开了。[72]
独臂贤者东方来的那伽朱那团的独臂贤者在对抗漆黑兽潮后前往地下考察坎瑞亚遗迹，遇见了白鸽骑士当中的海杜菈。
