# 丝柯克

丝柯克出生在另一个星球，那颗星球上的人们过着朴素的生活。丝柯克小时候不喜欢学习，经常逃学去欣赏风景，憧憬着长大后去欣赏更多的美景。她母亲对她的行为不满，但父亲对此不置可否。当时，丝柯克所在星球的人们与天外到来的商船有所接触，居民们以友善的态度对待外星人，然而商船在勘察中发现丝柯克所在星球的人们拥有适应极度恶劣环境的天赋，外星人们将此认定为威胁，于是呼叫士兵杀害了所有的当地人。丝柯克的父母抵抗最为激烈，伤到了外星人的士兵，被侵略军的队长下令虐待后杀害。丝柯克逃课后回到村里，只发现了侵略军烧杀抢掠的一片火海，与哭喊着被屠杀的人们。丝柯克找到了奄奄一息的父母，父母鼓励她努力活下去。就在侵略军准备杀害丝柯克时，坎瑞亚五大罪人之一的「极恶骑」苏尔特洛奇路过此处，顺手消灭了侵略军，并按照维瑟弗尼尔的预言，收丝柯克做了徒弟。极恶骑对她的坚强感到满意，以魔鬼式的训练方式训练丝柯克，让她挥剑直到骨骼断裂、肢体破碎，然后自己再用深渊力量复原她的肢体，循环往复，永无止境[27]。
