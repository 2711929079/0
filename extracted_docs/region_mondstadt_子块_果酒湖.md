# 果酒湖

环绕蒙德城的天然淡水湖泊。宽广的湖面和粼粼的波光历经千百年不变，水澈见底，入口清冽甘甜。正是这样的水源成就了蒙德美酒的盛名，也让果酒湖成为了蒙德的重要标志之一。果酒湖是蒙德城重要的水源地，湖水会随着蒙德城的沟渠汇入城中。[2]
