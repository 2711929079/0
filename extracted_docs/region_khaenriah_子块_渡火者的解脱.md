# 渡火者的解脱

渊上调查烬寂海是为了寻找自己认定的家乡，发现烬寂海能前往时间线的每个角落后前往步过烬寂海[22]。传说一种鸟，生而孤独，能在火中高歌，人民将它作为图腾崇拜，君主以之为贵。火山之地的智者佩着它的羽毛，在烈火中退隐。孤独而来的他最终也孤独而去，下落从此不明。从此以后，人们时常听到烬寂海深处传来鸣唱。不知是浴火之鸟的鸣叫，还是渡火之贤者的吟叹呢。[86]
渡火者的智慧流浪烬寂海的智者（渊上）渡过能前往任何时间点的烬寂海到达许多过去的文明，换过不同的身分，期间身在四百年前某文明时身分为学徒，用赤色玛瑙打造了一尊冠冕，用以适应烈火与高温。智慧与灼热执念成就了防火的冠冕，却引来了同僚和前辈的恐惧妒忌。「这自大的年轻人，竟敢挑战烬寂海的烈焰，这是百年来未有的亵渎」「火海必将他吞噬蚕食净尽，连灰烬也被热浪扬入高空，终化做虚无」善妒的老师企图刁难他的门徒，便要他戴上冠冕，步入火海。渊上见眼前之人这副德性，知道这里不是心目中的故乡于是离去。那位導師眼见这冠冕的主人在岩浆中悠然漫步，消失在了视野之外。[87]（仅为关于渊上的时间线解读之一，有其他解读）
