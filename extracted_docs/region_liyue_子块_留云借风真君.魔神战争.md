# 留云借风真君.魔神战争

在魔神战争中追随帝君，守护璃月。当时大多以人类姿态现身于世，果敢伶俐，受人敬仰[65]。留云借风真君以人形参与魔神战争，并多次庇护受难的人群。战后隐居于奥藏山，无拘无束，多以鹤形态现世[47][66]。魔神战争當时没人在求仙問道，於是太山谷被仙人藏入水下封印起来。
铜雀摩拉克斯曾为夜叉中的一位小仙赐名“铜雀”，他喜欢吃“烤螭虎鱼”。但铜雀敢终因实力不济，战死于魔神战争[67]。
