# 久岐忍

久岐忍来到璃月的“同文学塾”进修法学，期间烟绯还替久岐忍的老师代过课。久岐忍在毕业前夕回到稻妻探亲，结果正好碰上稻妻锁国令颁布，无法回璃月拿毕业证[203][204][20]。
层岩巨渊兩年前，由于晶石影响，七星决定透过碧鉴封锁层岩巨渊，但实际上层岩巨渊的开采并没有彻底终止，部分领有特许执照的矿工仍在该地开采以备不时之需[205]。自从经历了大灾厄后，层岩巨渊地下最深处的天钉封印便发生了松动，导致深渊污染渗透而出，而数年前深渊教团盯上了巨渊之底里的某种秘密，暗中进行破坏，导致层岩巨渊发生了各种异变：地下出现了蕴含污染的黑泥，还出现了一种被称为流明石的神秘矿石；许多在地下工作的矿工产生了严重精神问题；还有许多丘丘人诡异地往巨渊之底聚集[7]。璃月高层不得不遣散工人，封锁层岩巨渊，暂停当地的矿业开采。与此同时，愚人众执行官公鸡获得了一些有关层岩地下污泥的情报，于是派出了一支军团「第九联队」前往璃月，协助调查层岩地下的异变。当时璃月和至冬关系尚好，这一批派往层岩地下的愚人众先遣团在出征时还得到过人们的祝福[206]。层岩巨渊大部分区域的关闭使得部份矿工失去工作，有许多矿工因而成为盗贼。因层岩巨渊关停，黑岩厂生产压力大增导致工人无法负荷[207]。层岩巨渊被封闭后，志琼决定成为一名冒险家[153]。
