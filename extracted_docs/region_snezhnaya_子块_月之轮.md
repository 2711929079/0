# 月之轮

那时，高天所降下的律法尚未结集，人理的边界尚未划定。神的子民散布在新造的园囿与山谷，其数目不可胜数。天空的使者行走在蒙昧的人群当中，播散繁荣与智慧。最后，神所许诺的繁荣启迪了智识，智识引发了怀疑。为了寻求答案，人们选出了祭司；因为恐惧答案，祭司膏立了君王。神的使者为凡人的愚妄而震怒，面对他们的疑问，御使们缄默不言。于是人们转而望向天空，在日光不及的黑夜中为三轮皓月献上祭礼。夜空之上，梦与诗歌的女儿们（三月女神）无时无刻不在观察着尘世的变迁。在阿兰若国的酣息中，在七十层的阶梯下，在银璨璨的舫舻边。直到她们听见，这些可怜小人儿的心房，也会随着月震而悸动。夜幕为大地垂下了眼眸，正如银星曾为龙众的天船亮起航灯。她们将疯狂与灵感带回月下，那是「秩序」无法给出回答。当晨星再度升起，自山巅归来的主祭带回了一枚小小的轮符。这枚「月之轮」，便是三月与众民订下的新约[2]。
月神信仰为将失乡的魂灵引入月上的金屋，银色的大船驶入梅洛皮斯的港湾。那曾为流转于元素之息中的生命预留的乐土，如今也对凡人敞开大门。追逐着潮汐的人们赞颂银光的君王，向月夜的三重冕献上欢歌与舞蹈。相比于烈怒无情的天父，三月的柔光总如慈母般抚慰所有迷途的灵魂。因此地上的众邦也对月夜的主母献上了最多的热枕。他们登上芬德尼尔的山巅，向皓月献上桂冠的诗篇；或者步入琅轩的祭场，用精雕的美玉映衬夜的容姿；又或是在阿卡荻亚的圣林，折取橡树林中的金枝桠。在太阳升起之前，一切僭越天空的爱欲皆从此滋生。但在永恒天的主宰为人之子预选的家园中，月光亦不过是圣爱的一缕。对天空的试探是绝不允许的，因为一切逾越规则的求索都将指向毁灭[2]。
