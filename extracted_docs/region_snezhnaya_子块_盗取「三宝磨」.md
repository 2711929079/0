# 盗取「三宝磨」

霍德望来到挪德卡莱后，利用雷德·米勒为受欺凌和侮辱的人们创造「乐园」的理想，劝诱其跟从自己[33]，成为自己的弟子，并用谎言让他替自己取得「三宝磨」[34]。流行于那夏镇的故事里，大盗雷德·米勒被围困之时，正是靠着霜月的圣女爱莉厄送给他作为定情信物的符石隐去身形，逃脱了追捕。霜月之子的追随者坚决否认这样的说法，将其斥为胡言乱语，毕竟现在的寒涌石也没有这样神奇的能力[15]。为免除三宝磨引来众人抢夺的后患，当雷德米勒与追随他的爱莉厄前往盗取三宝磨时，爱依菈默许了[42]。从对话来看，此时的爱依拉应仍屈居于娄维娅的阴影下，并对姐姐的离开有诸多埋怨。
在过去也被称作「大盗的七道具」的精妙装置，据传其发明人正是闻名诸国的大盗雷德·米勒。如同最初的盗宝团中汇聚了形形色色的人们，将诸多原本用于不同意图的小道具整合在一处，便能破除人为设下的界限，无论是锁链还是疆域。其中收纳的工具数量并非只有七类，只是人们习惯于用七之数来称呼种类繁多之物。世界的秩序被认为只有七类，或许也是出于同样的原因[47]。

海螺帮海螺帮最初是由在动乱的时岁（不止坎瑞亚灾变，艾维雷勒安就读军校时挪德卡莱便已有海螺帮）中失去家园、遁入寒林间相依为命的孩子们抱团取暖组成[48]。
在挪德卡莱流传的故事中，孩子们以一只海螺作为话语权的象征，并推举出一个同样是孩童却不畏惧力量、能够带他们走出困境的人作为领袖，以此抵抗将他们视为奴仆的大人。然而当大人们已被清扫，理想的秩序并没有到来，出于对权力的渴望，持有武器的少年从“矮子”处要来了海螺，最终也在多疑、易怒中死去。被其他孩子找回的“矮子”将卷曲了的刀刃扔进海浪里、将海螺放在了众孩童的中间，宣布「我们要在这里建立属于我们的城镇，属于少年们和少女们的乐园。此后的海螺并不会独属于一人，所有人可以轮番举起它，说出任何想要说的话语，作出想要作出的发言…总有新的男孩和女孩，如果他们不喜欢变成大人的我们，也该亲手打破我们的世界」[49]。
