# 芒索斯山麓

临近沫芒宫的滨海山麓，宁谧的微风萦绕着绵袤的花田。在古老的记载中，此处曾有以花为名的宏伟殿宇，于众水之上抚育弃遣的遗孤，颂祷爱的诲谕。昔日的高塔已然不复，繁花却依旧芬馥如初。
