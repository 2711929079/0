# 风龙「特瓦林」

古国灾厄降临的腥臭最终传到了这片土地，而飘荡在欢欣的风中的歌词也被毒龙的咆哮、魔物撼动大地的脚步声、叹息声与烈焰撕碎。
蒙德众人的祈祷最终再度唤出了蒙德的神——风之歌者。天空之琴再度拨动，风龙也再度被唤来。之余，骑士与骑士团也为了国土与家园而战。为了避免杜林的毒血的影响，风神用笛声将杜林引到雪山。战斗的结果是显而易见的，杜林尸骸至今还在蒙德南部雪岭之上。风龙最后获胜了，他咬住了恶龙的咽喉，却咽下了毒血。恶龙的毒血，正是被歪曲的黄金，让山峦崩塌、大地坏灭的力量。风龙守护了蒙德，他以为这样人们就会理解它了吧。于是它陷入了长眠。[25]
狼群兽径猎犬来到蒙德，与当地狼群展开争战，[104]最后兽径猎犬们因不敌而逃往鹤观，仅几只兽径幼兽在奔狼岭蛰伏[105]。兽径幼兽破坏奔狼领北风之狼领地周围的地脉树，导致该地有许多大型断树根，且依靠地脉才能传送的传送锚点离北风狼很远，因为更里面的地脉被破坏了。考虑到杜林也是从坎瑞亚一路跑到忍冬之树，兽径猎犬和黄金王兽跑到地脉树「栖木」所在的鹤观，鹤观雾散后它们在天下人之章第二幕跑去破坏雷樱树，因此它们可能会循着地脉行动。
