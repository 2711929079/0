# 留云借风

魔神战争前，留雲与归终、歌尘浪市真君、灶神等仙人交往甚密，好与归终较量机关术[43]。璃月曾发生一场非常严重的旱灾，名为“旱魃”的仙人为田地带来干早，还散播瘴气与瘟疫。后世被称力“留云借风真君”的仙人帮助人类铲除了旱魃。也正是那次事情让留云获得了留云借风真君的美名[47][2]。留云借风真君受削月筑阳真君和理水叠山真君相劝，再加上摩拉克斯、归终、歌尘浪市真君的赞同，留云借风真君借出了自己所居住的玉虚，并在这之上建造了一处亭榭，以造福后来的求取仙道者。凡人求取仙道需要上下求索，上天入地，而留云借风真君借出的王虚（就是庆云顶上空的那个浮空岛）则力“天”的试炼，“地”的试炼则是位于绝云间的“太山府”[6][48][49]。
炉灶之魔神「炉灶之魔神」马科修斯诞生于石块碰撞击出的火星。他司掌炉火与民生，是一位喜爱人类的魔神。千年前古人扩建城市，到平原上建立聚落，取名「归离集」。灶神惜护民生，化出分身潜入千家万户帮着生火做饭，助人团圆[50][44]。
