# 灾厄

因为坎瑞亚并不存在稳固的地脉，纳塔成为深渊侵蚀最严重的国家。在灾厄过后，纳塔人花费数百年的时间才把深渊造成的灾厄逐渐解决，但这只是表象而已。纳塔的地脉早已被侵蚀殆尽，被彻底深渊化只需500年时间。之后的纳塔不断遭遇深渊入侵，纳塔人为了守护纳塔也变得较不常前往其他国家。[10]娜娜丝梨原本是特诺奇兹托克的特产，后来在五百年前灭绝了[113]。传说伟大的祭司将手中的镜子放在了地上，于是形成了镜璧山，以抵挡漆黑的奔流[114]。维护镜璧山的涂鸦需要代代相传的努力，据说曾有睡在巨大涂鸦画下的人于睡梦中梦见巨鸟张开翅膀的模样[115]。镜璧山上的紫色鸟涂鸦，可能就是雷鸟的形象。这个涂鸦也很像卡帕奇莉，或许就是迁居清籁岛前的卡帕奇莉或她的同胞。
坎瑞亚火之神受天理号召，前往参加坎瑞亚之战。[116]
