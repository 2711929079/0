# 「螭」

彼时的璃月北方盘踞着一种名为「螭」的恶兽。根据传说，「轻策庄」的「轻策」一词来源于上古魔兽「螭」。如今的通用语称之为「螭」，而「轻策」是在蛮荒时代璃月先民口中的发音[59]。螭是一种没有角的龙，形似蛟，帝君对抗螭的时候，曾用到了缚龙之法。然而就算击败了对方，螭的身体却不死不灭，血滴落结为石头化作剧毒，污害世间，帝君只能选择将其镇压，束其神于正北、钉其骨于东南、锁其身于西北、缚其魂于东北，压其形于西南。不知过了多久，其骨已销、其神已失、其形其魂已灭、唯独其身仍存，为了封印它的残骸，摩拉克斯，用水将螭的遗骸封堵，又用宝器将其镇压在洞穴中心，才算是解决了这个棘手的难题。在击败螭之后，摩拉克斯教先民们做出了一种名为「岩尊像」的东西，用来镇压螭剩余的力量。轻策庄周围生长着的巨树，比其他地方的树都粗壮不少。当地人把这些树称为「螭血木」，认为这树吸收着螭血而生，所在的地方则是「螭眼」。[60]传说螭死后身躯蜷曲转为顽石，鲜血化为碧水，鳞片成为梯田，曾经的魔兽巢穴变成了如今的轻策山。但这片山区其实大部分由因外力冲击而破碎的巨岩组成，并没有水元素魔兽的生命痕迹存留，魔兽化作山峦的故事只是一个古老的传说[59]。
