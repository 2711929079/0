# 祭礼时期至葬火之战结束

古国第一文明此時高山比天空更高，大地比穹顶更大，稻妻岛屿数量更多，此时海平面较现在低。因没有像蒙德（风神修整山地）和璃月（岩神的岩枪、碧水河是移宵导天真君的血而成、沈玉谷决堤、归离集荻花洲被水淹等等）大规模的地形变迁，稻妻的第一文明遗迹较为完整，都是一地区完整的一片。除了实为第一文明祭坛的秘境外，现存鸣神岛的遗迹位于荒海，当时人们在地下兴建荒海地宫，里面有完整能容纳多个人的平地当作避难所（现如今为恒常机关阵列场地），推测是为了葬火之战時求生兴建。当时的文明会使用雷种子的原理，透过雷屏障来当作屏障。稻妻的第一文明会使用一种立方体机关，靠击打它能解开谜题。稻妻古时建筑大部份似乎倚海兴建，后来葬火之战发生，海平面上涨，大部分遗址被淹没，只有原本在原先的山上的遗址没有被淹没。因为稻妻岛屿遍布，稻妻第一文明的人会打造并使用浪船锚点。稻妻的第一文明的人最终变成丘丘人[3]，但这些丘丘人在五百年前（坎瑞亚灾变前）就全数肉体腐烂殆尽了，到五百年前坎瑞亚灾变发生后稻妻才再度出现丘丘人（原坎瑞亚人）[4]。鹤观前文明鹤观文明信仰着三月女神，与天空岛的使者有联系。他们非常了解三月女神，除了称他们为月亮，还将他们称为赫卡忒[5]。他们会用壁画与石根文字纪录历史。葬火之战后被降下天钉，导致鹤观天空被净化，但也被迷雾笼罩。后来该文明覆灭。[3]清籁文明覆灭的古文明，是第一文明支系。沉眠之庭是在清籁岛还没有如今的名字，没有被染上雷色之前就静静沉眠在此间的庭苑[6]。清籁文明的特殊机关较多，譬如特别的立方体机关，与数个踩格子机关，格子上刻有凯尔特三角结。流沙无名古国曾有一个无名的古国，是第一文明支系。古国的愚王在万古的流沙之上梦想筑造供养银白古树的高庭[7]。樱之庭遥远的古代，高山比天空更高，大地比穹顶更大。直到葬火之战明镜（恒月）破碎，海洋升起，稻妻的低洼处被淹没，远古的岛屿樱之庭与其他岛屿的联系因此断绝了。[8]
