# 林尼技能详细介绍

林尼是《原神》中的火元素五星角色，枫丹歌剧院的首席魔术师，称号"彩星戏法"。
元素战技「戏法飞弹」：
- 释放火元素能量，对周围敌人造成火元素伤害并施加"火元素附着"。
- 冷却时间：10秒。
- 能量消耗：无。
元素爆发「魔术盛宴」：
- 释放强大的火元素能量，对周围敌人造成火元素伤害，并创造一个持续10秒的火元素领域。
- 冷却时间：18秒。
- 能量消耗：60点。
固有天赋：
- "戏法的祝福"：戏法飞弹命中敌人后，有15%概率触发"燃烧"效果。
- "火元素增幅"：林尼的普通攻击和重击造成暴击时，恢复3点元素能量。
- "盛宴的威严"：元素爆发持续时间内，林尼的火元素伤害提升20%。
