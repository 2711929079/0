# 空月升起

漫天星光也无法从黯色迷雾的引力下逃脱，为了赶在这颗星球也被吞噬前找到答案，始源之龙决心独自踏上苦旅[2]。他用指爪细捻白垠的游空之土，自后世称为挪德卡莱的土地升起空月[3]，以代其施行周流运作的权能，牵引原初天球，自己则前往世界之外，消失在星间迷雾中。三轮空月分别为最大的「恒月」、最亮的「霜月」，以及最神秘的「虹月」，并各由一位女神执掌（详见三月）。光焰的主人（火龙王）曾率部下造访居月的意志们在三个月面的玉座，那时王位已然空荡，天穹之上投下阴霾，各龙主任意而行。光焰的主人曾希冀从她们那求得踏入星间迷雾的王的踪迹，但答复他的，唯有三座宫殿之内，化作了三重的沉默不语。自那以后，即使是万里的雪原化作焦土，焰山被极寒冰封，她们也未曾再开口，或不能。[4]
天理到来一个又一个千年过去，月的三姐妹始终没有等来她们的旧主，直到那一日，身披羽翼的第一降临者（三月女神称其为「高天的主宰」，「天理」为葬火之战后世人们所称）踏着晨星而来。那并非预言中将要熄灭群星的大敌，似乎也不打算掠走维系世界存续的泉源，因此三月女神并不排斥其到来，双方相安无事。「高天的主宰」逐渐开始对世界显露出改造与支配的欲望，它的规则中存在一种龙王不曾教给她们的「秩序」，「恒月」艾莉亚和「霜月」卡侬想要知道这种秩序对提瓦特来说是好是坏，而「虹月」桑娜妲对此持反对意见[3]，代行星球意志的三位守护者不知所从，争论不休。但无论臣服抑或反叛，战斗还是死亡，万军之王都将无情碾过阻碍它的一切。这是它为人之子选定的新世界，依循它的规划大地与天空皆会焕然一新。在白日的征服告终之前，夜的侍者唯有静观其变。[2]
