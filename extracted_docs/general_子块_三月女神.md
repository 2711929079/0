# 三月女神

当时的部分文明（包括鹤观前文明[18]）除了崇拜天空岛，也崇拜「三月女神」。她们分别是艾莉亚（掌管最大的「恒月」）、桑娜妲（掌管最神秘的虹月）、卡侬（掌管最亮的霜月）[19]。三月女神每到夜晚，就离开珍珠色的宫殿，行走在沙漠里。月莲在她们的脚边开放。[20]
蒙德.沙尔芬德尼尔蒙德大地被冰雪覆盖，生存条件极其恶劣。一位名叫法鲁希的男性带领族人来到了当时的雪山地区，出乎意料的是，这里并没有被风雪覆盖，而是保持着自然的苍绿葱翠，于是他们在此建立了王城沙尔·芬德尼尔。[21]芬德尼尔的居民信仰天空岛，并会与神使沟通[22][23][24]。
