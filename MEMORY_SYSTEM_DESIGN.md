# 荧的记忆系统设计

## 1. 系统概述

为《原神》角色"荧"设计的记忆系统，使AI能够记住用户的偏好、对话历史和重要信息，提供更加个性化和连贯的对话体验。

## 2. 记忆类型设计

### 2.1 短期记忆（Short-term Memory）
- **定义**：当前对话会话的临时记忆
- **存储内容**：用户输入和助手回复的完整对话历史
- **生命周期**：会话期间有效，会话结束后可选择性保留
- **容量限制**：最多保存50条对话记录
- **存储方式**：内存存储

### 2.2 长期记忆（Long-term Memory）
- **定义**：需要持久化存储的重要信息
- **存储内容**：用户偏好、重要事件、角色相关信息等
- **生命周期**：永久保存（除非手动清除）
- **存储方式**：JSON文件持久化存储
